## Decisions Made

| Decision | Rationale | Impact |
| --- | --- | --- |
| Skip-link global no `<body>` antes do Header | WCAG 2.4.1 Bypass Blocks; primeiro elemento focável precisa pular header repetitivo | `app/layout.tsx`, `app/globals.css` |
| `<main id="conteudo" tabIndex={-1}>` como alvo do skip-link | tabIndex=-1 permite foco programático sem entrar na ordem de Tab | `app/layout.tsx` |
| AudioReader não como subcomponente novo, mas via prop `audioTexto` em `<Equation>` | Reusa o AudioReader já existente da `EquacaoCanonica`; opt-in por equação evita ruído | `Equation.tsx`, MDX |
| PayoffChart: `<title>`+`<desc>`+`.sr-only` data list, NÃO data table | Lista narra os pontos-chave (S, K, prêmio, BE, max) sem inflar DOM; SVG `aria-labelledby` liga title | `PayoffChart.tsx` |
| DuasPortas: WAI-ARIA Tabs Pattern completo (setas/Home/End/foco circular), sem aria-live | Foco do tabpanel sinaliza mudança ao SR; aria-live duplica anúncio | `DuasPortas.tsx` |
| Tabelas MDX: customizar só `th` (scope=col default) via `useMDXComponents` | Mínimo invasivo; GFM gera `<th>` puro e `scope` é o que faz a diferença pra SR | `mdx-components.tsx` |

## Patterns Discovered

- **Áudio narrado para fórmulas** (Web Speech API + texto PT-BR pré-escrito) é a alternativa pragmática a MathML acessível — KaTeX já emite MathML mas leitor de tela varia muito; texto controlado pelo autor é determinístico.
- **SVG acessível**: `role="img"` + `<title>` + `<desc>` + `aria-labelledby="caption-id title-id"` permite combinar legenda visível com título técnico — ambos são lidos.
- **`.sr-only` para resumo numérico** abaixo de gráficos: o gráfico permanece visual, mas usuário com SR recebe os dados estruturados sem precisar de view alternativa.

## Gotchas

- **`prefers-reduced-motion`** precisa do `!important` global (`*, *::before, *::after`) pra suplantar transitions inline e do Tailwind.
- **`tabIndex={ativo ? 0 : -1}`** no roving tabindex das tabs — sem isso, Tab leva o foco a cada pill sequencialmente em vez de pular pro tabpanel.
- **`requestAnimationFrame`** ao mudar tab via teclado — `setIdade` é assíncrono; foco precisa esperar o re-render.
