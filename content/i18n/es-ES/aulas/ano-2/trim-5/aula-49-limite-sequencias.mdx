<Porta nivel="formal" titulo="Sucesiones, convergencia y teoremas">

<Definicao titulo="Límite de sucesión (ε-N)">
$(a_n) \to L$ si: $\forall \eps > 0, \exists N \in \mathbb{N} : n \geq N \Rightarrow |a_n - L| < \eps$.
</Definicao>

### Teoremas clave

| Teorema | Enunciado |
|---|---|
| **Unicidad** | El límite, si existe, es único |
| **Aritmética** | $\lim(a_n \pm b_n) = \lim a_n \pm \lim b_n$, etc. |
| **Sándwich** | $a_n \leq b_n \leq c_n$, $\lim a_n = \lim c_n = L \Rightarrow \lim b_n = L$ |
| **Monótona acotada** | Creciente acotada superiormente converge |
| **Bolzano-Weierstrass** | Sucesión acotada tiene subsucesión convergente |
| **Cauchy** | $a_n$ converge $\iff$ es de Cauchy |

### Cauchy

<Definicao titulo="Sucesión de Cauchy">
$(a_n)$ es de **Cauchy** si $\forall \eps > 0, \exists N : m, n \geq N \Rightarrow |a_m - a_n| < \eps$.
</Definicao>

**En $\mathbb{R}$**: Cauchy $\iff$ convergente (completitud).
**En $\mathbb{Q}$**: no toda Cauchy converge ($a_n$ aproximando $\sqrt 2$).

### Monotonía + acotación

- $(a_n)$ creciente y acotada superiormente $\Rightarrow$ converge a $\sup a_n$.
- $(a_n)$ decreciente y acotada inferiormente $\Rightarrow$ converge a $\inf a_n$.

### Subsucesiones

$(a_{n_k})$ es subsucesión de $(a_n)$ si $n_1 < n_2 < \ldots$.
$\lim a_n = L \Rightarrow \lim a_{n_k} = L$ para toda subsucesión.

### Sucesiones divergentes

- $a_n = (-1)^n$: oscila entre $-1$ y $1$. Las subsucesiones convergen (a $\pm 1$), pero $a_n$ no.
- $a_n = n$: crece sin límite ($\to +\infty$).
- $a_n = n \sin(n)$: comportamiento irregular, sin límite.

### Sucesiones recursivas

$a_{n+1} = f(a_n)$, $a_0$ dado. El límite, si existe, es **punto fijo** de $f$: $L = f(L)$.

</Porta>

<Porta nivel="5" titulo="Para un niño de 5 años">

Imagina una fila de números: $1, 1/2, 1/3, 1/4, \ldots$: van **disminuyendo** y se acercan a **cero**.

El límite es la "**adónde está yendo**" de la fila. Si la fila va a un sitio, ese sitio es el límite.

Algunas filas no van a ninguna parte (siguen saltando), y entonces no hay límite.

</Porta>

<Porta nivel="10" titulo="Para un niño de 10 años">

**Sucesión** = lista de números: $a_1, a_2, a_3, \ldots$

**Límite** $L$: el número al que la sucesión **llega en el infinito**. Por ejemplo:
- $1, 1/2, 1/3, \ldots$ → va a **cero**.
- $1, 4, 9, 16, \ldots$ → va a **$\infty$** (crece).
- $1, -1, 1, -1, \ldots$ → no va a ninguna parte (alternando).

**Definición rigurosa**: $a_n \to L$ si consigues quedar **tan cerca de $L$ como quieras**, basta con avanzar mucho en la sucesión. Para cualquier "burbuja" $\eps$ alrededor de $L$, hay un $N$ tal que **todos** los términos después de $a_N$ están en la burbuja.

**Truco útil**: si $a_n$ siempre está **creciendo** y tiene un **techo**, ¡**converge**! (Monótona acotada.) Ni siquiera hay que saber el límite, solo saber que existe.

</Porta>

<Porta nivel="15" titulo="Para un adolescente de 15 años">

Las sucesiones son **funciones con dominio en $\mathbb{N}$**. La teoría del límite es la misma de la Lección 41 ε-δ, pero usamos $N$ en lugar de $\delta$.

### Por qué importan las sucesiones

- **Iteración de Newton** para raíces: $a_{n+1} = a_n - f(a_n)/f'(a_n)$.
- **Series** = límite de sumas parciales: $\sum a_n = \lim s_N$.
- **Integral** = límite de sumas de Riemann.
- **Probabilidad**: ley de los grandes números $\bar X_n \to E[X]$.

### Límites notables de sucesiones

| Sucesión | Límite |
|---|---|
| $1/n$ | $0$ |
| $(1 + 1/n)^n$ | $e$ |
| $n^{1/n}$ | $1$ |
| $n!/n^n$ | $0$ |
| $a^n$ ($|a| < 1$) | $0$ |
| $a^n$ ($a > 1$) | $\infty$ |
| $(-1)^n$ | no existe |

### Jerarquía de crecimiento (¡importante!)

$$\log n \ll n^a \ll a^n \ll n! \ll n^n.$$

Cada uno crece **mucho más despacio** que el siguiente. Herramienta usada todo el tiempo en complejidad de algoritmos.

### Teoremas operacionales

**Sándwich**: comprimir $b_n$ entre $a_n$ y $c_n$ que van al mismo sitio.

**Ejemplo**: $\lim_{n \to \infty} \sin n / n$. $-1/n \leq \sin n / n \leq 1/n$. Por sándwich, $\to 0$.

**Monótona acotada**: una sucesión creciente y acotada converge: ¡no hace falta hallar el límite, solo saber que existe!

**Ejemplo**: $a_1 = 1$, $a_{n+1} = (a_n + 2/a_n)/2$. Se demuestra que $a_n$ es decreciente tras $a_1$, acotada inferiormente por $\sqrt 2$. Luego converge. El límite $L$ satisface $L = (L + 2/L)/2$, es decir, $L = \sqrt 2$.

### Bolzano-Weierstrass

**Toda sucesión acotada tiene subsucesión convergente**. Útil para argumentos de existencia (compacidad).

</Porta>

<Porta nivel="25" titulo="Para un estudiante de ingeniería">

Las sucesiones son **la base de algoritmos iterativos**: Newton, gradient descent, EM, MCMC.

### Demostración: $\lim 1/n = 0$ vía ε-N

Dado $\eps > 0$, elige $N = \lceil 1/\eps \rceil + 1$. Entonces $n \geq N \Rightarrow 1/n \leq 1/N < \eps$. ∎

### Demostración: monótona acotada converge

Sea $(a_n)$ creciente, acotada superiormente. Toma $L = \sup\{a_n\}$ (existe por completitud). Para $\eps > 0$, $L - \eps$ no es cota superior, luego $\exists N : a_N > L - \eps$. Como creciente, $a_n \geq a_N > L - \eps$ para $n \geq N$. Como $a_n \leq L$, vale $|a_n - L| < \eps$. ∎

### Demostración: Bolzano-Weierstrass (esbozo)

Sucesión acotada $(a_n) \subseteq [a, b]$. Bisecta: alguna mitad contiene infinitos términos. Repite. Construye subsucesión $(a_{n_k})$ con $a_{n_k}$ en la mitad $k$-ésima, intervalo de longitud $(b-a)/2^k$. Cauchy, luego convergente. ∎

### Demostración: Cauchy ⇔ convergente en $\mathbb{R}$

$(\Rightarrow)$ Convergente es Cauchy: $|a_m - a_n| \leq |a_m - L| + |L - a_n| < \eps$.

$(\Leftarrow)$ Cauchy es convergente: Cauchy $\Rightarrow$ acotada $\Rightarrow$ tiene subsucesión convergente (Bolzano-Weierstrass) $\Rightarrow$ la sucesión entera converge al mismo límite (porque es Cauchy).

### Sucesiones de Cauchy en $\mathbb{Q}$

$(a_n)$ aproximando $\sqrt 2$ por bisección es Cauchy en $\mathbb{Q}$, pero **no converge** en $\mathbb{Q}$ (el límite es irracional). $\mathbb{R}$ es la **completación** de $\mathbb{Q}$ por la métrica $|\cdot|$.

### Aplicaciones en algoritmos

- **Gradient descent**: $w_{n+1} = w_n - \eta \nabla f(w_n)$. Converge si $\eta < 2/L$ (Lipschitz constant), $f$ convexa.
- **Newton**: $x_{n+1} = x_n - f(x_n)/f'(x_n)$. Convergencia **cuadrática** cerca de raíz simple.
- **Heron** (raíz cuadrada): $a_{n+1} = (a_n + N/a_n)/2$. Cuadrática para $\sqrt N$. Viene de Newton en $f(x) = x^2 - N$.
- **EM**: aumenta monotónicamente la likelihood. Converge a punto crítico (no necesariamente máx. global).

### Convergencia de series

$\sum a_n$ converge $\iff$ las sumas parciales $s_N = \sum_{n=1}^N a_n$ forman sucesión convergente.

**Criterios**:
- Cauchy: $\forall \eps, \exists N : m > n \geq N \Rightarrow |s_m - s_n| < \eps$.
- Razón (D'Alembert): $\lim |a_{n+1}/a_n| = L < 1 \Rightarrow$ converge.
- Raíz (Cauchy): $\lim |a_n|^{1/n} = L < 1 \Rightarrow$ converge.

### Ley de los grandes números

$X_1, X_2, \ldots$ iid con $E[X] = \mu$, $V[X] < \infty$. Entonces $\bar X_n = (1/n) \sum X_i \to \mu$ (en probabilidad: débil; ae: fuerte).

### Aplicaciones en sistemas dinámicos

Sistema discreto $x_{n+1} = T(x_n)$. Comportamiento: punto fijo, ciclo, atractor extraño (caos). El **mapa logístico** $x_{n+1} = r x_n (1 - x_n)$ exhibe transición al caos en $r \approx 3{,}57$.

### Sucesiones en $\mathbb{R}^n$

Convergencia componente a componente. En norma: $\|x_n - L\| \to 0$. Equivalente en $\mathbb{R}^n$ (pero no en dimensión infinita).

</Porta>

<Porta nivel="40" titulo="Para un profesional sénior">

Las sucesiones son centrales en **análisis funcional**, **espacios métricos**, **topología**.

### Espacios de sucesiones

- $\ell^p$ = sucesiones con $\sum |a_n|^p < \infty$. Banach con norma $\|a\|_p = (\sum |a_n|^p)^{1/p}$.
- $\ell^\infty$ = sucesiones acotadas. Banach con $\|a\|_\infty = \sup |a_n|$.
- $c_0$ = sucesiones $a_n \to 0$. Subespacio cerrado de $\ell^\infty$.

### Convergencia débil vs fuerte

- **Fuerte**: $\|x_n - x\| \to 0$.
- **Débil**: $f(x_n) \to f(x)$ para todo $f \in X^*$ (dual).

En dimensión finita, equivalentes. En $\ell^2$: la base ortonormal $e_n$ converge débilmente a 0 pero no fuertemente ($\|e_n\| = 1$).

### Cauchy en espacios métricos

$(X, d)$ es **completo** si toda Cauchy converge. Banach = espacio normado completo. Hilbert = inner product space completo.

**Completación**: todo espacio métrico tiene completación única. $\mathbb{R}$ es completación de $\mathbb{Q}$. $L^2$ es completación de $C[a,b]$ por la norma $L^2$.

### Compacidad

En espacios métricos: **secuencialmente compacto** = toda sucesión tiene subsucesión convergente. En $\mathbb{R}^n$, equivalente a cerrado y acotado (Heine-Borel). **En dimensión infinita**, equivalente a cerrado, acotado **y** totalmente acotado (Riesz).

### Topología

Caracterización secuencial de continuidad vale en **espacios métricos** (Heine). En general, requiere redes o filtros.

### Operadores compactos

$T : X \to Y$ es **compacto** si mapea la bola unidad en un conjunto precompacto. Las subsucesiones convergentes son clave en demostraciones.

### Teoría espectral

Sucesión de autovalores $\lambda_n$ de operador compacto autoadjunto: o finita, o $\lambda_n \to 0$. Spectral theorem descompone vía base de autovectores.

### Métodos iterativos avanzados

- **Aceleración de Aitken**: $\hat a_n = a_n - (a_{n+1} - a_n)^2/(a_{n+2} - 2 a_{n+1} + a_n)$. Acelera convergencia lineal.
- **Romberg integration**: extrapolación de Richardson en integración.
- **GMRES, Krylov subspaces**: para sistemas lineales grandes.

### Aplicaciones en ML / RL

- **Q-learning convergence**: $Q_n \to Q^*$ bajo exploración $\epsilon$-greedy adecuada.
- **Policy gradient convergence**: análisis vía aproximación estocástica (Robbins-Monro).
- **MCMC**: sucesión de estados $\to$ distribución estacionaria. Convergencia vía teoremas ergódicos.

### Categorías

Los límites en categorías generalizan los límites de sucesiones. Direct/inverse limit, colimit, etc. Misma estructura algebraica.

</Porta>

<Porta nivel="pratica" titulo="Aplicación práctica">

**Receta: cómo calcular el límite de una sucesión.**

### Algoritmo

1. **Sustitución directa**: si $a_n = f(n)$ con $f$ función, calcula $\lim_{x \to \infty} f(x)$.
2. **Recursiva**: asume límite $L$, resuelve $L = f(L)$.
3. **Aritmética**: separa en partes conocidas.
4. **Sándwich**: comprime entre conocidos.
5. **Monótona acotada**: prueba la existencia, calcula después (o ni siquiera calcules).

### SymPy

```python
import sympy as sp
n = sp.symbols('n', positive=True, integer=True)
print(sp.limit(1/n, n, sp.oo))                # 0
print(sp.limit((1 + 1/n)**n, n, sp.oo))       # E
print(sp.limit(n**(1/n), n, sp.oo))           # 1
print(sp.limit(sp.factorial(n)/n**n, n, sp.oo))  # 0
```

### Jerarquía de crecimiento (en la punta de la lengua)

$$\log n \ll n^a \ll a^n \ll n! \ll n^n.$$

Úsala para resolver razones: $\lim n!/2^n = \infty$, $\lim 2^n/n! = 0$, $\lim n^{100}/2^n = 0$.

### Sucesiones recursivas

Para $a_{n+1} = f(a_n)$:

1. Verifica la convergencia (monotonía + acotación, o contracción vía Banach).
2. Si converge a $L$, entonces $L = f(L)$.

```python
import numpy as np
def heron(N, a0=1, n_iter=10):
    a = a0
    for _ in range(n_iter):
        a = (a + N/a)/2
    return a
print(heron(2))   # ~1.41421356 (sqrt 2)
```

### Cuando el límite NO existe

- **Oscilación**: $(-1)^n$, $\sin n$.
- **Divergencia**: $a_n \to \pm \infty$.
- **No monótona y no Cauchy**: caos.

### Verificación numérica

Implementar y graficar $a_n$ vs $n$ es la forma más rápida de **conjeturar** el límite. Después, demuéstralo.

### Cuidados

- Un límite recursivo presupone **existencia**. Sin demostración, puede ser falaz.
- $(1 + 1/n)^n \neq 1$ en el infinito! Indeterminación $1^\infty$.
- Una sucesión **eventualmente creciente** puede no ser monótona: cuidado en demostraciones.
