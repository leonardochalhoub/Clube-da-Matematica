### Límites trigonométricos fundamentales

| Límite | Valor |
|---|---|
| $\lim_{x \to 0} \sin x / x$ | $1$ |
| $\lim_{x \to 0} \sin(kx)/x$ | $k$ |
| $\lim_{x \to 0} \tan x / x$ | $1$ |
| $\lim_{x \to 0} (1 - \cos x)/x$ | $0$ |
| $\lim_{x \to 0} (1 - \cos x)/x^2$ | $1/2$ |
| $\lim_{x \to 0} \sin^2 x / x^2$ | $1$ |
| $\lim_{x \to 0} \arcsin x / x$ | $1$ |
| $\lim_{x \to 0} \arctan x / x$ | $1$ |

### Identidades útiles

- $1 - \cos x = 2\sin^2(x/2)$
- $\sin(a + b) = \sin a \cos b + \cos a \sin b$
- $\cos(a + b) = \cos a \cos b - \sin a \sin b$
- $1 - \cos x = (1 - \cos^2 x)/(1 + \cos x) = \sin^2 x / (1 + \cos x)$

### Técnicas

1. **Sustitución $u$**: $\sin(2x)/x = 2 \cdot \sin(2x)/(2x) \to 2$.
2. **Identidades** para reescribir en forma fundamental.
3. **Conjugado**: $1 - \cos x = \sin^2 x / (1 + \cos x)$.
4. **Sándwich**: $-1 \leq \sin x \leq 1$, multiplica por algo $\to 0$.

### Demostración de $\lim (1 - \cos x)/x^2 = 1/2$

$\frac{1 - \cos x}{x^2} = \frac{2 \sin^2(x/2)}{x^2} = \frac{1}{2} \cdot \frac{\sin^2(x/2)}{(x/2)^2} \to \frac{1}{2} \cdot 1 = \frac{1}{2}$.

### Demostración de $\lim \tan x / x = 1$

$\tan x / x = (\sin x / x) \cdot (1 / \cos x) \to 1 \cdot 1 = 1$.

<Porta nivel="5" titulo="Para un niño de 5 años">

Cuando el ángulo es **muy pequeño**, su **seno** es casi igual al propio ángulo. Tipo, $\sin 0{,}01 \approx 0{,}01$.

Por eso $\sin x / x \to 1$: se vuelven casi iguales.

Y **uno menos coseno** es casi igual a **la mitad del cuadrado del ángulo**. Las cosas pequeñitas siguen reglas simples.

</Porta>

<Porta nivel="10" titulo="Para un niño de 10 años">

Para **ángulos pequeños** (en radianes):
- $\sin x \approx x$
- $\cos x \approx 1 - x^2/2$
- $\tan x \approx x$

De ahí salen todos los límites:
- $\sin(3x)/x \to 3$ (porque $\sin(3x) \approx 3x$)
- $(1 - \cos x)/x^2 \to 1/2$ (porque $1 - \cos x \approx x^2/2$)
- $\tan x / x \to 1$ (porque $\tan x \approx x$)

**Aplicación en física**: péndulo de pequeñas oscilaciones. La ecuación real es $\ddot \theta = -(g/L) \sin \theta$. Para $\theta$ pequeño, $\sin \theta \approx \theta$, y queda $\ddot \theta = -(g/L) \theta$: oscilador simple.

**Sin esa aproximación**, el problema se vuelve muchísimo más difícil (involucra integrales elípticas). Por eso los ingenieros aman los pequeños ángulos.

</Porta>

<Porta nivel="15" titulo="Para un adolescente de 15 años">

Los límites trig con $x \to 0$ se reducen a tres casos vía manipulación:

### Caso 1: $\sin(?)/?$

Si consigues escribirlo en la forma $\sin(\text{algo})/\text{algo}$, va a 1. Usa cambio de variable $u = $ argumento.

**Ejemplo**: $\lim_{x \to 0} \sin(7x)/(3x) = (7/3) \lim_{x \to 0} \sin(7x)/(7x) = 7/3$.

### Caso 2: $(1 - \cos)/?$

Usa la identidad $1 - \cos x = 2 \sin^2(x/2)$ o conjugado.

**Ejemplo**: $\lim_{x \to 0} (1 - \cos x)/(x \sin x) = (x^2/2)/(x \cdot x) = 1/2$ (usando $\sin x \approx x$).

### Caso 3: combinaciones $\tan, \sec, \cot$

Reescribe en $\sin, \cos$.

**Ejemplo**: $\lim_{x \to 0} \tan(2x)/\sin(3x) = \lim (2x)/(3x) = 2/3$ (aproximación).

### Trucos de cambio de variable

Para $\lim_{x \to a}$ trig, usa $u = x - a$ para llevarlo a cero.

**Ejemplo**: $\lim_{x \to \pi} \sin x / (x - \pi)$. Sea $u = x - \pi$. Entonces $\sin x = \sin(u + \pi) = -\sin u$. Resultado: $\lim_{u \to 0} -\sin u / u = -1$.

### Tabla de aproximaciones para $x$ pequeño

| Función | Aproximación (hasta $x^3$) |
|---|---|
| $\sin x$ | $x - x^3/6$ |
| $\cos x$ | $1 - x^2/2$ |
| $\tan x$ | $x + x^3/3$ |
| $\arcsin x$ | $x + x^3/6$ |
| $\arctan x$ | $x - x^3/3$ |

Úsala para resolver más rápido.

### Ejemplo desafiante

$\lim_{x \to 0} (\tan x - \sin x)/x^3$:

$\tan x - \sin x = \sin x (1/\cos x - 1) = \sin x \cdot (1 - \cos x)/\cos x$.

Por tanto: $\lim = (\sin x / x) \cdot ((1 - \cos x)/x^2) \cdot (1/\cos x) = 1 \cdot (1/2) \cdot 1 = 1/2$.

</Porta>

<Porta nivel="25" titulo="Para un estudiante de ingeniería">

Los límites trigonométricos son esenciales en **derivadas** ($\sin', \cos', \tan'$), **series de Fourier**, **análisis armónico**.

### Demostraciones clave

**$\lim \sin x / x = 1$**: argumento geométrico: área de triángulo, sector, triángulo. Por sándwich, se concluye.

**$(1 - \cos x)/x^2 \to 1/2$**:
$$\frac{1 - \cos x}{x^2} = \frac{(1-\cos x)(1+\cos x)}{x^2 (1+\cos x)} = \frac{\sin^2 x}{x^2(1+\cos x)} = \frac{\sin^2 x}{x^2} \cdot \frac{1}{1+\cos x} \to 1 \cdot \frac{1}{2} = \frac{1}{2}.$$

**$\arcsin x / x \to 1$**: $u = \arcsin x$, $x = \sin u$, $u \to 0$ cuando $x \to 0$. $\arcsin x / x = u/\sin u \to 1$.

### Conexión con derivadas

- $(\sin x)' = \cos x$ viene de $\lim_{h \to 0} (\sin(x+h) - \sin x)/h$. La manipulación usa $\sin(x+h) = \sin x \cos h + \cos x \sin h$, entonces la razón $= \sin x (\cos h - 1)/h + \cos x \sin h / h \to 0 + \cos x = \cos x$.
- $(\cos x)' = -\sin x$ análogo.
- $(\tan x)' = \sec^2 x$ vía cociente.

### Series de Taylor (Trimestre 7)

$$\sin x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n+1}}{(2n+1)!}, \quad \cos x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n}}{(2n)!}.$$

Truncadas dan aproximaciones de pequeños ángulos. Convergencia uniforme en compactos.

### Aplicaciones en ingeniería

- **Pequeños ángulos en mecánica**: péndulo $T = 2\pi\sqrt{L/g}$ vale para $\theta \lesssim 10°$. El error relativo crece como $\theta^2/16$.
- **Óptica gaussiana** (paraxial): $\sin \theta \approx \theta$. Lentes finas, refracción.
- **Linealización**: sistemas dinámicos $\dot x = f(x)$ linealizados vía jacobiano en punto de equilibrio.
- **Modulación**: $\cos(\omega t + \phi(t))$ con $\phi$ pequeño expandido para AM-equivalente.

### Análisis de Fourier

Coeficientes de Fourier $a_n = \frac{1}{T}\int f(t) \cos(2\pi nt/T) dt$. Límite $\sum a_n^2$ converge (Parseval). Los límites trig aparecen en verificaciones de convergencia punto a punto (Dini, Dirichlet).

### Sándwich avanzado

$\lim_{x \to \infty} \sin x$ no existe, pero $\lim_{x \to \infty} (\sin x)/x = 0$ por sándwich $-1/x \leq \sin x/x \leq 1/x$.

$\lim_{x \to 0} x \sin(1/x) = 0$ por sándwich. **No** es lo mismo que $\lim x \cdot \lim \sin(1/x)$: el segundo ni siquiera existe. El sándwich resuelve sin esa factorización.

### En circuitos AC

- Impedancia $Z(j\omega) = R + jX$. Magnitud $|Z| = \sqrt{R^2 + X^2}$. Límites en pequeña frec./alta frec.
- $\sin(\omega t)$ en $\omega = 0$: va a cero. En $\omega \to \infty$: oscila, sin límite.

<Porta nivel="40" titulo="Para un profesional sénior">

Los límites trigonométricos son manifestaciones de **estructura analítica** profunda: $\sin, \cos$ son funciones enteras con patrón específico de ceros.

### Factorización de Weierstrass

$$\sin(\pi x) = \pi x \prod_{n=1}^\infty \left(1 - \frac{x^2}{n^2}\right).$$

Tomando logaritmo y derivando, da la identidad de Mittag-Leffler:
$$\pi \cot(\pi x) = \frac{1}{x} + \sum_{n=1}^\infty \frac{2x}{x^2 - n^2}.$$

Conexión con $\zeta(2k)$.

### Caracterización funcional

$\sin, \cos$ son las únicas funciones $\mathbb{R} \to \mathbb{R}$ que satisfacen:
- $f(x+y) = f(x)g(y) + g(x)f(y)$, $g(x+y) = g(x)g(y) - f(x)f(y)$
- $f'(0) = 1$, $g(0) = 1$
- $f, g$ continuas

Equivalente: solución de $y'' + y = 0$ con condiciones iniciales.

### Análisis complejo

$e^{ix} = \cos x + i \sin x$ (Euler). Los límites trig se vuelven límites de $e^{iz}$ en $z \to 0$. Holomorfía en $\mathbb{C}$ entero. Factorización de Hadamard.

### Aplicaciones en PDE

- **Ecuación de la onda** $u_{tt} = c^2 u_{xx}$: solución $u(x, t) = f(x - ct) + g(x + ct)$ involucra $\sin, \cos$ vía separación de variables.
- **Schrödinger libre**: $\psi(x, t) = \int A(k) e^{i(kx - \omega t)} dk$, $\omega = k^2/2m$. Paquetes vía $\sin, \cos$.
- **Calor con BCs Dirichlet**: solución $\sum b_n \sin(n\pi x/L) e^{-n^2 \pi^2 D t/L^2}$.

### Análisis armónico

Teorema de Carleson: la serie de Fourier converge ae para $f \in L^2$. Demostración depende de estimaciones finas que involucran límites trig.

### Aproximación de Padé

Para $\sin x$, el aproximante Padé $[m/n]$ puede ser mejor que Taylor. En particular, $\tan x \approx x(1 - x^2/15)/(1 - 2x^2/5)$ tiene error menor que Taylor truncada del mismo grado.

### Cálculo de operadores

- **Función de matriz**: $\sin A = (e^{iA} - e^{-iA})/(2i)$. Bien definida para cualquier matriz.
- **Spectral mapping**: spec($\sin A$) = $\{\sin \lambda : \lambda \in $ spec $A\}$.
- **Aplicación**: la solución de $\ddot x = -\Omega^2 x$ es $x(t) = \cos(\Omega t) x_0 + \Omega^{-1} \sin(\Omega t) v_0$.

### Numérico

- **Argument reduction**: para $\sin(10^{15})$, reduce mod $\pi$ con **Payne-Hanek** (alta precisión).
- **CORDIC**: hardware-friendly, calcula $\sin, \cos$ por iteración de rotaciones.
- **Cancelación numérica**: $1 - \cos x$ en float32 para $x$ pequeño tiene error relativo grande. Usar `2 * sin(x/2)**2` en su lugar.

<Porta nivel="pratica" titulo="Aplicación práctica">

**Receta: límites trigonométricos.**

### Algoritmo

1. **Identifica la forma**: $\sin/?, \cos$, $(1 - \cos)/?, \tan/?$, etc.
2. **Reescribe** en forma fundamental vía:
   - Cambio de variable.
   - Identidades $\sin^2 + \cos^2 = 1$, $1 - \cos x = 2\sin^2(x/2)$, etc.
3. **Comprueba numéricamente** o vía SymPy.

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
# Lineal en el argumento
print(sp.limit(sp.sin(7*x)/x, x, 0))            # 7
# Razón de senos
print(sp.limit(sp.sin(3*x)/sp.sin(5*x), x, 0))  # 3/5
# (1 - cos)/x^2
print(sp.limit((1 - sp.cos(x))/x**2, x, 0))     # 1/2
# Difíciles
print(sp.limit((sp.tan(x) - sp.sin(x))/x**3, x, 0))  # 1/2
```

### Cheat sheet

Para $x \to 0$:

| Forma | Límite |
|---|---|
| $\sin(ax)/x$ | $a$ |
| $\sin(ax)/\sin(bx)$ | $a/b$ |
| $\sin(ax)/\tan(bx)$ | $a/b$ |
| $(1 - \cos(ax))/x^2$ | $a^2/2$ |
| $\arcsin(ax)/x$ | $a$ |
| $\arctan(ax)/x$ | $a$ |
| $(\sin x)^2/x^2$ | 1 |
| $\sin(\sin x)/x$ | 1 |

### Cuándo NO usar estas

- Límite involucra $x \to a \neq 0$: haz cambio $u = x - a$.
- Función con $1/x$ en el argumento ($\sin(1/x)$): aprovecha que oscila: sándwich.

### Cuidados comunes

- $\sin(2x)/x \neq 2 \sin x / x$: debe ser $2 \cdot \sin(2x)/(2x) \to 2$.
- $\sin(x^2)/x \neq \sin x \cdot x$: debe ser $\sin(x^2)/x = x \cdot \sin(x^2)/x^2 \to 0$.
- $(1 - \cos x)/x \neq (1 - \cos x)/x^2$: el primero va a 0, el segundo a 1/2.
