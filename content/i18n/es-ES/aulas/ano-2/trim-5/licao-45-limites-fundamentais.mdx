<Porta nivel="formal" titulo="Límites fundamentales y técnicas">

### Los 3 fundamentales

1. $\displaystyle\lim_{x \to 0} \frac{\sin x}{x} = 1$ (geométrico, sándwich).
2. $\displaystyle\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$ (define $e$).
3. $\displaystyle\lim_{x \to 0} \frac{e^x - 1}{x} = 1$ (deriva del anterior).

### Variantes útiles (Tabla)

| Límite | Valor |
|---|---|
| $\lim_{x \to 0} \sin x / x$ | $1$ |
| $\lim_{x \to 0} \sin(kx)/x$ | $k$ |
| $\lim_{x \to 0} \tan x / x$ | $1$ |
| $\lim_{x \to 0} (1-\cos x)/x^2$ | $1/2$ |
| $\lim_{x \to 0} (1-\cos x)/x$ | $0$ |
| $\lim_{x \to 0} \arcsin x / x$ | $1$ |
| $\lim_{x \to 0} \arctan x / x$ | $1$ |
| $\lim_{x \to 0} (e^x - 1)/x$ | $1$ |
| $\lim_{x \to 0} (a^x - 1)/x$ | $\ln a$ |
| $\lim_{x \to 0} \ln(1+x)/x$ | $1$ |
| $\lim_{x \to 0} ((1+x)^a - 1)/x$ | $a$ |
| $\lim_{x \to \infty} (1 + a/x)^x$ | $e^a$ |
| $\lim_{x \to 0^+} x^x$ | $1$ |
| $\lim_{x \to 0^+} x \ln x$ | $0$ |

### Indeterminaciones comunes (7)

$0/0$, $\infty/\infty$, $\infty - \infty$, $0 \cdot \infty$, $1^\infty$, $0^0$, $\infty^0$: todas resolubles por manipulación o L'Hôpital (Lección 64).

### Técnicas

- **Factorizar y cancelar** ($0/0$ polinómica).
- **Conjugado** ($0/0$ con raíces).
- **Trigonométrica**: usar $\sin x \approx x$ ($x$ pequeño).
- **Cambio de variable**: $u = 1/x$, $u = x - a$.
- **Logaritmo**: $A^B = e^{B \ln A}$ convierte $1^\infty$, $0^0$ en $0/0$.

<Porta nivel="5" titulo="Para un niño de 5 años">

Existen tres límites **mágicos** que aparecen todo el tiempo:

1. **$\sin x / x$** se va a **uno** cuando $x$ es pequeño (porque para ángulos pequeñitos, el seno y el ángulo son casi iguales).
2. **$(1 + 1/x)^x$** se va a un número especial llamado **$e$** cuando $x$ es grande (unos $2{,}71...$).
3. **$(e^x - 1)/x$** también se va a **uno** cuando $x$ es pequeño.

Esos tres son como las **tres tablas mágicas** de los límites.

</Porta>

<Porta nivel="10" titulo="Para un niño de 10 años">

Cuando tienes $0/0$ o $\infty/\infty$, el límite **puede ser cualquier cosa**: depende. Pero tres límites fundamentales siempre dan la misma respuesta:

1. **$\sin x / x \to 1$**: para ángulo pequeño, el seno es casi igual al ángulo (en radianes).
2. **$(1 + 1/n)^n \to e$**: número $e \approx 2{,}71828...$, aparece en intereses compuestos.
3. **$(e^x - 1)/x \to 1$**: deriva del anterior.

**Uso práctico**: quieres calcular $\lim_{x \to 0} \sin(5x)/x$. Reescribe: $5 \cdot \sin(5x)/(5x)$. Cuando $x \to 0$, $5x \to 0$, y $\sin(5x)/(5x) \to 1$. Respuesta: $5$.

**Truco de los intereses**: depositar con tasa $r$ por año, capitalizando $n$ veces al año: $(1 + r/n)^n$. La capitalización continua ($n \to \infty$) da $e^r$.

</Porta>

<Porta nivel="15" titulo="Para un adolescente de 15 años">

Los tres límites fundamentales son **fórmulas base** que combinas mediante álgebra para resolver todo tipo de indeterminación.

### Fórmula 1: $\lim \sin x / x = 1$

**Por qué vale**: por el círculo unitario, $\sin x < x < \tan x$ para $x \in (0, \pi/2)$. Dividiendo por $\sin x$: $1 < x/\sin x < 1/\cos x$. Por sándwich, $x/\sin x \to 1$, luego $\sin x / x \to 1$.

**Variantes**: $\lim \sin(kx)/x = k$ (cambio de variable $u = kx$).

### Fórmula 2: $\lim (1 + 1/x)^x = e$

**Por qué vale**: define el número $e \approx 2{,}71828$. Aparece en intereses compuestos continuos, decaimiento exponencial, distribución normal.

**Variantes**:
- $\lim (1 + a/x)^x = e^a$
- $\lim (1 + x)^{1/x} = e$ (con $x \to 0$)

### Fórmula 3: $\lim (e^x - 1)/x = 1$

**Por qué vale**: la derivada de $e^x$ en $x = 0$ es $e^0 = 1$. Conexión con derivada vía la definición $f'(0) = \lim (f(x) - f(0))/x$.

**Variantes**:
- $\lim (a^x - 1)/x = \ln a$
- $\lim \ln(1+x)/x = 1$
- $\lim ((1+x)^a - 1)/x = a$

### Tabla de aproximaciones para $x$ pequeño

| Función | Aproximación |
|---|---|
| $\sin x$ | $x - x^3/6$ |
| $\cos x$ | $1 - x^2/2$ |
| $\tan x$ | $x + x^3/3$ |
| $e^x$ | $1 + x + x^2/2$ |
| $\ln(1+x)$ | $x - x^2/2$ |
| $(1+x)^a$ | $1 + ax$ |

**Aplicación**: pequeños ángulos en física (péndulo, óptica), intereses continuos, decaimiento radiactivo.

</Porta>

<Porta nivel="25" titulo="Para un estudiante de ingeniería">

Los límites fundamentales son consecuencia de la **estructura analítica** de las funciones elementales: derivada y serie de Taylor.

### Demostración de $\lim \sin x / x = 1$

Argumento geométrico (Cauchy 1821):

En $x \in (0, \pi/2)$, considera el triángulo de área $\frac{1}{2}\sin x$, el sector circular de área $\frac{1}{2}x$, y el triángulo externo de área $\frac{1}{2}\tan x$.

$$\frac{\sin x}{2} < \frac{x}{2} < \frac{\tan x}{2} \Rightarrow \cos x < \frac{\sin x}{x} < 1$$

Por sándwich, $\sin x / x \to 1$ cuando $x \to 0^+$. Por la paridad, vale también $x \to 0^-$. ∎

### Demostración de $\lim (1 + 1/n)^n = e$

Se demuestra que $a_n = (1 + 1/n)^n$ es monótona creciente y acotada, luego converge. El límite **define** $e$.

Por binomio de Newton:
$$\left(1 + \frac{1}{n}\right)^n = \sum_{k=0}^n \binom{n}{k} \frac{1}{n^k} = \sum_{k=0}^n \frac{1}{k!} \cdot \frac{n!}{n^k (n-k)!}$$

Como $n \to \infty$, cada término $\to 1/k!$, y la suma va a $\sum_{k=0}^\infty 1/k! = e$.

### Demostración de $\lim (e^x - 1)/x = 1$

Usa $\lim_{x \to 0} (1 + x)^{1/x} = e$. Toma logaritmo: $\ln((1+x)^{1/x}) = \ln(1+x)/x \to 1$. Sustituyendo $y = e^x - 1$ (entonces $x = \ln(1+y)$): $(e^x - 1)/x = y/\ln(1+y) \to 1$. ∎

### Aplicaciones en ingeniería

- **Capitalización continua**: $V(T) = V_0 e^{rT}$, derivado de $\lim_n (1 + r/n)^{nT} = e^{rT}$.
- **Decaimiento radiactivo**: $N(t) = N_0 e^{-\lambda t}$: vida media $T_{1/2} = \ln 2 / \lambda$.
- **Aproximación para pequeños ángulos**: péndulo $\ddot \theta + (g/L) \sin \theta = 0$ se vuelve $\ddot \theta + (g/L) \theta = 0$ (oscilador armónico), válido hasta ~10°.
- **Estabilidad BIBO**: $\lim_{n \to \infty} h[n] = 0$ ⇒ sistema estable.
- **Frecuencia angular**: $\omega = \lim_{T \to 0} 2\pi / T$, conexión entre periodo y frecuencia.

### Aplicación a la derivada

Derivada de $e^x$ en cero:
$$\frac{d}{dx} e^x \Big|_{x=0} = \lim_{h \to 0} \frac{e^h - e^0}{h} = \lim_{h \to 0} \frac{e^h - 1}{h} = 1.$$

Por tanto $(e^x)' = e^x$ en todas partes (vía $e^{x+h} = e^x e^h$).

### Indeterminaciones por logaritmo

$\lim_{x \to 0^+} x^x$: ¿$1^\infty$ o $0^0$? Reescribe: $x^x = e^{x \ln x}$, y $x \ln x \to 0$ (demuestra por sándwich/L'Hôpital). Por tanto $x^x \to e^0 = 1$.

<Porta nivel="40" titulo="Para un profesional sénior">

Los límites fundamentales son **manifestaciones de estructuras analíticas profundas**.

### $\sin x / x$ y funciones enteras

$\sin x / x$ se extiende a una función **entera** $\text{sinc}(x)$, con serie
$$\text{sinc}(x) = \prod_{n=1}^\infty \left(1 - \frac{x^2}{n^2 \pi^2}\right).$$
Esta es la **factorización de Weierstrass** de $\sin$. Conexión con $\zeta(2) = \pi^2/6$ vía expansión.

### $e$ y el álgebra

$e = \exp(1)$, donde $\exp(z) = \sum z^n/n!$ es la única función entera con $f' = f$ y $f(0) = 1$. **Caracterización única** por ecuación diferencial.

### Generalizaciones en álgebra

- **Exponencial de matriz**: $e^A = \sum A^n/n!$. Resuelve $\dot x = Ax$.
- **Exponencial en grupos de Lie**: $\exp : \mathfrak{g} \to G$, mapeando álgebra al grupo.
- **Funciones Gamma**: $\Gamma(z) = \lim_n (n! n^z)/(z(z+1)\cdots(z+n))$ vía producto.

### Operadores y cálculo simbólico

- **Cálculo umbral / Sheffer**: la derivada $\partial$ tiene "exponencial" $e^{a\partial}$ que actúa como traslación por $a$.
- **Cálculo de Heaviside**: $D = d/dt$, $e^{-Dh}$ es traslación. Los límites fundamentales aparecen en la expansión.

### Análisis funcional

- $\sinh, \cosh, \sin, \cos$ forman **base** del espacio de soluciones de $y'' = \pm y$.
- $e^{i\omega t}$ es **autovector** del operador $d/dt$: base del análisis de Fourier.

### Demostración vía serie (rigurosa)

$\sin x = x - x^3/6 + x^5/120 - \ldots$ Entonces
$$\frac{\sin x}{x} = 1 - \frac{x^2}{6} + \frac{x^4}{120} - \ldots \to 1.$$

$e^x = 1 + x + x^2/2 + \ldots$ Entonces
$$\frac{e^x - 1}{x} = 1 + \frac{x}{2} + \frac{x^2}{6} + \ldots \to 1.$$

### Aplicaciones en probabilidad

- **Función característica**: $\varphi(t) = E[e^{itX}]$. $\varphi(0) = 1$, $\varphi'(0) = i E[X]$.
- **Ley de Poisson**: $P(X = k) = e^{-\lambda} \lambda^k / k!$. Límite de binomial $(1 + \lambda/n)^n$.
- **Fórmula de Stirling**: $n! \sim \sqrt{2\pi n}(n/e)^n$. Aproximación asintótica.

### Renormalización y física

En QFT, **regularización zeta**: $\sum_{n=1}^\infty n = \zeta(-1) = -1/12$. Los límites fundamentales aparecen en continuación analítica.

### Numérico

- Los algoritmos calculan $e^x$, $\sin x$ vía serie truncada + reducción de argumento (CORDIC).
- Cuidado con la cancelación numérica en $(e^x - 1)/x$ para $x$ pequeño: usa `expm1(x)` (built-in en C, Python, MATLAB).

</Porta>

<Porta nivel="pratica" titulo="Aplicación práctica">

**Receta: cómo usar los límites fundamentales.**

### Pasos para cualquier indeterminación

1. **Identifica el tipo**: $0/0$, $\infty/\infty$, $1^\infty$, etc.
2. **Intenta reescribir** en forma de límite fundamental:
   - $\sin(\text{algo}) / \text{algo}$ → usa $\sin x / x$.
   - $(1 + \text{algo})^{1/\text{algo}}$ → usa $e$.
   - $(e^{\text{algo}} - 1) / \text{algo}$ → usa $(e^x - 1)/x$.
3. **Cambia de variable** si es necesario ($u = $ argumento dentro).
4. **Comprueba numéricamente** o con SymPy.

### Casos típicos

| Indeterminación | Estrategia |
|---|---|
| $0/0$ trig | Usar $\sin x \approx x$, $1 - \cos x \approx x^2/2$ |
| $0/0$ exp/log | Usar $(e^x - 1)/x \to 1$, $\ln(1+x)/x \to 1$ |
| $1^\infty$ | $A^B = e^{B \ln A}$, manipular $B \ln A$ |
| $0^0$ o $\infty^0$ | Igual: $A^B = e^{B \ln A}$, $\ln A \to -\infty$ o $+\infty$ |
| $\infty - \infty$ | Factor común, conjugado o racional común |

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
print(sp.limit(sp.sin(5*x)/x, x, 0))           # 5
print(sp.limit((1 + 2/x)**x, x, sp.oo))        # exp(2)
print(sp.limit((sp.exp(3*x) - 1)/x, x, 0))     # 3
print(sp.limit(x**x, x, 0, '+'))               # 1
print(sp.limit((1 - sp.cos(x))/x**2, x, 0))    # 1/2
```

### Wolfram Alpha

`limit (1 + 3/x)^x as x -> infinity` → $e^3$.

### Cuándo NO usar límites fundamentales

- **$x \to a \neq 0$** con expresión trig: puede ser que la sustitución directa funcione.
- **Funciones no relacionadas con $\sin, e, \ln$**: otras técnicas.

### Cuidados

- El **argumento** de $\sin$ debe ser igual al denominador. $\sin(5x)/x \neq 1$: es $5$.
- $(1 + 1/n)^n$ vs $(1 + 2/n)^n$ vs $(1 + 1/n)^{2n}$: todo da cosas distintas.
