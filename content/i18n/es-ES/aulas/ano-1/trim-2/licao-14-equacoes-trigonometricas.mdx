<Porta nivel="formal" titulo="Estructura de las soluciones">

### Ecuaciones con seno

Para $\sin x = a$ con $a \in [-1, 1]$:
$$x = \arcsin a + 2k\pi \quad \text{o} \quad x = \pi - \arcsin a + 2k\pi, \quad k \in \mathbb{Z}$$

Donde $\arcsin: [-1, 1] \to [-\pi/2, \pi/2]$ es la inversa principal.

### Ecuaciones con coseno

Para $\cos x = a$ con $a \in [-1, 1]$:
$$x = \pm \arccos a + 2k\pi, \quad k \in \mathbb{Z}$$

Donde $\arccos: [-1, 1] \to [0, \pi]$.

### Ecuaciones con tangente

Para $\tan x = a$ con $a \in \mathbb{R}$:
$$x = \arctan a + k\pi, \quad k \in \mathbb{Z}$$

Donde $\arctan: \mathbb{R} \to (-\pi/2, \pi/2)$.

### Receta general

Para resolver $f(x) = 0$ trigonométrica:

1. **Reducir** a ecuaciones básicas $\sin x = a$, $\cos x = a$, $\tan x = a$ vía identidades.
2. **Listar** todas las soluciones en el intervalo $[0, 2\pi)$ (o $[-\pi, \pi]$).
3. **Generalizar** sumando $2k\pi$ (o $k\pi$ para tangente).

### Tipos de ecuaciones reducibles

| Tipo | Ejemplo | Sustitución |
|---|---|---|
| Polinómica en $\sin$ | $2 \sin^2 x - 3 \sin x + 1 = 0$ | $u = \sin x$ |
| Polinómica en $\cos$ | $\cos^2 x = 1/2$ | $u = \cos x$ |
| $\sin x \pm \cos x$ | $\sin x + \cos x = 1$ | $\sqrt 2 \sin(x + \pi/4) = 1$ |
| Ángulo doble | $\sin 2x = \sin x$ | $2 \sin x \cos x = \sin x$ |
| Suma-producto | $\sin x + \sin 3x = 0$ | $2 \sin 2x \cos x = 0$ |

<Porta nivel="5" titulo="Para un niño de 5 años">

Imagina un reloj. ¿En qué momentos del día la manecilla grande está apuntando al **6** (abajo del todo)? Respuesta: **6:00 y 18:00** (y también 30:00 si quieres más vueltas, pero no hay 30:00 en un día normal).

Una ecuación trigonométrica es así: quieres saber **en qué ángulos** el seno (o coseno) tiene cierto valor. Y la respuesta es una **lista** de ángulos, porque la rueda gira siempre.

</Porta>

<Porta nivel="10" titulo="Para un niño de 10 años">

La ecuación $\sin x = 1/2$ no es como $x + 3 = 5$ (una sola respuesta). Tiene **infinitas respuestas** porque el seno se repite.

Imagina el círculo trigonométrico. ¿Qué puntos tienen $y = 1/2$? **Dos puntos**: uno en el primer cuadrante (ángulo $\pi/6$) y otro en el segundo (ángulo $5\pi/6$). Y como puedes girar el círculo cuantas veces quieras, cada uno de esos dos puntos se convierte en **infinitos**: $\pi/6, \pi/6 + 2\pi, \pi/6 + 4\pi, \ldots$

Por eso la respuesta de una ecuación trigonométrica viene en **conjunto**:
$$x = \pi/6 + 2k\pi \quad \text{o} \quad x = 5\pi/6 + 2k\pi$$

para $k$ entero cualquiera.

Normalmente el profesor pide la respuesta en $[0, 2\pi)$ — solo una vuelta. Ahí es fácil: lista las soluciones de ese intervalo y listo.

</Porta>

<Porta nivel="15" titulo="Para un adolescente de 15 años">

Resolver $\sin x = 1/2$ no es tan sencillo como resolver $x + 3 = 5$.

¿Por qué? Porque el seno se repite todo el tiempo. Si $x = \pi/6$ es solución, $x = \pi/6 + 2\pi$ también lo es. Y $x = \pi/6 + 4\pi$. Y... infinitas.

Además, $\sin(\pi - \pi/6) = \sin(5\pi/6) = 1/2$ también. Entonces en $[0, 2\pi)$ hay **dos** soluciones: $\pi/6$ y $5\pi/6$.

**Receta:**
1. Encuentra las soluciones en el círculo trigonométrico, en $[0, 2\pi)$.
2. Añade $2k\pi$ (todas las vueltas enteras).

Para la tangente es distinto: el periodo de $\tan$ es $\pi$ (no $2\pi$), porque $\tan(x + \pi) = \tan x$.

**Ecuaciones más elaboradas** se reducen a básicas vía identidades:
- $\cos 2x = \cos^2 x - \sin^2 x = 1 - 2 \sin^2 x$.
- $\sin 2x = 2 \sin x \cos x$.
- $\sin x + \cos x = \sqrt 2 \sin(x + \pi/4)$.

**Inecuaciones**: igual pero con intervalo. ¿$\sin x > 1/2$ en $[0, 2\pi)$? Los $x$ en los que $\sin x$ está por encima de $1/2$ — por la gráfica, eso es $(\pi/6, 5\pi/6)$.

</Porta>

<Porta nivel="25" titulo="Para un estudiante de ingeniería">

En la práctica, **no se resuelve trigonometría a mano** después del Bachillerato. Se usa calculadora o solver. Pero la estructura de las soluciones es importante porque controla la forma de las respuestas en problemas de ingeniería.

En control, el **acoplamiento de fase** entre dos señales viene dado por $\arctan$ del cociente parte imaginaria/parte real del cociente de las transferencias. En mecánica, el ángulo de equilibrio es raíz de una ecuación trigonométrica.

### Métodos numéricos

Para ecuaciones trigonométricas que no se reducen a básicas (p. ej., $x = \cos x$), se usa **Newton-Raphson** o **bisección**. **No hay fórmula cerrada**.

```python
from scipy.optimize import brentq
import numpy as np
sol = brentq(lambda x: x - np.cos(x), 0, 1)
print(sol)  # ~0.739085
```

### Soluciones complejas (vista previa)

Para $\sin z = 5$ (imposible en reales), hay solución en $\mathbb{C}$ usando Euler:
$$\sin z = (\e^{iz} - \e^{-iz})/(2i) = 5 \Rightarrow z = \arcsin 5 = \pi/2 - i \ln(5 + 2\sqrt 6)$$

### Aplicaciones en ingeniería

- **Equilibrio de fuerzas**: $\sum F_i \cos \theta_i = 0$ — ecuación trigonométrica en $\theta$.
- **Cinemática inversa**: el ángulo de articulación de un brazo robótico resuelve la ecuación $a \cos\theta + b \sin\theta = c$.
- **Acústica**: resonancia en sala cerrada cuando $\sin(k_x L_x) = 0$ — modos propios.
- **Sistemas de control**: ceros y polos en el círculo unitario — donde $z = \e^{j\omega T}$.

</Porta>

<Porta nivel="40" titulo="Para un profesional sénior">

Las ecuaciones trigonométricas son un caso particular de ecuaciones **trascendentes**. En general no admiten fórmula cerrada — solo fórmulas para casos polinómicos en $\sin, \cos, \tan$.

### Estructura algebraica

El **anillo de las funciones trigonométricas** $\mathbb{R}[\sin x, \cos x]$, módulo $\sin^2 x + \cos^2 x = 1$, es isomorfo a $\mathbb{R}[t, t^{-1}]/(t \bar t = 1)$ vía Euler $t = \e^{ix}$. Esto reduce toda ecuación polinómica en $\sin x, \cos x$ a una ecuación algebraica en $t$ en el círculo unitario complejo.

```python
import sympy as sp
x = sp.symbols('x', real=True)

# Solución general simbólica:
sp.solveset(sp.sin(x) - sp.Rational(1,2), x, sp.S.Reals)
# Union(ImageSet({2*pi*n + pi/6, n in Z}),
#       ImageSet({2*pi*n + 5*pi/6, n in Z}))

# En intervalo:
sp.solveset(sp.sin(x) - sp.Rational(1,2), x, sp.Interval(0, 2*sp.pi))
```

### Patrones avanzados

- **Ecuación de Kepler**: $M = E - e \sin E$ — resolución para órbita elíptica. Newton converge en $\sim 5$ iteraciones para $e < 0{,}9$.
- **Ecuación de Bessel**: ceros no cerrados, pero tabulables. Aparece en modos de vibración de tambor.
- **Identidades de Chebyshev**: $\cos(n\theta) = T_n(\cos\theta)$ con $T_n$ polinomio de grado $n$ — base de cuadratura de Gauss-Chebyshev.

### Pitfalls

- **`np.arcsin` devuelve en $[-\pi/2, \pi/2]$**: para obtener la solución en el cuadrante II, usa $\pi - \arcsin a$.
- **Modo de calculadora**: deg vs rad en CFD/FEA — fuente clásica de bug en validación.
- **Soluciones perdidas** al multiplicar/dividir por $\sin x$ o $\cos x$ (que pueden ser cero): comprueba siempre si existe solución adicional donde esa función se anula.

<Porta nivel="pratica" titulo="Aplicación práctica">

**Receta master para resolver una ecuación trigonométrica:**

1. **Intenta reducir a básica** ($\sin x = a$, $\cos x = a$, $\tan x = a$).
2. **Usa identidades** (Pitágoras, suma, ángulo doble) si es necesario.
3. **Sustituye** $u = \sin x$ o $u = \cos x$ si es polinómica.
4. **Resuelve** $u$ primero, después vuelve a $x$.
5. **Lista** las soluciones en el intervalo pedido.
6. **Comprueba** una de las soluciones en la ecuación original.

### Workflow Wolfram / SymPy

```python
import sympy as sp
x = sp.symbols('x', real=True)

# Ecuación simple:
sp.solveset(sp.sin(x) - sp.cos(x), x, sp.Interval(0, 2*sp.pi))
# {pi/4, 5*pi/4}

# Ecuación con identidad:
eq = sp.sin(2*x) - sp.sin(x)   # = 2 sin(x) cos(x) - sin(x)
sp.solveset(eq, x, sp.Interval(0, 2*sp.pi))

# Inecuación:
sp.reduce_inequalities(sp.sin(x) > sp.Rational(1, 2))   # cuidado: puede fallar
```

Wolfram Alpha: `solve sin(x) + cos(x) = 1, x in [0, 2pi]` devuelve $\{0, \pi/2\}$.

### ¿Cuándo preocuparse?

- **Soluciones espurias** al elevar al cuadrado: $\sin x = -\cos x$ tiene solución, pero $\sin^2 x = \cos^2 x$ tiene **más** soluciones (incluyendo $\sin x = +\cos x$). Verifica siempre.
- **Al dividir por $\cos x$**: pierdes soluciones con $\cos x = 0$. Trata ese caso aparte.
- **`tan` en punto crítico**: $\tan x$ va a $\pm \infty$ en $x = \pi/2$ — en la inecuación, ese punto **no está** en el dominio.
- **En problemas físicos**, restringe el dominio: ángulos negativos pueden no tener sentido.

### El ejercicio más importante

Los ejercicios 14.21 (sin + cos = 1) y 14.31 (sin x > cos x) — porque combinan identidades, intervalos e inecuaciones simultáneamente.
