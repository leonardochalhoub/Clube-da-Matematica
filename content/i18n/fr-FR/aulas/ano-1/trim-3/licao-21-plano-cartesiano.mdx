<Porta nivel="formal" titulo="Géométrie analytique dans ℝ²">

<Definicao titulo="Plan cartésien">
$\mathbb{R}^2 = \{(x, y) : x, y \in \mathbb{R}\}$. Chaque point est un couple ordonné. Les axes $x$ (horizontal) et $y$ (vertical) se croisent en $O = (0, 0)$, divisant le plan en 4 quadrants.
</Definicao>

### Distance

Pour $P = (x_1, y_1)$, $Q = (x_2, y_2)$ :
$$d(P, Q) = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

### Point milieu

$$M = \left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}\right)$$

### Division d'un segment dans le rapport $k$

Pour diviser $\overline{PQ}$ dans le rapport $PR/RQ = k$ :
$$R = \left(\frac{x_1 + k x_2}{1 + k}, \frac{y_1 + k y_2}{1 + k}\right)$$

### Calcul d'aires

Aire du triangle de sommets $A = (x_1, y_1)$, $B = (x_2, y_2)$, $C = (x_3, y_3)$ :
$$\text{Aire} = \frac{1}{2} |x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2)|$$

Équivalent à $\frac{1}{2} |\det M|$ où $M$ est la matrice des vecteurs $\vec{AB}, \vec{AC}$.

</Porta>

<Porta nivel="15" titulo="Pour un adolescent de 15 ans">

Plan cartésien : chaque point est décrit par 2 nombres — une coordonnée $x$ (horizontale) et une coordonnée $y$ (verticale).

**Distance entre 2 points** : applique Pythagore. Si tu as $P = (1, 2)$ et $Q = (4, 6)$ :

$$d = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

**Point milieu** : prends la moyenne des coordonnées. Le milieu entre $(1, 2)$ et $(4, 6)$ est $(2{,}5,\ 4)$.

**Pourquoi est-ce important ?** Parce que le plan cartésien est le langage qui **connecte algèbre et géométrie**. Une droite devient une équation. Un cercle devient une équation. Une parabole devient une équation. Tout ce qui était « géométrique » devient « algébrique » — et l'ordinateur peut calculer.

</Porta>

<Porta nivel="25" titulo="Pour un étudiant en ingénierie">

La géométrie analytique est la base de tout :

- **Infographie** : chaque pixel est un point, chaque objet 2D/3D est un ensemble de coordonnées.
- **GPS** : positionnement par coordonnées + triangulation.
- **Robotique** : la cinématique décrit la trajectoire des bras robotiques.
- **Génie civil** : le levé topographique produit des coordonnées.

### Distance dans $\mathbb{R}^n$

Se généralise à n'importe quelle dimension :
$$d(P, Q) = \sqrt{\sum_{i=1}^n (x_i - y_i)^2} = \|P - Q\|_2$$

C'est la **norme euclidienne**. En ML, c'est la « distance par défaut » utilisée en k-NN, k-means, etc.

### Autres métriques

- **Manhattan** ($\ell_1$) : $\sum |x_i - y_i|$ — distance en pâtés de maisons.
- **Tchebychev** ($\ell_\infty$) : $\max |x_i - y_i|$ — déplacement du roi aux échecs.
- **Métriques géodésiques** : sur la sphère (Terre), la distance suit les grands cercles.

</Porta>
