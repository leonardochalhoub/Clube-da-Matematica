<Porta nivel="formal" titulo="Axioms and formulas">

### Sample space

$\Omega$: set of all possible outcomes of the experiment. **Event**: subset of $\Omega$.

### Kolmogorov axioms (1933)

1. $P(A) \geq 0$ for every event $A$.
2. $P(\Omega) = 1$.
3. **$\sigma$-additivity**: if $A_1, A_2, \ldots$ disjoint: $P(\bigcup A_i) = \sum P(A_i)$.

### Classical probability (equiprobable space)

$$P(A) = \frac{|A|}{|\Omega|}$$

### Immediate properties

- $P(\emptyset) = 0$.
- $P(A^c) = 1 - P(A)$.
- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$ (inclusion-exclusion).
- $A \subseteq B \Rightarrow P(A) \leq P(B)$.
- $P(A) \in [0, 1]$.

### Conditional probability

$$P(A | B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0$$

### Independence

$A$ and $B$ are **independent** if $P(A \cap B) = P(A) P(B)$ (equivalent: $P(A|B) = P(A)$).

### Bayes' theorem

$$P(A | B) = \frac{P(B | A) P(A)}{P(B)}$$

### Total probability

If $\{A_i\}$ partition of $\Omega$:
$$P(B) = \sum_i P(B | A_i) P(A_i)$$

### Discrete random variable

$X: \Omega \to \mathbb R$. Distribution: $P(X = x_i) = p_i$ with $\sum p_i = 1$.

| Distribution | Formula | Appearance |
|---|---|---|
| Bernoulli | $P(X=1) = p$ | 1 trial |
| Binomial | $\binom{n}{k}p^k(1-p)^{n-k}$ | $n$ trials |
| Geometric | $(1-p)^{k-1}p$ | waiting time |
| Poisson | $e^{-\lambda}\lambda^k/k!$ | rare events |

### Expectation and variance

- $E[X] = \sum x_i p_i$
- $\text{Var}(X) = E[(X - E[X])^2] = E[X^2] - E[X]^2$

<Porta nivel="5" titulo="For a 5-year-old">

Probability is "**chance** of something happening". Goes from 0 (impossible) to 1 (certain).

Toss a coin: chance of heads is one half (1/2), because there are 2 outcomes and 1 of them is heads.

Roll a die: chance of getting 6 is one sixth (1/6), because the die has 6 sides and only 1 is the 6.

The **more possibilities**, the **harder** to hit.

<Porta nivel="10" titulo="For a 10-year-old">

**Probability** = chance of something happening, a number between 0 and 1.

**Classical rule** (when all outcomes are equally likely):
$$P(\text{event}) = \frac{\text{favorable cases}}{\text{total cases}}$$

Examples:
- Die: $P(\text{even}) = 3/6 = 1/2$.
- 2 coins: $P(\text{2 heads}) = 1/4$ (HH, HT, TH, TT — only HH).
- Cards: $P(\text{king}) = 4/52 = 1/13$.

**Conditional probability**: "if I already know that B happened, what's the chance of A?". Written $P(A|B)$.

**Independence**: the outcome of A doesn't affect B. Example: rolling 2 dice — the second die doesn't care about the first.

**Bayes**: there's a clever formula to "invert" probabilities. It's like discovering the cause from the effect.

**Where it shows up**: lottery, games, weather forecast, medical diagnosis, spam filter, AI.

</Porta>

<Porta nivel="15" titulo="For a 15-year-old teen">

Probability is "a number that measures the chance" of something happening. Goes from 0 (impossible) to 1 (certain).

**Classical case**: experiment with $|\Omega|$ equally likely outcomes (toss coin, roll die). $P(A)$ = "favorable cases divided by possible cases".

Example: roll a die, $P(\text{even}) = 3/6 = 1/2$. Toss a coin twice, $P(\text{2 heads}) = 1/4$.

**Conditional probability**: "given that B occurred, what's the probability of A?". $P(A|B) = P(A \cap B)/P(B)$.

**Independent events**: $P(A \cap B) = P(A) P(B)$. Roll 2 dice — the result of the 2nd doesn't affect the 1st.

**Bayes**: way to "invert" conditional. If you know $P(B|A)$, you can compute $P(A|B)$. Used in medical diagnosis, spam filter, ML.

**Inclusion-exclusion**: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$. When the events aren't disjoint, the intersection gets counted twice — subtract.

**Binomial distribution**: number of successes in $n$ independent trials:
$$P(X = k) = \binom{n}{k}p^k(1-p)^{n-k}$$

Example: toss a coin 10 times, what's the chance of exactly 5 heads? $\binom{10}{5}(1/2)^{10} = 252/1024 \approx 0.246$.

**Combinatorics + probability go hand in hand**: counting gives $|A|$ and $|\Omega|$, division gives $P(A)$.

</Porta>

<Porta nivel="25" titulo="For an engineering student">

Probability is the **basis** of data science, ML, stochastic control, quantitative finance.

### Random variable

$X: \Omega \to \mathbb{R}$. Numerical value associated with each outcome. **Discrete** or **continuous** (next series).

### Expectation

$E[X] = \sum_i x_i P(X = x_i)$. Linear: $E[aX + bY] = aE[X] + bE[Y]$ — always, even without independence.

### Variance

$\text{Var}(X) = E[(X - \mu)^2] = E[X^2] - \mu^2$. **Not linear**: $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)$.

If $X, Y$ independent: $\text{Cov} = 0$, so $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y)$.

### Binomial distribution

$X \sim \text{Bin}(n, p)$: number of "successes" in $n$ independent trials with prob. $p$:
$$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}$$

Appears in quality control, clinical trials, A/B testing.

$E[X] = np$, $\text{Var}(X) = np(1-p)$.

### Bayes in practice — medical diagnosis

Rare disease $D$ with $P(D) = 0.001$. Test: $P(+|D) = 0.99$ (sensitivity), $P(-|D^c) = 0.95$ (specificity). Probability of disease given positive test:
$$P(D | +) = \frac{P(+|D) P(D)}{P(+)} = \frac{0.99 \cdot 0.001}{0.99 \cdot 0.001 + 0.05 \cdot 0.999} \approx 0.019$$

Only **1.9%** — base rate fallacy explained by Bayes.

### Cross-domain

- **Engineering**: reliability — series/parallel systems, MTBF.
- **Signals**: detection in noise via Neyman-Pearson.
- **ML**: Bayesian classifiers, probabilistic neural networks.
- **Quant**: risk-neutral pricing, value-at-risk.
- **Statistics**: hypothesis tests, p-values, confidence intervals.

### Law of large numbers

$\bar X_n = \frac{1}{n}\sum X_i \xrightarrow{n\to\infty} E[X]$ (convergence in probability or almost sure). Justifies relative frequencies as probability estimates.

### Central limit theorem (preview)

$\frac{\bar X - \mu}{\sigma/\sqrt n} \xrightarrow{d} N(0, 1)$. Sum of many independent variables tends to Gaussian — explains the ubiquity of the normal.

</Porta>

<Porta nivel="40" titulo="For a senior professional">

Modern probability is built on **measure spaces** $(\Omega, \mathcal F, P)$ in Kolmogorov's formalism (1933). Every random variable is a measurable function.

### Wolfram tour

```mathematica
Probability[X >= 5, X \[Distributed] BinomialDistribution[10, 0.5]]
Mean[BinomialDistribution[10, 0.5]]
PDF[NormalDistribution[0, 1], x]
RandomVariate[BinomialDistribution[100, 0.3], 10]
```

### Kolmogorov and measure

$\sigma$-algebra $\mathcal F$ — measurable events. Required because in continuous $\Omega$ (e.g., $[0,1]$) not every subset is measurable (Vitali). In discrete, $\mathcal F = 2^\Omega$ works.

### Bayesian inference

Posterior $\propto$ prior $\times$ likelihood. MCMC (Metropolis-Hastings, HMC, NUTS) samples from arbitrary posteriors. Stan, PyMC3, Pyro, NumPyro. Production tools in modern data science.

### Quant — measure change

Risk-neutral pricing: $V_0 = e^{-rT} E^Q[\text{payoff}]$. Radon-Nikodym change from $P$ (real-world) to $Q$ (risk-neutral). Girsanov for continuous processes.

### Concentration inequalities

- **Markov**: $P(X \geq a) \leq E[X]/a$.
- **Chebyshev**: $P(|X-\mu| \geq k\sigma) \leq 1/k^2$.
- **Hoeffding**: $P(|\bar X_n - \mu| \geq t) \leq 2e^{-2nt^2/(b-a)^2}$ — basis of PAC learning.
- **Chernoff**: exponential tail — communication, RNG, hashing.

### ML and probability

- **Cross-entropy loss**: $-\sum p_i \log q_i$.
- **KL divergence**: $D_{KL}(p\|q) = \sum p_i \log(p_i/q_i)$ — basis of variational methods.
- **Score-based generative models** (diffusion): learn $\nabla \log p$.
- **Neural networks** can be understood as universal approximators of conditional distributions $P(Y|X)$.

### Research

- **Free probability** (Voiculescu): non-commutative algebra for large random matrices.
- **Optimal transport** (Villani, Cuturi): distance between distributions — Wasserstein.
- **Stein's method**: bounds errors in CLT and variations.
- **Markov chain mixing**: cutoff phenomena, Diaconis.

</Porta>

<Porta nivel="pratica" titulo="Practical application">

**Workflow for probability problems.**

1. **Define $\Omega$ and the event $A$ clearly**. 80% of errors come from imprecise definition.
2. **Equiprobable?** Use combinatorics ($|A|/|\Omega|$).
3. **Non-equiprobable?** Use properties + axioms.
4. **Conditional or Bayes?** Identify what is given and what is sought.
5. **Independence?** Multiply. Otherwise, condition.

### Python recipe

```python
import math
from math import comb
from scipy.stats import binom, norm, poisson

# Classical probability
def classical_prob(favorable, total):
    return favorable / total

print(classical_prob(comb(4, 2), comb(52, 5)))   # P(2 kings in 5 cards) — adapted

# Binomial distribution
n, p = 10, 0.5
print(binom.pmf(5, n, p))     # P(X=5)
print(binom.cdf(3, n, p))     # P(X<=3)
print(binom.mean(n, p))       # n*p

# Manual Bayes
def bayes(prior_A, likelihood_B_given_A, likelihood_B_given_notA):
    prior_notA = 1 - prior_A
    p_B = likelihood_B_given_A * prior_A + likelihood_B_given_notA * prior_notA
    return likelihood_B_given_A * prior_A / p_B

# Rare disease
print(bayes(0.001, 0.99, 0.05))   # P(D|+) ~ 0.019

# Monte Carlo simulation
import numpy as np
np.random.seed(42)
samples = np.random.choice([0, 1], size=10000, p=[0.5, 0.5])
print((samples.sum()) / 10000)    # ~ 0.5
```

### Bayes via table

```python
# For small problems, build a 2x2 table
import pandas as pd
df = pd.DataFrame({
    "D": [0.001 * 0.99, 0.001 * 0.01],   # has disease
    "notD": [0.999 * 0.05, 0.999 * 0.95]  # doesn't
}, index=["+", "-"])
print(df.div(df.sum(axis=1), axis=0))   # conditional
```

### Pitfalls

- **Distinguish $P(A|B)$ from $P(B|A)$**. Bayes is exactly about that inversion.
- **Independence != mutually exclusive**. Independent can co-occur; mutually exclusive cannot.
- **Non-equiprobable cases**: rolling 2 dice → 36 equiprobable outcomes, BUT sums 2 to 12 are NOT equiprobable.
- **Base rate fallacy**: ignores the base rate — the prior probability matters a lot.
