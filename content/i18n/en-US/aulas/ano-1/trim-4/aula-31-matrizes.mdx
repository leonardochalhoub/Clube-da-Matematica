<Definicao titulo="Matrix">
An $m \times n$ matrix over $\mathbb{R}$ is a function $A: \{1, \ldots, m\} \times \{1, \ldots, n\} \to \mathbb{R}$. Notation $A = (a_{ij})$ or $A_{m \times n}$.

Equality: $A = B \iff$ same dimension and $a_{ij} = b_{ij}$ for all $i, j$.
</Definicao>

### Special types

- **Square** ($m = n$): same number of rows and columns.
- **Row** ($1 \times n$): one row, row vector.
- **Column** ($m \times 1$): one column, column vector.
- **Diagonal**: square with $a_{ij} = 0$ for $i \neq j$.
- **Identity** ($I_n$): diagonal with diagonal entries $= 1$.
- **Zero**: all elements are 0.
- **Upper triangular**: $a_{ij} = 0$ for $i > j$.
- **Lower triangular**: $a_{ij} = 0$ for $i < j$.
- **Symmetric**: $A^T = A$, i.e. $a_{ij} = a_{ji}$.
- **Antisymmetric**: $A^T = -A$, i.e. $a_{ij} = -a_{ji}$.

### Diagonal of a square matrix

The **main diagonal** is $\{a_{ii}\}$. **Trace**: $\text{tr}(A) = \sum a_{ii}$.

</Porta>

<Porta nivel="15" titulo="For a 15-year-old">

A matrix is a **table** of numbers organized in rows and columns:

$$A = \begin{pmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{pmatrix}$$

This is a $2 \times 3$ matrix (2 rows, 3 columns). Each number has an **address**: $a_{12} = 2$ (row 1, column 2). $a_{23} = 6$ (row 2, column 3).

**What is it for?** To store and organize data:
- Student grades × subjects: student-subject matrix.
- Distances between cities: symmetric matrix.
- Pixels of an image: intensity matrix.
- Coefficients of a linear system: coefficient matrix.
- Weights of a neural network: vectors and matrices (tons).

**Important types**:
- **Square**: has the same number of rows and columns.
- **Identity**: has 1s on the diagonal and 0 everywhere else. Acts like the "1" of multiplication ($IA = A$).

</Porta>

<Porta nivel="25" titulo="For an engineering student">

The matrix is the **fundamental object** of scientific computing and ML. In Python/NumPy, "everything is an array". In MATLAB, "everything is a matrix".

### Common matrix structures

- **Sparse**: most elements are zero. CSR/CSC formats. Crucial in FEM, networks, NLP.
- **Band**: zeros outside a central "band". Useful in tridiagonal systems (heat, Crank-Nicolson).
- **Dense**: everything stored. Standard in ML, pure linear algebra.

### Engineering applications

- **Control systems**: state $\dot{\mathbf{x}} = A\mathbf{x} + B\mathbf{u}$.
- **Structures (FEM)**: global stiffness is a sparse matrix.
- **Image**: each RGB channel is an $H \times W$ matrix.
- **Neural networks**: each layer is a matrix multiplication.

</Porta>
