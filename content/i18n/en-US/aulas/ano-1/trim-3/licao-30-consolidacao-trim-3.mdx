### Synthesis of the Cartesian method

Descartes (1637) taught: **translate any geometric problem to algebra**, solve it, then translate it back. This lesson is the final exercise of that translation in 2D before going to 3D in Term 4.

<Porta nivel="40" titulo="For a senior professional">

Term 3 covers what in the English literature is "Coordinate Geometry / Analytic Geometry" — but the student probably doesn't realize they are learning **elementary linear algebra in disguise**.

### Formal connections

| Term 3 topic | Linear algebra concept |
|---|---|
| Cartesian plane | $\mathbb{R}^2$ as a vector space |
| Line equation | hyperplane in $\mathbb{R}^2$ |
| 2x2 system | linear map $\mathbb{R}^2 \to \mathbb{R}^2$ |
| Determinant | oriented volume, parallelogram area |
| Dot product | bilinear form / inner product |
| Direction vector | basis of subspace |
| Coincident / parallel | linear dependence / rank |
| Conic | quadratic form $\mathbf x^T A \mathbf x = 1$ |

### Applications in ML / quant

- **Attention in Transformers**: $\text{softmax}(QK^T/\sqrt d)V$. $QK^T$ is a matrix of dot products.
- **Markowitz** (1952, [Nobel 1990](https://www.nobelprize.org/prizes/economic-sciences/1990/)): minimize $\mathbf w^T \Sigma \mathbf w$ — generalized dot product. The frontier is a hyperbola.
- **Black-Scholes** (1973, [Nobel 1997](https://www.nobelprize.org/prizes/economic-sciences/1997/)): parabolic heat equation in ($\ln S$, $t$). Monte Carlo simulation works in $\mathbb{R}^N$.
- **Word embeddings**: similarity $= \cos\theta$ between vectors in $\mathbb{R}^{300}$.

### Wolfram / SymPy workflow

```python
import sympy as sp
from sympy.geometry import Point, Line, Circle, Triangle

# Typical problem example: tangent from external point to circle
C = Circle(Point(0, 0), 5)
P = Point(7, 0)
print(C.tangent_lines(P))      # 2 tangent lines
print(P.distance(C.center)**2 - C.radius**2)  # squared length of the tangent

# Geometry system
T = Triangle(Point(0,0), Point(6,0), Point(0,8))
print(T.incenter, T.incircle.radius)  # center and radius of the inscribed circle
```

### When to be careful?

- **Mixing contexts**: be careful not to apply the ellipse formula to a circle (they're equivalent when $a = b$, but the notation changes).
- **Units**: work in joules, force in newtons, distance in meters — always coherent.
- **Cosine sign**: a negative dot product means an obtuse angle, not an error.
- **Verify order of magnitude**: cable tension supporting 1000 kg must be ≥ 5000 N (gravity) — anything much smaller is an error.
