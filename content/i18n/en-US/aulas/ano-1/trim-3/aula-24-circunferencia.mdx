<Definicao titulo="Circle">
The set of points $(x, y) \in \mathbb{R}^2$ at distance $r > 0$ from a center $C = (a, b)$.
</Definicao>

### Standard form

$$(x - a)^2 + (y - b)^2 = r^2$$

### General form

Expanding: $x^2 + y^2 + Dx + Ey + F = 0$, with $D = -2a$, $E = -2b$, $F = a^2 + b^2 - r^2$.

Center recovered: $(a, b) = (-D/2, -E/2)$. Radius: $r = \sqrt{D^2/4 + E^2/4 - F}$ (valid if positive; otherwise, not a real circle).

### Line and circle

| Position | Criterion | Common points |
|---|---|---|
| Exterior | $d(C, r) > r$ | 0 |
| Tangent | $d(C, r) = r$ | 1 |
| Secant | $d(C, r) < r$ | 2 |

### Two circles

| Position | Criterion |
|---|---|
| Externally disjoint | $d(C_1, C_2) > r_1 + r_2$ |
| Externally tangent | $d = r_1 + r_2$ |
| Secant | $|r_1 - r_2| < d < r_1 + r_2$ |
| Internally tangent | $d = |r_1 - r_2|$ |
| One inside the other | $d < |r_1 - r_2|$ |

### Tangent-radius property

The tangent at $P$ is perpendicular to the radius $\overline{CP}$.

### Tangent equation

Tangent at $(x_0, y_0)$ to the circle $(x-a)^2 + (y-b)^2 = r^2$:
$$(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$$

Take a **string** and pin one end to the floor. Hold a piece of chalk at the other end and walk **always keeping the string taut**. You draw a **perfect wheel** — a **circle**!

Every point on the wheel is **the same distance** from the middle. That distance is the **radius**.

<Porta nivel="10" titulo="For a 10-year-old">

A circle is the **set of points at the same distance** from the center. Like the drawing a compass makes: the distance from the pin tip to the pencil tip is the **radius**.

The equation $(x - a)^2 + (y - b)^2 = r^2$ comes from the distance formula (Lesson 21):
- $(a, b)$ is the center.
- $r$ is the radius.
- For each point $(x, y)$ on the curve, the distance to the center is $r$. Squaring, you get the formula above.

**Concrete example**: a bicycle wheel with center on the axle. Every point of the rubber is at the same distance from the axle.

**Real applications**:
- Wi-Fi: range is a circle.
- Police siren: people who hear it are inside the circle.
- Track field: 400 m in circumference.

</Porta>

<Porta nivel="15" titulo="For a 15-year-old teen">

A circle is the set of points **equally distant** from a center. Distance $= r$ (radius).

Applying the distance formula from Lesson 21:
$$\sqrt{(x - a)^2 + (y - b)^2} = r$$

Squaring: $(x - a)^2 + (y - b)^2 = r^2$. **That's all there is to it**.

When the center is at the origin ($a = b = 0$): $x^2 + y^2 = r^2$. **Unit circle** ($r = 1$): $x^2 + y^2 = 1$ — what you used in Lesson 12 of the trigonometric circle!

**General form $\to$ standard form**: complete the square. Example: $x^2 + y^2 - 4x + 6y - 12 = 0$.
- In $x$: $x^2 - 4x = (x-2)^2 - 4$.
- In $y$: $y^2 + 6y = (y+3)^2 - 9$.
- Substitute: $(x-2)^2 - 4 + (y+3)^2 - 9 - 12 = 0 \Rightarrow (x-2)^2 + (y+3)^2 = 25$.
- Center $(2, -3)$, radius 5.

**Caution**: not every equation $x^2 + y^2 + Dx + Ey + F = 0$ is a circle. If $D^2/4 + E^2/4 - F < 0$, there is no real point. If $= 0$, it's a single point ("degenerate" circle).

</Porta>

<Porta nivel="25" titulo="For an engineering student">

The circle is the simplest example of a **locus**. Immediate generalization: **sphere** in $\mathbb{R}^3$ — $(x-a)^2 + (y-b)^2 + (z-c)^2 = r^2$. In $\mathbb{R}^n$: $\|x - c\|^2 = r^2$ is the $(n-1)$-sphere.

### Cross-domain applications

- **Telecommunications**: antenna range = circle. Multiple coverage = union/intersection of circles. Voronoi diagrams (each point assigned to the nearest antenna).
- **GPS / Trilateration**: each satellite defines a sphere; your location is the intersection of 4 spheres. 3 give position + time; the 4th removes ambiguity.
- **Robotics**: the reach of an arm with a spherical joint is a sphere.
- **Computer graphics**: bounding sphere for collision and culling.
- **Chemical engineering**: Gibbs equilibrium in a binary reactive system forms circles in the phase diagram.

### Demonstration: tangent is perpendicular to the radius

Let $P_0 = (x_0, y_0)$ be a point on the circle, center $C = (a, b)$. Vector $\vec{CP_0} = (x_0 - a, y_0 - b)$. The tangent has $\vec{CP_0}$ as its normal vector. So the equation:
$$(x_0 - a)(x - x_0) + (y_0 - b)(y - y_0) = 0$$

Expanding and using $(x_0 - a)^2 + (y_0 - b)^2 = r^2$:
$$(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$$

### Power of a point

For point $P_0$ and circle of center $C$, radius $r$:
$$\text{pow}(P_0) = |P_0 C|^2 - r^2$$

- Negative: $P_0$ inside.
- Zero: $P_0$ on the boundary.
- Positive: $P_0$ outside; $\sqrt{\text{pow}}$ is the length of the tangent from $P_0$.

### 2D Trilateration

3 circles passing through the same point: 2 linear equations (subtracting pairs of circle equations cancels the quadratic terms). 2×2 linear system solves it.

</Porta>

<Porta nivel="40" titulo="For a senior professional">

The circle is the **fundamental curve** of differential geometry — constant curvature $\kappa = 1/r$.

### Differential view

Parametrization: $\gamma(t) = (a + r\cos t, b + r\sin t)$. Tangent vector $\gamma'(t) = (-r\sin t, r\cos t)$ has magnitude $r$ — not unit, but constant. The curvature vector $\gamma''(t)/|\gamma'|^2$ points to the center with magnitude $1/r$.

### Generalization: osculating circle

For any plane curve $\gamma(t)$, at each point there is an osculating (best-fit) circle: same tangent and same curvature. Radius $\rho = 1/\kappa$. Crucial in road design (transitions between straights and curves via clothoids).

### Applications in ML / quant

- **$\ell_2$ regularization (Ridge)**: $\|\beta\|_2 \leq c$ is a Euclidean ball — a circle "head" in the coefficient space. Solution is where the MSE level curve touches the boundary of the circle.
- **Quant**: option pricing with Heston or SABR uses spheres in parameter space for stable calibration regions.
- **Trust Region optimization**: each step is restricted to a Euclidean ball.
- **Manifold learning**: t-SNE, UMAP preserve local neighborhoods (= small circles in the original space).

### Generalization to n-spheres

Volume of the n-sphere: $V_n(r) = \frac{\pi^{n/2}}{\Gamma(n/2 + 1)} r^n$. Concentration of measure: for large $n$, almost all of the volume sits in a thin "shell" near the surface — a key phenomenon in high-dimensional ML.

### Historical curiosity

Descartes (1643) describes the "circle of four circles" (Descartes circle theorem): for 4 mutually tangent circles with curvatures $k_1, k_2, k_3, k_4$, $(k_1 + k_2 + k_3 + k_4)^2 = 2(k_1^2 + k_2^2 + k_3^2 + k_4^2)$. Apollonian gaskets are fractals.

### SymPy

```python
import sympy as sp
from sympy import Circle, Point, Line
C = Circle(Point(2, 3), 4)
print(C.equation())                    # (x-2)**2 + (y-3)**2 - 16
L = Line(Point(0, 0), Point(1, 1))
print(C.intersection(L))               # intersection points
print(C.tangent_lines(Point(10, 3)))   # 2 tangent lines
```

<Porta nivel="pratica" titulo="Practical application">

**Recipe: extract info from a circle equation.**

1. **Standard form** $(x-a)^2 + (y-b)^2 = r^2$: read center $(a, b)$ and radius $r$ directly.
2. **General form** $x^2 + y^2 + Dx + Ey + F = 0$: complete the square (or use $a = -D/2$, $b = -E/2$, $r^2 = a^2 + b^2 - F$).
3. **Verification**: substitute a point and test — the values must not be absurd.

### Typical problem workflow

1. **Relative position line-circle**: compute $d(\text{center}, \text{line}) = |Aa + Bb + C|/\sqrt{A^2+B^2}$, compare with $r$.
2. **Tangent at a point of the circle**: use $(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$.
3. **External tangent from a point $P$**: length $= \sqrt{|PC|^2 - r^2}$.
4. **Circle through 3 points**: solve 3×3 system with $x^2 + y^2 + Dx + Ey + F = 0$.

### Wolfram / SymPy

```python
import sympy as sp
from sympy import Circle, Point
C = Circle(Point(0, 0), 5)
print(C.area)         # 25π
print(C.circumference) # 10π
P = Point(3, 4)
print(P in C)         # True (on the boundary)
```

### When to be careful?

- **General equation might not be a circle**: $D^2/4 + E^2/4 - F < 0$ → empty point set. $= 0$ → single point.
- **Numerical**: completing the square with coefficients near zero is unstable; compute center/radius directly via formulas.
- **Sphere vs circle**: in 3D, watch out — a circle in 3D is the intersection of a plane and a sphere (1D curve); it requires two equations.
