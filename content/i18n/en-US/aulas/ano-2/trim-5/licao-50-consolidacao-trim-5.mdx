### Theorems summary table

| Theorem | Hypothesis | Conclusion |
|---|---|---|
| Squeeze | $g \leq f \leq h$, $\lim g = \lim h = L$ | $\lim f = L$ |
| IVT | $f \in C([a,b])$, $k$ between $f(a), f(b)$ | $\exists c : f(c) = k$ |
| Weierstrass | $f \in C([a,b])$ | $f$ attains max and min |
| Bolzano-Weierstrass | bounded $(a_n)$ | Has convergent subsequence |
| Heine-Cantor | $f \in C([a,b])$ | $f$ uniformly continuous |
| Cauchy | $(a_n)$ Cauchy in $\mathbb{R}$ | Converges |
| Monotone bounded | increasing, bounded above | Converges |

### Next step

**Derivatives (Term 6)** — defined as a limit:
$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}.$$

All Term 5 fluency in limits will be used **directly** there.

### Manipulation cheat sheet

| Form | Technique |
|---|---|
| Polynomial $0/0$ | Factor and cancel |
| $0/0$ with roots | Conjugate |
| Trig $0/0$ | Fundamental limits |
| Rational $\infty/\infty$ | Divide by highest degree |
| $1^\infty$ | $A^B = e^{B \ln A}$ |
| $0 \cdot \infty$ | Rewrite as quotient |
| $\infty - \infty$ | Common factor, conjugate |

### Immediate engineering applications

- **Linear-system stability**: poles in left half-plane $\Leftrightarrow$ $\lim_{t \to \infty} h(t) = 0$.
- **Frequency response**: limits of $H(j\omega)$ as $\omega \to 0, \infty$ give Bode plot.
- **Algorithm analysis**: $O$, $\Omega$, $\Theta$ complexity via limits of ratios.
- **Reliability**: $\text{MTBF} = \int_0^\infty R(t) dt$ — integral convergence via limits.
- **Quantum mechanics**: $\langle \hat A \rangle = \lim_n \langle \psi_n | \hat A | \psi_n \rangle$ in stationary states.

### What you must **master**

- Quick limit computation (sub-30s for common ones).
- Indeterminate-form-type identification (under 5s).
- Fluent algebraic manipulation (factoring, conjugate, log).
- ε-δ arguments (at least for classical cases).
- Application of IVT and Weierstrass for existence.

<Porta nivel="40" titulo="For a senior professional">

Term 5 opens the doors to **functional analysis** and **topology** that you use in ML, optimal control, quantitative finance, theoretical physics.

### Next directions

- **Rigorous real analysis** (Rudin, Lebl): open, closed, compact sets. Bolzano-Weierstrass as sequential compactness.
- **General topology**: limits in non-metric spaces. Filters, nets, Moore-Smith convergence.
- **Functional analysis**: Banach spaces, Hilbert. Bounded operators. Compacts. Spectral theorem.
- **Measure theory**: limits in $L^p$ norm. Convergence a.e., in measure, dominated (Lebesgue).
- **Complex analysis**: limits of holomorphic functions. Identity principle, maximum modulus.

### Structures at play

- **Metrics, norms, semiforms**: multiple topologies can coexist.
- **Strong vs weak vs $w^*$ convergence**: matter in optimization and PDE.
- **Compactification**: $\beta \mathbb{N}$, $\overline{\mathbb{R}}$, Riemann sphere.
- **Completion**: completeness vs Cauchy.

### Applications in ML / RL

- **SGD convergence**: analysis via stochastic approximation (Robbins-Monro, Kushner-Yin).
- **Stability of training**: spectrum of Hessian, Lipschitz constants — analysis via limits in operators.
- **Generalization bounds**: PAC-Bayes, Rademacher complexity — uniform convergence in classes.
- **Bandit / RL convergence**: contraction mappings, Banach.

### Applications in finance

- **Black-Scholes**: PDE with BCs in payoff. Monte Carlo convergence $\to$ price.
- **Hedge convergence**: continuous limit of discrete delta-hedge.
- **Tail risk**: Value-at-Risk, Expected Shortfall via limits in quantiles.

### Final cheat sheet

```
Fundamental limits:
  sin x / x → 1                   (x → 0)
  (1 + 1/x)^x → e                 (x → ∞)
  (e^x - 1) / x → 1               (x → 0)
  (1 - cos x) / x^2 → 1/2         (x → 0)
  ln(1+x) / x → 1                 (x → 0)
  (a^x - 1) / x → ln a            (x → 0)

Manipulation table:
  0/0 polynomial   → factor and cancel
  0/0 roots        → conjugate
  0/0 trig         → fundamentals
  ∞/∞ rational     → divide by highest degree
  1^∞              → e^(B ln A)
  0·∞              → rewrite
  ∞-∞              → common factor

Continuity at a:
  1. f(a) defined
  2. lim exists
  3. lim = f(a)

Types of discontinuity:
  Removable: lim exists, ≠ f(a) or f(a) undef.
  Jump: one-sided exist, differ
  Infinite: ±∞
  Oscillatory: lim doesn't exist

Growth hierarchy:
  log n ≪ n^a ≪ a^n ≪ n! ≪ n^n
```
