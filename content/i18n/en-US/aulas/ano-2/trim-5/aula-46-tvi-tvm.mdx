<Porta nivel="formal" titulo="IVT and variants">

<Definicao titulo="Intermediate Value Theorem (IVT)">
If $f \in C([a, b])$ and $k$ is between $f(a)$ and $f(b)$, then there exists $c \in (a, b)$ such that $f(c) = k$.
</Definicao>

**Special case** ($k = 0$): $f(a) \cdot f(b) < 0 \Rightarrow \exists c \in (a, b) : f(c) = 0$.

### Application 1: Bisection (algorithm)

If $f(a) f(b) < 0$:
1. $m = (a+b)/2$.
2. If $f(m) = 0$, stop.
3. If $f(a) f(m) < 0$, repeat in $[a, m]$.
4. Otherwise, repeat in $[m, b]$.

**Convergence**: error halves per iteration, $|c - m_n| \leq (b-a)/2^n$.

### Application 2: Fixed point

If $g \in C([a, b])$ with $g([a, b]) \subseteq [a, b]$, then $\exists c$ with $g(c) = c$. Apply IVT to $f(x) = g(x) - x$.

### Application 3: Existence of equation solutions

$\cos x = x$, $e^x = 2x$, $\tan x = x$ have solutions by IVT.

### Proof via completeness

Let $S = \{x \in [a, b] : f(x) < k\}$ (assuming $f(a) < k < f(b)$). $S \neq \emptyset$ ($a \in S$) and bounded above. Take $c = \sup S$. By continuity and supremum properties, $f(c) = k$.

### Why continuity is needed

Counterexample: $f(x) = \mathrm{sgn}(x)$ on $[-1, 1]$. $f(-1) = -1$, $f(1) = 1$, but $f$ never takes the value $0.5$. Fails because $f$ is not continuous at 0.

### IVT vs MVT

**IVT**: existence of an **intermediate value** ($f$ continuous).
**MVT** (Mean Value Theorem, Lesson 56): existence of an **intermediate derivative** ($f$ differentiable).

Don't confuse them. MVT requires differentiability; IVT only continuity.

Imagine you're walking from a **low** height (the floor) to a **high** height (a hill). At some moment, you must pass **through every intermediate height** — you can't jump from the floor straight to the top without going through the middle.

<Porta nivel="10" titulo="For a 10-year-old">

**Intermediate value theorem**: if a continuous function starts **negative** and ends **positive**, at some moment it passes through **zero**.

Think of a **thermometer** reading $-2°$ in the morning and $+5°$ in the afternoon. At some moment it read exactly $0°$ (freezing point). It's inevitable — if temperature is continuous (no jumps).

**Application**: discover where an equation has a solution. For example, $f(x) = x^3 - x - 1$. Compute:
- $f(1) = 1 - 1 - 1 = -1$ (negative)
- $f(2) = 8 - 2 - 1 = 5$ (positive)

Since it changes sign, **there's a root between 1 and 2**! IVT guarantees it.

**Bisection method**: take the midpoint $1.5$. $f(1.5) = 0.875 > 0$. So the root is between $1$ and $1.5$. Take the midpoint $1.25$. And so on, **dividing the interval in half**. Each time gets closer.

</Porta>

<Porta nivel="15" titulo="For a 15-year-old teen">

IVT is the theorem that **guarantees the existence of solutions** for equations with a continuous function.

### Practical form

To show that $g(x) = h(x)$ has a solution in $[a, b]$:
1. Define $f(x) = g(x) - h(x)$.
2. Verify $f$ is continuous on $[a, b]$.
3. If $f(a) f(b) < 0$, there's a solution.

### Examples

**$\cos x = x$** on $[0, \pi/2]$:
- $f(x) = \cos x - x$.
- $f(0) = 1 > 0$, $f(\pi/2) = -\pi/2 < 0$.
- Solution exists. (It's $\approx 0.739$.)

**$e^x = 2x$**:
- $f(0) = 1 > 0$, $f(1) = e - 2 \approx 0.72 > 0$. Doesn't change sign here.
- $f(2) = e^2 - 4 \approx 3.39 > 0$. For large values, $e^x$ wins.
- $f(-1) = e^{-1} + 2 \approx 2.37 > 0$.
- For very negative $x$, $e^x \to 0$ and $-2x \to \infty$, so $f \to \infty$. But for $x = 0$, $f = 1$. Still no change. Conclusion: $e^x = 2x$ has no solution in small $\mathbb{R}$; may have for specific $x$.

### Bisection: the canonical application

| Iteration | $a$ | $b$ | $m$ | $f(m)$ | Next |
|---|---|---|---|---|---|
| 0 | 1 | 2 | $1.5$ | + | $[1,\ 1.5]$ |
| 1 | 1 | $1.5$ | $1.25$ | $-$ | $[1.25,\ 1.5]$ |
| 2 | $1.25$ | $1.5$ | $1.375$ | + | $[1.25,\ 1.375]$ |
| 3 | $1.25$ | $1.375$ | $1.3125$ | $-$ | $[1.3125,\ 1.375]$ |

Each iteration halves the error. After 10 iterations, error $< (b-a)/1024$.

### Odd-degree polynomials

**Always have a real root**! For $P(x) = a_n x^n + \ldots$ with odd $n$ and $a_n > 0$:
- $\lim_{x \to +\infty} P = +\infty$
- $\lim_{x \to -\infty} P = -\infty$
- By IVT, a root exists.

</Porta>

<Porta nivel="25" titulo="For an engineering student">

IVT is the **foundational** theorem for numerical analysis and existence methods.

### Proof via completeness

Let $f \in C([a, b])$ with $f(a) < 0 < f(b)$. Define $S = \{x \in [a, b] : f(x) < 0\}$.

- $S$ non-empty: $a \in S$.
- $S$ bounded: $\subseteq [a, b]$.
- By completeness of $\mathbb{R}$, $\exists c = \sup S$, with $c \in [a, b]$.

Claim: $f(c) = 0$.
- If $f(c) < 0$: by continuity, $f < 0$ in a neighborhood of $c$, contradicts $c = \sup$.
- If $f(c) > 0$: by continuity, $f > 0$ in a neighborhood of $c$, but $c = \sup S$, contradiction.

So $f(c) = 0$. ∎

### Why IVT **fails on $\mathbb{Q}$**

$f(x) = x^2 - 2$ continuous on $\mathbb{Q}$, $f(0) = -2 < 0$, $f(2) = 2 > 0$. By classical IVT, root in $(0, 2)$. But the root is $\sqrt 2 \notin \mathbb{Q}$. **Without completeness, IVT fails.**

### Engineering applications

- **Implied volatility** (Black-Scholes Lesson): equation $C(\sigma) = C_{\text{mkt}}$. $C$ continuous and monotone in $\sigma$. IVT + bisection give $\sigma$ in fewer than $30$ iterations.
- **Equilibrium in nonlinear circuits**: $f(V) = i_{\text{in}}$. Solved via Newton (fast) or bisection (robust).
- **Stability analysis**: characteristic polynomial $p(s)$ has a root in the right half-plane? IVT + Routh-Hurwitz.
- **Transfer polynomials**: zeros via Routh, poles via Jury (digital control).

### Bisection: error analysis

After $n$ iterations, $|c - m_n| \leq (b-a)/2^{n+1}$. For error $< \eps$, need $n > \log_2((b-a)/\eps) - 1$.

**Convergence guarantee**: bisection always converges if IVT holds. Other methods (Newton) may diverge.

### Brouwer's fixed point (1D via IVT)

If $g \in C([a, b])$ with $g([a, b]) \subseteq [a, b]$, then $g$ has a fixed point. Proof: define $f(x) = g(x) - x$. $f(a) = g(a) - a \geq 0$ ($g(a) \in [a, b]$). $f(b) = g(b) - b \leq 0$. By IVT, $\exists c : f(c) = 0$, i.e., $g(c) = c$. ∎

**General Brouwer** (dimension $n$): every continuous function $g : B^n \to B^n$ has a fixed point. Non-trivial — basis of equilibrium theory in economics (Nash), in adaptive control.

### Implicit equations

$F(x, y) = 0$ with $F$ continuous and $F(x_0, y_0) = 0$. For each nearby $x$, find $y$ via IVT on $F(x, \cdot)$.

</Porta>

<Porta nivel="40" titulo="For a senior professional">

IVT is a **topological property** more general: image of a connected set by a continuous map is connected.

### Topological generalization

$f : X \to Y$ continuous, $X$ connected $\Rightarrow$ $f(X)$ connected. In $\mathbb{R}$, connected sets are intervals. So $f([a, b])$ is an interval, contains $f(a), f(b)$, and therefore contains all values in between.

### IVT and total order

The proof via $\sup$ depends on $\mathbb{R}$ being complete **and** linearly ordered. In $\mathbb{R}^n$ ($n \geq 2$), there's no total order — literal IVT doesn't hold, but there are generalizations via algebraic topology.

### Generalization: Brouwer fixed point

$g : B^n \to B^n$ continuous has a fixed point. Topological proof:
- Suppose no fixed point. Define $r(x) = $ ray from $x$ to $g(x)$, projected onto the boundary $\partial B^n = S^{n-1}$.
- $r$ is a continuous retraction of $B^n$ onto $S^{n-1}$, contradicts $\pi_{n-1}(B^n) = 0 \neq \pi_{n-1}(S^{n-1})$.

### Applications in economics (Nash equilibrium)

In finite games, Nash equilibrium exists via Brouwer (or Kakutani). The "best response" function is continuous (multivalued), has a fixed point. **Nobel Prize** in Economics for [Nash in 1994](https://www.nobelprize.org/prizes/economic-sciences/1994/summary/).

### Applications in PDE

- **Schaefer fixed point theorem**: extension to compact operators in Banach.
- **Schauder fixed point**: for compact convex sets in Banach.
- **Navier-Stokes solution**: existence via fixed point in appropriate space.

### IVT in complex

$f$ holomorphic has no literal IVT (no order in $\mathbb{C}$), but has the **open mapping**: non-constant $f$ maps open sets to open sets. Combined with the **argument principle**, counts zeros via contour integration.

### Algorithms

- **Bisection**: $O(\log_2 1/\eps)$ — robust, slow.
- **Newton**: $O(\log \log 1/\eps)$ (quadratic) — fast, but may diverge.
- **Brent**: clever combination, best of both worlds.
- **TOMS algorithm 748** (state of the art): bisection + interpolation.

### Numerical in Python

```python
from scipy.optimize import brentq, bisect
import numpy as np
f = lambda x: x**3 - x - 1
print(brentq(f, 1, 2))   # ~1.3247
print(bisect(f, 1, 2))   # ~1.3247
```

<Porta nivel="pratica" titulo="Practical application">

**Recipe: how to apply IVT.**

### To show existence of a root

1. **Identify $f$** (perhaps as a difference $g - h$).
2. **Verify continuity** on $[a, b]$.
3. **Compute $f(a)$ and $f(b)$**. Confirm opposite signs.
4. **Conclude**: there is a root in $(a, b)$.

### Bisection: implementation

```python
def bisect(f, a, b, tol=1e-9):
    assert f(a) * f(b) < 0, "IVT not applicable"
    while b - a > tol:
        m = (a + b) / 2
        if f(m) == 0:
            return m
        if f(a) * f(m) < 0:
            b = m
        else:
            a = m
    return (a + b) / 2

print(bisect(lambda x: x**3 - x - 1, 1, 2))   # ~1.3247
```

### When bisection is the right choice

- **Continuous but not differentiable function**: Newton can't run.
- **Robustness** matters more than speed.
- **Multimodal**: bisection finds **one** root in the interval.

### Cases where IVT doesn't give uniqueness

IVT guarantees **existence**, not **uniqueness**. $f(x) = x^3 - x$ on $[-2, 2]$ has three roots. For uniqueness, monotonicity is needed.

### When IVT does NOT help

- Discontinuous function: IVT doesn't hold.
- $f(a) f(b) > 0$ (same sign): IVT doesn't detect — **but a root may still exist** (undetected).
- Double root: $f(a) = f(b) = 0$ doesn't count (no strict sign change).

### Iterative fixed point

```python
def fixed_point(g, x0, tol=1e-9, max_iter=100):
    x = x0
    for _ in range(max_iter):
        x_new = g(x)
        if abs(x_new - x) < tol:
            return x_new
        x = x_new
    return x

print(fixed_point(np.cos, 0.5))   # ~0.739 (cos x = x)
```

**Caution**: convergence requires $|g'(c)| < 1$ at the fixed point (Banach-Picard).
