### Fundamental trigonometric limits

| Limit | Value |
|---|---|
| $\lim_{x \to 0} \sin x / x$ | $1$ |
| $\lim_{x \to 0} \sin(kx)/x$ | $k$ |
| $\lim_{x \to 0} \tan x / x$ | $1$ |
| $\lim_{x \to 0} (1 - \cos x)/x$ | $0$ |
| $\lim_{x \to 0} (1 - \cos x)/x^2$ | $1/2$ |
| $\lim_{x \to 0} \sin^2 x / x^2$ | $1$ |
| $\lim_{x \to 0} \arcsin x / x$ | $1$ |
| $\lim_{x \to 0} \arctan x / x$ | $1$ |

### Useful identities

- $1 - \cos x = 2\sin^2(x/2)$
- $\sin(a + b) = \sin a \cos b + \cos a \sin b$
- $\cos(a + b) = \cos a \cos b - \sin a \sin b$
- $1 - \cos x = (1 - \cos^2 x)/(1 + \cos x) = \sin^2 x / (1 + \cos x)$

### Techniques

1. **$u$-substitution**: $\sin(2x)/x = 2 \cdot \sin(2x)/(2x) \to 2$.
2. **Identities** to rewrite in fundamental form.
3. **Conjugate**: $1 - \cos x = \sin^2 x / (1 + \cos x)$.
4. **Squeeze**: $-1 \leq \sin x \leq 1$, multiply by something $\to 0$.

### Proof of $\lim (1 - \cos x)/x^2 = 1/2$

$\frac{1 - \cos x}{x^2} = \frac{2 \sin^2(x/2)}{x^2} = \frac{1}{2} \cdot \frac{\sin^2(x/2)}{(x/2)^2} \to \frac{1}{2} \cdot 1 = \frac{1}{2}$.

### Proof of $\lim \tan x / x = 1$

$\tan x / x = (\sin x / x) \cdot (1 / \cos x) \to 1 \cdot 1 = 1$.

</Porta>

<Porta nivel="5" titulo="For a 5-year-old">

When the angle is **very small**, its **sine** is almost equal to the angle itself. Like, $\sin 0.01 \approx 0.01$.

That's why $\sin x / x \to 1$ — they become nearly equal.

And **one minus cosine** is almost equal to **half the square of the angle**. Tiny things follow simple rules.

</Porta>

<Porta nivel="10" titulo="For a 10-year-old">

For **small angles** (in radians):
- $\sin x \approx x$
- $\cos x \approx 1 - x^2/2$
- $\tan x \approx x$

From those come all the limits:
- $\sin(3x)/x \to 3$ (because $\sin(3x) \approx 3x$)
- $(1 - \cos x)/x^2 \to 1/2$ (because $1 - \cos x \approx x^2/2$)
- $\tan x / x \to 1$ (because $\tan x \approx x$)

**Application in physics**: small-oscillation pendulum. The real equation is $\ddot \theta = -(g/L) \sin \theta$. For small $\theta$, $\sin \theta \approx \theta$, and we get $\ddot \theta = -(g/L) \theta$ — simple oscillator.

**Without that approximation**, the problem becomes much harder (involves elliptic integrals). That's why engineers love small angles.

</Porta>

<Porta nivel="15" titulo="For a 15-year-old teen">

Trig limits with $x \to 0$ reduce to three cases via manipulation:

### Case 1: $\sin(?)/?$

If you can write in the form $\sin(\text{something})/\text{something}$, it goes to 1. Use a variable change $u = $ argument.

**Example**: $\lim_{x \to 0} \sin(7x)/(3x) = (7/3) \lim_{x \to 0} \sin(7x)/(7x) = 7/3$.

### Case 2: $(1 - \cos)/?$

Use the identity $1 - \cos x = 2 \sin^2(x/2)$ or conjugate.

**Example**: $\lim_{x \to 0} (1 - \cos x)/(x \sin x) = (x^2/2)/(x \cdot x) = 1/2$ (using $\sin x \approx x$).

### Case 3: combinations $\tan, \sec, \cot$

Rewrite in $\sin, \cos$.

**Example**: $\lim_{x \to 0} \tan(2x)/\sin(3x) = \lim (2x)/(3x) = 2/3$ (approximation).

### Variable-change tricks

For $\lim_{x \to a}$ trig, use $u = x - a$ to bring it to zero.

**Example**: $\lim_{x \to \pi} \sin x / (x - \pi)$. Let $u = x - \pi$. Then $\sin x = \sin(u + \pi) = -\sin u$. Result: $\lim_{u \to 0} -\sin u / u = -1$.

### Approximation table for small $x$

| Function | Approximation (up to $x^3$) |
|---|---|
| $\sin x$ | $x - x^3/6$ |
| $\cos x$ | $1 - x^2/2$ |
| $\tan x$ | $x + x^3/3$ |
| $\arcsin x$ | $x + x^3/6$ |
| $\arctan x$ | $x - x^3/3$ |

Use it to solve faster.

### Challenging example

$\lim_{x \to 0} (\tan x - \sin x)/x^3$:

$\tan x - \sin x = \sin x (1/\cos x - 1) = \sin x \cdot (1 - \cos x)/\cos x$.

So: $\lim = (\sin x / x) \cdot ((1 - \cos x)/x^2) \cdot (1/\cos x) = 1 \cdot (1/2) \cdot 1 = 1/2$.

</Porta>

<Porta nivel="25" titulo="For an engineering student">

Trigonometric limits are essential in **derivatives** ($\sin', \cos', \tan'$), **Fourier series**, **harmonic analysis**.

### Key proofs

**$\lim \sin x / x = 1$**: geometric argument — area of triangle, sector, triangle. By squeeze, conclude.

**$(1 - \cos x)/x^2 \to 1/2$**:
$$\frac{1 - \cos x}{x^2} = \frac{(1-\cos x)(1+\cos x)}{x^2 (1+\cos x)} = \frac{\sin^2 x}{x^2(1+\cos x)} = \frac{\sin^2 x}{x^2} \cdot \frac{1}{1+\cos x} \to 1 \cdot \frac{1}{2} = \frac{1}{2}.$$

**$\arcsin x / x \to 1$**: $u = \arcsin x$, $x = \sin u$, $u \to 0$ as $x \to 0$. $\arcsin x / x = u/\sin u \to 1$.

### Connection to derivatives

- $(\sin x)' = \cos x$ comes from $\lim_{h \to 0} (\sin(x+h) - \sin x)/h$. Manipulation uses $\sin(x+h) = \sin x \cos h + \cos x \sin h$, so the ratio $= \sin x (\cos h - 1)/h + \cos x \sin h / h \to 0 + \cos x = \cos x$.
- $(\cos x)' = -\sin x$ analogously.
- $(\tan x)' = \sec^2 x$ via the quotient.

### Taylor series (Term 7)

$$\sin x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n+1}}{(2n+1)!}, \quad \cos x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n}}{(2n)!}.$$

Truncated, they give small-angle approximations. Uniform convergence on compacts.

### Engineering applications

- **Small angles in mechanics**: pendulum $T = 2\pi\sqrt{L/g}$ holds for $\theta \lesssim 10°$. Relative error grows like $\theta^2/16$.
- **Gaussian (paraxial) optics**: $\sin \theta \approx \theta$. Thin lenses, refraction.
- **Linearization**: dynamic systems $\dot x = f(x)$ linearized via Jacobian at an equilibrium point.
- **Modulation**: $\cos(\omega t + \phi(t))$ with small $\phi$ expanded into AM-equivalent.

### Fourier analysis

Fourier coefficients $a_n = \frac{1}{T}\int f(t) \cos(2\pi nt/T) dt$. Limit $\sum a_n^2$ converges (Parseval). Trig limits appear in pointwise convergence checks (Dini, Dirichlet).

### Advanced squeeze

$\lim_{x \to \infty} \sin x$ does not exist, but $\lim_{x \to \infty} (\sin x)/x = 0$ by squeeze $-1/x \leq \sin x/x \leq 1/x$.

$\lim_{x \to 0} x \sin(1/x) = 0$ by squeeze. **Not** the same as $\lim x \cdot \lim \sin(1/x)$ — the second doesn't even exist. Squeeze solves it without that factoring.

### In AC circuits

- Impedance $Z(j\omega) = R + jX$. Magnitude $|Z| = \sqrt{R^2 + X^2}$. Limits at low/high freq.
- $\sin(\omega t)$ at $\omega = 0$: goes to zero. As $\omega \to \infty$: oscillates, no limit.

</Porta>

<Porta nivel="40" titulo="For a senior professional">

Trigonometric limits are manifestations of deep **analytic structure**: $\sin, \cos$ are entire functions with a specific zero pattern.

### Weierstrass factorization

$$\sin(\pi x) = \pi x \prod_{n=1}^\infty \left(1 - \frac{x^2}{n^2}\right).$$

Taking the log and differentiating gives Mittag-Leffler's identity:
$$\pi \cot(\pi x) = \frac{1}{x} + \sum_{n=1}^\infty \frac{2x}{x^2 - n^2}.$$

Connection with $\zeta(2k)$.

### Functional characterization

$\sin, \cos$ are the unique functions $\mathbb{R} \to \mathbb{R}$ satisfying:
- $f(x+y) = f(x)g(y) + g(x)f(y)$, $g(x+y) = g(x)g(y) - f(x)f(y)$
- $f'(0) = 1$, $g(0) = 1$
- $f, g$ continuous

Equivalent: solution of $y'' + y = 0$ with initial conditions.

### Complex analysis

$e^{ix} = \cos x + i \sin x$ (Euler). Trig limits become limits of $e^{iz}$ as $z \to 0$. Holomorphy in all of $\mathbb{C}$. Hadamard factorization.

### Applications in PDE

- **Wave equation** $u_{tt} = c^2 u_{xx}$: solution $u(x, t) = f(x - ct) + g(x + ct)$ involves $\sin, \cos$ via separation of variables.
- **Free Schrödinger**: $\psi(x, t) = \int A(k) e^{i(kx - \omega t)} dk$, $\omega = k^2/2m$. Wave packets via $\sin, \cos$.
- **Heat with Dirichlet BCs**: solution $\sum b_n \sin(n\pi x/L) e^{-n^2 \pi^2 D t/L^2}$.

### Harmonic analysis

Carleson's theorem: Fourier series converges a.e. for $f \in L^2$. Proof depends on fine estimates involving trig limits.

### Padé approximation

For $\sin x$, the $[m/n]$ Padé approximant can be better than Taylor. In particular, $\tan x \approx x(1 - x^2/15)/(1 - 2x^2/5)$ has smaller error than the truncated Taylor of the same degree.

### Operator calculus

- **Matrix function**: $\sin A = (e^{iA} - e^{-iA})/(2i)$. Well-defined for any matrix.
- **Spectral mapping**: spec($\sin A$) = $\{\sin \lambda : \lambda \in $ spec $A\}$.
- **Application**: solution of $\ddot x = -\Omega^2 x$ is $x(t) = \cos(\Omega t) x_0 + \Omega^{-1} \sin(\Omega t) v_0$.

### Numerical

- **Argument reduction**: for $\sin(10^{15})$, reduce mod $\pi$ with **Payne-Hanek** (high precision).
- **CORDIC**: hardware-friendly, computes $\sin, \cos$ via rotation iteration.
- **Numerical cancellation**: $1 - \cos x$ in float32 for small $x$ has large relative error. Use `2 * sin(x/2)**2` instead.

</Porta>

<Porta nivel="pratica" titulo="Practical application">

**Recipe: trigonometric limits.**

### Algorithm

1. **Identify the form**: $\sin/?, \cos$, $(1 - \cos)/?, \tan/?$, etc.
2. **Rewrite** in fundamental form via:
   - Variable change.
   - Identities $\sin^2 + \cos^2 = 1$, $1 - \cos x = 2\sin^2(x/2)$, etc.
3. **Verify numerically** or via SymPy.

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
# Linear in argument
print(sp.limit(sp.sin(7*x)/x, x, 0))            # 7
# Ratio of sines
print(sp.limit(sp.sin(3*x)/sp.sin(5*x), x, 0))  # 3/5
# (1 - cos)/x^2
print(sp.limit((1 - sp.cos(x))/x**2, x, 0))     # 1/2
# Hard ones
print(sp.limit((sp.tan(x) - sp.sin(x))/x**3, x, 0))  # 1/2
```

### Cheat sheet

For $x \to 0$:

| Form | Limit |
|---|---|
| $\sin(ax)/x$ | $a$ |
| $\sin(ax)/\sin(bx)$ | $a/b$ |
| $\sin(ax)/\tan(bx)$ | $a/b$ |
| $(1 - \cos(ax))/x^2$ | $a^2/2$ |
| $\arcsin(ax)/x$ | $a$ |
| $\arctan(ax)/x$ | $a$ |
| $(\sin x)^2/x^2$ | 1 |
| $\sin(\sin x)/x$ | 1 |

### When NOT to use these

- Limit involves $x \to a \neq 0$: do the change $u = x - a$.
- Function with $1/x$ in the argument ($\sin(1/x)$): take advantage of oscillation — squeeze.

### Common care

- $\sin(2x)/x \neq 2 \sin x / x$: must be $2 \cdot \sin(2x)/(2x) \to 2$.
- $\sin(x^2)/x \neq \sin x \cdot x$: must be $\sin(x^2)/x = x \cdot \sin(x^2)/x^2 \to 0$.
- $(1 - \cos x)/x \neq (1 - \cos x)/x^2$: the first goes to 0, the second to 1/2.
