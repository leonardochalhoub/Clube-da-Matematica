<Porta nivel="formal" titulo="Definitions and theorems">

<Definicao titulo="Continuity at a point">
$f$ is continuous at $a \in \text{dom}(f)$ if all three conditions hold:
1. $\lim_{x \to a} f(x)$ exists.
2. $f(a)$ is defined.
3. $\lim_{x \to a} f(x) = f(a)$.

$f$ is **continuous on** $I$ if continuous at every point of $I$.
</Definicao>

### Direct ε-δ form

$f$ continuous at $a$ $\iff$ $\forall \eps > 0, \exists \delta > 0 : |x - a| < \delta \Rightarrow |f(x) - f(a)| < \eps$.

(Note: here we may have $|x - a| = 0$, that is, $x = a$, since $|f(a) - f(a)| = 0 < \eps$ trivially.)

### Types of discontinuity

| Type | Characterization |
|---|---|
| **Removable** | $\lim$ exists, but $\neq f(a)$ or $f(a)$ undefined |
| **Jump** | One-sided limits exist but differ |
| **Essential** (infinite) | At least one one-sided is $\pm\infty$ |
| **Oscillatory** | One-sided limits do not exist (oscillates) |

### Intermediate Value Theorem (IVT)

If $f \in C([a, b])$ and $k$ is between $f(a)$ and $f(b)$, then $\exists c \in (a, b) : f(c) = k$.

### Weierstrass theorem (extrema)

If $f \in C([a, b])$ (closed and bounded interval), then $f$ attains a **maximum and minimum** on $[a, b]$.

### Uniform continuity

$f$ is uniformly continuous on $I$ if $\forall \eps, \exists \delta$ (the same $\delta$ for every $x$) such that $|x - y| < \delta \Rightarrow |f(x) - f(y)| < \eps$.

**Heine-Cantor**: $f \in C([a, b])$ ⇒ $f$ uniformly continuous.

### Operations preserving continuity

- Sum, product, quotient (denom. $\neq 0$).
- Composition: $f \circ g$ continuous if $g$ continuous at $a$ and $f$ at $g(a)$.
- $\max(f, g)$, $\min(f, g)$, $|f|$.

</Porta>

<Porta nivel="5" titulo="For a 5-year-old">

Imagine **drawing with a pencil on paper**. If you can draw the function **without lifting the pencil**, it's continuous.

If at some point you need to **lift the pencil and put it somewhere else** — there's a **hole** or **jump** — the function is not continuous there.

</Porta>

<Porta nivel="10" titulo="For a 10-year-old">

Continuous function = graph **without holes** and **without jumps**.

- $f(x) = x^2$: continuous on all $\mathbb{R}$. Graph fully connected.
- $f(x) = 1/x$: continuous on $\mathbb{R} \setminus \{0\}$. Hole at $x = 0$ (goes to infinity).
- $f(x) = \lfloor x \rfloor$ (floor): has **jumps** at every integer. Discontinuous at integers.

**Recipe to be continuous at $a$**:
1. $f(a)$ exists.
2. $\lim_{x \to a} f(x)$ exists.
3. The two are **equal**.

If any fails, the function is **discontinuous** at $a$.

**Practical application**: in measurements of temperature, pressure, voltage — nature tends to be continuous. Jumps usually come from **models** (like turn on/off) or measurement errors.

</Porta>

<Porta nivel="15" titulo="For a 15-year-old teen">

Continuity is the **friendly version** of a function: you can "compute by the graph", substitute directly, and the limit **doesn't trick you**.

### The 3 conditions

1. $f(a)$ is defined (value exists at the point).
2. $\lim_{x \to a} f(x)$ exists (limit exists).
3. The two coincide ($\lim = f(a)$).

If **any** fails, $f$ is not continuous at $a$.

### Numerical examples

| Function | Continuous at $a$? | Why |
|---|---|---|
| $f(x) = x^2$, $a = 3$ | Yes | $\lim = 9 = f(3)$ |
| $f(x) = 1/x$, $a = 0$ | No | $f(0)$ undef.; $\lim = \pm \infty$ |
| $f(x) = (x^2-1)/(x-1)$, $a = 1$ | No | $f(1)$ undef.; but $\lim = 2$ (removable) |
| $f(x) = \lfloor x \rfloor$, $a = 2$ | No | one-sided $\lim$ differ (jump) |

### Types of discontinuity

- **Removable**: limit exists, just redefine $f(a)$ to "fix it".
- **Jump**: one-sided differ, irreparable.
- **Infinite**: goes to $\pm\infty$ (vertical asymptote).

### IVT (Intermediate Value Theorem)

If $f$ is continuous on $[a, b]$ and $f(a) < 0 < f(b)$, then $f$ has a root in $(a, b)$.

**Existence guarantee for a root** — basis of the **bisection method** (which goes everywhere: numerical analysis, root finding in engineering).

### Physical application

A particle's position is continuous (no teleportation). Velocity may not be (shock, sudden brake = discontinuous derivative, but position remains continuous).

</Porta>

<Porta nivel="25" titulo="For an engineering student">

Continuity is the **minimal regularity** that guarantees operational properties: IVT, Weierstrass, Riemann integration, etc.

### Proof: sum of continuous is continuous

Let $f, g$ be continuous at $a$. For $\eps > 0$:
- $\exists \delta_1 : |x - a| < \delta_1 \Rightarrow |f(x) - f(a)| < \eps/2$.
- $\exists \delta_2 : |x - a| < \delta_2 \Rightarrow |g(x) - g(a)| < \eps/2$.

Take $\delta = \min(\delta_1, \delta_2)$. Then $|(f+g)(x) - (f+g)(a)| < \eps$. ∎

### Proof of IVT (sketch)

Let $S = \{x \in [a, b] : f(x) < k\}$. $S$ non-empty ($a \in S$ if $f(a) < k$) and bounded. Take $c = \sup S$. By continuity of $f$ and the supremum property, $f(c) = k$.

Details: uses **completeness** of $\mathbb{R}$ (every bounded set has a sup). IVT **fails on $\mathbb{Q}$**: $f(x) = x^2 - 2$ is continuous, $f(0) = -2$, $f(2) = 2$, but there's no rational root.

### Dirichlet function — discontinuous everywhere

$$D(x) = \begin{cases} 1 & x \in \mathbb{Q} \\ 0 & x \notin \mathbb{Q} \end{cases}$$

In any $a$, the neighborhood contains rationals and irrationals → limit doesn't exist → discontinuous.

### Thomae's function — continuous at the irrationals

$$T(x) = \begin{cases} 1/q & x = p/q \in \mathbb{Q} \text{ irreducible} \\ 0 & x \notin \mathbb{Q} \end{cases}$$

Surprising: continuous at every irrational, discontinuous at every rational.

### Engineering applications

- **Stable causal system**: impulse response $h(t)$ continuous → output $y = h * u$ continuous if input $u$ is continuous (convolution preserves continuity).
- **Transfer function with no poles on the $j\omega$ axis**: $|H(j\omega)|$ is a continuous function of $\omega$ — used in Bode diagrams.
- **Optimization**: Weierstrass guarantees the existence of a minimum on compacts. Without it, search methods can diverge.
- **Implicit equations**: $F(x, y) = 0$. If $F$ is continuous and $F(x_0, y_0) = 0$, with $\partial F/\partial y \neq 0$, there's a local continuous $y(x)$ (implicit function theorem — Term 6).

### Uniform continuity — when it matters

In **numerical integration**, uniform continuity on $[a, b]$ guarantees uniform-mesh Riemann convergence. Heine-Cantor automates this for compact $[a, b]$.

### Lipschitz continuity

$|f(x) - f(y)| \leq L |x - y|$ — strong case of uniform continuity. Used in ODEs (Picard-Lindelöf guarantees existence/uniqueness), in neural networks (regularization), in optimization (gradient descent).

</Porta>

<Porta nivel="40" titulo="For a senior professional">

Continuity is the **operational basis** of topology and functional analysis.

### Topological definition

$f: X \to Y$ is continuous if the **preimage** of every open set in $Y$ is open in $X$. Equivalent to the $\eps$-$\delta$ definition in metric spaces.

### Sequential characterization (Heine)

$f$ continuous at $a$ $\iff$ for every sequence $x_n \to a$, $f(x_n) \to f(a)$. Holds in metric spaces; in general topological spaces, requires **nets** or **filters**.

### Compactness preserved

$f$ continuous, $K$ compact ⇒ $f(K)$ compact. **Weierstrass** is a special case: $f$ continuous, $[a,b]$ compact ⇒ $f([a,b])$ is compact in $\mathbb{R}$ ⇒ contains max/min.

### Connectedness preserved

$f$ continuous, $C$ connected ⇒ $f(C)$ connected. **IVT** is a special case in $\mathbb{R}$: $[a,b]$ is connected, so $f([a,b])$ is an interval (connected in $\mathbb{R}$).

### Function spaces

- $C(K)$ = continuous functions on compact $K$ is a Banach space with $\|f\|_\infty = \sup_K |f|$.
- $C^k$, $C^\infty$ — degrees of regularity.
- **Stone-Weierstrass**: polynomials dense in $C([a,b])$ in the sup norm.
- **Arzelà-Ascoli**: characterizes compacts in $C(K)$ via equicontinuity.

### Continuous operators

In Banach spaces, $T : X \to Y$ linear is continuous $\iff$ bounded: $\sup_{\|x\|=1} \|Tx\| < \infty$. Operational topology $\to$ classical functional analysis.

### Sobolev functions

For PDE, $H^1$, $H^2$ etc. relax classical continuity to "weak-sense continuity". Navier-Stokes solution lives in $H^1$ — pointwise continuity not required.

### Applications in ML / robotics

- ReLU neural networks are continuous but not differentiable at zero. Subgradient preserves analysis.
- Continuous policy in RL (not discrete): allows deterministic gradient (DDPG, SAC).
- **Lipschitz constant** of a network: bounds how much the output can vary — used in GANs (Wasserstein), in adversarial robustness.

</Porta>

<Porta nivel="pratica" titulo="Practical application">

**Recipe: how to verify continuity.**

### Algorithm

1. **Is $f(a)$ defined?** If not, $f$ is discontinuous (or removable, if $\lim$ exists).
2. **Does $\lim_{x \to a} f(x)$ exist?** Compute one-sided limits. If they differ, jump.
3. **$\lim = f(a)$?** If not, removable.

### Common discontinuity symptoms

| Symptom | Type |
|---|---|
| Denominator zero at $a$ | Infinite or removable (depends on numerator) |
| Piecewise function changes at $a$ | Check one-sided |
| $\sin(1/x)$, $\cos(1/x)$ | Oscillatory at $x = 0$ |
| Floor $\lfloor x \rfloor$ | Jump at integers |

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
f = (x**2 - 4)/(x - 2)
# Continuity at x = 2?
limit = sp.limit(f, x, 2)        # 4
value = f.subs(x, 2)              # nan (0/0)
print(limit, value)
# Fix: define f(2) = 4
```

### Wolfram Alpha

`continuity of (x^2-4)/(x-2) at x=2` → shows removable discontinuity.

### When discontinuity doesn't matter

In **integration**, isolated discontinuities (set of measure zero) don't affect the integral's value. Function continuous almost everywhere is Lebesgue integrable.

### When it matters a lot

- ODE solution: small discontinuity in the source term → solution may not exist/unique.
- Control: discontinuity in setpoint causes saturation, oscillation.
- Numerical algorithms: bisection needs continuity for IVT to hold.
