### The 3 fundamentals

1. $\displaystyle\lim_{x \to 0} \frac{\sin x}{x} = 1$ (geometric — squeeze).
2. $\displaystyle\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$ (defines $e$).
3. $\displaystyle\lim_{x \to 0} \frac{e^x - 1}{x} = 1$ (derives from the previous).

### Useful variants (Table)

| Limit | Value |
|---|---|
| $\lim_{x \to 0} \sin x / x$ | $1$ |
| $\lim_{x \to 0} \sin(kx)/x$ | $k$ |
| $\lim_{x \to 0} \tan x / x$ | $1$ |
| $\lim_{x \to 0} (1-\cos x)/x^2$ | $1/2$ |
| $\lim_{x \to 0} (1-\cos x)/x$ | $0$ |
| $\lim_{x \to 0} \arcsin x / x$ | $1$ |
| $\lim_{x \to 0} \arctan x / x$ | $1$ |
| $\lim_{x \to 0} (e^x - 1)/x$ | $1$ |
| $\lim_{x \to 0} (a^x - 1)/x$ | $\ln a$ |
| $\lim_{x \to 0} \ln(1+x)/x$ | $1$ |
| $\lim_{x \to 0} ((1+x)^a - 1)/x$ | $a$ |
| $\lim_{x \to \infty} (1 + a/x)^x$ | $e^a$ |
| $\lim_{x \to 0^+} x^x$ | $1$ |
| $\lim_{x \to 0^+} x \ln x$ | $0$ |

### Common indeterminate forms (7)

$0/0$, $\infty/\infty$, $\infty - \infty$, $0 \cdot \infty$, $1^\infty$, $0^0$, $\infty^0$ — all solvable via manipulation or L'Hôpital (Lesson 64).

### Techniques

- **Factor and cancel** (polynomial $0/0$).
- **Conjugate** ($0/0$ with roots).
- **Trigonometric**: use $\sin x \approx x$ (small $x$).
- **Variable change**: $u = 1/x$, $u = x - a$.
- **Logarithm**: $A^B = e^{B \ln A}$ converts $1^\infty$, $0^0$ into $0/0$.

<Porta nivel="5" titulo="For a 5-year-old">

There are three **magic** limits that show up all the time:

1. **$\sin x / x$** goes to **one** when $x$ is small (because for tiny angles, sine and the angle are almost equal).
2. **$(1 + 1/x)^x$** goes to a special number called **$e$** when $x$ is large (about $2.71...$).
3. **$(e^x - 1)/x$** also goes to **one** when $x$ is small.

Those three are like the **three magic times-tables** of limits.

</Porta>

<Porta nivel="10" titulo="For a 10-year-old">

When you have $0/0$ or $\infty/\infty$, the limit **can be anything** — it depends. But three fundamental limits always give the same answer:

1. **$\sin x / x \to 1$**: for a small angle, sine almost equals the angle (in radians).
2. **$(1 + 1/n)^n \to e$**: number $e \approx 2.71828...$ — appears in compound interest.
3. **$(e^x - 1)/x \to 1$**: derives from the previous.

**Practical use**: you want to compute $\lim_{x \to 0} \sin(5x)/x$. Rewrite: $5 \cdot \sin(5x)/(5x)$. As $x \to 0$, $5x \to 0$, and $\sin(5x)/(5x) \to 1$. Answer: $5$.

**Interest trick**: depositing at rate $r$ per year, compounding $n$ times/year: $(1 + r/n)^n$. Continuous compounding ($n \to \infty$) gives $e^r$.

</Porta>

<Porta nivel="15" titulo="For a 15-year-old teen">

The three fundamental limits are **base formulas** that you combine via algebra to solve any kind of indeterminate form.

### Formula 1: $\lim \sin x / x = 1$

**Why it holds**: by the unit circle, $\sin x < x < \tan x$ for $x \in (0, \pi/2)$. Dividing by $\sin x$: $1 < x/\sin x < 1/\cos x$. By squeeze, $x/\sin x \to 1$, so $\sin x / x \to 1$.

**Variants**: $\lim \sin(kx)/x = k$ (change variable $u = kx$).

### Formula 2: $\lim (1 + 1/x)^x = e$

**Why it holds**: defines the number $e \approx 2.71828$. Appears in continuous compound interest, exponential decay, normal distribution.

**Variants**:
- $\lim (1 + a/x)^x = e^a$
- $\lim (1 + x)^{1/x} = e$ (with $x \to 0$)

### Formula 3: $\lim (e^x - 1)/x = 1$

**Why it holds**: derivative of $e^x$ at $x = 0$ is $e^0 = 1$. Connection with derivative via definition $f'(0) = \lim (f(x) - f(0))/x$.

**Variants**:
- $\lim (a^x - 1)/x = \ln a$
- $\lim \ln(1+x)/x = 1$
- $\lim ((1+x)^a - 1)/x = a$

### Approximation table for small $x$

| Function | Approximation |
|---|---|
| $\sin x$ | $x - x^3/6$ |
| $\cos x$ | $1 - x^2/2$ |
| $\tan x$ | $x + x^3/3$ |
| $e^x$ | $1 + x + x^2/2$ |
| $\ln(1+x)$ | $x - x^2/2$ |
| $(1+x)^a$ | $1 + ax$ |

**Application**: small angles in physics (pendulum, optics), continuous interest, radioactive decay.

</Porta>

<Porta nivel="25" titulo="For an engineering student">

Fundamental limits are a consequence of the **analytic structure** of elementary functions — derivative and Taylor series.

### Proof of $\lim \sin x / x = 1$

Geometric argument (Cauchy 1821):

For $x \in (0, \pi/2)$, consider the triangle of area $\frac{1}{2}\sin x$, the circular sector of area $\frac{1}{2}x$, and the outer triangle of area $\frac{1}{2}\tan x$.

$$\frac{\sin x}{2} < \frac{x}{2} < \frac{\tan x}{2} \Rightarrow \cos x < \frac{\sin x}{x} < 1$$

By squeeze, $\sin x / x \to 1$ as $x \to 0^+$. By parity, also valid for $x \to 0^-$. ∎

### Proof of $\lim (1 + 1/n)^n = e$

It's shown that $a_n = (1 + 1/n)^n$ is monotone increasing and bounded, so it converges. The limit **defines** $e$.

By Newton's binomial:
$$\left(1 + \frac{1}{n}\right)^n = \sum_{k=0}^n \binom{n}{k} \frac{1}{n^k} = \sum_{k=0}^n \frac{1}{k!} \cdot \frac{n!}{n^k (n-k)!}$$

As $n \to \infty$, each term $\to 1/k!$, and the sum goes to $\sum_{k=0}^\infty 1/k! = e$.

### Proof of $\lim (e^x - 1)/x = 1$

Use $\lim_{x \to 0} (1 + x)^{1/x} = e$. Take the log: $\ln((1+x)^{1/x}) = \ln(1+x)/x \to 1$. Substituting $y = e^x - 1$ (so $x = \ln(1+y)$): $(e^x - 1)/x = y/\ln(1+y) \to 1$. ∎

### Engineering applications

- **Continuous compounding**: $V(T) = V_0 e^{rT}$, derived from $\lim_n (1 + r/n)^{nT} = e^{rT}$.
- **Radioactive decay**: $N(t) = N_0 e^{-\lambda t}$ — half-life $T_{1/2} = \ln 2 / \lambda$.
- **Small-angle approximation**: pendulum $\ddot \theta + (g/L) \sin \theta = 0$ becomes $\ddot \theta + (g/L) \theta = 0$ (harmonic oscillator), valid up to ~10°.
- **BIBO stability**: $\lim_{n \to \infty} h[n] = 0$ ⇒ stable system.
- **Angular frequency**: $\omega = \lim_{T \to 0} 2\pi / T$, connection between period and frequency.

### Application to the derivative

Derivative of $e^x$ at zero:
$$\frac{d}{dx} e^x \Big|_{x=0} = \lim_{h \to 0} \frac{e^h - e^0}{h} = \lim_{h \to 0} \frac{e^h - 1}{h} = 1.$$

So $(e^x)' = e^x$ everywhere (via $e^{x+h} = e^x e^h$).

### Indeterminate forms via logarithm

$\lim_{x \to 0^+} x^x$: $1^\infty$ or $0^0$? Rewrite: $x^x = e^{x \ln x}$, and $x \ln x \to 0$ (show via squeeze/L'Hôpital). So $x^x \to e^0 = 1$.

</Porta>

<Porta nivel="40" titulo="For a senior professional">

The fundamental limits are **manifestations of deep analytic structures**.

### $\sin x / x$ and entire functions

$\sin x / x$ extends to an **entire function** $\text{sinc}(x)$, with the series
$$\text{sinc}(x) = \prod_{n=1}^\infty \left(1 - \frac{x^2}{n^2 \pi^2}\right).$$
This is the **Weierstrass factorization** of $\sin$. Connection to $\zeta(2) = \pi^2/6$ via the expansion.

### $e$ and algebra

$e = \exp(1)$, where $\exp(z) = \sum z^n/n!$ is the unique entire function with $f' = f$ and $f(0) = 1$. **Unique characterization** by differential equation.

### Generalizations in algebra

- **Matrix exponential**: $e^A = \sum A^n/n!$. Solves $\dot x = Ax$.
- **Lie group exponential**: $\exp : \mathfrak{g} \to G$, mapping algebra to group.
- **Gamma functions**: $\Gamma(z) = \lim_n (n! n^z)/(z(z+1)\cdots(z+n))$ via product.

### Operators and symbolic calculus

- **Umbral / Sheffer calculus**: derivative $\partial$ has "exponential" $e^{a\partial}$ acting as translation by $a$.
- **Heaviside calculus**: $D = d/dt$, $e^{-Dh}$ is translation. Fundamental limits appear in expansion.

### Functional analysis

- $\sinh, \cosh, \sin, \cos$ form a **basis** of the solution space of $y'' = \pm y$.
- $e^{i\omega t}$ is an **eigenvector** of the operator $d/dt$ — the basis of Fourier analysis.

### Rigorous series proof

$\sin x = x - x^3/6 + x^5/120 - \ldots$ Then
$$\frac{\sin x}{x} = 1 - \frac{x^2}{6} + \frac{x^4}{120} - \ldots \to 1.$$

$e^x = 1 + x + x^2/2 + \ldots$ Then
$$\frac{e^x - 1}{x} = 1 + \frac{x}{2} + \frac{x^2}{6} + \ldots \to 1.$$

### Applications in probability

- **Characteristic function**: $\varphi(t) = E[e^{itX}]$. $\varphi(0) = 1$, $\varphi'(0) = i E[X]$.
- **Poisson law**: $P(X = k) = e^{-\lambda} \lambda^k / k!$. Limit of binomial $(1 + \lambda/n)^n$.
- **Stirling's formula**: $n! \sim \sqrt{2\pi n}(n/e)^n$. Asymptotic approximation.

### Renormalization and physics

In QFT, **zeta regularization**: $\sum_{n=1}^\infty n = \zeta(-1) = -1/12$. Fundamental limits appear in analytic continuation.

### Numerical

- Algorithms compute $e^x$, $\sin x$ via truncated series + argument reduction (CORDIC).
- Beware of numerical cancellation in $(e^x - 1)/x$ for small $x$: use `expm1(x)` (built-in in C, Python, MATLAB).

</Porta>

<Porta nivel="pratica" titulo="Practical application">

**Recipe: how to use fundamental limits.**

### Steps for any indeterminate form

1. **Identify the type**: $0/0$, $\infty/\infty$, $1^\infty$, etc.
2. **Try to rewrite** as a fundamental-limit form:
   - $\sin(\text{something}) / \text{something}$ → use $\sin x / x$.
   - $(1 + \text{something})^{1/\text{something}}$ → use $e$.
   - $(e^{\text{something}} - 1) / \text{something}$ → use $(e^x - 1)/x$.
3. **Change variables** if needed ($u = $ inner argument).
4. **Verify numerically** or via SymPy.

### Typical cases

| Indeterminate | Strategy |
|---|---|
| Trig $0/0$ | Use $\sin x \approx x$, $1 - \cos x \approx x^2/2$ |
| Exp/log $0/0$ | Use $(e^x - 1)/x \to 1$, $\ln(1+x)/x \to 1$ |
| $1^\infty$ | $A^B = e^{B \ln A}$, manipulate $B \ln A$ |
| $0^0$ or $\infty^0$ | Same: $A^B = e^{B \ln A}$, $\ln A \to -\infty$ or $+\infty$ |
| $\infty - \infty$ | Common factor, conjugate, or common rational |

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
print(sp.limit(sp.sin(5*x)/x, x, 0))           # 5
print(sp.limit((1 + 2/x)**x, x, sp.oo))        # exp(2)
print(sp.limit((sp.exp(3*x) - 1)/x, x, 0))     # 3
print(sp.limit(x**x, x, 0, '+'))               # 1
print(sp.limit((1 - sp.cos(x))/x**2, x, 0))    # 1/2
```

### Wolfram Alpha

`limit (1 + 3/x)^x as x -> infinity` → $e^3$.

### When NOT to use fundamental limits

- **$x \to a \neq 0$** with a trig expression: direct substitution may work.
- **Functions unrelated to $\sin, e, \ln$**: other techniques.

### Care

- The **argument** of $\sin$ must equal the denominator. $\sin(5x)/x \neq 1$ — it is $5$.
- $(1 + 1/n)^n$ vs $(1 + 2/n)^n$ vs $(1 + 1/n)^{2n}$ — all give different things.
