<Porta nivel="formal" titulo="Operational properties">

Let $\lim_{x \to a} f(x) = L$ and $\lim_{x \to a} g(x) = M$. Then:

| Operation | Result |
|---|---|
| $\lim (f \pm g)$ | $L \pm M$ |
| $\lim (cf)$, $c$ const. | $cL$ |
| $\lim (fg)$ | $LM$ |
| $\lim (f/g)$, $M \neq 0$ | $L/M$ |
| $\lim f^n$, $n \in \mathbb{N}$ | $L^n$ |
| $\lim \sqrt[n]{f}$ | $\sqrt[n]{L}$ (when defined) |
| $\lim |f|$ | $|L|$ |

### Composition

If $\lim_{x \to a} g(x) = b$ and $f$ is continuous at $b$, then $\lim_{x \to a} f(g(x)) = f(b)$.

**Caution**: without continuity, composition can fail. Counterexample: $g(x) = 0$ constant and $f$ with a removable discontinuity at 0.

### Squeeze theorem (Sandwich)

If $g(x) \leq f(x) \leq h(x)$ in a neighborhood of $a$ (except possibly at $a$) and $\lim g = \lim h = L$, then $\lim f = L$.

### Classic application

$\lim_{x \to 0} x \sin(1/x) = 0$ by squeeze: $-|x| \leq x \sin(1/x) \leq |x|$ and $\lim |x| = 0$.

### When it doesn't work

- **0/0, âˆž/âˆž**: indeterminate forms, require manipulation.
- **Denominator â†’ 0**: may give $\pm \infty$ or not exist.
- **Composition with discontinuous $f$**: requires care.

</Porta>

<Porta nivel="10" titulo="For a 10-year-old">

Limits have **good manners** with operations. If you know where two things go, you also know where the sum, product, and quotient go.

**Simple rules**:

- Sum of two going to $L$ and $M$ goes to $L + M$.
- Multiplying by a constant: the constant "passes outside".
- Product: $L \cdot M$.
- Quotient: $L / M$, **as long as $M$ is not zero** (because dividing by zero is forbidden).

**Sandwich trick**: imagine you're squeezed between two equal things. You can only be equal too. If $g \leq f \leq h$ and $g, h$ go to the same place, $f$ goes along.

Example: $-1 \leq \sin(1/x) \leq 1$. Multiply by small positive $x$: $-x \leq x \sin(1/x) \leq x$. As $x \to 0$, both go to 0, so $x \sin(1/x) \to 0$.

</Porta>

<Porta nivel="15" titulo="For a 15-year-old teen">

Limits "behave well" with sum, product, quotient â€” provided **both limits exist** and the denominator is non-zero.

### Practical recipe

1. **Substitute directly**. If you get a number, that's the limit.
2. **If you get an indeterminate form ($0/0$, $\infty/\infty$)**, manipulate:
   - Factor and cancel.
   - Multiply by the conjugate.
   - Use notable limits ($\sin x / x \to 1$).
3. **Squeeze** (sandwich): if you can squeeze the function between two whose limits you know, you find the function's.

### Common manipulation table

| Indeterminate | Technique |
|---|---|
| Polynomial $0/0$ | Factor and cancel |
| $0/0$ with roots | Conjugate |
| Rational $\infty/\infty$ | Divide by highest degree |
| $0 \cdot \infty$ | Rewrite as $0/0$ or $\infty/\infty$ |
| $1^\infty$, $0^0$ | $A^B = e^{B \ln A}$ |

### Combined example

$\lim_{x \to 2} \frac{x^2 - 4}{x - 2}$: indeterminate $0/0$. Factor: $\frac{(x-2)(x+2)}{x-2} = x + 2 \to 4$.

### Why squeeze is powerful

You don't even need to know how to **compute** $f$ â€” just **trap** it between two easy functions. Common in series and oscillations.

</Porta>

<Porta nivel="25" titulo="For an engineering student">

Algebraic properties turn limit computation into **algebra**: you manipulate expressions instead of proving via Îµ-Î´ each time.

### Proof of the sum property

Given $\lim f = L$, $\lim g = M$. For $\eps > 0$:
- There exists $\delta_1$ : $|f - L| < \eps/2$ if $|x - a| < \delta_1$.
- There exists $\delta_2$ : $|g - M| < \eps/2$ if $|x - a| < \delta_2$.
- Take $\delta = \min(\delta_1, \delta_2)$. Then $|f + g - (L + M)| \leq |f - L| + |g - M| < \eps$. âˆŽ

### Proof of the product

More subtle â€” uses $|fg - LM| \leq |f||g - M| + |M||f - L|$ and local boundedness of $f$.

### Squeeze: proof

Given $\eps$, choose $\delta$ such that $|g - L|, |h - L| < \eps$. Since $g \leq f \leq h$:
$$L - \eps < g \leq f \leq h < L + \eps \Rightarrow |f - L| < \eps. \quad \blacksquare$$

### Engineering applications

- **Circuit analysis**: $\lim_{s \to 0} H(s) = $ DC gain. If $H = N/D$, it is $N(0)/D(0)$ via the quotient property (assuming $D(0) \neq 0$).
- **Signals and systems**: BIBO stability via $\lim_{n \to \infty} h[n] = 0$ analyzed by squeeze with exponential decay.
- **Probability**: convergence in distribution via continuity â€” $X_n \to X \Rightarrow E[g(X_n)] \to E[g(X)]$ for bounded continuous $g$.

### When indeterminate forms appear

$0/0$ is the **most useful** case â€” it means numerator and denominator have a common root. Cancel the common root (L'HÃ´pital is a special case).

### Limit â†” other operations commutation

- $\lim$ commutes with **continuous functions**: $\lim f(g(x)) = f(\lim g(x))$.
- $\lim$ commutes with **finite sum**: trivially.
- $\lim$ does NOT commute with **infinite sum** (series) without uniform conditions.
- $\lim$ does NOT commute with **integral** without theorems (Lebesgue, monotone, dominated).

</Porta>

<Porta nivel="40" titulo="For a senior professional">

Algebraic properties reflect that $\lim$ is a **continuous linear functional** in appropriate spaces.

### Algebraic structure

Consider $\mathcal{C}_a = \{f : \lim_{x \to a} f \text{ exists}\}$. Then:
- $\mathcal{C}_a$ is an **algebra** (closed under sum, product).
- $\lim_a : \mathcal{C}_a \to \mathbb{R}$ is an **algebra homomorphism** (preserves $+, \cdot, $ scalar multiplication).
- The kernel $\{f : \lim_a f = 0\}$ is an **ideal** of $\mathcal{C}_a$.

### Generalization: limits over nets (Moore-Smith)

In general topology, "tendency" uses **nets** ($x : I \to X$ where $I$ is directed) or **filters**. Algebraic properties are preserved when the codomain is a topological vector space.

### Limit commutation: classic counterexamples

$$\lim_{m \to \infty} \lim_{n \to \infty} \frac{m}{m+n} = 1, \quad \lim_{n \to \infty} \lim_{m \to \infty} \frac{m}{m+n} = 1$$

Here it works. But with $a_{m,n} = m/(m+n) - n/(m+n)$ order matters. **Moore's theorem**: commutes under uniform convergence.

### Differential and integral operators

The passage to the limit is the basis of:
- **Continuity of derivative operator** $D$ in $C^1[a,b]$ (not in $C[a,b]$).
- **Riemann integral**: $\int = \lim$ of sums, algebraic properties â†’ linearity.
- **Laplace transform**: $\mathcal{L}\{f + g\} = \mathcal{L}\{f\} + \mathcal{L}\{g\}$ via properties.

### Non-standard analysis

In $^*\mathbb{R}$, algebraic properties transfer via the **transfer principle**: every first-order sentence true in $\mathbb{R}$ is true in $^*\mathbb{R}$.

### Numerical and ML

In deep learning, **chain rule** is an application of limit composition â€” the derivative of the composite uses continuity. Backpropagation depends on algebraic properties to decompose gradients into products of Jacobians.

</Porta>

<Porta nivel="pratica" titulo="Practical application">

**Recipe: how to apply properties to any limit.**

### Computation algorithm

1. **Direct substitution**: $f(a)$ exists and finite? That's the limit (if $f$ continuous).
2. **Indeterminate**? Identify the type: $0/0$, $\infty/\infty$, $\infty - \infty$, $0 \cdot \infty$, $1^\infty$, $0^0$, $\infty^0$.
3. **Manipulate** by type.
4. **Verify** with Wolfram/SymPy.

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
# Sum
print(sp.limit(sp.sin(x) + x, x, 0))           # 0
# Product
print(sp.limit(x * sp.sin(1/x), x, 0))         # 0 (squeeze)
# Quotient with cancellation
print(sp.limit((x**2 - 4)/(x - 2), x, 2))      # 4
# Composition
print(sp.limit(sp.exp(sp.sin(x)/x), x, 0))     # E (sin x/x â†’ 1)
```

### Cases where properties DON'T apply directly

- **Different one-sided limits**: the sum property requires both limits to exist.
- **Denominator â†’ 0**: $\lim (1/g)$ may give $\pm \infty$ or oscillate.
- **Composition with discontinuous**: $\lim f(g)$ may fail if $f$ is not continuous at $\lim g$.

### Squeeze in practice

When you see $\sin$, $\cos$, or oscillating function multiplied by something going to zero: **squeeze** almost always solves it.

Example: $\lim_{x \to \infty} (\sin x)/x$: $-1/x \leq \sin x / x \leq 1/x$, both $\to 0$, so $\to 0$.

Fluency with properties is what separates those who **know limits** from those who **memorize rules**.
