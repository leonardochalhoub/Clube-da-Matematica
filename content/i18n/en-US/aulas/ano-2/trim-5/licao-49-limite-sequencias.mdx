<Porta nivel="formal" titulo="Sequences, convergence, and theorems">

<Definicao titulo="Sequence limit (ε-N)">
$(a_n) \to L$ if: $\forall \eps > 0, \exists N \in \mathbb{N} : n \geq N \Rightarrow |a_n - L| < \eps$.
</Definicao>

### Key theorems

| Theorem | Statement |
|---|---|
| **Uniqueness** | The limit, if it exists, is unique |
| **Arithmetic** | $\lim(a_n \pm b_n) = \lim a_n \pm \lim b_n$, etc. |
| **Squeeze** | $a_n \leq b_n \leq c_n$, $\lim a_n = \lim c_n = L \Rightarrow \lim b_n = L$ |
| **Monotone bounded** | Increasing bounded above converges |
| **Bolzano-Weierstrass** | A bounded sequence has a convergent subsequence |
| **Cauchy** | $a_n$ converges $\iff$ it is Cauchy |

### Cauchy

<Definicao titulo="Cauchy sequence">
$(a_n)$ is **Cauchy** if $\forall \eps > 0, \exists N : m, n \geq N \Rightarrow |a_m - a_n| < \eps$.
</Definicao>

**In $\mathbb{R}$**: Cauchy $\iff$ convergent (completeness).
**In $\mathbb{Q}$**: not every Cauchy converges ($a_n$ approximating $\sqrt 2$).

### Monotonicity + boundedness

- $(a_n)$ increasing and bounded above $\Rightarrow$ converges to $\sup a_n$.
- $(a_n)$ decreasing and bounded below $\Rightarrow$ converges to $\inf a_n$.

### Subsequences

$(a_{n_k})$ is a subsequence of $(a_n)$ if $n_1 < n_2 < \ldots$.
$\lim a_n = L \Rightarrow \lim a_{n_k} = L$ for every subsequence.

### Divergent sequences

- $a_n = (-1)^n$: oscillates between $-1$ and $1$. Subsequences converge (to $\pm 1$), but $a_n$ does not.
- $a_n = n$: grows without bound ($\to +\infty$).
- $a_n = n \sin(n)$: irregular behavior, no limit.

### Recursive sequences

$a_{n+1} = f(a_n)$, $a_0$ given. The limit, if it exists, is a **fixed point** of $f$: $L = f(L)$.

</Porta>

<Porta nivel="5" titulo="For a 5-year-old">

Imagine a line of numbers: $1, 1/2, 1/3, 1/4, \ldots$ — they keep **getting smaller** and approaching **zero**.

The limit is the "**where it's going**" of the line. If the line goes somewhere, that place is the limit.

Some lines don't go anywhere (they keep jumping), and then there's no limit.

</Porta>

<Porta nivel="10" titulo="For a 10-year-old">

**Sequence** = list of numbers: $a_1, a_2, a_3, \ldots$

**Limit** $L$: the number the sequence **arrives at infinity**. For example:
- $1, 1/2, 1/3, \ldots$ → goes to **zero**.
- $1, 4, 9, 16, \ldots$ → goes to **$\infty$** (grows).
- $1, -1, 1, -1, \ldots$ → doesn't go anywhere (alternating).

**Rigorous definition**: $a_n \to L$ if you can stay **as close to $L$ as you want**, just go far enough in the sequence. For any "bubble" $\eps$ around $L$, there's an $N$ such that **all** terms after $a_N$ lie in the bubble.

**Useful trick**: if $a_n$ is always **growing** and has a **ceiling**, it **converges**! (Monotone bounded.) You don't even need to know the limit — just that it exists.

</Porta>

<Porta nivel="15" titulo="For a 15-year-old teen">

Sequences are **functions with domain $\mathbb{N}$**. The limit theory is the same as the ε-δ in Lesson 41, but we use $N$ instead of $\delta$.

### Why sequences matter

- **Newton iteration** for roots: $a_{n+1} = a_n - f(a_n)/f'(a_n)$.
- **Series** = limit of partial sums: $\sum a_n = \lim s_N$.
- **Integral** = limit of Riemann sums.
- **Probability**: law of large numbers $\bar X_n \to E[X]$.

### Notable sequence limits

| Sequence | Limit |
|---|---|
| $1/n$ | $0$ |
| $(1 + 1/n)^n$ | $e$ |
| $n^{1/n}$ | $1$ |
| $n!/n^n$ | $0$ |
| $a^n$ ($|a| < 1$) | $0$ |
| $a^n$ ($a > 1$) | $\infty$ |
| $(-1)^n$ | doesn't exist |

### Growth hierarchy (important!)

$$\log n \ll n^a \ll a^n \ll n! \ll n^n.$$

Each grows **much slower** than the next. A tool used all the time in algorithm complexity.

### Operational theorems

**Squeeze**: pinch $b_n$ between $a_n$ and $c_n$ that go to the same place.

**Example**: $\lim_{n \to \infty} \sin n / n$. $-1/n \leq \sin n / n \leq 1/n$. By squeeze, $\to 0$.

**Monotone bounded**: an increasing and bounded sequence converges — you don't need to find the limit, just know it exists!

**Example**: $a_1 = 1$, $a_{n+1} = (a_n + 2/a_n)/2$. Show that $a_n$ is decreasing after $a_1$, bounded below by $\sqrt 2$. Then it converges. The limit $L$ satisfies $L = (L + 2/L)/2$, that is, $L = \sqrt 2$.

### Bolzano-Weierstrass

**Every bounded sequence has a convergent subsequence**. Useful for existence arguments (compactness).

</Porta>

<Porta nivel="25" titulo="For an engineering student">

Sequences are **the basis of iterative algorithms** — Newton, gradient descent, EM, MCMC.

### ε-N proof: $\lim 1/n = 0$

Given $\eps > 0$, choose $N = \lceil 1/\eps \rceil + 1$. Then $n \geq N \Rightarrow 1/n \leq 1/N < \eps$. ∎

### Proof: monotone bounded converges

Let $(a_n)$ be increasing, bounded above. Take $L = \sup\{a_n\}$ (exists by completeness). For $\eps > 0$, $L - \eps$ is not an upper bound, so $\exists N : a_N > L - \eps$. Since increasing, $a_n \geq a_N > L - \eps$ for $n \geq N$. Since $a_n \leq L$, $|a_n - L| < \eps$. ∎

### Proof: Bolzano-Weierstrass (sketch)

Bounded sequence $(a_n) \subseteq [a, b]$. Bisect: some half contains infinitely many terms. Repeat. Construct subsequence $(a_{n_k})$ with $a_{n_k}$ in the $k$-th half, interval of length $(b-a)/2^k$. Cauchy, hence convergent. ∎

### Proof: Cauchy ⇔ convergent in $\mathbb{R}$

$(\Rightarrow)$ Convergent is Cauchy: $|a_m - a_n| \leq |a_m - L| + |L - a_n| < \eps$.

$(\Leftarrow)$ Cauchy is convergent: Cauchy $\Rightarrow$ bounded $\Rightarrow$ has convergent subsequence (Bolzano-Weierstrass) $\Rightarrow$ the whole sequence converges to the same limit (because it's Cauchy).

### Cauchy sequences in $\mathbb{Q}$

$(a_n)$ approximating $\sqrt 2$ by bisection is Cauchy in $\mathbb{Q}$, but **does not converge** in $\mathbb{Q}$ (limit is irrational). $\mathbb{R}$ is the **completion** of $\mathbb{Q}$ under the metric $|\cdot|$.

### Algorithm applications

- **Gradient descent**: $w_{n+1} = w_n - \eta \nabla f(w_n)$. Converges if $\eta < 2/L$ (Lipschitz constant), $f$ convex.
- **Newton**: $x_{n+1} = x_n - f(x_n)/f'(x_n)$. **Quadratic** convergence near a simple root.
- **Heron** (square root): $a_{n+1} = (a_n + N/a_n)/2$. Quadratic for $\sqrt N$. Comes from Newton on $f(x) = x^2 - N$.
- **EM**: monotonically increases the likelihood. Converges to a critical point (not necessarily global max).

### Series convergence

$\sum a_n$ converges $\iff$ partial sums $s_N = \sum_{n=1}^N a_n$ form a convergent sequence.

**Tests**:
- Cauchy: $\forall \eps, \exists N : m > n \geq N \Rightarrow |s_m - s_n| < \eps$.
- Ratio (D'Alembert): $\lim |a_{n+1}/a_n| = L < 1 \Rightarrow$ converges.
- Root (Cauchy): $\lim |a_n|^{1/n} = L < 1 \Rightarrow$ converges.

### Law of large numbers

$X_1, X_2, \ldots$ iid with $E[X] = \mu$, $V[X] < \infty$. Then $\bar X_n = (1/n) \sum X_i \to \mu$ (in probability — weak; a.s. — strong).

### Applications in dynamic systems

Discrete system $x_{n+1} = T(x_n)$. Behavior: fixed point, cycle, strange attractor (chaos). **Logistic map** $x_{n+1} = r x_n (1 - x_n)$ exhibits transition to chaos at $r \approx 3.57$.

### Sequences in $\mathbb{R}^n$

Componentwise convergence. In norm: $\|x_n - L\| \to 0$. Equivalent in $\mathbb{R}^n$ (but not in infinite dimension).

<Porta nivel="40" titulo="For a senior professional">

Sequences are central in **functional analysis**, **metric spaces**, **topology**.

### Sequence spaces

- $\ell^p$ = sequences with $\sum |a_n|^p < \infty$. Banach with norm $\|a\|_p = (\sum |a_n|^p)^{1/p}$.
- $\ell^\infty$ = bounded sequences. Banach with $\|a\|_\infty = \sup |a_n|$.
- $c_0$ = sequences $a_n \to 0$. Closed subspace of $\ell^\infty$.

### Weak vs strong convergence

- **Strong**: $\|x_n - x\| \to 0$.
- **Weak**: $f(x_n) \to f(x)$ for every $f \in X^*$ (dual).

In finite dimension, equivalent. In $\ell^2$: orthonormal basis $e_n$ converges weakly to 0 but not strongly ($\|e_n\| = 1$).

### Cauchy in metric spaces

$(X, d)$ is **complete** if every Cauchy converges. Banach = complete normed space. Hilbert = complete inner product space.

**Completion**: every metric space has a unique completion. $\mathbb{R}$ is the completion of $\mathbb{Q}$. $L^2$ is the completion of $C[a,b]$ in the $L^2$ norm.

### Compactness

In metric spaces: **sequentially compact** = every sequence has a convergent subsequence. In $\mathbb{R}^n$, equivalent to closed and bounded (Heine-Borel). **In infinite dimension**, equivalent to closed, bounded **and** totally bounded (Riesz).

### Topology

Sequential characterization of continuity holds in **metric spaces** (Heine). In general, requires nets or filters.

### Compact operators

$T : X \to Y$ is **compact** if it maps the unit ball to a precompact set. Convergent subsequences are key in proofs.

### Spectral theory

Sequence of eigenvalues $\lambda_n$ of a compact self-adjoint operator: either finite, or $\lambda_n \to 0$. Spectral theorem decomposes via eigenvector basis.

### Advanced iterative methods

- **Aitken acceleration**: $\hat a_n = a_n - (a_{n+1} - a_n)^2/(a_{n+2} - 2 a_{n+1} + a_n)$. Accelerates linear convergence.
- **Romberg integration**: Richardson extrapolation in integration.
- **GMRES, Krylov subspaces**: for large linear systems.

### Applications in ML / RL

- **Q-learning convergence**: $Q_n \to Q^*$ under appropriate $\epsilon$-greedy exploration.
- **Policy gradient convergence**: analysis via stochastic approximation (Robbins-Monro).
- **MCMC**: state sequence $\to$ stationary distribution. Convergence via ergodic theorems.

### Categories

Limits in categories generalize sequence limits. Direct/inverse limit, colimit, etc. Same algebraic structure.

</Porta>

<Porta nivel="pratica" titulo="Practical application">

**Recipe: how to compute a sequence limit.**

### Algorithm

1. **Direct substitution**: if $a_n = f(n)$ for a function $f$, compute $\lim_{x \to \infty} f(x)$.
2. **Recursive**: assume limit $L$, solve $L = f(L)$.
3. **Arithmetic**: split into known parts.
4. **Squeeze**: pinch between knowns.
5. **Monotone bounded**: prove existence, compute later (or don't).

### SymPy

```python
import sympy as sp
n = sp.symbols('n', positive=True, integer=True)
print(sp.limit(1/n, n, sp.oo))                # 0
print(sp.limit((1 + 1/n)**n, n, sp.oo))       # E
print(sp.limit(n**(1/n), n, sp.oo))           # 1
print(sp.limit(sp.factorial(n)/n**n, n, sp.oo))  # 0
```

### Growth hierarchy (at the tip of your tongue)

$$\log n \ll n^a \ll a^n \ll n! \ll n^n.$$

Use it to solve ratios: $\lim n!/2^n = \infty$, $\lim 2^n/n! = 0$, $\lim n^{100}/2^n = 0$.

### Recursive sequences

For $a_{n+1} = f(a_n)$:

1. Verify convergence (monotonicity + boundedness, or contraction via Banach).
2. If it converges to $L$, then $L = f(L)$.

```python
import numpy as np
def heron(N, a0=1, n_iter=10):
    a = a0
    for _ in range(n_iter):
        a = (a + N/a)/2
    return a
print(heron(2))   # ~1.41421356 (sqrt 2)
```

### When the limit DOES NOT exist

- **Oscillation**: $(-1)^n$, $\sin n$.
- **Divergence**: $a_n \to \pm \infty$.
- **Non-monotone and non-Cauchy**: chaos.

### Numerical verification

Implementing and plotting $a_n$ vs $n$ is the fastest way to **conjecture** the limit. Then prove it.

### Care

- Recursive limits assume **existence**. Without proof, it can be fallacious.
- $(1 + 1/n)^n \neq 1$ at infinity! Indeterminate $1^\infty$.
- An **eventually increasing** sequence may not be monotone — be careful in proofs.
