<Porta nivel="formal" titulo="Folgen, Konvergenz und Sätze">

<Definicao titulo="Folgengrenzwert (ε-N)">
$(a_n) \to L$, falls $\forall \eps > 0, \exists N \in \mathbb{N} : n \geq N \Rightarrow |a_n - L| < \eps$.
</Definicao>

### Schlüsselsätze

| Satz | Aussage |
|---|---|
| **Eindeutigkeit** | Grenzwert, falls existent, ist eindeutig |
| **Arithmetik** | $\lim(a_n \pm b_n) = \lim a_n \pm \lim b_n$ usw. |
| **Einschnürung** | $a_n \leq b_n \leq c_n$, $\lim a_n = \lim c_n = L \Rightarrow \lim b_n = L$ |
| **Monoton beschränkt** | Wachsende, nach oben beschränkte Folge konvergiert |
| **Bolzano-Weierstrass** | Beschränkte Folge hat konvergente Teilfolge |
| **Cauchy** | $a_n$ konvergiert $\iff$ ist Cauchy |

### Cauchy

<Definicao titulo="Cauchy-Folge">
$(a_n)$ ist **Cauchy**, falls $\forall \eps > 0, \exists N : m, n \geq N \Rightarrow |a_m - a_n| < \eps$.
</Definicao>

**In $\mathbb{R}$**: Cauchy $\iff$ konvergent (Vollständigkeit).
**In $\mathbb{Q}$**: nicht jede Cauchy-Folge konvergiert (z. B. $a_n$, das $\sqrt 2$ approximiert).

### Monotonie + Beschränktheit

- $(a_n)$ wachsend und nach oben beschränkt $\Rightarrow$ konvergiert gegen $\sup a_n$.
- $(a_n)$ fallend und nach unten beschränkt $\Rightarrow$ konvergiert gegen $\inf a_n$.

### Teilfolgen

$(a_{n_k})$ ist Teilfolge von $(a_n)$, wenn $n_1 < n_2 < \ldots$.
$\lim a_n = L \Rightarrow \lim a_{n_k} = L$ für jede Teilfolge.

### Divergente Folgen

- $a_n = (-1)^n$: oszilliert zwischen $-1$ und $1$. Teilfolgen konvergieren (zu $\pm 1$), aber $a_n$ nicht.
- $a_n = n$: wächst unbegrenzt ($\to +\infty$).
- $a_n = n \sin(n)$: irreguläres Verhalten, kein Grenzwert.

### Rekursive Folgen

$a_{n+1} = f(a_n)$, $a_0$ gegeben. Der Grenzwert (falls existent) ist **Fixpunkt** von $f$: $L = f(L)$.

</Porta>

<Porta nivel="5" titulo="Für ein 5-jähriges Kind">

Stell dir eine Reihe von Zahlen vor: $1, 1/2, 1/3, 1/4, \ldots$ — sie werden **kleiner** und nähern sich der **Null**.

Der Grenzwert ist „**wohin die Reihe geht**". Wenn die Reihe an einen Ort geht, ist dieser Ort der Grenzwert.

Manche Reihen gehen nirgendwohin (springen herum), und dann gibt es keinen Grenzwert.

</Porta>

<Porta nivel="10" titulo="Für ein 10-jähriges Kind">

**Folge** = Liste von Zahlen: $a_1, a_2, a_3, \ldots$

**Grenzwert** $L$: Die Zahl, gegen die die Folge **im Unendlichen läuft**. Beispiele:
- $1, 1/2, 1/3, \ldots$ → geht gegen **null**.
- $1, 4, 9, 16, \ldots$ → geht gegen **$\infty$** (wächst).
- $1, -1, 1, -1, \ldots$ → geht nirgendwohin (alternierend).

**Strenge Definition**: $a_n \to L$, wenn man **so nah wie man will** an $L$ kommen kann, einfach weit genug in der Folge gehen. Für jede „Blase" $\eps$ um $L$ gibt es $N$, sodass **alle** Glieder ab $a_N$ in der Blase liegen.

**Nützlicher Trick**: Wenn $a_n$ ständig **wächst** und eine **Decke** hat, **konvergiert sie**! (Monoton beschränkt.) Du musst den Grenzwert nicht kennen — nur wissen, dass er existiert.

</Porta>

<Porta nivel="15" titulo="Für einen 15-jährigen Jugendlichen">

Folgen sind **Funktionen mit Definitionsbereich $\mathbb{N}$**. Die Grenzwerttheorie ist dieselbe wie bei ε-δ aus Lektion 41, mit $N$ statt $\delta$.

### Warum Folgen wichtig sind

- **Newton-Iteration** für Wurzeln: $a_{n+1} = a_n - f(a_n)/f'(a_n)$.
- **Reihen** = Grenzwert von Teilsummen: $\sum a_n = \lim s_N$.
- **Integral** = Grenzwert Riemann-Summen.
- **Wahrscheinlichkeit**: Gesetz der großen Zahlen $\bar X_n \to E[X]$.

### Bemerkenswerte Folgengrenzwerte

| Folge | Grenzwert |
|---|---|
| $1/n$ | $0$ |
| $(1 + 1/n)^n$ | $e$ |
| $n^{1/n}$ | $1$ |
| $n!/n^n$ | $0$ |
| $a^n$ ($|a| < 1$) | $0$ |
| $a^n$ ($a > 1$) | $\infty$ |
| $(-1)^n$ | existiert nicht |

### Wachstumshierarchie (wichtig!)

$$\log n \ll n^a \ll a^n \ll n! \ll n^n.$$

Jeder wächst **viel langsamer** als der nächste. Standardwerkzeug in der Algorithmenkomplexität.

### Operationale Sätze

**Einschnürung**: $b_n$ zwischen $a_n$ und $c_n$ klemmen, die zum gleichen Ort gehen.

**Beispiel**: $\lim_{n \to \infty} \sin n / n$. $-1/n \leq \sin n / n \leq 1/n$. Per Einschnürung $\to 0$.

**Monoton beschränkt**: wachsende, beschränkte Folge konvergiert — du musst den Grenzwert nicht finden, nur wissen, dass er existiert!

**Beispiel**: $a_1 = 1$, $a_{n+1} = (a_n + 2/a_n)/2$. Man zeigt, $a_n$ ist nach $a_1$ fallend, nach unten durch $\sqrt 2$ beschränkt. Konvergiert. $L$ erfüllt $L = (L + 2/L)/2$, also $L = \sqrt 2$.

### Bolzano-Weierstrass

**Jede beschränkte Folge hat eine konvergente Teilfolge**. Nützlich für Existenzargumente (Kompaktheit).

</Porta>

<Porta nivel="25" titulo="Für einen Ingenieurstudenten">

Folgen sind **die Grundlage iterativer Algorithmen** — Newton, Gradient Descent, EM, MCMC.

### Beweis: $\lim 1/n = 0$ via ε-N

Bei $\eps > 0$ wähle $N = \lceil 1/\eps \rceil + 1$. Dann $n \geq N \Rightarrow 1/n \leq 1/N < \eps$. ∎

### Beweis: monoton beschränkte Folge konvergiert

Sei $(a_n)$ wachsend und nach oben beschränkt. Setze $L = \sup\{a_n\}$ (existiert per Vollständigkeit). Bei $\eps > 0$ ist $L - \eps$ keine obere Schranke, also $\exists N : a_N > L - \eps$. Da wachsend $a_n \geq a_N > L - \eps$ für $n \geq N$. Da $a_n \leq L$, gilt $|a_n - L| < \eps$. ∎

### Bolzano-Weierstrass (Skizze)

Beschränkte Folge $(a_n) \subseteq [a, b]$. Halbieren: eine Hälfte enthält unendlich viele Glieder. Wiederholen. Konstruiere Teilfolge $(a_{n_k})$ mit $a_{n_k}$ in der $k$-ten Hälfte, Intervalllänge $(b-a)/2^k$. Cauchy, also konvergent. ∎

### Cauchy ⇔ konvergent in $\mathbb{R}$

$(\Rightarrow)$ Konvergent ist Cauchy: $|a_m - a_n| \leq |a_m - L| + |L - a_n| < \eps$.

$(\Leftarrow)$ Cauchy ist konvergent: Cauchy $\Rightarrow$ beschränkt $\Rightarrow$ hat konvergente Teilfolge (Bolzano-Weierstrass) $\Rightarrow$ ganze Folge konvergiert zum gleichen Grenzwert (weil Cauchy).

### Cauchy-Folgen in $\mathbb{Q}$

$(a_n)$ approximiert $\sqrt 2$ per Bisektion ist Cauchy in $\mathbb{Q}$, **konvergiert aber nicht** in $\mathbb{Q}$ (Grenzwert irrational). $\mathbb{R}$ ist die **Vervollständigung** von $\mathbb{Q}$ bezüglich $|\cdot|$.

### Anwendungen in Algorithmen

- **Gradient Descent**: $w_{n+1} = w_n - \eta \nabla f(w_n)$. Konvergiert für $\eta < 2/L$ (Lipschitz-Konstante), $f$ konvex.
- **Newton**: $x_{n+1} = x_n - f(x_n)/f'(x_n)$. **Quadratische** Konvergenz nahe einfacher Nullstelle.
- **Heron** (Quadratwurzel): $a_{n+1} = (a_n + N/a_n)/2$. Quadratisch für $\sqrt N$. Folgt aus Newton in $f(x) = x^2 - N$.
- **EM**: erhöht Likelihood monoton. Konvergiert zu kritischem Punkt (nicht zwingend globales Max).

### Reihenkonvergenz

$\sum a_n$ konvergiert $\iff$ Teilsummen $s_N = \sum_{n=1}^N a_n$ bilden konvergente Folge.

**Kriterien**:
- Cauchy: $\forall \eps, \exists N : m > n \geq N \Rightarrow |s_m - s_n| < \eps$.
- Quotient (D'Alembert): $\lim |a_{n+1}/a_n| = L < 1 \Rightarrow$ konvergiert.
- Wurzel (Cauchy): $\lim |a_n|^{1/n} = L < 1 \Rightarrow$ konvergiert.

### Gesetz der großen Zahlen

$X_1, X_2, \ldots$ iid mit $E[X] = \mu$, $V[X] < \infty$. Dann $\bar X_n = (1/n) \sum X_i \to \mu$ (in Wahrscheinlichkeit — schwach; ae — stark).

### Anwendungen in dynamischen Systemen

Diskretes System $x_{n+1} = T(x_n)$. Verhalten: Fixpunkt, Zyklus, seltsamer Attraktor (Chaos). **Logistische Abbildung** $x_{n+1} = r x_n (1 - x_n)$ zeigt Übergang zum Chaos bei $r \approx 3{,}57$.

### Folgen in $\mathbb{R}^n$

Konvergenz komponentenweise. In Norm: $\|x_n - L\| \to 0$. Äquivalent in $\mathbb{R}^n$ (aber nicht in unendlicher Dimension).

</Porta>

<Porta nivel="40" titulo="Für einen erfahrenen Profi">

Folgen sind zentral in der **Funktionalanalysis**, **metrischen Räumen**, **Topologie**.

### Folgenräume

- $\ell^p$ = Folgen mit $\sum |a_n|^p < \infty$. Banach mit Norm $\|a\|_p = (\sum |a_n|^p)^{1/p}$.
- $\ell^\infty$ = beschränkte Folgen. Banach mit $\|a\|_\infty = \sup |a_n|$.
- $c_0$ = Folgen $a_n \to 0$. Abgeschlossener Unterraum von $\ell^\infty$.

### Schwache vs. starke Konvergenz

- **Stark**: $\|x_n - x\| \to 0$.
- **Schwach**: $f(x_n) \to f(x)$ für alle $f \in X^*$ (Dual).

In endlicher Dimension äquivalent. In $\ell^2$: Orthonormalbasis $e_n$ konvergiert schwach gegen 0, aber nicht stark ($\|e_n\| = 1$).

### Cauchy in metrischen Räumen

$(X, d)$ ist **vollständig**, falls jede Cauchy-Folge konvergiert. Banach = vollständig normierter Raum. Hilbert = vollständiger Skalarproduktraum.

**Vervollständigung**: jeder metrische Raum hat eine eindeutige Vervollständigung. $\mathbb{R}$ ist Vervollständigung von $\mathbb{Q}$. $L^2$ ist Vervollständigung von $C[a,b]$ in $L^2$-Norm.

### Kompaktheit

In metrischen Räumen: **folgenkompakt** = jede Folge hat konvergente Teilfolge. In $\mathbb{R}^n$ äquivalent zu abgeschlossen und beschränkt (Heine-Borel). **In unendlicher Dimension** äquivalent zu abgeschlossen, beschränkt **und** total beschränkt (Riesz).

### Topologie

Folgenkriterium der Stetigkeit gilt in **metrischen Räumen** (Heine). Allgemein braucht man Netze oder Filter.

### Kompakte Operatoren

$T : X \to Y$ ist **kompakt**, wenn das Bild der Einheitskugel präkompakt ist. Konvergente Teilfolgen sind Schlüssel in Beweisen.

### Spektraltheorie

Eigenwertfolge $\lambda_n$ eines kompakten selbstadjungierten Operators: entweder endlich oder $\lambda_n \to 0$. Spektralsatz zerlegt über Eigenvektorbasis.

### Fortgeschrittene iterative Methoden

- **Aitken-Beschleunigung**: $\hat a_n = a_n - (a_{n+1} - a_n)^2/(a_{n+2} - 2 a_{n+1} + a_n)$. Beschleunigt lineare Konvergenz.
- **Romberg-Integration**: Richardson-Extrapolation in der Integration.
- **GMRES, Krylov-Unterräume**: für große lineare Systeme.

### Anwendungen in ML / RL

- **Q-Learning-Konvergenz**: $Q_n \to Q^*$ unter geeigneter $\epsilon$-greedy-Erkundung.
- **Policy-Gradient-Konvergenz**: Analyse via stochastischer Approximation (Robbins-Monro).
- **MCMC**: Zustandsfolge $\to$ stationäre Verteilung. Konvergenz über ergodische Sätze.

### Kategorien

Grenzwerte in Kategorien verallgemeinern Folgengrenzwerte. Direkter/inverser Limes, Kolimes usw. Gleiche algebraische Struktur.

</Porta>

<Porta nivel="pratica" titulo="Praktische Anwendung">

**Rezept: Folgengrenzwerte berechnen.**

### Algorithmus

1. **Direkt einsetzen**: Wenn $a_n = f(n)$ mit Funktion $f$, berechne $\lim_{x \to \infty} f(x)$.
2. **Rekursiv**: Grenzwert $L$ annehmen, $L = f(L)$ lösen.
3. **Arithmetik**: in bekannte Teile zerlegen.
4. **Einschnürung**: zwischen Bekannten klemmen.
5. **Monoton beschränkt**: Existenz beweisen, Wert später (oder gar nicht) berechnen.

### SymPy

```python
import sympy as sp
n = sp.symbols('n', positive=True, integer=True)
print(sp.limit(1/n, n, sp.oo))                # 0
print(sp.limit((1 + 1/n)**n, n, sp.oo))       # E
print(sp.limit(n**(1/n), n, sp.oo))           # 1
print(sp.limit(sp.factorial(n)/n**n, n, sp.oo))  # 0
```

### Wachstumshierarchie (auf Abruf)

$$\log n \ll n^a \ll a^n \ll n! \ll n^n.$$

Für Quotienten verwenden: $\lim n!/2^n = \infty$, $\lim 2^n/n! = 0$, $\lim n^{100}/2^n = 0$.

### Rekursive Folgen

Für $a_{n+1} = f(a_n)$:

1. Konvergenz prüfen (Monotonie + Beschränktheit oder Banach-Kontraktion).
2. Falls Konvergenz zu $L$, dann $L = f(L)$.

```python
import numpy as np
def heron(N, a0=1, n_iter=10):
    a = a0
    for _ in range(n_iter):
        a = (a + N/a)/2
    return a
print(heron(2))   # ~1.41421356 (sqrt 2)
```

### Wann der Grenzwert NICHT existiert

- **Oszillation**: $(-1)^n$, $\sin n$.
- **Divergenz**: $a_n \to \pm \infty$.
- **Nicht monoton, nicht Cauchy**: Chaos.

### Numerische Verifikation

Implementieren und $a_n$ vs. $n$ plotten ist die schnellste Methode, den Grenzwert zu **vermuten**. Dann beweisen.

### Vorsichten

- Rekursiver Grenzwert setzt **Existenz** voraus. Ohne Beweis kann es falsch sein.
- $(1 + 1/n)^n \neq 1$ im Unendlichen! Unbestimmte Form $1^\infty$.
- **Schließlich wachsende** Folge muss nicht monoton sein — Vorsicht in Beweisen.
