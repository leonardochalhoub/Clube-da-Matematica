<Porta nivel="formal" titulo="ZWS und Varianten">

<Definicao titulo="Zwischenwertsatz (ZWS)">
Wenn $f \in C([a, b])$ und $k$ zwischen $f(a)$ und $f(b)$ liegt, dann existiert $c \in (a, b)$ mit $f(c) = k$.
</Definicao>

**Spezialfall** ($k = 0$): $f(a) \cdot f(b) < 0 \Rightarrow \exists c \in (a, b) : f(c) = 0$.

### Anwendung 1: Bisektion (Algorithmus)

Bei $f(a) f(b) < 0$:
1. $m = (a+b)/2$.
2. Falls $f(m) = 0$, abbrechen.
3. Falls $f(a) f(m) < 0$, in $[a, m]$ wiederholen.
4. Sonst in $[m, b]$ wiederholen.

**Konvergenz**: Fehler halbiert sich pro Iteration, $|c - m_n| \leq (b-a)/2^n$.

### Anwendung 2: Fixpunkt

Wenn $g \in C([a, b])$ mit $g([a, b]) \subseteq [a, b]$, dann $\exists c$ mit $g(c) = c$. Wende ZWS auf $f(x) = g(x) - x$ an.

### Anwendung 3: Existenz von Gleichungslösungen

$\cos x = x$, $e^x = 2x$, $\tan x = x$ haben Lösungen über ZWS.

### Beweis über Vollständigkeit

Sei $S = \{x \in [a, b] : f(x) < k\}$ (annehmend $f(a) < k < f(b)$). $S \neq \emptyset$ ($a \in S$) und nach oben beschränkt. Setze $c = \sup S$. Aus Stetigkeit und Supremumseigenschaft folgt $f(c) = k$.

### Warum Stetigkeit erforderlich ist

Gegenbeispiel: $f(x) = \mathrm{sgn}(x)$ auf $[-1, 1]$. $f(-1) = -1$, $f(1) = 1$, aber $f$ nimmt nie den Wert $0{,}5$ an. Scheitert, weil $f$ in 0 unstetig ist.

### ZWS vs. MWS

**ZWS**: Existenz eines **Zwischenwerts** ($f$ stetig).
**MWS** (Mittelwertsatz, Lektion 56): Existenz einer **Zwischen-Ableitung** ($f$ differenzierbar).

Nicht verwechseln. MWS braucht Differenzierbarkeit, ZWS nur Stetigkeit.

</Porta>

<Porta nivel="5" titulo="Für ein 5-jähriges Kind">

Stell dir vor, du läufst von einer **niedrigen** Höhe (Boden) zu einer **hohen** Höhe (Hügelspitze). Irgendwann musst du **alle Zwischenhöhen** durchlaufen — du kannst nicht direkt vom Boden zum Gipfel springen, ohne durch die Mitte zu gehen.

Eine stetige Funktion ist genauso: Wenn sie von **minus** zu **plus** geht, ist sie irgendwann **null**.

</Porta>

<Porta nivel="10" titulo="Für ein 10-jähriges Kind">

**Zwischenwertsatz**: Wenn eine stetige Funktion **negativ** beginnt und **positiv** endet, geht sie irgendwann durch **null**.

Stell dir ein **Thermometer** vor, das morgens $-2°$ und nachmittags $+5°$ anzeigt. Irgendwann zeigte es genau $0°$ (Gefrierpunkt). Unausweichlich — solange die Temperatur stetig ist (nicht springt).

**Anwendung**: Herausfinden, wo eine Gleichung eine Lösung hat. Beispiel $f(x) = x^3 - x - 1$:
- $f(1) = 1 - 1 - 1 = -1$ (negativ)
- $f(2) = 8 - 2 - 1 = 5$ (positiv)

Da Vorzeichenwechsel, **gibt es eine Nullstelle zwischen 1 und 2**! ZWS garantiert.

**Bisektionsverfahren**: Nimm den Mittelpunkt $1{,}5$. $f(1{,}5) = 0{,}875 > 0$. Also liegt die Nullstelle zwischen $1$ und $1{,}5$. Nimm die Mitte $1{,}25$. Und so weiter, **immer in der Mitte teilen**. Jedes Mal näher dran.

</Porta>

<Porta nivel="15" titulo="Für einen 15-jährigen Jugendlichen">

ZWS ist der Satz, der **die Existenz von Lösungen** für Gleichungen mit stetiger Funktion garantiert.

### Praktische Form

Um zu zeigen, dass $g(x) = h(x)$ eine Lösung in $[a, b]$ hat:
1. Definiere $f(x) = g(x) - h(x)$.
2. Verifiziere $f$ stetig auf $[a, b]$.
3. Wenn $f(a) f(b) < 0$, gibt es eine Lösung.

### Beispiele

**$\cos x = x$** auf $[0, \pi/2]$:
- $f(x) = \cos x - x$.
- $f(0) = 1 > 0$, $f(\pi/2) = -\pi/2 < 0$.
- Lösung existiert. (Es ist $\approx 0{,}739$.)

**$e^x = 2x$**:
- $f(0) = 1 > 0$, $f(1) = e - 2 \approx 0{,}72 > 0$. Hier kein Vorzeichenwechsel.
- $f(2) = e^2 - 4 \approx 3{,}39 > 0$. Für große Werte gewinnt $e^x$.
- $f(-1) = e^{-1} + 2 \approx 2{,}37 > 0$.

(Mit Graph reanalysieren.)

### Bisektion: die kanonische Anwendung

| Iteration | $a$ | $b$ | $m$ | $f(m)$ | Nächstes |
|---|---|---|---|---|---|
| 0 | 1 | 2 | $1{,}5$ | + | $[1;\ 1{,}5]$ |
| 1 | 1 | $1{,}5$ | $1{,}25$ | $-$ | $[1{,}25;\ 1{,}5]$ |
| 2 | $1{,}25$ | $1{,}5$ | $1{,}375$ | + | $[1{,}25;\ 1{,}375]$ |
| 3 | $1{,}25$ | $1{,}375$ | $1{,}3125$ | $-$ | $[1{,}3125;\ 1{,}375]$ |

Jede Iteration halbiert den Fehler. Nach 10 Iterationen Fehler $< (b-a)/1024$.

### Polynome ungeraden Grades

**Haben immer eine reelle Nullstelle**! Für $P(x) = a_n x^n + \ldots$ mit ungeradem $n$ und $a_n > 0$:
- $\lim_{x \to +\infty} P = +\infty$
- $\lim_{x \to -\infty} P = -\infty$
- Per ZWS existiert eine Nullstelle.

</Porta>

<Porta nivel="25" titulo="Für einen Ingenieurstudenten">

Der ZWS ist der **fundamentale** Satz für numerische Analysis und Existenzmethoden.

### Beweis über Vollständigkeit

Sei $f \in C([a, b])$ mit $f(a) < 0 < f(b)$. Definiere $S = \{x \in [a, b] : f(x) < 0\}$.

- $S$ nicht leer: $a \in S$.
- $S$ beschränkt: $\subseteq [a, b]$.
- Aus Vollständigkeit von $\mathbb{R}$: $\exists c = \sup S$, mit $c \in [a, b]$.

Behauptung: $f(c) = 0$.
- Wenn $f(c) < 0$: Aus Stetigkeit ist $f < 0$ in einer Umgebung von $c$, Widerspruch zu $c = \sup$.
- Wenn $f(c) > 0$: Aus Stetigkeit ist $f > 0$ in einer Umgebung von $c$, aber $c = \sup S$, Widerspruch.

Also $f(c) = 0$. ∎

### Warum ZWS **in $\mathbb{Q}$ versagt**

$f(x) = x^2 - 2$ stetig auf $\mathbb{Q}$, $f(0) = -2 < 0$, $f(2) = 2 > 0$. Nach klassischem ZWS Nullstelle in $(0, 2)$. Aber die Nullstelle ist $\sqrt 2 \notin \mathbb{Q}$. **Ohne Vollständigkeit versagt der ZWS.**

### Anwendungen im Engineering

- **Implizite Volatilität** (Black-Scholes-Lektion): Gleichung $C(\sigma) = C_{\text{mkt}}$. $C$ stetig und monoton in $\sigma$. ZWS + Bisektion liefern $\sigma$ in unter $30$ Iterationen.
- **Gleichgewicht in nichtlinearen Schaltkreisen**: $f(V) = i_{\text{in}}$. Mittels Newton (schnell) oder Bisektion (robust).
- **Stabilitätsanalyse**: Charakteristisches Polynom $p(s)$ — Nullstelle in der rechten Halbebene? ZWS + Routh-Hurwitz.
- **Übertragungsfunktionspolynome**: Nullstellen via Routh, Pole via Jury (digitale Regelung).

### Bisektion: Fehleranalyse

Nach $n$ Iterationen $|c - m_n| \leq (b-a)/2^{n+1}$. Für Fehler $< \eps$ braucht man $n > \log_2((b-a)/\eps) - 1$.

**Konvergenzgarantie**: Bisektion konvergiert immer, falls ZWS gilt. Andere Methoden (Newton) können divergieren.

### Brouwer-Fixpunkt (1D via ZWS)

Wenn $g \in C([a, b])$ mit $g([a, b]) \subseteq [a, b]$, dann hat $g$ einen Fixpunkt. Beweis: Definiere $f(x) = g(x) - x$. $f(a) = g(a) - a \geq 0$ ($g(a) \in [a, b]$). $f(b) = g(b) - b \leq 0$. Nach ZWS $\exists c : f(c) = 0$, also $g(c) = c$. ∎

**Brouwer allgemein** (Dimension $n$): jede stetige $g : B^n \to B^n$ hat einen Fixpunkt. Nicht trivial — Grundlage der Gleichgewichtstheorie in der Ökonomie (Nash), in der adaptiven Regelung.

### Implizite Gleichungen

$F(x, y) = 0$ mit stetigem $F$ und $F(x_0, y_0) = 0$. Für jedes $x$ in der Nähe suche $y$ via ZWS in $F(x, \cdot)$.

</Porta>

<Porta nivel="40" titulo="Für einen erfahrenen Profi">

Der ZWS ist eine **topologische Eigenschaft**: das Bild eines Zusammenhängenden unter einer stetigen Abbildung ist zusammenhängend.

### Topologische Verallgemeinerung

$f : X \to Y$ stetig, $X$ zusammenhängend $\Rightarrow$ $f(X)$ zusammenhängend. In $\mathbb{R}$ sind Zusammenhängende Intervalle. Also ist $f([a, b])$ ein Intervall, enthält $f(a), f(b)$ und damit alle Zwischenwerte.

### ZWS und totale Ordnung

Der Beweis über $\sup$ hängt davon ab, dass $\mathbb{R}$ vollständig **und** linear geordnet ist. In $\mathbb{R}^n$ ($n \geq 2$) gibt es keine totale Ordnung — der wörtliche ZWS gilt nicht, aber es gibt Verallgemeinerungen über algebraische Topologie.

### Verallgemeinerung: Brouwer-Fixpunkt

Stetige $g : B^n \to B^n$ hat einen Fixpunkt. Topologischer Beweis:
- Annahme: kein Fixpunkt. Definiere $r(x) = $ Strahl von $x$ durch $g(x)$ projiziert auf den Rand $\partial B^n = S^{n-1}$.
- $r$ ist stetige Retraktion von $B^n$ auf $S^{n-1}$, Widerspruch zu $\pi_{n-1}(B^n) = 0 \neq \pi_{n-1}(S^{n-1})$.

### Anwendungen in der Ökonomie (Nash-Gleichgewicht)

In endlichen Spielen existiert das Nash-Gleichgewicht via Brouwer (oder Kakutani). Die Best-Response-Funktion ist stetig (mehrwertig) und hat einen Fixpunkt. **Wirtschaftsnobelpreis** für [Nash 1994](https://www.nobelprize.org/prizes/economic-sciences/1994/summary/).

### Anwendungen in PDEs

- **Schaefer-Fixpunkttheorem**: Erweiterung für kompakte Operatoren in Banach.
- **Schauder-Fixpunkt**: für konvexe kompakte Mengen in Banach.
- **Lösung von Navier-Stokes**: Existenz über Fixpunkt im geeigneten Raum.

### ZWS in den komplexen Zahlen

$f$ holomorph hat keinen wörtlichen ZWS (keine Ordnung in $\mathbb{C}$), aber **offene Abbildung**: nicht-konstantes $f$ bildet Offene auf Offene ab. Kombiniert mit dem **Argumentprinzip**, zählt Nullstellen über Konturintegration.

### Algorithmik

- **Bisektion**: $O(\log_2 1/\eps)$ — robust, langsam.
- **Newton**: $O(\log \log 1/\eps)$ (quadratisch) — schnell, kann divergieren.
- **Brent**: kluge Kombination, das Beste aus beiden.
- **TOMS-Algorithmus 748** (Stand der Technik): Bisektion + Interpolation.

### Numerisch in Python

```python
from scipy.optimize import brentq, bisect
import numpy as np
f = lambda x: x**3 - x - 1
print(brentq(f, 1, 2))   # ~1.3247
print(bisect(f, 1, 2))   # ~1.3247
```

<Porta nivel="pratica" titulo="Praktische Anwendung">

**Rezept: ZWS anwenden.**

### Existenz einer Nullstelle zeigen

1. **Identifiziere $f$** (evtl. als Differenz $g - h$).
2. **Stetigkeit auf $[a, b]$** prüfen.
3. **$f(a)$ und $f(b)$** berechnen. Vorzeichenwechsel bestätigen.
4. **Schließen**: Es existiert eine Nullstelle in $(a, b)$.

### Bisektion: Implementierung

```python
def bissect(f, a, b, tol=1e-9):
    assert f(a) * f(b) < 0, "ZWS nicht anwendbar"
    while b - a > tol:
        m = (a + b) / 2
        if f(m) == 0:
            return m
        if f(a) * f(m) < 0:
            b = m
        else:
            a = m
    return (a + b) / 2

print(bissect(lambda x: x**3 - x - 1, 1, 2))   # ~1.3247
```

### Wann Bisektion die richtige Wahl ist

- **Stetige, aber nicht differenzierbare Funktion**: Newton geht nicht.
- **Robustheit** wichtiger als Geschwindigkeit.
- **Multimodal**: Bisektion findet **eine** Nullstelle im Intervall.

### Wann der ZWS keine Eindeutigkeit liefert

Der ZWS garantiert **Existenz**, nicht **Eindeutigkeit**. $f(x) = x^3 - x$ auf $[-2, 2]$ hat drei Nullstellen. Für Eindeutigkeit Monotonie nötig.

### Wann der ZWS NICHT hilft

- Unstetige Funktion: ZWS gilt nicht.
- $f(a) f(b) > 0$ (gleiches Vorzeichen): ZWS detektiert nicht — **kann aber trotzdem Nullstelle haben** (nicht erkannt).
- Doppelte Nullstelle: $f(a) = f(b) = 0$ zählt nicht (kein strikter Vorzeichenwechsel).

### Iterativer Fixpunkt

```python
def fixpunkt(g, x0, tol=1e-9, max_iter=100):
    x = x0
    for _ in range(max_iter):
        x_new = g(x)
        if abs(x_new - x) < tol:
            return x_new
        x = x_new
    return x

print(fixpunkt(np.cos, 0.5))   # ~0.739 (cos x = x)
```

**Vorsicht**: Konvergenz erfordert $|g'(c)| < 1$ am Fixpunkt (Banach-Picard).
