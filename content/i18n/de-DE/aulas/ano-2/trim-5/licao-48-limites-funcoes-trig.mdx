### Fundamentale trigonometrische Grenzwerte

| Grenzwert | Wert |
|---|---|
| $\lim_{x \to 0} \sin x / x$ | $1$ |
| $\lim_{x \to 0} \sin(kx)/x$ | $k$ |
| $\lim_{x \to 0} \tan x / x$ | $1$ |
| $\lim_{x \to 0} (1 - \cos x)/x$ | $0$ |
| $\lim_{x \to 0} (1 - \cos x)/x^2$ | $1/2$ |
| $\lim_{x \to 0} \sin^2 x / x^2$ | $1$ |
| $\lim_{x \to 0} \arcsin x / x$ | $1$ |
| $\lim_{x \to 0} \arctan x / x$ | $1$ |

### Nützliche Identitäten

- $1 - \cos x = 2\sin^2(x/2)$
- $\sin(a + b) = \sin a \cos b + \cos a \sin b$
- $\cos(a + b) = \cos a \cos b - \sin a \sin b$
- $1 - \cos x = (1 - \cos^2 x)/(1 + \cos x) = \sin^2 x / (1 + \cos x)$

### Techniken

1. **Variablensubstitution $u$**: $\sin(2x)/x = 2 \cdot \sin(2x)/(2x) \to 2$.
2. **Identitäten** zum Umschreiben in Grundform.
3. **Konjugierte**: $1 - \cos x = \sin^2 x / (1 + \cos x)$.
4. **Einschnürung**: $-1 \leq \sin x \leq 1$, multipliziere mit etwas $\to 0$.

### Beweis von $\lim (1 - \cos x)/x^2 = 1/2$

$\frac{1 - \cos x}{x^2} = \frac{2 \sin^2(x/2)}{x^2} = \frac{1}{2} \cdot \frac{\sin^2(x/2)}{(x/2)^2} \to \frac{1}{2} \cdot 1 = \frac{1}{2}$.

### Beweis von $\lim \tan x / x = 1$

$\tan x / x = (\sin x / x) \cdot (1 / \cos x) \to 1 \cdot 1 = 1$.

<Porta nivel="5" titulo="Für ein 5-jähriges Kind">

Wenn der Winkel **sehr klein** ist, ist sein **Sinus** fast gleich dem Winkel selbst. Wie z. B. $\sin 0{,}01 \approx 0{,}01$.

Daher $\sin x / x \to 1$ — sie sind fast gleich.

Und **eins minus Kosinus** ist fast gleich der **Hälfte des Winkelquadrats**. Kleine Dinge folgen einfachen Regeln.

</Porta>

<Porta nivel="10" titulo="Für ein 10-jähriges Kind">

Für **kleine Winkel** (im Bogenmaß):
- $\sin x \approx x$
- $\cos x \approx 1 - x^2/2$
- $\tan x \approx x$

Daraus folgen alle Grenzwerte:
- $\sin(3x)/x \to 3$ (weil $\sin(3x) \approx 3x$)
- $(1 - \cos x)/x^2 \to 1/2$ (weil $1 - \cos x \approx x^2/2$)
- $\tan x / x \to 1$ (weil $\tan x \approx x$)

**Anwendung in der Physik**: Pendel mit kleinen Schwingungen. Die wahre Gleichung ist $\ddot \theta = -(g/L) \sin \theta$. Für kleines $\theta$ ist $\sin \theta \approx \theta$, und es wird $\ddot \theta = -(g/L) \theta$ — einfacher Oszillator.

**Ohne diese Approximation** wird das Problem viel schwieriger (elliptische Integrale). Daher lieben Ingenieure kleine Winkel.

</Porta>

<Porta nivel="15" titulo="Für einen 15-jährigen Jugendlichen">

Trigonometrische Grenzwerte mit $x \to 0$ reduzieren sich durch Manipulation auf drei Fälle:

### Fall 1: $\sin(?)/?$

Wenn du in die Form $\sin(\text{etwas})/\text{etwas}$ schreiben kannst, geht es gegen 1. Variablen­wechsel $u = $ Argument.

**Beispiel**: $\lim_{x \to 0} \sin(7x)/(3x) = (7/3) \lim_{x \to 0} \sin(7x)/(7x) = 7/3$.

### Fall 2: $(1 - \cos)/?$

Verwende Identität $1 - \cos x = 2 \sin^2(x/2)$ oder Konjugiertes.

**Beispiel**: $\lim_{x \to 0} (1 - \cos x)/(x \sin x) = (x^2/2)/(x \cdot x) = 1/2$ (mit $\sin x \approx x$).

### Fall 3: Kombinationen $\tan, \sec, \cot$

In $\sin, \cos$ umschreiben.

**Beispiel**: $\lim_{x \to 0} \tan(2x)/\sin(3x) = \lim (2x)/(3x) = 2/3$ (Approximation).

### Tricks beim Variablenwechsel

Für $\lim_{x \to a}$ trig: $u = x - a$, um auf null zu reduzieren.

**Beispiel**: $\lim_{x \to \pi} \sin x / (x - \pi)$. Sei $u = x - \pi$. Dann $\sin x = \sin(u + \pi) = -\sin u$. Ergebnis: $\lim_{u \to 0} -\sin u / u = -1$.

### Tabelle der Approximationen für kleines $x$

| Funktion | Approximation (bis $x^3$) |
|---|---|
| $\sin x$ | $x - x^3/6$ |
| $\cos x$ | $1 - x^2/2$ |
| $\tan x$ | $x + x^3/3$ |
| $\arcsin x$ | $x + x^3/6$ |
| $\arctan x$ | $x - x^3/3$ |

Für schnelleres Lösen.

### Anspruchsvolles Beispiel

$\lim_{x \to 0} (\tan x - \sin x)/x^3$:

$\tan x - \sin x = \sin x (1/\cos x - 1) = \sin x \cdot (1 - \cos x)/\cos x$.

Also: $\lim = (\sin x / x) \cdot ((1 - \cos x)/x^2) \cdot (1/\cos x) = 1 \cdot (1/2) \cdot 1 = 1/2$.

</Porta>

<Porta nivel="25" titulo="Für einen Ingenieurstudenten">

Trigonometrische Grenzwerte sind essentiell in **Ableitungen** ($\sin', \cos', \tan'$), **Fourier-Reihen**, **harmonischer Analyse**.

### Schlüsselbeweise

**$\lim \sin x / x = 1$**: geometrisches Argument — Dreiecksfläche, Sektor, Dreieck. Per Einschnürung folgt es.

**$(1 - \cos x)/x^2 \to 1/2$**:
$$\frac{1 - \cos x}{x^2} = \frac{(1-\cos x)(1+\cos x)}{x^2 (1+\cos x)} = \frac{\sin^2 x}{x^2(1+\cos x)} = \frac{\sin^2 x}{x^2} \cdot \frac{1}{1+\cos x} \to 1 \cdot \frac{1}{2} = \frac{1}{2}.$$

**$\arcsin x / x \to 1$**: $u = \arcsin x$, $x = \sin u$, $u \to 0$ wenn $x \to 0$. $\arcsin x / x = u/\sin u \to 1$.

### Verbindung zu Ableitungen

- $(\sin x)' = \cos x$ folgt aus $\lim_{h \to 0} (\sin(x+h) - \sin x)/h$. Manipulation über $\sin(x+h) = \sin x \cos h + \cos x \sin h$, dann Quotient $= \sin x (\cos h - 1)/h + \cos x \sin h / h \to 0 + \cos x = \cos x$.
- $(\cos x)' = -\sin x$ analog.
- $(\tan x)' = \sec^2 x$ via Quotient.

### Taylor-Reihen (Quartal 7)

$$\sin x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n+1}}{(2n+1)!}, \quad \cos x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n}}{(2n)!}.$$

Trunziert ergeben Approximationen für kleine Winkel. Gleichmäßige Konvergenz auf Kompakta.

### Anwendungen im Engineering

- **Kleine Winkel in der Mechanik**: Pendel $T = 2\pi\sqrt{L/g}$ gilt für $\theta \lesssim 10°$. Relativer Fehler $\sim \theta^2/16$.
- **Gauß-Optik** (paraxial): $\sin \theta \approx \theta$. Dünne Linsen, Brechung.
- **Linearisierung**: Dynamische Systeme $\dot x = f(x)$ um Gleichgewichtspunkt via Jacobi linearisieren.
- **Modulation**: $\cos(\omega t + \phi(t))$ mit kleinem $\phi$ in AM-Äquivalent entwickeln.

### Fourier-Analyse

Fourier-Koeffizienten $a_n = \frac{1}{T}\int f(t) \cos(2\pi nt/T) dt$. Grenzwert $\sum a_n^2$ konvergiert (Parseval). Trig-Grenzwerte erscheinen in Konvergenzbeweisen punktweise (Dini, Dirichlet).

### Fortgeschrittene Einschnürung

$\lim_{x \to \infty} \sin x$ existiert nicht, aber $\lim_{x \to \infty} (\sin x)/x = 0$ per Einschnürung $-1/x \leq \sin x/x \leq 1/x$.

$\lim_{x \to 0} x \sin(1/x) = 0$ per Einschnürung. **Nicht** dasselbe wie $\lim x \cdot \lim \sin(1/x)$ — der zweite existiert nicht. Einschnürung löst, ohne diese Faktorisierung.

### In AC-Schaltkreisen

- Impedanz $Z(j\omega) = R + jX$. Betrag $|Z| = \sqrt{R^2 + X^2}$. Grenzwerte bei niedriger/hoher Frequenz.
- $\sin(\omega t)$ in $\omega = 0$: gegen null. In $\omega \to \infty$: oszilliert, kein Grenzwert.

</Porta>

<Porta nivel="40" titulo="Für einen erfahrenen Profi">

Trigonometrische Grenzwerte sind Manifestationen tiefer **analytischer Struktur**: $\sin, \cos$ sind ganze Funktionen mit spezifischem Nullstellenmuster.

### Weierstrass-Faktorisierung

$$\sin(\pi x) = \pi x \prod_{n=1}^\infty \left(1 - \frac{x^2}{n^2}\right).$$

Logarithmieren und Ableiten ergibt Mittag-Lefflers Identität:
$$\pi \cot(\pi x) = \frac{1}{x} + \sum_{n=1}^\infty \frac{2x}{x^2 - n^2}.$$

Verbindung mit $\zeta(2k)$.

### Funktionalcharakterisierung

$\sin, \cos$ sind die einzigen Funktionen $\mathbb{R} \to \mathbb{R}$, die folgendes erfüllen:
- $f(x+y) = f(x)g(y) + g(x)f(y)$, $g(x+y) = g(x)g(y) - f(x)f(y)$
- $f'(0) = 1$, $g(0) = 1$
- $f, g$ stetig

Äquivalent: Lösung von $y'' + y = 0$ mit Anfangsbedingungen.

### Komplexe Analysis

$e^{ix} = \cos x + i \sin x$ (Euler). Trig-Grenzwerte werden zu Grenzwerten von $e^{iz}$ in $z \to 0$. Holomorph in ganz $\mathbb{C}$. Hadamard-Faktorisierung.

### Anwendungen in PDEs

- **Wellengleichung** $u_{tt} = c^2 u_{xx}$: Lösung $u(x, t) = f(x - ct) + g(x + ct)$ enthält $\sin, \cos$ über Variablentrennung.
- **Freie Schrödinger-Gleichung**: $\psi(x, t) = \int A(k) e^{i(kx - \omega t)} dk$, $\omega = k^2/2m$. Wellenpakete via $\sin, \cos$.
- **Wärmegleichung mit Dirichlet-Randbedingung**: Lösung $\sum b_n \sin(n\pi x/L) e^{-n^2 \pi^2 D t/L^2}$.

### Harmonische Analyse

Carleson-Theorem: Fourier-Reihe konvergiert ae für $f \in L^2$. Beweis basiert auf feinen Schätzungen mit trig-Grenzwerten.

### Padé-Approximation

Für $\sin x$ kann der Padé-Approximant $[m/n]$ besser sein als Taylor. Insbesondere $\tan x \approx x(1 - x^2/15)/(1 - 2x^2/5)$ hat geringeren Fehler als gleichgradiges Taylor.

### Operatorkalkül

- **Matrixfunktion**: $\sin A = (e^{iA} - e^{-iA})/(2i)$. Wohldefiniert für jede Matrix.
- **Spektrale Abbildung**: spec($\sin A$) = $\{\sin \lambda : \lambda \in $ spec $A\}$.
- **Anwendung**: Lösung von $\ddot x = -\Omega^2 x$ ist $x(t) = \cos(\Omega t) x_0 + \Omega^{-1} \sin(\Omega t) v_0$.

### Numerisch

- **Argumentreduktion**: Für $\sin(10^{15})$ mod $\pi$ reduzieren mit **Payne-Hanek** (hochpräzise).
- **CORDIC**: hardwarefreundlich, berechnet $\sin, \cos$ über Rotationsiteration.
- **Numerische Auslöschung**: $1 - \cos x$ in float32 bei kleinem $x$ hat großen relativen Fehler. Stattdessen `2 * sin(x/2)**2`.

</Porta>

<Porta nivel="pratica" titulo="Praktische Anwendung">

**Rezept: trigonometrische Grenzwerte.**

### Algorithmus

1. **Form erkennen**: $\sin/?, \cos$, $(1 - \cos)/?, \tan/?$, etc.
2. **Umschreiben** in Grundform via:
   - Variablenwechsel.
   - Identitäten $\sin^2 + \cos^2 = 1$, $1 - \cos x = 2\sin^2(x/2)$ usw.
3. **Numerisch oder mit SymPy verifizieren**.

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
# Linear im Argument
print(sp.limit(sp.sin(7*x)/x, x, 0))            # 7
# Sinusquotient
print(sp.limit(sp.sin(3*x)/sp.sin(5*x), x, 0))  # 3/5
# (1 - cos)/x^2
print(sp.limit((1 - sp.cos(x))/x**2, x, 0))     # 1/2
# Schwierig
print(sp.limit((sp.tan(x) - sp.sin(x))/x**3, x, 0))  # 1/2
```

### Cheat Sheet

Für $x \to 0$:

| Form | Grenzwert |
|---|---|
| $\sin(ax)/x$ | $a$ |
| $\sin(ax)/\sin(bx)$ | $a/b$ |
| $\sin(ax)/\tan(bx)$ | $a/b$ |
| $(1 - \cos(ax))/x^2$ | $a^2/2$ |
| $\arcsin(ax)/x$ | $a$ |
| $\arctan(ax)/x$ | $a$ |
| $(\sin x)^2/x^2$ | 1 |
| $\sin(\sin x)/x$ | 1 |

### Wann diese NICHT verwenden

- Grenzwert bei $x \to a \neq 0$: Variablenwechsel $u = x - a$.
- Funktion mit $1/x$ im Argument ($\sin(1/x)$): nutze die Oszillation — Einschnürung.

### Häufige Vorsichten

- $\sin(2x)/x \neq 2 \sin x / x$: muss $2 \cdot \sin(2x)/(2x) \to 2$ sein.
- $\sin(x^2)/x \neq \sin x \cdot x$: muss $\sin(x^2)/x = x \cdot \sin(x^2)/x^2 \to 0$ sein.
- $(1 - \cos x)/x \neq (1 - \cos x)/x^2$: Erstes geht gegen 0, zweites gegen 1/2.
