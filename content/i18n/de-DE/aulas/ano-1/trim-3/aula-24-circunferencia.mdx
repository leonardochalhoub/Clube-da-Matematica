<Definicao titulo="Kreis">
Menge der Punkte $(x, y) \in \mathbb{R}^2$, die einen Abstand $r > 0$ von einem Mittelpunkt $C = (a, b)$ haben.
</Definicao>

### Normalform

$$(x - a)^2 + (y - b)^2 = r^2$$

### Allgemeine Form

Ausmultipliziert: $x^2 + y^2 + Dx + Ey + F = 0$ mit $D = -2a$, $E = -2b$, $F = a^2 + b^2 - r^2$.

Mittelpunkt zurückgewonnen: $(a, b) = (-D/2, -E/2)$. Radius: $r = \sqrt{D^2/4 + E^2/4 - F}$ (gültig, falls positiv; sonst ist es kein reeller Kreis).

### Gerade und Kreis

| Lage | Kriterium | Gemeinsame Punkte |
|---|---|---|
| Außerhalb | $d(C, r) > r$ | 0 |
| Tangente | $d(C, r) = r$ | 1 |
| Sekante | $d(C, r) < r$ | 2 |

### Zwei Kreise

| Lage | Kriterium |
|---|---|
| Äußerlich getrennt | $d(C_1, C_2) > r_1 + r_2$ |
| Äußerlich tangierend | $d = r_1 + r_2$ |
| Schneidend | $\lvert r_1 - r_2 \rvert \lt d \lt r_1 + r_2$ |
| Innerlich tangierend | $d = \lvert r_1 - r_2 \rvert$ |
| Einer im anderen | $d \lt \lvert r_1 - r_2 \rvert$ |

### Tangente-Radius-Eigenschaft

Die Tangente in $P$ steht senkrecht auf dem Radius $\overline{CP}$.

### Tangentengleichung

Tangente in $(x_0, y_0)$ an den Kreis $(x-a)^2 + (y-b)^2 = r^2$:
$$(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$$

Nimm einen **Faden** und befestige ein Ende auf dem Boden. Halte am anderen Ende eine Kreide und gehe **immer mit gespanntem Faden** im Kreis. Du zeichnest ein **perfektes Rad** — einen **Kreis**!

Jeder Punkt des Rades hat **denselben Abstand** zur Mitte. Dieser Abstand ist der **Radius**.

<Porta nivel="10" titulo="Für ein 10-jähriges Kind">

Ein Kreis ist die **Menge der Punkte mit gleichem Abstand** zum Mittelpunkt. Wie die Zeichnung, die ein Zirkel macht: Der Abstand von der Spitze zum Bleistift ist der **Radius**.

Die Gleichung $(x - a)^2 + (y - b)^2 = r^2$ folgt aus der Abstandsformel (Lektion 21):
- $(a, b)$ ist der Mittelpunkt.
- $r$ ist der Radius.
- Für jeden Punkt $(x, y)$ der Kurve ist der Abstand zum Mittelpunkt $r$. Beim Quadrieren erhält man die Gleichung oben.

**Konkretes Beispiel**: Ein Fahrradreifen mit Mittelpunkt auf der Achse. Jeder Gummipunkt ist gleich weit von der Achse entfernt.

**Reale Anwendungen**:
- WLAN: Reichweite ist ein Kreis.
- Polizeisirene: Personen, die sie hören, sind innerhalb des Kreises.
- Leichtathletikbahn: 400 m Umfang.

</Porta>

<Porta nivel="15" titulo="Für einen 15-jährigen Jugendlichen">

Der Kreis ist die Menge der Punkte mit **gleichem Abstand** zu einem Mittelpunkt. Abstand $= r$ (Radius).

Anwendung der Abstandsformel aus Lektion 21:
$$\sqrt{(x - a)^2 + (y - b)^2} = r$$

Nach Quadrieren: $(x - a)^2 + (y - b)^2 = r^2$. **Mehr ist es nicht**.

Wenn der Mittelpunkt im Ursprung liegt ($a = b = 0$): $x^2 + y^2 = r^2$. **Einheitskreis** ($r = 1$): $x^2 + y^2 = 1$ — den hast du in Lektion 12 zum Einheitskreis bereits verwendet!

**Allgemeine Form $\to$ Normalform**: Quadratische Ergänzung. Beispiel: $x^2 + y^2 - 4x + 6y - 12 = 0$.
- In $x$: $x^2 - 4x = (x-2)^2 - 4$.
- In $y$: $y^2 + 6y = (y+3)^2 - 9$.
- Einsetzen: $(x-2)^2 - 4 + (y+3)^2 - 9 - 12 = 0 \Rightarrow (x-2)^2 + (y+3)^2 = 25$.
- Mittelpunkt $(2, -3)$, Radius 5.

**Vorsicht**: Nicht jede Gleichung $x^2 + y^2 + Dx + Ey + F = 0$ ist ein Kreis. Wenn $D^2/4 + E^2/4 - F < 0$, gibt es keinen reellen Punkt. Wenn $= 0$, ist es ein einzelner Punkt (entarteter „Kreis").

</Porta>

<Porta nivel="25" titulo="Für einen Ingenieurstudenten">

Der Kreis ist das einfachste Beispiel eines **geometrischen Ortes** (Locus). Direkte Verallgemeinerung: **Kugel** in $\mathbb{R}^3$ — $(x-a)^2 + (y-b)^2 + (z-c)^2 = r^2$. In $\mathbb{R}^n$: $\|x - c\|^2 = r^2$ ist die $(n-1)$-Sphäre.

### Anwendungen über Domänen hinweg

- **Telekommunikation**: Antennenreichweite = Kreis. Mehrfachabdeckung = Vereinigung/Schnitt von Kreisen. Voronoi-Diagramme (jeder Punkt der nächsten Antenne zugeordnet).
- **GPS / Trilateration**: Jeder Satellit definiert eine Kugel; Ihre Position ist der Schnitt von 4 Kugeln. 3 liefern Position + Zeit, der 4. eliminiert die Mehrdeutigkeit.
- **Robotik**: Reichweite eines Arms mit Kugelgelenk ist eine Kugel.
- **Computergrafik**: Bounding Sphere für Kollision und Culling.
- **Verfahrenstechnik**: Gibbs-Gleichgewicht in einem binären Reaktionssystem bildet Kreise im Phasendiagramm.

### Beweis: Tangente steht senkrecht auf dem Radius

Sei $P_0 = (x_0, y_0)$ Punkt auf dem Kreis, Mittelpunkt $C = (a, b)$. Vektor $\vec{CP_0} = (x_0 - a, y_0 - b)$. Die Tangente hat als Normalenvektor genau diesen $\vec{CP_0}$. Somit Gleichung:
$$(x_0 - a)(x - x_0) + (y_0 - b)(y - y_0) = 0$$

Ausmultiplizieren und nutzen, dass $(x_0 - a)^2 + (y_0 - b)^2 = r^2$:
$$(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$$

### Potenz eines Punktes

Für Punkt $P_0$ und Kreis mit Mittelpunkt $C$, Radius $r$:
$$\text{pot}(P_0) = |P_0 C|^2 - r^2$$

- Negativ: $P_0$ innerhalb.
- Null: $P_0$ auf dem Rand.
- Positiv: $P_0$ außerhalb; $\sqrt{\text{pot}}$ ist die Länge der Tangente von $P_0$.

### 2D-Trilateration

3 Kreise durch denselben Punkt: 2 lineare Gleichungen (Subtraktion von Kreisgleichungspaaren eliminiert quadratische Terme). Lineares 2×2-System löst.

</Porta>

<Porta nivel="40" titulo="Für einen erfahrenen Profi">

Der Kreis ist die **fundamentale Kurve** der Differentialgeometrie — konstante Krümmung $\kappa = 1/r$.

### Differentialgeometrische Sicht

Parametrisierung: $\gamma(t) = (a + r\cos t, b + r\sin t)$. Tangentenvektor $\gamma'(t) = (-r\sin t, r\cos t)$ hat Betrag $r$ — nicht Einheits-, aber konstant. Krümmungsvektor $\gamma''(t)/|\gamma'|^2$ zeigt zum Mittelpunkt mit Betrag $1/r$.

### Verallgemeinerung: Schmiegekreis

Für eine beliebige Kurve $\gamma(t)$ in der Ebene gibt es an jedem Punkt einen Schmiegekreis (Best-Fit Circle): tangential zur Kurve mit gleicher Krümmung. Radius $\rho = 1/\kappa$. Entscheidend im Straßenbau (Übergang zwischen Geraden und Kurven über Klothoiden).

### Anwendungen in ML / Quant

- **$\ell_2$-Regularisierung (Ridge)**: $\|\beta\|_2 \leq c$ ist eine euklidische Kugel — Querschnitt der Kreis im Koeffizientenraum. Die Lösung liegt dort, wo die MSE-Niveaukurve die Kreisgrenze berührt.
- **Quant**: Optionspreismodelle wie Heston oder SABR nutzen Kugeln im Parameterraum für Bereiche stabiler Kalibrierung.
- **Trust-Region-Optimierung**: Jeder Schritt ist auf eine euklidische Kugel beschränkt.
- **Manifold Learning**: t-SNE und UMAP bewahren lokale Nachbarschaften (= kleine Kreise im Originalraum).

### Verallgemeinerung auf n-dimensionale Sphären

Volumen der n-Sphäre: $V_n(r) = \frac{\pi^{n/2}}{\Gamma(n/2 + 1)} r^n$. Maßkonzentration: Für großes $n$ liegt fast das gesamte Volumen in einer dünnen „Schale" nahe der Oberfläche — ein Schlüsselphänomen im hochdimensionalen ML.

### Historische Anekdote

Descartes (1643) beschreibt den „Kreis der vier Kreise" (Descartes-Kreissatz): Für 4 sich gegenseitig tangierende Kreise mit Krümmungen $k_1, k_2, k_3, k_4$ gilt $(k_1 + k_2 + k_3 + k_4)^2 = 2(k_1^2 + k_2^2 + k_3^2 + k_4^2)$. Apollonische Kreispackungen — Fraktal.

### SymPy

```python
import sympy as sp
from sympy import Circle, Point, Line
C = Circle(Point(2, 3), 4)
print(C.equation())                    # (x-2)**2 + (y-3)**2 - 16
L = Line(Point(0, 0), Point(1, 1))
print(C.intersection(L))               # Schnittpunkte
print(C.tangent_lines(Point(10, 3)))   # 2 Tangenten
```

<Porta nivel="pratica" titulo="Praktische Anwendung">

**Rezept: Informationen aus einer Kreisgleichung extrahieren.**

1. **Normalform** $(x-a)^2 + (y-b)^2 = r^2$: Mittelpunkt $(a, b)$ und Radius $r$ direkt ablesen.
2. **Allgemeine Form** $x^2 + y^2 + Dx + Ey + F = 0$: Quadratische Ergänzung (oder $a = -D/2$, $b = -E/2$, $r^2 = a^2 + b^2 - F$).
3. **Verifizierung**: Setzen Sie einen Punkt ein und testen Sie — keine absurden Werte.

### Workflow eines typischen Problems

1. **Lagebeziehung Gerade-Kreis**: Berechnen Sie $d(\text{Mittelpunkt}, \text{Gerade}) = |Aa + Bb + C|/\sqrt{A^2+B^2}$, vergleichen Sie mit $r$.
2. **Tangente in einem Punkt des Kreises**: Verwenden Sie $(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$.
3. **Externe Tangente von Punkt $P$**: Länge $= \sqrt{|PC|^2 - r^2}$.
4. **Kreis durch 3 Punkte**: Lösen Sie das 3×3-System mit $x^2 + y^2 + Dx + Ey + F = 0$.

### Wolfram / SymPy

```python
import sympy as sp
from sympy import Circle, Point
C = Circle(Point(0, 0), 5)
print(C.area)         # 25π
print(C.circumference) # 10π
P = Point(3, 4)
print(P in C)         # True (auf dem Rand)
```

### Wann sollten Sie aufpassen?

- **Allgemeine Gleichung ist nicht immer Kreis**: $D^2/4 + E^2/4 - F < 0$ → leere Menge. $= 0$ → einzelner Punkt.
- **Numerisch**: Quadratische Ergänzung mit fast null Koeffizienten ist instabil; berechnen Sie Mittelpunkt/Radius direkt über die Formeln.
- **Kugel vs. Kreis**: In 3D Vorsicht — ein Kreis im 3D ist der Schnitt von Ebene und Kugel (1D-Kurve); braucht zwei Gleichungen.
