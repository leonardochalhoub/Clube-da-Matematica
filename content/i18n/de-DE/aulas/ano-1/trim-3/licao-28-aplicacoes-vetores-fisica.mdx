<Porta nivel="formal" titulo="Vektoren in der Mechanik">

### Grundprinzipien

1. **Statisches Gleichgewicht**: $\sum \vec F_i = \vec 0$ (Objekt in Ruhe oder in gleichförmig-geradliniger Bewegung).
2. **2. Newtonsches Gesetz**: $\sum \vec F_i = m \vec a$.
3. **Zerlegung**: Auf einer Rampe mit Neigung $\theta$ wird die Schwerkraft $\vec g = -g\hat\jmath$ zerlegt in $-g\sin\theta$ parallel (Hangabtrieb) und $-g\cos\theta$ normal.

### Arbeit und Energie

- Arbeit: $W = \vec F \cdot \vec d$.
- Leistung: $P = \vec F \cdot \vec v$.
- Kinetische Energie: $K = \frac{1}{2} m |\vec v|^2$.
- Arbeit-Energie-Theorem: $W_{\text{gesamt}} = \Delta K$.

### Vektorielle Kinematik

- Position: $\vec r(t)$.
- Geschwindigkeit: $\vec v(t) = d\vec r/dt$.
- Beschleunigung: $\vec a(t) = d\vec v/dt$.

Für gleichmäßig beschleunigte Bewegung:
$$\vec r(t) = \vec r_0 + \vec v_0 t + \frac{1}{2} \vec a t^2$$

### Klassische Kräfte

| Kraft | Ausdruck | Anmerkungen |
|---|---|---|
| Gewicht | $\vec P = m\vec g$ | immer vertikal nach unten |
| Normalkraft | $\vec N \perp$ Oberfläche | Reaktion auf Druckbeanspruchung |
| Reibung | $f = \mu N$ | entgegen der Bewegung |
| Spannung | $T$ entlang des Seils | gleich an beiden Enden (ideales Seil) |
| Feder | $\vec F = -k\vec x$ | Hooke |
| Zentripetal | $F_c = mv^2/r$ | zum Mittelpunkt |

### Gleichgewicht in 2D

3 unabhängige Gleichungen: $\sum F_x = 0$, $\sum F_y = 0$, $\sum M_O = 0$ (Moment um einen beliebigen Punkt $O$).

</Porta>

<Porta nivel="5" titulo="Für ein 5-jähriges Kind">

Stell dir zwei Jungen vor, die an einer **Kiste** ziehen. Einer zieht **nach rechts**, der andere **nach links**, mit gleicher Kraft. **Die Kiste bewegt sich nicht** — sie heben sich auf.

Jetzt zieht einer **nach oben** und der andere **zur Seite**. Die Kiste bewegt sich **diagonal**, weil sich die Kräfte wie Pfeile **addieren**.

Wenn alle Pfeile zusammen **null** ergeben, bleibt die Kiste **stehen**. Das ist das **Gleichgewicht**!

</Porta>

<Porta nivel="10" titulo="Für ein 10-jähriges Kind">

In der Physik **hat jede Kraft eine Richtung**. Daher ist sie ein Vektor.

**Newton** entdeckte: $\vec F = m\vec a$. Je größer die Kraft, desto größer die Beschleunigung. Je größer die Masse, desto schwerer ist es zu beschleunigen.

**Gleichgewicht**: Wenn sich alle Kräfte aufheben (Summe null), bleibt das Objekt stehen oder bewegt sich geradlinig mit konstanter Geschwindigkeit.

**Rampen-Trick**: Die Schwerkraft zieht nach unten, aber nur ein **Teil** davon lässt das Objekt die Rampe hinunterrutschen. Der andere Teil drückt nur gegen die Rampe. Bei einer 30°-Rampe ist $\sin 30° = 0{,}5$ — nur die Hälfte der Schwerkraft zieht das Objekt die Rampe hinunter.

**Im Alltag**: Brückenkabel, Aufzugsspannung, Bremsen eines Autos, Basketballwurf — alles sind sich addierende Vektoren.

</Porta>

<Porta nivel="15" titulo="Für einen 15-jährigen Jugendlichen">

In der Physik ist **jede Größe mit Richtung ein Vektor**: Kraft, Geschwindigkeit, Beschleunigung, Verschiebung. Und die Vektorsumme **respektiert die Richtung**: 50 N nach rechts + 50 N nach links = 0 N (heben sich auf).

**Gleichgewicht**: Objekt in Ruhe $\Rightarrow$ alle Kräfte heben sich auf.

**Newton**: Resultierende Kraft = Masse × Beschleunigung. Mehr Kraft → mehr Beschleunigung. Mehr Masse → schwerer zu beschleunigen.

**Zerlegung** ist der nützlichste Trick: Auf einer 30°-Rampe teilt sich die Schwerkraft $g$ in $g \cos 30°$ Druck auf die Rampe (Normalkomponente) + $g \sin 30°$ Hangabtrieb (parallele Komponente). Es ist dieser parallele Anteil, der das Objekt rutschen lässt.

**Schräger Wurf**: $\vec v_0 = (v_0\cos\theta, v_0\sin\theta)$. Beschleunigung $\vec a = (0, -g)$. Parabolische Bahn:
- Flugzeit: $t = 2v_0\sin\theta / g$.
- Reichweite: $R = v_0^2 \sin(2\theta)/g$ — maximal bei $\theta = 45°$.
- Maximale Höhe: $h = v_0^2\sin^2\theta/(2g)$.

**Verbindung zu Lektion 27**: Arbeit $W = \vec F \cdot \vec d$ verwendet das Skalarprodukt.

</Porta>

<Porta nivel="25" titulo="Für einen Ingenieurstudenten">

Vektorielle Mechanik ist die Grundlage der Statik (Technische Mechanik I) und Dynamik (Technische Mechanik II) im Ingenieurstudium. Jede Analyse von:

- **Fachwerken**: Gleichgewicht an jedem Knoten (koplanare Kraftsysteme).
- **Balken**: Verteilung von Moment und Querkraft (Vektoren entlang der Länge).
- **Maschinen**: Mechanismenanalyse, kinematisches Paar.
- **Fahrzeugen**: Längs- und Querdynamik (Bremsen, Kurvenfahrt).
- **Bauwerken**: Gebäude, Brücken, Flugzeuge, Schiffe — alle gelöst über $\sum \vec F = 0$ + $\sum M = 0$.

### Vektorielle Gleichgewichtsanalyse

In 2D-Problemen: 2 Gleichungen ($\sum F_x = 0, \sum F_y = 0$) + 1 Moment ($\sum M = 0$). In 3D: 6 Gleichungen.

### Vollständige schiefe Ebene

Block der Masse $m$ auf einer Rampe $\theta$ mit Reibung $\mu$:

| Kraft | Parallele Komponente | Normale Komponente |
|---|---|---|
| Gewicht | $-mg\sin\theta$ | $-mg\cos\theta$ |
| Normal | 0 | $+N$ |
| Reibung | $\pm \mu N$ | 0 |

Normalgleichgewicht: $N = mg\cos\theta$. Maximalreibung daher $\mu mg\cos\theta$.

Block rutscht $\iff mg\sin\theta > \mu_s mg\cos\theta \iff \tan\theta > \mu_s$.

### Schräger Wurf

$\vec r(t) = (v_0\cos\theta \cdot t, v_0\sin\theta \cdot t - \frac{1}{2}g t^2)$. Kartesische Gleichung:
$$y = x\tan\theta - \frac{g x^2}{2 v_0^2 \cos^2\theta}$$

— eine **Parabel** (Lektion 25).

### Kreisbewegung

Zentripetalbeschleunigung: $\vec a_c = -\omega^2 \vec r$. Betrag $a_c = v^2/r$. Die Zentripetalkraft muss bereitgestellt werden (Seil, Reibung, Gravitation).

### Seile und Spannungen

Ideales Seil (masselos, undehnbar): Spannung ist über die gesamte Länge gleich. Ideale Rolle: ändert die Richtung, ohne den Betrag zu ändern.

### Verbindung zum [Nobelpreis](https://nobelprize.org)

Die vektorielle Mechanik ist Grundlage für die Bahnmechanik (Newton 1687) und die Relativitätstheorie (Einstein 1916). Die Detektion von Gravitationswellen (LIGO, [Nobel 2017](https://www.nobelprize.org/prizes/physics/2017/)) verwendet die vektorielle Zerlegung von Energie-Impuls-Tensoren.

</Porta>

<Porta nivel="40" titulo="Für einen erfahrenen Profi">

Die klassische Mechanik in vektorieller Form ist ein Spezialfall der **Lagrange- / Hamilton-Mechanik** — koordinatenwechselinvarianter Formalismus.

### Lagrange-Sicht

Lagrange-Funktion $L(q, \dot q, t) = T - V$ (kinetische minus potentielle Energie). Euler-Lagrange-Gleichungen:
$$\frac{d}{dt}\frac{\partial L}{\partial \dot q_i} - \frac{\partial L}{\partial q_i} = 0$$

Newton in kartesischen Koordinaten ist ein Sonderfall. In krummlinigen Koordinaten (polar, sphärisch, zylindrisch) wird Newton hässlich; Lagrange wird trivial.

### Anwendungen in Engineering / Quant

- **Robotik**: direkte/inverse Kinematik von Armen ist reine Vektorarithmetik. Quaternionen vermeiden Gimbal Lock.
- **Aerospace**: fünf Kräfte an einer Rakete (Schwerkraft, Schub, Widerstand, Auftrieb, Querkraft). 3D-Vektoranalyse.
- **Quant**: Multivariate Derivate (mehrdimensional Black-Scholes-Merton) verwenden Vektorarithmetik. Greeks (Delta, Gamma) sind Vektoren / Jacobi-Matrizen.
- **CFD**: Navier-Stokes-Gleichungen sind vektoriell. Diskretisierung über finite Volumen ist Vektorarithmetik auf einem Gitter.

### Vielteilchensysteme

Für $N$ Teilchen hat der Phasenraum die Dimension $6N$. In der statistischen Mechanik definieren Integrale vektorwertiger Funktionen über diesen Raum Temperatur, Druck etc.

### Numerisches

ODE-Integratoren für die Mechanik:
- **Explizites Euler**: $\vec r_{n+1} = \vec r_n + \vec v_n \Delta t$. Einfach, aber instabil.
- **Verlet/Leapfrog**: symplektisch, erhalten Energie in Hamiltonschen Systemen.
- **Runge-Kutta 4**: hohe Ordnung, aber nicht symplektisch.

Für Planetensimulation, Spiele, Proteinfaltung: Verlet > Euler.

### NumPy / SciPy

```python
import numpy as np
from scipy.integrate import solve_ivp

def projektil(t, y):
    g = 9.81
    return [y[2], y[3], 0, -g]   # dx/dt, dy/dt, dvx/dt, dvy/dt

v0, theta = 50, np.radians(60)
y0 = [0, 0, v0*np.cos(theta), v0*np.sin(theta)]
sol = solve_ivp(projektil, [0, 10], y0, dense_output=True, rtol=1e-9)
```

### Kuriosität

Newton veröffentlichte 1687 die *Principia* ohne moderne Vektornotation — die gesamte Formulierung ist geometrisch. Die moderne Vektoranalysis kam Ende des 19. Jahrhunderts mit Gibbs/Heaviside. Die **vektorielle Reformulierung der Principia** ist eine klassische didaktische Übung in Aufbaustudien.

<Porta nivel="pratica" titulo="Praktische Anwendung">

**Rezept: ein Mechanikproblem in 5 Schritten lösen.**

1. **Freikörperdiagramm (FKD)**: Zeichnen Sie das Objekt und alle daran wirkenden Kräfte.
2. **Achsen wählen**: Richten Sie sie an der Oberfläche (Rampe) oder an der Bewegung aus; dies vereinfacht die Zerlegung.
3. **Zerlegen Sie jede Kraft** in den Achsen.
4. **Wenden Sie an**: $\sum F_x = ma_x$, $\sum F_y = ma_y$ (und Moment, falls relevant).
5. **Lösen Sie das System**.

### Workflow Wolfram / NumPy

```python
import numpy as np

# Block auf einer Rampe mit Reibung
m, theta, mu, g = 5, np.radians(30), 0.2, 9.81
F_grav_par = -m*g*np.sin(theta)
F_normal   =  m*g*np.cos(theta)
F_reibung  = -mu*F_normal*np.sign(F_grav_par)  # entgegen der Bewegung
F_par_gesamt = F_grav_par + F_reibung
a = F_par_gesamt / m
print(f"a = {a:.2f} m/s²")
```

### Wann sollten Sie aufpassen?

- **Statische vs. dynamische Reibung**: unterschiedliche Koeffizienten; statisch bis zum Bewegungsbeginn, dann dynamisch.
- **Nicht-inertiales System**: Im Bezugssystem eines beschleunigenden Fahrzeugs erscheint eine „fiktive" Kraft $-m\vec a_0$.
- **Dehnbares Seil**: Spannung variiert entlang seiner Länge. Nicht ideal.
- **Vorzeichen**: Achsenorientierung kann Vorzeichen umkehren. Immer konsistent mit dem FKD.
