### Parallelität und Orthogonalität

Gegeben Geraden $r: y = m_r x + n_r$ und $s: y = m_s x + n_s$:

| Lage | Kriterium (Normalform) | Kriterium (allgemeine Form) |
|---|---|---|
| Parallel, verschieden | $m_r = m_s$, $n_r \neq n_s$ | $A_r/A_s = B_r/B_s \neq C_r/C_s$ |
| Identisch | $m_r = m_s$, $n_r = n_s$ | $A_r/A_s = B_r/B_s = C_r/C_s$ |
| Schneidend | $m_r \neq m_s$ | $A_r B_s - A_s B_r \neq 0$ |
| Senkrecht | $m_r \cdot m_s = -1$ | $A_r A_s + B_r B_s = 0$ |

Sonderfälle mit vertikalen Geraden: $x = a$ ist vertikal, $y = b$ ist horizontal. Vertikale untereinander sind parallel; vertikal und horizontal sind senkrecht.

### Winkel zwischen zwei Geraden

$$\tan\theta = \left|\frac{m_r - m_s}{1 + m_r m_s}\right|$$

(unter der Annahme, dass keine Gerade vertikal ist).

### Abstand Punkt-Gerade

Punkt $P_0 = (x_0, y_0)$ und Gerade $Ax + By + C = 0$:

$$d(P_0, r) = \frac{|A x_0 + B y_0 + C|}{\sqrt{A^2 + B^2}}$$

### Abstand zwischen parallelen Geraden

Für $r: Ax + By + C_1 = 0$ und $s: Ax + By + C_2 = 0$:
$$d(r, s) = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$

### Mittelsenkrechte

Menge der Punkte, die gleich weit von $A$ und $B$ entfernt sind. Es ist die Gerade, die senkrecht auf $\overline{AB}$ steht und durch den Mittelpunkt verläuft.

</Porta>

<Porta nivel="5" titulo="Für ein 5-jähriges Kind">

Stell dir **zwei Eisenbahnschienen** vor. Sie verlaufen nebeneinander, **treffen sich nie**. Das ist **parallel**.

Jetzt stell dir zwei **Lineale vor, die ein Pluszeichen** $+$ bilden. Sie kreuzen sich in einer **rechten Ecke**. Das ist **senkrecht**.

Wenn zwei Geraden weder parallel noch senkrecht sind, **kreuzen sie sich einfach in einem schiefen Punkt**.

</Porta>

<Porta nivel="10" titulo="Für ein 10-jähriges Kind">

Zwei Geraden in der Ebene können 4 Dinge tun:

1. **Parallel**: treffen sich nie (gleiche Steigung).
2. **Identisch**: sind dieselbe Gerade.
3. **Schneidend**: kreuzen sich in einem Punkt.
4. **Senkrecht**: kreuzen sich im rechten Winkel.

**Wie erkennt man das?** Schau auf die Steigung $m$:
- $m_1 = m_2$: parallel (oder identisch, wenn auch $n$ gleich).
- $m_1 \cdot m_2 = -1$: senkrecht.
- Sonst: schneiden sich in irgendeinem Punkt.

**Konkretes Beispiel**: $y = 2x + 1$ und $y = 2x - 5$ sind parallel (beide haben Steigung 2). $y = 2x$ und $y = -x/2$ sind senkrecht ($2 \cdot (-1/2) = -1$).

</Porta>

<Porta nivel="15" titulo="Für einen 15-jährigen Jugendlichen">

Zwei Geraden in der Ebene können:

1. **Parallel sein** (gleiche Steigung, kreuzen sich nie).
2. **Senkrecht sein** (bilden einen 90°-Winkel).
3. **Sich in einem Punkt schneiden** (schneidend).
4. **Dieselbe Gerade sein** (identisch).

**Parallelitätskriterium**: $m_r = m_s$ — gleiche Steigung.

**Orthogonalitätskriterium**: $m_r \cdot m_s = -1$. Warum? Denk an $m_1 = \tan\alpha_1$. Wenn $\alpha_2 = \alpha_1 + 90°$, dann $\tan\alpha_2 = -\cot\alpha_1 = -1/m_1$. Also $m_1 m_2 = -1$.

**Vollständiges numerisches Beispiel**: 
- $r: y = 2x + 1$ und $s: y = -x/2 + 5$. Produkt $2 \cdot (-1/2) = -1$. **Senkrecht**.
- Schnittpunkt: $2x + 1 = -x/2 + 5 \Rightarrow 5x/2 = 4 \Rightarrow x = 8/5$. Einsetzen: $y = 16/5 + 1 = 21/5$. Sie schneiden sich in $(8/5, 21/5)$.

**Der Abstand Punkt-Gerade** ist grundlegend — er wird in fast jedem nachfolgenden geometrischen Problem verwendet. Die Formel $|Ax_0 + By_0 + C|/\sqrt{A^2 + B^2}$ entsteht durch die Projektion des Punktes auf die Normalenrichtung der Geraden.

</Porta>

<Porta nivel="25" titulo="Für einen Ingenieurstudenten">

In der Praxis ist das **Lösen von 2x2-linearen Gleichungssystemen** der Schnitt von Geraden. Cramer und Gauß-Verfahren beginnen hier.

### 2x2-System und geometrische Interpretation

$$\begin{cases} a_1 x + b_1 y = c_1 \\ a_2 x + b_2 y = c_2 \end{cases}$$

| Fall | Geometrie | Determinante |
|---|---|---|
| Eindeutige Lösung | schneidende Geraden | $\det \neq 0$ |
| Unendlich viele Lösungen | identische Geraden | $\det = 0$ + konsistent |
| Keine Lösung | verschiedene parallele Geraden | $\det = 0$ + inkonsistent |

### Unmittelbare Anwendungen

- **Lineare Regression**: Die bestpassende Gerade steht senkrecht auf der Richtung der maximalen Fehlervarianz.
- **CG / SVG**: Polygon-Clipping (Sutherland-Hodgman) testet Geradenseiten über das Vorzeichen von $Ax + By + C$.
- **Robotik**: Bahnplanung umgeht Hindernisse, die durch Halbebenen dargestellt werden.
- **GIS**: Nähe-Analyse — Abstand jedes Punktes zum Straßennetz.

### Beweis der Formel $m_1 m_2 = -1$

Seien $\vec u_1 = (1, m_1)$ und $\vec u_2 = (1, m_2)$ Richtungsvektoren. Orthogonalität $\iff \vec u_1 \cdot \vec u_2 = 0 \iff 1 + m_1 m_2 = 0 \iff m_1 m_2 = -1$.

### Mittelsenkrechte als geometrischer Ort

Punkte $(x, y)$ gleich weit entfernt von $A = (x_A, y_A)$ und $B = (x_B, y_B)$:
$$(x - x_A)^2 + (y - y_A)^2 = (x - x_B)^2 + (y - y_B)^2$$

Beim Ausmultiplizieren heben sich die quadratischen Terme weg, übrig bleibt eine Gerade $\Rightarrow$ sie steht senkrecht auf $\overline{AB}$ und verläuft durch den Mittelpunkt.

</Porta>

<Porta nivel="40" titulo="Für einen erfahrenen Profi">

Der Begriff der „Orthogonalität" zwischen Geraden verallgemeinert sich auf die **Orthogonalität von Unterräumen** in jedem Skalarproduktraum.

### Sicht der linearen Algebra

Gerade = 1D affiner Unterraum. Die Richtung ist ein Vektor $\vec v$. Zwei Geraden $r_1, r_2$ parallel $\iff \vec v_1 \parallel \vec v_2$. Senkrecht $\iff \langle \vec v_1, \vec v_2 \rangle = 0$.

In $\mathbb{R}^n$ ist das **orthogonale Komplement** eines Unterraums $V$ definiert als $V^\perp = \{w : \langle v, w \rangle = 0 \forall v \in V\}$. $V \oplus V^\perp = \mathbb{R}^n$. Grundlage für Gram-Schmidt, QR und SVD.

### Anwendungen in ML / Quant

- **Lineare Regression**: Der Residuenvektor steht orthogonal auf dem Spaltenraum von $X$ (Orthogonalitätsprinzip). Formal projiziert $\hat\beta = (X^TX)^{-1} X^Ty$ den Vektor $y$ orthogonal.
- **PCA**: Hauptkomponenten sind orthogonal — Eigenwerte der Kovarianzmatrix.
- **Quant Finance**: Portfolio-Rebalancing nutzt die orthogonale Projektion erwarteter Renditen auf den Constraint-Raum.
- **Quanten**: Orthogonale Zustände $\langle \psi | \phi \rangle = 0$ entsprechen „gegenseitig ausschließenden" Messungen.

### Abstand über Projektion

Die Formel $d = |Ax_0 + By_0 + C|/\sqrt{A^2+B^2}$ ist der Betrag der Projektion des „Punkt-zu-Gerade"-Vektors auf den Einheitsnormalenvektor $(A, B)/\sqrt{A^2+B^2}$. Verallgemeinert sich auf den Abstand Punkt-Hyperebene in $\mathbb{R}^n$.

### Numerisches

Vorsicht bei nahezu parallelen Geraden in linearen Systemen — die Konditionszahl $\kappa(A) = \cot(\theta/2)$ explodiert für $\theta \to 0$. Partielle Pivotierung in Gauß + Skalierung mildert dies.

### SymPy

```python
import sympy as sp
from sympy import Line, Point
r = Line(Point(0, 0), Point(1, 2))
s = Line(Point(0, 5), Point(1, 7))
print(r.is_parallel(s))      # True
print(r.is_perpendicular(s)) # False
print(r.angle_between(s))    # 0
print(s.distance(Point(0, 0)))
```

<Porta nivel="pratica" titulo="Praktische Anwendung">

**Rezept: zwei Geraden in 3 Schritten klassifizieren.**

1. **Vergleichen Sie die Steigungen** $m_1, m_2$:
   - Gleich? Dann parallel/identisch.
   - Produkt $-1$? Senkrecht.
   - Sonst? Schneidend — Schnittpunkt finden.
2. **Falls parallel**: Vergleichen Sie $n_1, n_2$. Gleich $\Rightarrow$ identisch; sonst verschieden.
3. **Für den Winkel**: Verwenden Sie $\tan\theta = |(m_1 - m_2)/(1 + m_1 m_2)|$.

### Vollständiger Workflow

```python
import sympy as sp
from sympy import Line, Point, pi
r = Line(Point(0, 1), Point(1, 3))   # y = 2x + 1
s = Line(Point(0, 5), Point(2, 4))   # y = -x/2 + 5
print(r.is_perpendicular(s))                   # True
print(r.intersection(s))                       # Schnittpunkt
print(sp.deg(r.angle_between(s)))              # 90
print(s.distance(Point(0, 0)))                 # Abstand Ursprung-Gerade
```

### Wann sollten Sie aufpassen?

- **Vertikale Geraden**: $m$ undefiniert — verwenden Sie die allgemeine Form.
- **Nahezu parallel**: Eine kleine Störung verändert den Schnittpunkt stark. In der Praxis $\det$ berechnen und verwerfen, falls $|\det| < \epsilon$.
- **Abstand vs. Vorzeichen**: Die Formel $|Ax_0 + By_0 + C|$ enthält den Betrag. Ohne Betrag liefert sie die **Seite** der Geraden — nützlich in Clipping-Algorithmen.
