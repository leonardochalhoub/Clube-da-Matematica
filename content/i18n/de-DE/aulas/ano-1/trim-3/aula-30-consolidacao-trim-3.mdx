<Porta nivel="10" titulo="Für ein 10-jähriges Kind">

**Wiederholung** der Themen:
1. **Kartesische Ebene**: Punkte mit Koordinaten $(x, y)$.
2. **Gerade**: Gleichung $y = mx + n$.
3. **Kreis**: $(x - a)^2 + (y - b)^2 = r^2$.
4. **Kegelschnitte**: Ellipse, Parabel, Hyperbel.
5. **Vektoren**: Pfeile in der Ebene, die addiert werden.
6. **Skalarprodukt**: Kombiniert zwei Vektoren und ergibt eine Zahl.
7. **Lineare Gleichungssysteme**: Mehrere Gleichungen gleichzeitig.

**In dieser Lektion** Probleme, die alles zusammen verwenden. Zum Beispiel: Schnittpunkte einer Geraden mit einem Kreis finden (Lektionen 22 + 24). Oder Spannungen in Seilen berechnen, die eine Last halten (Vektoren + lineares System).

</Porta>

### Querverbindungen

- **Quant Finance + Kegelschnitte** (Lektionen 25, 27): Markowitz ist eine Hyperbel in der Risiko-Rendite-Ebene.

### Synthese der kartesischen Methode

Descartes (1637) lehrte: **Übersetzen Sie jedes geometrische Problem in Algebra**, lösen Sie und übersetzen Sie zurück. Diese Lektion ist die letzte Übung dieser Übersetzung in 2D, bevor es im Quartal 4 in 3D weitergeht.

### Die nächsten Quartale verwenden all dies

- **Quartal 4 (Raumgeometrie)**: 3D-Abstand, Ebenen, Sphären — direkte Erweiterung.
- **Quartal 5 (Funktionen)**: Untersucht Graph-Transformationen ($f(x-a) + b$) — Vektor-Translationen.
- **Quartal 6 (Analysis)**: Ableitung als Steigung, Integral als Fläche. Alles ist beschleunigte analytische Geometrie.
- **Quartal 12 (Formale lineare Algebra)**: Vektoren und Systeme in $\mathbb{R}^n$ rigoros.

| Kegelschnitt | quadratische Form $\mathbf x^T A \mathbf x = 1$ |

### Anwendungen in ML / Quant

- **Attention in Transformern**: $\text{softmax}(QK^T/\sqrt d)V$. $QK^T$ ist eine Matrix von Skalarprodukten.
- **Markowitz** (1952, [Nobel 1990](https://www.nobelprize.org/prizes/economic-sciences/1990/)): minimiert $\mathbf w^T \Sigma \mathbf w$ — verallgemeinertes Skalarprodukt. Die Frontier ist eine Hyperbel.
- **Black-Scholes** (1973, [Nobel 1997](https://www.nobelprize.org/prizes/economic-sciences/1997/)): parabolische Wärmeleitungsgleichung in ($\ln S$, $t$). Monte-Carlo-Simulation arbeitet in $\mathbb{R}^N$.
- **Word Embeddings**: Ähnlichkeit $= \cos\theta$ zwischen Vektoren in $\mathbb{R}^{300}$.

### Workflow Wolfram / SymPy

```python
import sympy as sp
from sympy.geometry import Point, Line, Circle, Triangle

# Beispiel typisches Problem: Tangente von externem Punkt an Kreis
C = Circle(Point(0, 0), 5)
P = Point(7, 0)
print(C.tangent_lines(P))      # 2 Tangenten
print(P.distance(C.center)**2 - C.radius**2)  # Tangentenlänge^2

# Geometriesystem
T = Triangle(Point(0,0), Point(6,0), Point(0,8))
print(T.incenter, T.incircle.radius)  # Inkreismittelpunkt und Radius
```

### Wann sollten Sie aufpassen?

- **Kontexte mischen**: Vorsicht, keine Ellipsenformel auf einen Kreis anwenden (sie sind äquivalent für $a = b$, aber die Notation ändert sich).
- **Einheiten**: Arbeit in Joule, Kraft in Newton, Abstand in Metern — immer konsistent.
- **Vorzeichen des Kosinus**: Negatives Skalarprodukt bedeutet stumpfen Winkel, nicht Fehler.
- **Größenordnung prüfen**: Spannung in einem Seil, das 1000 kg trägt, muss $\geq 5000$ N sein (Schwerkraft) — viel weniger ist ein Fehler.
