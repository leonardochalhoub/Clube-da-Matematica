<Porta nivel="formal" titulo="Axiome und Formeln">

### Stichprobenraum

$\Omega$: Menge aller möglichen Ergebnisse des Experiments. **Ereignis**: Teilmenge von $\Omega$.

### Kolmogorov-Axiome (1933)

1. $P(A) \geq 0$ für jedes Ereignis $A$.
2. $P(\Omega) = 1$.
3. **$\sigma$-Additivität**: bei disjunkten $A_1, A_2, \ldots$: $P(\bigcup A_i) = \sum P(A_i)$.

### Klassische Wahrscheinlichkeit (gleichwahrscheinlich)

$$P(A) = \frac{|A|}{|\Omega|}$$

### Unmittelbare Eigenschaften

- $P(\emptyset) = 0$.
- $P(A^c) = 1 - P(A)$.
- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$ (Inklusion-Exklusion).
- $A \subseteq B \Rightarrow P(A) \leq P(B)$.
- $P(A) \in [0, 1]$.

### Bedingte Wahrscheinlichkeit

$$P(A | B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0$$

### Unabhängigkeit

$A$ und $B$ sind **unabhängig**, wenn $P(A \cap B) = P(A) P(B)$ (äquivalent: $P(A|B) = P(A)$).

### Bayes-Theorem

$$P(A | B) = \frac{P(B | A) P(A)}{P(B)}$$

### Totale Wahrscheinlichkeit

Wenn $\{A_i\}$ eine Partition von $\Omega$ ist:
$$P(B) = \sum_i P(B | A_i) P(A_i)$$

### Diskrete Zufallsvariable

$X: \Omega \to \mathbb R$. Verteilung: $P(X = x_i) = p_i$ mit $\sum p_i = 1$.

| Verteilung | Formel | Auftreten |
|---|---|---|
| Bernoulli | $P(X=1) = p$ | 1 Versuch |
| Binomial | $\binom{n}{k}p^k(1-p)^{n-k}$ | $n$ Versuche |
| Geometrisch | $(1-p)^{k-1}p$ | Wartezeit |
| Poisson | $e^{-\lambda}\lambda^k/k!$ | seltene Ereignisse |

### Erwartungswert und Varianz

- $E[X] = \sum x_i p_i$
- $\text{Var}(X) = E[(X - E[X])^2] = E[X^2] - E[X]^2$

Wahrscheinlichkeit ist die „**Chance**, dass etwas passiert". Sie geht von 0 (unmöglich) bis 1 (sicher).

Wirf eine Münze: Chance auf Kopf ist die Hälfte (1/2), denn es gibt 2 Ergebnisse und eines davon ist Kopf.

Wirf einen Würfel: Chance auf eine 6 ist ein Sechstel (1/6), denn der Würfel hat 6 Seiten und nur eine ist die 6.

Je **mehr Möglichkeiten**, desto **schwieriger** zu treffen.

<Porta nivel="10" titulo="Für ein 10-jähriges Kind">

**Wahrscheinlichkeit** = Chance, dass etwas passiert, eine Zahl zwischen 0 und 1.

**Klassische Regel** (wenn alle Ergebnisse gleichwahrscheinlich sind):
$$P(\text{Ereignis}) = \frac{\text{günstige Fälle}}{\text{Gesamtzahl der Fälle}}$$

Beispiele:
- Würfel: $P(\text{gerade}) = 3/6 = 1/2$.
- Münze × 2: $P(\text{2 Köpfe}) = 1/4$ (KK, KZ, ZK, ZZ — nur KK).
- Karten: $P(\text{König}) = 4/52 = 1/13$.

**Bedingte Wahrscheinlichkeit**: „Wenn ich schon weiß, dass B passiert ist, was ist die Chance auf A?". Schreibweise $P(A|B)$.

**Unabhängigkeit**: Das Ergebnis von A beeinflusst B nicht. Beispiel: 2 Würfel werfen — der zweite Würfel kümmert sich nicht um den ersten.

**Bayes**: Es gibt eine clevere Formel zum „Umkehren" der Wahrscheinlichkeiten. Wie das Erkennen der Ursache aus der Wirkung.

**Wo es vorkommt**: Lotterien, Spiele, Wettervorhersage, medizinische Diagnose, Spam-Filter, KI.

</Porta>

<Porta nivel="15" titulo="Für einen 15-jährigen Jugendlichen">

Wahrscheinlichkeit ist „eine Zahl, die die Chance misst", dass etwas passiert. Sie geht von 0 (unmöglich) bis 1 (sicher).

**Klassischer Fall**: Experiment mit $|\Omega|$ gleichwahrscheinlichen Ergebnissen (Münzwurf, Würfeln). $P(A)$ = „günstige Fälle geteilt durch mögliche Fälle".

Beispiel: Würfel, $P(\text{gerade}) = 3/6 = 1/2$. Münze 2-mal, $P(\text{2 Köpfe}) = 1/4$.

**Bedingte Wahrscheinlichkeit**: „Gegeben dass B eingetreten ist, was ist die Wahrscheinlichkeit von A?". $P(A|B) = P(A \cap B)/P(B)$.

**Unabhängige Ereignisse**: $P(A \cap B) = P(A) P(B)$. 2 Würfel werfen — Ergebnis des 2. beeinflusst den 1. nicht.

**Bayes**: Möglichkeit, das Bedingte „umzukehren". Wenn Sie $P(B|A)$ kennen, können Sie $P(A|B)$ berechnen. Verwendet in medizinischer Diagnose, Spam-Filter, ML.

**Inklusion-Exklusion**: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$. Wenn die Ereignisse nicht disjunkt sind, wird der Schnitt zweimal gezählt — abziehen.

**Binomialverteilung**: Anzahl der Erfolge in $n$ unabhängigen Versuchen:
$$P(X = k) = \binom{n}{k}p^k(1-p)^{n-k}$$

Beispiel: Münze 10-mal werfen, was ist die Chance auf genau 5 Köpfe? $\binom{10}{5}(1/2)^{10} = 252/1024 \approx 0{,}246$.

**Kombinatorik + Wahrscheinlichkeit gehen Hand in Hand**: Zählen liefert $|A|$ und $|\Omega|$, Division liefert $P(A)$.

</Porta>

<Porta nivel="25" titulo="Für einen Ingenieurstudenten">

Wahrscheinlichkeit ist die **Grundlage** von Data Science, ML, stochastischer Regelung, quantitativer Finanzmathematik.

### Zufallsvariable

$X: \Omega \to \mathbb{R}$. Numerischer Wert, der jedem Ergebnis zugeordnet ist. **Diskret** oder **stetig** (in den nächsten Reihen).

### Erwartungswert

$E[X] = \sum_i x_i P(X = x_i)$. Linear: $E[aX + bY] = aE[X] + bE[Y]$ — immer, auch ohne Unabhängigkeit.

### Varianz

$\text{Var}(X) = E[(X - \mu)^2] = E[X^2] - \mu^2$. **Nicht linear**: $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)$.

Wenn $X, Y$ unabhängig: $\text{Cov} = 0$, daher $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y)$.

### Binomialverteilung

$X \sim \text{Bin}(n, p)$: Anzahl „Erfolge" in $n$ unabhängigen Versuchen mit Wahrscheinlichkeit $p$:
$$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}$$

Erscheint in der Qualitätskontrolle, in klinischen Studien, im A/B-Testing.

$E[X] = np$, $\text{Var}(X) = np(1-p)$.

### Bayes in der Praxis — medizinische Diagnose

Seltene Krankheit $D$ mit $P(D) = 0{,}001$. Test: $P(+|D) = 0{,}99$ (Sensitivität), $P(-|D^c) = 0{,}95$ (Spezifität). Wahrscheinlichkeit der Krankheit bei positivem Test:
$$P(D | +) = \frac{P(+|D) P(D)}{P(+)} = \frac{0{,}99 \cdot 0{,}001}{0{,}99 \cdot 0{,}001 + 0{,}05 \cdot 0{,}999} \approx 0{,}019$$

Nur **1,9 %** — Base-Rate-Fallacy, erklärt durch Bayes.

### Querverbindungen

- **Engineering**: Zuverlässigkeit — Serien-/Parallelsysteme, MTBF.
- **Signale**: Detektion im Rauschen über Neyman-Pearson.
- **ML**: Bayessche Klassifikatoren, probabilistische neuronale Netze.
- **Quant**: risk-neutrales Pricing, Value-at-Risk.
- **Statistik**: Hypothesentests, p-Werte, Konfidenzintervalle.

### Gesetz der großen Zahlen

$\bar X_n = \frac{1}{n}\sum X_i \xrightarrow{n\to\infty} E[X]$ (Konvergenz in Wahrscheinlichkeit oder fast sicher). Rechtfertigt relative Häufigkeiten als Wahrscheinlichkeitsschätzungen.

### Zentraler Grenzwertsatz (Vorschau)

$\frac{\bar X - \mu}{\sigma/\sqrt n} \xrightarrow{d} N(0, 1)$. Summe vieler unabhängiger Variablen tendiert zur Gauß-Verteilung — erklärt die Allgegenwart der Normalverteilung.

Die moderne Wahrscheinlichkeitstheorie ist auf **Maßräumen** $(\Omega, \mathcal F, P)$ im Formalismus von Kolmogorov (1933) aufgebaut. Jede Zufallsvariable ist eine messbare Funktion.

### Kolmogorov und Maß

$\sigma$-Algebra $\mathcal F$ — messbare Ereignisse. Notwendig, weil in stetigem $\Omega$ (z. B. $[0,1]$) nicht jede Teilmenge messbar ist (Vitali). Im diskreten Fall reicht $\mathcal F = 2^\Omega$.

### Bayessche Inferenz

Posterior $\propto$ Prior $\times$ Likelihood. MCMC (Metropolis-Hastings, HMC, NUTS) sampelt aus beliebigen Posteriors. Stan, PyMC3, Pyro, NumPyro. Standardtools moderner Data Science.

### Quant — Maßwechsel

Risk-neutrales Pricing: $V_0 = e^{-rT} E^Q[\text{Payoff}]$. Radon-Nikodym-Maßwechsel von $P$ (Real-World) zu $Q$ (risk-neutral). Girsanov für stetige Prozesse.

### Konzentrationsungleichungen

- **Markov**: $P(X \geq a) \leq E[X]/a$.
- **Tschebyschew**: $P(|X-\mu| \geq k\sigma) \leq 1/k^2$.
- **Hoeffding**: $P(|\bar X_n - \mu| \geq t) \leq 2e^{-2nt^2/(b-a)^2}$ — Grundlage des PAC-Lernens.
- **Chernoff**: exponentielle Schranke — Kommunikation, RNG, Hashing.

### ML und Wahrscheinlichkeit

- **Cross-Entropy-Loss**: $-\sum p_i \log q_i$.
- **KL-Divergenz**: $D_{KL}(p\|q) = \sum p_i \log(p_i/q_i)$ — Grundlage variationaler Methoden.
- **Score-Based Generative Models** (Diffusion): lernen $\nabla \log p$.
- **Neuronale Netze** können als universelle Approximatoren bedingter Verteilungen $P(Y|X)$ verstanden werden.

### Forschung

- **Freie Wahrscheinlichkeit** (Voiculescu): nicht-kommutative Algebra für große Zufallsmatrizen.
- **Optimal Transport** (Villani, Cuturi): Distanz zwischen Verteilungen — Wasserstein.
- **Steins Methode**: Schranken für Fehler in CLT und Varianten.
- **Markov Chain Mixing**: Cutoff-Phänomene, Diaconis.

<Porta nivel="pratica" titulo="Praktische Anwendung">

**Workflow für Wahrscheinlichkeitsprobleme.**

1. **$\Omega$ und das Ereignis $A$ klar definieren**. 80 % der Fehler kommen von ungenauer Definition.
2. **Gleichwahrscheinlich?** Verwende Kombinatorik ($|A|/|\Omega|$).
3. **Nicht gleichwahrscheinlich?** Verwende Eigenschaften + Axiome.
4. **Bedingt oder Bayes?** Identifiziere, was gegeben und was gesucht ist.
5. **Unabhängigkeit?** Multiplizieren. Sonst bedingen.

### Python-Rezept

```python
import math
from math import comb
from scipy.stats import binom, norm, poisson

# Klassische Wahrscheinlichkeit
def klassisch(guenstig, gesamt):
    return guenstig / gesamt

print(klassisch(comb(4, 2), comb(52, 5)))   # P(2 Könige in 5 Karten) — angepasst

# Binomialverteilung
n, p = 10, 0.5
print(binom.pmf(5, n, p))     # P(X=5)
print(binom.cdf(3, n, p))     # P(X<=3)
print(binom.mean(n, p))       # n*p

# Bayes manuell
def bayes(prior_A, likelihood_B_given_A, likelihood_B_given_notA):
    prior_notA = 1 - prior_A
    p_B = likelihood_B_given_A * prior_A + likelihood_B_given_notA * prior_notA
    return likelihood_B_given_A * prior_A / p_B

# Seltene Krankheit
print(bayes(0.001, 0.99, 0.05))   # P(D|+) ~ 0.019

# Monte-Carlo-Simulation
import numpy as np
np.random.seed(42)
samples = np.random.choice([0, 1], size=10000, p=[0.5, 0.5])
print((samples.sum()) / 10000)    # ~ 0.5
```

### Bayes über eine Tabelle

```python
# Für kleine Probleme eine 2x2-Tabelle aufstellen
import pandas as pd
df = pd.DataFrame({
    "D": [0.001 * 0.99, 0.001 * 0.01],   # hat Krankheit
    "notD": [0.999 * 0.05, 0.999 * 0.95]  # nicht
}, index=["+", "-"])
print(df.div(df.sum(axis=1), axis=0))   # bedingt
```

### Fallstricke

- **$P(A|B)$ und $P(B|A)$ unterscheiden**. Bayes dreht sich genau um diese Umkehr.
- **Unabhängigkeit ≠ sich gegenseitig ausschließend**. Unabhängige können koexistieren; sich ausschließende nicht.
- **Nicht gleichwahrscheinliche Fälle**: 2 Würfel werfen → 36 gleichwahrscheinliche Ergebnisse, ABER Summen 2 bis 12 sind NICHT gleichwahrscheinlich.
- **Base-Rate-Fallacy**: ignoriert die Basis — die a-priori-Wahrscheinlichkeit zählt enorm.
