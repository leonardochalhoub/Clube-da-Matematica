<Porta nivel="formal" titulo="証明と利用">

### 正弦定理

<Equation explicacao="任意の三角形(鋭角、鈍角、直角)に対して成立する。Rは外接円の半径。">
{`\\frac{a}{\\sin A} = \\frac{b}{\\sin B} = \\frac{c}{\\sin C} = 2R`}
</Equation>

**証明**(鋭角三角形の場合):頂点$C$から辺$\overline{AB}$への高さ$h$を作図する。すると$h = b \sin A = a \sin B$。したがって$a/\sin A = b/\sin B$。$c$についても同様。∎

**特殊な場合**($C$で直角):$\sin C = 1$、よって$c = 2R$ — 斜辺は外接円の直径である。**ターレスの定理**(幾何学的)。

### 余弦定理

<Equation explicacao="ピタゴラスの定理を一般化したもの。C = 90°のとき、cos C = 0となり、c² = a² + b²が回復される。">
{`c^2 = a^2 + b^2 - 2ab\\cos C`}
</Equation>

**証明**:ベクトルの内積$\vec{CB} - \vec{CA} = \vec{AB}$より:
$$|\vec{AB}|^2 = |\vec{CB}|^2 + |\vec{CA}|^2 - 2 \vec{CB} \cdot \vec{CA}$$

$\vec{CB} \cdot \vec{CA} = |\vec{CB}||\vec{CA}|\cos C = ab \cos C$より、$c^2 = a^2 + b^2 - 2ab \cos C$が得られる。∎

### どちらの定理を使うか

| あるもの | 求めたいもの | 使う |
|---|---|---|
| 2角 + 1辺 (AAS, ASA) | 他の辺 | **正弦定理** |
| 2辺 + 一方の対角 (SSA) | 残り(曖昧!) | **正弦定理** |
| 2辺 + その間の角 (SAS) | 第3の辺 | **余弦定理** |
| 3辺 (SSS) | ある角 | **余弦定理**を逆に |

### 曖昧な場合 (SSA)

$a$、$b$、$A$($a$の対角)が与えられている場合:三角形は0個、1個、または2個存在し得る。判定:
- $a \geq b$なら:1個。
- $a < b \sin A$なら:0個(幾何学的に不可能)。
- $b \sin A < a < b$なら:2個。

</Porta>

<Porta nivel="15" titulo="15歳の青少年向け">

第11課で直角三角形を「解く」ことを学んだ:1辺と1角が与えられれば、他の辺を求められる。

しかし、三角形に**90°の角がない**場合はどうだろう?その場合、ピタゴラスの定理は使えず、直角三角形の正弦/余弦も使えない。

解決策:**正弦定理**と**余弦定理**。これらは**任意の**三角形で機能する。

**正弦定理**:各辺と対角の正弦の比は一定。
$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}$$

**余弦定理**:$c^2 = a^2 + b^2 - 2ab\cos C$。これは辺$a$と$b$の間の角の余弦によって「修正された」ピタゴラスの定理。

実践ではこうだ:3つの測定値(3辺、または2辺と1角など)があり、残りを求めたい。正しい定理を適用し、計算する。

**例**:北に5 km歩き、その後60°東に向きを変えてさらに3 km歩く。出発点からの距離は?

余弦定理より:$d^2 = 5^2 + 3^2 - 2 \cdot 5 \cdot 3 \cdot \cos(120°) = 25 + 9 + 15 = 49$。よって$d = 7$ km。

(注:経路の偏向により、三角形の内角は$180° - 60° = 120°$である。)

</Porta>

<Porta nivel="25" titulo="工学部学生向け">

正弦定理と余弦定理は以下で基本的:

- **測量 / 地籍**:直接アクセスできない地点の三角測量。
- **航海**:GPSは4個以上の衛星を組み合わせた3D三角測量を行う — 余弦定理の多次元バージョン。
- **天文学**:恒星視差は正弦定理を使用。
- **ロボット工学**:2セグメントのロボットアームの逆運動学は余弦定理で解ける。

### 三角形の面積

$$\text{面積} = \frac{1}{2} ab \sin C$$

「底辺 × 高さ / 2」で高さ $= a \sin C$から得られる。$C = 90°$のとき、「直角辺 × 直角辺 / 2」が回復される。

### ヘロンの公式

3辺$a, b, c$と半周長$s = (a+b+c)/2$が与えられたとき:
$$\text{面積} = \sqrt{s(s-a)(s-b)(s-c)}$$

余弦定理 + 恒等式$\sin^2 + \cos^2 = 1$の組み合わせで証明可能。

</Porta>
