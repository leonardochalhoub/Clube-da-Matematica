<Porta nivel="formal" titulo="定義と性質">

<Definicao titulo="一次関数">
$f: \mathbb{R} \to \mathbb{R}$ は、$f(x) = ax + b$ となる $a, b \in \mathbb{R}$ が存在するとき**一次関数**という。
</Definicao>

- $a$：**傾き**（slope）
- $b$：**y切片**
- グラフ：直線。$a > 0$：単調増加。$a < 0$：単調減少。$a = 0$：定数。

<Equation
  numero="1"
  audioTexto="式1。f(x_2) - f(x_1) を x_2 - x_1 で割ると a になる。一次関数の任意の2点間の変化率は一定で傾きに等しい。"
  explicacao="直線の2点間の変化率。一次関数では、この値は一定 — どの2点を選んでも依存しません。この一定性が他のすべての関数の中で一次関数を特徴づけます。">
{`\\frac{f(x_2) - f(x_1)}{x_2 - x_1} = a`}
</Equation>

### 関数の零点と切片

$f(x) = 0 \iff x = -b/a$（$a \neq 0$ のとき）。$(0, b)$ は **y切片**。$(-b/a, 0)$ は**零点**（または x切片）。

### 2点による一意決定定理

<Teorema titulo="2点が一次関数を決める">
$x_1 \neq x_2$ かつ異なる2点 $(x_1, y_1)$、$(x_2, y_2)$ が与えられたとき、両方を通る一次関数 $f(x) = ax + b$ は一意に存在する。係数は：
$$a = \frac{y_2 - y_1}{x_2 - x_1}, \quad b = y_1 - a x_1$$
</Teorema>

**証明（概略）。** 存在性：上の公式で $a$、$b = y_1 - a x_1$ と定義する。構成より $f(x_1) = y_1$、また $f(x_2) = a(x_2 - x_1) + y_1 = (y_2 - y_1) + y_1 = y_2$ が確認される。一意性：$g(x) = a' x + b'$ も $g(x_i) = y_i$ を満たすなら、$a' = (y_2 - y_1)/(x_2 - x_1) = a$ かつ $b' = y_1 - a' x_1 = b$。∎

### 合成と演算

$f(x) = a_1 x + b_1$、$g(x) = a_2 x + b_2$ とする。すると：

- **和**：$(f + g)(x) = (a_1 + a_2) x + (b_1 + b_2)$ — 一次、傾きが和。
- **合成**：$(f \circ g)(x) = a_1 (a_2 x + b_2) + b_1 = a_1 a_2 x + (a_1 b_2 + b_1)$ — 一次、傾きが積。
- **逆関数**（$a_1 \neq 0$ のとき）：$f^{-1}(y) = (y - b_1)/a_1$ — これも一次、傾き $1/a_1$。

**可逆**な一次関数（$a \neq 0$）の集合は合成について群をなす — 構造 $(\mathbb{R}^* \times \mathbb{R}, \circ)$。この観察は線形代数（第31講以降）と仿射幾何で使われます。

### 解いた数値例

**問題。** $(2, 5)$ と $(7, -10)$ を通る一次関数を求めよ。

**解法。** 定理を適用：
$$a = \frac{-10 - 5}{7 - 2} = \frac{-15}{5} = -3, \quad b = 5 - (-3)(2) = 5 + 6 = 11.$$

よって $f(x) = -3x + 11$。**検証：** $f(2) = -6 + 11 = 5$ ✓ かつ $f(7) = -21 + 11 = -10$ ✓。

<Porta nivel="5" titulo="5歳の子供向け">

階段を想像してください。各段は**同じ高さ**です。

- 各段が 20 cm なら、1段で 20 cm 上がります。
- 2段で 40 cm 上がります。
- 5段で 100 cm = 1 メートル上がります。

あなたがいる高さは**何段上がったか**で決まります。そして1段ごとに**同じ量**上がります。

これが一次関数です。高さは $f(n) = 20 \cdot n$、$n$ は段数。

もし階段が床から 50 cm 上にある（小さなステージのような）なら、$f(n) = 20n + 50$ となります。50は**スタート地点**（何も上がる前の位置）。20は**1段あたりの上昇量**。

</Porta>

<Porta nivel="10" titulo="10歳の子供向け">

映画館に行きます。入場料は**固定 R\$ 30**。ポップコーン1袋は **R\$ 8**。$n$ 袋買うなら：

$$\text{総支出} = 30 + 8n$$

これが一次関数です！ ここで：
- **30** は、ポップコーンを **0 個買っても** かかる費用（入場料）。これが式 $f(x) = ax + b$ の **b**。
- **8** は、ポップコーンを1袋増やすごとに **増える金額**。これが式の **a**。

「固定部分 + 同じペースで増える部分」のあらゆる関数で、一次関数があります。他の例：

- **Uber 料金**：初乗り（R\$ 6 固定）+ km あたり R\$ 2 ⟹ $f(d) = 6 + 2d$。
- **電気代**：基本料金（R\$ 25）+ kWh あたり R\$ 0.80 ⟹ $f(k) = 25 + 0.80\,k$。
- **電動キックボードのレンタル**：解錠 R\$ 3 + 分あたり R\$ 0.80。

グラフの直線がこの式を視覚的に**表現**します。直線が急なほど、料金の伸びが速くなります。

</Porta>

<Porta nivel="15" titulo="15歳の中高生向け">

一次関数は最も単純な「興味深い」関数で、微積分への扉を開きます。

規則は：$f(x) = ax + b$。$x$ の値を入れて、$a$ をかけて、$b$ を足すと、$f(x)$ が出てきます。それだけ。

**なぜ重要か？** グラフが**直線**で、その直線の傾きがちょうど $a$ だからです。$x$ 軸で**右に1歩**進むと、$y$ 軸で **a 歩**上がります。常に。直線上のどの点でも。この**一定性**が一次関数を「線形」と動作させるものです。

数講後、**瞬間変化率**（微分）を学びます。そして見ることになる：一次関数では、微分は一定で $a$ に等しい。他の関数（放物線、指数）では、「傾き」は $x$ で変わる — そこで微積分が登場します。しかし直線では簡単：傾きは $a$、それで終わり。

**具体例。** 一定速度 60 km/h で車を運転する。走行距離は $d(t) = 60t$（$t$ は時間）。「傾き」 $a = 60$ は速度。理にかなっている — 直線が急なほど、速く進む。

</Porta>

<Porta nivel="25" titulo="工学部の学生向け">

一次関数は、ほとんどの物理モデルが**局所的に線形化される**点です。非線形現象（空気抵抗を伴う自由落下、非ニュートン力学）でさえ、1次近似では一次関数で近似される — それが微分が形式化するものです。

### 簡約形 vs 一般形

- 簡約：$y = ax + b$
- 一般：$Ax + By + C = 0$
- パラメトリック：$\{(x_0 + t\vec{u}_x, y_0 + t\vec{u}_y) : t \in \mathbb{R}\}$

それぞれ異なる文脈で有用。CADではパラメトリック。線形最適化では一般形。統計（回帰）では簡約形。

### 直線フィッティング（線形回帰のプレビュー）

データ $(x_1, y_1), \ldots, (x_n, y_n)$ について、最もよく通る直線は？ **最小二乗法**：

$$a = \frac{n \sum x_i y_i - \sum x_i \sum y_i}{n \sum x_i^2 - (\sum x_i)^2}, \quad b = \bar{y} - a \bar{x}$$

第11学期（推測統計）で詳しく学びます。

### 応用（部分リスト）

- **フックの法則**：$F = kx$ — 伸びに対する弾性力（$b = 0$ の一次、つまり線形）
- **オームの法則**：$V = RI$ — 電流に対する電圧（線形）
- **摂氏 → 華氏変換**：$F = 1{,}8C + 32$（一次、$a = 1{,}8$、$b = 32$）
- **等速運動の運動方程式**：$s(t) = s_0 + vt$（$t$ について一次、$b = s_0$、$a = v$）

</Porta>

<Porta nivel="40" titulo="シニアプロフェッショナル向け">

実務では「**線形は局所**」。モデルを伴う工学的活動の大半は、結局のところ直線フィッティング：

- **制御工学のパラメータ推定**：正弦波励起によるプラント同定とラプラス空間での線形フィッティング。
- **テイラー線形化**：$x_0$ 周りの任意の滑らかな関数は局所的に一次 — $f(x) \approx f(x_0) + f'(x_0)(x - x_0)$。動的システムの安定性解析はすべてここに依存。
- **OLS線形回帰**：$n$ 点の二乗誤差を最小化する直線は、計量経済学者（GDP〜投資）、薬剤師（用量〜反応）、農学者（雨量〜収穫）の基礎。
- **ロジスティック回帰**：log-odds の「線形化」 — 確率比の対数は共変量の一次関数。分類するすべての AI（スパムフィルタ、不正検出）は心臓部にロジスティック回帰を持つ。

### 注意：不等分散と偽の線形性

実運用で残差をチェックせずに $y = ax + b$ をデータにフィットさせるのは災難のレシピ。回帰の前提（ガウス・マルコフ）：

1. データだけでなく、関係そのものの線形性。
2. 一定分散の残差（等分散性）。
3. 残差が無相関。
4. 残差の平均が0。

これらが破れると、$\hat a$ は推定量のままだが、効率性や一致性の特性を失います。**チェックなしの OLS はカーゴカルト**。

### 線形化 vs 変数変換

関係が一次でないとき、(a) 非線形モデルでフィット、(b) 一次になるまで**変数を変換**の二つの選択肢があります。古典的な例：

- $y = a x^b$ ⟹ $\ln y = \ln a + b \ln x$ — log-log 線形（べき乗則）
- $y = a \e^{bx}$ ⟹ $\ln y = \ln a + bx$ — 半対数線形（指数）
- $y = a/x$ ⟹ $y = a \cdot (1/x)$ — $1/x$ について一次

log-log や semi-log スケールでプロットして直線になるか確認するのが最初の診断ツール。

</Porta>

<Porta nivel="pratica" titulo="実践 — 3つの実際のシナリオ">

### シナリオ1：CEMIG 電気代（ミナスジェライス、2026年4月）

家庭用 B1 タリフ（推定）：R\$ 0.82/kWh + 黄バンド R\$ 0.02/kWh + 消費に対する ICMS 25% + 固定可用性料金 R\$ 35.40（三相住宅）。消費 $k$（kWh）について：

$$C(k) = (0{,}82 + 0{,}02) \cdot 1{,}25 \cdot k + 35{,}40 = 1{,}05 k + 35{,}40$$

**一次関数**、$a = 1.05$ R\$/kWh、$b = 35.40$ R\$。$k = 200$ kWh のとき、料金は $C(200) = 245.40$ レアル。

**限界分析**：追加 1 kWh は厳密に R\$ 1.05 かかります。R\$ 35.40 の「固定部分」は消費に依存しない — 1か月ブレーカーを切って旅行しても**支払う**ものです。

### シナリオ2：亀裂進展速度（破壊力学）

疲労試験（パリス曲線）では、金属の亀裂進展を以下のように線形化できます：

$$\log(da/dN) = \log C + m \log(\Delta K)$$

$da/dN$ はサイクルあたりの亀裂速度、$\Delta K$ は応力拡大係数の変動、$C, m$ は材料定数。log-log スケールで傾き $m$ の直線 — **パリス指数**と呼ばれます。

典型的な構造鋼で $m \approx 3$。これは：$\Delta K$ を倍にすると進展速度が $2^3 = 8$ 倍 — log-log スケールで線形になる劇的な非線形挙動。

### シナリオ3：集計需要曲線（ミクロ経済）

財の需要はしばしば一次関数 $Q(p) = \alpha - \beta p$（$\alpha, \beta > 0$、$p$ は価格）でモデル化される。需要の価格弾力性は $\epsilon = -\beta p / Q$ — 関数が一次でも曲線に沿って変動する。

数値例：$Q(p) = 1000 - 25 p$。$p = 20$ で $Q = 500$、弾力性 $|\epsilon| = 25 \cdot 20 / 500 = 1$（単位弾力的）。$p = 30$ で $Q = 250$、弾力性 $|\epsilon| = 3$（弾力的）。$p = 10$ で $Q = 750$、$|\epsilon| = 0.33$（非弾力的）。**業務上の観察**：非弾力的な財の価格を上げる企業は収益増；弾力的な財では損失。
