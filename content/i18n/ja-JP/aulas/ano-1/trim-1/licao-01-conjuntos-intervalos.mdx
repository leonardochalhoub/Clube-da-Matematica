### 基本的な数集合

<Definicao titulo="実数の階層">

- **自然数** $\mathbb{N} = \{0, 1, 2, 3, \ldots\}$。（ブラジルの慣例では0を含む。フランスの慣例では除く。）
- **整数** $\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$ — 加法、減法、乗法について閉じている。
- **有理数** $\mathbb{Q} = \{p/q \mid p \in \mathbb{Z},\ q \in \mathbb{Z}_{\neq 0}\}$ — 整数の比として表せるすべての数。
- **実数** $\mathbb{R}$ — 数直線を完備化する（$\sqrt{2}$、$\pi$、$\e$ のような「穴」を埋める）。

</Definicao>

### 集合の演算

<Definicao titulo="和、共通部分、差、補集合">

- **和集合**：$A \cup B = \{x : x \in A \text{ または } x \in B\}$
- **共通部分**：$A \cap B = \{x : x \in A \text{ かつ } x \in B\}$
- **差集合**：$A \setminus B = \{x \in A : x \notin B\}$
- **補集合**（全体集合 $U$ において）：$A^c = U \setminus A$

<Porta nivel="5" titulo="5歳の子供向け">

**おもちゃ**が入った箱があると想像してみてください。もう一つは**本**が入った箱です。

- **和集合**（∪）は、二つの箱を一つにまとめることです。おもちゃも本もあります。
- **共通部分**（∩）は、両方の箱に同時にあるものです。たいてい空っぽです — おもちゃは本ではありません。
- でも、もし一つの箱に「**赤いボール**」、もう一つに「**小さいボール**」が入っていたら、共通部分は**赤くて小さい**ボールです。

</Porta>

<Porta nivel="10" titulo="10歳の子供向け">

数学者は数を家族にまとめるのが大好きです。

- **自然数** ℕ の家族は、物を数えるときに使うものだけ：0, 1, 2, 3, ...
- **整数** ℤ の家族は負の数を加えます：..., −2, −1, 0, 1, 2, ...
- **有理数** ℚ の家族は分数を加えます：1/2, 3/4, −5/7, ...
- **実数** ℝ の家族は **π**（3.14159...）や √2（1.41421...）のような変わった数を加えます。

そして**区間**は「これとこれの間にあるすべての数」を短く表す方法です：
- `[0, 5]` は「0から5までのすべての数で、0と5を**含む**」を意味します。
- `(0, 5)` は「0から5までのすべての数で、0も5も**含まない**」を意味します。

</Porta>

<Porta nivel="15" titulo="15歳の中高生向け">

この講義は、これから3年間使う**言語**を確立するものです。微積分、線形代数、統計学などのすべての高度な数学はこの記法を使います。これなしでは、文字を知らずに物理の本を読もうとするようなものです。

**4つの中心的な数集合：**

| 記号 | 名称 | 例 |
|---|---|---|
| $\mathbb{N}$ | 自然数 | 0, 1, 2, 3, ... |
| $\mathbb{Z}$ | 整数 | ..., −3, −2, −1, 0, 1, 2, ... |
| $\mathbb{Q}$ | 有理数 | 1/2, 3/4, −7/5, 0.333... = 1/3 |
| $\mathbb{R}$ | 実数 | これら全部 + $\pi$, $\sqrt{2}$, $\e$ |

そして重要な観察：**すべての実数が有理数というわけではありません**。無理数（$\sqrt{2}$ や $\pi$ など）は整数の分数として書くことが**できない**実数です。これはピタゴラス時代（紀元前5世紀）から知られている定理であり、優雅な証明はこの講義の演習問題28で見ることになります。

**区間**は「これらの限界の間のすべての実数」の略記です：

- `[3, 7]` = $\{x \in \mathbb{R} : 3 \leq x \leq 7\}$（閉区間、境界を含む）
- `(3, 7)` = $\{x : 3 < x < 7\}$（開区間、境界なし）

先生が黒板に $x \in [3, 7]$ と書いたら、それは「xは3から7までの任意の実数で、3と7を含む」という意味です。極めてコンパクトです。これを文章で書くのと比べてみてください。

<Insight>

**なぜ $[3, 7]$ と $(3, 7)$ に違いがあるのか？** 高度な数学では、「点を含む」と「達することなく近づく」は異なる現象だからです。数か月後に**極限**を学ぶとき、$\lim_{x \to 3^+}$ は「xが3になることなく3に近づくとき何が起こるか」を問うものだとわかります。開と閉の違いはここから始まります、今、15歳のあなたとともに。

</Insight>

</Porta>

<Porta nivel="25" titulo="工学部の学生向け">

この講義は日本のMath I（第1章）+ ドイツのKlasse 10の一部に大体対応します。一見些細な内容ですが、ここでの記法の誤りは12学期すべてに伝播します。

### 重要なポイント

**1. ℕ は0を含むか？** ブラジル（BNCC）とフランスの慣例：YES。アメリカといくつかの古典的な慣例：NO。BNCCに合わせて、このプログラム全体で **0 ∈ ℕ** を採用します。外国の本を読むときは、著者の慣例を確認してください。

**2. 区間記法のブラジル式 vs アングロサクソン式：**
- BR：開区間に $]a, b[$、閉区間に $[a, b]$（フランス由来のブルバキ記法）
- アングロ：開区間に $(a, b)$、閉区間に $[a, b]$

`livros/CATALOG.md` で参照される90%のオープンブックとの互換性のため、**アングロ記法**（開区間に括弧）を採用します。

**3. ℝ における ℚ の稠密性。** 任意の異なる二つの実数 $a < b$ の間には、$a < q < b$ となる有理数 $q$ が存在します。**証明**（演習問題26）：アルキメデス性を用いる — 任意の $\eps > 0$ に対し、$1/n < \eps$ となる $n \in \mathbb{N}$ が存在する。

**4. 有限集合 vs 可算無限集合 vs 非可算集合。** カントール（1874）は ℝ が**非可算**で ℚ が**可算**であることを証明しました — どちらも無限ですが、濃度が異なります。この事実は大学の解析学（微積分IV / 実解析）で使うことになります。

### 分野横断の接続

- **微積分**：関数の定義域は常に ℝ の部分集合。
- **確率論**：標本空間は集合。
- **線形代数**：ベクトルは ℝⁿ または ℂⁿ のタプル。
- **計算機科学**：プリミティブ型 `int`, `float`, `bool` はそれぞれ ℤ（境界付き）、ℝ（有限精度）、$\{V, F\}$ に対応。

### 数値例

$|2x - 3| \leq 5$ を解き、結果を区間記法で表せ：

$$|2x - 3| \leq 5 \iff -5 \leq 2x - 3 \leq 5 \iff -2 \leq 2x \leq 8 \iff -1 \leq x \leq 4$$

答え：$x \in [-1, 4]$。

</Porta>

<Porta nivel="40" titulo="シニアプロフェッショナル向け">

集合論はZFC（ツェルメロ・フレンケル + 選択公理）という基礎理論で、現代数学全体を支えています。この講義で示す直感的な「定義」には既知の落とし穴があります：

- **ラッセルのパラドックス**：自身を含まないすべての集合の集合。それが自身を含むなら、含まない。含まないなら、含む。これがカントールの素朴集合論を破壊し（1901）、ツェルメロが内包公理を制限して理論を公理化することにつながった。

- **連続体仮説**：カントールは $|\mathbb{N}|$ と $|\mathbb{R}|$ の間に濃度がないと予想した。ゲーデル（1940）とコーエン（1963）はこの仮説がZFC公理から**独立**であることを証明 — 仮説とその否定の両方がZFCと整合的。

- **選択公理**：一見無害だが、バナッハ・タルスキー（球を分解して元と同じ二つを再構成）のような奇妙な結果を導く。ACの採用は実用的だが哲学的には繊細な決定。

実務（エンジニアリング、応用科学、金融）ではこれらは現れません。$\mathbb{R}^n$ をユークリッド空間として使い、進めるだけです。しかし、いつか解析、論理、集合論で大学院に進学するなら、これらのパラドックスに戻ってきます。

### 「実数」とは本当に何か？

計算的には、ℝ は存在しません — すべてのデジタル実装は**浮動小数点**（IEEE 754）を使い、約7桁の精度（float32）または約15桁（float64）です。浮動小数点誤差は数値計算で伝播します（REAMATによる第10学期で扱う）。Pythonで `0.1 + 0.2` と書くと `0.30000000000000004` が得られます。ℚ ⊊ ℝ は数学的抽象。コンピュータは ℚ よりはるかに小さい有限部分集合で動作します。

</Porta>

<Porta nivel="pratica" titulo="実践 — 実データの例">

### 例1：大学入試（ENEM）の年齢層

ENEM 2025の登録は、第1試験当日までに16歳になる候補者を受け付けます。記法では：

$$\text{候補者の年齢} \in [16, +\infty)$$

右に半開区間であることに注意（最大年齢の制限はない）。

### 例2：最低賃金と所得税の免除

2025年のブラジルでは、所得税完全免除：

$$\text{月収} \in [0,\ 2\,824]$$

（R\$単位の範囲。）これを超えると累進税率。

### 例3：空腹時血糖値の範囲（医学）

| 範囲 | 区間（mg/dL） | 診断 |
|---|---|---|
| 正常 | $[0, 99]$ | 健康 |
| 糖尿病前段階 | $[100, 125]$ | 注意 |
| 糖尿病 | $[126, +\infty)$ | 治療 |

すべての医学ガイドラインは、可能な値の集合を区間で**分割**して表現されます。生医学エンジニアや医療ソフトウェア開発者になるなら、これらの範囲を毎日扱うことになります。

### 例4：費用関数の定義域

$q$ 単位の製造に対する費用関数 $C(q) = 50 + 12q + 0.5q^2$。

数学的定義域：$C: \mathbb{R} \to \mathbb{R}$。
物理的（実際の）定義域：$q \in \{0, 1, 2, 3, \ldots\} = \mathbb{N}$（自動車の半分は製造できない）。

「最適点」の計算で $q^* = 17.3$ が出たら、エンジニアとして17か18に丸めるかを決めるのはあなたの仕事です — 連続的数学は ℝ の解を生み、応用は ℕ の解を要求します。この**離散 vs 連続**の緊張は、Rezende（UFF, 2003）によると微積分における5つの認識論的困難の源の一つです。
