<Porta nivel="formal" titulo="合成と逆関数">

<Definicao titulo="合成">
$f: B \to C$ と $g: A \to B$ が与えられたとき、合成 $f \circ g: A \to C$ は $(f \circ g)(x) = f(g(x))$ で定義される。
</Definicao>

<Definicao titulo="逆関数">
$f: A \to B$ が**逆関数**を持つのは $f$ が全単射のとき、かつそのときのみである。この場合、$f^{-1}: B \to A$ は次を満たす:
$$f^{-1}(f(x)) = x \quad \forall x \in A, \qquad f(f^{-1}(y)) = y \quad \forall y \in B$$
</Definicao>

<Porta nivel="5" titulo="5歳児向け">

**2つの機械**を持っているとしよう。

最初の機械は数を取って**1を足す**。3を入れると4が出る。7を入れると8が出る。

2番目の機械は数を取って**2倍にする**。4を入れると8が出る。5を入れると10が出る。

さて、2つを順番につなぐ:最初の機械から出たものが**そのまま2番目に入る**。最初の機械に3を入れる → 4が出る → 2番目に入る → 8が出る。

これが関数の**合成**:2つを順番に使うこと。

そして**逆関数**は、別の機械を取り消す機械。1を足す機械があれば、その逆は1を引く。足してから逆で引けば、元の数に戻る。

</Porta>

<Porta nivel="10" titulo="10歳児向け">

公式がある:「数を取って3倍し、それから5を足す」。数学では $f(x) = 3x + 5$。

**合成**は2つの公式を順に行うこと。例えば $g(x) = x - 2$ と $f(x) = 3x + 5$ なら、$f$ を $g$ と合成したもの $(f \circ g)(x)$ は:

1. $x$ に $g$ を適用:$x - 2$ になる。
2. その結果を $f$ に入れる:$3(x-2) + 5 = 3x - 1$。

**逆関数**は「取り消す」こと。公式が $f(x) = 2x + 3$ なら、取り消すには各演算を逆順に逆にする:3を引いて、2で割る。結果:$f^{-1}(x) = (x - 3)/2$。

**視覚的なコツ。** グラフ上で、逆関数は元のものを**対角線** $y = x$ で反射したもの。鏡で関数を見るようなもの。

**注意。** 全ての関数に逆関数があるわけではない。$f(x) = x^2$ には逆関数がない — なぜなら $f(2) = 4$ かつ $f(-2) = 4$ でもあるから。「4」だけ与えられても、機械に2を入れたか−2を入れたかわからない。十分な情報がなければ、取り消せない。

</Porta>

<Porta nivel="15" titulo="15歳児向け">

**合成**は2つの演算を順に行うこと。「数を取って2倍し、それから1を足す」: $g(x) = 2x$ と $h(y) = y + 1$ の合成 — $h(g(x)) = h(2x) = 2x + 1$ と書く。

**逆関数**は取り消すこと。$f(x) = 2x + 1$ が3を取って7を返すなら、逆関数 $f^{-1}$ は7を取って3を返す。

逆関数を求めるには:方程式で $x$ を $y$ に置き換え、$y$ について解く。

$$y = 2x + 1 \quad \to \quad x = 2y + 1 \quad \to \quad y = \frac{x - 1}{2}$$

よって $f^{-1}(x) = (x-1)/2$。確認:$f^{-1}(7) = 6/2 = 3$。✓

**注意**:全ての関数に逆関数があるわけではない。$f(x) = x^2$ にはない、なぜなら $f(2) = f(-2) = 4$ — 「元の入力」が2か−2かわからない。

</Porta>

<Porta nivel="25" titulo="工学部の学生向け">

合成は基本的 — あらゆる「複雑な」関数は「単純な」関数の合成に分解できる。この分解こそが**連鎖律**(第6学期)を計算可能にする。

### 逆関数と全単射

$f$ が逆関数を持つ $\iff$ $f$ が全単射。実用上:
- **単射でない?** 定義域を制限する。$f(x) = x^2$ は $[0, +\infty)$ で単射になり、逆関数は $\sqrt{y}$。
- **全射でない?** 終域を像に制限する。

### 応用

- **暗号**: $E$ が暗号化、$D$ が復号化。$D = E^{-1}$。逆関数なしには復号化はない。
- **制御工学**:入出力関係とプラント同定。
- **熱力学**:$P-V$、$T-S$ などの関係は、可逆性のために単射でなければならない。

### 表記

$f^{-1}$ は $1/f$ では**ない**。学生は大学2年生になっても混同する。注意。

### 演算子の単位元

恒等関数 $I(x) = x$ は合成の**単位元**:$f \circ I = I \circ f = f$。逆関数はこの演算の**逆元**:$f \circ f^{-1} = f^{-1} \circ f = I$。これにより $A$ から $A$ への全単射関数は合成のもとで**群**となる — $A$ が有限のとき**対称群** $\text{Sym}(A)$ と呼ばれる。これは大学の抽象代数で出会う最初の代数構造である。

### 合成は性質を保存する

$f$ と $g$ が連続なら $f \circ g$ は連続。両方が微分可能なら $f \circ g$ は微分可能(連鎖律)。しかし:**凸**関数の合成は必ずしも凸でない — 符号と単調性に依存する。

</Porta>

<Porta nivel="40" titulo="シニアプロフェッショナル向け">

合成は関数型プログラミングの**基本演算子**。Haskell では `(f . g) x = f (g x)`。Python(関数型)では `compose(f, g)(x) = f(g(x))`。あらゆるデータ変換は合成の連鎖 — Pandas `.pipe`、dplyr `%>%`、sklearn `Pipeline`。

### 不適切問題における逆関数

逆問題工学(トモグラフィー、地震学、逆制御)では、問題は:**$y$ が与えられたとき、$f(x) = y$ となる $x$ を見つける**。$f$ が単射でないか測定にノイズがあれば、「$f^{-1}$」は不適切定義。これに対処するのが**正則化** — Tikhonov、ridge、lasso。あらゆる統計的推論は正則化された逆問題の一形態。

### 線形変換の逆関数

$f(x) = Ax$(行列)について、$f$ が可逆 $\iff$ $A$ が可逆 $\iff \det A \neq 0$ $\iff$ すべての固有値が非零。これは第33-35回(第4学期)と第111-120回(第3学年 — 第12学期)で探求される。

### 反復関数と力学

$f^n(x) = f(f(\ldots f(x)\ldots))$($n$ 回)。軌道、不動点、引き込み域を研究する — **力学系**の基礎。マンデルブロ集合は反復の発散により定義される。MLでは、**深層ネットワーク**は合成 $f_L \circ f_{L-1} \circ \ldots \circ f_1$ — あらゆるニューラルネットワークは合成関数。

### 注意:産業規模での逆関数

$f^{-1}$ を解析的に計算することは稀。本番環境では、各 $y$ に対して $f(x) = y$ を解くのに**二分法**または**ニュートン・ラフソン法**を使う(第10学期の数値解析参照)。例えば、ブラック・ショールズのインプライド・ボラティリティ計算(ページ `/financas/.../black-scholes`)はまさに BS 公式を反転すること — 解析的でなく数値的に行われる。

</Porta>

<Porta nivel="pratica" titulo="実践 — 3つの実際のシナリオ">

### シナリオ1:データ変換パイプライン(sklearnスタイル)

センチメートル単位の身長データがあり、z-score(平均0、標準偏差1)に変換したい:

1. $g(x) = x - \bar x$(中心化)
2. $f(y) = y / \sigma$(スケーリング)

合成:$(f \circ g)(x) = (x - \bar x)/\sigma$。逆関数(予測を逆変換するため):$g^{-1}(z) = z + \bar x$、$f^{-1}(z) = z \cdot \sigma$。合成の逆:$(f \circ g)^{-1}(z) = (g^{-1} \circ f^{-1})(z) = z \cdot \sigma + \bar x$。**順序の反転に注意!**

### シナリオ2:温度変換(直列合成)

セ氏 → ケルビン変換:$K(C) = C + 273{,}15$。
ケルビン → ランキン変換:$R(K) = K \cdot 1{,}8$。
C → R 合成:$(R \circ K)(C) = (C + 273{,}15) \cdot 1{,}8 = 1{,}8 C + 491{,}67$。

ランキン → セ氏の逆:$(R \circ K)^{-1}(r) = (r - 491{,}67)/1{,}8$。

### シナリオ3:パスワードハッシュ(暗号)

銀行のパスワードは**ハッシュ**関数 $h$(SHA-256、bcrypt、argon2)を通る。$h$ は意図的に**計算可能な逆関数を持たない**ように設計されている。$h(\text{パスワード})$ を素早く計算できるが、ハッシュ $y$ が与えられて $h(\text{パスワード}) = y$ となる $\text{パスワード}$ を見つけることは計算的に不可能。この**可逆性の非対称性**があらゆる現代認証の基礎。$h$ に簡単な逆関数があれば、データベース漏洩は致命的。

</Porta>
