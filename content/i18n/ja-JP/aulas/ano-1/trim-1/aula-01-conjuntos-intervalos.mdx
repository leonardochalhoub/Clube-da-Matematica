### 基本的な数集合

<Definicao titulo="実数の階層">

- **自然数** $\mathbb{N} = \{0, 1, 2, 3, \ldots\}$。(ブラジルの慣例ではゼロを含む;フランスの慣例では除外。)
- **整数** $\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$ — 加法、減法、乗法に関して閉じている。
- **有理数** $\mathbb{Q} = \{p/q \mid p \in \mathbb{Z},\ q \in \mathbb{Z}_{\neq 0}\}$ — 整数の比として表現可能なすべての数。
- **実数** $\mathbb{R}$ — 軸を完成させる($\sqrt{2}$、$\pi$、$\e$ のような「穴」を埋める)。

</Definicao>

### 集合演算

<Definicao titulo="和、積、差、補">

- **和集合**:$A \cup B = \{x : x \in A \text{ または } x \in B\}$
- **共通部分**:$A \cap B = \{x : x \in A \text{ かつ } x \in B\}$
- **差集合**:$A \setminus B = \{x \in A : x \notin B\}$
- **補集合**(全体集合 $U$ における):$A^c = U \setminus A$

**おもちゃ**の入った箱があると想像してください。**本**の入った別の箱があります。

- **和**(∪)は両方の箱を一つにまとめることです。おもちゃと本があります。
- **積**(∩)は両方の箱に同時にあるものです。普通は空っぽ — おもちゃは本ではありません。
- でも、一つの箱に「**赤いボール**」、もう一つに「**小さなボール**」があるなら、積は**赤くて小さい**ボールです。

</Porta>

<Porta nivel="10" titulo="10歳の子供のために">

数学者は数を家族に整理するのが大好きです。

- **自然数** ℕ の家族は、物を数えるときに使うものだけです:0, 1, 2, 3, ...
- **整数** ℤ の家族は負の数を加えます:..., −2, −1, 0, 1, 2, ...
- **有理数** ℚ の家族は分数を加えます:1/2, 3/4, −5/7, ...
- **実数** ℝ の家族は **π**(3.14159...)や √2(1.41421...)のような奇妙な数を加えます。

そして**区間**は「これとあれの間のすべての数」を短く言う方法です:
- `[0, 5]` は「0 と 5 を**含めて**、0 から 5 までのすべての数」を意味します。
- `(0, 5)` は「0 と 5 を**除いて**、0 から 5 までのすべての数」を意味します。

</Porta>

<Porta nivel="15" titulo="15歳の青少年のために">

これは今後3年間で使う**言語**を確立する授業です。すべての高度な数学 — 微積分、線形代数、統計学 — がこの記法を使います。これなしでは、文字を知らずに物理学の本を読もうとするようなものです。

**4つの中心的な数集合:**

| 記号 | 名前 | 例 |
|---|---|---|
| $\mathbb{N}$ | 自然数 | 0, 1, 2, 3, ... |
| $\mathbb{Z}$ | 整数 | ..., −3, −2, −1, 0, 1, 2, ... |
| $\mathbb{Q}$ | 有理数 | 1/2, 3/4, −7/5, 0.333... = 1/3 |
| $\mathbb{R}$ | 実数 | これらすべて + $\pi$, $\sqrt{2}$, $\e$ |

そして重要な観察:**すべての実数が有理数とは限らない**。無理数($\sqrt{2}$ や $\pi$ のような)は整数の分数として書くことが**できない**実数です。これはピタゴラス(紀元前5世紀)以来知られている定理です — この授業の演習28で見るエレガントな証明です。

**区間**は「ある限界の間のすべての実数」の略記です:

- `[3, 7]` = $\{x \in \mathbb{R} : 3 \leq x \leq 7\}$(閉区間、境界を含む)
- `(3, 7)` = $\{x : 3 < x < 7\}$(開区間、境界を含まない)

先生が黒板に $x \in [3, 7]$ と書くとき、彼は「x は 3 と 7 を含む 3 から 7 までの任意の実数」と言っています。非常にコンパクト。これを言葉で書き出してみてください。

<Insight>

**なぜ $[3, 7]$ と $(3, 7)$ の間に違いがあるのか?** なぜなら高度な数学では、「点を含める」ことと「到達せずに近づく」ことは異なる現象だからです。数ヶ月後に**極限**を学ぶとき、$\lim_{x \to 3^+}$ は *x が 3 になることなく 3 に近づくとき* 何が起こるかを尋ねていることがわかります。開と閉の違いは、ここで、今、15歳のあなたから始まります。

</Insight>

</Porta>

<Porta nivel="25" titulo="工学部の学生のために">

この授業はおおよそ日本の数学I(第1章)+ ドイツの Klasse 10 の一部に対応します。一見些細な内容ですが — ここでの記法の誤りは12学期すべてに伝播します。

### 重要な点

**1. ℕ は 0 を含むか?** ブラジル(BNCC)とフランスの慣例:はい。アメリカの慣例といくつかの古典:いいえ。BNCC に沿って、このプログラム全体を通じて **0 ∈ ℕ** を採用します。外国の本を読むときは、著者の慣例を確認してください。

**2. ブラジル vs. アングロサクソン式区間記法:**
- BR:開は $]a, b[$、閉は $[a, b]$(フランスから受け継いだブルバキ記法)
- アングロ:開は $(a, b)$、閉は $[a, b]$

`livros/CATALOG.md` で参照されているオープンブックの 90% との互換性のため、**アングロ記法**(開には丸括弧)を採用します。

**3. ℝ における ℚ の稠密性。** 任意の異なる2つの実数 $a < b$ の間には、$a < q < b$ を満たす有理数 $q$ が存在します。**証明**(演習26):アルキメデスの性質を使う — 任意の $\eps > 0$ に対して、$1/n < \eps$ となる $n \in \mathbb{N}$ が存在する。

**4. 有限集合 vs. 可算無限集合 vs. 非可算集合。** カントール(1874)は ℝ が**非可算**であるが ℚ は**可算**であることを証明した — 両者とも無限であるが、異なる濃度を持つ。この事実は大学の解析学(微積分 IV / 実解析)で使うことになる。

### 領域横断的な接続

- **微積分**:関数の定義域は常に ℝ の部分集合。
- **確率**:標本空間は集合。
- **線形代数**:ベクトルは ℝⁿ または ℂⁿ のタプル。
- **計算機**:プリミティブ型 `int`、`float`、`bool` はそれぞれ ℤ(境界あり)、ℝ(有限精度)、$\{T, F\}$ に対応する。

### 数値例

$|2x - 3| \leq 5$ を解き、結果を区間記法で表す:

$$|2x - 3| \leq 5 \iff -5 \leq 2x - 3 \leq 5 \iff -2 \leq 2x \leq 8 \iff -1 \leq x \leq 4$$

答え:$x \in [-1, 4]$。

</Porta>

<Porta nivel="40" titulo="シニアプロフェッショナルのために">

集合論はすべての現代数学を支える ZFC 基盤(ツェルメロ・フレンケル + 選択公理)です。この授業で示した直感的な「定義」には既知の落とし穴があります:

- **ラッセルのパラドックス**:自分自身を含まないすべての集合の集合。それが自分自身を含むなら、含まない。含まないなら、含む。これがカントールの素朴理論を破り(1901)、ツェルメロが分出公理によって理論を公理化することにつながった。

- **連続体仮説**:カントールは $|\mathbb{N}|$ と $|\mathbb{R}|$ の間に濃度が存在しないと予想した。ゲーデル(1940)とコーエン(1963)はこの仮説が ZFC 公理から**独立**であることを証明した — 仮説とその否定の両方が ZFC と整合する。

- **選択公理**:一見無害だが、バナッハ・タルスキー(球を分解して原球と等しい2つの球を組み立てる)のような奇妙な結果を含意する。AC を採用する決定は実用的だが哲学的に微妙。

専門的な実践(工学、応用科学、金融)では、これらは現れません。$\mathbb{R}^n$ をユークリッド空間として使い、進みます。しかし、いつか解析、論理、または集合論の大学院に進むなら、これらのパラドックスに戻ることになります。

### 「実数」とは本当に何か?

計算機上では ℝ は存在しません — すべてのデジタル実装は**浮動小数点**(IEEE 754)を使い、約7桁の精度(float32)または約15桁(float64)を持ちます。浮動小数点エラーは数値計算で伝播します(REAMAT で第10学期に扱う)。Python で `0.1 + 0.2` と書くと `0.30000000000000004` が得られます。ℚ ⊊ ℝ は数学的抽象;コンピューターは ℚ よりはるかに小さい有限部分集合で動作します。

</Porta>

<Porta nivel="pratica" titulo="練習 — 実データの例">

### 例1:ENEM の年齢範囲

ENEM 2025 の出願は、最初の試験日までに満16歳の候補者を受け入れます。記法では:

$$\text{候補者の年齢} \in [16, +\infty)$$

右半開区間に注意(上限なし)。

### 例2:最低賃金と所得税免除

2025年のブラジルでは、所得の IR 完全免除:

$$\text{月額所得} \in [0,\ 2\,824]$$

(R\$ での範囲。)これを超えると、累進税率。

### 例3:空腹時血糖値の範囲(医学)

| 範囲 | 区間(mg/dL) | 診断 |
|---|---|---|
| 正常 | $[0, 99]$ | 健康 |
| 糖尿病前症 | $[100, 125]$ | 注意 |
| 糖尿病 | $[126, +\infty)$ | 治療 |

すべての医療ガイドラインは、可能な値の集合の区間への**分割**として表現されます。生体医工学エンジニアや医療ソフトウェア開発者になるなら、これらの範囲を毎日扱うことになります。

### 例4:コスト関数の定義域

$q$ 単位の製造のコスト関数 $C(q) = 50 + 12q + 0.5q^2$。

数学的定義域:$C: \mathbb{R} \to \mathbb{R}$。
物理的(実)定義域:$q \in \{0, 1, 2, 3, \ldots\} = \mathbb{N}$(車を半分作ることはない)。

「最適点」の計算が $q^* = 17.3$ を出す場合、エンジニアとしてのあなたの仕事は17に丸めるか18に丸めるかを決めることです — 連続数学は ℝ で解を生成します;応用は ℕ で解を要求します。この**離散 vs. 連続**の緊張は、Rezende(UFF, 2003)による微積分の認識論的困難の5つの源の1つです。
