<Porta nivel="formal" titulo="ℝ²における解析幾何">

<Definicao titulo="デカルト平面">
$\mathbb{R}^2 = \{(x, y) : x, y \in \mathbb{R}\}$。各点は順序対です。$x$軸(水平)と$y$軸(垂直)は$O = (0, 0)$で交わり、平面を4つの象限に分けます。
</Definicao>

### 距離

$P = (x_1, y_1)$、$Q = (x_2, y_2)$ について:
$$d(P, Q) = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

### 中点

$$M = \left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}\right)$$

### 比$k$による線分の分割

$\overline{PQ}$ を比 $PR/RQ = k$ で分割するには:
$$R = \left(\frac{x_1 + k x_2}{1 + k}, \frac{y_1 + k y_2}{1 + k}\right)$$

### 面積の計算

頂点 $A = (x_1, y_1)$、$B = (x_2, y_2)$、$C = (x_3, y_3)$ の三角形の面積:
$$\text{面積} = \frac{1}{2} |x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2)|$$

これは $\frac{1}{2} |\det M|$ と同等で、$M$ はベクトル $\vec{AB}, \vec{AC}$ の行列です。

</Porta>

<Porta nivel="15" titulo="15歳の青少年へ">

デカルト平面:各点は2つの数で記述されます — $x$座標(水平)と$y$座標(垂直)。

**2点間の距離**:ピタゴラスを適用します。$P = (1, 2)$ と $Q = (4, 6)$ がある場合:

$$d = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

**中点**:座標の平均を取ります。$(1, 2)$ と $(4, 6)$ の中点は $(2{,}5,\ 4)$ です。

**なぜ重要なのか?** デカルト平面は**代数と幾何をつなぐ**言語だからです。直線が方程式になります。円が方程式になります。放物線が方程式になります。「幾何的」だったものすべてが「代数的」になり — コンピュータが計算できるようになります。

</Porta>

<Porta nivel="25" titulo="工学部の学生へ">

解析幾何はすべての基礎です:

- **コンピュータグラフィックス**:各ピクセルは点であり、各2D/3Dオブジェクトは座標の集合。
- **GPS**:座標+三角測量による位置決定。
- **ロボット工学**:運動学はロボットアームの軌道を記述。
- **土木工学**:地形測量は座標を生成。

### $\mathbb{R}^n$ における距離

任意の次元に一般化:
$$d(P, Q) = \sqrt{\sum_{i=1}^n (x_i - y_i)^2} = \|P - Q\|_2$$

これは**ユークリッドノルム**です。MLでは、k-NN、k-meansなどで使用される「標準距離」です。

### その他のメトリック

- **マンハッタン** ($\ell_1$):$\sum |x_i - y_i|$ — 街区での距離。
- **チェビシェフ** ($\ell_\infty$):$\max |x_i - y_i|$ — チェスにおけるキングの動き。
- **測地メトリック**:球面(地球)上では、距離は大円に従う。

</Porta>
