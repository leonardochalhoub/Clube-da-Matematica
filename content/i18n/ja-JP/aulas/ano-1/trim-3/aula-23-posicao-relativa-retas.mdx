<Porta nivel="formal" titulo="判定基準と公式">

### 平行性と垂直性

直線 $r: y = m_r x + n_r$ と $s: y = m_s x + n_s$ に対して:

- **平行 (一致しない)**: $m_r = m_s$ かつ $n_r \neq n_s$。
- **一致**: $m_r = m_s$ かつ $n_r = n_s$。
- **交わる**: $m_r \neq m_s$。
- **垂直**: $m_r \cdot m_s = -1$ (どちらも垂直線でないとする)。

垂直線を含む場合: $x = a$ は垂直、$y = b$ は水平。垂直線同士は平行; 垂直と水平は垂直。

### 直線間の角度

$$\tan\theta = \left|\frac{m_r - m_s}{1 + m_r m_s}\right|$$

(どちらも垂直線でないとする。)

### 点と直線の距離

点 $P_0 = (x_0, y_0)$ と直線 $Ax + By + C = 0$:

$$d(P_0, r) = \frac{|A x_0 + B y_0 + C|}{\sqrt{A^2 + B^2}}$$

### 平行な直線間の距離

$r: Ax + By + C_1 = 0$ と $s: Ax + By + C_2 = 0$ に対して:
$$d(r, s) = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$

</Porta>

<Porta nivel="15" titulo="15歳のティーンエイジャーに">

平面上の2直線は次のいずれかの関係にあります:

1. **平行** (同じ傾き、決して交わらない)。
2. **垂直** (90°の角度を成す)。
3. **1点で交わる** (交わる)。
4. **同じ直線** (一致)。

**平行の判定基準**: $m_r = m_s$ — 同じ傾き。

**垂直の判定基準**: $m_r \cdot m_s = -1$。奇妙だが、これは2つの変位が成す三角形にピタゴラスの定理を適用することから導かれる。

**例**: $y = 2x + 1$ と $y = -x/2 + 5$ は垂直 ($2 \cdot (-1/2) = -1$)。

</Porta>

<Porta nivel="25" titulo="工学部生に">

実用上、**2x2線形系の解法**は直線の交点を求めることである。クラメル法と消去法はここから始まる。

### 2x2 系と幾何学的解釈

$$\begin{cases} a_1 x + b_1 y = c_1 \\ a_2 x + b_2 y = c_2 \end{cases}$$

- **唯一解**: 交わる直線。$\det \neq 0$。
- **無数の解**: 一致する直線。行列式 $= 0$ かつ 系が無矛盾。
- **解なし**: 異なる平行な直線。行列式 $= 0$ かつ 系が矛盾。

</Porta>
