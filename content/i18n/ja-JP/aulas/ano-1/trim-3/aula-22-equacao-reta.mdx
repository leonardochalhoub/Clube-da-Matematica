<Porta nivel="formal" titulo="直線の方程式の形式">

### 傾き切片形

$y = mx + n$、ここで $m$ は **傾き** 、$n$ は **$y$切片** です。

### 一般形

$Ax + By + C = 0$、ただし $(A, B) \neq (0, 0)$。垂直でない直線 ($B \neq 0$) では: $m = -A/B$、$n = -C/B$。

### 点-傾き形

$(x_0, y_0)$ を通り傾き $m$ の直線:
$$y - y_0 = m(x - x_0)$$

### 媒介変数形

方向ベクトル $\vec u = (a, b)$ を持ち $(x_0, y_0)$ を通る直線:
$$\begin{cases} x = x_0 + at \\ y = y_0 + bt \end{cases}, \quad t \in \mathbb{R}$$

### 2点形

$(x_1, y_1)$ と $(x_2, y_2)$ を通る直線:
$$\frac{y - y_1}{x - x_1} = \frac{y_2 - y_1}{x_2 - x_1}$$

</Porta>

<Porta nivel="15" titulo="15歳のティーンエイジャーに">

直線の方程式は代数と幾何をつなぐものです。平面上の各直線には、その**すべての点**を記述する方程式 $y = mx + n$ があります。

- $m$ は **傾き** (直線がどれだけ「上昇」または「下降」するか)。
- $n$ は直線が $y$ 軸と交わる場所。

**2点** $(x_1, y_1)$ と $(x_2, y_2)$ があるとき、傾きを次のように計算します:

$$m = \frac{y_2 - y_1}{x_2 - x_1}$$

そしていずれかの点を代入して $n$ を求めます。

**一般形** $Ax + By + C = 0$ はより「対称的」で、$x$ と $y$ を等しく扱います。直線が垂直 ($x = $ 定数) の場合に役立ちます。これは傾き切片形では表現できません。

</Porta>

<Porta nivel="25" titulo="工学部生に">

直線の方程式は以下の基礎です:

- **線形計画法**: $Ax \leq b$ の条件下で $cx$ を最大化 — 線形制約 (直線で囲まれた半空間)。
- **単純線形回帰**: 最小二乗法で $\hat y = \hat m x + \hat n$ を当てはめ。
- **運動学**: 直線運動における位置は $s(t) = s_0 + vt$。
- **コンピュータグラフィックス**: 直線のラスタライズ (ブレゼンハムのアルゴリズム)。

### 点と直線の距離

直線 $Ax + By + C = 0$ と点 $P_0 = (x_0, y_0)$ について:
$$d(P_0, \text{reta}) = \frac{|A x_0 + B y_0 + C|}{\sqrt{A^2 + B^2}}$$

</Porta>
