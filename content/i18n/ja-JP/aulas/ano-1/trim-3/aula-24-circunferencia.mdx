<Porta nivel="formal" titulo="方程式と性質">

<Definicao titulo="円">
中心 $C = (a, b)$ から距離 $r > 0$ にある点 $(x, y) \in \mathbb{R}^2$ の集合。
</Definicao>

### 標準形

$$(x - a)^2 + (y - b)^2 = r^2$$

### 一般形

展開すると：$x^2 + y^2 + Dx + Ey + F = 0$、ここで $D = -2a$、$E = -2b$、$F = a^2 + b^2 - r^2$。

中心の復元：$(a, b) = (-D/2, -E/2)$。半径：$r = \sqrt{D^2/4 + E^2/4 - F}$（正なら有効、そうでなければ実円ではない）。

### 直線と円

直線 $r$ は：
- 円と**交わらない**：$d(\text{中心}, r) > r$。
- **接する**（$d = r$）：ちょうど 1 点。
- **割線**（$d < r$）：2 つの交点。

### 接線と半径の性質

$P$ における接線は半径 $\overline{CP}$ に垂直。

</Porta>

<Porta nivel="15" titulo="15歳のために">

円は中心から**等距離**にある点の集合です。距離 $= r$（半径）。

第21課の距離公式を適用：
$$\sqrt{(x - a)^2 + (y - b)^2} = r$$

二乗して：$(x - a)^2 + (y - b)^2 = r^2$。**これだけです**。

中心が原点にあるとき（$a = b = 0$）：$x^2 + y^2 = r^2$。**単位円**（$r = 1$）：$x^2 + y^2 = 1$ — 第12課の三角円で使ったものです！

</Porta>

<Porta nivel="25" titulo="工学部の学生のために">

円は**幾何学的軌跡**（locus）の最も単純な例です。直接の一般化：$\mathbb{R}^3$ における**球面** — $(x-a)^2 + (y-b)^2 + (z-c)^2 = r^2$。

応用：
- **通信**：アンテナの到達範囲 = 円。複数カバレッジ = 円の和集合。
- **GPS**：各衛星が球面を定義する；あなたの位置は 4 つの球面の交点。
- **ロボティクス**：球面関節を持つアームの到達範囲は球面。

</Porta>
