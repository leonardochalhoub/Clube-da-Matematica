<Definicao titulo="二次函数">

<Equation numero="1" explicacao="著名的求解公式。Δ = b² − 4ac 是判别式。Δ > 0：两个不同实根。Δ = 0：一个重根。Δ < 0：无实根（两个共轭复根）。">

<Equation numero="2" explicacao="顶点的 x 坐标 — 临界点（a < 0 时为最大值，a > 0 时为最小值）。可由配方法或根的平均证明。">

<Porta nivel="5" titulo="给5岁孩子">

拿一个球向高处扔。它升啊升啊升，然后落啊落啊落。

如果你画出球走过的路，那是一条曲线。这个图有名字：**抛物线**。

每次你把东西扔向空中 — 球、石头、火箭 — 它的路径都画出一条抛物线。

数学家发现了一个用字母和数字描述这条路径的公式：$y = ax^2 + bx + c$。现在不用记 — 只要知道**每条抛物线都有这样的公式**就好。

</Porta>

<Porta nivel="10" titulo="给10岁孩子">

抛物线是球在空中画出的曲线。但它也是这些东西的形状：

- **抛物面天线**（卫星电视的那个抛物面天线）
- **汽车前灯**（抛物面反射镜）
- **拱桥结构**
- **烟花的轨迹**

为什么这条曲线这么常见？因为它有个特别的性质：如果一束光线平行于抛物线的轴射入，它击中表面后**反射到唯一一点**，叫做**焦点**。所以抛物面天线能把信号集中到一个小接收器上 — 所有进来的信号变成一个。

公式是 $y = ax^2 + bx + c$。$a$ 控制抛物线"张开"的程度：$a$ 大时变窄，$a$ 小时变宽。$a$ 的符号控制方向：$a > 0$ 朝上（笑脸），$a < 0$ 朝下（哭脸）。

</Porta>

<Porta nivel="15" titulo="给15岁青少年">

二次函数 $f(x) = ax^2 + bx + c$ 是我们正式学习的第一个"曲线"函数。图象是**抛物线** — $a > 0$ 时开口向上，$a < 0$ 时向下。

**要知道的三件事：**

1. **根**：抛物线穿过 x 轴的位置。求根公式：
   $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

2. **顶点**：最高点（$a < 0$ 时）或最低点（$a > 0$ 时）。x 坐标：$x_V = -b/(2a)$。

3. **凹凸性**：$a > 0$ 是"微笑"（cup），$a < 0$ 是"皱眉"（cap）。

抛出球的轨迹是抛物线。抛物面天线是这个形状。汽车前灯也是。**为什么抛物线在那么多地方出现？** 因为它们有一个几何性质：每条与轴平行的光线都反射到焦点。（或反过来：从焦点出发的光线平行射出。）你将在微积分3中看到。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

二次函数**远不止**预备微积分：

- 常系数二阶线性 ODE（Trim 10）化归为二次**特征**方程 $\lambda^2 + b\lambda + c = 0$。
- 二次函数的无约束优化是平凡的**凸规划** — 回归的基础。
- 求根公式可推广到卡尔达诺（三次）和费拉里（四次）。伽罗瓦（1832）证明了次数 ≥ 5 的一般方程没有根式解。
- 判别式 $\Delta = b^2 - 4ac$ 决定解是实数还是复数。出现在振动力学中：$\Delta > 0$ 是过阻尼（无振荡），$\Delta = 0$ 是临界，$\Delta < 0$ 是欠阻尼（振荡）。

### 求根公式的证明（配方法）

$ax^2 + bx + c = 0 \Rightarrow x^2 + \frac{b}{a}x = -\frac{c}{a}$

两边加 $\left(\frac{b}{2a}\right)^2$：

$\left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a^2}$

开方：$x + \frac{b}{2a} = \pm\frac{\sqrt{b^2 - 4ac}}{2a}$

所以 $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$。∎

### 根的和与积（韦达定理）

若 $x_1, x_2$ 是 $ax^2 + bx + c = 0$ 的根，则：

$$x_1 + x_2 = -\frac{b}{a}, \quad x_1 \cdot x_2 = \frac{c}{a}$$

（在不需单独求出根，而只需其和或积的问题中很有用。）

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

二次函数在工业规模上重新出现。列出实践中重要的内容：

### 二次规划（QP）

形式为 $\min \frac{1}{2} x^T Q x + c^T x$ 受 $Ax \leq b$ 约束的优化。当 $Q$ 半正定时，是凸的且解唯一（KKT）。应用：

- **马科维茨投资组合**：$\min \mathbf{w}^T \Sigma \mathbf{w}$ — 给定预期收益的最小方差组合。
- **约束最小二乘**：$\sum w_i = 1$，$w_i \geq 0$ 下的回归。
- **线性 SVM**：最大间隔分类器是 $\mathbb{R}^n$ 上的 QP。
- **MPC**（模型预测控制）：每个控制步求解一个 QP。

标准求解器：OSQP、Gurobi、CPLEX。最坏情况计算开销 $O(n^3)$（内点法）；现代硬件可接受 $n \sim 10^4$。

### 非线性优化中的抛物线近似

优化中的牛顿-拉夫森：每一步将目标函数用二次近似（泰勒二阶截断）并最小化该二次函数。**每次牛顿迭代就是求解一个局部二次函数。** 这就是为什么掌握求根公式是高级优化的基础。

### 抛物线曲线拟合

实验室中，将 $y = ax^2 + bx + c$ 拟合到 $n$ 个点是变量 $\{1, x, x^2\}$ 上的 OLS。系数由正规方程 $X^T X \beta = X^T y$ 得出，$X$ 是设计矩阵。标准残差：$\hat\sigma^2 = \text{SSR}/(n-3)$。**注意**：在采样区间外推会灾难性失败 — 外推的二次函数会迅速冲向 $\pm\infty$。

### 微分方程中的二次

ODE $y'' + b y' + c y = 0$ 的特征方程为 $\lambda^2 + b\lambda + c = 0$。根决定解的形式：

| $\Delta$ | 行为 | 应用 |
|---|---|---|
| $\Delta > 0$ | 过阻尼（无振荡） | 汽车悬挂 |
| $\Delta = 0$ | 临界阻尼 | 自动门（无碰撞返回） |
| $\Delta < 0$ | 欠阻尼（振荡） | 摆、RLC 电路 |

机械振动和电气振荡的所有分析都经过这里 — Trim 10 详细说明。

</Porta>

<Porta nivel="pratica" titulo="实践 — 3个真实场景">

### 场景 1：弹道轨迹（弹丸发射）

弹丸以初速度 $v_0 = 25$ m/s 在地面以 $\theta = 45°$ 角发射。分量：

$v_{0x} = v_0 \cos\theta \approx 17.68$ m/s，$v_{0y} = v_0 \sin\theta \approx 17.68$ m/s。

高度作为时间的函数：$h(t) = v_{0y} t - \tfrac{1}{2} g t^2 = 17.68 t - 4.9 t^2$。

**分析。** 抛物线的顶点：$t_V = -b/(2a) = 17.68/9.8 \approx 1.80$ s，$h_V = h(t_V) \approx 15.96$ m。落地：$h(t) = 0 \Rightarrow t = 0$ 或 $t \approx 3.61$ s。水平射程：$x = v_{0x} \cdot 3.61 \approx 63.8$ m。

### 场景 2：无盖矩形盒（经典优化）

边长 30 cm 的正方形纸板。在角落剪去边长 $x$ 的正方形并折成盒子。体积：

$$V(x) = x \cdot (30 - 2x)^2 = 4x^3 - 120 x^2 + 900 x$$

不是二次的 — 是三次的！但其**导数**是二次的：$V'(x) = 12 x^2 - 240 x + 900 = 0$。由求根公式：$x = (240 \pm \sqrt{57600 - 43200})/24 = (240 \pm 120)/24$。所以 $x = 5$（有效最大值）或 $x = 15$（退化，放不下）。最大体积：$V(5) = 5 \cdot 400 = 2{,}000$ cm³。

### 场景 3：垄断企业利润（微观经济学）

垄断企业面临需求 $Q(p) = 1000 - 50 p$ 和单位成本 $c = 5$。利润：

$$L(p) = (p - 5) \cdot Q(p) = (p - 5)(1000 - 50p) = -50 p^2 + 1250 p - 5000$$

关于 $p$ 的二次函数，$a = -50$（开口向下）。顶点：$p^* = -1250/(2 \cdot (-50)) = 12.5$。最大利润：$L(12.5) = 50 \cdot 12.5^2 \cdot ... = 2{,}812.50$ 货币单位。**操作答案**：定价 12.50 即使损失一半客户（卖出 375 单位 vs. 潜在 750）也能最大化利润。

</Porta>
