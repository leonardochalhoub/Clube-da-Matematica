### 基本数集

<Definicao titulo="实数的层次结构">

- **自然数** $\mathbb{N} = \{0, 1, 2, 3, \ldots\}$。(巴西约定包括零;法国约定不包括。)
- **整数** $\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$ — 在加法、减法和乘法下封闭。
- **有理数** $\mathbb{Q} = \{p/q \mid p \in \mathbb{Z},\ q \in \mathbb{Z}_{\neq 0}\}$ — 每一个可表示为整数比的数。
- **实数** $\mathbb{R}$ — 完成数轴(填补"空洞"如 $\sqrt{2}$、$\pi$、$\e$)。

</Definicao>

### 集合运算

<Definicao titulo="并、交、差、补">

- **并集**:$A \cup B = \{x : x \in A \text{ 或 } x \in B\}$
- **交集**:$A \cap B = \{x : x \in A \text{ 且 } x \in B\}$
- **差集**:$A \setminus B = \{x \in A : x \notin B\}$
- **补集**(在全集 $U$ 中):$A^c = U \setminus A$

对于任意集合 $A, B$:

<Equation numero="DM" explicacao="并集的补集是补集的交集。反之亦然。">

</Equation>

<Porta nivel="5" titulo="给5岁的孩子">

想象你有一个装着**玩具**的盒子。另一个装着**书**的盒子。

- **并集**(∪)就是把两个盒子合并成一个。里面有玩具和书。
- **交集**(∩)就是同时在两个盒子里的东西。通常是空的——玩具不是书。
- 但如果一个盒子有"**红色珠子**",另一个有"**小珠子**",交集就是**既红又小**的珠子。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

数学家喜欢把数字组织成家族。

- **自然数** ℕ 家族就是你用来数东西的:0, 1, 2, 3, ...
- **整数** ℤ 家族增加了负数:..., −2, −1, 0, 1, 2, ...
- **有理数** ℚ 家族增加了分数:1/2, 3/4, −5/7, ...
- **实数** ℝ 家族增加了奇怪的数,如 **π**(3.14159...)和 √2(1.41421...)。

而**区间**是简短地说"某某之间所有的数"的方式:
- `[0, 5]` 表示"从0到5的每一个数,**包括** 0 和 5"。
- `(0, 5)` 表示"从0到5的每一个数,**不包括** 0 和 5"。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

这是建立**语言**的课,我们将在接下来的3年里使用这种语言。所有高级数学——微积分、线性代数、统计学——都使用这种记号。没有它,就像试图在不知道字母的情况下读物理书。

**4个核心数集:**

| 符号 | 名称 | 例子 |
|---|---|---|
| $\mathbb{N}$ | 自然数 | 0, 1, 2, 3, ... |
| $\mathbb{Z}$ | 整数 | ..., −3, −2, −1, 0, 1, 2, ... |
| $\mathbb{Q}$ | 有理数 | 1/2, 3/4, −7/5, 0.333... = 1/3 |
| $\mathbb{R}$ | 实数 | 所有这些 + $\pi$, $\sqrt{2}$, $\e$ |

还有一个重要的观察:**并非每个实数都是有理数**。无理数(如 $\sqrt{2}$ 和 $\pi$)是不能写成整数分数的实数。这是自毕达哥拉斯(公元前5世纪)以来已知的定理——你将在本课的练习28中看到一个优雅的证明。

**区间**是"某些限制之间所有实数"的缩写:

- `[3, 7]` = $\{x \in \mathbb{R} : 3 \leq x \leq 7\}$(闭区间,包括边界)
- `(3, 7)` = $\{x : 3 < x < 7\}$(开区间,不包括边界)

当老师在黑板上写 $x \in [3, 7]$ 时,他在说:"x 是 3 到 7 之间的任何实数,包括 3 和 7"。非常紧凑。和写出来比较一下。

<Insight>

**为什么 $[3, 7]$ 和 $(3, 7)$ 之间存在差异?** 因为在高级数学中,"包括点"和"接近它但不到达"是不同的现象。当你几个月后学习**极限**时,你会看到 $\lim_{x \to 3^+}$ 问的是 *当 x 接近 3 但不是 3 时*会发生什么。开和闭之间的差异从这里开始,现在,在你15岁的时候。

</Insight>

</Porta>

<Porta nivel="25" titulo="给工程学生">

这一课大致对应日本 Math I(第1章)+ 德国 Klasse 10 的一部分。看似平凡的内容——但这里的记号错误会传播到所有12个学期。

### 关键点

**1. ℕ 包括 0 吗?** 巴西约定(BNCC)和法国:是。美国约定和几个经典:否。我在整个课程中采用 **0 ∈ ℕ**,与 BNCC 一致。读外文书时,请检查作者的约定。

**2. 巴西 vs. 英美区间记号:**
- BR:$]a, b[$ 表示开,$[a, b]$ 表示闭(从法国继承的 Bourbaki 记号)
- 英美:$(a, b)$ 表示开,$[a, b]$ 表示闭

我采用**英美记号**(圆括号表示开),以与 `livros/CATALOG.md` 中引用的 90% 开放书籍兼容。

**3. ℚ 在 ℝ 中的稠密性。** 在任意两个不同的实数 $a < b$ 之间,存在有理数 $q$ 使 $a < q < b$。**证明**(练习26):使用阿基米德性质 — 对于每个 $\eps > 0$,存在 $n \in \mathbb{N}$ 使 $1/n < \eps$。

**4. 有限 vs. 可数无限 vs. 不可数集合。** 康托尔(1874)证明 ℝ 是**不可数的**,但 ℚ 是**可数的**——两者都是无限的,但有不同的基数。你将在大学分析(微积分 IV / 实分析)中使用这个事实。

### 跨领域联系

- **微积分**:函数的定义域始终是 ℝ 的子集。
- **概率**:样本空间是一个集合。
- **线性代数**:向量是 ℝⁿ 或 ℂⁿ 中的元组。
- **计算**:基本类型 `int`、`float`、`bool` 对应于 ℤ(有界)、ℝ(有限精度)、$\{真, 假\}$。

### 数值示例

求解 $|2x - 3| \leq 5$ 并以区间记号表示结果:

$$|2x - 3| \leq 5 \iff -5 \leq 2x - 3 \leq 5 \iff -2 \leq 2x \leq 8 \iff -1 \leq x \leq 4$$

答案:$x \in [-1, 4]$。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

集合论是支撑所有现代数学的 ZFC 基础(策梅洛-弗兰克尔 + 选择公理)。我们在本课中提出的直观"定义"有已知的陷阱:

- **罗素悖论**:所有不包含自身的集合的集合。如果它包含自身,则它不包含。如果它不包含,则它包含。这打破了康托尔的朴素理论(1901),并使策梅洛用受限分离公理化该理论。

- **连续统假设**:康托尔猜想在 $|\mathbb{N}|$ 和 $|\mathbb{R}|$ 之间不存在基数。哥德尔(1940)和科恩(1963)证明该假设**独立**于 ZFC 公理 — 它和它的否定都与 ZFC 一致。

- **选择公理**:看似无害,但暗示奇怪的结果如巴拿赫-塔斯基(分解一个球并重新组装两个与原球相等的球)。采用 AC 的决定是实用的,但在哲学上微妙。

在专业实践中(工程、应用科学、金融),这些都不会出现。你使用 $\mathbb{R}^n$ 作为欧几里得空间然后继续。但是,如果有一天你做分析、逻辑或集合论的研究生工作,你会回到这些悖论。

### "实数"到底是什么?

在计算上,ℝ 不存在 — 每个数字实现都使用**浮点**(IEEE 754),精度约为 7 位(float32)或约 15 位(float64)。浮点错误在数值计算中传播(在第 10 学期与 REAMAT 一起讨论)。当你在 Python 中写 `0.1 + 0.2` 时,你会得到 `0.30000000000000004`。ℚ ⊊ ℝ 是数学抽象;计算机在比 ℚ 小得多的有限子集中工作。

</Porta>

### 示例 1:ENEM 年龄范围

ENEM 2025 注册接受第一次考试日期前年满 16 岁的考生。以记号:

$$\text{考生年龄} \in [16, +\infty)$$

### 示例 2:最低工资和所得税豁免

2025 年在巴西,收入完全免征 IR:

$$\text{月收入} \in [0,\ 2\,824]$$

(范围以 R\$ 计。)超过此范围,采用累进税率。

### 示例 3:空腹血糖范围(医学)

| 范围 | 区间(mg/dL) | 诊断 |
|---|---|---|
| 正常 | $[0, 99]$ | 健康 |
| 糖尿病前期 | $[100, 125]$ | 注意 |
| 糖尿病 | $[126, +\infty)$ | 治疗 |

每个医学指南都被表示为可能值集合的**划分**为区间。如果你成为生物医学工程师或医疗软件开发人员,你将每天使用这些范围。

### 示例 4:成本函数的定义域

成本函数 $C(q) = 50 + 12q + 0.5q^2$ 用于制造 $q$ 个单位。

数学定义域:$C: \mathbb{R} \to \mathbb{R}$。
物理(实际)定义域:$q \in \{0, 1, 2, 3, \ldots\} = \mathbb{N}$(你不会制造半辆汽车)。

当"最佳点"的计算给出 $q^* = 17.3$ 时,作为工程师你的工作是决定四舍五入为 17 或 18 — 连续数学产生 ℝ 中的解;应用需要 ℕ 中的解。这种**离散 vs. 连续**的张力是根据 Rezende(UFF, 2003)在微积分中认识论困难的五个来源之一。
