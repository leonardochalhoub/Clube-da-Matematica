<Porta nivel="formal" titulo="定义和性质">

<Definicao titulo="仿射函数">
$f: \mathbb{R} \to \mathbb{R}$ 是**仿射函数**，如果存在 $a, b \in \mathbb{R}$ 使得 $f(x) = ax + b$。
</Definicao>

- $a$：**斜率系数**（slope，倾斜度）
- $b$：**线性系数**（y 截距）
- 图像：直线。$a > 0$：递增。$a < 0$：递减。$a = 0$：常数。

<Equation
  numero="1"
  audioTexto="方程一。F of x 二减 f of x 一，除以 x 二减 x 一，等于 a。仿射函数任意两点间的变化率是常数，等于斜率系数。"
  explicacao="直线两点之间的变化率。对于仿射函数，这个值是常数——不依赖于你选择哪对点。正是这种恒定性将仿射函数与所有其他函数区分开。">
{`\\frac{f(x_2) - f(x_1)}{x_2 - x_1} = a`}
</Equation>

### 函数零点和截距

$f(x) = 0 \iff x = -b/a$（当 $a \neq 0$ 时）。点对 $(0, b)$ 是**纵截距**。点对 $(-b/a, 0)$ 是**零点**（或横截距）。

### 两点确定唯一性定理

<Teorema titulo="两点确定一个仿射函数">
给定两个不同的点 $(x_1, y_1)$ 和 $(x_2, y_2)$ 满足 $x_1 \neq x_2$，存在唯一的仿射函数 $f(x) = ax + b$ 经过这两点。系数为：
$$a = \frac{y_2 - y_1}{x_2 - x_1}, \quad b = y_1 - a x_1$$
</Teorema>

**证明（概述）。** 存在性：用上面的公式定义 $a$，并令 $b = y_1 - a x_1$。由构造可验证 $f(x_1) = y_1$，且 $f(x_2) = a(x_2 - x_1) + y_1 = (y_2 - y_1) + y_1 = y_2$。唯一性：若 $g(x) = a' x + b'$ 也满足 $g(x_i) = y_i$，则 $a' = (y_2 - y_1)/(x_2 - x_1) = a$ 且 $b' = y_1 - a' x_1 = b$。∎

### 复合与运算

设 $f(x) = a_1 x + b_1$ 和 $g(x) = a_2 x + b_2$。则：

- **和**：$(f + g)(x) = (a_1 + a_2) x + (b_1 + b_2)$ — 仿射函数，斜率相加。
- **复合**：$(f \circ g)(x) = a_1 (a_2 x + b_2) + b_1 = a_1 a_2 x + (a_1 b_2 + b_1)$ — 仿射函数，斜率相乘。
- **逆**（若 $a_1 \neq 0$）：$f^{-1}(y) = (y - b_1)/a_1$ — 也是仿射函数，斜率为 $1/a_1$。

**可逆**仿射函数（$a \neq 0$）在复合运算下构成一个群——结构 $(\mathbb{R}^* \times \mathbb{R}, \circ)$。该观察将在线性代数（第31课起）和仿射几何中使用。

### 数值示例

**问题。** 求经过 $(2, 5)$ 和 $(7, -10)$ 的仿射函数。

**解。** 应用定理：
$$a = \frac{-10 - 5}{7 - 2} = \frac{-15}{5} = -3, \quad b = 5 - (-3)(2) = 5 + 6 = 11.$$

所以 $f(x) = -3x + 11$。**验证：** $f(2) = -6 + 11 = 5$ ✓ 和 $f(7) = -21 + 11 = -10$ ✓。

<Porta nivel="5" titulo="给5岁的孩子">

想象一架楼梯。每个台阶**高度相同**。

- 如果每个台阶高 20 cm，上 1 个台阶你就上升了 20 cm。
- 上 2 个台阶后，上升了 40 cm。
- 上 5 个台阶后，上升了 100 cm = 1 米。

你所在的高度取决于**上了多少个台阶**。每个台阶都增加**相同的量**。

这就是仿射函数。高度是 $f(n) = 20 \cdot n$，其中 $n$ 是台阶数。

如果楼梯从地面以上 50 cm 的楼层开始（像一个小舞台），那么就是 $f(n) = 20n + 50$。50 是**起点**（你还没上之前所在的位置）。20 是**每个台阶上升的量**。

<Porta nivel="10" titulo="给10岁的孩子">

你去看电影。门票**固定 R\$ 30**。每包爆米花 **R\$ 8**。如果你买 $n$ 包：

$$\text{总花费} = 30 + 8n$$

这就是仿射函数！其中：
- **30** 是**即使不买爆米花**也要花的钱（门票）。这就是公式 $f(x) = ax + b$ 中的 **b**。
- **8** 是**每多买一包爆米花账单增加的金额**。这就是公式中的 **a**。

任何"有一个固定部分 + 一个以相同节奏增长的部分"类型的函数，你就有仿射函数。其他例子：

- **Uber 账单**：起步价（R\$ 6 固定）+ R\$ 2 每公里。⟹ $f(d) = 6 + 2d$。
- **电费**：使用费（R\$ 25）+ R\$ 0.80 每千瓦时。⟹ $f(k) = 25 + 0.80\,k$。
- **滑板车租赁**：R\$ 3 解锁 + R\$ 0.80 每分钟。

图上的直线**直观地**表示这个公式。直线越倾斜，你的账单增长越快。

<Porta nivel="15" titulo="给15岁的青少年">

仿射函数是存在的最简单的"有趣"函数类型——也是开启微积分大门的函数。

规则是：$f(x) = ax + b$。输入 $x$ 值，乘以 $a$，加 $b$，输出 $f(x)$。就这么简单。

**为什么重要？** 因为图像是**直线**，且该直线的斜率恰好是 $a$。当你在 $x$ 轴上**向右走 1 步**时，在 y 轴上**上升 a 步**。在直线上的任何点都是如此。这种**恒定性**使仿射函数在行为上是"线性的"。

几节课之后你将学习**瞬时变化率**（导数）。你会看到：对于仿射函数，导数是常数且等于 $a$。对于其他函数（抛物线、指数函数），"斜率"随 $x$ 变化——这正是微积分出现的地方。但对于直线，很简单：斜率是 $a$，就这样。

**具体例子。** 你以恒定速度 60 km/h 开车。行驶距离是 $d(t) = 60t$（其中 $t$ 以小时为单位）。"斜率" $a = 60$ 就是速度。这有道理——直线越倾斜，你走得越快。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

仿射函数是大多数物理模型在**局部线性化**的关键。即使是非线性现象（带空气阻力的自由落体、非牛顿动力学），在一阶近似下也是仿射的——这正是导数所形式化的。

### 简化形式 vs 一般形式

- 简化形式：$y = ax + b$
- 一般形式：$Ax + By + C = 0$
- 参数形式：$\{(x_0 + t\vec{u}_x, y_0 + t\vec{u}_y) : t \in \mathbb{R}\}$

每种形式在不同情境下有用。在 CAD 中，参数形式。在线性优化中，一般形式。在统计（回归）中，简化形式。

### 直线拟合（线性回归预览）

给定数据 $(x_1, y_1), \ldots, (x_n, y_n)$，最佳拟合直线是什么？**最小二乘法**：

$$a = \frac{n \sum x_i y_i - \sum x_i \sum y_i}{n \sum x_i^2 - (\sum x_i)^2}, \quad b = \bar{y} - a \bar{x}$$

你将在第11学期（推断统计学）中详细学习。

### 应用（部分列表）

- **胡克定律**：$F = kx$ — 弹性力作为伸长的函数（仿射函数，$b = 0$，即线性）
- **欧姆定律**：$V = RI$ — 电压作为电流的函数（线性）
- **摄氏 → 华氏转换**：$F = 1.8C + 32$（仿射，$a = 1.8$，$b = 32$）
- **匀速运动方程**：$s(t) = s_0 + vt$（关于 $t$ 的仿射，$b = s_0$，$a = v$）

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

在实践中，"**线性是局部的**"。涉及模型的大多数工程技术活动最终归结为直线拟合：

- **控制工程中的参数估计**：通过正弦激励和拉普拉斯空间中的线性拟合识别对象。
- **泰勒线性化**：任何在 $x_0$ 周围光滑的函数局部都是仿射的——$f(x) \approx f(x_0) + f'(x_0)(x - x_0)$。所有动力系统稳定性分析都依赖于此。
- **线性回归 OLS**：使 $n$ 个点的平方误差最小化的直线是计量经济学家（GDP ~ 投资）、药剂师（剂量 ~ 反应）、农学家（降雨 ~ 收成）的基础。
- **逻辑回归**：log-odds 的"线性化"——概率比的对数是协变量的仿射函数。所有分类的 AI（垃圾邮件过滤器、欺诈检测）都将逻辑回归作为核心。

### 注意：异方差性和虚假线性

在生产中，未检查残差就将 $y = ax + b$ 拟合到数据是灾难的配方。回归的前提（高斯-马尔可夫）：

1. 关系本身是线性的——不仅仅是数据。
2. 残差具有恒定方差（同方差性）。
3. 残差不相关。
4. 残差均值为零。

当这些失败时，$\hat a$ 仍然是估计量，但失去了效率或一致性的属性。**未经检验的 OLS 是邪教。**

### 线性化 vs 变换

当关系**不是**仿射时，有两种解决方案：(a) 用非线性模型拟合，(b) **变换变量**直到变成仿射。经典例子：

- $y = a x^b$ ⟹ $\ln y = \ln a + b \ln x$ — log-log 线性（幂律）
- $y = a \e^{bx}$ ⟹ $\ln y = \ln a + bx$ — 半对数线性（指数）
- $y = a/x$ ⟹ $y = a \cdot (1/x)$ — 关于 $1/x$ 仿射

绘制 log-log 或半对数图并查看是否变成直线是首要的诊断工具。

<Porta nivel="pratica" titulo="实际应用 — 3 个真实场景">

### 场景1：CEMIG 电费（米纳斯吉拉斯，2026年4月）

B1 类住宅电价（估计）：R\$ 0.82/kWh + 黄色旗 R\$ 0.02/kWh + 消费的 25% ICMS + 固定可用费 R\$ 35.40（三相住宅）。对于消耗 $k$（kWh）：

$$C(k) = (0.82 + 0.02) \cdot 1.25 \cdot k + 35.40 = 1.05 k + 35.40$$

**仿射函数**，其中 $a = 1.05$ R\$/kWh 和 $b = 35.40$ R\$。对于 $k = 200$ kWh，账单为 $C(200) = 245.40$ 雷亚尔。

**边际分析**：每多用一度电正好花费 R\$ 1.05。R\$ 35.40 的"固定部分"不依赖于消耗——这是即使**整个月旅行而断电**也要支付的金额。

### 场景2：裂纹扩展速度（断裂力学）

在疲劳测试（Paris 曲线）中，金属中裂纹的扩展可以线性化为：

$$\log(da/dN) = \log C + m \log(\Delta K)$$

其中 $da/dN$ 是每周期裂纹速度，$\Delta K$ 是应力强度因子的变化，$C, m$ 是材料常数。在 log-log 尺度上这是斜率为 $m$ 的直线——称为 **Paris 指数**。

对典型结构钢，$m \approx 3$。这意味着：将 $\Delta K$ 加倍会使扩展速度乘以 $2^3 = 8$ — 在 log-log 尺度上变线性的剧烈非线性行为。

### 场景3：聚合需求曲线（微观经济学）

商品需求通常被建模为仿射函数：$Q(p) = \alpha - \beta p$，其中 $\alpha, \beta > 0$，$p$ 是价格。需求的价格弹性是 $\epsilon = -\beta p / Q$ — 即使函数是仿射的，沿曲线变化。

数值例子：$Q(p) = 1000 - 25 p$。对 $p = 20$，$Q = 500$，弹性 $|\epsilon| = 25 \cdot 20 / 500 = 1$（单位）。对 $p = 30$，$Q = 250$，弹性 $|\epsilon| = 3$（弹性）。对 $p = 10$，$Q = 750$，$|\epsilon| = 0.33$（无弹性）。**操作观察**：提高无弹性商品价格的公司增加收入；对弹性商品，则失去收入。
