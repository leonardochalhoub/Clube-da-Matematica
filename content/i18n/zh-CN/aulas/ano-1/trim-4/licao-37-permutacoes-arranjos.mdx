<Porta nivel="formal" titulo="定义">

### 阶乘

$n! = n \cdot (n-1) \cdot (n-2) \cdots 1$。约定：$0! = 1$。

增长：

| $n$ | $n!$ |
|---|---|
| 5 | 120 |
| 10 | 3 628 800 |
| 20 | $\approx 2{,}4 \times 10^{18}$ |
| 70 | $> 10^{100}$ |
| 170 | float64 溢出 |

斯特林近似：$n! \approx \sqrt{2\pi n}(n/e)^n$。

### 简单全排列

$P_n = n!$——$n$ 个不同对象排成一行的方式。

### 重复排列

$n$ 个对象中 $n_1$ 个 1 型，$n_2$ 个 2 型，...，$n_k$ 个 $k$ 型：
$$P_n^{n_1, n_2, \ldots, n_k} = \frac{n!}{n_1! \cdot n_2! \cdots n_k!}$$

"ARARA"（3 A，2 R）的字谜：$5!/(3! \cdot 2!) = 10$。

### 简单有序选择

$$A_n^p = n(n-1)(n-2) \cdots (n-p+1) = \frac{n!}{(n-p)!}$$

从 $n$ 个中选 $p$ 个并排序的方式。

### 圆排列

$n$ 物围圆：$(n-1)!$。原因："首位"是任意的。

### 排列与有序选择的差别

- **排列**：用所有 $n$ 个对象。
- **有序选择**：选 $p \leq n$ 个并排序。

当 $p = n$：有序选择与排列重合。

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

想象你有**3 个玩具**排一行。能排出多少种？

可以 A, B, C。或 A, C, B。或 B, A, C。或 B, C, A。或 C, A, B。或 C, B, A。**6 种**！

这个 "6" 是 $3 \times 2 \times 1$——3 的阶乘，写作 $3!$。每加一个玩具就乘。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

**排列** = 改变整组的顺序。

例：你有 5 本书想排在书架上。**多少种？** $5! = 5 \times 4 \times 3 \times 2 \times 1 = 120$。

**有序选择**类似，但你只用**部分**项，不全用。

例：10 名运动员要颁奖（1, 2, 3 名）。**$10 \times 9 \times 8 = 720$**。

**注意**：两者中**顺序都重要**。在领奖台上 A 与 B 调换是不同的。

**字母重复**：PARA 的字谜 = $4!/2! = 12$（因为 A 重复 2 次——除以）。

**圆桌**特殊：5 人围桌 = $4! = 24$（不是 $5!$）。为什么？整体一起转不算新排法。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

**排列** = 全部排序。5 本书在书架上的方式？$5! = 120$。

**有序选择** = 选取并排序。10 名运动员中颁奖 1, 2, 3 名的方式？$A_{10}^3 = 10 \cdot 9 \cdot 8 = 720$。

**直接用公式**：两者中顺序都重要。第一第二名 A 与 B 调换是不同的。

**重复排列**特殊："AMOR" 字谜 $4! = 24$，但 "AMA" 字谜 $3!/2! = 3$（不是 $6$，因为 AA 等于 AA）。

**经典案例**：
- "MISSISSIPPI"（11 字母：1 M，4 I，4 S，2 P）：$11!/(1!\cdot 4!\cdot 4!\cdot 2!) = 34\,650$。
- 8 人圆桌：$7! = 5\,040$。

**阶乘增长快**：$5! = 120$，$10! \approx 360$ 万，$20!$ 已超 $10^{18}$。**所以排列问题"爆炸"**。

**有序选择 vs PFC**：$A_n^p = n(n-1)(n-2)\cdots(n-p+1)$ 只是带"无重复"限制的 PFC。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

排列和有序选择是枚举和概率估计的组合原语。

### 证明：$P_n = n!$

PFC：第1位 $n$ 选项，第2位 $n-1$，...，最后 1。乘积：$n!$。

### 通过 PFC 证明有序选择

$A_n^p = n(n-1)(n-2)\cdots(n-p+1) = n!/(n-p)!$。

### 阶乘增长

斯特林：$\ln(n!) = n\ln n - n + O(\ln n)$。所以 $n! \sim \sqrt{2\pi n}(n/e)^n$。在 ML 中对高斯对数似然、分类模型熵很重要。

### 跨领域

- **TSP（旅行商问题）**：$n!$ 路径 → NP 难。启发式（2-opt、Lin-Kernighan、Concorde）攻克实际情形。
- **洗牌**：$52! \approx 8\times 10^{67}$——洗牌均匀化需 $\log_2(52!) \approx 226$ 次（Bayer-Diaconis：7 次基本均匀）。
- **排序**：比较排序最坏情况 $\Omega(n \log n)$，因决策树有 $\geq n!$ 叶。
- **生物信息学**：序列比对涉及排列；替换密码是字母表的排列。

### 排列作为 $S_n$ 元素

$S_n$ 是 $n!$ 阶对称群。群结构——复合、循环、符号——在抽象代数、伽罗瓦理论、物理中核心。

- **符号**：$\text{sgn}(\sigma) \in \{+1, -1\}$，由换位次数奇偶性。
- **循环分解**：除顺序外唯一。
- **出现在 $\det$**：$\det A = \sum_\sigma \text{sgn}(\sigma) \prod a_{i,\sigma(i)}$。

### 在 ML/量化中的应用

- **置换检验**：不假设正态性的统计显著性。
- **随机森林**：通过排列对特征和数据子采样。
- **交叉验证**：洗牌分割。
- **MCMC**：许多链遍历 $S_n$（模拟退火 TSP）。

</Porta>

排列是**对称群** $S_n$ 的实例——代数、几何、物理中的核心对象。

### Wolfram 速览

```mathematica

```

### $S_n$ 的结构

- $|S_n| = n!$。
- **生成元**：相邻换位 $(i, i+1)$。
- $n \geq 3$ 时 $S_n$ 非阿贝尔。
- **交错子群** $A_n$：符号 +1 的排列，$|A_n| = n!/2$。
- $n \geq 5$ 时 $S_n$ 不可解——故 5 次以上多项式无求根公式（伽罗瓦）。

### 量化——定价

- **隐马尔可夫**：隐状态构成字母表，观察依赖。标签的排列是模型对称性。
- **随机折现因子**：通过排列枚举历史情景做 bootstrap。

### 现代 ML 中的排列

- **Transformer 注意力**对词元排列等变——故位置编码。
- **置换不变网络**：DeepSets、PointNet——通过和/最大聚合特征。
- **Sinkhorn-Knopp**：通过双随机矩阵学习软排列。

### 组合生成

- **Lehmer 码**：$S_n \leftrightarrow \prod \{0, \ldots, n-1\}$ 双射。允许不生成列表的随机索引。
- **Heap 算法**：在 $O(n!)$ 总时间生成所有 $n!$ 排列。
- **Steinhaus-Johnson-Trotter**：每次只换两个相邻元素的顺序生成（$S_n$ 的 Gray 码）。

### 研究

- 排列中的**模式回避**——Wilf 等价、Stanley-Wilf 猜想。
- **随机化**：洗牌、cutoff 现象（Diaconis）。
- **排列多面体**——几何。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**排列/有序选择工作流程。**

1. **顺序重要？** 是 → 在这里。
2. **用所有元素？** 排列。**子集？** 有序选择。
3. **有重复？** 除以重数的阶乘。
4. **圆形？** $(n-1)!$。

### Python 配方

```python
import math
import itertools

# 阶乘
math.factorial(10)        # 3628800

# A(n, p)
math.perm(10, 3)          # 720

# 生成排列
list(itertools.permutations([1, 2, 3]))   # 6 元组

# 长度 k 的排列
list(itertools.permutations([1, 2, 3, 4], 2))   # A(4,2) = 12

# 重复排列：生成并去重
def perm_with_rep(letters):
    return set(itertools.permutations(letters))

print(len(perm_with_rep("ARARA")))    # 10

# 大阶乘的斯特林
def stirling(n):
    import math
    return math.sqrt(2*math.pi*n) * (n/math.e)**n
```

### SymPy

```python
import sympy as sp
sp.factorial(20)              # 精确，2.4e18
sp.binomial(10, 3) * sp.factorial(3)   # 通过组合 × p! 得有序选择
```

### 圆桌等

```python
def round_table(n):
    return math.factorial(n - 1)

print(round_table(8))   # 5040
```

### 陷阱

- 字符串中 **`A == B`** vs 多重集中 **`A == B`**：含相同字母的字谜需特殊处理。
- 排列中 **0 vs 1 索引**。
- **溢出**：$n > 170$ 时 $n! > 10^{300}$。用 `math.factorial`（Python 任意整数）或对数。
- **"至少一个相邻"** 通常用补集更易。
