### 矩阵形式

$\begin{cases} a_1 x + b_1 y + c_1 z = d_1 \\ a_2 x + b_2 y + c_2 z = d_2 \\ a_3 x + b_3 y + c_3 z = d_3 \end{cases}$ ⟺ $A\mathbf{x} = \mathbf{b}$，$A_{3 \times 3}$，$\mathbf{x}, \mathbf{b} \in \mathbb{R}^3$。

### 方法1——高斯消元

初等变换（不改变解）：
1. 交换两行。
2. 一行乘以非零标量。
3. 一行倍数加到另一行。

目标：将 $[A | \mathbf{b}]$ 三角化为阶梯形。然后回代。

### 方法2——克拉默

对 $A\mathbf{x} = \mathbf{b}$ 且 $\det A \neq 0$：
$$x_i = \frac{\det A_i}{\det A}$$

其中 $A_i$ 是 $A$ 的第 $i$ 列被 $\mathbf{b}$ 替换。

### 方法3——逆

$\mathbf{x} = A^{-1} \mathbf{b}$。可通过 $[A | I] \to [I | A^{-1}]$ 行变换计算 $A^{-1}$。

### 何时使用

| 方法 | 代价 | 何时使用 |
|---|---|---|
| 克拉默 | $O(n^4)$ | $n \leq 3$，教学 |
| 高斯 | $O(n^3)$ | 实际标准 |
| 逆 | $O(n^3)$ + 每方程组 $O(n^2)$ | 多个 $\mathbf b$ 同 $A$ |
| LU | $O(n^3)$ + 每方程组 $O(n^2)$ | 多方程组，比逆好 |

### 方程组分类

- **确定的（SPD）**：唯一解（$\det A \neq 0$，或 $\text{rank}(A) = \text{rank}([A|\mathbf b]) = n$）。
- **不定的（SPI）**：无穷多解（$\text{rank}(A) = \text{rank}([A|\mathbf b]) < n$）。
- **不可解（SI）**：无解（$\text{rank}(A) < \text{rank}([A|\mathbf b])$）。

### 鲁谢-卡佩利定理

$A\mathbf x = \mathbf b$ 有解 ⟺ $\text{rank}(A) = \text{rank}([A|\mathbf b])$。两者 = 未知数个数时解唯一。

<Porta nivel="5" titulo="给5岁的孩子">

方程组就是**同时几个方程**。比如："我有3个苹果和2个梨，共13雷亚尔。我有1个苹果和4个梨，共16雷亚尔。每个多少钱？"

要解，你**聪明地组合**方程**消去**一种水果，找出另一个的价格，再回代。是有规则的拼图。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

**线性方程组** = 几个方程几个未知数同时为真。

例：
$$\begin{cases} x + y = 5 \\ x - y = 1 \end{cases}$$

怎么解？
- **两式相加**：$2x = 6 \Rightarrow x = 3$。
- **相减**：$2y = 4 \Rightarrow y = 2$。

完成：$x = 3, y = 2$。

**对大方程组**，手算麻烦。所以**用矩阵组织**：
$$\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} 5 \\ 1 \end{pmatrix}$$

并用**消元法**（标准方式）或**克拉默法则**（用行列式）或**逆**（$\mathbf x = A^{-1}\mathbf b$）。

**何时出错？**
- 方程组**无解**：方程相互矛盾。
- 方程组有**无穷解**：方程冗余。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

矩阵形式的线性方程组：把系数装到 $A$，未知数装到 $\mathbf{x}$，结果装到 $\mathbf{b}$。方程组变为 $A\mathbf{x} = \mathbf{b}$。

**克拉默**（教学，小型）：$x_i = \det(A_i)/\det(A)$。例：
$$\begin{cases} 2x + y = 5 \\ x - 3y = -2 \end{cases} \Rightarrow A = \begin{pmatrix} 2 & 1 \\ 1 & -3 \end{pmatrix}, \det A = -7$$

$x = \det\begin{pmatrix} 5 & 1 \\ -2 & -3 \end{pmatrix}/(-7) = (-13)/(-7) = 13/7$。

$y = \det\begin{pmatrix} 2 & 5 \\ 1 & -2 \end{pmatrix}/(-7) = (-9)/(-7) = 9/7$。

**高斯**（标准）：做初等变换三角化。然后从下到上代入。

$$[A | \mathbf b] = \begin{pmatrix} 2 & 1 & | & 5 \\ 1 & 3 & | & 6 \end{pmatrix} \xrightarrow{L_2 \leftarrow L_2 - L_1/2} \begin{pmatrix} 2 & 1 & | & 5 \\ 0 & 5/2 & | & 7/2 \end{pmatrix}$$

第二行：$y = 7/5$。代入第一行：$2x = 5 - 7/5 \Rightarrow x = 9/5$。

**分类**：
- $\det A \neq 0$ → **确定**（唯一解）。
- $\det A = 0$ + 一致 → **不定**（无穷）。
- $\det A = 0$ + 不一致 → **不可解**。

**齐次方程组** $A\mathbf x = \mathbf 0$：总有 $\mathbf x = \mathbf 0$（平凡）。有非平凡解 ⟺ $\det A = 0$。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

求解 $A\mathbf x = \mathbf b$ 是所有科学计算中**最常见**的运算。FEM/CFD 仿真约 80% 时间花在线性求解器上。

### 求解器层次

1. **直接方法**（高斯、LU、QR、楚列斯基）：$O(n^3)$，机器精度内精确。
2. **迭代方法**（雅可比、高斯-赛德尔、CG、GMRES、BiCGSTAB）：每次迭代 $O(n^2)$，需收敛，理想用于稀疏。
3. **专用方法**：循环矩阵的 FFT、PDE 的多重网格。

### 部分主元 LU

$PA = LU$——$P$ 置换，$L$ 单位下三角，$U$ 上三角。通过以下解 $A\mathbf x = \mathbf b$：
1. $L\mathbf y = P\mathbf b$（前向代入，$O(n^2)$）。
2. $U\mathbf x = \mathbf y$（后向代入，$O(n^2)$）。

主元对稳定性至关重要——避免除以小数。

### 病态方程组

$\kappa(A) = \|A\|\|A^{-1}\|$。在 float64 下高斯消元损失 $\log_{10}\kappa$ 位。若 $\kappa = 10^{16}$，解纯属垃圾。

### 最小二乘（超定方程组）

$\min_\mathbf x \|A\mathbf x - \mathbf b\|^2$。正规方程：$A^TA\mathbf x = A^T\mathbf b$。**别用！** 条件数平方化。用 QR：$A = QR$ ⟹ $R\mathbf x = Q^T\mathbf b$。

### 跨领域

- **电路**：KCL/KVL 生成线性方程组——改进节点分析。
- **FEM**：$K\mathbf u = \mathbf f$——百万未知数，稀疏，对称正定。
- **优化**：每次牛顿迭代解 $H\Delta\mathbf x = -\nabla f$。
- **统计**：回归 $X^TX\hat\beta = X^T\mathbf y$（但用 QR/SVD）。
- **滤波器（卡尔曼）**：每步预测-修正涉及求解器。

### 数值——最佳实践

- **用 `solve`，不用 `inv`**。
- 对 SPD：楚列斯基。
- 对稀疏：`scipy.sparse.linalg.spsolve` 或带预条件子的迭代法。
- 对 $> 10^4$ 稠密：GPU 或迭代方法。

</Porta>

### 矩阵类与最优算法

| 类型 | 求解器 |
|---|---|
| 一般稠密 | LU（高斯+主元） |
| 稠密 SPD | 楚列斯基 |
| 稠密矩形 | QR |
| 稀疏对称不定 | MINRES、SYMMLQ |
| 稀疏 SPD | CG（带预条件子） |
| 稀疏非对称 | GMRES、BiCGSTAB |
| 结构化（托普利茨） | 基于 FFT，$O(n \log n)$ |
| 三对角 | Thomas，$O(n)$ |
| 带状 | 专用 LU $O(nb^2)$ |

### 现代迭代

- **Krylov 子空间**：$\mathcal K_k = \text{span}\{\mathbf b, A\mathbf b, \ldots, A^{k-1}\mathbf b\}$。CG（SPD）、GMRES（一般）在 $\mathcal K_k$ 中构造最优近似。
- **收敛**：CG $O(\sqrt{\kappa})$ 次迭代。预条件 $\widetilde A = M^{-1}A$ 使 $\kappa(\widetilde A) \ll \kappa(A)$ 是艺术。

### 大数据/稀疏

- **Hypre、PETSc、Trilinos**：$> 10^9$ 系统的工业库。
- **多重网格**：椭圆问题 $O(n)$。
- **域分解**：通过分解 $\Omega$ 为子域并行化。

### 量化——风险中的方程组

模型校准（BS、Heston、SABR）生成病态方程组。**Levenberg-Marquardt**、**Tikhonov 正则化**、**截断 SVD** 是标准。

### 当前研究

- **随机数值线性代数**：求解前用 sketching 降维。Drineas、Mahoney。
- **混合精度迭代精化**：FP32 求解，FP64 精化。现代硬件 10× 增益。
- **量子算法（HHL）**：某些类的量子比特 $O(\log n)$——仍理论。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**实际求解 $A\mathbf x = \mathbf b$ 的工作流程。**

1. 看结构：对称？稀疏？三角？
2. 选择兼容的求解器。
3. 若目标是求解，**绝不用显式逆**。
4. 检查条件数；始终验证残差 $\|A\mathbf x - \mathbf b\|$。

### NumPy 配方

```python
import numpy as np
from scipy.linalg import lu, cho_factor, cho_solve, qr, solve_triangular

A = np.array([[2, 1], [1, 3]], dtype=float)
b = np.array([5, 7], dtype=float)

# 标准
x = np.linalg.solve(A, b)

# SPD
c, lo = cho_factor(A)
x = cho_solve((c, lo), b)

# 最小二乘（超定）
A_rect = np.random.randn(100, 5)
b_rect = np.random.randn(100)
x_lsq, residuals, rank, sv = np.linalg.lstsq(A_rect, b_rect, rcond=None)

# 稀疏
from scipy.sparse import csr_matrix
from scipy.sparse.linalg import spsolve
A_sp = csr_matrix(A)
x_sp = spsolve(A_sp, b)

# 迭代（SPD 的 CG）
from scipy.sparse.linalg import cg
x_cg, info = cg(A_sp, b)
```

### 手动 2×2 克拉默

```python
def cramer_2x2(A, b):
    detA = A[0,0]*A[1,1] - A[0,1]*A[1,0]
    if abs(detA) < 1e-14:
        return None
    Ax = np.array([[b[0], A[0,1]], [b[1], A[1,1]]])
    Ay = np.array([[A[0,0], b[0]], [A[1,0], b[1]]])
    return np.array([np.linalg.det(Ax)/detA, np.linalg.det(Ay)/detA])
```

### 陷阱

- 若 `solve` 报错 `LinAlgError: singular matrix`：用 `lstsq` 或 `pinv`。
- 即使 $\det \approx 0$ 也可能有解——取决于 $\mathbf b$ 是否在列空间。
- **始终验证**残差：`np.linalg.norm(A @ x - b)`。
