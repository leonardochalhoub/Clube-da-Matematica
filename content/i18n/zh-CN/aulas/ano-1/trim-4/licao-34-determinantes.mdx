### 2x2

$$\det \begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc$$

### 3x3——萨吕斯法则

$$\det \begin{pmatrix} a & b & c \\ d & e & f \\ g & h & i \end{pmatrix} = aei + bfg + cdh - ceg - bdi - afh$$

（在右边重复前两列，3 个下行乘积 − 3 个上行乘积。）

### n×n——拉普拉斯展开（代数余子式）

$$\det A = \sum_{j=1}^n (-1)^{i+j} a_{ij} M_{ij}$$

其中 $M_{ij}$ 是**子式**（去掉第 $i$ 行第 $j$ 列的子矩阵的行列式）。递归：将 $n \times n$ 简化为 $(n-1) \times (n-1)$ 之和。

### 通过置换的定义（莱布尼茨）

$$\det A = \sum_{\sigma \in S_n} \text{sgn}(\sigma) \prod_{i=1}^n a_{i,\sigma(i)}$$

对所有 $n!$ 置换求和。

### 性质

| # | 性质 |
|---|---|
| 1 | $\det(I) = 1$ |
| 2 | $\det(A^T) = \det(A)$ |
| 3 | $\det(AB) = \det(A) \cdot \det(B)$（柯西-比内） |
| 4 | $A_{n\times n}$ 时 $\det(\alpha A) = \alpha^n \det(A)$ |
| 5 | $\det(A^{-1}) = 1/\det(A)$ |
| 6 | 交换两行/列改变符号 |
| 7 | 零行 ⟹ $\det = 0$ |
| 8 | 两行相等 ⟹ $\det = 0$ |
| 9 | 行成比例 ⟹ $\det = 0$ |
| 10 | 一行的倍数加到另一行**不改变** $\det$ |
| 11 | 一行乘以 $\alpha$ 使 $\det$ 乘以 $\alpha$ |
| 12 | 三角：$\det = $ 对角线乘积 |

### 几何解释

- $|\det A|$ = 由 $A$ 的列生成的平行六面体的体积。
- $\det A > 0$：保持定向。$\det A < 0$：反转定向。
- $\det A = 0$：列线性相关（"压扁"的平行六面体）。

### 可逆性判据

$A$ 可逆 $\iff \det A \neq 0$。

<Porta nivel="5" titulo="给5岁的孩子">

想象你的矩阵是一个玩具**纸盒**。**行列式**就是**盒子的大小**——里面能装多少。

如果盒子像纸一样**扁**（被压扁），大小就是零——什么也装不下。

有些盒子在你乘以东西时会"翻面"——这时大小变成负数。这只是表示盒子翻过来了。

<Porta nivel="10" titulo="给10岁的孩子">

行列式是衡量矩阵**大小**的数字：
- 对 $2 \times 2$ 矩阵，是列形成的平行四边形的**面积**。
- 对 $3 \times 3$，是平行六面体的**体积**。

**2×2 直接计算**：$\det\begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc$。乘主对角线，减次对角线。

**3×3 用萨吕斯**：把前两列复制到右边，加3个下行乘积，减3个上行乘积。

**行列式告诉我们的酷事**：
- $\det = 0$ → 矩阵"扁了"，没有逆。
- $\det \neq 0$ → 矩阵有逆。
- $\det$ 为负 → 矩阵把空间"翻面"。

**应用**：克拉默法则用行列式给出方程组的解——漂亮，但对大方程组太慢。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

行列式是从方阵计算出的**一个数**。

对 2x2：
$$\det \begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc$$

对 3x3 用**萨吕斯**：
$$\det\begin{pmatrix} a & b & c \\ d & e & f \\ g & h & i \end{pmatrix} = aei + bfg + cdh - ceg - bdi - afh$$

对 4×4 或更大用**拉普拉斯**（代数余子式展开）或消元法（更高效）。

**为什么重要？**

1. **可逆性**：$\det A \neq 0$ ⟺ $A$ 可逆。
2. **克拉默**：$A\mathbf{x} = \mathbf{b}$ 的解用行列式之比。
3. **面积和体积**：对 2x2，$|\det A|$ 是列形成的平行四边形**面积**。对 3x3 是**体积**。

**几何上**：若 $A$ 的列是"指向相同方向的向量"，平行四边形很瘦，$\det A$ 小。若垂直，平行四边形"胖"，$\det A$ 大。

**必背的基本性质**：
- $\det(AB) = \det A \cdot \det B$——可乘！
- 交换两行改变符号。
- 零行或两行相等 ⟹ $\det = 0$。
- 三角矩阵：对角乘积。

**计算技巧**：对大矩阵，消元到三角——性质10保持行列式。然后乘对角线。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

行列式是**唯一**的多线性、交错、单位元处取1的泛函。这种公理化刻画（外尔斯特拉斯）唯一地定义 $\det$。

### 公理（外尔斯特拉斯）

1. 行（和列）**多线性**。
2. **交错**：交换两行改变符号。
3. **归一**：$\det I = 1$。

其余皆推论。

### 证明：$\det(AB) = \det A \det B$

固定 $B$。定义 $f(A) = \det(AB)$。则 $f$ 在 $A$ 的行上多线性且交错。所以 $f(A) = c \det A$ 对某 $c$。取 $A = I$：$f(I) = \det B = c$。所以 $\det(AB) = \det A \det B$。

### 代数余子式与伴随矩阵

$\text{adj}(A) = (C_{ij})^T$，$C_{ij} = (-1)^{i+j} M_{ij}$。基本恒等式：
$$A \cdot \text{adj}(A) = \det(A) \cdot I$$

所以 $A^{-1} = \frac{1}{\det A} \text{adj}(A)$——理论上有用，数值上很差。

### 跨领域

- **CG**：雅可比 $J = \det\frac{\partial(u,v)}{\partial(x,y)}$ 在坐标变换中修正面积。曲面/体积积分。
- **力学**：物体→参考系变换的雅可比衡量变形。
- **量子**：费米子的斯莱特行列式（反对称性）。
- **线性 ODE**：朗斯基 $W(t) = \det\begin{pmatrix} y_1 & y_2 \\ y_1' & y_2' \end{pmatrix}$——解的独立性。
- **动力系统**：$\det A$ = 特征值乘积。$\det$ 的符号关联到稳定性。

### 有向体积

在 $\mathbb R^n$ 中，$|\det A|$ = 由列生成的 $n$ 维体积。通过 SVD：$|\det A| = \prod \sigma_i$（奇异值）。

### 数值计算

- **拉普拉斯** $O(n!)$——只对小 $n$。
- **通过 LU**：$\det A = \pm \prod u_{ii}$（$O(n^3)$）。**标准。**
- 对大 $n$ 注意下溢：使用 $\log|\det A|$。

### 与特征值的联系

$$\det A = \prod_i \lambda_i, \quad \text{tr}(A) = \sum_i \lambda_i$$

刻画矩阵超出表面。在 $2\times 2$ 中：$\lambda_{1,2} = \frac{\text{tr} \pm \sqrt{\text{tr}^2 - 4\det}}{2}$。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

行列式是**唯一**自然推广到微分几何中**体积形式**的矩阵不变量——通过 $\det$ 在 $\mathbb R^n$ 上定义标准体积形式 $dx_1 \wedge \cdots \wedge dx_n$。

### 张量视角

$\det A = \frac{1}{n!}\epsilon^{i_1\cdots i_n} \epsilon_{j_1 \cdots j_n} A^{j_1}_{i_1} \cdots A^{j_n}_{i_n}$——与列维-奇维塔的缩并。在任意域上的任意空间上定义 $\det$（直至正特征，有保留）。

### Wolfram 速览

```mathematica
A = {{1, 2, 3}, {4, 5, 6}, {7, 8, 10}};
Det[A]
Minors[A, 2]                    (* 所有 2x2 子式 *)
Cofactor[A, {1, 1}]
Eigenvalues[A]
```

### 雅可比恒等式

$$\frac{d}{dt}\det A(t) = \det A(t) \cdot \text{tr}\left(A(t)^{-1} \dot A(t)\right)$$

在连续介质力学（体积变化率）、随机过程、哈密顿流中至关重要。

### 推广的柯西-比内

对 $A_{m\times n}, B_{n\times m}$ 且 $m \leq n$：
$$\det(AB) = \sum_{S \in \binom{[n]}{m}} \det(A_{[m],S}) \det(B_{S,[m]})$$

出现在统计（Gram 矩阵）和组合物理中。

### 量化/金融——风险与 copula

- $\det(\Sigma)$：投资组合总聚合方差度量。
- 通过 Radon-Nikodym 测度变换使用雅可比。
- 多维 copula：混合偏导 = 相关 Hessian 的 $\det$。

### 现代计算

- **通过舒尔补求行列式**：对块 $\begin{pmatrix} A & B \\ C & D \end{pmatrix}$，$\det = \det(A) \det(D - CA^{-1}B)$。
- **对数行列式**避免溢出：通过 LU $\log\det = \sum \log|u_{ii}|$。ML 中标准（高斯先验）。

### 研究与现代应用

- **行列式点过程（DPPs）**：$P(S) \propto \det(K_S)$。在推荐系统、量子中建模多样性。
- **计算复杂度**：Valiant 的 $\#P$ 类——permanent 是难的，行列式是简单的。基本区别。
- **Pfaffian**：反对称 $A$ 时 $\text{Pf}(A)^2 = \det(A)$——完美匹配的组合数学。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**计算行列式的工作流程。**

1. **2×2**：直接 $ad - bc$。4次乘法 + 1次减法。
2. **3×3**：萨吕斯。比拉普拉斯快。
3. **n×n**（n ≥ 4）：消元 → 三角 → 对角线乘积。
4. **数值上**：用 LU；生产中绝不实现拉普拉斯。

### NumPy 配方

```python
import numpy as np

A = np.array([[1, 2, 3], [4, 5, 6], [7, 8, 10]], dtype=float)

# 行列式
print(np.linalg.det(A))        # 内部用 LU

# 对数行列式（稳定）
sign, logdet = np.linalg.slogdet(A)
print(sign * np.exp(logdet))   # 等价于 det

# 病态矩阵优先用 slogdet
```

### SymPy 符号

```python
import sympy as sp
A = sp.Matrix([[1, 2], [3, 4]])
print(A.det())                 # 闭形式精确
```

### 显式 2×2 克拉默

```python
def cramer_2x2(A, b):
    a, B, c, d = A[0, 0], A[0, 1], A[1, 0], A[1, 1]
    det = a * d - B * c
    if abs(det) < 1e-12:
        raise ValueError("奇异系统")
    x = (b[0] * d - B * b[1]) / det
    y = (a * b[1] - b[0] * c) / det
    return np.array([x, y])
```

### 常见陷阱

- 对整数矩阵 **`np.linalg.det` 返回 float64**——可能有舍入误差。用 SymPy 求精确。
- **大行列式**可能溢出——用 `slogdet`。
- **克拉默不可扩展**：朴素 $O(n^4)$，优化 $O(n^3)$，vs. 直接高斯 $O(n^3)$。
- **萨吕斯**对 $4\times 4$ 或更大**不适用**——只对 $3\times 3$。
