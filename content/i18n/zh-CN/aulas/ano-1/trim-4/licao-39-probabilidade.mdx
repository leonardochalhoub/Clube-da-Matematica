<Porta nivel="formal" titulo="公理与公式">

### 样本空间

$\Omega$：实验所有可能结果的集合。**事件**：$\Omega$ 的子集。

### 柯尔莫哥洛夫公理（1933）

1. 对每个事件 $A$，$P(A) \geq 0$。
2. $P(\Omega) = 1$。
3. **$\sigma$-可加性**：$A_1, A_2, \ldots$ 不相交时：$P(\bigcup A_i) = \sum P(A_i)$。

### 古典概率（等概空间）

$$P(A) = \frac{|A|}{|\Omega|}$$

### 直接性质

- $P(\emptyset) = 0$。
- $P(A^c) = 1 - P(A)$。
- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$（容斥）。
- $A \subseteq B \Rightarrow P(A) \leq P(B)$。
- $P(A) \in [0, 1]$。

### 条件概率

$$P(A | B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0$$

### 独立性

$A$ 与 $B$ **独立**若 $P(A \cap B) = P(A) P(B)$（等价：$P(A|B) = P(A)$）。

### 贝叶斯定理

$$P(A | B) = \frac{P(B | A) P(A)}{P(B)}$$

### 全概率

若 $\{A_i\}$ 为 $\Omega$ 的划分：
$$P(B) = \sum_i P(B | A_i) P(A_i)$$

### 离散随机变量

$X: \Omega \to \mathbb R$。分布：$P(X = x_i) = p_i$，$\sum p_i = 1$。

| 分布 | 公式 | 出现 |
|---|---|---|
| 伯努利 | $P(X=1) = p$ | 1 次试验 |
| 二项 | $\binom{n}{k}p^k(1-p)^{n-k}$ | $n$ 次试验 |
| 几何 | $(1-p)^{k-1}p$ | 等待时间 |
| 泊松 | $e^{-\lambda}\lambda^k/k!$ | 罕见事件 |

### 期望与方差

- $E[X] = \sum x_i p_i$
- $\text{Var}(X) = E[(X - E[X])^2] = E[X^2] - E[X]^2$

<Porta nivel="5" titulo="给5岁的孩子">

概率是"某事发生的**机会**"。从 0（不可能）到 1（一定）。

抛硬币：正面机会一半（1/2），因为 2 个结果中 1 个是正面。

掷骰子：掷 6 的机会六分之一（1/6），骰子 6 面只有 1 是 6。

**可能性越多**，**越难**猜中。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

**概率** = 某事发生的机会，0 到 1 之间的数。

**古典规则**（结果等概率时）：
$$P(\text{事件}) = \frac{\text{有利情况}}{\text{总情况}}$$

例：
- 骰子：$P(\text{偶数}) = 3/6 = 1/2$。
- 2 硬币：$P(\text{2 正面}) = 1/4$（HH, HT, TH, TT——只有 HH）。
- 牌：$P(\text{K}) = 4/52 = 1/13$。

**条件概率**："如果已知 B 发生，A 的机会？" 写为 $P(A|B)$。

**独立**：A 的结果不影响 B。例：抛 2 骰子——第二个不在乎第一个。

**贝叶斯**：聪明地"反转"概率的公式。从结果找原因。

**出现在**：抽奖、游戏、天气预报、医学诊断、垃圾邮件过滤、AI。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

概率是"测量某事机会"的数。0（不可能）到 1（确定）。

**古典情形**：含 $|\Omega|$ 等概结果的实验（抛硬币，掷骰子）。$P(A)$ = "有利情况除以可能情况"。

例：掷骰子，$P(\text{偶数}) = 3/6 = 1/2$。抛 2 硬币，$P(\text{2 正面}) = 1/4$。

**条件概率**："已知 B 发生，A 的概率？" $P(A|B) = P(A \cap B)/P(B)$。

**独立事件**：$P(A \cap B) = P(A) P(B)$。掷 2 骰子——第 2 个结果不影响第 1 个。

**贝叶斯**："反转"条件的方法。若知 $P(B|A)$，可算 $P(A|B)$。用于医学诊断、垃圾邮件过滤、ML。

**容斥**：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$。事件不相交时，交集多算了一次——减去。

**二项分布**：$n$ 次独立试验中成功数：
$$P(X = k) = \binom{n}{k}p^k(1-p)^{n-k}$$

例：抛硬币 10 次，恰 5 正面机会？$\binom{10}{5}(1/2)^{10} = 252/1024 \approx 0{,}246$。

**组合学+概率并行**：计数给 $|A|$ 和 $|\Omega|$，相除给 $P(A)$。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

概率是数据科学、ML、随机控制、量化金融的**基础**。

### 随机变量

$X: \Omega \to \mathbb{R}$。每个结果对应数值。**离散**或**连续**（下学期）。

### 期望

$E[X] = \sum_i x_i P(X = x_i)$。线性：$E[aX + bY] = aE[X] + bE[Y]$——总是，即使不独立。

### 方差

$\text{Var}(X) = E[(X - \mu)^2] = E[X^2] - \mu^2$。**非线性**：$\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)$。

若 $X, Y$ 独立：$\text{Cov} = 0$，所以 $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y)$。

### 二项分布

$X \sim \text{Bin}(n, p)$：$n$ 次独立试验中 prob $p$ 的"成功"数：
$$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}$$

出现在质量控制、临床试验、A/B 测试。

$E[X] = np$，$\text{Var}(X) = np(1-p)$。

### 实践中的贝叶斯——医学诊断

罕见疾病 $D$，$P(D) = 0{,}001$。检测：$P(+|D) = 0{,}99$（敏感度），$P(-|D^c) = 0{,}95$（特异度）。检测阳性时疾病概率：
$$P(D | +) = \frac{P(+|D) P(D)}{P(+)} = \frac{0{,}99 \cdot 0{,}001}{0{,}99 \cdot 0{,}001 + 0{,}05 \cdot 0{,}999} \approx 0{,}019$$

仅 **1.9%**——基率谬误由贝叶斯解释。

### 跨领域

- **工程**：可靠性——串/并联系统、MTBF。
- **信号**：通过 Neyman-Pearson 在噪声中检测。
- **ML**：贝叶斯分类器、概率神经网络。
- **量化**：风险中性定价、风险价值。
- **统计**：假设检验、p 值、置信区间。

### 大数定律

$\bar X_n = \frac{1}{n}\sum X_i \xrightarrow{n\to\infty} E[X]$（依概率或几乎必然收敛）。证明相对频率作为概率估计。

### 中心极限定理（预告）

$\frac{\bar X - \mu}{\sigma/\sqrt n} \xrightarrow{d} N(0, 1)$。许多独立变量之和趋于高斯——解释正态分布的普遍性。

<Porta nivel="40" titulo="给资深专业人士">

现代概率建立在 Kolmogorov（1933）形式中的**测度空间** $(\Omega, \mathcal F, P)$ 上。每个随机变量都是可测函数。

### Wolfram 速览

```mathematica
Probability[X >= 5, X \[Distributed] BinomialDistribution[10, 0.5]]
Mean[BinomialDistribution[10, 0.5]]
PDF[NormalDistribution[0, 1], x]
RandomVariate[BinomialDistribution[100, 0.3], 10]
```

### 柯尔莫哥洛夫与测度

$\sigma$-代数 $\mathcal F$——可测事件。必要因为连续 $\Omega$（如 $[0,1]$）中并非每个子集可测（Vitali）。离散中 $\mathcal F = 2^\Omega$ 可行。

### 贝叶斯推断

后验 $\propto$ 先验 $\times$ 似然。MCMC（Metropolis-Hastings、HMC、NUTS）从任意后验抽样。Stan、PyMC3、Pyro、NumPyro。现代数据科学的生产工具。

### 量化——测度变换

风险中性定价：$V_0 = e^{-rT} E^Q[\text{payoff}]$。从 $P$（真实世界）到 $Q$（风险中性）的 Radon-Nikodym 变换。连续过程用 Girsanov。

### 集中不等式

- **马尔可夫**：$P(X \geq a) \leq E[X]/a$。
- **切比雪夫**：$P(|X-\mu| \geq k\sigma) \leq 1/k^2$。
- **Hoeffding**：$P(|\bar X_n - \mu| \geq t) \leq 2e^{-2nt^2/(b-a)^2}$——PAC 学习基础。
- **Chernoff**：指数尾——通信、RNG、哈希。

### ML 与概率

- **交叉熵损失**：$-\sum p_i \log q_i$。
- **KL 散度**：$D_{KL}(p\|q) = \sum p_i \log(p_i/q_i)$——变分方法基础。
- **基于分数的生成模型**（扩散）：学习 $\nabla \log p$。
- **神经网络**可理解为对条件分布 $P(Y|X)$ 的通用近似。

### 研究

- **自由概率**（Voiculescu）：大随机矩阵的非交换代数。
- **最优传输**（Villani, Cuturi）：分布间距离——Wasserstein。
- **Stein 方法**：限制 CLT 及变种的误差。
- **马尔可夫链混合**：cutoff 现象，Diaconis。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**概率问题工作流程。**

1. **明确定义 $\Omega$ 与事件 $A$**。80% 错误来自定义不清。
2. **等概？** 用组合（$|A|/|\Omega|$）。
3. **不等概？** 用性质 + 公理。
4. **条件还是贝叶斯？** 识别给定与所求。
5. **独立？** 乘。否则条件化。

### Python 配方

```python
import math
from math import comb
from scipy.stats import binom, norm, poisson

# 古典概率
def prob_classica(favoraveis, totais):
    return favoraveis / totais

print(prob_classica(comb(4, 2), comb(52, 5)))   # P(5 牌 2 K) — 适配

# 二项分布
n, p = 10, 0.5
print(binom.pmf(5, n, p))     # P(X=5)
print(binom.cdf(3, n, p))     # P(X<=3)
print(binom.mean(n, p))       # n*p

# 手动贝叶斯
def bayes(prior_A, likelihood_B_given_A, likelihood_B_given_notA):
    prior_notA = 1 - prior_A
    p_B = likelihood_B_given_A * prior_A + likelihood_B_given_notA * prior_notA
    return likelihood_B_given_A * prior_A / p_B

# 罕见疾病
print(bayes(0.001, 0.99, 0.05))   # P(D|+) ~ 0.019

# 蒙特卡洛模拟
import numpy as np
np.random.seed(42)
samples = np.random.choice([0, 1], size=10000, p=[0.5, 0.5])
print((samples.sum()) / 10000)    # ~ 0.5
```


### 陷阱

- **区分 $P(A|B)$ 与 $P(B|A)$**。贝叶斯正是关于这种反转。
- **独立 != 互斥**。独立可共发生；互斥不可。
- **非等概情形**：掷 2 骰子 → 36 等概结果，但和 2 到 12 **不**等概。
- **基率谬误**：忽略基率——先验概率非常重要。
