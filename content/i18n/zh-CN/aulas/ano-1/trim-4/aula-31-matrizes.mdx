<Porta nivel="formal" titulo="定义与类型">

<Definicao titulo="矩阵">
$\mathbb{R}$ 上的 $m \times n$ 矩阵是函数 $A: \{1, \ldots, m\} \times \{1, \ldots, n\} \to \mathbb{R}$。记号 $A = (a_{ij})$ 或 $A_{m \times n}$。

相等：$A = B \iff$ 维度相同且对所有 $i, j$ 有 $a_{ij} = b_{ij}$。
</Definicao>

### 特殊类型

| 类型 | 定义 |
|---|---|
| 方阵 | $m = n$ |
| 行矩阵 | $1 \times n$ |
| 列矩阵 | $m \times 1$ |
| 对角矩阵 | 方阵且 $i \neq j$ 时 $a_{ij} = 0$ |
| 单位矩阵 $I_n$ | 对角阵且 $a_{ii} = 1$ |
| 零矩阵 $O$ | 所有元素均为零 |
| 上三角 | $i > j$ 时 $a_{ij} = 0$ |
| 下三角 | $i < j$ 时 $a_{ij} = 0$ |
| 对称 | $A^T = A$，即 $a_{ij} = a_{ji}$ |
| 反对称 | $A^T = -A$，即 $a_{ij} = -a_{ji}$ |
| 数量矩阵 | 对角阵且 $a_{ii} = k$ 常数 |

### 方阵的对角

**主对角**为 $\{a_{ii}\}$。**迹**：$\text{tr}(A) = \sum_{i=1}^n a_{ii}$。

### 形成法则

通常用公式 $a_{ij} = f(i, j)$ 定义 $A$。例：
- $a_{ij} = i + j$
- $a_{ij} = (-1)^{i+j}$
- $a_{ij} = \delta_{ij}$（克罗内克 delta——生成单位矩阵）

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

想象一个**鸡蛋盒**。它有行和列。每个格子放一个蛋（也就是一个数字）。

**矩阵**就是这样：一个充满数字的方形或矩形盒子，整齐组织——每个数字都有自己的位置。要说一个数字在哪里，你说出行和列：" 第2行，第3列 "——找到了。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

你在课上见过**表格**。一张成绩表：每个学生是一行，每门学科是一列。**矩阵就是这个**——数字的表格。

日常例子：
- **成绩单**：30个学生 × 5门学科 = 30×5 矩阵。
- **电脑屏幕**：每个像素有颜色值。1920×1080 屏幕是巨大的矩阵。
- **距离表**：城市 × 城市的表。圣保罗行、里约列 = 430 km。

每个数字有**地址**：$a_{ij}$ 是"第 $i$ 行，第 $j$ 列"。$i$ 总是先。

经常出现的特殊类型：
- **方阵**：行数等于列数。
- **单位矩阵**：相当于数字1——只有对角线上是1，其余都是0。
- **零矩阵**：全是零。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

矩阵是按行和列组织的数字**表格**：

$$A = \begin{pmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{pmatrix}$$

这是 $2 \times 3$ 矩阵（2行3列）。每个数字有**地址**：$a_{12} = 2$（第1行第2列）。$a_{23} = 6$（第2行第3列）。

**有什么用？** 用来储存和组织数据，主要是**批量做计算**：
- 学生成绩 × 学科：学生-学科矩阵。
- 城市间距离：对称矩阵。
- 图像像素：强度矩阵。
- 线性方程组的系数：系数矩阵。
- 神经网络的权重：向量和矩阵（成吨）。

**相等**：两个矩阵相等当且仅当**维度相同**且**对应元素全部相等**。$2 \times 3$ 不能与 $3 \times 2$ 比较。

**重要类型**：
- **方阵**：行数等于列数。
- **单位矩阵** $I_n$：对角线上全为1，其余为0。乘法的"1"（$IA = A$）。
- **对角矩阵**：主对角线之外全为零。
- **对称**：$A^T = A$——沿对角线完美镜像。

**形成法则**：可由公式构造矩阵。若 $a_{ij} = i + j$，则 $a_{12} = 3$，$a_{21} = 3$，$a_{33} = 6$ 等。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

矩阵是科学计算和 ML 的**基本对象**。在 Python/NumPy 中，"一切皆数组"。在 MATLAB 中，"一切皆矩阵"。术语来自 **Cayley（1858）**，但结构在拉格朗日和高斯那里已经隐含。

### 作为线性映射的视角

$A_{m \times n}$ 表示通过 $T(\mathbf x) = A\mathbf x$ 的线性映射 $T: \mathbb{R}^n \to \mathbb{R}^m$。**列**是标准基向量的像。这种解读使矩阵无处不在。

### 常见矩阵结构

- **稀疏**：大部分元素为零。CSR/CSC 格式只存储非零。在 FEM、网络、NLP、百万节点图中至关重要。
- **带状**：除中央带外为零。在三对角系统中有用（热方程、Crank-Nicolson、三次样条）。
- **稠密**：全部存储。ML、纯线性代数、BLAS-3 的标准。
- **块对角**：对角线上是方块——解耦的方程组。
- **托普利茨**：$a_{ij}$ 仅依赖 $i - j$——卷积、信号处理。

### 跨领域应用

- **控制系统**：状态 $\dot{\mathbf{x}} = A\mathbf{x} + B\mathbf{u}$，$\mathbf y = C\mathbf x$——A 是"动力学"。
- **结构（FEM）**：全局刚度矩阵 $K$ 是稀疏、对称、正定的。
- **数字图像**：每个 RGB 通道是 $H \times W$ 强度矩阵 $\{0, \ldots, 255\}$。
- **神经网络**：每个稠密层是 $\mathbf y = \sigma(W\mathbf x + \mathbf b)$。
- **量子**：状态是向量，可观测量是厄米矩阵 $A = A^*$。
- **统计**：协方差矩阵 $\Sigma$ 是对称半正定的。
- **图**：邻接矩阵 $A_{ij} = 1$ 若有边——特征值编码结构。

### 向量空间 $M_{m \times n}(\mathbb R)$

$m \times n$ 矩阵集合是**维度** $mn$ 的向量空间——标准基 $\{E_{ij}\}$，$E_{ij}$ 在 $(i,j)$ 位置为1，其余为0。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

矩阵是向量空间上**线性算子**的有限实例。一般理论存在于 $\text{Hom}(V, W)$ 中，对 $V, W$ 向量空间——固定基后 $\text{Hom}(V, W) \cong M_{m \times n}$。

### 结构视角

- **$M_n(\mathbb R)$ 是单位元为 $I_n$ 的非交换结合代数**。
- **谱分解**：SVD（$A = U\Sigma V^T$）、特征值（$A = PDP^{-1}$ 若可对角化）、Schur、Jordan——每种揭示不同方面。
- **矩阵群**（$GL_n$、$SO_n$、$U_n$、$Sp_{2n}$）是几何、物理、表示论的核心对象。

### 现代数值方法

- **BLAS/LAPACK**：GEMM（乘法）是计算中最优化的内核。理论 $O(n^3)$，但内存/缓存主导——GPU 上浮点运算"免费"。
- **自动微分和 ML**：库（PyTorch、JAX）将张量视为广义矩阵。反向传播是矩阵运算图上的链式法则。
- **随机化**：随机投影（Johnson-Lindenstrauss）、sketching、随机化 SVD 在大数据中降低复杂度。

### 量化/金融

- **协方差矩阵** $\Sigma$ 收益是马科维茨的基础。鲁棒估计（收缩、Ledoit-Wolf）是核心问题。
- **马尔可夫链转移矩阵**对信用评级（S&P、Moody's）建模。
- **楚列斯基** $\Sigma = LL^T$ 在蒙特卡洛中生成相关高斯样本。

### Wolfram/SymPy 实践

```python
import numpy as np
A = np.array([[1, 2], [3, 4]])
print(A.shape, A.T, np.trace(A))
```


### 推广

- **张量**（$T_{ijk\ldots}$）：将矩阵推广到多个指标。分解（CP、Tucker）是 SVD 的类比。
- **无限维算子**：Fredholm 积分、紧算子——连续类比。
- **有限域上的矩阵**：纠错码、密码学（Niederreiter、McEliece）、组合数学。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**实际操作矩阵的工作流程。**

1. **识别结构**：维度？对称？稀疏？三角？结构节省内存和浮点运算。
2. **选择表示**：NumPy 数组（稠密）、`scipy.sparse`（稀疏）、`pandas.DataFrame`（带标签）、`xarray`（带名称的多维）。
3. **向量化**：永远不要用 `for` 循环做逐元素运算——使用广播。

### NumPy——配方

```python
import numpy as np

# 构造
A = np.array([[1, 2, 3], [4, 5, 6]])     # 2x3 显式
I = np.eye(3)                             # 3x3 单位矩阵
Z = np.zeros((4, 5))                      # 4x5 零矩阵
D = np.diag([2, -1, 5])                   # 对角

# 形成法则
i, j = np.indices((3, 3))
A_lei = i + j                             # a_ij = i + j

# 访问
print(A[0, 2])      # 第0行第2列元素（0为基的索引！）
print(A.shape)      # (2, 3)
print(A.T)          # 转置
print(np.trace(np.eye(5)))   # 迹
```

### SymPy——符号计算

```python
import sympy as sp
A = sp.Matrix([[1, 2], [3, 4]])
print(A.shape, A.T, A.trace())
```

### 对称性检查

```python
def is_symmetric(A, tol=1e-9):
    return np.allclose(A, A.T, atol=tol)
```

### 常见陷阱

- Python/NumPy 中**0 为基的索引** vs 数学中的 **1 为基**。翻译公式时要小心。
- **行 vs 列**：`A[i]` 是第 $i$ 行。列用 `A[:, j]`。
- **复制 vs 视图**：`B = A` 不复制！用 `B = A.copy()`。
- **`np.matrix` 已弃用**。用 `np.ndarray`（`@` 表示矩阵乘法）。
