<Porta nivel="formal" titulo="直线方程的各种形式">

### 斜截式

$y = mx + n$，其中 $m$ 是**斜率**（倾斜度），$n$ 是**线性系数**（在 $y$ 轴上的截距）。

### 一般式

$Ax + By + C = 0$，且 $(A, B) \neq (0, 0)$。对于非垂直直线（$B \neq 0$）：$m = -A/B$，$n = -C/B$。

### 点斜式

过 $(x_0, y_0)$ 且斜率为 $m$ 的直线：
$$y - y_0 = m(x - x_0)$$

### 参数式

方向向量 $\vec u = (a, b)$ 过 $(x_0, y_0)$ 的直线：
$$\begin{cases} x = x_0 + at \\ y = y_0 + bt \end{cases}, \quad t \in \mathbb{R}$$

### 两点式方程

过 $(x_1, y_1)$ 和 $(x_2, y_2)$ 的直线：
$$\frac{y - y_1}{x - x_1} = \frac{y_2 - y_1}{x_2 - x_1}$$

### 由两点求斜率

$$m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\Delta y}{\Delta x} = \tan\alpha$$

其中 $\alpha$ 是直线与 $x$ 轴的夹角。

### 5种形式表

| 形式 | 方程 | 何时使用 |
|---|---|---|
| 斜截式 | $y = mx + n$ | 作图，$y$ 显式 |
| 一般式 | $Ax + By + C = 0$ | 垂直线，对称性 |
| 点斜式 | $y - y_0 = m(x - x_0)$ | 给定点+斜率 |
| 参数式 | $(x_0 + at, y_0 + bt)$ | 轨迹，动画 |
| 截距式 | $x/p + y/q = 1$ | 截距 $(p, 0), (0, q)$ |

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

想象一个**小斜坡**。它向上或向下，但**始终保持同样的倾斜度**——没有弯曲。这就是**直线**。

要描述这个斜坡，只需要知道两件事：**它从哪里开始**（截距）和**它有多陡**（斜率）。这样就可以猜出斜坡在任何地方的位置了。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

笛卡尔平面上的直线由 $y = mx + n$ 描述。**$m$** 是斜率（每走一步 $x$ 上升多少）。**$n$** 是直线与垂直轴交叉的位置。

**具体例子**：$y = 2x + 3$。当 $x = 0$ 时，$y = 3$——从 $y$ 轴上的3开始。当 $x$ 增加1时，$y$ 增加2——上升很快。

**实际应用**：出租车费率。起步价 R\$ 5 + 每千米 R\$ 2。方程：$y = 2x + 5$。其中 $y$ 是价格，$x$ 是距离。完美的直线。

**求两点间直线的窍门**：
- 计算 $m = \frac{y_2 - y_1}{x_2 - x_1}$。
- 将一个点代入 $y = mx + n$ 求 $n$。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

直线方程是连接代数与几何的纽带。平面上每条直线都有方程 $y = mx + n$，描述其上的**所有点**。

- $m$ 是**斜率**（直线"上升"或"下降"的程度）。$m = \tan\alpha$，其中 $\alpha$ 是与 $x$ 轴的夹角。
- $n$ 是直线与 $y$ 轴的交点。

当你有**两个点** $(x_1, y_1)$ 和 $(x_2, y_2)$ 时，斜率由下式计算：

$$m = \frac{y_2 - y_1}{x_2 - x_1}$$

然后通过代入其中一个点求出 $n$。

**一般式** $Ax + By + C = 0$ 更"对称"——平等对待 $x$ 和 $y$。当直线是垂直的（$x = $ 常数）时很有用，斜截式无法表达这种情况。

**与第03课的联系**：你已经见过仿射函数 $f(x) = ax + b$。直线就是这个函数的图像，$a = m$，$b = n$。同一件事，不同的语言。

**完整的数值例子**：过 $(2, 3)$ 和 $(5, 9)$ 的直线。$m = (9-3)/(5-2) = 2$。代入 $(2, 3)$：$3 = 2 \cdot 2 + n \Rightarrow n = -1$。方程：$y = 2x - 1$。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

直线方程是所有线性分析的基础。在 $\mathbb{R}^n$ 中，等价物是**超平面** $\sum a_i x_i = b$。

### 直接应用

- **线性规划**：在 $Ax \leq b$ 约束下最大化 $c^Tx$——线性约束（由超平面界定的半空间）。单纯形法在顶点之间游走。
- **简单线性回归**：通过最小二乘法拟合 $\hat y = \hat m x + \hat n$。OLS 估计：
$$\hat m = \frac{\sum (x_i - \bar x)(y_i - \bar y)}{\sum (x_i - \bar x)^2}$$
- **运动学**：匀速直线运动中位置为 $s(t) = s_0 + vt$——$(t, s)$ 平面上的直线。
- **计算机图形学**：直线光栅化（Bresenham 算法，1965 — IBM）。
- **支持向量机**：分类器找到类间最大间隔的超平面。

### 点到直线的距离

对于直线 $Ax + By + C = 0$ 和点 $P_0 = (x_0, y_0)$：
$$d(P_0, \text{直线}) = \frac{|A x_0 + B y_0 + C|}{\sqrt{A^2 + B^2}}$$

通过向量投影证明——向量 $(A, B)$ 与直线垂直（法向量）。

### 黑塞标准式

$$x \cos\theta + y \sin\theta = p$$

其中 $p \geq 0$ 是原点到直线的距离，$\theta$ 是法向量的角度。在计算几何中很有用。

### 与微积分的联系

曲线 $y = f(x)$ 在点 $(a, f(a))$ 处的切线为：
$$y = f(a) + f'(a)(x - a)$$

——局部**线性化**。牛顿-拉夫森法、泰勒级数、梯度下降优化都基于这一思想。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

直线 = $\mathbb{R}^n$ 的1维仿射子空间。所有现代射影几何都从这里开始。

### 射影视角

在齐次坐标 $(X, Y, W)$ 中，$x = X/W$，$y = Y/W$，直线变为：
$$AX + BY + CW = 0$$

——**纯线性形式**。无穷远点（$W = 0$）变得可访问。仿射中两条平行线在射影中相交于无穷远点。这是 **OpenGL**、**PBR** 和所有现代 3D 变换（旋转、平移、透视投影）的基础。

### 对偶性

在射影平面上，**点和直线是对偶的**：每个关于点的定理都有关于直线的版本，将"通过"↔"包含"互换。帕斯卡↔布列安桑。在代数几何中至关重要。

### 在 ML / 量化中的应用

- **线性回归**推广为 $\hat y = X\beta$。正规方程 $X^TX\beta = X^Ty$ 通过 QR 或 SVD 求解。
- **PCA**：投影到使残差方差最小化的超平面。
- **量化金融**：资本资产定价模型（CAPM）：$E[R_i] = R_f + \beta_i (E[R_m] - R_f)$——$(\beta, E[R])$ 平面上的直线。马科维茨前沿是包络；CAPM 是切线。法马-弗伦奇模型扩展到3+维超平面。
- **梯度下降**：每一步用切线近似函数，沿下降方向移动。所有一阶优化都是"斜率为 $-\nabla f$ 的直线"。

### SymPy

```python
import sympy as sp
from sympy import Line, Point
L = Line(Point(2, 3), Point(5, 9))
print(L.equation())     # 2x - y - 1
print(L.slope)          # 2
print(L.distance(Point(0, 0)))  # 1/sqrt(5)
```

### 数值方面

在 IEEE 754 中，几乎垂直的直线（$|m|$ 巨大）会遭受灾难性抵消。使用一般式 $Ax + By + C = 0$ 且 $(A, B)$ 单位化以保持稳定性。鲁棒计算几何（CGAL，Shewchuk 谓词）使用精确算术避免错误。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**配方：在任何场景下找直线方程。**

| 给定 | 策略 |
|---|---|
| 2点 | $m = \Delta y / \Delta x$，然后代入 |
| 1点 + 斜率 | 直接用点斜式 |
| 1点 + 平行直线 | 同 $m$，新 $n$ |
| 1点 + 垂直直线 | $m \to -1/m$ |
| 一般式 $\to$ 斜截式 | $y = -A/B \cdot x - C/B$ |
| 斜截式 $\to$ 一般式 | $mx - y + n = 0$ |

### Wolfram / SymPy 工作流

```python
import sympy as sp
x, y, t = sp.symbols('x y t')

# 过两点的直线
P, Q = sp.Point(1, 2), sp.Point(4, 8)
L = sp.Line(P, Q)
print(L.equation())   # 一般式

# 参数式
print(L.arbitrary_point(t))
```

### 何时需要小心？

- **垂直线**（$x = c$）没有定义的 $m$。使用一般式。
- **重合点**（$x_1 = x_2$，$y_1 = y_2$）：不能定义唯一直线。输入错误。
- **数值上无穷的斜率**：抵消问题——优先使用 $\Delta x \cdot (y - y_0) = \Delta y \cdot (x - x_0)$。

完成**所有**30个练习。直线是解析几何的原子单元——所有曲线方程在微积分中都是"局部切线"。
