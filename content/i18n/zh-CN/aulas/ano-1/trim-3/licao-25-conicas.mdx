<Porta nivel="formal" titulo="标准方程">

### 椭圆

到2个焦点的距离之和为常数：
$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$$

其中 $a > b > 0$。焦点位于 $(\pm c, 0)$，$c = \sqrt{a^2 - b^2}$。长轴 $= 2a$，短轴 $= 2b$。离心率 $e = c/a \in [0, 1)$。当 $a = b$ 时，是圆。

### 抛物线

到焦点的距离 $=$ 到准线的距离：
$$y^2 = 4px$$

焦点在 $(p, 0)$，准线 $x = -p$。

### 双曲线

到2个焦点的距离之差为常数：
$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$$

焦点位于 $(\pm c, 0)$，$c = \sqrt{a^2 + b^2}$。离心率 $e = c/a > 1$。渐近线 $y = \pm (b/a) x$。

### 三种圆锥曲线表

| 曲线 | 标准方程 | 离心率 | 焦点 | 定义 |
|---|---|---|---|---|
| 圆 | $x^2 + y^2 = r^2$ | $e = 0$ | 圆心 | 到圆心距离 $= r$ |
| 椭圆 | $x^2/a^2 + y^2/b^2 = 1$ | $0 < e < 1$ | $(\pm c, 0)$ | $r_1 + r_2 = 2a$ |
| 抛物线 | $y^2 = 4px$ | $e = 1$ | $(p, 0)$ | $r = $ 到准线距离 |
| 双曲线 | $x^2/a^2 - y^2/b^2 = 1$ | $e > 1$ | $(\pm c, 0)$ | $\|r_1 - r_2\| = 2a$ |

### 圆锥曲线一般式

$$Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0$$

判别式 $\Delta = B^2 - 4AC$：
- $\Delta < 0$：椭圆（若 $A=C, B=0$ 则为圆）。
- $\Delta = 0$：抛物线。
- $\Delta > 0$：双曲线。

拿一个**手电筒**指向地板。光形成一个**光的圆圈**！

现在**稍微倾斜手电筒**。圆圈变成**椭圆形**——一个**椭圆**。

**倾斜很多**：光分成**两条曲线**远去——一个**双曲线**。

它们就是四种**圆锥曲线**！

<Porta nivel="10" titulo="给10岁的孩子">

**圆锥曲线**是用平面切圆锥时出现的曲线。有4种：

1. **圆**：水平切。
2. **椭圆**：斜切（蛋形）。
3. **抛物线**：与圆锥的母线平行切。
4. **双曲线**：垂直切，截到双锥的两边。

**生活中的应用**：
- **地球绕太阳的轨道**：椭圆（几乎是圆，但不完美）。
- **篮球罚球**：抛物线。
- **电视抛物面天线**：抛物线——把所有信号集中到一个点（焦点）。
- **超音速飞机的声音**：冲击波的截面是双曲线。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

**圆锥曲线**是用平面切圆锥时出现的3条曲线：

1. **椭圆**——斜切（包括 $a = b$ 时的圆）。
2. **抛物线**——与母线平行切。
3. **双曲线**——切到双锥的两半。

**自然界中出现在哪里？**
- 行星轨道是**椭圆**（开普勒，1609——由牛顿推导）。
- 弹道轨迹是**抛物线**（伽利略）。
- 抛物面天线和车灯：抛物线形以集中/分散光/信号。
- 双曲线出现在音爆、能量为正的彗星轨道中。

**离心率 $e$** 衡量圆锥曲线的"扁平度"：
- $e = 0$：圆。
- $0 < e < 1$：椭圆。
- $e = 1$：抛物线。
- $e > 1$：双曲线。

**统一定义（焦点-准线）**：圆锥曲线是满足 $|PF|/d(P, \ell) = e$ 为常数的点 $P$ 的轨迹。

**数值例子**：椭圆 $x^2/25 + y^2/9 = 1$。$a = 5, b = 3, c = \sqrt{25 - 9} = 4$。焦点在 $(\pm 4, 0)$。离心率 $e = 4/5 = 0{,}8$。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

圆锥曲线是**开普勒方程的解**（$1/r^2$ 引力），出现在几乎所有应用领域。

### 在轨道（开普勒-牛顿）中的应用

围绕中心天体的引力轨道，机械能 $E$ 决定类型：

| 能量 | 离心率 | 圆锥曲线 | 例子 |
|---|---|---|---|
| $E < 0$ | $0 \leq e < 1$ | 椭圆 | 行星，人造卫星 |
| $E = 0$ | $e = 1$ | 抛物线 | 逃逸彗星 |
| $E > 0$ | $e > 1$ | 双曲线 | 星际探测器（旅行者号） |

离心率：
- 地球：$e \approx 0{,}0167$（几乎圆）。
- 火星：$e \approx 0{,}0934$。
- 哈雷彗星：$e \approx 0{,}967$（高度拉长的椭圆）。
- 1I/'Oumuamua（2017年星际访客）：$e \approx 1{,}2$（双曲线——逃离了太阳）。

### 抛物面天线

将平行于轴的射线反射到焦点。方程：$y^2 = 4px$。焦点距顶点 $p$。**卫星电视、反射望远镜、GPS**——都使用抛物线。1888年赫兹专利。

### 双曲线在 LORAN/双曲线 GPS 中

两个信号之间的时间差定义双曲线；3+条双曲线的交点定位。在 GPS 之前是主流系统。仍用于军用备份。

### 椭圆标准方程的证明

设 $F_1 = (-c, 0)$ 和 $F_2 = (c, 0)$。距离之和 $= 2a$：
$$\sqrt{(x+c)^2 + y^2} + \sqrt{(x-c)^2 + y^2} = 2a$$

隔离一个根号并平方两次：
$$\frac{x^2}{a^2} + \frac{y^2}{a^2 - c^2} = 1$$

定义 $b^2 = a^2 - c^2$，得 $x^2/a^2 + y^2/b^2 = 1$。证毕。

### 耳语长廊

在椭圆形房间里，焦点 $F_1$ 处发出的声音会汇聚到 $F_2$。美国国会大厦就有这个特性。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

圆锥曲线是**2次代数曲线**——代数几何最简单的非平凡情形。

### 射影视角

在齐次坐标 $(X, Y, W)$ 中，所有圆锥曲线都变为：
$$\sum a_{ij} X_i X_j = 0$$

——二次型。按**矩阵符号**（西尔维斯特）分类：
- $(2, 0)$ 或 $(0, 2)$：椭圆/圆。
- $(1, 1)$：双曲线。
- 退化：抛物线。

在射影中，**所有非退化圆锥曲线都等价**（通过射影坐标变换）。椭圆、抛物线、双曲线之间的差别在于**它们与无穷远直线的相交方式**——分别为 0、1 或 2 个点。

### 在 ML / 量化金融中的应用

- **二次优化**：二次函数的等高线是椭圆（正定）、双曲线（鞍点）或退化（奇异）。牛顿法实际上处理参数空间中的椭圆。
- **量化**：波动率曲面（波动率微笑/偏斜）通常用 strike-vol 中的圆锥曲线建模。Heston 模型 ↦ $(\rho, \sigma)$ 平面上的椭圆。
- **卡尔曼滤波**：协方差是不确定性椭圆；贝叶斯更新产生新椭圆。
- **马科维茨前沿**：风险-收益（均值-方差）平面上的双曲线。切线是 CAPM。

### 哈密顿-雅可比方程与共焦圆锥曲线

刘维尔可积系统：相同焦点的圆锥曲线形成正交（椭圆）坐标。在轴对称问题的经典力学中至关重要。雅可比 1838。

### 趣闻

阿波罗尼斯（约公元前200年）写了8卷《圆锥曲线论》——7卷流传至今。几乎所有现代术语（椭圆、抛物线、双曲线、焦点）都来自他。自开普勒以来的物理学[诺贝尔奖](https://nobelprize.org)得主们隐含地都使用着他的思想。

<Porta nivel="pratica" titulo="实际应用">

**配方：识别和分析圆锥曲线。**

1. **看 $x^2$ 和 $y^2$ 项的符号**：
   - 同号，系数相等 → 圆。
   - 同号，系数不同 → 椭圆。
   - 异号 → 双曲线。
   - 只有一个二次项 → 抛物线。
2. **中心**（若椭圆/双曲线）：配方法。
3. **计算 $a, b, c$**：$c^2 = a^2 - b^2$（椭圆），$c^2 = a^2 + b^2$（双曲线）。
4. **离心率** $e = c/a$。

### 典型问题工作流

```python
import sympy as sp
from sympy.geometry import Ellipse, Point, Line

# 给定顶点和焦点，求椭圆
E = Ellipse(center=Point(0,0), hradius=5, vradius=3)
print(E.foci)             # 焦点
print(E.eccentricity)     # 4/5
print(E.equation())       # 标准式

# 一点处的切线
P = Point(3, 0)  # 顶点
print(E.tangent_lines(P))
```

### 何时需要小心？

- **旋转的圆锥曲线**（$Bxy \neq 0$）：需要坐标变换。角度 $\tan(2\theta) = B/(A - C)$。
- **退化圆锥曲线**：2条直线（相交、平行或重合）、单点或空集。当行列式 $= 0$ 时发生。
- **CG 中的应用**：抛物线和椭圆通过 NURBS 插值。注意贝塞尔近似。
