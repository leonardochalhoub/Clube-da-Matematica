<Definicao titulo="圆">
$(x, y) \in \mathbb{R}^2$ 中到圆心 $C = (a, b)$ 距离为 $r > 0$ 的点的集合。
</Definicao>

### 标准式

$$(x - a)^2 + (y - b)^2 = r^2$$

### 一般式

展开：$x^2 + y^2 + Dx + Ey + F = 0$，其中 $D = -2a$，$E = -2b$，$F = a^2 + b^2 - r^2$。

恢复圆心：$(a, b) = (-D/2, -E/2)$。半径：$r = \sqrt{D^2/4 + E^2/4 - F}$（若为正则有效；否则不是实圆）。

### 直线与圆

| 位置 | 准则 | 公共点 |
|---|---|---|
| 相离 | $d(C, r) > r$ | 0 |
| 相切 | $d(C, r) = r$ | 1 |
| 相交 | $d(C, r) < r$ | 2 |

### 两个圆

| 位置 | 准则 |
|---|---|
| 外离 | $d(C_1, C_2) > r_1 + r_2$ |
| 外切 | $d = r_1 + r_2$ |
| 相交 | $\lvert r_1 - r_2 \rvert \lt d \lt r_1 + r_2$ |
| 内切 | $d = \lvert r_1 - r_2 \rvert$ |
| 内含 | $d \lt \lvert r_1 - r_2 \rvert$ |

### 切线-半径性质

$P$ 处的切线垂直于半径 $\overline{CP}$。

### 切线方程

圆 $(x-a)^2 + (y-b)^2 = r^2$ 在 $(x_0, y_0)$ 处的切线：
$$(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$$

拿一根**绳子**，把一头固定在地上。另一头拿支粉笔，**始终拉直绳子**走。你会画出一个**完美的轮子**——**圆**！

轮子上的每一个点都到中心**距离相同**。这个距离就是**半径**。

<Porta nivel="10" titulo="给10岁的孩子">

圆是到圆心**距离相同的点的集合**。就像圆规画出的图：圆规两脚之间的距离就是**半径**。

方程 $(x - a)^2 + (y - b)^2 = r^2$ 来自距离公式（第21课）：
- $(a, b)$ 是圆心。
- $r$ 是半径。
- 对于曲线上每个点 $(x, y)$，到圆心的距离是 $r$。两边平方就得到上面的方程。

**具体例子**：自行车轮以轴为中心。轮胎上每个点到轴的距离相同。

**实际应用**：
- Wi-Fi：覆盖范围是圆形。
- 警笛：能听到的人在圆内。
- 田径跑道：周长 400m。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

圆是到圆心**距离相等**的点的集合。距离 $= r$（半径）。

应用第21课的距离公式：
$$\sqrt{(x - a)^2 + (y - b)^2} = r$$

两边平方：$(x - a)^2 + (y - b)^2 = r^2$。**就是这样**。

当圆心在原点时（$a = b = 0$）：$x^2 + y^2 = r^2$。**单位圆**（$r = 1$）：$x^2 + y^2 = 1$——你在第12课的三角圆中用过的！

**一般式 $\to$ 标准式**：配方法。例：$x^2 + y^2 - 4x + 6y - 12 = 0$。
- 关于 $x$：$x^2 - 4x = (x-2)^2 - 4$。
- 关于 $y$：$y^2 + 6y = (y+3)^2 - 9$。
- 代入：$(x-2)^2 - 4 + (y+3)^2 - 9 - 12 = 0 \Rightarrow (x-2)^2 + (y+3)^2 = 25$。
- 圆心 $(2, -3)$，半径 5。

**注意**：并非每个 $x^2 + y^2 + Dx + Ey + F = 0$ 都是圆。若 $D^2/4 + E^2/4 - F < 0$，则没有实点。若 $= 0$，则是单点（"退化"圆）。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

圆是**几何轨迹**（locus）最简单的例子。直接推广：$\mathbb{R}^3$ 中的**球**——$(x-a)^2 + (y-b)^2 + (z-c)^2 = r^2$。在 $\mathbb{R}^n$ 中：$\|x - c\|^2 = r^2$ 是 $(n-1)$-球面。

### 跨领域应用

- **电信**：天线覆盖 = 圆。多重覆盖 = 圆的并/交。Voronoi 图（每点关联最近的天线）。
- **GPS / 三边测量**：每颗卫星定义一个球；你的位置是4个球的交点。3个给位置+时间，第4个去除歧义。
- **机器人**：球形关节的机械臂活动范围是球。
- **计算机图形学**：碰撞检测和剔除的包围球。
- **化学工程**：二元反应系统的吉布斯平衡在相图上形成圆。

### 证明：切线与半径垂直

设 $P_0 = (x_0, y_0)$ 为圆上一点，圆心 $C = (a, b)$。向量 $\vec{CP_0} = (x_0 - a, y_0 - b)$。切线的法向量就是这个 $\vec{CP_0}$。因此方程：
$$(x_0 - a)(x - x_0) + (y_0 - b)(y - y_0) = 0$$

展开并使用 $(x_0 - a)^2 + (y_0 - b)^2 = r^2$：
$$(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$$

### 点的幂

对于点 $P_0$ 和圆心 $C$、半径 $r$ 的圆：
$$\text{pow}(P_0) = |P_0 C|^2 - r^2$$

- 负：$P_0$ 在圆内。
- 零：$P_0$ 在圆周上。
- 正：$P_0$ 在圆外；$\sqrt{\text{pow}}$ 是从 $P_0$ 出发切线的长度。

### 2D 三边测量

3个圆过同一点：2个线性方程（用圆方程对相减消去二次项）。2×2 线性系统求解。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

圆是微分几何的**基本曲线**——常曲率 $\kappa = 1/r$。

### 微分视角

参数化：$\gamma(t) = (a + r\cos t, b + r\sin t)$。切向量 $\gamma'(t) = (-r\sin t, r\cos t)$ 模长为 $r$——非单位但常数。曲率向量 $\gamma''(t)/|\gamma'|^2$ 指向圆心，模长为 $1/r$。

### 推广：密切圆

对平面上任意曲线 $\gamma(t)$，每点都有一个密切圆（最佳拟合圆）：切线与曲线相同，曲率相同。半径 $\rho = 1/\kappa$。在道路设计中至关重要（直线与曲线之间通过回旋曲线过渡）。

### 在 ML / 量化中的应用

- **$\ell_2$ 正则化（Ridge）**：$\|\beta\|_2 \leq c$ 是欧氏球——系数空间中的圆。解是 MSE 等高线触及圆边界处。
- **量化**：用 Heston 或 SABR 进行期权定价时，参数空间中使用球面定义稳定校准区域。
- **信赖域优化**：每步限制在欧氏球内。
- **流形学习**：t-SNE、UMAP 保留局部邻域（= 原空间中的小圆）。

### 推广到 n 维球面

n-球的体积：$V_n(r) = \frac{\pi^{n/2}}{\Gamma(n/2 + 1)} r^n$。测度集中：$n$ 较大时，几乎所有体积都集中在表面附近的薄"壳"中——高维 ML 的关键现象。

### 历史趣闻

笛卡尔（1643）描述了"四圆之圆"（笛卡尔圆定理）：4个相互相切圆的曲率 $k_1, k_2, k_3, k_4$ 满足 $(k_1 + k_2 + k_3 + k_4)^2 = 2(k_1^2 + k_2^2 + k_3^2 + k_4^2)$。阿波罗尼垫片分形。

### SymPy

```python
import sympy as sp
from sympy import Circle, Point, Line
C = Circle(Point(2, 3), 4)
print(C.equation())                    # (x-2)**2 + (y-3)**2 - 16
L = Line(Point(0, 0), Point(1, 1))
print(C.intersection(L))               # 交点
print(C.tangent_lines(Point(10, 3)))   # 2条切线
```

<Porta nivel="pratica" titulo="实际应用">

**配方：从圆方程中提取信息。**

1. **标准式** $(x-a)^2 + (y-b)^2 = r^2$：直接读出圆心 $(a, b)$ 和半径 $r$。
2. **一般式** $x^2 + y^2 + Dx + Ey + F = 0$：配方法（或用 $a = -D/2$，$b = -E/2$，$r^2 = a^2 + b^2 - F$）。
3. **验证**：代入一个点检验——不应给出荒谬值。

### 典型问题工作流

1. **直线-圆相对位置**：计算 $d(\text{圆心}, \text{直线}) = |Aa + Bb + C|/\sqrt{A^2+B^2}$，与 $r$ 比较。
2. **圆上点的切线**：用 $(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$。
3. **从点 $P$ 引外切线**：长度 $= \sqrt{|PC|^2 - r^2}$。
4. **过3点的圆**：用 $x^2 + y^2 + Dx + Ey + F = 0$ 解 3×3 系统。

### Wolfram / SymPy

```python
import sympy as sp
from sympy import Circle, Point
C = Circle(Point(0, 0), 5)
print(C.area)         # 25π
print(C.circumference) # 10π
P = Point(3, 4)
print(P in C)         # True (在边上)
```

### 何时需要小心？

- **一般方程可能不是圆**：$D^2/4 + E^2/4 - F < 0$ → 空集。$= 0$ → 单点。
- **数值方面**：在系数接近零时配方法不稳定；直接通过公式计算圆心/半径。
- **球 vs 圆**：在3D中要小心——3D中的圆是平面与球的交（1D 曲线）；需要两个方程。
