<Porta nivel="formal" titulo="ℝ²中的向量">

<Definicao titulo="向量">
$\vec v \in \mathbb{R}^2$ 是有序对 $(v_1, v_2)$。在几何上，表示**位移**或**有向方向**。
</Definicao>

### 运算

- **加法**：$\vec u + \vec v = (u_1 + v_1, u_2 + v_2)$。
- **数乘**：$\alpha \vec v = (\alpha v_1, \alpha v_2)$。
- **减法**：$\vec u - \vec v = (u_1 - v_1, u_2 - v_2)$。

### 模（范数）

$$|\vec v| = \sqrt{v_1^2 + v_2^2}$$

### 单位向量

$\hat v = \vec v / |\vec v|$ 模为1。**单位向量**。

### 标准基向量

$\hat\imath = (1, 0)$，$\hat\jmath = (0, 1)$。每个向量：$\vec v = v_1 \hat\imath + v_2 \hat\jmath$。

### 极坐标形式

$\vec v = (|\vec v|\cos\theta, |\vec v|\sin\theta)$，其中 $\theta$ 是与 $x$ 正轴的夹角。

### 性质（向量空间的8条公理）

| 性质 | 表达式 |
|---|---|
| 交换律 | $\vec u + \vec v = \vec v + \vec u$ |
| 结合律 | $(\vec u + \vec v) + \vec w = \vec u + (\vec v + \vec w)$ |
| 单位元 | $\vec v + \vec 0 = \vec v$ |
| 相反元 | $\vec v + (-\vec v) = \vec 0$ |
| 分配律（标量/向量） | $\alpha(\vec u + \vec v) = \alpha\vec u + \alpha\vec v$ |
| 分配律（向量/标量） | $(\alpha + \beta)\vec v = \alpha\vec v + \beta\vec v$ |
| 兼容性 | $\alpha(\beta\vec v) = (\alpha\beta)\vec v$ |
| 标量单位元 | $1 \cdot \vec v = \vec v$ |

这8条性质刻画了**向量空间**——在线性代数（第12学期）中正式介绍。

### 平行四边形法则

$\vec u + \vec v$ 是由 $\vec u$ 和 $\vec v$ 形成的平行四边形的对角线。

<Porta nivel="5" titulo="给5岁的孩子">

拿起纸上画的**小箭头**。它有**长度**并指向某个**方向**。这就是**向量**！

如果你想合并两个小箭头，**把第二个的尾部接到第一个的头部**。从最初起点到最终终点的小箭头就是**结果**。

<Porta nivel="10" titulo="给10岁的孩子">

向量是平面上的"**箭头**"。$(3, 4)$ 这个箭头向右3，向上4。**它的长度**：用勾股定理 → $\sqrt{9 + 16} = 5$。

**向量加法**：把箭头连接起来，"**头接尾**"。在坐标中，逐分量相加：$(2, 1) + (3, 4) = (5, 5)$。

**乘以标量**：拉伸或压缩箭头。$3 \cdot (2, 1) = (6, 3)$。

**出现在哪里**：
- **风速**：每小时50公里向东是向量。
- **拳击的一拳**：力有方向。
- **GPS**：位置是向量（纬度，经度）。
- **游戏手柄**：同时向上和向右 = 对角线向量。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

向量是平面上的"一支箭头"。它有：
- **模**（箭头长度）
- **方向**（所在直线）
- **指向**（直线哪一端）

在坐标中，它就是一个数对 $(a, b)$。$(3, 4)$ 是向右3、向上4的箭头。模：$\sqrt{9 + 16} = 5$。

**加法**是把一支箭头接在另一支的尾部。$(2,1) + (3,4) = (5, 5)$——逐分量相加。

**数乘**：拉伸或压缩。$3 \cdot (2, 1) = (6, 3)$——大三倍。

**单位向量**：除以模。对于 $(3, 4)$，模为5，单位向量为 $(3/5, 4/5)$。**方向**保持，**模**为1。

**出现在哪里**：物理中所有有方向的东西都是向量。速度、力、加速度、电场、角动量。在游戏中，角色位置；在 ML 中，词嵌入是100+维向量。

**与之前课程的联系**：
- 距离 $|\vec u - \vec v|$ 推广了点之间的距离。
- 直线的方向向量：$(x_2 - x_1, y_2 - y_1)$ 参数式。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

向量是所有应用数学中最重要的对象。在 $\mathbb{R}^n$ 中：

### 跨领域应用

- **力学**：力是向量，力的合成是合力（平行四边形法则）。平衡：$\sum \vec F = \vec 0$。
- **运动学**：速度 $\vec v$ 和加速度 $\vec a$ 是向量；$\vec v(t) = \vec v_0 + \vec a t$（向量合成）。
- **静力学**：平衡 = $\sum \vec F = \vec 0$——向量方程组。
- **电磁学**：电场/磁场；洛伦兹力 $\vec F = q\vec E + q\vec v \times \vec B$。
- **信号**：音频/图像样本是 $\mathbb{R}^n$ 中的向量，$n$ 巨大。
- **计算机图形学**：物体姿态 = 位置（3D）+ 朝向（3D）= 6D 向量。

### 分解

$\mathbb{R}^2$ 中每个向量都可以**唯一**分解为给定方向上的平行分量和垂直分量——斜面上力的分解的基础。推广为在任意子空间上的投影（第27课）。

### 线性组合与基

$\hat\imath, \hat\jmath$ 形成 $\mathbb{R}^2$ 的**标准基**。每个 $\vec v = v_1 \hat\imath + v_2 \hat\jmath$。更一般：两个**线性无关**的向量 $\vec a, \vec b$ 形成基；每个向量 $= \alpha\vec a + \beta\vec b$。

向量是**线性相关**的若 $\alpha\vec a + \beta\vec b = \vec 0$ 有非平凡解。在 $\mathbb{R}^2$ 中：相关 $\iff$ 平行 $\iff \det\begin{pmatrix} a_1 & b_1 \\ a_2 & b_2 \end{pmatrix} = 0$。

### 范数与三角不等式

$$|\vec u + \vec v| \leq |\vec u| + |\vec v|$$

等号 $\iff$ 同向平行向量。通过柯西-施瓦茨证明（第27课）。

### 极坐标

$\vec v = (r\cos\theta, r\sin\theta)$，$r = |\vec v|$，$\theta = \arctan(v_2/v_1)$（注意象限——使用 `atan2`）。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

$\mathbb{R}^2$ 配上加法和数乘是**实向量空间**最简单的例子——线性代数的基本范畴。

### 抽象视角

域 $\mathbb{F}$ 上的向量空间 $V$：满足8条公理的具有加法和数乘的集合。重要例子：

- $\mathbb{R}^n$——数列。
- $\mathbb{C}^n$——复向量（量子力学）。
- $L^2(\Omega)$——平方可积函数。
- 次数 $\leq n$ 的多项式。
- $m \times n$ 矩阵。

### 在 ML / 量化金融中的应用

- **嵌入**：word2vec、GloVe、BERT 将词元表示为 $\mathbb{R}^{300}$ 到 $\mathbb{R}^{12288}$ 中的向量。`king - man + woman ≈ queen` 等运算字面上就是向量代数。
- **量化金融**：收益向量是 $\mathbb{R}^N$ 的元素（$N$ = 资产数）。投资组合是权重向量。通过协方差范数衡量风险。
- **强化学习**：状态和动作是向量。策略网络映射 $\mathbb{R}^d \to \mathbb{R}^k$。
- **量子**：状态是复希尔伯特空间中的向量。算子是厄米矩阵。
- **计算机视觉**：$1024\times 1024$ 的 RGB 图像是 $\mathbb{R}^{3 \cdot 10^6}$ 中的向量。

### 推广：环上的模

当"标量域"只是一个环时（例如 $\mathbb{Z}$），就有了**模**。出现在代数几何、密码学（格）、纠错码中。

### 数值方面

在 IEEE 754 中，高度不平衡向量的求和（Kahan 求和）有抵消误差。BLAS 1 级（saxpy 等）以优化的汇编实现。对于 $n = 10^9$，朴素 `for` 与 BLAS 之间的差异是 100×。

### NumPy / SymPy

```python
import numpy as np
u = np.array([3, 4])
v = np.array([1, 2])
print(u + v)              # [4 6]
print(np.linalg.norm(u))  # 5
print(u / np.linalg.norm(u))  # 单位向量

# SymPy 符号计算
import sympy as sp
v = sp.Matrix([3, 4])
print(v.norm())           # 5
```

### 趣闻

哈密顿（1843）试图将复数推广到3D时发明了四元数。在3D中失败（无法做除法），但在4D中成功。如今，四元数是CG和机器人中表示3D旋转而避免万向节锁的标准。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**配方：实际操作向量。**

1. **将每个向量写成数对** $(v_1, v_2)$——清晰第一。
2. **加法/差/标量**：逐分量操作。
3. **模**：$\sqrt{v_1^2 + v_2^2}$。
4. **单位向量**：除以模。
5. **与 $x$ 的夹角**：$\theta = \text{atan2}(v_2, v_1)$（不是 $\arctan$——会丢失象限）。

### Wolfram / NumPy 工作流

```python
import numpy as np

# 初始位置和速度
r0 = np.array([0., 0.])
v  = np.array([3., 4.])

# t秒后
def posicao(t):
    return r0 + v * t

print(posicao(2))   # [6 8]
print(np.linalg.norm(v))  # 5

# 与水平方向的角度（弧度和度）
theta = np.arctan2(v[1], v[0])
print(np.degrees(theta))  # 53.13...
```

### 何时需要小心？

- **零向量 $(0,0)$**：模为0，方向不确定。归一化时小心。
- **`atan` vs `atan2`**：`atan(y/x)` 丢失象限。始终使用 `atan2(y, x)`。
- **不同尺度的求和**：高度向量（米）+ 时间向量（秒）没有意义——分析单位。
- **高维向量**：大多数对几乎正交（测度集中）。范数随 $\sqrt n$ 增长。
