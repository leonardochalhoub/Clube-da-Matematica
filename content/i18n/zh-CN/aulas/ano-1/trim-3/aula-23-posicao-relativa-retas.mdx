<Porta nivel="formal" titulo="判定标准与公式">

### 平行与垂直

给定直线 $r: y = m_r x + n_r$ 和 $s: y = m_s x + n_s$:

- **平行(不重合)**:$m_r = m_s$ 且 $n_r \neq n_s$。
- **重合**:$m_r = m_s$ 且 $n_r = n_s$。
- **相交**:$m_r \neq m_s$。
- **垂直**:$m_r \cdot m_s = -1$(假定两条都不是垂直线)。

含垂直直线的情形:$x = a$ 是垂直,$y = b$ 是水平。两条垂直直线相互平行;垂直与水平相互垂直。

### 直线之间的夹角

$$\tan\theta = \left|\frac{m_r - m_s}{1 + m_r m_s}\right|$$

(假定两条都不是垂直线。)

### 点到直线距离

点 $P_0 = (x_0, y_0)$ 与直线 $Ax + By + C = 0$:

$$d(P_0, r) = \frac{|A x_0 + B y_0 + C|}{\sqrt{A^2 + B^2}}$$

### 平行直线之间的距离

对于 $r: Ax + By + C_1 = 0$ 与 $s: Ax + By + C_2 = 0$:
$$d(r, s) = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

平面上的两条直线可能:

1. **平行**(斜率相同,永不相交)。
2. **垂直**(夹角为 90°)。
3. **相交于一点**(相交)。
4. **是同一条直线**(重合)。

**平行判定标准**:$m_r = m_s$ — 斜率相同。

**垂直判定标准**:$m_r \cdot m_s = -1$。看似奇怪,但来自将勾股定理应用于两次位移构成的三角形。

**例子**:$y = 2x + 1$ 与 $y = -x/2 + 5$ 互相垂直 ($2 \cdot (-1/2) = -1$)。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

实务中,**求解 2x2 线性方程组**就是求直线的交点。克拉默法则与消元法都从这里开始。

### 2x2 方程组与几何解读

$$\begin{cases} a_1 x + b_1 y = c_1 \\ a_2 x + b_2 y = c_2 \end{cases}$$

- **唯一解**:相交直线。$\det \neq 0$。
- **无穷多解**:重合直线。行列式 $= 0$ + 方程组相容。
- **无解**:不同的平行直线。行列式 $= 0$ + 方程组不相容。

</Porta>
