<Porta nivel="formal" titulo="判别准则与公式">

### 平行与垂直

给定直线 $r: y = m_r x + n_r$ 和 $s: y = m_s x + n_s$：

| 位置 | 判别准则（斜截式） | 判别准则（一般式） |
|---|---|---|
| 不重合的平行 | $m_r = m_s$，$n_r \neq n_s$ | $A_r/A_s = B_r/B_s \neq C_r/C_s$ |
| 重合 | $m_r = m_s$，$n_r = n_s$ | $A_r/A_s = B_r/B_s = C_r/C_s$ |
| 相交 | $m_r \neq m_s$ | $A_r B_s - A_s B_r \neq 0$ |
| 垂直 | $m_r \cdot m_s = -1$ | $A_r A_s + B_r B_s = 0$ |

含垂直直线的情形：$x = a$ 是垂直的，$y = b$ 是水平的。两条垂直线之间相互平行；垂直线与水平线相互垂直。

### 直线之间的夹角

$$\tan\theta = \left|\frac{m_r - m_s}{1 + m_r m_s}\right|$$

（假设没有垂直线。）

### 点到直线的距离

点 $P_0 = (x_0, y_0)$ 与直线 $Ax + By + C = 0$：

$$d(P_0, r) = \frac{|A x_0 + B y_0 + C|}{\sqrt{A^2 + B^2}}$$

### 平行直线之间的距离

对于 $r: Ax + By + C_1 = 0$ 和 $s: Ax + By + C_2 = 0$：
$$d(r, s) = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$

### 中垂线

到 $A$ 和 $B$ 等距的点的集合。是过 $\overline{AB}$ 中点且与之垂直的直线。

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

想象**两条火车铁轨**。它们并排前进，**永远不会相遇**。这就是**平行**。

现在想象两把**形成加号的尺子**，$+$。它们交叉形成**直角**。这就是**垂直**。

当两条直线既不平行也不垂直时，它们只是**在一个点上斜着**相交。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

平面上两条直线可以做4件事：

1. **平行**：永不相遇（斜率相同）。
2. **重合**：是同一条直线。
3. **相交**：交于一点。
4. **垂直**：交叉成90°。

**怎么判断？** 看斜率 $m$：
- $m_1 = m_2$：平行（如果 $n$ 也相同则重合）。
- $m_1 \cdot m_2 = -1$：垂直。
- 否则：交于某一点。

**具体例子**：$y = 2x + 1$ 和 $y = 2x - 5$ 是平行的（斜率都是2）。$y = 2x$ 和 $y = -x/2$ 是垂直的（$2 \cdot (-1/2) = -1$）。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

平面上两条直线可能：

1. **平行**（斜率相同，永不相交）。
2. **垂直**（成90°角）。
3. **交于一点**（相交）。
4. **是同一条直线**（重合）。

**平行准则**：$m_r = m_s$——斜率相同。

**垂直准则**：$m_r \cdot m_s = -1$。为什么？想想 $m_1 = \tan\alpha_1$。如果 $\alpha_2 = \alpha_1 + 90°$，则 $\tan\alpha_2 = -\cot\alpha_1 = -1/m_1$。因此 $m_1 m_2 = -1$。

**完整数值例子**：
- $r: y = 2x + 1$ 和 $s: y = -x/2 + 5$。乘积 $2 \cdot (-1/2) = -1$。**垂直**。
- 找交点：$2x + 1 = -x/2 + 5 \Rightarrow 5x/2 = 4 \Rightarrow x = 8/5$。代入：$y = 16/5 + 1 = 21/5$。交于 $(8/5, 21/5)$。

**点到直线的距离**至关重要——几乎所有后续的几何问题都用得上。公式 $|Ax_0 + By_0 + C|/\sqrt{A^2 + B^2}$ 来自将点投影到直线的法线方向。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

实际上，**2x2 线性方程组的求解**就是直线的相交。克拉默法则和高斯消元法从这里开始。

### 2x2 系统及几何解释

$$\begin{cases} a_1 x + b_1 y = c_1 \\ a_2 x + b_2 y = c_2 \end{cases}$$

| 情形 | 几何 | 行列式 |
|---|---|---|
| 唯一解 | 相交直线 | $\det \neq 0$ |
| 无穷多解 | 重合直线 | $\det = 0$ + 一致 |
| 无解 | 不重合的平行直线 | $\det = 0$ + 不一致 |

### 直接应用

- **线性回归**：最佳拟合直线垂直于误差最大方差方向。
- **CG / SVG**：多边形裁剪（Sutherland-Hodgman）通过 $Ax + By + C$ 的符号测试直线两侧。
- **机器人**：轨迹规划避开由半平面表示的障碍物。
- **GIS**：邻近性分析——每个点到道路网络的距离。

### $m_1 m_2 = -1$ 公式的证明

设 $\vec u_1 = (1, m_1)$ 和 $\vec u_2 = (1, m_2)$ 为方向向量。垂直 $\iff \vec u_1 \cdot \vec u_2 = 0 \iff 1 + m_1 m_2 = 0 \iff m_1 m_2 = -1$。

### 中垂线作为几何轨迹

到 $A = (x_A, y_A)$ 和 $B = (x_B, y_B)$ 等距的点 $(x, y)$：
$$(x - x_A)^2 + (y - y_A)^2 = (x - x_B)^2 + (y - y_B)^2$$

展开后，二次项相消，剩下直线 $\Rightarrow$ 过 $\overline{AB}$ 中点且与之垂直。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

直线之间的"正交性"概念可以推广到任何内积空间中**子空间的正交性**。

### 线性代数视角

直线 = 1D 仿射子空间。方向是向量 $\vec v$。两条直线 $r_1, r_2$ 平行 $\iff \vec v_1 \parallel \vec v_2$。垂直 $\iff \langle \vec v_1, \vec v_2 \rangle = 0$。

在 $\mathbb{R}^n$ 中，子空间 $V$ 的**正交补**是 $V^\perp = \{w : \langle v, w \rangle = 0 \forall v \in V\}$。$V \oplus V^\perp = \mathbb{R}^n$。Gram-Schmidt、QR 和 SVD 的基础。

### 在 ML / 量化中的应用

- **线性回归**：残差向量与 $X$ 的列空间正交（正交性原理）。形式上，$\hat\beta = (X^TX)^{-1} X^Ty$ 将 $y$ 正交投影。
- **PCA**：主成分是正交的——协方差矩阵的特征值。
- **量化金融**：投资组合再平衡使用预期收益在约束空间上的正交投影。
- **量子**：正交态 $\langle \psi | \phi \rangle = 0$ 对应于测量的"互斥性"。

### 通过投影计算距离

公式 $d = |Ax_0 + By_0 + C|/\sqrt{A^2+B^2}$ 是"点到直线"向量在法线 $(A, B)/\sqrt{A^2+B^2}$ 上投影的大小。推广到 $\mathbb{R}^n$ 中点到超平面的距离。

### 数值方面

注意线性方程组中几乎平行的直线——条件数 $\kappa(A) = \cot(\theta/2)$ 在 $\theta \to 0$ 时爆炸。高斯消元中的部分主元加上重新缩放可以缓解。

### SymPy

```python
import sympy as sp
from sympy import Line, Point
r = Line(Point(0, 0), Point(1, 2))
s = Line(Point(0, 5), Point(1, 7))
print(r.is_parallel(s))      # True
print(r.is_perpendicular(s)) # False
print(r.angle_between(s))    # 0
print(s.distance(Point(0, 0)))
```

<Porta nivel="pratica" titulo="实际应用">

**配方：3步分类两条直线。**

1. **比较斜率** $m_1, m_2$：
   - 相等？走平行/重合分支。
   - 乘积 $-1$？垂直。
   - 否则？相交——找交点。
2. **若平行**：比较 $n_1, n_2$。相等 $\Rightarrow$ 重合；否则不重合。
3. **求夹角**：用 $\tan\theta = |(m_1 - m_2)/(1 + m_1 m_2)|$。

### 完整工作流

```python
import sympy as sp
from sympy import Line, Point, pi
r = Line(Point(0, 1), Point(1, 3))   # y = 2x + 1
s = Line(Point(0, 5), Point(2, 4))   # y = -x/2 + 5
print(r.is_perpendicular(s))                   # True
print(r.intersection(s))                       # 交点
print(sp.deg(r.angle_between(s)))              # 90
print(s.distance(Point(0, 0)))                 # 原点到直线距离
```

### 何时需要小心？

- **垂直线**：$m$ 未定义——使用一般式。
- **几乎平行**：小扰动会大幅改变交点。实际上，计算 $\det$ 并在 $|\det| < \epsilon$ 时舍弃。
- **距离 vs 符号**：公式 $|Ax_0 + By_0 + C|$ 取绝对值。不取绝对值时，给出**直线的哪一侧**——在裁剪算法中很有用。
