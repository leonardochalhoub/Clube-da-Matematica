<Definicao titulo="笛卡尔平面">
$\mathbb{R}^2 = \{(x, y) : x, y \in \mathbb{R}\}$。每个点是一个有序对。$x$轴（水平）与$y$轴（垂直）相交于$O = (0, 0)$，将平面分成4个象限。
</Definicao>

### 距离

对于 $P = (x_1, y_1)$，$Q = (x_2, y_2)$：
$$d(P, Q) = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

### 中点

$$M = \left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}\right)$$

### 按比例 $k$ 分割线段

将 $\overline{PQ}$ 按 $PR/RQ = k$ 的比例分割：
$$R = \left(\frac{x_1 + k x_2}{1 + k}, \frac{y_1 + k y_2}{1 + k}\right)$$

### 面积计算

顶点为 $A = (x_1, y_1)$、$B = (x_2, y_2)$、$C = (x_3, y_3)$ 的三角形面积：
$$\text{面积} = \frac{1}{2} |x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2)|$$

等价于 $\frac{1}{2} |\det M|$，其中 $M$ 是向量 $\vec{AB}, \vec{AC}$ 构成的矩阵。

### 性质表

| 概念 | 公式 | 来源 |
|---|---|---|
| 距离 | $\sqrt{\Delta x^2 + \Delta y^2}$ | 勾股定理 |
| 中点 | $((x_1+x_2)/2, (y_1+y_2)/2)$ | 算术平均 |
| 重心 | $((x_1+x_2+x_3)/3, ...)$ | 3顶点平均 |
| 三角形面积 | $\frac{1}{2}\|x_1(y_2-y_3)+\ldots\|$ | 行列式 |
| 共线性 | 面积 $= 0$ | det $= 0$ |

想象客厅地板上的**寻宝游戏**。你说："向右走3步，向前走4步"。好啦，找到宝藏了！这两个数字就是宝藏的**坐标**。

要知道小鸡从客厅角落到宝藏需要**直线走**多少，只要想象一个**三角形**：向右3步，向前4步，对角线就是……5步！这就是**距离**。

<Porta nivel="10" titulo="给10岁的孩子">

**笛卡尔平面**就像无限的方格纸。每个小点有两个数字——向右多少（$x$）和向上多少（$y$）。

**两点之间的距离**：用**勾股定理**。想象你画一个直角三角形，两点作为相对的顶点。直角边是 $x$ 的差和 $y$ 的差。斜边就是距离！

**例子**：从点 $(1, 1)$ 到点 $(4, 5)$。$x$ 的差是3。$y$ 的差是4。距离 $= \sqrt{9 + 16} = \sqrt{25} = 5$。

**中点**：两点之间的中间点。只要**取坐标的平均**。$(2, 4)$ 和 $(6, 10)$ 的中点是 $(4, 7)$。

**用在哪里**：GPS、游戏地图、在地板上行走的机器人、Wi-Fi 覆盖范围。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

笛卡尔平面：每个点由2个数字描述——$x$坐标（水平）和$y$坐标（垂直）。**笛卡尔**于1637年统一了代数与几何——故有"笛卡尔"这一形容词。

**两点之间的距离**：应用勾股定理。如果有 $P = (1, 2)$ 和 $Q = (4, 6)$：

$$d = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

**中点**：取坐标的平均。$(1, 2)$ 和 $(4, 6)$ 的中点是 $(2{,}5,\ 4)$。

**三角形重心**：3个顶点的平均。对于 $A, B, C$：$G = \frac{A + B + C}{3}$——按坐标分量计算。

**为什么重要？** 因为笛卡尔平面是**连接代数与几何**的语言。直线变成方程。圆变成方程。抛物线变成方程。所有"几何的"东西都变成"代数的"——计算机就能计算了。

**物理联系**：在2D运动学中，物体的位置是一对 $(x(t), y(t))$。轨迹是这些点的集合。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

解析几何几乎是所有现代应用数学的基础。

### 推广

$\mathbb{R}^n$ 中的距离：
$$d(P, Q) = \sqrt{\sum_{i=1}^n (x_i - y_i)^2} = \|P - Q\|_2$$

这是**欧几里得范数**（$\ell_2$）。在机器学习中，它是 k-NN、k-means、t-SNE、神经网络（MSE 损失）使用的"标准距离"。

### 其他度量

| 度量 | 公式 | 用途 |
|---|---|---|
| 曼哈顿（$\ell_1$） | $\sum \|x_i - y_i\|$ | 街区，对异常值的鲁棒性 |
| 切比雪夫（$\ell_\infty$） | $\max \|x_i - y_i\|$ | 国际象棋，规划 |
| 马哈拉诺比斯 | $\sqrt{(x-\mu)^T \Sigma^{-1}(x-\mu)}$ | 多元相关性 |
| 测地线 | 大圆 | 球面上的距离 |
| 余弦 | $1 - \frac{\vec u \cdot \vec v}{\|u\|\|v\|}$ | NLP，推荐系统 |

### 有符号面积（有向）

公式 $\frac{1}{2}(x_1(y_2 - y_3) + \ldots)$ 在**没有绝对值**时给出**有符号面积**——若 $A, B, C$ 为逆时针方向则为正，反之为负。在计算机图形学（背面剔除）和 ConvexHull 等算法（Graham 扫描）中至关重要。

### 跨领域应用

- **计算机图形学**：每个像素是一个点，每个2D/3D对象是坐标的集合。光线追踪在 $\mathbb{R}^3$ 中工作。
- **GPS**：通过坐标 + 4颗卫星的三边测量定位。
- **机器人**：通过变换的复合实现机器人手臂的正向/逆向运动学。
- **土木工程**：地形测量产生 UTM 坐标（笛卡尔的变体）。
- **统计学**：散点图字面意义上就是笛卡尔平面，线性回归寻找最佳拟合直线。

### 勾股定理 → 距离的证明

构造一个直角三角形，水平直角边为 $|x_2 - x_1|$，垂直直角边为 $|y_2 - y_1|$。斜边是线段 $\overline{PQ}$。由勾股定理：$d^2 = (x_2-x_1)^2 + (y_2-y_1)^2$。证毕。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

"距离公式"只是 $\mathbb{R}^n$ 中**由标准内积诱导的范数**。

### 泛函分析视角

定义 $\langle u, v \rangle = \sum u_i v_i$。范数 $\|x\| = \sqrt{\langle x, x \rangle}$ 满足：

1. **正定性**：$\|x\| \geq 0$，$= 0 \iff x = 0$。
2. **齐次性**：$\|\alpha x\| = |\alpha| \|x\|$。
3. **三角不等式**：$\|x + y\| \leq \|x\| + \|y\|$（柯西-施瓦茨）。

**度量** $d(x, y) = \|x - y\|$ 继承这些性质。$(\mathbb{R}^n, d)$ 是**完备度量空间**——每个柯西序列都收敛。

### 推广：内积空间

在任意希尔伯特空间 $H$（例如 $L^2(\Omega)$——平方可积函数）中，$f$ 与 $g$ 之间的"距离"是 $\|f - g\|_2 = \sqrt{\int |f - g|^2}$。与 $\mathbb{R}^n$ **同样的形式**——信号工程、量子力学等。

### 在 ML / 量化金融中的应用

- **嵌入**：GPT/Claude 将词元表示为 $\mathbb{R}^{4096}$ 或 $\mathbb{R}^{12288}$ 中的向量。"语义相似度" $=$ 欧几里得距离或余弦距离。
- **聚类**：k-means 收敛到平方距离之和的局部极小值。仍是主力算法。
- **量化金融**：收益率序列之间的距离定义相关性；层次聚类（Mantegna 1999）使用 $\sqrt{2(1-\rho)}$ 作为资产之间的度量。
- **高斯-博内**：在流形上，局部度量是张量；测地线推广了"直线"。

### 历史趣闻

笛卡尔于1637年作为《方法论》的附录出版了《几何学》。费马独立地发展了平行的思想。现代记号 $(x, y)$ 是后来才出现的——笛卡尔使用字母表末尾的字母作为变量（故有 $x, y, z$）。

<Porta nivel="pratica" titulo="实际应用">

**配方：如何解决任何基本解析几何问题。**

1. **画出坐标轴系统**（即使是粗略的）。标出各点。
2. **识别问题要求**：距离？中点？面积？几何轨迹？
3. **应用相应公式**，不跳过步骤。
4. **用几何检验**（数量级是否合理？）。

### Wolfram Alpha / SymPy 工作流

```python
import sympy as sp
from sympy import Point, Segment, Triangle

A, B, C = Point(0,0), Point(4,0), Point(0,3)
T = Triangle(A, B, C)
print(T.area)          # 6
print(T.centroid)      # Point2D(4/3, 1)
print(T.perimeter)     # 12
```

使用 Wolfram Alpha 进行可视化：`distance from (1,2) to (4,6)` 会一步步生成公式。

### 何时需要小心？

- **坐标混乱**：注意 $\Delta y$ 的符号——公式中是平方，但几何解释不同。
- **重合点**：$d(P, P) = 0$。简单但常常是输入错误。
- **坐标轴尺度不同**：如果 $x$ 以千米计，$y$ 以米计，必须先归一化才能计算"几何"距离。
- **非欧度量**：在球形地球的 GPS 上，地图上的直线距离**不是**实际距离——使用 Haversine 公式。
