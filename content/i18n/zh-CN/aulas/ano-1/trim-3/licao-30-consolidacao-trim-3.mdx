<Porta nivel="10" titulo="给10岁的孩子">

我们学过的**复习**：
1. **笛卡尔平面**：坐标 $(x, y)$ 的点。
2. **直线**：方程 $y = mx + n$。
3. **圆**：$(x - a)^2 + (y - b)^2 = r^2$。
4. **圆锥曲线**：椭圆、抛物线、双曲线。
5. **向量**：平面上能相加的箭头。
6. **点积**：组合两个向量得到一个数。
7. **线性方程组**：同时几个方程。

**本课中**，把所有内容综合的题目。例如：找直线和圆的交点（第22+24课）。或求支撑负载的缆绳张力（向量+线性方程组）。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

本课巩固**整个第三学期**。每个题目综合多课的思想：

- **解析几何 + 直线**（第21-23课）：距离、中垂线、夹角。
- **圆锥曲线**（24-25）：圆、椭圆、抛物线在应用情境。
- **向量 + 点积**（26-27）：物理、ML、投影。
- **力学**（28）：力、功、斜面。
- **线性方程组**（29）：混合、桁架、经济。

**解综合题的策略**：
1. **识别涉及的主题**。
2. **几何上画出题目**。
3. **写出每部分的方程**。
4. **求解所得方程组**。
5. **解释并验证**答案。

**重要联系**：
- 直线 $\to$ 参数式 $\to$ 方向向量（第22、26课）。
- 圆 $\to$ 椭圆的特例（24、25）。
- 直线相对位置 $\to$ 2x2 方程组（23、29）。
- 力学功 $\to$ 点积（27、28）。

<Porta nivel="25" titulo="给工程系学生">

第三学期是**解析几何基础**，在所有工程学中再次出现：

### 跨领域联系

- **2D 运动学 + 抛物线**（第25、28课）：弹道轨迹。
- **静力学 + 线性方程组**（28、29）：桁架平衡是每个节点 $\sum F_x = 0, \sum F_y = 0$ 的线性方程组。
- **CG + 线性变换**（26、29）：GPU 中的旋转、缩放、平移。
- **ML + 向量**（26、27）：Transformer 中的嵌入和注意力。
- **计量经济学 + 方程组**（22、29）：线性回归是正规方程组。
- **量化金融 + 圆锥曲线**（25、27）：马科维茨在风险-收益平面上是双曲线。

### 笛卡尔方法综合

笛卡尔（1637）教导：**将任何几何问题转换为代数**，求解，然后翻译回来。本课是这个翻译在 2D 中的最后练习，进入第四学期 3D 之前。

### 接下来的学期都用这些

- **第4学期（立体几何）**：3D 距离、平面、球——直接扩展。
- **第5学期（函数）**：研究图形变换（$f(x-a) + b$）——向量平移。
- **第6学期（微积分）**：导数即斜率，积分即面积。一切都是加速的解析几何。
- **第12学期（正式线性代数）**：$\mathbb{R}^n$ 中的向量和方程组的严格处理。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

第三学期涵盖了英文文献中的"Coordinate Geometry / Analytic Geometry"——但学生可能没意识到自己正在学习**伪装的初等线性代数**。

### 形式联系

| 第三学期主题 | 线性代数概念 |
|---|---|
| 笛卡尔平面 | $\mathbb{R}^2$ 作为向量空间 |
| 直线方程 | $\mathbb{R}^2$ 中的超平面 |
| 2x2 方程组 | 线性映射 $\mathbb{R}^2 \to \mathbb{R}^2$ |
| 行列式 | 有向体积，平行四边形面积 |
| 点积 | 双线性形式/内积 |
| 方向向量 | 子空间的基 |
| 重合/平行 | 线性相关/秩 |
| 圆锥曲线 | 二次型 $\mathbf x^T A \mathbf x = 1$ |

### 在 ML / 量化中的应用

- **Transformer 中的注意力**：$\text{softmax}(QK^T/\sqrt d)V$。$QK^T$ 是点积矩阵。
- **马科维茨**（1952，[1990年诺贝尔奖](https://www.nobelprize.org/prizes/economic-sciences/1990/)）：最小化 $\mathbf w^T \Sigma \mathbf w$——广义点积。前沿是双曲线。
- **Black-Scholes**（1973，[1997年诺贝尔奖](https://www.nobelprize.org/prizes/economic-sciences/1997/)）：($\ln S$, $t$) 中的抛物型热方程。蒙特卡洛模拟在 $\mathbb{R}^N$ 中。
- **词嵌入**：相似度 $= \cos\theta$，$\mathbb{R}^{300}$ 中的向量。

### 第三学期后的路线图

掌握第三学期后的下一步：

1. **第4学期**：扩展到 $\mathbb{R}^3$（平面、球、3D 向量、向量积）。
2. **第6学期**：微分——导数作为切线的局部斜率。
3. **第12学期**：正式线性代数——抽象向量空间、特征值、SVD。

### 趣闻

自1969年以来，几乎所有**经济学诺贝尔奖**（[nobelprize.org](https://www.nobelprize.org/prizes/lists/all-prizes-in-economic-sciences/)）都以向量数学和线性/二次优化为基础。马科维茨（1990）、夏普（1990）、Black-Scholes/Merton（1997）、Fama（2013）、Hansen（2013）——都从 $\mathbb{R}^n$ 出发。

<Porta nivel="pratica" titulo="实际应用">

**配方：应对综合题。**

1. **读两遍**：明确题目要什么。
2. **列出涉及主题**。
3. **画草图**。
4. **选最直接的方法**（克拉默？代入？向量夹角？）。
5. **求解并验证**。

### Wolfram / SymPy 工作流

```python
import sympy as sp
from sympy.geometry import Point, Line, Circle, Triangle

# 典型问题示例：从外部点到圆的切线
C = Circle(Point(0, 0), 5)
P = Point(7, 0)
print(C.tangent_lines(P))      # 2条切线
print(P.distance(C.center)**2 - C.radius**2)  # 切线长度的平方

# 几何方程组
T = Triangle(Point(0,0), Point(6,0), Point(0,8))
print(T.incenter, T.incircle.radius)  # 内切圆中心和半径
```

### 何时需要小心？

- **混淆情境**：注意不要把椭圆公式套用到圆上（在 $a = b$ 时等价，但记号不同）。
- **单位**：功用焦耳，力用牛顿，距离用米——始终一致。
- **余弦的符号**：负的点积意味着钝角，不是错误。
- **检验数量级**：支撑1000 kg的缆绳张力应 ≥ 5000 N（重力）——远小于此就是错。
