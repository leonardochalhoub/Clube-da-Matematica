<Porta nivel="formal" titulo="力学中的向量">

### 基本原理

1. **静态平衡**：$\sum \vec F_i = \vec 0$（物体静止或匀速直线运动）。
2. **牛顿第二定律**：$\sum \vec F_i = m \vec a$。
3. **分解**：在倾角 $\theta$ 的斜面上，重力 $\vec g = -g\hat\jmath$ 分解为平行分量 $-g\sin\theta$（下滑）和法向分量 $-g\cos\theta$。

### 功与能量

- 功：$W = \vec F \cdot \vec d$。
- 功率：$P = \vec F \cdot \vec v$。
- 动能：$K = \frac{1}{2} m |\vec v|^2$。
- 动能定理：$W_{\text{total}} = \Delta K$。

### 向量运动学

- 位置：$\vec r(t)$。
- 速度：$\vec v(t) = d\vec r/dt$。
- 加速度：$\vec a(t) = d\vec v/dt$。

匀加速运动：
$$\vec r(t) = \vec r_0 + \vec v_0 t + \frac{1}{2} \vec a t^2$$

### 经典力

| 力 | 表达式 | 备注 |
|---|---|---|
| 重力 | $\vec P = m\vec g$ | 始终竖直向下 |
| 法向力 | $\vec N \perp$ 表面 | 对压缩的反作用 |
| 摩擦力 | $f = \mu N$ | 与运动方向相反 |
| 张力 | $T$ 沿绳 | 两端相等（理想绳） |
| 弹簧力 | $\vec F = -k\vec x$ | 胡克定律 |
| 向心力 | $F_c = mv^2/r$ | 指向中心 |

### 2D 中的平衡

3 个独立方程：$\sum F_x = 0$，$\sum F_y = 0$，$\sum M_O = 0$（关于任一点 $O$ 的力矩）。

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

想象两个男孩在拉一个**箱子**。一个**向右**拉，另一个**向左**拉，力相同。**箱子不动**——他们相互抵消了。

现在一个**向上**拉，另一个**向旁边**拉。箱子沿**对角线**移动，因为两个力像小箭头一样**相加**。

当所有的小箭头**加起来等于零**时，箱子**静止**。这就是**平衡**！

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

在物理学中，**每个力都有方向**。所以是向量。

**牛顿**发现：$\vec F = m\vec a$。力越大，加速度越大。质量越大，越难加速。

**平衡**：当所有力相互抵消（和为零）时，物体静止或以恒定速度做直线运动。

**斜面技巧**：重力向下拉，但只有**一部分**让物体沿斜面下滑。另一部分只是把物体压在斜面上。在30°斜面上，$\sin 30° = 0{,}5$——只有重力的一半在斜面上拉物体下滑。

**实际生活中**：桥梁缆索、电梯张力、汽车制动、篮球投掷——都是向量相加。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

在物理学中，**每个有方向的量都是向量**：力、速度、加速度、位移。向量的加法**遵守方向**：50N 向右 + 50N 向左 = 0N（零，相互抵消）。

**平衡**：物体静止 $\Rightarrow$ 所有力相互抵消。

**牛顿**：合力 = 质量 × 加速度。力越大 → 加速度越大。质量越大 → 越难加速。

**分解**是最有用的技巧：在30°斜面上，重力 $g$ 分为压在斜面上的 $g \cos 30°$（法向）+ 沿斜面下拉的 $g \sin 30°$（平行）。沿斜面的部分使物体下滑。

**斜抛运动**：$\vec v_0 = (v_0\cos\theta, v_0\sin\theta)$。加速度 $\vec a = (0, -g)$。抛物线轨迹：
- 飞行时间：$t = 2v_0\sin\theta / g$。
- 射程：$R = v_0^2 \sin(2\theta)/g$——在 $\theta = 45°$ 时最大。
- 最大高度：$h = v_0^2\sin^2\theta/(2g)$。

**与第27课的联系**：功 $W = \vec F \cdot \vec d$ 使用点积。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

向量力学是工程静力学（应用力学I）和动力学（应用力学II）的基础。所有以下分析都用到：

- **桁架**：每个节点的平衡（共面力系）。
- **梁**：弯矩和剪力的分布（沿长度方向的向量）。
- **机器**：机构分析、运动副。
- **车辆**：纵向和横向动力学（制动、转弯）。
- **结构**：建筑、桥梁、飞机、船舶——都通过 $\sum \vec F = 0$ + $\sum M = 0$ 求解。

### 平衡的向量分析

2D 问题中：2 个方程（$\sum F_x = 0, \sum F_y = 0$）+ 1 个力矩（$\sum M = 0$）。3D 中：6 个方程。

### 完整斜面问题

质量为 $m$ 的滑块在倾角 $\theta$、摩擦系数 $\mu$ 的斜面上：

| 力 | 平行分量 | 法向分量 |
|---|---|---|
| 重力 | $-mg\sin\theta$ | $-mg\cos\theta$ |
| 法向 | 0 | $+N$ |
| 摩擦 | $\pm \mu N$ | 0 |

法向平衡：$N = mg\cos\theta$。所以最大摩擦力 $\mu mg\cos\theta$。

滑块下滑 $\iff mg\sin\theta > \mu_s mg\cos\theta \iff \tan\theta > \mu_s$。

### 斜抛运动

$\vec r(t) = (v_0\cos\theta \cdot t, v_0\sin\theta \cdot t - \frac{1}{2}g t^2)$。直角坐标方程：
$$y = x\tan\theta - \frac{g x^2}{2 v_0^2 \cos^2\theta}$$

——**抛物线**（第25课）。

### 圆周运动

向心加速度：$\vec a_c = -\omega^2 \vec r$。大小 $a_c = v^2/r$。向心力必须由（绳、摩擦、重力）提供。

### 缆绳与张力

理想缆绳（零质量、不可伸长）：张力沿长度处处相同。理想滑轮：改变方向不改变大小。

### 与[诺贝尔奖](https://nobelprize.org)的联系

向量力学是轨道力学（牛顿1687）和相对论（爱因斯坦1916）的基础。引力波探测（LIGO，[2017年诺贝尔奖](https://www.nobelprize.org/prizes/physics/2017/)）使用应力-能量张量的向量分解。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

向量形式的经典力学是**拉格朗日/哈密顿力学**的特例——坐标变换不变的形式。

### 拉格朗日视角

拉格朗日量 $L(q, \dot q, t) = T - V$（动能减势能）。欧拉-拉格朗日方程：
$$\frac{d}{dt}\frac{\partial L}{\partial \dot q_i} - \frac{\partial L}{\partial q_i} = 0$$

笛卡尔坐标下的牛顿是特例。在曲线坐标（极坐标、球坐标、柱坐标）中，牛顿很丑陋；拉格朗日则很简洁。

### 在工程/量化中的应用

- **机器人**：机械臂的正向/逆向运动学是纯向量代数。四元数避免万向节锁。
- **航空航天**：火箭上的五种力（重力、推力、阻力、升力、侧力）。3D 向量分析。
- **量化**：多元衍生品（多维 Black-Scholes-Merton）使用向量代数。Greeks（delta、gamma）是向量/雅可比矩阵。
- **CFD**：纳维-斯托克斯方程是向量方程。通过有限体积离散是网格上的向量代数。

### 多粒子系统

对 $N$ 个粒子，相空间维度为 $6N$。在统计力学中，该空间上向量函数的积分定义温度、压力等。

### 数值方面

力学的 ODE 积分器：
- **显式欧拉法**：$\vec r_{n+1} = \vec r_n + \vec v_n \Delta t$。简单但不稳定。
- **Verlet/Leapfrog**：辛性，在哈密顿系统中保持能量。
- **Runge-Kutta 4**：高阶，但非辛性。

对于行星模拟、游戏、蛋白质折叠：Verlet > Euler。

### NumPy / SciPy

```python
import numpy as np
from scipy.integrate import solve_ivp

def projetil(t, y):
    g = 9.81
    return [y[2], y[3], 0, -g]   # dx/dt, dy/dt, dvx/dt, dvy/dt

v0, theta = 50, np.radians(60)
y0 = [0, 0, v0*np.cos(theta), v0*np.sin(theta)]
sol = solve_ivp(projetil, [0, 10], y0, dense_output=True, rtol=1e-9)
```

### 趣闻

牛顿1687年出版《自然哲学的数学原理》时未使用现代向量符号——所有公式都是几何的。现代向量代数始于19世纪末的 Gibbs/Heaviside。**《原理》的向量重述**是研究生经典的教学练习。

<Porta nivel="pratica" titulo="实际应用">

**配方：5步解决力学问题。**

1. **自由体图（DCL）**：画出物体及其上的所有力。
2. **选择坐标轴**：与表面（斜面）或运动方向对齐；简化分解。
3. **将每个力分解**到坐标轴。
4. **应用** $\sum F_x = ma_x$，$\sum F_y = ma_y$（如适用还有力矩）。
5. **求解方程组**。

### Wolfram / NumPy 工作流

```python
import numpy as np

# 带摩擦力的斜面滑块
m, theta, mu, g = 5, np.radians(30), 0.2, 9.81
F_grav_par = -m*g*np.sin(theta)
F_normal   =  m*g*np.cos(theta)
F_atrito   = -mu*F_normal*np.sign(F_grav_par)  # 与运动相反
F_paralela_total = F_grav_par + F_atrito
a = F_paralela_total / m
print(f"a = {a:.2f} m/s²")
```

### 何时需要小心？

- **静摩擦 vs 动摩擦**：系数不同；运动开始前是静摩擦，之后是动摩擦。
- **非惯性系**：在加速车辆参考系中，会出现"虚拟"力 $-m\vec a_0$。
- **可拉伸的绳**：张力沿绳变化。非理想。
- **符号**：坐标轴方向可能翻转符号。始终与 DCL 一致。
