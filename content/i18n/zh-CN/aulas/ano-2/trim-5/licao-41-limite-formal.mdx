<Porta nivel="formal" titulo="严格定义">

<Definicao titulo="ε-δ 极限（柯西 1821，Weierstrass 1872 形式化）">
$\lim_{x \to a} f(x) = L$ 意味：对每个 $\eps > 0$，存在 $\delta > 0$（依赖于 $\eps$）使得若 $0 < |x - a| < \delta$，则 $|f(x) - L| < \eps$。

注：$a$ 不必在 $f$ 的定义域中；重要的是 $a$ 附近的行为。
</Definicao>

### 通过 ε-δ 证明极限

1. 用 $|x - a|$ 表示 $|f(x) - L|$。
2. 任意 $\eps > 0$，找 $\delta = \delta(\eps)$ 使 $0 < |x - a| < \delta$ 时 $|f(x) - L| < \eps$。

### 模板例：$\lim_{x \to 2} (3x + 1) = 7$

$|3x + 1 - 7| = |3x - 6| = 3|x - 2|$。给定 $\eps > 0$，取 $\delta = \eps/3$。则 $|x - 2| < \delta \Rightarrow 3|x - 2| < \eps$。∎

### 单侧极限

- $\lim_{x \to a^+} f(x) = L$：$0 < x - a < \delta$（仅右侧）。
- $\lim_{x \to a^-} f(x) = L$：$0 < a - x < \delta$（仅左侧）。
- $\lim_{x \to a} f(x) = L \iff \lim_{x \to a^+} = \lim_{x \to a^-} = L$。

### 无穷极限

$\lim_{x \to a} f(x) = +\infty$：$\forall M > 0, \exists \delta > 0 : 0 < |x - a| < \delta \Rightarrow f(x) > M$。

$\lim_{x \to \infty} f(x) = L$：$\forall \eps > 0, \exists N : x > N \Rightarrow |f(x) - L| < \eps$。

### 4 种量化表

| 类型 | 量化 |
|---|---|
| $\lim_{x \to a} = L$ | $\forall \eps, \exists \delta$ |
| $\lim_{x \to a} = \infty$ | $\forall M, \exists \delta$ |
| $\lim_{x \to \infty} = L$ | $\forall \eps, \exists N$ |
| $\lim_{x \to \infty} = \infty$ | $\forall M, \exists N$ |

### 基本性质

- 和的极限 = 极限的和。
- 积的极限 = 极限的积。
- 商的极限 = 商，分母 $\neq 0$。

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

想象一个梯子接近上面的小星星。

我们想台阶**非常接近**星星。你说："我想离得近到能用手指碰到。" 我答："好，多走几步就行。"

你说："现在我想超级超级近。" 我答："那就再多走**几步**。"

对你想出的**任何**小距离，我能告诉你**走多少步**。这就是极限。

<Porta nivel="10" titulo="给10岁的孩子">

想象你在 GPS 上开车，目的地是地图上 $L$ 点。车永不到达，但越来越近。

**极限**：对你在目的地周围画的**任意**泡泡（不论多小），车**进入**泡泡并**保持**在内。不管泡泡 1 米还是 1 毫米——只需等够久。

希腊字母 ε（epsilon）是目的地周围**泡泡的大小**。δ（delta）是到达时间周围（$x$ 轴上）**泡泡的大小**。

规则：**你先选 ε**（任意小值），**我保证 δ**（$x$ 轴上的距离）使其工作。

若我总能给 δ 答案，极限**存在**。若某个 ε 破坏游戏，极限**不存在**。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

第19课你直观看过极限：$a_n$ 在 $n$ 增长时"接近" $L$。

这里形式化。柯西定义（1821）用两个希腊字母：ε（epsilon）和 δ（delta）。

思想：你想证 $f(x)$ 在 $x$ 接近 $a$ 时**非常接近** $L$。多近？**任意接近**。对任意 ε（"我想多近"），存在 δ（"$x$ 需在 $a$ 这距离内"）使得 $|x - a| < δ$ 时 $|f(x) - L| < ε$。

**对话**：
- "我想 $f(x)$ 距 $L$ $0{,}001$" → "好，$x$ 距 $a$ $0{,}0003$ 即可"
- "现在我想 $0{,}000001$" → "$0{,}0000003$"
- 对任何挑战，我都能给 δ 答案。

### 数值例：$\lim_{x \to 1} x^2 = 1$

$x = 1$ 附近的表：

| $x$ | $x^2$ | $|x^2 - 1|$ |
|---|---|---|
| $0{,}9$ | $0{,}81$ | $0{,}19$ |
| $0{,}99$ | $0{,}9801$ | $0{,}0199$ |
| $0{,}999$ | $0{,}998001$ | $0{,}001999$ |
| $1{,}001$ | $1{,}002001$ | $0{,}002001$ |
| $1{,}01$ | $1{,}0201$ | $0{,}0201$ |

为保 $|x^2 - 1| < \eps$，只需 $|x - 1| < \eps/3$（用 $|x + 1| \leq 3$ 当 $x$ 接近 1）。所以 $\delta = \min(1, \eps/3)$ 工作。

### 为何需要？

无 ε-δ 无法回答：$\lim$ 存在吗？值多少？怎么证？严格定义**裁决**所有这些。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

$\eps$-$\delta$ 定义是**严格微积分的基础**。在它之前（18世纪），牛顿和莱布尼茨用"无穷小"——比任意正数小但非零的量。实践有效但**逻辑可疑**（贝克莱 1734 攻击："已逝量的鬼魂"）。

柯西 1821 形式化（[**Cours d'analyse**](https://gallica.bnf.fr/ark:/12148/bpt6k90196z)）。Weierstrass 1872 完善。**所有现代分析从此始。**

### 模板证明：$\lim_{x \to 3} x^2 = 9$

要 $|x^2 - 9| < \eps$，即 $|x - 3| \cdot |x + 3| < \eps$。限制 $|x - 3| < 1$，则 $|x + 3| < 7$。所以 $|x - 3| < \eps/7$ 且 $|x - 3| < 1$ 即可。取 $\delta = \min(1, \eps/7)$。∎

### 夹逼定理（三明治）

若 $a$ 附近 $g(x) \leq f(x) \leq h(x)$ 且 $\lim g = \lim h = L$，则 $\lim f = L$。

**经典应用**：$\lim_{x \to 0} x \sin(1/x) = 0$（与 $\pm |x|$ 夹逼）。

### 重要极限（重访）

- $\lim_{x \to 0} \sin x / x = 1$（几何 + 夹逼）。
- $\lim_{x \to 0} (1 - \cos x)/x = 0$。
- $\lim_{x \to 0} (e^x - 1)/x = 1$。
- $\lim_{x \to \infty} (1 + 1/x)^x = e$。

### 不定式

$0/0$、$\infty/\infty$、$\infty - \infty$、$0 \cdot \infty$、$1^\infty$、$0^0$、$\infty^0$——需代数操作或洛必达（第7学期）。

### 物理与工程应用

- **瞬时速度**：$v(t) = \lim_{\Delta t \to 0} \Delta s/\Delta t$。
- **稳态响应**：$V(t) \to V_\infty$ 在 RC 电路。
- **稳定性**：线性系统稳定 $\Leftrightarrow$ $\lim_{t \to \infty} y(t) = 0$ 零输入。
- **离散化误差**：$\lim_{h \to 0} \frac{f(x+h) - f(x)}{h} - f'(x) = 0$ 验证数值方法。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

ε-δ 定义是**度量拓扑**的实例：度量空间 $(X, d)$ 中的极限。

### 度量推广

若 $f: (X, d_X) \to (Y, d_Y)$，则 $\lim_{x \to a} f(x) = L$ 意味：$\forall \eps > 0, \exists \delta > 0 : 0 < d_X(x, a) < \delta \Rightarrow d_Y(f(x), L) < \eps$。定义不依赖 $\mathbb{R}$。

### 拓扑（无度量）

一般拓扑空间，通过**滤子**或**邻域**定义极限：$f(x) \to L$ 当 $x \to a$ 若对每个 $L$ 邻域 $V$，存在 $a$ 邻域 $U$ 使 $f(U \setminus \{a\}) \subseteq V$。

### 序列刻画（海涅）

度量空间中：$\lim_{x \to a} f(x) = L \iff$ 对每个 $x_n \to a$ 且 $x_n \neq a$ 的序列，$f(x_n) \to L$。**关键**——将函数极限换成序列极限（更易操作）。

### 一致 vs 逐点收敛

$f_n \to f$ 逐点：$\forall x, \forall \eps, \exists N(\eps, x)$。$f_n \to f$ 一致：$\forall \eps, \exists N(\eps), \forall x$。**量词顺序改变一切**——一致保连续性，逐点不一定。

### 在泛函分析中的应用

- **巴拿赫空间**：线性算子 $T: X \to Y$ 的连续性由 $\sup_{\|x\| = 1} \|Tx\|< \infty$ 刻画。
- **谱理论**：$\lim_{n \to \infty} \|T^n x\|^{1/n}$ 定义谱半径。
- **施瓦茨分布**：通过 $C_c^\infty$ 中连续线性泛函的极限。

### 非标准：罗宾逊无穷小分析

ε-δ 可被严格的**无穷小**通过**超实数** $^*\mathbb{R}$ 替代（罗宾逊 1960）。$\lim_{x \to a} f(x) = L \iff f(a + \delta) \approx L$ 对所有非零无穷小 $\delta$。等价，但记号更接近莱布尼茨。

### 数值

实际计算中，ε-δ 变成**误差估计**：库（NumPy、MATLAB）用预定义 ε 的 `atol`、`rtol` 容差。迭代收敛只在误差低于预设 ε 时声明。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**配方：如何计算或证明极限。**

### 数值计算

1. **直接代入**。若 $f(a)$ 存在且 $f$ 连续，$\lim_{x \to a} f(x) = f(a)$。
2. **若得不定式（$0/0$、$\infty/\infty$）**，操作：
   - **因式分解约简**：$x \neq 2$ 时 $(x^2 - 4)/(x - 2) = x + 2$。
   - **乘共轭**：$(\sqrt{x+1} - 1)/x \cdot (\sqrt{x+1} + 1)/(\sqrt{x+1} + 1)$。
   - **用重要极限**：$\sin x \approx x$，$1 - \cos x \approx x^2/2$。
   - **变量替换**：$u = 1/x$，$u = x - a$ 等。
3. **洛必达**（第7学期）是最后选择。

### Wolfram Alpha / SymPy

```python
import sympy as sp
x = sp.symbols('x')
print(sp.limit(sp.sin(x)/x, x, 0))         # 1
print(sp.limit((1 + 1/x)**x, x, sp.oo))    # E
print(sp.limit(1/x, x, 0, '+'))            # +oo
print(sp.limit(1/x, x, 0, '-'))            # -oo
```

用于**核对**手算。

### 极限不存在

- **单侧极限不同**：$\lim_{x \to 0} 1/x$ 不存在（右 $+\infty$，左 $-\infty$）。
- **振荡**：$\lim_{x \to 0} \sin(1/x)$ 不存在（在 $-1$ 与 $1$ 之间振荡）。
- **不可去间断**：整数处的 $\lfloor x \rfloor$。

### ε-δ 证明（工程）

实际很少需要——工程用性质。但会证明**证实理解**。用纸笔做练习 41.21–41.30。

### 最重要的练习

完成**所有**30个练习。极限是后续一切（导数、积分、级数）的基础。不要跳。
