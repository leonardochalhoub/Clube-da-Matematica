### 单侧极限

<Definicao titulo="右极限">
$\lim_{x \to a^+} f(x) = L$ 若 $\forall \eps > 0, \exists \delta > 0 : 0 < x - a < \delta \Rightarrow |f(x) - L| < \eps$。
</Definicao>

<Definicao titulo="左极限">
$\lim_{x \to a^-} f(x) = L$ 若 $\forall \eps > 0, \exists \delta > 0 : 0 < a - x < \delta \Rightarrow |f(x) - L| < \eps$。
</Definicao>

### 通过单侧的存在性定理

$\lim_{x \to a} f(x) = L \iff \lim_{x \to a^+} f(x) = \lim_{x \to a^-} f(x) = L$。

### 无穷极限

$\lim_{x \to a} f(x) = +\infty$：$\forall M > 0, \exists \delta > 0 : 0 < |x - a| < \delta \Rightarrow f(x) > M$。

$-\infty$ 与单侧类似。

### 无穷处极限

$\lim_{x \to \infty} f(x) = L$：$\forall \eps > 0, \exists N : x > N \Rightarrow |f(x) - L| < \eps$。

$\lim_{x \to \infty} f(x) = +\infty$：$\forall M, \exists N : x > N \Rightarrow f(x) > M$。

### 量化表

| 类型 | 形式 |
|---|---|
| $\lim_{x \to a} = L$ | $\forall \eps, \exists \delta$ |
| $\lim_{x \to a^+} = L$ | $\forall \eps, \exists \delta$，$x > a$ |
| $\lim_{x \to a} = \infty$ | $\forall M, \exists \delta$ |
| $\lim_{x \to \infty} = L$ | $\forall \eps, \exists N$ |
| $\lim_{x \to \infty} = \infty$ | $\forall M, \exists N$ |

### $x \to \infty$ 时 $P/Q$ 的"直接"规则

| $\deg P$ vs $\deg Q$ | $\lim$ |
|---|---|
| $\deg P > \deg Q$ | $\pm\infty$ |
| $\deg P = \deg Q$ | 首项系数比 |
| $\deg P < \deg Q$ | $0$ |

<Porta nivel="5" titulo="给5岁的孩子">

想象一扇门。可**从街上来**（右侧）或**从家里来**（左侧）。

**单侧极限**就是看你从哪边来。

若每边都看到**相同**，"普通"极限存在。若看到**不同**，无极限——只有单侧的。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

想想**上楼梯** vs **下楼梯**。某些点取决于你从哪边来。

例：$f(x) = 1/x$。从右接近 $0$（$x = 0{,}1, 0{,}01$ 等），$f(x)$ 去 $+\infty$（升很多）。从左来（$x = -0{,}1, -0{,}01$ 等），去 $-\infty$（降很多）。

由于结果不同，$x \to 0$ 时**无极限**。但有**单侧极限**：

- $\lim_{x \to 0^+} 1/x = +\infty$
- $\lim_{x \to 0^-} 1/x = -\infty$

**金科玉律**：普通极限存在**当且仅当**两单侧均存在**且相等**。

**无穷处极限**：$x$ 去很远时 $f(x)$ 怎样？例如 $1/x$，$x \to +\infty$ 时去零。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

函数在某点两侧**行为不同**时，需**单侧**看。

### 经典例

| $f(x)$ | $\lim_{x \to 0^+}$ | $\lim_{x \to 0^-}$ | $\lim_{x \to 0}$ 存在？ |
|---|---|---|---|
| $1/x$ | $+\infty$ | $-\infty$ | 否 |
| $|x|/x$ | $1$ | $-1$ | 否（跳） |
| $\sqrt x$ | $0$ | 未定义 | 仅右 |
| $x \cdot \mathrm{sgn}(x)$ | $0$ | $0$ | 是（$= 0$） |

### 无穷极限：垂直渐近线

当 $\lim_{x \to a^\pm} f(x) = \pm \infty$ 时，存在**垂直渐近线** $x = a$（第47课）。

例：$1/(x-2)$ 在 $x = 2$ 有 AV。$\tan x$ 在 $x = \pi/2$。

### 无穷处极限：渐近行为

$\lim_{x \to \infty} f(x)$ 可：
- 有限 $L$ → 水平渐近线 $y = L$。
- $\pm \infty$ → 无限增/减。
- 不存在（如 $\sin x$ 振荡）。

### 多项式比表

对 $\lim_{x \to \infty} (a_n x^n + ...)/(b_m x^m + ...)$：

- $n < m$：$\to 0$。
- $n = m$：$\to a_n/b_m$。
- $n > m$：$\to \pm\infty$。

### 数值例

$\lim_{x \to \infty} (3x^2 + 1)/(x^2 - 1) = 3$（同次，比 3/1）。

$\lim_{x \to \infty} x/(x^2 + 1) = 0$（分母大）。

$\lim_{x \to \infty} (x^3 + 1)/x = +\infty$（分子大）。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

单侧与无穷极限是**系统分析**、**频率响应**、**稳定性**的关键。

### ε-δ 证明：$\lim_{x \to 0^+} 1/x = +\infty$

给 $M > 0$，取 $\delta = 1/M$。$0 < x < \delta = 1/M$ 时 $1/x > M$。∎

### "直接"规则证明

对 $\deg P = \deg Q = n$ 的 $P(x)/Q(x)$：
$$\frac{a_n x^n + ...}{b_n x^n + ...} = \frac{a_n + a_{n-1}/x + ...}{b_n + b_{n-1}/x + ...} \xrightarrow{x \to \infty} \frac{a_n}{b_n}.$$

### 系统应用

- **DC 增益**：线性系统 $H(0) = \lim_{s \to 0} H(s)$。$H = (s+1)/((s+2)(s+3))$ 时 $H(0) = 1/6$。
- **高频增益**：$\lim_{|s| \to \infty} |H(s)|$。低通去零。高通有限。
- **阶跃响应**：$y(\infty) = \lim_{t \to \infty} y(t) = $ 终值（终值定理）。
- **虚轴极点**：$H(s)$ 极点 $j\omega_0$ → $s \to j\omega_0$ 时 $|H(s)| \to \infty$（共振）。

### 序列渐近行为

- $\lim_{n \to \infty} n^k/a^n = 0$（$a > 1$）（指数胜多项式）。
- $\lim_{n \to \infty} n!/n^n = 0$（阶乘**不**胜 $n^n$）。
- $\lim_{n \to \infty} a^n/n! = 0$（阶乘胜指数）。

**增长层级**：$\log < \text{多项式} < \text{指数} < \text{阶乘} < n^n$。

### 符号函数与应用

控制中 $\mathrm{sgn}(x)$：在 $x = 0$ 不连续 → 在滑模控制中引入**抖振**。解：替换为 $\tanh(kx)$（光滑）。

### 无穷极限 vs 渐近线

注意：$\lim_{x \to \infty} \sin x$ **不存在**（振荡），但 $\lim_{x \to \infty} (\sin x)/x = 0$（夹逼）。水平渐近线 $y = 0$。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

单侧与无穷极限有深刻的拓扑解释。

### 扩展实直线拓扑

$\overline{\mathbb{R}} = \mathbb{R} \cup \{-\infty, +\infty\}$ 配适当拓扑（$+\infty$ 邻域为 $(N, +\infty]$）。无穷极限变 $\overline{\mathbb{R}}$ 中极限。

### 紧化

$\overline{\mathbb{R}}$ 紧（通过 $x/\sqrt{1+x^2}$ 同胚于 $[-1, 1]$）。序列极限通过 Bolzano-Weierstrass 在 $\overline{\mathbb{R}}$ 中总存在。

### Lim sup, lim inf

无极限的序列：$\limsup a_n = \lim_{n \to \infty} \sup_{k \geq n} a_k$，$\liminf$ 类似。在 $\overline{\mathbb{R}}$ 中总存在。$\lim$ 存在 ⇔ $\limsup = \liminf$。

### $\mathbb{R}^n$ 中方向极限

高维中，"单侧"推广为**方向**：$\lim_{t \to 0^+} f(a + t v)$ 对向量 $v$。多维极限存在性需所有方向重合。

经典反例：$f(x, y) = xy/(x^2 + y^2)$ 在 $(0, 0)$。方向极限存在（依赖方向），但**无全局极限**。

### 留数与极点

$f(z)$ 全纯且在 $z_0$ 有极点：$\lim_{z \to z_0} f(z) = \infty$ 在黎曼球面。留数 $\text{Res}_{z_0} f = \lim_{z \to z_0} (z - z_0) f(z)$ 刻画单极点。

### PDE 应用

- **边界条件**：有界域 PDE 解必须在边界有良定极限。
- **基本解**：格林函数源处无穷，远处有限。
- **拉普拉斯渐近**：$M \to \infty$ 时 $\int e^{Mf(x)} dx \sim$ $f$ 极大贡献。

### 拓扑中滤子与网

单侧极限是 $a$ 处**滤子**特例：右滤子 = $a$ 右邻域滤子。推广至一般拓扑。

### 数值

- **条件 vs 绝对收敛**：$\sum a_n$ 可收敛但 $\sum |a_n|$ 不——黎曼证重排改和。级数中方向极限。
- **解析延拓**：复极限定义级数不收敛处的 $\zeta(s)$、$\Gamma(s)$。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**配方：单侧与无穷极限。**

### 何时需单侧

- 函数在一点变化（分段定义）。
- 函数有**洞**或**渐近线**。
- 验证双侧极限存在性。

### $\lim_{x \to a^\pm}$ 策略

1. 考虑 $x = a + \eps$（小 $\eps > 0$）求 $a^+$。
2. 考虑 $x = a - \eps$（小 $\eps > 0$）求 $a^-$。
3. 计算并看是否重合。

### $\lim_{x \to \infty}$ 策略

1. **多项式比**：除以分母最高次。
2. **指数 vs 多项式**：指数总赢。
3. **对数**：增长慢于任何多项式。

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
print(sp.limit(1/x, x, 0, '+'))            # +oo
print(sp.limit(1/x, x, 0, '-'))            # -oo
print(sp.limit((3*x + 1)/(x + 5), x, sp.oo))  # 3
print(sp.limit(x**2 / sp.exp(x), x, sp.oo))   # 0（exp 胜）
```

### Wolfram Alpha

`limit (x^2-1)/(x-1) as x -> 1 from below` → 2。

### 极限不存在情形

- 单侧存在但不同（跳）。
- 某单侧 $\pm\infty$（但若两边重合可写 $\lim = \pm\infty$）。
- 振荡：$\sin(1/x)$ 在 0，$\sin x$ 在 $\infty$。

### 常见注意

- 忘检查单侧 → 不连续处出错。
- 混淆"去无穷"与"极限不存在"——一般，**无穷**极限严格意义上是**不存在**情形（极限不是数），但按惯例书写。
