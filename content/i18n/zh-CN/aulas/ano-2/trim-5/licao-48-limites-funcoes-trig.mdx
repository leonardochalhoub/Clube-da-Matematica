### 基本三角极限

| 极限 | 值 |
|---|---|
| $\lim_{x \to 0} \sin x / x$ | $1$ |
| $\lim_{x \to 0} \sin(kx)/x$ | $k$ |
| $\lim_{x \to 0} \tan x / x$ | $1$ |
| $\lim_{x \to 0} (1 - \cos x)/x$ | $0$ |
| $\lim_{x \to 0} (1 - \cos x)/x^2$ | $1/2$ |
| $\lim_{x \to 0} \sin^2 x / x^2$ | $1$ |
| $\lim_{x \to 0} \arcsin x / x$ | $1$ |
| $\lim_{x \to 0} \arctan x / x$ | $1$ |

### 有用恒等式

- $1 - \cos x = 2\sin^2(x/2)$
- $\sin(a + b) = \sin a \cos b + \cos a \sin b$
- $\cos(a + b) = \cos a \cos b - \sin a \sin b$
- $1 - \cos x = (1 - \cos^2 x)/(1 + \cos x) = \sin^2 x / (1 + \cos x)$

### 技巧

1. **替换 $u$**：$\sin(2x)/x = 2 \cdot \sin(2x)/(2x) \to 2$。
2. **恒等式**重写为基本形式。
3. **共轭**：$1 - \cos x = \sin^2 x / (1 + \cos x)$。
4. **夹逼**：$-1 \leq \sin x \leq 1$，乘以 $\to 0$ 的东西。

### $\lim (1 - \cos x)/x^2 = 1/2$ 证明

$\frac{1 - \cos x}{x^2} = \frac{2 \sin^2(x/2)}{x^2} = \frac{1}{2} \cdot \frac{\sin^2(x/2)}{(x/2)^2} \to \frac{1}{2} \cdot 1 = \frac{1}{2}$。

### $\lim \tan x / x = 1$ 证明

$\tan x / x = (\sin x / x) \cdot (1 / \cos x) \to 1 \cdot 1 = 1$。

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

角度**很小**时，它的**正弦**几乎等于角度本身。如 $\sin 0{,}01 \approx 0{,}01$。

故 $\sin x / x \to 1$——它们几乎相等。

且**1 减余弦**几乎等于**角度平方的一半**。小东西遵循简单规则。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

**小角度**（弧度）时：
- $\sin x \approx x$
- $\cos x \approx 1 - x^2/2$
- $\tan x \approx x$

由此得所有极限：
- $\sin(3x)/x \to 3$（因 $\sin(3x) \approx 3x$）
- $(1 - \cos x)/x^2 \to 1/2$（因 $1 - \cos x \approx x^2/2$）
- $\tan x / x \to 1$（因 $\tan x \approx x$）

**物理应用**：小振荡钟摆。实方程 $\ddot \theta = -(g/L) \sin \theta$。小 $\theta$ 时 $\sin \theta \approx \theta$，变 $\ddot \theta = -(g/L) \theta$——简单振子。

**无此近似**问题困难得多（涉及椭圆积分）。所以工程师爱小角度。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

$x \to 0$ 三角极限通过操作归为三种：

### 情形 1：$\sin(?)/?$

若能写成 $\sin(\text{某})/\text{某}$，去 1。变量替换 $u = $ 参数。

**例**：$\lim_{x \to 0} \sin(7x)/(3x) = (7/3) \lim_{x \to 0} \sin(7x)/(7x) = 7/3$。

### 情形 2：$(1 - \cos)/?$

用恒等式 $1 - \cos x = 2 \sin^2(x/2)$ 或共轭。

**例**：$\lim_{x \to 0} (1 - \cos x)/(x \sin x) = (x^2/2)/(x \cdot x) = 1/2$（用 $\sin x \approx x$）。

### 情形 3：$\tan, \sec, \cot$ 组合

重写为 $\sin, \cos$。

**例**：$\lim_{x \to 0} \tan(2x)/\sin(3x) = \lim (2x)/(3x) = 2/3$（近似）。

### 变量替换技巧

$\lim_{x \to a}$ 三角，用 $u = x - a$ 拉到零。

**例**：$\lim_{x \to \pi} \sin x / (x - \pi)$。设 $u = x - \pi$。则 $\sin x = \sin(u + \pi) = -\sin u$。结果：$\lim_{u \to 0} -\sin u / u = -1$。

### 小 $x$ 近似表

| 函数 | 近似（至 $x^3$） |
|---|---|
| $\sin x$ | $x - x^3/6$ |
| $\cos x$ | $1 - x^2/2$ |
| $\tan x$ | $x + x^3/3$ |
| $\arcsin x$ | $x + x^3/6$ |
| $\arctan x$ | $x - x^3/3$ |

用于快速求解。

### 挑战例

$\lim_{x \to 0} (\tan x - \sin x)/x^3$：

$\tan x - \sin x = \sin x (1/\cos x - 1) = \sin x \cdot (1 - \cos x)/\cos x$。

故：$\lim = (\sin x / x) \cdot ((1 - \cos x)/x^2) \cdot (1/\cos x) = 1 \cdot (1/2) \cdot 1 = 1/2$。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

三角极限在**导数**（$\sin', \cos', \tan'$）、**傅里叶级数**、**调和分析**中关键。

### 关键证明

**$\lim \sin x / x = 1$**：几何论证——三角形、扇形、三角形面积。夹逼推出。

**$(1 - \cos x)/x^2 \to 1/2$**：
$$\frac{1 - \cos x}{x^2} = \frac{(1-\cos x)(1+\cos x)}{x^2 (1+\cos x)} = \frac{\sin^2 x}{x^2(1+\cos x)} = \frac{\sin^2 x}{x^2} \cdot \frac{1}{1+\cos x} \to 1 \cdot \frac{1}{2} = \frac{1}{2}.$$

**$\arcsin x / x \to 1$**：$u = \arcsin x$，$x = \sin u$，$x \to 0$ 时 $u \to 0$。$\arcsin x / x = u/\sin u \to 1$。

### 与导数联系

- $(\sin x)' = \cos x$ 来自 $\lim_{h \to 0} (\sin(x+h) - \sin x)/h$。操作用 $\sin(x+h) = \sin x \cos h + \cos x \sin h$，则比 $= \sin x (\cos h - 1)/h + \cos x \sin h / h \to 0 + \cos x = \cos x$。
- $(\cos x)' = -\sin x$ 类似。
- $(\tan x)' = \sec^2 x$ 通过商法则。

### 泰勒级数（第7学期）

$$\sin x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n+1}}{(2n+1)!}, \quad \cos x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n}}{(2n)!}.$$

截断给小角度近似。紧集上一致收敛。

### 工程应用

- **力学小角度**：钟摆 $T = 2\pi\sqrt{L/g}$ 在 $\theta \lesssim 10°$ 有效。相对误差按 $\theta^2/16$ 增长。
- **高斯光学**（傍轴）：$\sin \theta \approx \theta$。薄透镜、折射。
- **线性化**：动力系统 $\dot x = f(x)$ 在平衡点用雅可比线性化。
- **调制**：$\cos(\omega t + \phi(t))$ 小 $\phi$ 展开到 AM 等价。

### 傅里叶分析

傅里叶系数 $a_n = \frac{1}{T}\int f(t) \cos(2\pi nt/T) dt$。极限 $\sum a_n^2$ 收敛（帕塞瓦尔）。三角极限出现在逐点收敛验证（Dini、Dirichlet）。

### 高级夹逼

$\lim_{x \to \infty} \sin x$ 不存在，但 $\lim_{x \to \infty} (\sin x)/x = 0$ 通过 $-1/x \leq \sin x/x \leq 1/x$ 夹逼。

$\lim_{x \to 0} x \sin(1/x) = 0$ 通过夹逼。**不**等于 $\lim x \cdot \lim \sin(1/x)$——后者不存在。夹逼无需此分解。

### AC 电路中

- 阻抗 $Z(j\omega) = R + jX$。模 $|Z| = \sqrt{R^2 + X^2}$。低/高频极限。
- $\sin(\omega t)$ 在 $\omega = 0$：去零。$\omega \to \infty$：振荡，无极限。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

三角极限是深刻**分析结构**的体现：$\sin, \cos$ 是带特定零点模式的整函数。

### Weierstrass 因式分解

$$\sin(\pi x) = \pi x \prod_{n=1}^\infty \left(1 - \frac{x^2}{n^2}\right).$$

取对数并求导，得 Mittag-Leffler 恒等式：
$$\pi \cot(\pi x) = \frac{1}{x} + \sum_{n=1}^\infty \frac{2x}{x^2 - n^2}.$$

与 $\zeta(2k)$ 联系。

### 函数刻画

$\sin, \cos$ 是 $\mathbb{R} \to \mathbb{R}$ 满足以下的唯一函数：
- $f(x+y) = f(x)g(y) + g(x)f(y)$，$g(x+y) = g(x)g(y) - f(x)f(y)$
- $f'(0) = 1$，$g(0) = 1$
- $f, g$ 连续

等价：$y'' + y = 0$ 配初值的解。

### 复分析

$e^{ix} = \cos x + i \sin x$（欧拉）。三角极限变 $z \to 0$ 时 $e^{iz}$ 极限。整 $\mathbb{C}$ 全纯。Hadamard 因式分解。

### PDE 应用

- **波动方程** $u_{tt} = c^2 u_{xx}$：解 $u(x, t) = f(x - ct) + g(x + ct)$ 通过分离变量涉 $\sin, \cos$。
- **自由薛定谔**：$\psi(x, t) = \int A(k) e^{i(kx - \omega t)} dk$，$\omega = k^2/2m$。波包通过 $\sin, \cos$。
- **Dirichlet BC 热方程**：解 $\sum b_n \sin(n\pi x/L) e^{-n^2 \pi^2 D t/L^2}$。

### 调和分析

Carleson 定理：$f \in L^2$ 的傅里叶级数 ae 收敛。证明依细致估计涉三角极限。

### Padé 近似

$\sin x$ 的 Padé 近似 $[m/n]$ 可比泰勒好。$\tan x \approx x(1 - x^2/15)/(1 - 2x^2/5)$ 误差比同阶截断泰勒小。

### 算子计算

- **矩阵函数**：$\sin A = (e^{iA} - e^{-iA})/(2i)$。任何矩阵良定。
- **谱映射**：spec($\sin A$) = $\{\sin \lambda : \lambda \in $ spec $A\}$。
- **应用**：$\ddot x = -\Omega^2 x$ 解 $x(t) = \cos(\Omega t) x_0 + \Omega^{-1} \sin(\Omega t) v_0$。

### 数值

- **参数缩减**：$\sin(10^{15})$ 要 mod $\pi$ 用 **Payne-Hanek**（高精度）。
- **CORDIC**：硬件友好，通过旋转迭代算 $\sin, \cos$。
- **数值抵消**：float32 中小 $x$ 的 $1 - \cos x$ 相对误差大。改用 `2 * sin(x/2)**2`。

<Porta nivel="pratica" titulo="实际应用">

**配方：三角极限。**

### 算法

1. **识别形式**：$\sin/?, \cos$、$(1 - \cos)/?, \tan/?$ 等。
2. 通过以下**重写**为基本形式：
   - 变量替换。
   - 恒等式 $\sin^2 + \cos^2 = 1$、$1 - \cos x = 2\sin^2(x/2)$ 等。
3. **数值或 SymPy 验证**。

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
# 参数线性
print(sp.limit(sp.sin(7*x)/x, x, 0))            # 7
# 正弦比
print(sp.limit(sp.sin(3*x)/sp.sin(5*x), x, 0))  # 3/5
# (1 - cos)/x^2
print(sp.limit((1 - sp.cos(x))/x**2, x, 0))     # 1/2
# 难
print(sp.limit((sp.tan(x) - sp.sin(x))/x**3, x, 0))  # 1/2
```

### 备忘录

$x \to 0$ 时：

| 形式 | 极限 |
|---|---|
| $\sin(ax)/x$ | $a$ |
| $\sin(ax)/\sin(bx)$ | $a/b$ |
| $\sin(ax)/\tan(bx)$ | $a/b$ |
| $(1 - \cos(ax))/x^2$ | $a^2/2$ |
| $\arcsin(ax)/x$ | $a$ |
| $\arctan(ax)/x$ | $a$ |
| $(\sin x)^2/x^2$ | 1 |
| $\sin(\sin x)/x$ | 1 |

### 不用以上情形

- $x \to a \neq 0$：变 $u = x - a$。
- 含 $1/x$ 参数（$\sin(1/x)$）：用振荡——夹逼。

### 常见注意

- $\sin(2x)/x \neq 2 \sin x / x$：应为 $2 \cdot \sin(2x)/(2x) \to 2$。
- $\sin(x^2)/x \neq \sin x \cdot x$：应为 $\sin(x^2)/x = x \cdot \sin(x^2)/x^2 \to 0$。
- $(1 - \cos x)/x \neq (1 - \cos x)/x^2$：第一个 → 0，第二个 → 1/2。
