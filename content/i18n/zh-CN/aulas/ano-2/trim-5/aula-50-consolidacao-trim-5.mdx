### 定理总表

| 定理 | 假设 | 结论 |
|---|---|---|
| 夹逼 | $g \leq f \leq h$，$\lim g = \lim h = L$ | $\lim f = L$ |
| TVI | $f \in C([a,b])$，$k$ 在 $f(a), f(b)$ 间 | $\exists c : f(c) = k$ |
| Weierstrass | $f \in C([a,b])$ | $f$ 取 max 与 min |
| Bolzano-Weierstrass | $(a_n)$ 有界 | 有收敛子列 |
| Heine-Cantor | $f \in C([a,b])$ | $f$ 一致连续 |
| 柯西 | $\mathbb{R}$ 中 $(a_n)$ 柯西 | 收敛 |
| 单调有界 | 递增上界 | 收敛 |

### 下一步

**导数（第6学期）**——定义为极限：
$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}.$$

第5学期的极限熟练度将**直接**用于此。

### 操作备忘录

| 形式 | 技巧 |
|---|---|
| 多项式 $0/0$ | 因式分解约简 |
| 含根 $0/0$ | 共轭 |
| 三角 $0/0$ | 重要极限 |
| 有理 $\infty/\infty$ | 除以最高次 |
| $1^\infty$ | $A^B = e^{B \ln A}$ |
| $0 \cdot \infty$ | 重写为商 |
| $\infty - \infty$ | 公因子、共轭 |

### 第6学期内容

**导数**：$f'(x) = \lim_{h \to 0} (f(x+h) - f(x))/h$。注意：**一个极限**。

**导数应用**：极值、画图、相关变化率、优化。一切建于极限上。

### 自测

若你能：
- 不用洛必达解 $\lim_{x \to 0} (e^{2x} - \cos x)/x$；
- 用 ε-δ 证 $x = 2$ 处 $\lim x^2 = 4$；
- 画 $f(x) = (x^2 - 1)/(x^2 - 4)$ 识别 2 AV 与 1 AH；
- 用 TVI 证 $e^x = 5x$ 在 $(0, \infty)$ 有解；
- 算 $\lim n!/n^n$；

你为第6学期准备好了。

<Porta nivel="25" titulo="给工程系学生">

第5学期是**分析基石**，微积分、微分方程、系统分析、优化、概率建于其上。

### 跨学期联系

**第6学期（导数）**：通过极限定义。所有规则（链式、积、商）通过极限证。

**第7学期（洛必达，级数）**：系统解决不定式。通过部分和极限的级数收敛。

**第8学期（积分）**：$\int = \lim$ 黎曼和。TFC 联系积分与导数。

**第9学期（应用）**：优化、ODE——边界条件与稳态中的极限。

**第10学期（数值）**：二分、牛顿、迭代法——收敛序列。

### 工程直接应用

- **线性系统稳定性**：左半平面极点 $\Leftrightarrow$ $\lim_{t \to \infty} h(t) = 0$。
- **频率响应**：$\omega \to 0, \infty$ 时 $H(j\omega)$ 极限给 Bode 图。
- **算法分析**：通过比的极限的复杂度 $O$、$\Omega$、$\Theta$。
- **可靠性**：$\text{MTBF} = \int_0^\infty R(t) dt$——通过极限的积分收敛。
- **量子力学**：稳态中 $\langle \hat A \rangle = \lim_n \langle \psi_n | \hat A | \psi_n \rangle$。

### 你需要**掌握**

- 快速极限计算（常见 30 秒内）。
- 不定式类型识别（5 秒内）。
- 流畅代数操作（因式分解、共轭、对数）。
- ε-δ 论证（至少经典情形）。
- 存在性中 TVI 与 Weierstrass 应用。

### 推荐工作流

任何问题：
1. **直接代入**。
2. **识别不定式**。
3. **操作**（因式分解、共轭、基本）。
4. 用 Wolfram/SymPy **确认**。
5. 量纲/物理**验证**。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

第5学期为你打开**泛函分析**与**拓扑**的大门，用于 ML、最优控制、量化金融、理论物理。

### 下一方向

- **严格实分析**（Rudin、Lebl）：开集、闭集、紧集。Bolzano-Weierstrass 作序列紧性。
- **一般拓扑**：非度量空间中极限。滤子、网、Moore-Smith 收敛。
- **泛函分析**：巴拿赫、希尔伯特空间。有界算子。紧。谱定理。
- **测度论**：$L^p$ 范数中极限。ae、依测度、控制（勒贝格）收敛。
- **复分析**：全纯函数极限。同一性原理、最大模。

### 涉及结构

- **度量、范数、半范**：多种拓扑共存。
- **强 vs 弱 vs $w^*$ 收敛**：优化与 PDE 中重要。
- **紧化**：$\beta \mathbb{N}$、$\overline{\mathbb{R}}$、黎曼球面。
- **完备化**：完备 vs 柯西。

### ML / RL 应用

- **SGD 收敛**：通过随机近似（Robbins-Monro、Kushner-Yin）分析。
- **训练稳定性**：Hessian 谱、Lipschitz 常数——通过算子极限分析。
- **泛化界**：PAC-Bayes、Rademacher 复杂度——类上一致收敛。
- **bandit / RL 收敛**：压缩映射、巴拿赫。

### 金融应用

- **Black-Scholes**：含 payoff BC 的 PDE。蒙特卡洛收敛 $\to$ 价格。
- **对冲收敛**：离散 delta 对冲的连续极限。
- **尾部风险**：通过分位数极限的 VaR、ES。

### 研究中

极限与拓扑出现在**所有**数学化学科。掌握者起飞。

### 推荐资源

- **Rudin, Principles of Mathematical Analysis**（"baby Rudin"）：经典。
- **Royden, Real Analysis**：勒贝格。
- **Reed-Simon, Methods of Mathematical Physics**：应用泛函分析。
- **Folland, A Guide to Advanced Real Analysis**：简明回顾。

### 最终备忘录

```
重要极限：
  sin x / x → 1                   (x → 0)
  (1 + 1/x)^x → e                 (x → ∞)
  (e^x - 1) / x → 1               (x → 0)
  (1 - cos x) / x^2 → 1/2         (x → 0)
  ln(1+x) / x → 1                 (x → 0)
  (a^x - 1) / x → ln a            (x → 0)

操作表：
  多项式 0/0     → 因式分解约简
  含根 0/0       → 共轭
  三角 0/0       → 重要极限
  有理 ∞/∞       → 除以最高次
  1^∞            → e^(B ln A)
  0·∞            → 重写
  ∞-∞            → 公因子

a 处连续：
  1. f(a) 已定义
  2. lim 存在
  3. lim = f(a)

不连续类型：
  可去：lim 存在，≠ f(a) 或 f(a) 未定义
  跳跃：单侧存在，不同
  无穷：±∞
  振荡：lim 不存在

增长层级：
  log n ≪ n^a ≪ a^n ≪ n! ≪ n^n
```
