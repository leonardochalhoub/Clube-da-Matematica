### 三个基本

1. $\displaystyle\lim_{x \to 0} \frac{\sin x}{x} = 1$（几何——夹逼）。
2. $\displaystyle\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$（定义 $e$）。
3. $\displaystyle\lim_{x \to 0} \frac{e^x - 1}{x} = 1$（由前者推出）。

### 有用变体（表）

| 极限 | 值 |
|---|---|
| $\lim_{x \to 0} \sin x / x$ | $1$ |
| $\lim_{x \to 0} \sin(kx)/x$ | $k$ |
| $\lim_{x \to 0} \tan x / x$ | $1$ |
| $\lim_{x \to 0} (1-\cos x)/x^2$ | $1/2$ |
| $\lim_{x \to 0} (1-\cos x)/x$ | $0$ |
| $\lim_{x \to 0} \arcsin x / x$ | $1$ |
| $\lim_{x \to 0} \arctan x / x$ | $1$ |
| $\lim_{x \to 0} (e^x - 1)/x$ | $1$ |
| $\lim_{x \to 0} (a^x - 1)/x$ | $\ln a$ |
| $\lim_{x \to 0} \ln(1+x)/x$ | $1$ |
| $\lim_{x \to 0} ((1+x)^a - 1)/x$ | $a$ |
| $\lim_{x \to \infty} (1 + a/x)^x$ | $e^a$ |
| $\lim_{x \to 0^+} x^x$ | $1$ |
| $\lim_{x \to 0^+} x \ln x$ | $0$ |

### 7 种常见不定式

$0/0$、$\infty/\infty$、$\infty - \infty$、$0 \cdot \infty$、$1^\infty$、$0^0$、$\infty^0$——通过操作或洛必达（第64课）解决。

### 技巧

- **因式分解约简**（多项式 $0/0$）。
- **共轭**（含根 $0/0$）。
- **三角**：用 $\sin x \approx x$（小 $x$）。
- **变量替换**：$u = 1/x$，$u = x - a$。
- **取对数**：$A^B = e^{B \ln A}$ 把 $1^\infty$、$0^0$ 转 $0/0$。

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

有三个**魔法**极限始终出现：

1. **$\sin x / x$** 在小 $x$ 时去**一**（小角度时正弦与角度几乎相等）。
2. **$(1 + 1/x)^x$** 在大 $x$ 时去特殊数 **$e$**（约 $2{,}71...$）。
3. **$(e^x - 1)/x$** 在小 $x$ 时也去**一**。

这三个像极限的**三个魔法乘法表**。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

得 $0/0$ 或 $\infty/\infty$ 时极限**可能为任何值**——视情况。但三个重要极限总给同样答：

1. **$\sin x / x \to 1$**：小角度时正弦几乎等于角度（弧度）。
2. **$(1 + 1/n)^n \to e$**：$e \approx 2{,}71828...$——出现在复利。
3. **$(e^x - 1)/x \to 1$**：由前者推出。

**实用**：你想算 $\lim_{x \to 0} \sin(5x)/x$。重写：$5 \cdot \sin(5x)/(5x)$。$x \to 0$ 时 $5x \to 0$，$\sin(5x)/(5x) \to 1$。答：$5$。

**复利窍门**：年利率 $r$，年复利 $n$ 次：$(1 + r/n)^n$。连续复利（$n \to \infty$）得 $e^r$。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

三个重要极限是**基础公式**，组合代数求解各种不定式。

### 公式 1：$\lim \sin x / x = 1$

**为何**：单位圆上 $x \in (0, \pi/2)$ 时 $\sin x < x < \tan x$。除以 $\sin x$：$1 < x/\sin x < 1/\cos x$。夹逼，$x/\sin x \to 1$，故 $\sin x / x \to 1$。

**变体**：$\lim \sin(kx)/x = k$（替换 $u = kx$）。

### 公式 2：$\lim (1 + 1/x)^x = e$

**为何**：定义 $e \approx 2{,}71828$。出现在连续复利、指数衰减、正态分布。

**变体**：
- $\lim (1 + a/x)^x = e^a$
- $\lim (1 + x)^{1/x} = e$（$x \to 0$）

### 公式 3：$\lim (e^x - 1)/x = 1$

**为何**：$e^x$ 在 $x = 0$ 的导数是 $e^0 = 1$。通过 $f'(0) = \lim (f(x) - f(0))/x$ 与导数联系。

**变体**：
- $\lim (a^x - 1)/x = \ln a$
- $\lim \ln(1+x)/x = 1$
- $\lim ((1+x)^a - 1)/x = a$

### 小 $x$ 近似表

| 函数 | 近似 |
|---|---|
| $\sin x$ | $x - x^3/6$ |
| $\cos x$ | $1 - x^2/2$ |
| $\tan x$ | $x + x^3/3$ |
| $e^x$ | $1 + x + x^2/2$ |
| $\ln(1+x)$ | $x - x^2/2$ |
| $(1+x)^a$ | $1 + ax$ |

**应用**：物理小角度（钟摆、光学）、连续复利、放射衰变。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

重要极限是基本函数**分析结构**——导数与泰勒级数——的结果。

### $\lim \sin x / x = 1$ 证明

几何论证（柯西 1821）：

$x \in (0, \pi/2)$，考虑面积 $\frac{1}{2}\sin x$ 的三角形、面积 $\frac{1}{2}x$ 的扇形、外面积 $\frac{1}{2}\tan x$ 的三角形。

$$\frac{\sin x}{2} < \frac{x}{2} < \frac{\tan x}{2} \Rightarrow \cos x < \frac{\sin x}{x} < 1$$

夹逼，$x \to 0^+$ 时 $\sin x / x \to 1$。由奇偶性，$x \to 0^-$ 也成立。∎

### $\lim (1 + 1/n)^n = e$ 证明

证 $a_n = (1 + 1/n)^n$ 单调递增有界，故收敛。极限**定义** $e$。

牛顿二项式：
$$\left(1 + \frac{1}{n}\right)^n = \sum_{k=0}^n \binom{n}{k} \frac{1}{n^k} = \sum_{k=0}^n \frac{1}{k!} \cdot \frac{n!}{n^k (n-k)!}$$

$n \to \infty$ 时每项 $\to 1/k!$，和去 $\sum_{k=0}^\infty 1/k! = e$。

### $\lim (e^x - 1)/x = 1$ 证明

用 $\lim_{x \to 0} (1 + x)^{1/x} = e$。取对数：$\ln((1+x)^{1/x}) = \ln(1+x)/x \to 1$。代入 $y = e^x - 1$（则 $x = \ln(1+y)$）：$(e^x - 1)/x = y/\ln(1+y) \to 1$。∎

### 工程应用

- **连续复利**：$V(T) = V_0 e^{rT}$，由 $\lim_n (1 + r/n)^{nT} = e^{rT}$ 推出。
- **放射衰变**：$N(t) = N_0 e^{-\lambda t}$——半衰期 $T_{1/2} = \ln 2 / \lambda$。
- **小角度近似**：钟摆 $\ddot \theta + (g/L) \sin \theta = 0$ 变 $\ddot \theta + (g/L) \theta = 0$（谐振子），约 10° 内有效。
- **BIBO 稳定性**：$\lim_{n \to \infty} h[n] = 0$ ⇒ 系统稳定。
- **角频率**：$\omega = \lim_{T \to 0} 2\pi / T$，周期与频率联系。

### 应用于导数

零处 $e^x$ 导数：
$$\frac{d}{dx} e^x \Big|_{x=0} = \lim_{h \to 0} \frac{e^h - e^0}{h} = \lim_{h \to 0} \frac{e^h - 1}{h} = 1.$$

故 $(e^x)' = e^x$ 处处（通过 $e^{x+h} = e^x e^h$）。

### 通过对数处理不定式

$\lim_{x \to 0^+} x^x$：$1^\infty$ 还是 $0^0$？重写：$x^x = e^{x \ln x}$，$x \ln x \to 0$（用夹逼/洛必达）。故 $x^x \to e^0 = 1$。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

重要极限是**深刻分析结构的体现**。

### $\sin x / x$ 与整函数

$\sin x / x$ 延伸为**整函数** $\text{sinc}(x)$，级数
$$\text{sinc}(x) = \prod_{n=1}^\infty \left(1 - \frac{x^2}{n^2 \pi^2}\right).$$
这是 $\sin$ 的 **Weierstrass 因式分解**。通过展开与 $\zeta(2) = \pi^2/6$ 联系。

### $e$ 与代数

$e = \exp(1)$，$\exp(z) = \sum z^n/n!$ 是唯一满足 $f' = f$、$f(0) = 1$ 的整函数。**微分方程的唯一刻画**。

### 代数推广

- **矩阵指数**：$e^A = \sum A^n/n!$。解 $\dot x = Ax$。
- **李群指数**：$\exp : \mathfrak{g} \to G$，代数到群的映射。
- **Gamma 函数**：$\Gamma(z) = \lim_n (n! n^z)/(z(z+1)\cdots(z+n))$ 通过乘积。

### 算子与符号微积分

- **影子微积分/Sheffer**：导数 $\partial$ 有"指数" $e^{a\partial}$ 作为 $a$ 平移。
- **Heaviside 微积分**：$D = d/dt$，$e^{-Dh}$ 是平移。重要极限出现在展开。

### 泛函分析

- $\sinh, \cosh, \sin, \cos$ 形成 $y'' = \pm y$ 解空间的**基**。
- $e^{i\omega t}$ 是 $d/dt$ 算子的**特征向量**——傅里叶分析基础。

### 严格级数证明

$\sin x = x - x^3/6 + x^5/120 - \ldots$ 故
$$\frac{\sin x}{x} = 1 - \frac{x^2}{6} + \frac{x^4}{120} - \ldots \to 1.$$

$e^x = 1 + x + x^2/2 + \ldots$ 故
$$\frac{e^x - 1}{x} = 1 + \frac{x}{2} + \frac{x^2}{6} + \ldots \to 1.$$

### 概率应用

- **特征函数**：$\varphi(t) = E[e^{itX}]$。$\varphi(0) = 1$，$\varphi'(0) = i E[X]$。
- **泊松定律**：$P(X = k) = e^{-\lambda} \lambda^k / k!$。二项极限 $(1 + \lambda/n)^n$。
- **斯特林公式**：$n! \sim \sqrt{2\pi n}(n/e)^n$。渐近近似。

### 重整化与物理

QFT 中，**zeta 正则化**：$\sum_{n=1}^\infty n = \zeta(-1) = -1/12$。重要极限出现在解析延拓。

### 数值

- 算法通过截断级数 + 参数缩减（CORDIC）算 $e^x$、$\sin x$。
- 小 $x$ 时 $(e^x - 1)/x$ 注意数值抵消：用 `expm1(x)`（C、Python、MATLAB 内置）。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**配方：使用重要极限。**

### 任何不定式的步骤

1. **识别类型**：$0/0$、$\infty/\infty$、$1^\infty$ 等。
2. **尝试重写**为重要极限形式：
   - $\sin(\text{某}) / \text{某}$ → 用 $\sin x / x$。
   - $(1 + \text{某})^{1/\text{某}}$ → 用 $e$。
   - $(e^{\text{某}} - 1) / \text{某}$ → 用 $(e^x - 1)/x$。
3. **变量替换**若需（$u = $ 内层参数）。
4. **数值或 SymPy 核对**。

### 典型情形

| 不定式 | 策略 |
|---|---|
| $0/0$ 三角 | 用 $\sin x \approx x$，$1 - \cos x \approx x^2/2$ |
| $0/0$ 指/对 | 用 $(e^x - 1)/x \to 1$，$\ln(1+x)/x \to 1$ |
| $1^\infty$ | $A^B = e^{B \ln A}$，操作 $B \ln A$ |
| $0^0$ 或 $\infty^0$ | 同：$A^B = e^{B \ln A}$，$\ln A \to -\infty$ 或 $+\infty$ |
| $\infty - \infty$ | 公因子、共轭或公分母 |

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
print(sp.limit(sp.sin(5*x)/x, x, 0))           # 5
print(sp.limit((1 + 2/x)**x, x, sp.oo))        # exp(2)
print(sp.limit((sp.exp(3*x) - 1)/x, x, 0))     # 3
print(sp.limit(x**x, x, 0, '+'))               # 1
print(sp.limit((1 - sp.cos(x))/x**2, x, 0))    # 1/2
```

### Wolfram Alpha

`limit (1 + 3/x)^x as x -> infinity` → $e^3$。

### 不用重要极限的情形

- $x \to a \neq 0$ 含三角表达式：可能直接代入。
- 与 $\sin, e, \ln$ 无关函数：其他技巧。

### 注意

- $\sin$ 的**参数**应等分母。$\sin(5x)/x \neq 1$——是 $5$。
- $(1 + 1/n)^n$ vs $(1 + 2/n)^n$ vs $(1 + 1/n)^{2n}$——结果不同。
