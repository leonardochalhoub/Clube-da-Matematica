<Porta nivel="formal" titulo="序列、收敛与定理">

<Definicao titulo="序列极限（ε-N）">
$(a_n) \to L$ 若：$\forall \eps > 0, \exists N \in \mathbb{N} : n \geq N \Rightarrow |a_n - L| < \eps$。
</Definicao>

### 关键定理

| 定理 | 陈述 |
|---|---|
| **唯一性** | 极限若存在，唯一 |
| **算术** | $\lim(a_n \pm b_n) = \lim a_n \pm \lim b_n$ 等 |
| **夹逼** | $a_n \leq b_n \leq c_n$，$\lim a_n = \lim c_n = L \Rightarrow \lim b_n = L$ |
| **单调有界** | 上界递增收敛 |
| **Bolzano-Weierstrass** | 有界序列有收敛子列 |
| **柯西** | $a_n$ 收敛 $\iff$ 是柯西 |

### 柯西

<Definicao titulo="柯西序列">
$(a_n)$ 是**柯西**若 $\forall \eps > 0, \exists N : m, n \geq N \Rightarrow |a_m - a_n| < \eps$。
</Definicao>

**$\mathbb{R}$ 中**：柯西 $\iff$ 收敛（完备性）。
**$\mathbb{Q}$ 中**：并非每个柯西都收敛（$a_n$ 逼近 $\sqrt 2$）。

### 单调 + 有界

- $(a_n)$ 递增上界 $\Rightarrow$ 收敛于 $\sup a_n$。
- $(a_n)$ 递减下界 $\Rightarrow$ 收敛于 $\inf a_n$。

### 子序列

$(a_{n_k})$ 是 $(a_n)$ 的子序列若 $n_1 < n_2 < \ldots$。
$\lim a_n = L \Rightarrow $ 每个子列 $\lim a_{n_k} = L$。

### 发散序列

- $a_n = (-1)^n$：在 $-1$ 与 $1$ 间振荡。子列收敛（$\pm 1$），但 $a_n$ 不。
- $a_n = n$：无界增长（$\to +\infty$）。
- $a_n = n \sin(n)$：不规则，无极限。

### 递归序列

$a_{n+1} = f(a_n)$，给定 $a_0$。若极限存在，是 $f$ 的**不动点**：$L = f(L)$。

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

想象数字队列：$1, 1/2, 1/3, 1/4, \ldots$——它们**变小**接近**零**。

极限是队列**去往何处**。若队列去某地，那地是极限。

某些队列哪里都不去（跳来跳去），无极限。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

**序列** = 数字列表：$a_1, a_2, a_3, \ldots$

**极限** $L$：序列**在无穷处到达**的数。例：
- $1, 1/2, 1/3, \ldots$ → 去**零**。
- $1, 4, 9, 16, \ldots$ → 去 **$\infty$**（增长）。
- $1, -1, 1, -1, \ldots$ → 哪儿不去（交替）。

**严格定义**：$a_n \to L$ 若你能**任意接近 $L$**，只要在序列中走得够远。对任何 $L$ 周围的"泡泡" $\eps$，有 $N$ 使 $a_N$ 之后的**所有**项在泡泡里。

**有用窍门**：若 $a_n$ 始终**增长**且有**顶**，它**收敛**！（单调有界。）甚至不必知极限——只需知存在。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

序列是**$\mathbb{N}$ 上的函数**。极限理论同第41课的 ε-δ，但用 $N$ 代 $\delta$。

### 序列重要性

- **求根的牛顿迭代**：$a_{n+1} = a_n - f(a_n)/f'(a_n)$。
- **级数** = 部分和的极限：$\sum a_n = \lim s_N$。
- **积分** = 黎曼和的极限。
- **概率**：大数定律 $\bar X_n \to E[X]$。

### 著名序列极限

| 序列 | 极限 |
|---|---|
| $1/n$ | $0$ |
| $(1 + 1/n)^n$ | $e$ |
| $n^{1/n}$ | $1$ |
| $n!/n^n$ | $0$ |
| $a^n$（$|a| < 1$） | $0$ |
| $a^n$（$a > 1$） | $\infty$ |
| $(-1)^n$ | 不存在 |

### 增长层级（重要！）

$$\log n \ll n^a \ll a^n \ll n! \ll n^n.$$

各远慢于下一个。算法复杂度中常用工具。

### 操作定理

**夹逼**：把 $b_n$ 夹在去同处的 $a_n$ 与 $c_n$ 间。

**例**：$\lim_{n \to \infty} \sin n / n$。$-1/n \leq \sin n / n \leq 1/n$。夹逼，$\to 0$。

**单调有界**：递增有界序列收敛——不必找极限，只需知存在！

**例**：$a_1 = 1$，$a_{n+1} = (a_n + 2/a_n)/2$。$a_1$ 后递减，下界 $\sqrt 2$。故收敛。极限 $L$ 满足 $L = (L + 2/L)/2$，即 $L = \sqrt 2$。

### Bolzano-Weierstrass

**每个有界序列有收敛子列**。存在性论证（紧性）有用。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

序列是**迭代算法的基础**——牛顿、梯度下降、EM、MCMC。

### ε-N 证明 $\lim 1/n = 0$

给 $\eps > 0$，取 $N = \lceil 1/\eps \rceil + 1$。则 $n \geq N \Rightarrow 1/n \leq 1/N < \eps$。∎

### 单调有界收敛证明

设 $(a_n)$ 递增上界。取 $L = \sup\{a_n\}$（完备性存在）。$\eps > 0$ 时 $L - \eps$ 非上界，故 $\exists N : a_N > L - \eps$。递增，$n \geq N$ 时 $a_n \geq a_N > L - \eps$。$a_n \leq L$，故 $|a_n - L| < \eps$。∎

### Bolzano-Weierstrass 证明（梗概）

有界序列 $(a_n) \subseteq [a, b]$。二分：某半含无限项。重复。构子列 $(a_{n_k})$，$a_{n_k}$ 在第 $k$ 半，长 $(b-a)/2^k$。柯西，故收敛。∎

### $\mathbb{R}$ 中柯西 ⇔ 收敛证明

$(\Rightarrow)$ 收敛即柯西：$|a_m - a_n| \leq |a_m - L| + |L - a_n| < \eps$。

$(\Leftarrow)$ 柯西即收敛：柯西 $\Rightarrow$ 有界 $\Rightarrow$ 有收敛子列（B-W）$\Rightarrow$ 整序列收敛于同极限（因柯西）。

### $\mathbb{Q}$ 中柯西序列

二分逼近 $\sqrt 2$ 的 $(a_n)$ 是 $\mathbb{Q}$ 柯西，但 $\mathbb{Q}$ 中**不收敛**（极限无理）。$\mathbb{R}$ 是 $\mathbb{Q}$ 按 $|\cdot|$ 度量的**完备化**。

### 算法应用

- **梯度下降**：$w_{n+1} = w_n - \eta \nabla f(w_n)$。$\eta < 2/L$（Lipschitz 常数）、$f$ 凸时收敛。
- **牛顿**：$x_{n+1} = x_n - f(x_n)/f'(x_n)$。简单根附近**二次**收敛。
- **海伦**（平方根）：$a_{n+1} = (a_n + N/a_n)/2$。$\sqrt N$ 二次。来自牛顿对 $f(x) = x^2 - N$。
- **EM**：单调增似然。收敛于临界点（不必全局最大）。

### 级数收敛

$\sum a_n$ 收敛 $\iff$ 部分和 $s_N = \sum_{n=1}^N a_n$ 形成收敛序列。

**判据**：
- 柯西：$\forall \eps, \exists N : m > n \geq N \Rightarrow |s_m - s_n| < \eps$。
- 比值（达朗贝尔）：$\lim |a_{n+1}/a_n| = L < 1 \Rightarrow$ 收敛。
- 根（柯西）：$\lim |a_n|^{1/n} = L < 1 \Rightarrow$ 收敛。

### 大数定律

$X_1, X_2, \ldots$ iid，$E[X] = \mu$，$V[X] < \infty$。则 $\bar X_n = (1/n) \sum X_i \to \mu$（依概率——弱；ae——强）。

### 动力系统应用

离散系统 $x_{n+1} = T(x_n)$。行为：不动点、循环、奇异吸引子（混沌）。**逻辑斯蒂映射** $x_{n+1} = r x_n (1 - x_n)$ 在 $r \approx 3{,}57$ 转混沌。

### $\mathbb{R}^n$ 中序列

按分量收敛。范数下：$\|x_n - L\| \to 0$。$\mathbb{R}^n$ 中等价（无穷维不等价）。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

序列在**泛函分析**、**度量空间**、**拓扑**中核心。

### 序列空间

- $\ell^p$ = $\sum |a_n|^p < \infty$ 序列。范数 $\|a\|_p = (\sum |a_n|^p)^{1/p}$ 巴拿赫。
- $\ell^\infty$ = 有界序列。$\|a\|_\infty = \sup |a_n|$ 巴拿赫。
- $c_0$ = $a_n \to 0$ 序列。$\ell^\infty$ 闭子空间。

### 弱 vs 强收敛

- **强**：$\|x_n - x\| \to 0$。
- **弱**：对所有 $f \in X^*$（对偶），$f(x_n) \to f(x)$。

有限维等价。$\ell^2$：标准正交基 $e_n$ 弱收敛于 0 但非强（$\|e_n\| = 1$）。

### 度量空间中柯西

$(X, d)$ **完备**若每柯西收敛。巴拿赫 = 完备赋范空间。希尔伯特 = 完备内积空间。

**完备化**：每个度量空间有唯一完备化。$\mathbb{R}$ 是 $\mathbb{Q}$ 完备化。$L^2$ 是 $C[a,b]$ 按 $L^2$ 范数完备化。

### 紧性

度量空间中：**序列紧** = 每序列有收敛子列。$\mathbb{R}^n$ 中等价于闭有界（Heine-Borel）。**无穷维**等价于闭、有界**且**完全有界（Riesz）。

### 拓扑

序列连续刻画在**度量空间**有效（海涅）。一般需网或滤子。

### 紧算子

$T : X \to Y$ **紧**若把单位球映为预紧集。子列收敛是证明关键。

### 谱论

紧自伴算子的特征值序列 $\lambda_n$：或有限，或 $\lambda_n \to 0$。谱定理通过特征向量基分解。

### 高级迭代方法

- **Aitken 加速**：$\hat a_n = a_n - (a_{n+1} - a_n)^2/(a_{n+2} - 2 a_{n+1} + a_n)$。加速线性收敛。
- **Romberg 积分**：积分中的 Richardson 外推。
- **GMRES、Krylov 子空间**：大型线性方程组。

### ML / RL 应用

- **Q 学习收敛**：适当 $\epsilon$-greedy 探索下 $Q_n \to Q^*$。
- **策略梯度收敛**：通过随机近似（Robbins-Monro）分析。
- **MCMC**：状态序列 $\to$ 平稳分布。通过遍历定理收敛。

### 范畴

范畴中极限推广序列极限。直接/反向极限、余极限等。同代数结构。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**配方：算序列极限。**

### 算法

1. **直接代入**：若 $a_n = f(n)$，$f$ 是函数，算 $\lim_{x \to \infty} f(x)$。
2. **递归**：假设极限 $L$，解 $L = f(L)$。
3. **算术**：拆已知部分。
4. **夹逼**：夹在已知间。
5. **单调有界**：证存在性，再算（或不算）。

### SymPy

```python
import sympy as sp
n = sp.symbols('n', positive=True, integer=True)
print(sp.limit(1/n, n, sp.oo))                # 0
print(sp.limit((1 + 1/n)**n, n, sp.oo))       # E
print(sp.limit(n**(1/n), n, sp.oo))           # 1
print(sp.limit(sp.factorial(n)/n**n, n, sp.oo))  # 0
```

### 增长层级（脱口而出）

$$\log n \ll n^a \ll a^n \ll n! \ll n^n.$$

用于求比：$\lim n!/2^n = \infty$，$\lim 2^n/n! = 0$，$\lim n^{100}/2^n = 0$。

### 递归序列

对 $a_{n+1} = f(a_n)$：

1. 验收敛（单调+有界，或巴拿赫压缩）。
2. 若收敛于 $L$，则 $L = f(L)$。

```python
import numpy as np
def heron(N, a0=1, n_iter=10):
    a = a0
    for _ in range(n_iter):
        a = (a + N/a)/2
    return a
print(heron(2))   # ~1.41421356 (sqrt 2)
```

### 极限不存在情形

- **振荡**：$(-1)^n$、$\sin n$。
- **发散**：$a_n \to \pm \infty$。
- **非单调非柯西**：混沌。

### 数值验证

实现并绘 $a_n$ vs $n$ 是**猜测**极限最快方式。然后证。

### 注意

- 递归极限设**存在**。无证明可能谬误。
- $(1 + 1/n)^n \neq 1$ 在无穷！$1^\infty$ 不定式。
- 序列**最终递增**可能非单调——证明小心。
