<Porta nivel="formal" titulo="渐近线——定义与计算">

### 垂直渐近线（AV）

<Definicao titulo="垂直渐近线">
直线 $x = a$ 是 $f$ 的 AV 若 $\lim_{x \to a^+} f(x) = \pm\infty$ 或 $\lim_{x \to a^-} f(x) = \pm\infty$。
</Definicao>

**出现处**：$f$ 有极点的点（分母为零、$\log 0$ 等）。

### 水平渐近线（AH）

<Definicao titulo="水平渐近线">
直线 $y = L$ 是 $f$ 的 AH 若 $\lim_{x \to +\infty} f(x) = L$ 或 $\lim_{x \to -\infty} f(x) = L$。
</Definicao>

**可有两条 AH**（一条 $+\infty$，一条 $-\infty$）：例 $\arctan x$ 有 $y = \pi/2$ 和 $y = -\pi/2$。

### 斜渐近线（AO）

<Definicao titulo="斜渐近线">
直线 $y = mx + b$（$m \neq 0$）是 AO 若 $\lim_{x \to \pm\infty} (f(x) - (mx + b)) = 0$。
</Definicao>

**计算**：
$$m = \lim_{x \to \pm\infty} \frac{f(x)}{x}, \quad b = \lim_{x \to \pm\infty} (f(x) - mx).$$

两极限均存在且 $m$ 有限非零，则有 AO。

### $P(x)/Q(x)$ 渐近线表

| 情形 | 渐近线 |
|---|---|
| $\deg P < \deg Q$ | AH $y = 0$ |
| $\deg P = \deg Q$ | AH $y = $ 首项系数比 |
| $\deg P = \deg Q + 1$ | AO（$y = $ 商） |
| $\deg P > \deg Q + 1$ | 无 AO（增长更快） |

### 其他函数

| 函数 | 渐近线 |
|---|---|
| $1/x$ | AV $x = 0$，AH $y = 0$ |
| $e^x$ | AH $y = 0$（$x \to -\infty$） |
| $\ln x$ | AV $x = 0$ |
| $\arctan x$ | AH $y = \pm\pi/2$ |
| $\tan x$ | AV $\pi/2 + k\pi$ |
| $1/\sin x$ | AV $k\pi$ |

### 一般步骤

1. **AV**：找 $f$ 未定义点（分母 = 0）。算单侧极限。
2. **AH**：算 $\lim_{x \to \pm\infty} f$。
3. **AO**：若无 AH，通过公式算 $m, b$。

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

想象一条路靠近**墙**但永不接触。墙是**渐近线**——函数接近但不触。

三种墙：
- **垂直**（站立）：函数在附近升降很多。
- **水平**（横躺）：函数末端平直。
- **斜**（倾斜）：函数远处变倾斜直线。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

把**图像拉远**：远处看像简单东西（水平、垂直或倾斜直线）。这种"简化"是**渐近线**。

**例**：

- $f(x) = 1/x$：
  - **AV**：$x = 0$——零附近函数爆。
  - **AH**：$y = 0$——大 $x$ 时 $1/x \to 0$。

- $f(x) = (x+1)/(x-1)$：
  - **AV**：$x = 1$。
  - **AH**：$y = 1$（$x \to \infty$ 时分母分子都 $\to \infty$，比 $\to 1$）。

- $f(x) = (x^2 + 1)/x$：
  - **AV**：$x = 0$。
  - **AO**：$y = x$（因 $(x^2+1)/x = x + 1/x$，$1/x \to 0$）。

**用于画图**：识别渐近线，标出，画曲线接近。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

渐近分析是画图的**标准工具**。四步：

### 画图步骤

1. **定义域**：$f$ 在哪定义？
2. **垂直渐近线**：定义域外点。算 $\lim_{x \to a^\pm} f$。
3. **水平/斜渐近线**：$\pm\infty$ 行为。
4. **特殊点**：零点、局部极值。

### 完整例：$f(x) = (x^2 + 1)/(x - 2)$

**1. 定义域**：$\mathbb{R} \setminus \{2\}$。

**2. AV**：$x = 2$。$\lim_{x \to 2^+} = +\infty$，$\lim_{x \to 2^-} = -\infty$。AV：$x = 2$。

**3. AO**：多项式除：$(x^2+1)/(x-2) = x + 2 + 5/(x-2)$。$x \to \pm\infty$ 时 $5/(x-2) \to 0$。AO：$y = x + 2$。

**4. 草图**：远处接近 $y = x + 2$，$x = 2$ 附近"爆"。

### 特殊情形

- **偶/奇函数**：渐近线对称镜像。$f(-x) = f(x)$ → $\pm a$ 处渐近线。
- **周期**：$\tan x$ 在每 $\pi/2 + k\pi$ 有 AV。
- **无穷振荡函数**：$\sin x$ 无 AH（$-1$ 与 $1$ 间振荡，不去单一值）。

### 有理函数记忆表

$f(x) = P(x)/Q(x)$，$\deg P = m$，$\deg Q = n$：

| 情形 | AH | AO |
|---|---|---|
| $m < n$ | $y = 0$ | 无 |
| $m = n$ | $y = a_m/b_n$ | 无 |
| $m = n+1$ | 无 | $y = $ 商 |
| $m > n+1$ | 无 | 无（但有多项式渐近行为） |

</Porta>

<Porta nivel="25" titulo="给工程系学生">

渐近分析在**伯德图**、**高频响应**、**稳定性**中核心。

### AO 严格计算

$f(x) = mx + b + g(x)$，$g(x) \to 0$。则 $m = \lim f/x$，$b = \lim (f - mx)$。

**证明**：
- $f/x = m + b/x + g(x)/x \to m$。
- $f - mx = b + g(x) \to b$。

每个极限**单独**必存在有限（$m$ 可零，则变 AH）。

### AV 通过奇异性

有理 $f = P/Q$：AV 在 $Q$ 的零点但**不**在 $P$ 的零点（否则可去）。

$f = \tan x = \sin x / \cos x$：AV 在 $\cos x$ 零点（即 $\pi/2 + k\pi$）。

### 动力系统应用

- **传递函数**：$H(s) = (s+1)/((s+2)(s+3))$。$s = -2, -3$ 处极点（复平面 AV）。$|s| \to \infty$ 时 $\to 0$。
- **Bode 图**：log-log 中 $|H(j\omega)|$ vs $\omega$。高低频斜率是 log 尺度上的**斜渐近线**。

**Bode 例**：$H(s) = K/(s+1)$。$\omega \to 0$：$|H| \to K$（低频 AH）。$\omega \to \infty$：$|H| \approx K/\omega$，即 $\log|H| \approx \log K - \log \omega$（斜率 $-20$ dB/dec 的 AO）。

### 控制应用

- **0 型**：$\lim_{s \to 0} H(s) = K$（有限）——斜坡稳态误差 = $\infty$。
- **1 型**：$\lim_{s \to 0} sH(s) = K$——阶跃误差 = 0，斜坡 = $1/K$。
- **2 型**：$\lim_{s \to 0} s^2 H(s) = K$——斜坡误差 = 0。

### 金融应用

Black-Scholes 在 $T \to 0$：期权 $\to \max(S - K, 0)$。**渐近**捕捉内在支付。$T \to \infty$：复杂，依赖波动率。

### 代数曲线

$x^2/a^2 + y^2/b^2 = 1$（椭圆）：无渐近线。
$x^2/a^2 - y^2/b^2 = 1$（双曲线）：渐近线 $y = \pm (b/a) x$。
$y^2 = x^3 - x$（三次）：渐近线依分支行为。

### 多维分析

$\mathbb{R}^n$ 中，渐近线变**渐近方向**或**渐近平面**。$f(x, y) \to L$ 当 $\|(x, y)\| \to \infty$，$z = L$ 是**水平渐近平面**。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

渐近分析是应用数学一整领域——远超基本渐近线。

### 大 O / 小 o 记号

$f(x) = O(g(x))$：邻域内 $|f| \leq C|g|$。
$f(x) = o(g(x))$：$\lim f/g = 0$。
$f \sim g$：$\lim f/g = 1$。

允许严格比较**增长尺度**。**斯特林**：$n! \sim \sqrt{2\pi n} (n/e)^n$。

### 拉普拉斯渐近分析

$\int e^{Mf(x)} dx$ 在 $M \to \infty$，贡献来自 $f$ 极大：
$$\int e^{Mf(x)} dx \sim \sqrt{\frac{2\pi}{M|f''(x_0)|}} e^{Mf(x_0)}.$$

应用：罕见事件概率、统计力学。

### 驻相方法

$\int e^{iMf(x)} g(x) dx$，贡献在 $f'(x) = 0$ 点（驻相）。应用：光学、QM。

### 渐近展开（不收敛）

$\Gamma$ 的斯特林级数：
$$\ln \Gamma(z) \sim (z - 1/2) \ln z - z + \ln \sqrt{2\pi} + \sum_{k=1}^\infty \frac{B_{2k}}{2k(2k-1) z^{2k-1}}.$$

发散，但**截断**给出极佳近似。

### PDE 渐近分析

- **边界层**（普朗特尔）：Navier-Stokes 在 $\nu \to 0$——奇异极限。
- **WKB 方法**在薛定谔方程：$\hbar \to 0$。
- **均匀化**：精尺度振荡系数 PDE。

### 格林函数与渐近线

无穷域椭圆 PDE 的格林解以特定速率衰减（如 $\mathbb{R}^n$ 中 $1/r^{n-2}$）。径向渐近线正是此。

### 数值

- **Padé 近似**：优化的有理近似——比泰勒更好捕捉渐近线。
- **重求和**：从发散级数提取信息的技术（Borel、Padé、切比雪夫）。
- **渐近加速**：Richardson、Aitken、Wynn。

### ML 应用

- **超参缩放**：LLM 的缩放律（Kaplan、Hoffmann、Chinchilla）是 $N, D, C$ 中渐近分析。
- **SGD 收敛**：依制度 $\|w_t - w^*\| \sim C/\sqrt t$ 或 $\sim e^{-\rho t}$。
- **泛化间隔**：通过集中 $O(\sqrt{d/n})$。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**配方：找 $f$ 的渐近线。**

### 算法

1. **定义域**：识别外部点。
2. **AV**：每个 $a$ 在定义域外，算 $\lim_{x \to a^\pm} f$。无穷则 $x = a$ 是 AV。
3. **AH**：$\lim_{x \to +\infty} f$ 与 $\lim_{x \to -\infty} f$。有限则各为 AH。
4. **AO**：若无 $\pm\infty$ 处 AH，算 $m = \lim f/x$ 与 $b = \lim (f - mx)$。

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
f = (x**2 + 1)/(x - 2)
# AV
print(sp.limit(f, x, 2, '+'))     # +oo
print(sp.limit(f, x, 2, '-'))     # -oo
# AO
m = sp.limit(f/x, x, sp.oo)        # 1
b = sp.limit(f - m*x, x, sp.oo)    # 2
print(f"AO: y = {m}x + {b}")       # y = x + 2
```

### 画图（步骤）

$f(x) = x^3/(x^2 - 1)$：

1. **定义域**：$\mathbb{R} \setminus \{-1, 1\}$。
2. **AV**：$x = -1$ 与 $x = 1$。
3. **AH**：$\lim_{x \to \infty} f = \infty$，无 AH。
4. **AO**：$f/x = x^2/(x^2-1) \to 1$，$f - x = x^3/(x^2-1) - x = x/(x^2-1) \to 0$。AO：$y = x$。
5. **$f$ 零点**：$x = 0$。
6. **对称**：$f(-x) = -f(x)$，奇。

按此画。

### 无标准渐近线的情形

- **$f$ 无穷处振荡**：$\sin x$、$x \sin x$。
- **增长快于线性**：$x^2$ 无 AO。
- **分支**：$\sqrt{x^2 + 1}$ 在 $+\infty$ 与 $-\infty$ 有**不同** AO（$y = x$ 与 $y = -x$）。

### 多渐近线

$f(x) = \arctan x$ 有**两** AH：$y = \pi/2$（$x \to +\infty$）与 $y = -\pi/2$（$x \to -\infty$）。
