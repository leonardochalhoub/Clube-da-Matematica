<Porta nivel="formal" titulo="TVI 与变体">

<Definicao titulo="中间值定理（TVI）">
若 $f \in C([a, b])$ 且 $k$ 在 $f(a)$ 与 $f(b)$ 之间，则存在 $c \in (a, b)$ 使 $f(c) = k$。
</Definicao>

**特例**（$k = 0$）：$f(a) \cdot f(b) < 0 \Rightarrow \exists c \in (a, b) : f(c) = 0$。

### 应用 1：二分法（算法）

若 $f(a) f(b) < 0$：
1. $m = (a+b)/2$。
2. 若 $f(m) = 0$，停。
3. 若 $f(a) f(m) < 0$，在 $[a, m]$ 重复。
4. 否则在 $[m, b]$ 重复。

**收敛**：每次迭代误差减半，$|c - m_n| \leq (b-a)/2^n$。

### 应用 2：不动点

若 $g \in C([a, b])$ 且 $g([a, b]) \subseteq [a, b]$，则 $\exists c$ 使 $g(c) = c$。对 $f(x) = g(x) - x$ 用 TVI。

### 应用 3：方程解的存在性

$\cos x = x$、$e^x = 2x$、$\tan x = x$ 由 TVI 有解。

### 通过完备性证明

设 $S = \{x \in [a, b] : f(x) < k\}$（设 $f(a) < k < f(b)$）。$S \neq \emptyset$（$a \in S$）有上界。取 $c = \sup S$。由连续性与上确界性质，$f(c) = k$。

### 为何需要连续性

反例：$f(x) = \mathrm{sgn}(x)$ 在 $[-1, 1]$。$f(-1) = -1$，$f(1) = 1$，但 $f$ 永不取 $0{,}5$。失败因 $f$ 在 0 不连续。

### TVI vs TVM

**TVI**：**中间值**存在性（$f$ 连续）。
**TVM**（中值定理，第56课）：**中间导数**存在性（$f$ 可微）。

不要混淆。TVM 需可微；TVI 仅需连续。

连续函数同样：从**负**到**正**时，某时它**为零**。

<Porta nivel="10" titulo="给10岁的孩子">

**中间值定理**：连续函数从**负**始到**正**终，某时穿过**零**。

想想**温度计**：早上 $-2°$，下午 $+5°$。某时必标 $0°$（冰点）——不可避免，若温度连续（不跳）。

**应用**：找方程解所在。例 $f(x) = x^3 - x - 1$。算：
- $f(1) = 1 - 1 - 1 = -1$（负）
- $f(2) = 8 - 2 - 1 = 5$（正）

由于变号，**1 与 2 之间有根**！TVI 保证。

**二分法**：取中 $1{,}5$。$f(1{,}5) = 0{,}875 > 0$。所以根在 $1$ 与 $1{,}5$ 之间。取中 $1{,}25$。如此**对半分**。每次更近。

<Porta nivel="15" titulo="给15岁的青少年">

TVI 是**保证连续函数方程解存在性**的定理。

### 实用形式

证 $g(x) = h(x)$ 在 $[a, b]$ 有解：
1. 定义 $f(x) = g(x) - h(x)$。
2. 验 $f$ 在 $[a, b]$ 连续。
3. 若 $f(a) f(b) < 0$，有解。

### 例

**$\cos x = x$** 在 $[0, \pi/2]$：
- $f(x) = \cos x - x$。
- $f(0) = 1 > 0$，$f(\pi/2) = -\pi/2 < 0$。
- 有解。（约 $0{,}739$。）

**$e^x = 2x$**：
- $f(0) = 1 > 0$，$f(1) = e - 2 \approx 0{,}72 > 0$。这里不变号。
- $f(2) = e^2 - 4 \approx 3{,}39 > 0$。大值时 $e^x$ 胜。
- $f(-1) = e^{-1} + 2 \approx 2{,}37 > 0$。
- 大负 $x$ 时 $e^x \to 0$、$-2x \to \infty$，故 $f \to \infty$。但 $x = 0$ 时 $f = 1$。仍不变。结论：$e^x = 2x$ 在小 $\mathbb{R}$ 无解；可能在特定 $x$ 有。

（用图重析。）

### 二分法：典型应用

| 迭代 | $a$ | $b$ | $m$ | $f(m)$ | 下个 |
|---|---|---|---|---|---|
| 0 | 1 | 2 | $1{,}5$ | + | $[1;\ 1{,}5]$ |
| 1 | 1 | $1{,}5$ | $1{,}25$ | $-$ | $[1{,}25;\ 1{,}5]$ |
| 2 | $1{,}25$ | $1{,}5$ | $1{,}375$ | + | $[1{,}25;\ 1{,}375]$ |
| 3 | $1{,}25$ | $1{,}375$ | $1{,}3125$ | $-$ | $[1{,}3125;\ 1{,}375]$ |

每次迭代误差减半。10 次后误差 $< (b-a)/1024$。

### 奇次多项式

**总有实根**！$P(x) = a_n x^n + \ldots$，$n$ 奇，$a_n > 0$：
- $\lim_{x \to +\infty} P = +\infty$
- $\lim_{x \to -\infty} P = -\infty$
- 由 TVI，根存在。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

TVI 是数值分析与存在性方法的**基础**定理。

### 通过完备性证明

$f \in C([a, b])$，$f(a) < 0 < f(b)$。定义 $S = \{x \in [a, b] : f(x) < 0\}$。

- $S$ 非空：$a \in S$。
- $S$ 有界：$\subseteq [a, b]$。
- 由 $\mathbb{R}$ 完备性，$\exists c = \sup S$，$c \in [a, b]$。

断：$f(c) = 0$。
- 若 $f(c) < 0$：连续性，$f$ 在 $c$ 邻域 $< 0$，矛盾 $c = \sup$。
- 若 $f(c) > 0$：连续性，$f$ 在 $c$ 邻域 $> 0$，但 $c = \sup S$，矛盾。

故 $f(c) = 0$。∎

### 为何 TVI **在 $\mathbb{Q}$ 失败**

$f(x) = x^2 - 2$ 在 $\mathbb{Q}$ 连续，$f(0) = -2 < 0$，$f(2) = 2 > 0$。经典 TVI 在 $(0, 2)$ 有根。但根是 $\sqrt 2 \notin \mathbb{Q}$。**无完备性，TVI 失败。**

### 工程应用

- **隐含波动率**（Black-Scholes 课）：方程 $C(\sigma) = C_{\text{mkt}}$。$C$ 关于 $\sigma$ 连续单调。TVI + 二分法 30 次内得 $\sigma$。
- **非线性电路平衡**：$f(V) = i_{\text{in}}$。通过牛顿（快）或二分法（鲁棒）求解。
- **稳定性分析**：特征多项式 $p(s)$ 在右半平面有根？TVI + Routh-Hurwitz。
- **传递多项式**：通过 Routh 找零点，Jury 找极点（数字控制）。

### 二分法：误差分析

$n$ 次迭代后 $|c - m_n| \leq (b-a)/2^{n+1}$。误差 $< \eps$ 需 $n > \log_2((b-a)/\eps) - 1$。

**收敛保证**：TVI 成立时二分法总收敛。其他方法（牛顿）可能发散。

### Brouwer 不动点（通过 TVI 1D）

若 $g \in C([a, b])$ 且 $g([a, b]) \subseteq [a, b]$，$g$ 有不动点。证明：定义 $f(x) = g(x) - x$。$f(a) = g(a) - a \geq 0$（$g(a) \in [a, b]$）。$f(b) = g(b) - b \leq 0$。由 TVI，$\exists c : f(c) = 0$，即 $g(c) = c$。∎

**一般 Brouwer**（$n$ 维）：每个连续 $g : B^n \to B^n$ 有不动点。非平凡——经济均衡（纳什）、自适应控制基础。

### 隐方程

$F(x, y) = 0$，$F$ 连续，$F(x_0, y_0) = 0$。每个邻近 $x$，通过对 $F(x, \cdot)$ 用 TVI 求 $y$。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

TVI 是更一般的**拓扑性质**：连续映射下连通的像连通。

### 拓扑推广

$f : X \to Y$ 连续，$X$ 连通 $\Rightarrow$ $f(X)$ 连通。$\mathbb{R}$ 中连通是区间。所以 $f([a, b])$ 是区间，含 $f(a), f(b)$，含其间所有值。

### TVI 与全序

通过 $\sup$ 的证明依赖 $\mathbb{R}$ 完备**且**线序。$\mathbb{R}^n$（$n \geq 2$）无全序——字面 TVI 不成立，但有代数拓扑推广。

### 推广：Brouwer 不动点

$g : B^n \to B^n$ 连续有不动点。拓扑证明：
- 设无不动点。定义 $r(x) = $ 从 $x$ 到 $g(x)$ 投影到 $\partial B^n = S^{n-1}$ 的射线。
- $r$ 是 $B^n$ 到 $S^{n-1}$ 的连续收缩，矛盾 $\pi_{n-1}(B^n) = 0 \neq \pi_{n-1}(S^{n-1})$。

### 经济应用（纳什均衡）

有限博弈中纳什均衡通过 Brouwer（或 Kakutani）存在。"最佳响应"函数连续（多值），有不动点。1994 年[纳什诺贝尔经济学奖](https://www.nobelprize.org/prizes/economic-sciences/1994/summary/)。

### PDE 应用

- **Schaefer 不动点定理**：扩展到巴拿赫紧算子。
- **Schauder 不动点**：巴拿赫凸紧集。
- **Navier-Stokes 解**：通过适当空间不动点存在。

### 复 TVI

$f$ 全纯无字面 TVI（$\mathbb{C}$ 无序），但有**开映射**：非常 $f$ 把开映为开。结合**辐角原理**通过围道积分数零点。

### 算法

- **二分法**：$O(\log_2 1/\eps)$——鲁棒、慢。
- **牛顿**：$O(\log \log 1/\eps)$（二次）——快但可能发散。
- **Brent**：智能组合，两者优。
- **TOMS 748 算法**（最先进）：二分 + 插值。

### Python 数值

```python
from scipy.optimize import brentq, bisect
import numpy as np
f = lambda x: x**3 - x - 1
print(brentq(f, 1, 2))   # ~1.3247
print(bisect(f, 1, 2))   # ~1.3247
```

<Porta nivel="pratica" titulo="实际应用">

**配方：应用 TVI。**

### 证根存在

1. **识别 $f$**（可能为差 $g - h$）。
2. 在 $[a, b]$ **验证连续性**。
3. **算 $f(a)$ 与 $f(b)$**。确认异号。
4. **结论**：$(a, b)$ 中有根。

### 二分法：实现

```python
def bissect(f, a, b, tol=1e-9):
    assert f(a) * f(b) < 0, "TVI 不适用"
    while b - a > tol:
        m = (a + b) / 2
        if f(m) == 0:
            return m
        if f(a) * f(m) < 0:
            b = m
        else:
            a = m
    return (a + b) / 2

print(bissect(lambda x: x**3 - x - 1, 1, 2))   # ~1.3247
```

### 二分法适用情形

- **连续但不可微**：牛顿不行。
- **鲁棒性**比速度重要。
- **多模**：二分法在区间找**一**根。

### TVI 不给唯一性

TVI 保**存在**，不保**唯一**。$f(x) = x^3 - x$ 在 $[-2, 2]$ 有三根。唯一性需单调。

### TVI 不帮的情形

- 不连续函数：TVI 不成立。
- $f(a) f(b) > 0$（同号）：TVI 不检测——**但仍可能有根**（未检测）。
- 重根：$f(a) = f(b) = 0$ 不算（无严格变号）。

### 不动点迭代

```python
def ponto_fixo(g, x0, tol=1e-9, max_iter=100):
    x = x0
    for _ in range(max_iter):
        x_new = g(x)
        if abs(x_new - x) < tol:
            return x_new
        x = x_new
    return x

print(ponto_fixo(np.cos, 0.5))   # ~0.739 (cos x = x)
```

**注意**：收敛需不动点处 $|g'(c)| < 1$（巴拿赫-皮卡）。
