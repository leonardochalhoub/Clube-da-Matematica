<Porta nivel="formal" titulo="运算性质">

设 $\lim_{x \to a} f(x) = L$，$\lim_{x \to a} g(x) = M$。则：

| 运算 | 结果 |
|---|---|
| $\lim (f \pm g)$ | $L \pm M$ |
| $\lim (cf)$，$c$ 常数 | $cL$ |
| $\lim (fg)$ | $LM$ |
| $\lim (f/g)$，$M \neq 0$ | $L/M$ |
| $\lim f^n$，$n \in \mathbb{N}$ | $L^n$ |
| $\lim \sqrt[n]{f}$ | $\sqrt[n]{L}$（定义时） |
| $\lim |f|$ | $|L|$ |

### 复合

若 $\lim_{x \to a} g(x) = b$ 且 $f$ 在 $b$ 连续，则 $\lim_{x \to a} f(g(x)) = f(b)$。

**注意**：无连续性，复合可能失败。反例：$g(x) = 0$ 常数，$f$ 在 0 处可去间断。

### 夹逼定理（三明治）

若 $a$ 邻域（除 $a$ 外）$g(x) \leq f(x) \leq h(x)$ 且 $\lim g = \lim h = L$，则 $\lim f = L$。

### 经典应用

$\lim_{x \to 0} x \sin(1/x) = 0$ 由夹逼：$-|x| \leq x \sin(1/x) \leq |x|$ 且 $\lim |x| = 0$。

### 何时不行

- **0/0、∞/∞**：不定式，需操作。
- **分母 → 0**：可能 $\pm \infty$ 或不存在。
- **与不连续 $f$ 复合**：需谨慎。

</Porta>

<Porta nivel="5" titulo="给5岁的孩子">

若 Pati 有 3 苹果，João 有 2，**合起来**他们有 5。和的极限是极限的和：两件事各自接近，合起来就求和。

乘法一样：3 乘 2 是 6——直接。

除法要小心：除以**零**坏掉。所以分母不能趋于零。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

极限对运算**讲礼貌**。若你知两件事去哪儿，你也知和、积、商去哪儿。

**简单规则**：

- 两个去 $L$ 与 $M$ 的和去 $L + M$。
- 乘常数：常数"提出来"。
- 积：$L \cdot M$。
- 商：$L / M$，**前提 $M$ 非零**（除以零禁止）。

**三明治窍门**：你被两个相同的东西夹住，你也只能相同。若 $g \leq f \leq h$ 且 $g, h$ 去同地方，$f$ 同行。

例：$-1 \leq \sin(1/x) \leq 1$。乘以小正 $x$：$-x \leq x \sin(1/x) \leq x$。$x \to 0$ 时两边都 $\to 0$，所以 $x \sin(1/x) \to 0$。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

极限对加、积、商"行为良好"——前提**两个极限存在**且分母非零。

### 实用配方

1. **直接代入**。给数则极限。
2. **若得不定式（$0/0$、$\infty/\infty$）**，操作：
   - 因式分解约简。
   - 乘共轭。
   - 用重要极限（$\sin x / x \to 1$）。
3. **夹逼**（三明治）：若你能把函数夹在两个已知极限的函数间，得函数的极限。

### 常见操作表

| 不定式 | 技巧 |
|---|---|
| 多项式 $0/0$ | 因式分解 |
| 含根 $0/0$ | 共轭 |
| 有理 $\infty/\infty$ | 除以最高次 |
| $0 \cdot \infty$ | 重写为 $0/0$ 或 $\infty/\infty$ |
| $1^\infty$、$0^0$ | $A^B = e^{B \ln A}$ |

### 综合例

$\lim_{x \to 2} \frac{x^2 - 4}{x - 2}$：$0/0$ 不定式。因式分解：$\frac{(x-2)(x+2)}{x-2} = x + 2 \to 4$。

### 夹逼为何强大

你不需要知道**怎么算** $f$——只需把它**夹在**两个易函数间。在级数和振荡中常见。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

代数性质把极限计算变成**代数**：你操作表达式而非每次用 ε-δ 证明。

### 和性质证明

给定 $\lim f = L$，$\lim g = M$。对 $\eps > 0$：
- 存在 $\delta_1$：$|x - a| < \delta_1$ 时 $|f - L| < \eps/2$。
- 存在 $\delta_2$：$|x - a| < \delta_2$ 时 $|g - M| < \eps/2$。
- 取 $\delta = \min(\delta_1, \delta_2)$。则 $|f + g - (L + M)| \leq |f - L| + |g - M| < \eps$。∎

### 积证明

更微妙——用 $|fg - LM| \leq |f||g - M| + |M||f - L|$ 和 $f$ 局部有界。

### 夹逼：证明

给定 $\eps$，选 $\delta$ 使 $|g - L|, |h - L| < \eps$。因 $g \leq f \leq h$：
$$L - \eps < g \leq f \leq h < L + \eps \Rightarrow |f - L| < \eps. \quad \blacksquare$$

### 工程应用

- **电路分析**：$\lim_{s \to 0} H(s) = $ DC 增益。若 $H = N/D$，是 $N(0)/D(0)$（假设 $D(0) \neq 0$）。
- **信号系统**：BIBO 稳定通过 $\lim_{n \to \infty} h[n] = 0$ 与指数衰减夹逼分析。
- **概率**：分布收敛通过连续性——$X_n \to X \Rightarrow E[g(X_n)] \to E[g(X)]$ 对连续有界 $g$。

### 何时出现不定式

$0/0$ 是**最有用**情形——意味分子分母有公根。约去公根（洛必达是特例）。

### 极限 ↔ 其他运算的交换

- $\lim$ 与**连续函数**交换：$\lim f(g(x)) = f(\lim g(x))$。
- $\lim$ 与**有限和**交换：平凡。
- $\lim$ **不与**无限和（级数）交换，无一致条件。
- $\lim$ **不与**积分交换，无定理（勒贝格、单调、控制）。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

代数性质反映 $\lim$ 是适当空间上的**连续线性泛函**。

### 代数结构

考虑 $\mathcal{C}_a = \{f : \lim_{x \to a} f \text{ 存在}\}$。则：
- $\mathcal{C}_a$ 是**代数**（对加、积封闭）。
- $\lim_a : \mathcal{C}_a \to \mathbb{R}$ 是**代数同态**（保 $+, \cdot$、数乘）。
- 核 $\{f : \lim_a f = 0\}$ 是 $\mathcal{C}_a$ 的**理想**。

### 推广：网中的极限（摩尔-史密斯）

一般拓扑中，"趋向"用**网**（$x : I \to X$，$I$ 有向）或**滤子**。当上域为拓扑向量空间时代数性质保持。

### 极限交换：经典反例

$$\lim_{m \to \infty} \lim_{n \to \infty} \frac{m}{m+n} = 1, \quad \lim_{n \to \infty} \lim_{m \to \infty} \frac{m}{m+n} = 1$$

这里行。但 $a_{m,n} = m/(m+n) - n/(m+n)$ 时顺序重要。**Moore 定理**：一致收敛下交换。

### 微分与积分算子

极限是以下基础：
- **导数算子** $D$ 在 $C^1[a,b]$（不在 $C[a,b]$）连续。
- **黎曼积分**：$\int = \lim$ 和，代数性质 → 线性。
- **拉普拉斯变换**：$\mathcal{L}\{f + g\} = \mathcal{L}\{f\} + \mathcal{L}\{g\}$ 通过性质。

### 非标准分析

在 $^*\mathbb{R}$ 中，代数性质通过**转移原理**传递：$\mathbb{R}$ 中真的所有一阶句子在 $^*\mathbb{R}$ 中真。

### 数值与 ML

深度学习中，**链式法则**是极限复合的应用——复合的导数用连续性。反向传播依赖代数性质把梯度分解为雅可比积。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**配方：在任何极限中应用性质。**

### 计算算法

1. **直接代入**：$f(a)$ 存在有限？是极限（若 $f$ 连续）。
2. **不定式**？识别类型：$0/0$、$\infty/\infty$、$\infty - \infty$、$0 \cdot \infty$、$1^\infty$、$0^0$、$\infty^0$。
3. 按类型**操作**。
4. 用 Wolfram/SymPy **核对**。

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
# 和
print(sp.limit(sp.sin(x) + x, x, 0))           # 0
# 积
print(sp.limit(x * sp.sin(1/x), x, 0))         # 0（夹逼）
# 商带约简
print(sp.limit((x**2 - 4)/(x - 2), x, 2))      # 4
# 复合
print(sp.limit(sp.exp(sp.sin(x)/x), x, 0))     # E (sin x/x → 1)
```

### 性质不直接适用的情形

- **单侧极限不同**：和性质需两个极限存在。
- **分母 → 0**：$\lim (1/g)$ 可能 $\pm \infty$ 或振荡。
- **与不连续复合**：若 $f$ 在 $\lim g$ 处不连续，$\lim f(g)$ 可能失败。

### 实践中的夹逼

见 $\sin$、$\cos$ 或振荡函数乘以趋零的东西时：**夹逼**几乎总有效。

例：$\lim_{x \to \infty} (\sin x)/x$：$-1/x \leq \sin x / x \leq 1/x$，两边 $\to 0$，所以 $\to 0$。

### 完成所有练习

性质流畅度区分**懂极限**和**死记规则**。

</Porta>
