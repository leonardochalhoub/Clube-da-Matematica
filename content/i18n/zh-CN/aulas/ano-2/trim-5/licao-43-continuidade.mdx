<Porta nivel="formal" titulo="定义与定理">

<Definicao titulo="一点处的连续性">
$f$ 在 $a \in \text{dom}(f)$ 连续若三条件成立：
1. $\lim_{x \to a} f(x)$ 存在。
2. $f(a)$ 已定义。
3. $\lim_{x \to a} f(x) = f(a)$。

$f$ 在 $I$ 上**连续**若在 $I$ 每点连续。
</Definicao>

### 直接 ε-δ 形式

$f$ 在 $a$ 连续 $\iff$ $\forall \eps > 0, \exists \delta > 0 : |x - a| < \delta \Rightarrow |f(x) - f(a)| < \eps$。

（注：这里允许 $|x - a| = 0$，即 $x = a$，因 $|f(a) - f(a)| = 0 < \eps$ 平凡。）

### 不连续类型

| 类型 | 刻画 |
|---|---|
| **可去** | $\lim$ 存在，但 $\neq f(a)$ 或 $f(a)$ 未定义 |
| **跳跃** | 单侧极限存在但不同 |
| **本质**（无穷） | 至少一个单侧 $\pm\infty$ |
| **振荡** | 单侧极限不存在（振荡） |

### 中间值定理（TVI）

若 $f \in C([a, b])$ 且 $k$ 在 $f(a)$ 与 $f(b)$ 之间，则 $\exists c \in (a, b) : f(c) = k$。

### Weierstrass 极值定理

若 $f \in C([a, b])$（闭有界区间），则 $f$ 在 $[a, b]$ 上达到**最大值与最小值**。

### 一致连续性

$f$ 在 $I$ 上一致连续若 $\forall \eps, \exists \delta$（同一 $\delta$ 对所有 $x$）使 $|x - y| < \delta \Rightarrow |f(x) - f(y)| < \eps$。

**Heine-Cantor**：$f \in C([a, b])$ ⇒ $f$ 一致连续。

### 保连续性的运算

- 加、积、商（分母 $\neq 0$）。
- 复合：$g$ 在 $a$ 连续，$f$ 在 $g(a)$ 连续 ⇒ $f \circ g$ 在 $a$ 连续。
- $\max(f, g)$、$\min(f, g)$、$|f|$。

<Porta nivel="5" titulo="给5岁的孩子">

想象**纸上铅笔的轨迹**。若能**不抬起铅笔**画出函数，它连续。

若某点需要**抬起铅笔放别处**——有**洞**或**跳**——函数在那不连续。

</Porta>

<Porta nivel="10" titulo="给10岁的孩子">

连续函数 = 图像**无洞**且**无跳**。

- $f(x) = x^2$：在 $\mathbb{R}$ 全连续。图全连。
- $f(x) = 1/x$：在 $\mathbb{R} \setminus \{0\}$ 连续。$x = 0$ 处洞（去无穷）。
- $f(x) = \lfloor x \rfloor$（取整）：每整数处**跳**。整数处不连续。

**在 $a$ 处连续配方**：
1. $f(a)$ 存在。
2. $\lim_{x \to a} f(x)$ 存在。
3. 二者**相等**。

某条失败，函数在 $a$ **不连续**。

**实际应用**：温度、压力、电压测量——自然倾向连续。跳通常来自**模型**（如开/关）或测量误差。

</Porta>

<Porta nivel="15" titulo="给15岁的青少年">

连续性是**友好版**函数：你可"凭图计算"，直接代入，极限**不骗**。

### 3 条件

1. $f(a)$ 已定义（点处有值）。
2. $\lim_{x \to a} f(x)$ 存在（极限存在）。
3. 二者重合（$\lim = f(a)$）。

**任何**失败，$f$ 在 $a$ 不连续。

### 数值例

| 函数 | 在 $a$ 连续？ | 原因 |
|---|---|---|
| $f(x) = x^2$，$a = 3$ | 是 | $\lim = 9 = f(3)$ |
| $f(x) = 1/x$，$a = 0$ | 否 | $f(0)$ 未定义；$\lim = \pm \infty$ |
| $f(x) = (x^2-1)/(x-1)$，$a = 1$ | 否 | $f(1)$ 未定义；但 $\lim = 2$（可去） |
| $f(x) = \lfloor x \rfloor$，$a = 2$ | 否 | 单侧极限不同（跳） |

### 不连续类型

- **可去**：极限存在，重定义 $f(a)$ 即可"修"。
- **跳跃**：单侧不同，不可修。
- **无穷**：去 $\pm\infty$（垂直渐近线）。

### TVI（中间值定理）

若 $f$ 在 $[a, b]$ 连续且 $f(a) < 0 < f(b)$，则 $f$ 在 $(a, b)$ 有根。

**根存在性保证**——**二分法**基础（在数值分析、工程根查找中）。

### 物理应用

粒子位置连续（不能传送）。速度可能不连续（碰撞、急刹 = 导数不连续，但位置仍连续）。

</Porta>

<Porta nivel="25" titulo="给工程系学生">

连续性是保证操作性质（TVI、Weierstrass、黎曼积分等）的**最低规整性**。

### 证明：连续之和连续

设 $f, g$ 在 $a$ 连续。对 $\eps > 0$：
- $\exists \delta_1 : |x - a| < \delta_1 \Rightarrow |f(x) - f(a)| < \eps/2$。
- $\exists \delta_2 : |x - a| < \delta_2 \Rightarrow |g(x) - g(a)| < \eps/2$。

取 $\delta = \min(\delta_1, \delta_2)$。则 $|(f+g)(x) - (f+g)(a)| < \eps$。∎

### TVI 证明（梗概）

设 $S = \{x \in [a, b] : f(x) < k\}$。$S$ 非空（若 $f(a) < k$ 则 $a \in S$）有界。取 $c = \sup S$。由 $f$ 连续与上确界性质，$f(c) = k$。

细节：用 $\mathbb{R}$ 的**完备性**（每个有界集有上确界）。TVI 在 $\mathbb{Q}$ **失败**：$f(x) = x^2 - 2$ 连续，$f(0) = -2$，$f(2) = 2$，但无有理根。

### 狄利克雷函数——处处不连续

$$D(x) = \begin{cases} 1 & x \in \mathbb{Q} \\ 0 & x \notin \mathbb{Q} \end{cases}$$

任 $a$ 邻域含有理与无理 → 极限不存在 → 不连续。

### Thomae 函数——无理处连续

$$T(x) = \begin{cases} 1/q & x = p/q \in \mathbb{Q} \text{ 既约} \\ 0 & x \notin \mathbb{Q} \end{cases}$$

惊讶：所有无理处连续，所有有理处不连续。

### 工程应用

- **因果稳定系统**：脉冲响应 $h(t)$ 连续 → 输出 $y = h * u$ 在输入 $u$ 连续时连续（卷积保连续）。
- **$j\omega$ 轴上无极点的传递函数**：$|H(j\omega)|$ 是 $\omega$ 的连续函数——伯德图用。
- **优化**：Weierstrass 保紧集上极小值存在。否则搜索方法可能发散。
- **隐方程**：$F(x, y) = 0$。若 $F$ 连续且 $F(x_0, y_0) = 0$，$\partial F/\partial y \neq 0$，存在局部连续 $y(x)$（隐函数定理——第6学期）。

### 一致连续——何时重要

在**数值积分**中，$[a, b]$ 上一致连续保黎曼方法在网格上一致收敛。Heine-Cantor 在紧 $[a, b]$ 上自动化。

### Lipschitz 连续

$|f(x) - f(y)| \leq L |x - y|$——一致连续的强情形。在 ODE（Picard-Lindelöf 保存在/唯一）、神经网络（正则化）、优化（梯度下降）中用。

</Porta>

<Porta nivel="40" titulo="给资深专业人士">

连续性是拓扑与泛函分析的**操作基础**。

### 拓扑定义

$f: X \to Y$ 连续若 $Y$ 中每个开集的**原像**在 $X$ 中开。在度量空间中等价于 $\eps$-$\delta$ 定义。

### 序列刻画（海涅）

$f$ 在 $a$ 连续 $\iff$ 对每个 $x_n \to a$ 序列，$f(x_n) \to f(a)$。在度量空间成立；一般拓扑空间需**网**或**滤子**。

### 紧性保持

$f$ 连续，$K$ 紧 ⇒ $f(K)$ 紧。**Weierstrass** 是特例：$f$ 连续，$[a,b]$ 紧 ⇒ $f([a,b])$ 在 $\mathbb{R}$ 中紧 ⇒ 含 max/min。

### 连通性保持

$f$ 连续，$C$ 连通 ⇒ $f(C)$ 连通。**TVI** 在 $\mathbb{R}$ 中是特例：$[a,b]$ 连通，故 $f([a,b])$ 是区间（$\mathbb{R}$ 中连通）。

### 函数空间

- $C(K)$ = 紧 $K$ 上连续函数是带 $\|f\|_\infty = \sup_K |f|$ 的巴拿赫空间。
- $C^k$、$C^\infty$——规整度。
- **Stone-Weierstrass**：多项式在 $C([a,b])$ 中按上确界范数稠密。
- **Arzelà-Ascoli**：通过等度连续性刻画 $C(K)$ 中紧集。

### 连续算子

巴拿赫空间中，$T : X \to Y$ 线性连续 $\iff$ 有界：$\sup_{\|x\|=1} \|Tx\| < \infty$。算子拓扑 $\to$ 经典泛函分析。

### Sobolev 函数

PDE 中，$H^1$、$H^2$ 等放宽古典连续到"弱意义连续"。Navier-Stokes 解在 $H^1$——不要求逐点连续。

### ML/机器人应用

- ReLU 神经网络连续但在零不可微。次梯度保持分析。
- RL 中连续策略 $\pi(s)$（非离散）：允许确定性梯度（DDPG、SAC）。
- 网络的 **Lipschitz 常数**：限制输出变化——GAN（Wasserstein）、对抗鲁棒性中用。

</Porta>

<Porta nivel="pratica" titulo="实际应用">

**配方：如何检验连续性。**

### 算法

1. **$f(a)$ 已定义？** 否则 $f$ 不连续（或可去，若 $\lim$ 存在）。
2. **$\lim_{x \to a} f(x)$ 存在？** 算单侧极限。不同则跳。
3. **$\lim = f(a)$？** 否则可去。

### 不连续常见症状

| 症状 | 类型 |
|---|---|
| 分母在 $a$ 为零 | 无穷或可去（依分子） |
| 分段函数在 $a$ 处变 | 检查单侧 |
| $\sin(1/x)$、$\cos(1/x)$ | $x = 0$ 振荡 |
| 取整 $\lfloor x \rfloor$ | 整数处跳 |

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
f = (x**2 - 4)/(x - 2)
# 在 x = 2 连续？
limite = sp.limit(f, x, 2)        # 4
valor = f.subs(x, 2)              # nan (0/0)
print(limite, valor)
# 修：定义 f(2) = 4
```

### Wolfram Alpha

`continuity of (x^2-4)/(x-2) at x=2` → 显示可去不连续。

### 不连续何时不重要

**积分**中，孤立不连续（零测集）不影响积分值。几乎处处连续函数勒贝格可积。

### 何时很重要

- ODE 解：源项小不连续 → 解可能不存在/唯一。
- 控制：设定点不连续致饱和、振荡。
- 数值算法：二分法需连续性使 TVI 有效。

完成 30 题。连续性是形式微积分与应用微积分的**过滤器**——不掌握会犯微妙错误。
