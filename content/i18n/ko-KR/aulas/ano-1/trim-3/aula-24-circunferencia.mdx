<Porta nivel="formal" titulo="방정식과 성질">

<Definicao titulo="원">
중심 $C = (a, b)$로부터 거리 $r > 0$인 점 $(x, y) \in \mathbb{R}^2$의 집합.
</Definicao>

### 표준형

$$(x - a)^2 + (y - b)^2 = r^2$$

### 일반형

전개하면: $x^2 + y^2 + Dx + Ey + F = 0$, 여기서 $D = -2a$, $E = -2b$, $F = a^2 + b^2 - r^2$.

중심 복원: $(a, b) = (-D/2, -E/2)$. 반지름: $r = \sqrt{D^2/4 + E^2/4 - F}$ (양수일 때 유효; 그렇지 않으면 실원이 아님).

### 직선과 원

직선 $r$은:
- 원과 **만나지 않음**: $d(\text{중심}, r) > r$.
- **접함** ($d = r$): 정확히 1점.
- **할선** ($d < r$): 2개의 교점.

### 접선-반지름 성질

$P$에서의 접선은 반지름 $\overline{CP}$에 수직이다.

</Porta>

<Porta nivel="15" titulo="15세를 위해">

원은 중심으로부터 **같은 거리**에 있는 점들의 집합입니다. 거리 $= r$ (반지름).

21강의 거리 공식을 적용하면:
$$\sqrt{(x - a)^2 + (y - b)^2} = r$$

제곱하면: $(x - a)^2 + (y - b)^2 = r^2$. **이게 전부입니다**.

중심이 원점일 때 ($a = b = 0$): $x^2 + y^2 = r^2$. **단위원** ($r = 1$): $x^2 + y^2 = 1$ — 12강의 삼각원에서 사용한 것!

</Porta>

<Porta nivel="25" titulo="공학도를 위해">

원은 **기하학적 자취**(locus)의 가장 간단한 예입니다. 즉시 일반화: $\mathbb{R}^3$에서의 **구면** — $(x-a)^2 + (y-b)^2 + (z-c)^2 = r^2$.

응용:
- **통신**: 안테나 도달 범위 = 원. 다중 커버리지 = 원의 합집합.
- **GPS**: 각 위성이 구면을 정의; 당신의 위치는 4개 구면의 교점.
- **로봇공학**: 구형 관절을 가진 팔의 도달 범위는 구면.

</Porta>
