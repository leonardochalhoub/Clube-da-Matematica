### 평행과 수직

직선 $r: y = m_r x + n_r$과 $s: y = m_s x + n_s$에 대해:

- **평행 (일치하지 않음)**: $m_r = m_s$이고 $n_r \neq n_s$.
- **일치**: $m_r = m_s$이고 $n_r = n_s$.
- **교차**: $m_r \neq m_s$.
- **수직**: $m_r \cdot m_s = -1$ (둘 다 수직선이 아님 가정).

수직선이 포함된 경우: $x = a$는 수직, $y = b$는 수평. 수직선끼리는 평행; 수직과 수평은 수직.

### 직선 사이의 각

$$\tan\theta = \left|\frac{m_r - m_s}{1 + m_r m_s}\right|$$

(둘 다 수직선이 아님 가정.)

### 점-직선 거리

점 $P_0 = (x_0, y_0)$과 직선 $Ax + By + C = 0$:

$$d(P_0, r) = \frac{|A x_0 + B y_0 + C|}{\sqrt{A^2 + B^2}}$$

### 평행한 직선 사이의 거리

$r: Ax + By + C_1 = 0$과 $s: Ax + By + C_2 = 0$에 대해:
$$d(r, s) = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$

</Porta>

<Porta nivel="15" titulo="15세 청소년을 위해">

평면 위의 두 직선은 다음 중 하나입니다:

1. **평행** (같은 기울기, 결코 만나지 않음).
2. **수직** (90° 각도를 이룸).
3. **한 점에서 만남** (교차).
4. **같은 직선** (일치).

**평행 기준**: $m_r = m_s$ — 같은 기울기.

**수직 기준**: $m_r \cdot m_s = -1$. 이상하지만, 두 변위가 만드는 삼각형에 피타고라스를 적용하면 나옵니다.

**예**: $y = 2x + 1$과 $y = -x/2 + 5$는 수직 ($2 \cdot (-1/2) = -1$).

</Porta>

<Porta nivel="25" titulo="공학부 학생을 위해">

실제로, **2x2 선형 시스템 풀이**는 직선의 교점 구하기입니다. 크라메르와 소거법은 여기서 시작됩니다.

### 2x2 시스템과 기하학적 해석

$$\begin{cases} a_1 x + b_1 y = c_1 \\ a_2 x + b_2 y = c_2 \end{cases}$$

- **유일 해**: 교차하는 직선. $\det \neq 0$.
- **무한 해**: 일치하는 직선. 행렬식 $= 0$ + 시스템이 모순 없음.
- **해 없음**: 서로 다른 평행한 직선. 행렬식 $= 0$ + 시스템이 모순.

</Porta>
