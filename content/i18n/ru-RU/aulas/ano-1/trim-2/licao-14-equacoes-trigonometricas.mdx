<Porta nivel="formal" titulo="Структура решений">

### Уравнения с синусом

Для $\sin x = a$ при $a \in [-1, 1]$:
$$x = \arcsin a + 2k\pi \quad \text{ou} \quad x = \pi - \arcsin a + 2k\pi, \quad k \in \mathbb{Z}$$

Где $\arcsin: [-1, 1] \to [-\pi/2, \pi/2]$ — главная обратная функция.

### Уравнения с косинусом

Для $\cos x = a$ при $a \in [-1, 1]$:
$$x = \pm \arccos a + 2k\pi, \quad k \in \mathbb{Z}$$

Где $\arccos: [-1, 1] \to [0, \pi]$.

### Уравнения с тангенсом

Для $\tan x = a$ при $a \in \mathbb{R}$:
$$x = \arctan a + k\pi, \quad k \in \mathbb{Z}$$

Где $\arctan: \mathbb{R} \to (-\pi/2, \pi/2)$.

### Общий рецепт

Чтобы решить тригонометрическое $f(x) = 0$:

1. **Свести** к основным уравнениям $\sin x = a$, $\cos x = a$, $\tan x = a$ через тождества.
2. **Перечислить** все решения на промежутке $[0, 2\pi)$ (или $[-\pi, \pi]$).
3. **Обобщить** прибавлением $2k\pi$ (или $k\pi$ для тангенса).

</Porta>

<Porta nivel="15" titulo="Для 15-летнего подростка">

Решить $\sin x = 1/2$ не так просто, как решить $x + 3 = 5$.

Почему? Потому что синус всё время повторяется. Если $x = \pi/6$ — решение, то и $x = \pi/6 + 2\pi$ — тоже. И $x = \pi/6 + 4\pi$. И... бесконечно много.

Кроме того, $\sin(\pi - \pi/6) = \sin(5\pi/6) = 1/2$ тоже! Поэтому в $[0, 2\pi)$ есть **два** решения: $\pi/6$ и $5\pi/6$.

**Рецепт:**
1. Найди решения на тригонометрическом круге, в $[0, 2\pi)$.
2. Прибавь $2k\pi$ (все целые обороты).

Для тангенса по-другому: период $\tan$ равен $\pi$ (а не $2\pi$), потому что $\tan(x + \pi) = \tan x$.

</Porta>

<Porta nivel="25" titulo="Для студента-инженера">

На практике **тригонометрию вручную не решают** после школы. Используют калькулятор или решатель. Но структура решений важна, потому что она определяет форму ответов в инженерных задачах.

В теории управления, **фазовая связь** между двумя сигналами задаётся $\arctan$ отношения мнимая/действительная часть частного передаточных функций. В механике, угол равновесия — корень тригонометрического уравнения.

### Численные методы

Для тригонометрических уравнений, которые не сводятся к основным (например, $x = \cos x$), используют метод Ньютона-Рафсона или дихотомию. **Замкнутой формулы нет.**

</Porta>
