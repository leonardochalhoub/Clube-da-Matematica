<Porta nivel="formal" titulo="Manipulação trig">

Técnicas para limites $0/0$ trigonométricos:

1. **Substituição $u$**: $\sin(2x)/x = 2 \cdot \sin(2x)/(2x) \to 2$.
2. **Identidades**: $1 - \cos x = 2\sin^2(x/2)$.
3. **Conjugado**: $1 - \cos x = (1 - \cos^2 x)/(1 + \cos x) = \sin^2 x /(1 + \cos x)$.

### Tabela útil

| Limite | Valor |
|---|---|
| $\sin x / x$ ($x \to 0$) | $1$ |
| $\sin(kx)/x$ | $k$ |
| $(1-\cos x)/x$ | $0$ |
| $(1-\cos x)/x^2$ | $1/2$ |
| $\tan x / x$ | $1$ |
| $\arcsin x / x$ | $1$ |
| $\arctan x / x$ | $1$ |
