### Limites trigonomĂ©tricos fundamentais

| Limite | Valor |
|---|---|
| $\lim_{x \to 0} \sin x / x$ | $1$ |
| $\lim_{x \to 0} \sin(kx)/x$ | $k$ |
| $\lim_{x \to 0} \tan x / x$ | $1$ |
| $\lim_{x \to 0} (1 - \cos x)/x$ | $0$ |
| $\lim_{x \to 0} (1 - \cos x)/x^2$ | $1/2$ |
| $\lim_{x \to 0} \sin^2 x / x^2$ | $1$ |
| $\lim_{x \to 0} \arcsin x / x$ | $1$ |
| $\lim_{x \to 0} \arctan x / x$ | $1$ |

### Identidades Ășteis

- $1 - \cos x = 2\sin^2(x/2)$
- $\sin(a + b) = \sin a \cos b + \cos a \sin b$
- $\cos(a + b) = \cos a \cos b - \sin a \sin b$
- $1 - \cos x = (1 - \cos^2 x)/(1 + \cos x) = \sin^2 x / (1 + \cos x)$

### TĂ©cnicas

1. **SubstituiĂ§ĂŁo $u$**: $\sin(2x)/x = 2 \cdot \sin(2x)/(2x) \to 2$.
2. **Identidades** para reescrever em forma fundamental.
3. **Conjugado**: $1 - \cos x = \sin^2 x / (1 + \cos x)$.
4. **Confronto**: $-1 \leq \sin x \leq 1$, multiplique por algo $\to 0$.

### DemonstraĂ§ĂŁo de $\lim (1 - \cos x)/x^2 = 1/2$

$\frac{1 - \cos x}{x^2} = \frac{2 \sin^2(x/2)}{x^2} = \frac{1}{2} \cdot \frac{\sin^2(x/2)}{(x/2)^2} \to \frac{1}{2} \cdot 1 = \frac{1}{2}$.

### DemonstraĂ§ĂŁo de $\lim \tan x / x = 1$

$\tan x / x = (\sin x / x) \cdot (1 / \cos x) \to 1 \cdot 1 = 1$.

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Limites trig com $x \to 0$ se reduzem a trĂȘs casos via manipulaĂ§ĂŁo:

### Caso 1: $\sin(?)/?$

Se vocĂȘ consegue escrever na forma $\sin(\text{algo})/\text{algo}$, vai pra 1. Use mudanĂ§a de variĂĄvel $u = $ argumento.

**Exemplo**: $\lim_{x \to 0} \sin(7x)/(3x) = (7/3) \lim_{x \to 0} \sin(7x)/(7x) = 7/3$.

### Caso 2: $(1 - \cos)/?$

Use identidade $1 - \cos x = 2 \sin^2(x/2)$ ou conjugado.

**Exemplo**: $\lim_{x \to 0} (1 - \cos x)/(x \sin x) = (x^2/2)/(x \cdot x) = 1/2$ (usando $\sin x \approx x$).

### Caso 3: combinaĂ§Ă”es $\tan, \sec, \cot$

Reescreva em $\sin, \cos$.

**Exemplo**: $\lim_{x \to 0} \tan(2x)/\sin(3x) = \lim (2x)/(3x) = 2/3$ (aproximaĂ§ĂŁo).

### Truques de mudanĂ§a de variĂĄvel

Para $\lim_{x \to a}$ trig, use $u = x - a$ pra trazer pra zero.

**Exemplo**: $\lim_{x \to \pi} \sin x / (x - \pi)$. Seja $u = x - \pi$. EntĂŁo $\sin x = \sin(u + \pi) = -\sin u$. Resultado: $\lim_{u \to 0} -\sin u / u = -1$.

### Tabela de aproximaĂ§Ă”es para $x$ pequeno

| FunĂ§ĂŁo | AproximaĂ§ĂŁo (atĂ© $x^3$) |
|---|---|
| $\sin x$ | $x - x^3/6$ |
| $\cos x$ | $1 - x^2/2$ |
| $\tan x$ | $x + x^3/3$ |
| $\arcsin x$ | $x + x^3/6$ |
| $\arctan x$ | $x - x^3/3$ |

Use pra resolver mais rĂĄpido.

### Exemplo desafiador

$\lim_{x \to 0} (\tan x - \sin x)/x^3$:

$\tan x - \sin x = \sin x (1/\cos x - 1) = \sin x \cdot (1 - \cos x)/\cos x$.

Logo: $\lim = (\sin x / x) \cdot ((1 - \cos x)/x^2) \cdot (1/\cos x) = 1 \cdot (1/2) \cdot 1 = 1/2$.

<Porta nivel="25" titulo="Pra estudante de engenharia">

Limites trigonomĂ©tricos sĂŁo essenciais em **derivadas** ($\sin', \cos', \tan'$), **sĂ©ries de Fourier**, **anĂĄlise harmĂŽnica**.

### DemonstraĂ§Ă”es chave

**$\lim \sin x / x = 1$**: argumento geomĂ©trico â ĂĄrea de triĂąngulo, setor, triĂąngulo. Pelo confronto, conclui.

**$(1 - \cos x)/x^2 \to 1/2$**:
$$\frac{1 - \cos x}{x^2} = \frac{(1-\cos x)(1+\cos x)}{x^2 (1+\cos x)} = \frac{\sin^2 x}{x^2(1+\cos x)} = \frac{\sin^2 x}{x^2} \cdot \frac{1}{1+\cos x} \to 1 \cdot \frac{1}{2} = \frac{1}{2}.$$

**$\arcsin x / x \to 1$**: $u = \arcsin x$, $x = \sin u$, $u \to 0$ quando $x \to 0$. $\arcsin x / x = u/\sin u \to 1$.

### ConexĂŁo com derivadas

- $(\sin x)' = \cos x$ vem de $\lim_{h \to 0} (\sin(x+h) - \sin x)/h$. ManipulaĂ§ĂŁo usa $\sin(x+h) = \sin x \cos h + \cos x \sin h$, entĂŁo razĂŁo $= \sin x (\cos h - 1)/h + \cos x \sin h / h \to 0 + \cos x = \cos x$.
- $(\cos x)' = -\sin x$ anĂĄlogo.
- $(\tan x)' = \sec^2 x$ via quociente.

### Series de Taylor (Trim 7)

$$\sin x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n+1}}{(2n+1)!}, \quad \cos x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n}}{(2n)!}.$$

Truncadas dĂŁo aproximaĂ§Ă”es de pequenos Ăąngulos. ConvergĂȘncia uniforme em compactos.

### AplicaĂ§Ă”es em engenharia

- **Pequenos Ăąngulos em mecĂąnica**: pĂȘndulo $T = 2\pi\sqrt{L/g}$ vale para $\theta \lesssim 10Â°$. Erro relativo cresce como $\theta^2/16$.
- **Ăptica gaussiana** (paraxial): $\sin \theta \approx \theta$. Lentes finas, refraĂ§ĂŁo.
- **LinearizaĂ§ĂŁo**: sistemas dinĂąmicos $\dot x = f(x)$ linearizados via Jacobiano em ponto de equilĂ­brio.
- **ModulaĂ§ĂŁo**: $\cos(\omega t + \phi(t))$ com $\phi$ pequeno expandido para AM-equivalente.

### AnĂĄlise de Fourier

Coeficientes de Fourier $a_n = \frac{1}{T}\int f(t) \cos(2\pi nt/T) dt$. Limite $\sum a_n^2$ converge (Parseval). Limites trig aparecem em verificaĂ§Ă”es de convergĂȘncia ponto a ponto (Dini, Dirichlet).

### Confronto avanĂ§ado

$\lim_{x \to \infty} \sin x$ nĂŁo existe, mas $\lim_{x \to \infty} (\sin x)/x = 0$ por confronto $-1/x \leq \sin x/x \leq 1/x$.

$\lim_{x \to 0} x \sin(1/x) = 0$ por confronto. **NĂŁo** Ă© o mesmo que $\lim x \cdot \lim \sin(1/x)$ â o segundo nem existe. Confronto resolve sem essa fatoraĂ§ĂŁo.

### Em circuitos AC

- ImpedĂąncia $Z(j\omega) = R + jX$. Magnitude $|Z| = \sqrt{R^2 + X^2}$. Limites em pequena freq./alta freq.
- $\sin(\omega t)$ em $\omega = 0$: vai pra zero. Em $\omega \to \infty$: oscila, sem limite.

<Porta nivel="40" titulo="Pra profissional sĂȘnior">

Limites trigonomĂ©tricos sĂŁo manifestaĂ§Ă”es de **estrutura analĂ­tica** profunda: $\sin, \cos$ sĂŁo funĂ§Ă”es inteiras com padrĂŁo especĂ­fico de zeros.

### FatorizaĂ§ĂŁo de Weierstrass

$$\sin(\pi x) = \pi x \prod_{n=1}^\infty \left(1 - \frac{x^2}{n^2}\right).$$

Tomando logaritmo e derivando, dĂĄ identidade de Mittag-Leffler:
$$\pi \cot(\pi x) = \frac{1}{x} + \sum_{n=1}^\infty \frac{2x}{x^2 - n^2}.$$

ConexĂŁo com $\zeta(2k)$.

### CaracterizaĂ§ĂŁo funcional

$\sin, \cos$ sĂŁo as Ășnicas funĂ§Ă”es $\mathbb{R} \to \mathbb{R}$ satisfazendo:
- $f(x+y) = f(x)g(y) + g(x)f(y)$, $g(x+y) = g(x)g(y) - f(x)f(y)$
- $f'(0) = 1$, $g(0) = 1$
- $f, g$ contĂ­nuas

Equivalente: soluĂ§ĂŁo de $y'' + y = 0$ com condiĂ§Ă”es iniciais.

### AnĂĄlise complexa

$e^{ix} = \cos x + i \sin x$ (Euler). Limites trig viram limites de $e^{iz}$ em $z \to 0$. Holomorfia em $\mathbb{C}$ inteiro. FatorizaĂ§ĂŁo de Hadamard.

### AplicaĂ§Ă”es em PDE

- **EquaĂ§ĂŁo da onda** $u_{tt} = c^2 u_{xx}$: soluĂ§ĂŁo $u(x, t) = f(x - ct) + g(x + ct)$ envolve $\sin, \cos$ via separaĂ§ĂŁo de variĂĄveis.
- **SchrĂ¶dinger livre**: $\psi(x, t) = \int A(k) e^{i(kx - \omega t)} dk$, $\omega = k^2/2m$. Pacotes via $\sin, \cos$.
- **Calor com BCs Dirichlet**: soluĂ§ĂŁo $\sum b_n \sin(n\pi x/L) e^{-n^2 \pi^2 D t/L^2}$.

### AnĂĄlise harmĂŽnica

Teorema de Carleson: sĂ©rie de Fourier converge ae para $f \in L^2$. DemonstraĂ§ĂŁo depende de estimativas finas envolvendo limites trig.

### AproximaĂ§ĂŁo de PadĂ©

Para $\sin x$, aproximante PadĂ© $[m/n]$ pode ser melhor que Taylor. Em particular, $\tan x \approx x(1 - x^2/15)/(1 - 2x^2/5)$ tem erro menor que Taylor truncada do mesmo grau.

### CĂĄlculo de operadores

- **FunĂ§ĂŁo de matriz**: $\sin A = (e^{iA} - e^{-iA})/(2i)$. Bem definido para qualquer matriz.
- **Spectral mapping**: spec($\sin A$) = $\{\sin \lambda : \lambda \in $ spec $A\}$.
- **AplicaĂ§ĂŁo**: soluĂ§ĂŁo de $\ddot x = -\Omega^2 x$ Ă© $x(t) = \cos(\Omega t) x_0 + \Omega^{-1} \sin(\Omega t) v_0$.

### NumĂ©rico

- **Argument reduction**: pra $\sin(10^{15})$, reduce mod $\pi$ com **Payne-Hanek** (alta precisĂŁo).
- **CORDIC**: hardware-friendly, calcula $\sin, \cos$ por iteraĂ§ĂŁo de rotaĂ§Ă”es.
- **Cancelamento numĂ©rico**: $1 - \cos x$ em float32 pra $x$ pequeno tem erro relativo grande. Usar `2 * sin(x/2)**2` em vez disso.

### Cheat sheet

Para $x \to 0$:

| Forma | Limite |
|---|---|
| $\sin(ax)/x$ | $a$ |
| $\sin(ax)/\sin(bx)$ | $a/b$ |
| $\sin(ax)/\tan(bx)$ | $a/b$ |
| $(1 - \cos(ax))/x^2$ | $a^2/2$ |
| $\arcsin(ax)/x$ | $a$ |
| $\arctan(ax)/x$ | $a$ |
| $(\sin x)^2/x^2$ | 1 |
| $\sin(\sin x)/x$ | 1 |

### Quando NĂO usar essas

- Limite envolve $x \to a \neq 0$: faĂ§a mudanĂ§a $u = x - a$.
- FunĂ§ĂŁo com $1/x$ no argumento ($\sin(1/x)$): aproveite que oscila â confronto.

### Cuidados comuns

- $\sin(2x)/x \neq 2 \sin x / x$: deve ser $2 \cdot \sin(2x)/(2x) \to 2$.
- $\sin(x^2)/x \neq \sin x \cdot x$: deve ser $\sin(x^2)/x = x \cdot \sin(x^2)/x^2 \to 0$.
- $(1 - \cos x)/x \neq (1 - \cos x)/x^2$: o primeiro vai a 0, o segundo a 1/2.
