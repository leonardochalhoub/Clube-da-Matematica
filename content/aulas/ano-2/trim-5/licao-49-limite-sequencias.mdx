<Porta nivel="formal" titulo="Sequências, convergência e teoremas">

<Definicao titulo="Limite de sequência (ε-N)">
$(a_n) \to L$ se: $\forall \eps > 0, \exists N \in \mathbb{N} : n \geq N \Rightarrow |a_n - L| < \eps$.
</Definicao>

### Teoremas-chave

| Teorema | Enunciado |
|---|---|
| **Unicidade** | Limite, se existe, é único |
| **Aritmética** | $\lim(a_n \pm b_n) = \lim a_n \pm \lim b_n$, etc. |
| **Confronto** | $a_n \leq b_n \leq c_n$, $\lim a_n = \lim c_n = L \Rightarrow \lim b_n = L$ |
| **Monótona limitada** | Crescente limitada superiormente converge |
| **Bolzano-Weierstrass** | Sequência limitada tem subsequência convergente |
| **Cauchy** | $a_n$ converge $\iff$ é de Cauchy |

### Cauchy

<Definicao titulo="Sequência de Cauchy">
$(a_n)$ é de **Cauchy** se $\forall \eps > 0, \exists N : m, n \geq N \Rightarrow |a_m - a_n| < \eps$.
</Definicao>

**Em $\mathbb{R}$**: Cauchy $\iff$ convergente (completude).
**Em $\mathbb{Q}$**: nem toda Cauchy converge ($a_n$ aproximando $\sqrt 2$).

### Monotonia + limitação

- $(a_n)$ crescente e limitada superiormente $\Rightarrow$ converge para $\sup a_n$.
- $(a_n)$ decrescente e limitada inferiormente $\Rightarrow$ converge para $\inf a_n$.

### Subsequências

$(a_{n_k})$ é subsequência de $(a_n)$ se $n_1 < n_2 < \ldots$.
$\lim a_n = L \Rightarrow \lim a_{n_k} = L$ para toda subsequência.

### Sequências divergentes

- $a_n = (-1)^n$: oscila entre $-1$ e $1$. Subsequências convergem (a $\pm 1$), mas $a_n$ não.
- $a_n = n$: cresce sem limite ($\to +\infty$).
- $a_n = n \sin(n)$: comportamento irregular, sem limite.

### Sequências recursivas

$a_{n+1} = f(a_n)$, $a_0$ dado. Limite, se existe, é **ponto fixo** de $f$: $L = f(L)$.

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Imagina uma fila de números: $1, 1/2, 1/3, 1/4, \ldots$ — eles vão **diminuindo** e se aproximando de **zero**.

Limite é a "**onde está indo**" da fila. Se a fila vai pra um lugar, esse lugar é o limite.

Algumas filas não vão pra lugar nenhum (ficam pulando), e aí não tem limite.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

**Sequência** = lista de números: $a_1, a_2, a_3, \ldots$

**Limite** $L$: o número pra onde a sequência **chega no infinito**. Por exemplo:
- $1, 1/2, 1/3, \ldots$ → vai pra **zero**.
- $1, 4, 9, 16, \ldots$ → vai pra **$\infty$** (cresce).
- $1, -1, 1, -1, \ldots$ → não vai pra lugar nenhum (alternando).

**Definição rigorosa**: $a_n \to L$ se você consegue ficar **tão perto de $L$ quanto quiser**, basta avançar muito na sequência. Para qualquer "bolha" $\eps$ em torno de $L$, há um $N$ tal que **todos** os termos depois de $a_N$ estão na bolha.

**Truque útil**: se $a_n$ está sempre **crescendo** e tem um **teto**, ela **converge**! (Monótona limitada.) Não precisa nem saber o limite — só saber que existe.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Sequências são **funções com domínio em $\mathbb{N}$**. A teoria de limite é mesma da Aula 41 ε-δ, mas usamos $N$ no lugar de $\delta$.

### Por que sequências importam

- **Iteração de Newton** pra raízes: $a_{n+1} = a_n - f(a_n)/f'(a_n)$.
- **Séries** = limite de somas parciais: $\sum a_n = \lim s_N$.
- **Integral** = limite de somas de Riemann.
- **Probabilidade**: lei dos grandes números $\bar X_n \to E[X]$.

### Limites notáveis de sequências

| Sequência | Limite |
|---|---|
| $1/n$ | $0$ |
| $(1 + 1/n)^n$ | $e$ |
| $n^{1/n}$ | $1$ |
| $n!/n^n$ | $0$ |
| $a^n$ ($|a| < 1$) | $0$ |
| $a^n$ ($a > 1$) | $\infty$ |
| $(-1)^n$ | não existe |

### Hierarquia de crescimento (importante!)

$$\log n \ll n^a \ll a^n \ll n! \ll n^n.$$

Cada um cresce **muito mais devagar** que o seguinte. Ferramenta usada o tempo todo em complexidade de algoritmos.

### Teoremas operacionais

**Confronto**: prensar $b_n$ entre $a_n$ e $c_n$ que vão pro mesmo lugar.

**Exemplo**: $\lim_{n \to \infty} \sin n / n$. $-1/n \leq \sin n / n \leq 1/n$. Pelo confronto, $\to 0$.

**Monótona limitada**: sequência crescente e limitada converge — não precisa achar o limite, só saber que existe!

**Exemplo**: $a_1 = 1$, $a_{n+1} = (a_n + 2/a_n)/2$. Mostra-se que $a_n$ é decrescente após $a_1$, limitada inferior por $\sqrt 2$. Logo converge. O limite $L$ satisfaz $L = (L + 2/L)/2$, ou seja, $L = \sqrt 2$.

### Bolzano-Weierstrass

**Toda sequência limitada tem subsequência convergente**. Útil para argumentos de existência (compacidade).

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Sequências são **a base de algoritmos iterativos** — Newton, gradient descent, EM, MCMC.

### Demonstração: $\lim 1/n = 0$ via ε-N

Dado $\eps > 0$, escolha $N = \lceil 1/\eps \rceil + 1$. Então $n \geq N \Rightarrow 1/n \leq 1/N < \eps$. ∎

### Demonstração: monótona limitada converge

Seja $(a_n)$ crescente, limitada superiormente. Tome $L = \sup\{a_n\}$ (existe por completude). Para $\eps > 0$, $L - \eps$ não é cota superior, logo $\exists N : a_N > L - \eps$. Como crescente, $a_n \geq a_N > L - \eps$ para $n \geq N$. Como $a_n \leq L$, vale $|a_n - L| < \eps$. ∎

### Demonstração: Bolzano-Weierstrass (esboço)

Sequência limitada $(a_n) \subseteq [a, b]$. Bisecte: alguma metade contém infinitos termos. Repita. Construa subsequência $(a_{n_k})$ com $a_{n_k}$ na metade $k$-ésima, intervalo de comprimento $(b-a)/2^k$. Cauchy, logo convergente. ∎

### Demonstração: Cauchy ⇔ convergente em $\mathbb{R}$

$(\Rightarrow)$ Convergente é Cauchy: $|a_m - a_n| \leq |a_m - L| + |L - a_n| < \eps$.

$(\Leftarrow)$ Cauchy é convergente: Cauchy $\Rightarrow$ limitada $\Rightarrow$ tem subsequência convergente (Bolzano-Weierstrass) $\Rightarrow$ a sequência inteira converge para o mesmo limite (porque é Cauchy).

### Sequências de Cauchy em $\mathbb{Q}$

$(a_n)$ aproximando $\sqrt 2$ por bisseção é Cauchy em $\mathbb{Q}$, mas **não converge** em $\mathbb{Q}$ (limite é irracional). $\mathbb{R}$ é a **completação** de $\mathbb{Q}$ pela métrica $|\cdot|$.

### Aplicações em algoritmos

- **Gradient descent**: $w_{n+1} = w_n - \eta \nabla f(w_n)$. Converge se $\eta < 2/L$ (Lipschitz constant), $f$ convexa.
- **Newton**: $x_{n+1} = x_n - f(x_n)/f'(x_n)$. Convergência **quadrática** perto de raiz simples.
- **Heron** (raiz quadrada): $a_{n+1} = (a_n + N/a_n)/2$. Quadrática para $\sqrt N$. Vem de Newton em $f(x) = x^2 - N$.
- **EM**: monotonicamente aumenta likelihood. Converge a ponto crítico (não necessariamente global max).

### Convergência de séries

$\sum a_n$ converge $\iff$ somas parciais $s_N = \sum_{n=1}^N a_n$ formam sequência convergente.

**Critérios**:
- Cauchy: $\forall \eps, \exists N : m > n \geq N \Rightarrow |s_m - s_n| < \eps$.
- Razão (D'Alembert): $\lim |a_{n+1}/a_n| = L < 1 \Rightarrow$ converge.
- Raiz (Cauchy): $\lim |a_n|^{1/n} = L < 1 \Rightarrow$ converge.

### Lei dos grandes números

$X_1, X_2, \ldots$ iid com $E[X] = \mu$, $V[X] < \infty$. Então $\bar X_n = (1/n) \sum X_i \to \mu$ (em probabilidade — fraca; ae — forte).

### Aplicações em sistemas dinâmicos

Sistema discreto $x_{n+1} = T(x_n)$. Comportamento: ponto fixo, ciclo, atrator estranho (caos). **Mapa logístico** $x_{n+1} = r x_n (1 - x_n)$ exibe transição para caos em $r \approx 3{,}57$.

### Sequências em $\mathbb{R}^n$

Convergência componente a componente. Em norma: $\|x_n - L\| \to 0$. Equivalente em $\mathbb{R}^n$ (mas não em dimensão infinita).

<Porta nivel="40" titulo="Pra profissional sênior">

Sequências são central em **análise funcional**, **espaços métricos**, **topologia**.

### Espaços de sequências

- $\ell^p$ = sequências com $\sum |a_n|^p < \infty$. Banach com norma $\|a\|_p = (\sum |a_n|^p)^{1/p}$.
- $\ell^\infty$ = sequências limitadas. Banach com $\|a\|_\infty = \sup |a_n|$.
- $c_0$ = sequências $a_n \to 0$. Subespaço fechado de $\ell^\infty$.

### Convergência fraca vs forte

- **Forte**: $\|x_n - x\| \to 0$.
- **Fraca**: $f(x_n) \to f(x)$ para todo $f \in X^*$ (dual).

Em dimensão finita, equivalentes. Em $\ell^2$: base ortonormal $e_n$ converge fracamente para 0 mas não fortemente ($\|e_n\| = 1$).

### Cauchy em espaços métricos

$(X, d)$ é **completo** se toda Cauchy converge. Banach = espaço normado completo. Hilbert = inner product space completo.

**Completação**: todo espaço métrico tem completação única. $\mathbb{R}$ é completação de $\mathbb{Q}$. $L^2$ é completação de $C[a,b]$ pela norma $L^2$.

### Compacidade

Em espaços métricos: **sequencialmente compacto** = toda sequência tem subsequência convergente. Em $\mathbb{R}^n$, equivalente a fechado e limitado (Heine-Borel). **Em dimensão infinita**, equivalente a fechado, limitado **e** totalmente limitado (Riesz).

### Topologia

Caracterização sequencial de continuidade vale em **espaços métricos** (Heine). Em geral, precisa de redes ou filtros.

### Operadores compactos

$T : X \to Y$ é **compacto** se mapa bola unitária em conjunto pré-compacto. Subsequências convergentes são chave em demonstrações.

### Teoria espectral

Sequência de autovalores $\lambda_n$ de operador compacto autoadjunto: ou finita, ou $\lambda_n \to 0$. Spectral theorem decompõe via base de autovetores.

### Métodos iterativos avançados

- **Aceleração de Aitken**: $\hat a_n = a_n - (a_{n+1} - a_n)^2/(a_{n+2} - 2 a_{n+1} + a_n)$. Acelera convergência linear.
- **Romberg integration**: extrapolação de Richardson em integração.
- **GMRES, Krylov subspaces**: para sistemas lineares grandes.

### Aplicações em ML / RL

- **Q-learning convergence**: $Q_n \to Q^*$ sob exploração $\epsilon$-greedy adequada.
- **Policy gradient convergence**: análise via aproximação estocástica (Robbins-Monro).
- **MCMC**: sequência de estados $\to$ distribuição estacionária. Convergência via teoremas ergódicos.

### Categorias

Limites em categorias generalizam limites de sequências. Direct/inverse limit, colimit, etc. Mesma estrutura algébrica.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita: como calcular limite de sequência.**

### Algoritmo

1. **Substituição direta**: se $a_n = f(n)$ com $f$ função, calcule $\lim_{x \to \infty} f(x)$.
2. **Recursiva**: assuma limite $L$, resolva $L = f(L)$.
3. **Aritmética**: separe em partes conhecidas.
4. **Confronto**: prensar entre conhecidos.
5. **Monótona limitada**: prove existência, calcule depois (ou nem calcule).

### SymPy

```python
import sympy as sp
n = sp.symbols('n', positive=True, integer=True)
print(sp.limit(1/n, n, sp.oo))                # 0
print(sp.limit((1 + 1/n)**n, n, sp.oo))       # E
print(sp.limit(n**(1/n), n, sp.oo))           # 1
print(sp.limit(sp.factorial(n)/n**n, n, sp.oo))  # 0
```

### Hierarquia de crescimento (na ponta da língua)

$$\log n \ll n^a \ll a^n \ll n! \ll n^n.$$

Use pra resolver razões: $\lim n!/2^n = \infty$, $\lim 2^n/n! = 0$, $\lim n^{100}/2^n = 0$.

### Sequências recursivas

Pra $a_{n+1} = f(a_n)$:

1. Verifique convergência (monotonicidade + limitação, ou contração via Banach).
2. Se converge para $L$, então $L = f(L)$.

```python
import numpy as np
def heron(N, a0=1, n_iter=10):
    a = a0
    for _ in range(n_iter):
        a = (a + N/a)/2
    return a
print(heron(2))   # ~1.41421356 (sqrt 2)
```

### Quando o limite NÃO existe

- **Oscilação**: $(-1)^n$, $\sin n$.
- **Divergência**: $a_n \to \pm \infty$.
- **Não monotônica e não Cauchy**: caos.

### Verificação numérica

Implementar e plotar $a_n$ vs $n$ é a forma mais rápida de **conjecturar** o limite. Depois prove.

### Cuidados

- Limite recursivo pressupõe **existência**. Sem prova, pode ser falacioso.
- $(1 + 1/n)^n \neq 1$ no infinito! Indeterminação $1^\infty$.
- Sequência **eventualmente crescente** pode não ser monótona — cuidado em demonstrações.
