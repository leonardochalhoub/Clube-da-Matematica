<Porta nivel="formal" titulo="Limites fundamentais e técnicas">

### Os 3 fundamentais

1. $\displaystyle\lim_{x \to 0} \frac{\sin x}{x} = 1$ (geométrico — confronto).
2. $\displaystyle\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$ (define $e$).
3. $\displaystyle\lim_{x \to 0} \frac{e^x - 1}{x} = 1$ (deriva do anterior).

### Variantes úteis (Tabela)

| Limite | Valor |
|---|---|
| $\lim_{x \to 0} \sin x / x$ | $1$ |
| $\lim_{x \to 0} \sin(kx)/x$ | $k$ |
| $\lim_{x \to 0} \tan x / x$ | $1$ |
| $\lim_{x \to 0} (1-\cos x)/x^2$ | $1/2$ |
| $\lim_{x \to 0} (1-\cos x)/x$ | $0$ |
| $\lim_{x \to 0} \arcsin x / x$ | $1$ |
| $\lim_{x \to 0} \arctan x / x$ | $1$ |
| $\lim_{x \to 0} (e^x - 1)/x$ | $1$ |
| $\lim_{x \to 0} (a^x - 1)/x$ | $\ln a$ |
| $\lim_{x \to 0} \ln(1+x)/x$ | $1$ |
| $\lim_{x \to 0} ((1+x)^a - 1)/x$ | $a$ |
| $\lim_{x \to \infty} (1 + a/x)^x$ | $e^a$ |
| $\lim_{x \to 0^+} x^x$ | $1$ |
| $\lim_{x \to 0^+} x \ln x$ | $0$ |

### Indeterminações comuns (7)

$0/0$, $\infty/\infty$, $\infty - \infty$, $0 \cdot \infty$, $1^\infty$, $0^0$, $\infty^0$ — todas resolvíveis por manipulação ou L'Hôpital (Aula 64).

### Técnicas

- **Fatorar e cancelar** ($0/0$ polinomial).
- **Conjugado** ($0/0$ com raízes).
- **Trigonométrica**: usar $\sin x \approx x$ ($x$ pequeno).
- **Mudança de variável**: $u = 1/x$, $u = x - a$.
- **Logaritmo**: $A^B = e^{B \ln A}$ converte $1^\infty$, $0^0$ em $0/0$.

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Existem três limites **mágicos** que aparecem o tempo todo:

1. **$\sin x / x$** vai pra **um** quando $x$ é pequeno (porque pra ângulos pequeninhos, seno e o ângulo são quase iguais).
2. **$(1 + 1/x)^x$** vai pra um número especial chamado **$e$** quando $x$ é grande (uns $2{,}71...$).
3. **$(e^x - 1)/x$** também vai pra **um** quando $x$ é pequeno.

Esses três são tipo as **três tabuadas mágicas** dos limites.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

Quando você tem $0/0$ ou $\infty/\infty$, o limite **pode ser qualquer coisa** — depende. Mas três limites fundamentais sempre dão a mesma resposta:

1. **$\sin x / x \to 1$**: pra ângulo pequeno, seno quase igual ao ângulo (em radianos).
2. **$(1 + 1/n)^n \to e$**: número $e \approx 2{,}71828...$ — aparece em juros compostos.
3. **$(e^x - 1)/x \to 1$**: deriva do anterior.

**Uso prático**: você quer calcular $\lim_{x \to 0} \sin(5x)/x$. Reescreve: $5 \cdot \sin(5x)/(5x)$. Quando $x \to 0$, $5x \to 0$, e $\sin(5x)/(5x) \to 1$. Resposta: $5$.

**Truque dos juros**: depositar com taxa $r$ por ano, capitalizando $n$ vezes/ano: $(1 + r/n)^n$. Capitalização contínua ($n \to \infty$) dá $e^r$.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Os três limites fundamentais são **fórmulas-base** que você combina via álgebra para resolver todo tipo de indeterminação.

### Fórmula 1: $\lim \sin x / x = 1$

**Por que vale**: pelo círculo unitário, $\sin x < x < \tan x$ para $x \in (0, \pi/2)$. Dividindo por $\sin x$: $1 < x/\sin x < 1/\cos x$. Pelo confronto, $x/\sin x \to 1$, logo $\sin x / x \to 1$.

**Variantes**: $\lim \sin(kx)/x = k$ (mude de variável $u = kx$).

### Fórmula 2: $\lim (1 + 1/x)^x = e$

**Por que vale**: define o número $e \approx 2{,}71828$. Aparece em juros compostos contínuos, decaimento exponencial, distribuição normal.

**Variantes**:
- $\lim (1 + a/x)^x = e^a$
- $\lim (1 + x)^{1/x} = e$ (com $x \to 0$)

### Fórmula 3: $\lim (e^x - 1)/x = 1$

**Por que vale**: derivada de $e^x$ em $x = 0$ é $e^0 = 1$. Conexão com derivada via definição $f'(0) = \lim (f(x) - f(0))/x$.

**Variantes**:
- $\lim (a^x - 1)/x = \ln a$
- $\lim \ln(1+x)/x = 1$
- $\lim ((1+x)^a - 1)/x = a$

### Tabela de aproximações pra $x$ pequeno

| Função | Aproximação |
|---|---|
| $\sin x$ | $x - x^3/6$ |
| $\cos x$ | $1 - x^2/2$ |
| $\tan x$ | $x + x^3/3$ |
| $e^x$ | $1 + x + x^2/2$ |
| $\ln(1+x)$ | $x - x^2/2$ |
| $(1+x)^a$ | $1 + ax$ |

**Aplicação**: pequenos ângulos em física (pêndulo, óptica), juros contínuos, decaimento radioativo.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Os limites fundamentais são consequência da **estrutura analítica** de funções elementares — derivada e série de Taylor.

### Demonstração de $\lim \sin x / x = 1$

Argumento geométrico (Cauchy 1821):

Em $x \in (0, \pi/2)$, considere o triângulo de área $\frac{1}{2}\sin x$, o setor circular de área $\frac{1}{2}x$, e o triângulo externo de área $\frac{1}{2}\tan x$.

$$\frac{\sin x}{2} < \frac{x}{2} < \frac{\tan x}{2} \Rightarrow \cos x < \frac{\sin x}{x} < 1$$

Pelo confronto, $\sin x / x \to 1$ quando $x \to 0^+$. Pela paridade, vale também $x \to 0^-$. ∎

### Demonstração de $\lim (1 + 1/n)^n = e$

Mostra-se que $a_n = (1 + 1/n)^n$ é monótona crescente e limitada, logo converge. O limite **define** $e$.

Por binômio de Newton:
$$\left(1 + \frac{1}{n}\right)^n = \sum_{k=0}^n \binom{n}{k} \frac{1}{n^k} = \sum_{k=0}^n \frac{1}{k!} \cdot \frac{n!}{n^k (n-k)!}$$

Como $n \to \infty$, cada termo $\to 1/k!$, e a soma vai para $\sum_{k=0}^\infty 1/k! = e$.

### Demonstração de $\lim (e^x - 1)/x = 1$

Use $\lim_{x \to 0} (1 + x)^{1/x} = e$. Tome logaritmo: $\ln((1+x)^{1/x}) = \ln(1+x)/x \to 1$. Substituindo $y = e^x - 1$ (então $x = \ln(1+y)$): $(e^x - 1)/x = y/\ln(1+y) \to 1$. ∎

### Aplicações em engenharia

- **Capitalização contínua**: $V(T) = V_0 e^{rT}$, derivado de $\lim_n (1 + r/n)^{nT} = e^{rT}$.
- **Decaimento radioativo**: $N(t) = N_0 e^{-\lambda t}$ — meia-vida $T_{1/2} = \ln 2 / \lambda$.
- **Aproximação para pequenos ângulos**: pêndulo $\ddot \theta + (g/L) \sin \theta = 0$ vira $\ddot \theta + (g/L) \theta = 0$ (oscilador harmônico), válido até ~10°.
- **Estabilidade BIBO**: $\lim_{n \to \infty} h[n] = 0$ ⇒ sistema estável.
- **Frequência angular**: $\omega = \lim_{T \to 0} 2\pi / T$, conexão entre período e frequência.

### Aplicação à derivada

Derivada de $e^x$ em zero:
$$\frac{d}{dx} e^x \Big|_{x=0} = \lim_{h \to 0} \frac{e^h - e^0}{h} = \lim_{h \to 0} \frac{e^h - 1}{h} = 1.$$

Logo $(e^x)' = e^x$ em todo lugar (via $e^{x+h} = e^x e^h$).

### Indeterminações por logaritmo

$\lim_{x \to 0^+} x^x$: $1^\infty$ ou $0^0$? Reescreva: $x^x = e^{x \ln x}$, e $x \ln x \to 0$ (mostre por confronto/L'Hôpital). Logo $x^x \to e^0 = 1$.

<Porta nivel="40" titulo="Pra profissional sênior">

Os limites fundamentais são **manifestações de estruturas analíticas profundas**.

### $\sin x / x$ e funções inteiras

$\sin x / x$ se estende a uma função **inteira** $\text{sinc}(x)$, com série
$$\text{sinc}(x) = \prod_{n=1}^\infty \left(1 - \frac{x^2}{n^2 \pi^2}\right).$$
Esta é a **fatorização de Weierstrass** de $\sin$. Conexão com $\zeta(2) = \pi^2/6$ via expansão.

### $e$ e a álgebra

$e = \exp(1)$, onde $\exp(z) = \sum z^n/n!$ é a única função inteira com $f' = f$ e $f(0) = 1$. **Caracterização única** por equação diferencial.

### Generalizações em álgebra

- **Exponencial de matriz**: $e^A = \sum A^n/n!$. Resolve $\dot x = Ax$.
- **Exponencial em grupos de Lie**: $\exp : \mathfrak{g} \to G$, mapeando álgebra para grupo.
- **Funções Gamma**: $\Gamma(z) = \lim_n (n! n^z)/(z(z+1)\cdots(z+n))$ via produto.

### Operadores e cálculo simbólico

- **Cálculo umbral / Sheffer**: derivada $\partial$ tem "exponencial" $e^{a\partial}$ que age como translação por $a$.
- **Cálculo de Heaviside**: $D = d/dt$, $e^{-Dh}$ é translação. Limites fundamentais aparecem na expansão.

### Análise funcional

- $\sinh, \cosh, \sin, \cos$ formam **base** do espaço de soluções de $y'' = \pm y$.
- $e^{i\omega t}$ é **autovetor** do operador $d/dt$ — base da análise de Fourier.

### Demonstração via série (rigorosa)

$\sin x = x - x^3/6 + x^5/120 - \ldots$ Então
$$\frac{\sin x}{x} = 1 - \frac{x^2}{6} + \frac{x^4}{120} - \ldots \to 1.$$

$e^x = 1 + x + x^2/2 + \ldots$ Então
$$\frac{e^x - 1}{x} = 1 + \frac{x}{2} + \frac{x^2}{6} + \ldots \to 1.$$

### Aplicações em probabilidade

- **Função característica**: $\varphi(t) = E[e^{itX}]$. $\varphi(0) = 1$, $\varphi'(0) = i E[X]$.
- **Lei de Poisson**: $P(X = k) = e^{-\lambda} \lambda^k / k!$. Limite de binomial $(1 + \lambda/n)^n$.
- **Fórmula de Stirling**: $n! \sim \sqrt{2\pi n}(n/e)^n$. Aproximação assintótica.

### Renormalização e física

Em QFT, **regularização zeta**: $\sum_{n=1}^\infty n = \zeta(-1) = -1/12$. Limites fundamentais aparecem em continuação analítica.

### Numérico

- Algoritmos computam $e^x$, $\sin x$ via série truncada + redução de argumento (CORDIC).
- Cuidado com cancelamento numérico em $(e^x - 1)/x$ para $x$ pequeno: use `expm1(x)` (built-in em C, Python, MATLAB).

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita: como usar limites fundamentais.**

### Passos para qualquer indeterminação

1. **Identifique o tipo**: $0/0$, $\infty/\infty$, $1^\infty$, etc.
2. **Tente reescrever** em forma de limite fundamental:
   - $\sin(\text{algo}) / \text{algo}$ → use $\sin x / x$.
   - $(1 + \text{algo})^{1/\text{algo}}$ → use $e$.
   - $(e^{\text{algo}} - 1) / \text{algo}$ → use $(e^x - 1)/x$.
3. **Mude de variável** se necessário ($u = $ argumento dentro).
4. **Confira numericamente** ou via SymPy.

### Casos típicos

| Indeterminação | Estratégia |
|---|---|
| $0/0$ trig | Usar $\sin x \approx x$, $1 - \cos x \approx x^2/2$ |
| $0/0$ exp/log | Usar $(e^x - 1)/x \to 1$, $\ln(1+x)/x \to 1$ |
| $1^\infty$ | $A^B = e^{B \ln A}$, manipular $B \ln A$ |
| $0^0$ ou $\infty^0$ | Igual: $A^B = e^{B \ln A}$, $\ln A \to -\infty$ ou $+\infty$ |
| $\infty - \infty$ | Fator comum, conjugado, ou racional comum |

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
print(sp.limit(sp.sin(5*x)/x, x, 0))           # 5
print(sp.limit((1 + 2/x)**x, x, sp.oo))        # exp(2)
print(sp.limit((sp.exp(3*x) - 1)/x, x, 0))     # 3
print(sp.limit(x**x, x, 0, '+'))               # 1
print(sp.limit((1 - sp.cos(x))/x**2, x, 0))    # 1/2
```

### Wolfram Alpha

`limit (1 + 3/x)^x as x -> infinity` → $e^3$.

### Quando NÃO usar limites fundamentais

- **$x \to a \neq 0$** com expressão trig: pode ser que substituição direta funciona.
- **Funções não relacionadas a $\sin, e, \ln$**: outras técnicas.

### Cuidados

- O **argumento** de $\sin$ deve ser igual ao denominador. $\sin(5x)/x \neq 1$ — é $5$.
- $(1 + 1/n)^n$ vs $(1 + 2/n)^n$ vs $(1 + 1/n)^{2n}$ — tudo dá coisas diferentes.
