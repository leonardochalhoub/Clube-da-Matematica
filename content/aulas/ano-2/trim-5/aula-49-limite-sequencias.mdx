<Porta nivel="formal" titulo="Sequências e teoremas">

### Definição

$(a_n) \to L$ se: $\forall \eps > 0, \exists N \in \mathbb{N} : n \geq N \Rightarrow |a_n - L| < \eps$.

### Teoremas-chave

- **Unicidade**: limite, se existe, é único.
- **Aritmética**: $\lim(a_n \pm b_n) = \lim a_n \pm \lim b_n$, etc.
- **Confronto**: $a_n \leq b_n \leq c_n$ e $\lim a_n = \lim c_n = L \Rightarrow \lim b_n = L$.
- **Monótona limitada**: toda monótona crescente limitada superiormente converge.
- **Bolzano-Weierstrass**: toda sequência limitada tem subsequência convergente.
