### Limites trigonométricos fundamentais

| Limite | Valor |
|---|---|
| $\lim_{x \to 0} \sin x / x$ | $1$ |
| $\lim_{x \to 0} \sin(kx)/x$ | $k$ |
| $\lim_{x \to 0} \tan x / x$ | $1$ |
| $\lim_{x \to 0} (1 - \cos x)/x$ | $0$ |
| $\lim_{x \to 0} (1 - \cos x)/x^2$ | $1/2$ |
| $\lim_{x \to 0} \sin^2 x / x^2$ | $1$ |
| $\lim_{x \to 0} \arcsin x / x$ | $1$ |
| $\lim_{x \to 0} \arctan x / x$ | $1$ |

### Identidades úteis

- $1 - \cos x = 2\sin^2(x/2)$
- $\sin(a + b) = \sin a \cos b + \cos a \sin b$
- $\cos(a + b) = \cos a \cos b - \sin a \sin b$
- $1 - \cos x = (1 - \cos^2 x)/(1 + \cos x) = \sin^2 x / (1 + \cos x)$

### Técnicas

1. **Substituição $u$**: $\sin(2x)/x = 2 \cdot \sin(2x)/(2x) \to 2$.
2. **Identidades** para reescrever em forma fundamental.
3. **Conjugado**: $1 - \cos x = \sin^2 x / (1 + \cos x)$.
4. **Confronto**: $-1 \leq \sin x \leq 1$, multiplique por algo $\to 0$.

### Demonstração de $\lim (1 - \cos x)/x^2 = 1/2$

$\frac{1 - \cos x}{x^2} = \frac{2 \sin^2(x/2)}{x^2} = \frac{1}{2} \cdot \frac{\sin^2(x/2)}{(x/2)^2} \to \frac{1}{2} \cdot 1 = \frac{1}{2}$.

### Demonstração de $\lim \tan x / x = 1$

$\tan x / x = (\sin x / x) \cdot (1 / \cos x) \to 1 \cdot 1 = 1$.

<Porta nivel="5" titulo="Pra criança de 5 anos">

Quando o ângulo é **muito pequeno**, o **seno** dele é quase igual ao próprio ângulo. Tipo, $\sin 0{,}01 \approx 0{,}01$.

Por isso $\sin x / x \to 1$ — eles ficam quase iguais.

E **um menos cosseno** é quase igual a **metade do quadrado do ângulo**. Coisas pequeninhas seguem regras simples.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

Pra **ângulos pequenos** (em radianos):
- $\sin x \approx x$
- $\cos x \approx 1 - x^2/2$
- $\tan x \approx x$

Daí saem todos os limites:
- $\sin(3x)/x \to 3$ (porque $\sin(3x) \approx 3x$)
- $(1 - \cos x)/x^2 \to 1/2$ (porque $1 - \cos x \approx x^2/2$)
- $\tan x / x \to 1$ (porque $\tan x \approx x$)

**Aplicação na física**: pêndulo de pequenas oscilações. A equação real é $\ddot \theta = -(g/L) \sin \theta$. Pra $\theta$ pequeno, $\sin \theta \approx \theta$, e fica $\ddot \theta = -(g/L) \theta$ — oscilador simples.

**Sem essa aproximação**, o problema fica muito mais difícil (envolve integrais elípticas). Por isso engenheiros amam pequenos ângulos.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Limites trig com $x \to 0$ se reduzem a três casos via manipulação:

### Caso 1: $\sin(?)/?$

Se você consegue escrever na forma $\sin(\text{algo})/\text{algo}$, vai pra 1. Use mudança de variável $u = $ argumento.

**Exemplo**: $\lim_{x \to 0} \sin(7x)/(3x) = (7/3) \lim_{x \to 0} \sin(7x)/(7x) = 7/3$.

### Caso 2: $(1 - \cos)/?$

Use identidade $1 - \cos x = 2 \sin^2(x/2)$ ou conjugado.

**Exemplo**: $\lim_{x \to 0} (1 - \cos x)/(x \sin x) = (x^2/2)/(x \cdot x) = 1/2$ (usando $\sin x \approx x$).

### Caso 3: combinações $\tan, \sec, \cot$

Reescreva em $\sin, \cos$.

**Exemplo**: $\lim_{x \to 0} \tan(2x)/\sin(3x) = \lim (2x)/(3x) = 2/3$ (aproximação).

### Truques de mudança de variável

Para $\lim_{x \to a}$ trig, use $u = x - a$ pra trazer pra zero.

**Exemplo**: $\lim_{x \to \pi} \sin x / (x - \pi)$. Seja $u = x - \pi$. Então $\sin x = \sin(u + \pi) = -\sin u$. Resultado: $\lim_{u \to 0} -\sin u / u = -1$.

### Tabela de aproximações para $x$ pequeno

| Função | Aproximação (até $x^3$) |
|---|---|
| $\sin x$ | $x - x^3/6$ |
| $\cos x$ | $1 - x^2/2$ |
| $\tan x$ | $x + x^3/3$ |
| $\arcsin x$ | $x + x^3/6$ |
| $\arctan x$ | $x - x^3/3$ |

Use pra resolver mais rápido.

### Exemplo desafiador

$\lim_{x \to 0} (\tan x - \sin x)/x^3$:

$\tan x - \sin x = \sin x (1/\cos x - 1) = \sin x \cdot (1 - \cos x)/\cos x$.

Logo: $\lim = (\sin x / x) \cdot ((1 - \cos x)/x^2) \cdot (1/\cos x) = 1 \cdot (1/2) \cdot 1 = 1/2$.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Limites trigonométricos são essenciais em **derivadas** ($\sin', \cos', \tan'$), **séries de Fourier**, **análise harmônica**.

### Demonstrações chave

**$\lim \sin x / x = 1$**: argumento geométrico — área de triângulo, setor, triângulo. Pelo confronto, conclui.

**$(1 - \cos x)/x^2 \to 1/2$**:
$$\frac{1 - \cos x}{x^2} = \frac{(1-\cos x)(1+\cos x)}{x^2 (1+\cos x)} = \frac{\sin^2 x}{x^2(1+\cos x)} = \frac{\sin^2 x}{x^2} \cdot \frac{1}{1+\cos x} \to 1 \cdot \frac{1}{2} = \frac{1}{2}.$$

**$\arcsin x / x \to 1$**: $u = \arcsin x$, $x = \sin u$, $u \to 0$ quando $x \to 0$. $\arcsin x / x = u/\sin u \to 1$.

### Conexão com derivadas

- $(\sin x)' = \cos x$ vem de $\lim_{h \to 0} (\sin(x+h) - \sin x)/h$. Manipulação usa $\sin(x+h) = \sin x \cos h + \cos x \sin h$, então razão $= \sin x (\cos h - 1)/h + \cos x \sin h / h \to 0 + \cos x = \cos x$.
- $(\cos x)' = -\sin x$ análogo.
- $(\tan x)' = \sec^2 x$ via quociente.

### Series de Taylor (Trim 7)

$$\sin x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n+1}}{(2n+1)!}, \quad \cos x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n}}{(2n)!}.$$

Truncadas dão aproximações de pequenos ângulos. Convergência uniforme em compactos.

### Aplicações em engenharia

- **Pequenos ângulos em mecânica**: pêndulo $T = 2\pi\sqrt{L/g}$ vale para $\theta \lesssim 10°$. Erro relativo cresce como $\theta^2/16$.
- **Óptica gaussiana** (paraxial): $\sin \theta \approx \theta$. Lentes finas, refração.
- **Linearização**: sistemas dinâmicos $\dot x = f(x)$ linearizados via Jacobiano em ponto de equilíbrio.
- **Modulação**: $\cos(\omega t + \phi(t))$ com $\phi$ pequeno expandido para AM-equivalente.

### Análise de Fourier

Coeficientes de Fourier $a_n = \frac{1}{T}\int f(t) \cos(2\pi nt/T) dt$. Limite $\sum a_n^2$ converge (Parseval). Limites trig aparecem em verificações de convergência ponto a ponto (Dini, Dirichlet).

### Confronto avançado

$\lim_{x \to \infty} \sin x$ não existe, mas $\lim_{x \to \infty} (\sin x)/x = 0$ por confronto $-1/x \leq \sin x/x \leq 1/x$.

$\lim_{x \to 0} x \sin(1/x) = 0$ por confronto. **Não** é o mesmo que $\lim x \cdot \lim \sin(1/x)$ — o segundo nem existe. Confronto resolve sem essa fatoração.

### Em circuitos AC

- Impedância $Z(j\omega) = R + jX$. Magnitude $|Z| = \sqrt{R^2 + X^2}$. Limites em pequena freq./alta freq.
- $\sin(\omega t)$ em $\omega = 0$: vai pra zero. Em $\omega \to \infty$: oscila, sem limite.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

Limites trigonométricos são manifestações de **estrutura analítica** profunda: $\sin, \cos$ são funções inteiras com padrão específico de zeros.

### Fatorização de Weierstrass

$$\sin(\pi x) = \pi x \prod_{n=1}^\infty \left(1 - \frac{x^2}{n^2}\right).$$

Tomando logaritmo e derivando, dá identidade de Mittag-Leffler:
$$\pi \cot(\pi x) = \frac{1}{x} + \sum_{n=1}^\infty \frac{2x}{x^2 - n^2}.$$

Conexão com $\zeta(2k)$.

### Caracterização funcional

$\sin, \cos$ são as únicas funções $\mathbb{R} \to \mathbb{R}$ satisfazendo:
- $f(x+y) = f(x)g(y) + g(x)f(y)$, $g(x+y) = g(x)g(y) - f(x)f(y)$
- $f'(0) = 1$, $g(0) = 1$
- $f, g$ contínuas

Equivalente: solução de $y'' + y = 0$ com condições iniciais.

### Análise complexa

$e^{ix} = \cos x + i \sin x$ (Euler). Limites trig viram limites de $e^{iz}$ em $z \to 0$. Holomorfia em $\mathbb{C}$ inteiro. Fatorização de Hadamard.

### Aplicações em PDE

- **Equação da onda** $u_{tt} = c^2 u_{xx}$: solução $u(x, t) = f(x - ct) + g(x + ct)$ envolve $\sin, \cos$ via separação de variáveis.
- **Schrödinger livre**: $\psi(x, t) = \int A(k) e^{i(kx - \omega t)} dk$, $\omega = k^2/2m$. Pacotes via $\sin, \cos$.
- **Calor com BCs Dirichlet**: solução $\sum b_n \sin(n\pi x/L) e^{-n^2 \pi^2 D t/L^2}$.

### Análise harmônica

Teorema de Carleson: série de Fourier converge ae para $f \in L^2$. Demonstração depende de estimativas finas envolvendo limites trig.

### Aproximação de Padé

Para $\sin x$, aproximante Padé $[m/n]$ pode ser melhor que Taylor. Em particular, $\tan x \approx x(1 - x^2/15)/(1 - 2x^2/5)$ tem erro menor que Taylor truncada do mesmo grau.

### Cálculo de operadores

- **Função de matriz**: $\sin A = (e^{iA} - e^{-iA})/(2i)$. Bem definido para qualquer matriz.
- **Spectral mapping**: spec($\sin A$) = $\{\sin \lambda : \lambda \in $ spec $A\}$.
- **Aplicação**: solução de $\ddot x = -\Omega^2 x$ é $x(t) = \cos(\Omega t) x_0 + \Omega^{-1} \sin(\Omega t) v_0$.

### Numérico

- **Argument reduction**: pra $\sin(10^{15})$, reduce mod $\pi$ com **Payne-Hanek** (alta precisão).
- **CORDIC**: hardware-friendly, calcula $\sin, \cos$ por iteração de rotações.
- **Cancelamento numérico**: $1 - \cos x$ em float32 pra $x$ pequeno tem erro relativo grande. Usar `2 * sin(x/2)**2` em vez disso.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita: limites trigonométricos.**

### Algoritmo

1. **Identifique forma**: $\sin/?, \cos$, $(1 - \cos)/?, \tan/?$, etc.
2. **Reescreva** em forma fundamental via:
   - Mudança de variável.
   - Identidades $\sin^2 + \cos^2 = 1$, $1 - \cos x = 2\sin^2(x/2)$, etc.
3. **Confira numericamente** ou via SymPy.

### SymPy

```python
import sympy as sp
x = sp.symbols('x')
# Linear no argumento
print(sp.limit(sp.sin(7*x)/x, x, 0))            # 7
# Razão de senos
print(sp.limit(sp.sin(3*x)/sp.sin(5*x), x, 0))  # 3/5
# (1 - cos)/x^2
print(sp.limit((1 - sp.cos(x))/x**2, x, 0))     # 1/2
# Difíceis
print(sp.limit((sp.tan(x) - sp.sin(x))/x**3, x, 0))  # 1/2
```

### Cheat sheet

Para $x \to 0$:

| Forma | Limite |
|---|---|
| $\sin(ax)/x$ | $a$ |
| $\sin(ax)/\sin(bx)$ | $a/b$ |
| $\sin(ax)/\tan(bx)$ | $a/b$ |
| $(1 - \cos(ax))/x^2$ | $a^2/2$ |
| $\arcsin(ax)/x$ | $a$ |
| $\arctan(ax)/x$ | $a$ |
| $(\sin x)^2/x^2$ | 1 |
| $\sin(\sin x)/x$ | 1 |

### Quando NÃO usar essas

- Limite envolve $x \to a \neq 0$: faça mudança $u = x - a$.
- Função com $1/x$ no argumento ($\sin(1/x)$): aproveite que oscila — confronto.

### Cuidados comuns

- $\sin(2x)/x \neq 2 \sin x / x$: deve ser $2 \cdot \sin(2x)/(2x) \to 2$.
- $\sin(x^2)/x \neq \sin x \cdot x$: deve ser $\sin(x^2)/x = x \cdot \sin(x^2)/x^2 \to 0$.
- $(1 - \cos x)/x \neq (1 - \cos x)/x^2$: o primeiro vai a 0, o segundo a 1/2.
