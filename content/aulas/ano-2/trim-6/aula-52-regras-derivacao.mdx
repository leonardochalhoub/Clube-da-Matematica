### Tabela básica

| Função | Derivada |
|---|---|
| $c$ (constante) | $0$ |
| $x^n$ | $n x^{n-1}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $\tan x$ | $\sec^2 x$ |
| $e^x$ | $e^x$ |
| $\ln x$ | $1/x$ |
| $a^x$ | $a^x \ln a$ |
| $\log_a x$ | $1/(x \ln a)$ |

### Regras operatórias

- $(f + g)' = f' + g'$
- $(cf)' = c f'$
- $(fg)' = f'g + fg'$
- $(f/g)' = (f'g - fg')/g^2$
