<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Trimestre 6 é a **caixa de ferramentas completa de derivação clássica**. Pra cada problema novo:

1. **Reconheça o tipo**: explícita? composta? implícita? definida por inversa? superior?
2. **Aplique a técnica certa** — sempre cadeia + tabela.
3. **Simplifique algébricamente**.
4. **Verifique unidades / sinais / ordem de grandeza**.

### Workflow universal de problema de derivada

1. Escreva a função com clareza.
2. Identifique **tipo**.
3. Liste **ferramentas** necessárias (cadeia? produto? quociente? inversa?).
4. **Derive** uma camada por vez.
5. **Junte** e simplifique.

### Direções de aprofundamento

1. **Multivariado**: Jacobiano, Hessian, gradiente. Generalização sistemática de tudo que você fez.
2. **Funcional**: derivadas em espaços de função (Fréchet, calculus of variations).
3. **Geométrico**: derivada exterior $d$ em formas, conexões em variedades.
4. **Numérico**: autodiff (forward/reverse), spectral methods.
5. **Estocástico**: cálculo de Itô, Malliavin.
6. **Não-suave**: subgradiente, tangentes generalizados, B-derivative.

### Conhecimento operacional do quant/eng/ML moderno

- Backprop = cadeia.
- Greeks = derivadas parciais.
- Gradient descent = linearização.
- Newton = linearização + zero.
- Sensitivity analysis = derivada paramétrica.
- Implicit differentiation through optimization = TFI.

### Bibliografia do nível seguinte

### Wolfram / SymPy referência

```python
import sympy as sp
x, y, t = sp.symbols('x y t')

# Derivada simples
sp.diff(x**3 + sp.sin(x), x)

# Derivada implícita
y_func = sp.Function('y')(x)
expr = x**2 + y_func**2 - 25
yprime = sp.solve(sp.diff(expr, x), sp.diff(y_func, x))[0]

# Limite
sp.limit((x**2 - 4)/(x - 2), x, 2)

# Linearização
f = sp.sqrt(x)
L = f.subs(x, 4) + sp.diff(f, x).subs(x, 4) * (x - 4)
```
