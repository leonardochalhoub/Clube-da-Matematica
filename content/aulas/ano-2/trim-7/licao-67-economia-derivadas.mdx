### Definições

<Definicao titulo="Funções marginais">
Dadas funções de **custo** $C(q)$, **receita** $R(q)$, **lucro** $\pi(q) = R(q) - C(q)$:
$$MC(q) = C'(q), \quad MR(q) = R'(q), \quad M\pi(q) = \pi'(q).$$
$MC$ aproxima o custo de produzir a $(q+1)$-ésima unidade.
</Definicao>

### Maximização do lucro

$\pi'(q) = 0 \iff R'(q) = C'(q) \iff MR = MC$.

**Condição de segunda ordem**: $\pi''(q^*) < 0 \iff MR'(q^*) < MC'(q^*)$.

### Custo médio e custo marginal

$\bar C(q) = C(q)/q$ é o **custo médio**. 

$\bar C'(q) = (C'(q) q - C(q))/q^2 = (MC - \bar C)/q$.

Logo, $\bar C$ é mínimo onde $\bar C = MC$. **Curva de custo marginal cruza curva de custo médio em seu mínimo.**

### Elasticidade-preço da demanda

<Definicao titulo="Elasticidade">
Para demanda $q = D(p)$:
$$\varepsilon(p) = \frac{dq}{dp} \cdot \frac{p}{q} = \frac{p}{q} D'(p).$$
$|\varepsilon| > 1$: elástica. $|\varepsilon| < 1$: inelástica. $|\varepsilon| = 1$: unitária.
</Definicao>

### Receita e elasticidade

$R = pq$, então $dR/dp = q + p \cdot dq/dp = q(1 + \varepsilon)$.

- Elástica ($\varepsilon < -1$): $dR/dp < 0$, baixar preço aumenta receita.
- Inelástica ($-1 < \varepsilon < 0$): $dR/dp > 0$, subir preço aumenta receita.

### Mark-up rule

Para monopolista: $MR = p(1 + 1/\varepsilon)$. Lucro máximo: $p(1 + 1/\varepsilon) = MC \Rightarrow p = MC \cdot \varepsilon/(\varepsilon + 1)$.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

Em economia, **derivada** vira **marginal**:

- $C(q)$ = custo total de produzir $q$ unidades. $MC = C'$ = quanto custa fazer **mais uma**.
- $R(q)$ = receita (dinheiro que entra). $MR = R'$ = quanto entra de **mais uma**.
- Lucro $\pi = R - C$. **Lucro máximo**: produza mais até $MR = MC$. Se $MR > MC$, produza mais. Se $MR < MC$, pare.

**Exemplo**: $C(q) = 100 + 5q + 0{,}1q^2$, $p = 50$ (preço fixo). $R = 50q$, $MR = 50$. $MC = 5 + 0{,}2q$. $MR = MC \Rightarrow 50 = 5 + 0{,}2q \Rightarrow q = 225$.

**Elasticidade**: se você sobe preço $1\%$ e quantidade vendida cai $2\%$, a elasticidade é $-2$. Significa que **demanda é elástica** (sensível).

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Análise marginal é o **modelo central** da microeconomia neoclássica. Cada decisão se reduz a "vale a pena fazer mais uma unidade?"

### Funções básicas

- **Custo** $C(q)$: total para produzir $q$. Tipicamente $C(q) = F + V(q)$ (fixo + variável).
- **Receita** $R(q) = p(q) \cdot q$. Em concorrência perfeita, $p$ fixo. Em monopólio, $p$ depende de $q$.
- **Lucro** $\pi(q) = R(q) - C(q)$.

### Custo médio vs marginal

- $\bar C(q) = C/q$.
- $MC(q) = C'(q)$.
- $\bar C$ tem mínimo onde $\bar C = MC$. Geometria: tangente partindo da origem ao gráfico de $C$ tem inclinação $\bar C$; é mínima onde toca tangentemente.

### Elasticidade

$\varepsilon = (\Delta q\,/q)/(\Delta p/p) \approx D'(p) \cdot p/q$.

- $|\varepsilon| > 1$ elástica (luxo).
- $|\varepsilon| < 1$ inelástica (necessidade — sal, gasolina).

### Por que importa

Política de preço (ex.: governo aumenta imposto de cigarro: tem efeito em quantidade fumada? Depende da elasticidade.) Política tarifária. Maximização de receita por monopolistas. Decisões de investimento.

### Exemplo trabalhado

Demanda $q = 100 - 2p$. Receita $R(q) = pq$, com $p = (100-q)/2$, ou seja $R(q) = q(100-q)/2 = 50q - q^2/2$.

$MR(q) = 50 - q$.

Custo $C(q) = 10q + q^2/4$. $MC(q) = 10 + q/2$.

$MR = MC \Rightarrow 50 - q = 10 + q/2 \Rightarrow 3q/2 = 40 \Rightarrow q^* \approx 26{,}67$.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Análise marginal aparece tanto em economia clássica quanto em **engenharia econômica** (decisões de projeto), **finanças** (utilidade marginal), e **operations research** (precificação dinâmica).

### Função custo típica

$C(q) = F + a q + b q^2 + c q^3$ (cúbica) reflete: custo fixo $F$, custos variáveis lineares, eventual deseconomia de escala em $q^2$, $q^3$.

### Curvas U-shaped

$\bar C(q)$ tem formato $\cup$ típico em manuais: cai com escala, depois sobe com congestionamento.

### Monopólio vs concorrência

| | Concorrência perfeita | Monopólio |
|---|---|---|
| $p$ | dado pelo mercado | $p(q)$, escolhido |
| $MR$ | $p$ | $p + q\,dp/dq$ |
| Decisão | $MC = p$ | $MC = MR < p$ |

Daí o **markup**: $p^* = MC/(1 + 1/\varepsilon)$.

### Aplicação: pricing dinâmico

Uber, airlines, e-commerce ajustam preço em real-time pra maximizar receita esperada. Modelos: bandits, RL, mas o **fundamento** é elasticidade momentânea.

### Aplicação: sustentabilidade e externalidades

Pigou (1920) propõe imposto = custo marginal externo (ex.: poluição). Internaliza externalidade. Coase (1960) discute alocação de direitos de propriedade. Tudo manipulação de $MC$ percebido pelos agentes.

### Conexão com Lagrange

Em equilíbrio competitivo, $\lambda$ (multiplicador de Lagrange da restrição orçamentária) é a **utilidade marginal da renda**. Em macro, $\lambda$ vira a taxa de juros real (modelo Ramsey).

### Conexão com cálculo das variações

Em modelos dinâmicos (Ramsey-Cass-Koopmans), o agente maximiza $\int U(c(t)) e^{-\rho t} dt$ sujeito a EDO de capital. Equação de Euler-Lagrange dá $\dot c/c = (r - \rho)/\sigma$ — a famosa **regra de Keynes-Ramsey**.

### Onde isso falha

Modelo neoclássico assume agentes racionais maximizando utilidade. Behavioral economics (Kahneman, [Prêmio Nobel 2002](https://www.nobelprize.org/prizes/economic-sciences/2002/kahneman/facts/)) mostra desvios sistemáticos. Mas marginal **continua sendo** o framework dominante.

<Porta nivel="40" titulo="Pra profissional sênior">

Análise marginal é o ponto de partida; a economia moderna trabalha em equilíbrio geral, dinâmica estocástica e jogos.

### Equilíbrio geral

Walras (1874) — todos os mercados em equilíbrio simultâneo. Existência via teorema de ponto fixo (Arrow-Debreu, 1954, [Prêmio Nobel](https://www.nobelprize.org/prizes/economic-sciences/1972/)). Sistema é um sistema de equações $MC_i = p_i$ pra cada bem $i$.

### Teoria dos jogos

Quando agentes interagem, equilíbrio de Nash substitui o "monopolista isolado". Cournot, Bertrand, Stackelberg modelam diferentes timing/estrutura. **First-order condition** continua válida; mas é em derivada parcial em relação à própria escolha, fixando rivais.

### Tempo e incerteza

Hamilton-Jacobi-Bellman em controle ótimo estocástico. **Black-Scholes** ([prêmio Nobel 1997](https://www.nobelprize.org/prizes/economic-sciences/1997/)) é caso especial em finanças: preço de opção como solução de PDE com fronteira.

### Demonstração de Black-Scholes via marginal

Construir portfólio replicante: $\Pi = -V + \Delta S$ onde $\Delta = \partial V/\partial S$. Itô + replicação livre de risco $\Rightarrow$ $V_t + \frac{1}{2}\sigma^2 S^2 V_{SS} + rS V_S - rV = 0$.

### Mecanism design

Como projetar mercado tal que equilíbrio seja eficiente? Vickrey (1961, [Prêmio Nobel](https://www.nobelprize.org/prizes/economic-sciences/1996/)), Maskin, Myerson. Leilões VCG são caso central. Sempre via análise marginal.

### Crítica e horizontes

Stiglitz, Akerlof (informação assimétrica, [Prêmio Nobel 2001](https://www.nobelprize.org/prizes/economic-sciences/2001/)), Krugman (geografia econômica, [Prêmio Nobel 2008](https://www.nobelprize.org/prizes/economic-sciences/2008/)) mostraram limites do paradigma marginal puro. Behavioral, agent-based, network economics são extensões.

### Conexão com ML

Reinforcement learning em precificação dinâmica = generalização de marginal pra tomada de decisão sequencial sob incerteza. Bellman equation generaliza FOC marginal.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita prática:**

1. Identifique funções: custo $C(q)$, demanda $p(q)$ ou $q(p)$.
2. Calcule **marginais**: $MC = C'$, $MR = (pq)' = p + q\,dp/dq$.
3. Pra lucro: ache $q^*$ tal que $MR = MC$. Confirme $\pi'' < 0$.
4. Pra elasticidade: $\varepsilon = (dq/dp)(p/q)$.
5. Pra preço ótimo via markup: $p^* = MC/(1 + 1/\varepsilon)$.

### Conferência

- $C$ deve ser crescente, geralmente $C''(q) > 0$ pra $q$ grande (deseconomia).
- $R$ tem máximo onde $\varepsilon = -1$.
- $\pi$ máximo entre $\bar C_{\min}$ e ponto onde $MR = MC$.

### Verificação Python

```python
import sympy as sp
q = sp.symbols('q', positive=True)
C = 100 + 5*q + 0.1*q**2
R = 50*q  # preço fixo
profit = R - C
qstar = sp.solve(sp.diff(profit, q), q)
print(qstar)  # [225.0]
```
