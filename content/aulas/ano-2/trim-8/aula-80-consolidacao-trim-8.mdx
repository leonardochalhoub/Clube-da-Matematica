### Hierarquia conceitual

1. **Dados brutos** → estatísticas descritivas (média, $\sigma$, quartis).
2. **Modelo probabilístico** (binomial, Poisson, normal) → distribuições.
3. **TCL** → distribuição da média amostral.
4. **Inferência** (IC, teste de hipótese) → conclusões com incerteza.
5. **Bayes** → atualização de crenças com evidência.
6. **Regressão** → relações entre variáveis.

### Workflow padrão

```
Dados → EDA (boxplot, histograma) → 
    Estatística descritiva (μ, σ, r) → 
    Modelo (escolha distribuição) → 
    Estimação (MLE, Bayes) → 
    Diagnóstico (resíduos, Q-Q) → 
    Inferência (IC, p) → 
    Decisão
```

Vamos juntar tudo num **caso real**: você é cientista de dados de uma rede de pizzarias.

1. **Tempo de entrega** (dados): coleta 1000 entregas. Calcula **média = 32 min**, **$\sigma$ = 5 min** (Aula 72).

3. **Modelo**: tempo de entrega $\approx \mathcal N(32, 5^2)$ (Aula 76).
4. **TCL** (Aula 77): média de 25 pedidos diários $\approx \mathcal N(32, 1^2)$.
5. **SLA**: 95\% das entregas em até **40 min** ($z = (40-32)/5 = 1{,}6$ → 95\%-ish OK).
6. **Correlação** (Aula 78): trânsito vs. tempo de entrega: $r = 0{,}7$.
7. **Bayes** (Aula 79): se cliente reclama, $P(\text{atraso real} | \text{reclamação})$.

**Decisão**: investir em mais entregadores no bairro problemático.

### Erros típicos a evitar

1. **Aplicar normal sem checar**: testes rápidos (histograma, Q-Q).
2. **Confundir correlação com causalidade**: sempre questione.
3. **Ignorar prevalência** (base rate fallacy).
4. **Reportar média em distribuição assimétrica**: prefira mediana.
5. **TCL com $n$ pequeno** em distribuição assimétrica.
6. **Aproximar binomial por normal sem checar $np$ e $n(1-p)$**.

### Pipeline em Python

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
import statsmodels.api as sm

df = pd.read_csv('dados.csv')
print(df.describe())
sns.pairplot(df)
plt.show()

# Regressão
y = df['target']
X = sm.add_constant(df.drop('target', axis=1))
model = sm.OLS(y, X).fit()
print(model.summary())

# Diagnóstico
residuos = model.resid
plt.scatter(model.fittedvalues, residuos)
plt.axhline(0)
plt.show()
```

### Modernização do trimestre

Os tópicos clássicos cobertos são "estatística do século XX". Conexões com práticas atuais:

**1. Causal inference revolution**

Pearl, Rubin, Hernán: identificação de efeito causal via DAGs e do-calculus. **Angrist e Imbens — Nobel 2021** ([nobelprize.org](https://www.nobelprize.org/prizes/economic-sciences/2021/summary/)).

**2. Computational statistics**

Bayes operacional via MCMC (Stan, NUTS), variational inference, normalizing flows. Hamilton dynamics integra geometria diferencial e estatística.

**3. Machine learning bayesiano**

Gaussian processes, BNN, Bayesian deep learning. Predições com incerteza calibrada.

**4. Modelos de cauda pesada**

Finanças real exige distribuições estáveis (Lévy), copulas, EVT (extreme value theory). TCL clássico inadequado.

**5. Concentração não-assintótica**

Sub-gaussianidade, Hoeffding, McDiarmid, Talagrand. Bounds com $n$ finito — fundamento de generalização em ML.

**6. Robustez**

Estimadores M, métodos não-paramétricos (kernel density, kNN), distribuições contaminadas. ML adversarial.

**7. Replication crisis**

P-hacking, garden of forking paths, overfitting. Bayes Factor, pre-registration, registered reports. Cultura científica em transição.

### Recursos de estudo avançado

- **Murphy (2022)** "Probabilistic Machine Learning" — referência moderna.
- **Hastie-Tibshirani-Friedman** "Elements of Statistical Learning" — clássico.
- **Gelman et al.** "Bayesian Data Analysis" — bíblia bayesiana.
- **MIT OCW 18.650** — statistics for applications.
- **Stat 110 (Harvard)** — Joe Blitzstein on YouTube.

### Decisões em ambiente real

Estatística não é fim — é meio para **decisão sob incerteza**. Em produção:
- **Custo de falso positivo vs. negativo** define threshold.
- **Latência** de inferência importa.
- **Drift de dados**: modelos envelhecem.
- **Fairness**: vieses sistêmicos.
- **Explicabilidade**: stakeholders precisam entender.

Cultura DS moderna: probabilidade + ML + ética + engenharia.

### Pipeline mínimo (1 dia)

```python
# 1. Carregar
df = pd.read_csv('dataset.csv')

# 2. Descritiva
df.describe()
df.hist(bins=30, figsize=(12, 8))

# 3. Correlação
sns.heatmap(df.corr(), annot=True)

# 4. Outliers
for col in df.select_dtypes('number').columns:
    df.boxplot(col)

# 5. Modelo
from sklearn.linear_model import LinearRegression
X, y = df[['feature1', 'feature2']], df['target']
model = LinearRegression().fit(X, y)
print(model.score(X, y))  # R²

# 6. Resíduos
residuos = y - model.predict(X)
plt.scatter(model.predict(X), residuos)
plt.axhline(0)
```
