<Porta nivel="formal" titulo="Três classes fundamentais">

### Simétrica

$A^T = A$ (real) ou $A^* = A$ (Hermitiana, complexa).

**Teorema espectral**: $A$ simétrica real $\Rightarrow$ existe $Q$ ortogonal com $A = QDQ^T$. Autovalores reais. Autovetores de autovalores distintos são **ortogonais**.

### Ortogonal

$Q \in \mathcal{M}_n(\mathbb{R})$ com $Q^T Q = QQ^T = I$. Equivalentemente, $Q^{-1} = Q^T$.

Propriedades:
- Preserva norma: $\|Qv\| = \|v\|$.
- Preserva produto interno: $\langle Qu, Qv \rangle = \langle u, v \rangle$.
- $\det Q = \pm 1$. $\det Q = 1$: rotação. $\det Q = -1$: rotação + reflexão.
- Autovalores em $\mathbb{C}$ têm módulo 1.

Análogo complexo: **unitária** $U^*U = I$.

### Definida positiva (SPD)

$A$ simétrica com $x^T A x > 0$ para todo $x \neq 0$.

Equivalências:
- Todos autovalores $\lambda_i > 0$.
- Existe $L$ inversível com $A = LL^T$ (Cholesky).
- Determinantes principais (menor à esquerda) todos $> 0$ (Sylvester).
- Define produto interno $\langle x, y \rangle_A = x^T A y$.

**Semidefinida positiva** (PSD): $x^T A x \geq 0$. Autovalores $\geq 0$. Caso de matrizes de covariância e Gram.

### Decomposições derivadas

- **Cholesky**: $A = LL^T$ ($A$ SPD), $L$ triangular inferior.
- **Polar**: $A = QP$ ($Q$ ortogonal, $P$ SPD). Generaliza $z = re^{i\theta}$.
- **Eckart-Young**: melhor aproximação de baixo posto via SVD truncada.

### Forma quadrática

$Q(x) = x^T A x$, $A$ simétrica. Diagonalizando $A = Q D Q^T$ e mudando coordenadas $y = Q^T x$:

$$Q(x) = \sum \lambda_i y_i^2$$

— soma de termos quadráticos puros (forma canônica). Sinais dos $\lambda_i$ classificam: definida (todos $> 0$ ou $< 0$), indefinida (mistura).

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Tem **três tipos de matrizes mágicas**:

1. **Simétrica**: igual à própria sombra (mesmo na diagonal). Toda matriz simétrica é "**arrumadinha**".
2. **Ortogonal**: gira sem mudar tamanho. Tipo girar uma boneca sem esticá-la.
3. **Positiva definida**: faz tudo **crescer** quando você coloca pra dentro.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

Em álgebra linear, tem **3 tipos de matriz** que valem ouro:

**Simétrica** ($A = A^T$): a matriz é igual ao seu reflexo na diagonal. Exemplo: $\begin{pmatrix} 2 & 3 \\ 3 & 5 \end{pmatrix}$. Toda **matriz de distâncias** ou **covariância** é simétrica.

**Ortogonal** ($Q^TQ = I$): suas linhas são vetores unitários e perpendiculares. **Não distorce nada** — só gira ou reflete. Por isso é a matriz favorita pra computação gráfica.

**Positiva definida** ($x^T A x > 0$): leva todo vetor não-zero pra um **número positivo** quando aplicada via produto interno. Aparece em **energias** (sempre positiva), **covariâncias** (variância > 0), **distâncias quadradas**.

A grande regra (**teorema espectral**): toda matriz simétrica vira diagonal trocando pra base de autovetores ortogonais. **Tudo em estatística, mecânica e ML usa esse fato**.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Estas três classes não são "casos especiais" — são **a maioria** das matrizes que aparecem em problemas reais.

### Por que simétrica é tão comum?

- **Covariância** $\Sigma_{ij} = \text{Cov}(X_i, X_j) = \text{Cov}(X_j, X_i) = \Sigma_{ji}$.
- **Hessiana** $H_{ij} = \partial^2 f / \partial x_i \partial x_j$ — Schwarz $\Rightarrow$ simétrica.
- **Matriz de distâncias** $D_{ij} = d(x_i, x_j)$.
- **Matriz de adjacência** de grafo não-dirigido.
- **Hamiltoniano** quântico.

### Por que ortogonal é tão comum?

- **Rotação 3D**: matrizes de rotação são ortogonais com $\det = 1$ (grupo $SO(3)$).
- **Mudança de base ortonormal**: $Q$.
- **DFT/DCT**: matriz de Fourier discreta, normalizada.
- **Reflexões de Householder**: usadas em QR.

### Por que SPD é tão comum?

- **Variância** sempre positiva ($\geq 0$).
- **Energia** em sistemas físicos (cinética + potencial).
- **Métrica** Riemanniana (distâncias).
- **Hessiana em mínimo local** de função suave.
- **Matriz de Gram** $G_{ij} = \langle v_i, v_j \rangle$ — sempre PSD, SPD se $v_i$ LI.

### Teorema espectral em ação

$A = QDQ^T$ é equivalente a:
$$A = \sum_i \lambda_i q_i q_i^T$$

— soma de **projeções rank-1**. Truncando (manter só os $k$ maiores $\lambda_i$): aproximação ótima de posto $k$.

Esta ideia é base de:
- **PCA** (Aula 118): truncar covariância.
- **Compressão de imagem**: SVD = espectral aplicado a $A^T A$.
- **Recomendação Netflix**: completar matriz aproximando por baixo posto.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Estas três classes são **a moeda corrente** de toda engenharia computacional. Reconhecê-las muda completamente a estratégia computacional.

### Vantagens computacionais

| Classe | Custo decomposição | Algoritmo |
|---|---|---|
| Geral | $O(n^3)$ | LU, QR |
| Simétrica | $O(n^3 / 3)$ | Cholesky se SPD; senão LDL$^T$ |
| Tridiagonal simétrica | $O(n)$ | Recorrência |
| SPD | Cholesky $O(n^3/6)$, mais estável | Gaussian elimination s/ pivot |
| Ortogonal | Inversão = transposição | Householder, Givens |

### Aplicações ortogonais

- **Decomposição QR**: $A = QR$ via Gram-Schmidt ou Householder. Base de mínimos quadrados estáveis.
- **DFT/FFT**: matriz unitária $F$ com $F_{jk} = \omega^{jk}/\sqrt{N}$. FFT calcula $Fv$ em $O(N \log N)$.
- **Pose estimation**: rotação 3D em $SO(3)$ — matriz ortogonal.
- **Quaternions**: representação compacta de rotações.

### SPD em otimização

Newton: $x_{k+1} = x_k - H^{-1} \nabla f$. Se $H$ é SPD, direção é descendente. Senão, modifica-se $H$ (BFGS, Levenberg-Marquardt).

**Função quadrática** $f(x) = \frac{1}{2} x^T A x - b^T x$ com $A$ SPD: minimizador único $x^* = A^{-1} b$. Resolva via Cholesky.

### Forma quadrática e ML

Regularização Ridge: $\min \|y - X\beta\|^2 + \lambda \|\beta\|^2$. Solução: $\beta^* = (X^TX + \lambda I)^{-1} X^T y$. $X^TX + \lambda I$ é SPD (mesmo se $X^T X$ for só PSD).

Toda função de perda quadrática + regularização L2 vira problema SPD.

### Decomposição polar em CG

$A = QP$ ($Q$ ortogonal, $P$ SPD). Em **rigid body simulation**, separa rotação ($Q$) de deformação ($P$). Em rendering, usado para suavizar transformações.

### Em finanças

Covariância $\Sigma$ de retornos é PSD. SPD se ativos são linearmente independentes. **Markowitz**: $\min w^T \Sigma w$ — problema quadrático SPD-constrained.

### Mecânica quântica

Hamiltoniano $\hat H$ é Hermitiano → autovalores reais (energias) e autovetores ortogonais (estados próprios). Toda mecânica quântica vive sobre este teorema.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

### Cone de PSD

$\mathbb{S}^n_+ = \{A \succeq 0\}$ é cone convexo fechado. Otimização sobre este cone = **SDP (Semidefinite Programming)** — generalização de LP.

Aplicações modernas: relaxações convexas (Goemans-Williamson para Max-Cut), controle robusto, sum-of-squares hierarchies.

### Operadores positivos em Hilbert

$T \in B(H)$ é **positivo** se $\langle Tv, v\rangle \geq 0$. Generaliza SPD. Toda raiz quadrada $T^{1/2}$ existe (cálculo funcional).

Em **mecânica quântica**, **estado misto** = operador densidade $\rho$: positivo, traço 1. Subconjunto convexo do espaço dos operadores positivos.

### Decomposições estruturais

- **Spectral theorem**: $A = QDQ^T$ ($A$ simétrica) — generaliza para normais ($AA^* = A^*A$).
- **Schur**: $A = Q T Q^*$ ($Q$ unitária, $T$ triangular). **Sempre existe**, base do QR algorithm.
- **Polar**: $A = QP$, $A = P'Q$. $P, P'$ tem autovalores = singulares de $A$.
- **Cholesky modificada**: para PSD com posto deficiente.

### Pseudoespectro & matrizes não-normais

Matriz simétrica é **normal** ($AA^T = A^TA$) e portanto bem-comportada. Não-normais têm pseudoespectro diferente do espectro — fenômenos transientes (Reddy-Trefethen), sensibilidade a perturbações, dinâmica não-modal.

### Aplicações state-of-the-art

- **Random matrix theory**: teorema de Wigner para simétricas → distribuição semicircular.
- **Spectral graph theory**: laplaciano $L = D - W$ é PSD. Autovalores codificam estrutura do grafo.
- **Kernel methods**: matriz de Gram $K_{ij} = k(x_i, x_j)$ é PSD se $k$ é kernel positivo (Mercer).
- **Diffusion maps, t-SNE, UMAP**: embeddings via espectro de matrizes simétricas/PSD.
- **Matrix completion** (Netflix): low-rank PSD via SDP.

### Conexão com BS (Aula 119)

A **matriz de covariância** de retornos é PSD. Em precificação multi-asset, o argumento de **martingale-representation** depende de existir uma fatoração $\Sigma = LL^T$ (Cholesky) — base de **simulação Monte Carlo** correlacionada.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

### Como reconhecer

- **Simétrica**: $A^T = A$. Olhe — diagonal e elementos espelhados.
- **Ortogonal**: colunas (e linhas) ortonormais. Verifique $Q^TQ = I$.
- **SPD**: simétrica + autovalores positivos. **Atalhos**: determinantes principais positivos (Sylvester); Cholesky existe.

### Em código

```python
import numpy as np

# Verificar simétrica
np.allclose(A, A.T)

# Verificar ortogonal
np.allclose(Q.T @ Q, np.eye(n))

# Verificar SPD via Cholesky
try:
    L = np.linalg.cholesky(A)
    # SPD!
except np.linalg.LinAlgError:
    # Não é SPD
    pass

# Decomposição espectral (simétrica)
vals, Q = np.linalg.eigh(A)  # mais rápido e estável
A_reconstruida = Q @ np.diag(vals) @ Q.T

# Decomposição polar
from scipy.linalg import polar
Q_pol, P_pol = polar(A)
```

### Workflow comum

1. **Construa** matriz simétrica/SPD (covariância, Gram, etc.).
2. **Decomponha** via `np.linalg.eigh` (ou Cholesky).
3. **Trabalhe na base ortogonal** — projeções, mudanças de coordenadas.
4. **Reconstrua** se necessário.

### Quando preocupar?

- **Quase singular**: SPD com autovalor mínimo $\to 0$ — instável. Adicione regularização ($A + \lambda I$).
- **Numericamente assimétrica**: arredondamentos. Use $(A + A^T)/2$ pra simetrizar.
- **Não-PSD inesperado**: matriz de covariância amostral pode ter autovalores negativos minúsculos. Projete em PSD via $\max(\lambda, 0)$.
