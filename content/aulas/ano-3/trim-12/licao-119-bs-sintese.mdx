<Porta nivel="formal" titulo="Reconhecendo todo o programa">

Black-Scholes é o ponto onde **convergem 120 lições de matemática**. Vamos identificar cada conceito que você aprendeu nos últimos 3 anos:

### 1. Funções (Trim 1, Lições 1-10)

$C(S, t)$ é uma **função de duas variáveis**. Domínio: $S > 0$, $t \in [0, T]$. Imagem: $C \geq 0$.

### 2. Exponencial e logaritmo (Trim 1, Lições 6-7)

$e^{-r(T-t)}$ é desconto contínuo (Aula 8). Aparece em $d_1$:
$$d_1 = \frac{\ln(S/K) + (r + \sigma^2/2)(T - t)}{\sigma\sqrt{T - t}}$$

O $\ln(S/K)$ é o log-retorno necessário pra opção entrar in-the-money.

### 3. Trigonometria (Trim 2)

Não aparece diretamente, mas está embutida via $\sqrt 2$, $\pi$ na constante $1/\sqrt{2\pi}$ da normal.

### 4. Sequências e limite (Trim 2-5)

$e = \lim (1 + 1/n)^n$ define o desconto contínuo. Convergência rigorosa via Aulas 19, 41-50.

### 5. Geometria analítica (Trim 3)

A curva de payoff $\max(S_T - K, 0)$ é uma reta quebrada — tipo a função afim por partes da Aula 1.

### 6. Probabilidade (Trim 4 + Trim 8)

$N(d)$ é a função de distribuição acumulada da **normal padrão**:
$$N(d) = \frac{1}{\sqrt{2\pi}}\int_{-\infty}^d e^{-u^2/2}\, du$$

$N(d_2)$ é a probabilidade neutra a risco da opção terminar exercida.

### 7. Limites (Trim 5)

A solução de BS via lema de Itô requer **$\Delta t \to 0$** — limite formal $\eps$-$\delta$ da Aula 41.

### 8. Derivadas (Trim 6-7)

Os **Greeks** — Δ, Γ, ν (vega), Θ — são derivadas parciais:
$$\Delta = \frac{\partial C}{\partial S}, \quad \Gamma = \frac{\partial^2 C}{\partial S^2}, \quad \nu = \frac{\partial C}{\partial \sigma}, \quad \Theta = \frac{\partial C}{\partial t}$$

Toda mesa de derivativos calcula esses números **continuamente**.

### 9. Estatística (Trim 8)

A normal aparece via Teorema Central do Limite (Aula 77): retornos como soma de muitos pequenos incrementos → log-normal.

### 10. Integrais (Trim 9)

$N(d)$ é integral. Calculada numericamente via Simpson, Gauss-Hermite, ou aproximação racional (Abramowitz & Stegun).

### 11. EDPs (Trim 10)

A fórmula é solução fechada de uma **EDP** (não-EDO):
$$\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + rS\frac{\partial V}{\partial S} - rV = 0$$

EDP **parabólica** — análoga à equação do calor.

### 12. Inferência estatística (Trim 11)

**Vol implícita** $\sigma_\text{impl}$ é estimada invertendo BS — problema de inferência (Aulas 101-110).

### 13. Álgebra Linear (Trim 12)

Em portfólios multi-asset (Markowitz), risco = $\mathbf{w}^T \Sigma \mathbf{w}$ — forma quadrática com matriz de covariância $\Sigma$. **PCA** (Aula 118) reduz dimensionalidade para gestão de risco.

<Porta nivel="25" titulo="Pra estudante de engenharia">

Black-Scholes é o **arquétipo** da matemática aplicada moderna: um problema do mundo real (precificar opção) reduzido a uma EDP, resolvido em forma fechada, validado empiricamente, premiado com o [**Nobel de Economia em 1997**](https://www.nobelprize.org/prizes/economic-sciences/1997/summary/).

### Origem (Black, Scholes 1973; Merton 1973)

Suposições centrais:
1. Preço da ação segue **GBM**: $dS = \mu S \, dt + \sigma S \, dW$.
2. Mercado **sem atrito** (frictionless): pode-se comprar/vender qualquer quantidade, sem custos.
3. **Hedge contínuo**: portfólio replicante com $\Delta = \partial C / \partial S$ ações + 1 call vendida = livre de risco.
4. Não-arbitragem: portfólio livre de risco rende $r$.

A combinação dessas hipóteses gera a EDP de BS.

### A EDP

$$\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + rS\frac{\partial V}{\partial S} - rV = 0$$

Mudança de variáveis $x = \ln S$, $\tau = T - t$, $u = e^{r\tau} V$ converte BS na **equação do calor** $u_\tau = \frac{1}{2}\sigma^2 u_{xx}$.

### Solução fechada (call europeia)

Com payoff $V(S, T) = \max(S - K, 0)$:

$$C(S, t) = S\, N(d_1) - K\, e^{-r(T-t)} N(d_2)$$

$$d_1 = \frac{\ln(S/K) + (r + \sigma^2/2)(T-t)}{\sigma\sqrt{T-t}}, \quad d_2 = d_1 - \sigma\sqrt{T-t}$$

### Greeks (sensibilidades)

| Greek | Fórmula | Interpretação |
|---|---|---|
| $\Delta = \partial C / \partial S$ | $N(d_1)$ | Hedge ratio — # ações por call |
| $\Gamma = \partial^2 C / \partial S^2$ | $\phi(d_1) / (S\sigma\sqrt T)$ | Convexidade |
| $\nu = \partial C / \partial \sigma$ | $S\phi(d_1)\sqrt T$ | Sensibilidade a vol |
| $\Theta = \partial C / \partial t$ | (negativo) | Time decay |
| $\rho = \partial C / \partial r$ | $K T e^{-rT} N(d_2)$ | Taxa de juros |

### Paridade put-call

$$C - P = S - K e^{-r(T-t)}$$

— relação **livre de modelo**, válida em qualquer mercado sem arbitragem. Dá $P$ a partir de $C$ sem refazer BS.

### Implementação numérica

```python
from scipy.stats import norm
import numpy as np

def bs_call(S, K, r, sigma, T):
    d1 = (np.log(S/K) + (r + 0.5*sigma**2)*T) / (sigma*np.sqrt(T))
    d2 = d1 - sigma*np.sqrt(T)
    return S*norm.cdf(d1) - K*np.exp(-r*T)*norm.cdf(d2)

# PETR4 a R$46.60, strike 47, 30 dias, vol 35%, r=14.65%
print(bs_call(46.60, 47, 0.1465, 0.35, 30/365))
```

### Vol implícita

Inverte BS dado preço observado: $\sigma_\text{impl}$ tal que $\text{BS}(\sigma_\text{impl}) = C_\text{observado}$. Resolve via Newton-Raphson com seed Brenner-Subrahmanyam $\sigma_0 \approx \sqrt{2\pi/T} \cdot C_\text{ATM}/S$.

### Limitações

| Hipótese BS | Realidade | Mitigação |
|---|---|---|
| Log-normalidade | Cauda gorda, kurtosis alta | Heston, saltos (Merton, Kou) |
| $\sigma$ constante | Smile + termo + skew | Stochastic vol (SABR), Local Vol |
| Mercado completo | Custos, liquidez, capital | FVA, KVA, MVA |
| Hedge contínuo | Discreto | Análise de erro de réplica |

Black-Scholes não é uma fórmula — é um **paradigma**. O argumento de **replicação dinâmica** (long Δ ações + short 1 call → portfolio livre de risco → rende r → EDP) é a base de toda finança quantitativa moderna.

### Limitações conhecidas

| Hipótese BS | Realidade | Mitigação |
|---|---|---|
| Log-normalidade | Cauda gorda | Smile + saltos (Merton, Kou) |
| $\sigma$ constante | Stochastic vol | Heston, SABR, rough Bergomi |
| Mercado completo | Fricção, capital | FVA, KVA, MVA |
| Hedge contínuo | Discreto, custo | Análise de erro de réplica |

### Vol implícita: o que de fato se opera

Mesa não opera "preço do call". Opera **vol implícita** — superfície 3D (smile + termo + skew). $\sigma_\text{impl}(C, S, K, r, T) = \text{BS}^{-1}(C)$, resolvido numericamente (Newton-Raphson iniciado com fórmula de Brenner-Subrahmanyam).

### Modelos modernos

- **Heston (1993)**: vol estocástica, smile capturável.
- **Local Vol (Dupire 1994)**: $\sigma(S, t)$ determinística, calibra exata à smile.
- **SABR (Hagan 2002)**: padrão em FX e rates.
- **Rough Bergomi (Bayer-Friz-Gatheral 2016)**: vol da vol fractional, estado-da-arte research.

Todos **reduzem a BS** quando $\sigma$ constante — BS é o **modelo nulo** sobre o qual se constrói.

### Computacionalmente

Em produção:
- **Greeks via diferenciação automática** (autograd): mais rápido e preciso que finite differences.
- **Monte Carlo** para opções path-dependent (Asian, Lookback, Barrier).
- **Métodos de PDE** (Crank-Nicolson) para opções americanas e early exercise.

### Conexão multi-asset com Trim 12

Em portfólio com $n$ ativos, vol → matriz de covariância $\Sigma$. Decompondo $\Sigma = L L^T$ (Cholesky, Aula 116) gera **Brownianos correlacionados** para Monte Carlo. **PCA** (Aula 118) identifica fatores principais — top 3-5 PCs capturam ~99% da variância em renda fixa, base de **risk attribution** institucional.

### Itô e o lema chave

$$dC = \left(\frac{\partial C}{\partial t} + \mu S \frac{\partial C}{\partial S} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 C}{\partial S^2}\right) dt + \sigma S \frac{\partial C}{\partial S} dW$$

A presença do termo $\frac{1}{2}\sigma^2 S^2 \partial^2 C / \partial S^2$ — **convexidade × variância** — é o que torna calculus estocástico distinto de calculus clássico.

### Estado da arte (2024-2026)

- **Neural SDEs**: redes que parametrizam $\sigma(S, t, \theta)$ aprendido de dados.
- **Signature methods** (Lyons): rough paths.
- **Operadores de Koopman** estocásticos para análise espectral de processos.
- **Portfólios robustos via SDP** (Goldfarb, Iyengar): otimização sobre incerteza Knightiana.

### Como precificar uma call europeia (recipe)

1. **Reúna inputs**: $S, K, r, \sigma, T$ (em anos).
2. **Calcule** $d_1, d_2$.
3. **Calcule $N(d_1), N(d_2)$** (via tabela ou função `norm.cdf`).
4. **Aplique fórmula**: $C = S N(d_1) - K e^{-rT} N(d_2)$.
5. **Para o put**: use paridade $P = C - S + K e^{-rT}$.

### Workflow Python

```python
import numpy as np
from scipy.stats import norm

def black_scholes(S, K, r, sigma, T, option='call'):
    d1 = (np.log(S/K) + (r + 0.5*sigma**2)*T) / (sigma*np.sqrt(T))
    d2 = d1 - sigma*np.sqrt(T)
    if option == 'call':
        return S*norm.cdf(d1) - K*np.exp(-r*T)*norm.cdf(d2)
    else:
        return K*np.exp(-r*T)*norm.cdf(-d2) - S*norm.cdf(-d1)

def greeks(S, K, r, sigma, T):
    d1 = (np.log(S/K) + (r + 0.5*sigma**2)*T) / (sigma*np.sqrt(T))
    d2 = d1 - sigma*np.sqrt(T)
    return {
        'delta': norm.cdf(d1),
        'gamma': norm.pdf(d1) / (S*sigma*np.sqrt(T)),
        'vega': S*norm.pdf(d1)*np.sqrt(T),
        'theta': -(S*norm.pdf(d1)*sigma)/(2*np.sqrt(T)) - r*K*np.exp(-r*T)*norm.cdf(d2),
        'rho': K*T*np.exp(-r*T)*norm.cdf(d2)
    }
```

### Cenário B3 real

PETR4 em 30/abr/2026 a R\$ 46,60. Call strike 47, vencimento 30 dias, vol implícita 35%, taxa Selic 14,65%.

```
S = 46.60, K = 47, r = 0.1465, sigma = 0.35, T = 30/365
=> d1 ≈ 0.085, d2 ≈ -0.015
=> N(d1) ≈ 0.534, N(d2) ≈ 0.494
=> C ≈ 46.60 * 0.534 - 47 * exp(-0.1465*30/365) * 0.494
=> C ≈ R$ 1.94
```

Compare com preço de mercado. Diferença? Vol implícita real $\neq$ 35%.

### Quando preocupar?

- **Eventos**: BS assume vol constante. Antes de earnings, vol implícita explode.
- **Liquidez**: ATM tem boa liquidez; OTM extremo é menos confiável.
- **Dividendos**: ajustar $S \to S e^{-qT}$ com $q$ taxa de dividendo.
- **Opção americana**: exercício antecipado — BS não dá solução fechada. Use binomial (Cox-Ross-Rubinstein) ou Crank-Nicolson.
