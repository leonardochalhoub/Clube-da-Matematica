<Porta nivel="formal" titulo="Definição e cálculo">

### Definição

Seja $A: V \to V$ operador linear. **Autovetor** é $v \neq 0$ com $Av = \lambda v$ para algum escalar $\lambda$ (**autovalor**).

### Equação característica

$$Av = \lambda v \iff (A - \lambda I)v = 0 \iff \lambda \text{ raiz de } p_A(\lambda) = \det(A - \lambda I)$$

$p_A$ é o **polinômio característico**, grau $n$ (em dim $n$).

### Espectro

$\sigma(A) = \{\lambda_1, \ldots, \lambda_k\}$ — conjunto de autovalores. Sobre $\mathbb{C}$, $|\sigma(A)| \leq n$ (com multiplicidade, $= n$).

### Autoespaço

$E_\lambda = \ker(A - \lambda I)$ — subespaço de $V$. Dimensão = **multiplicidade geométrica** $m_g(\lambda)$.

**Multiplicidade algébrica** $m_a(\lambda)$ = ordem da raiz $\lambda$ em $p_A$.

Sempre $1 \leq m_g(\lambda) \leq m_a(\lambda)$.

### Propriedades fundamentais

- $\det A = \prod \lambda_i$.
- $\text{tr}(A) = \sum \lambda_i$.
- Autovetores de autovalores **distintos** são LI.
- $A$ similar a $B$ $\Rightarrow$ mesmo $p_A$, mesmo espectro.
- $\sigma(A^k) = \{\lambda^k : \lambda \in \sigma(A)\}$.
- $A$ invertível $\iff 0 \notin \sigma(A)$.

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Imagina um **brinquedo elástico** que estica em algumas direções e em outras não. Tem direções **especiais** onde o brinquedo só estica reto, sem dobrar. Essas direções são as **direções dos autovetores**. O quanto estica é o **autovalor**.

Se autovalor é 2, estica pra 2 vezes o tamanho. Se é negativo, vira pro outro lado. Se é zero, encolhe pra zero.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

Toda transformação linear (matriz) tem **direções favoritas**. Se você puxa um vetor nessa direção, ele **só fica maior ou menor**, mas continua na mesma direção. Não gira.

Essas direções são **autovetores**. O fator que multiplica é o **autovalor**.

Exemplo: a matriz que escala $x$ por 3 e $y$ por 2 tem dois autovetores óbvios — o eixo $x$ (com autovalor 3) e o eixo $y$ (com autovalor 2).

Já uma rotação (que gira tudo) **não tem autovetores reais** — toda direção muda. Mas tem **autovetores complexos**, que é como a matemática lida com a rotação.

Pra achar autovalores: monta o polinômio $\det(A - \lambda I) = 0$ e resolve.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Autovalores são **a propriedade mais reveladora** de uma matriz. Eles dizem o que a matriz "faz" no longo prazo — bem mais que os números individuais.

### Receita de cálculo

1. Monte $A - \lambda I$.
2. Calcule $\det(A - \lambda I)$ — polinômio em $\lambda$.
3. Iguale a zero. Raízes = autovalores.
4. Para cada $\lambda$, resolva $(A - \lambda I) v = 0$ — soluções não-nulas = autovetores.

### Exemplo

$A = \begin{pmatrix} 4 & 1 \\ 2 & 3 \end{pmatrix}$.

$\det(A - \lambda I) = (4 - \lambda)(3 - \lambda) - 2 = \lambda^2 - 7\lambda + 10 = (\lambda - 5)(\lambda - 2)$.

Autovalores: $\lambda_1 = 5$, $\lambda_2 = 2$.

Para $\lambda = 5$: $(A - 5I) v = \begin{pmatrix} -1 & 1 \\ 2 & -2 \end{pmatrix} v = 0 \Rightarrow v = (1, 1)$.

Para $\lambda = 2$: análogo, $v = (1, -2)$.

### Por que importa?

- $A^k$ no autovetor $v$: $A^k v = \lambda^k v$. Crescimento exponencial pelo autovalor.
- $A$ aplicada repetidamente: domínio do autovalor de **maior módulo** (Perron-Frobenius).
- **Estabilidade** de sistemas $\dot x = Ax$: depende de $\text{Re}(\lambda) < 0$ pra todos.

### Conexão com Trim 12

Autovalores ligam todas as próximas aulas: diagonalização (115), teorema espectral (116), SVD (117 — autovalores de $A^T A$), PCA (118 — autovalores de covariância).

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Autovalores são o **DNA computacional** de uma matriz. Em engenharia, todo sistema linear é caracterizado pelos seus.

### Aplicações canônicas

| Domínio | Aparição |
|---|---|
| **Mecânica** | Modos próprios de vibração — autovalores de matriz de rigidez/massa |
| **Controle** | Polos do sistema — autovalores de $A$ em $\dot x = Ax$. Estabilidade $\iff \text{Re}(\lambda) < 0$ |
| **Quântica** | Energias = autovalores do hamiltoniano $\hat H$ |
| **Estatística** | Componentes principais (PCA) = autovetores da covariância |
| **Grafos** | Espectro do laplaciano → conectividade, particionamento |
| **PageRank** | Distribuição estacionária = autovetor de $\lambda = 1$ |
| **Finanças** | Fatores de risco = autovetores principais da covariância |

### Cálculo numérico

Em produção, **NUNCA** se calcula via $\det(A - \lambda I) = 0$ (instável). Métodos:

- **Power iteration**: $v_{k+1} = A v_k / \|A v_k\|$ — converge pro autovetor dominante. Base do PageRank.
- **QR algorithm**: $A_k = Q_k R_k$, $A_{k+1} = R_k Q_k$. Converge pra forma de Schur. **Padrão indústria** (LAPACK).
- **Lanczos**: para matrizes grandes esparsas simétricas (Krylov).
- **Arnoldi**: caso geral.

```python
import numpy as np
A = np.array([[4, 1], [2, 3]])
vals, vecs = np.linalg.eig(A)  # geral
vals_h, vecs_h = np.linalg.eigh(A_sym)  # mais rápido se simétrica
```

### Interpretação geométrica

$A$ esticando ao longo de autovetores → forma de **elipsoide** se $A$ é positiva definida. Razão $\lambda_\max / \lambda_\min$ = **número de condicionamento** (cond. ruim → instabilidade numérica).

### Estabilidade de Lyapunov

$\dot x = Ax$ tem origem assintoticamente estável $\iff$ todos $\text{Re}(\lambda_i) < 0$. Toda análise de robustez de controle parte daqui.

### Spectral radius

$\rho(A) = \max |\lambda_i|$. Iteração $x_{k+1} = Ax_k$ converge pra zero $\iff \rho(A) < 1$. Usada em métodos iterativos (Jacobi, Gauss-Seidel).

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

### Análise espectral em dim infinita

Em Hilbert, $\sigma(T)$ pode ser:

- **Pontual** ($\sigma_p$): autovalores no sentido clássico.
- **Contínuo** ($\sigma_c$): $T - \lambda I$ injetiva mas não sobrejetiva, imagem densa.
- **Residual** ($\sigma_r$): imagem não densa.

Operadores compactos têm $\sigma$ enumerável com único ponto de acumulação em 0 (alternativa de Fredholm).

### Funções de matriz via espectro

Se $A = PDP^{-1}$ diagonalizável: $f(A) = P f(D) P^{-1}$ com $f(D) = \text{diag}(f(\lambda_i))$. Daí:

- $e^{At} = \sum \frac{t^k A^k}{k!}$ — solução de $\dot x = Ax$.
- $\log A$, $\sqrt{A}$, $\sin A$, etc.

Em geral, **cálculo funcional** via Cauchy: $f(A) = \frac{1}{2\pi i}\oint f(z)(zI - A)^{-1} dz$.

### Forma de Jordan

Se $A$ não é diagonalizável, existe $J$ (Jordan) tal que $A = PJP^{-1}$. $J$ é bloco-diagonal com blocos $J_k(\lambda) = \lambda I + N$ ($N$ nilpotente).

Mais usada teoricamente que numericamente — **forma de Schur** ($A = QTQ^*$, $T$ triangular) é mais estável computacionalmente.

### Conexão com BS (Aula 119)

Em finanças multi-asset, **fatores principais** são autovetores da matriz de covariância. **Risco sistemático** corresponde aos autovalores grandes; **risco idiossincrático** aos pequenos.

### Perron-Frobenius

Matriz $A \geq 0$ irredutível tem único autovalor positivo $\lambda^* = \rho(A)$ com autovetor positivo. **Base de toda teoria de cadeias de Markov, modelos input-output (Leontief), PageRank.**

### Aplicações sofisticadas

- **Spectral clustering**: autovetores do laplaciano $L = D - W$ revelam comunidades em grafos.
- **Quantum walks**: dinâmica via autovalores de operador unitário.
- **Random matrix theory**: distribuição de autovalores tem leis universais (Wigner, Tracy-Widom).
- **Koopman operator**: extensão linear de dinâmica não-linear via espectro infinito-dim. DMD (Dynamic Mode Decomposition) é versão data-driven.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

### Receita básica (a mão)

1. Calcule $\det(A - \lambda I) = 0$ — polinômio em $\lambda$.
2. Resolva (use Bhaskara ou fatoração).
3. Para cada autovalor $\lambda$, encontre $\ker(A - \lambda I)$.

### Em código

```python
import numpy as np

A = np.array([[4, 1], [2, 3]])
vals, vecs = np.linalg.eig(A)
print("Autovalores:", vals)
print("Autovetores (colunas):", vecs)

# Verificar
for i, l in enumerate(vals):
    v = vecs[:, i]
    print("Erro:", np.linalg.norm(A @ v - l * v))
```

### Atalhos úteis

- Triangular: autovalores na diagonal.
- $A^T$ tem mesmos autovalores que $A$.
- $A + cI$: autovalores deslocados em $c$.
- $A^k$: autovalores elevados a $k$.
- $A^{-1}$: autovalores invertidos.

### Quando preocupar?

- **Autovalores complexos**: matriz real pode ter autovalores complexos conjugados — útil em rotações.
- **Multiplicidade $> 1$**: cuidado com $m_g < m_a$ — não diagonalizável.
- **Autovalores próximos**: instabilidade numérica.
- **Autovalor zero**: matriz singular.

### Para o restante de Trim 12
