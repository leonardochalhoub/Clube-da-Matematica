<Porta nivel="formal" titulo="Definição e teorema">

### Teorema (existência da SVD)

Para toda $A \in \mathbb{R}^{m \times n}$, existem:

- $U \in \mathbb{R}^{m \times m}$ ortogonal,
- $V \in \mathbb{R}^{n \times n}$ ortogonal,
- $\Sigma \in \mathbb{R}^{m \times n}$ diagonal com $\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r > 0 = \sigma_{r+1} = \cdots$,

tais que $A = U \Sigma V^T$. O número $r$ é o **posto** de $A$.

### Forma compacta

$$A = U_r \Sigma_r V_r^T = \sum_{i=1}^r \sigma_i u_i v_i^T$$

— soma de $r$ matrizes de **posto 1**.

### Conexão com autovalores

- $A^T A = V \Sigma^T \Sigma V^T \Rightarrow $ autovalores de $A^T A$ são $\sigma_i^2$.
- $A A^T = U \Sigma \Sigma^T U^T \Rightarrow $ autovalores de $AA^T$ são $\sigma_i^2$.
- Vetores singulares à direita ($V$) = autovetores de $A^T A$.
- Vetores singulares à esquerda ($U$) = autovetores de $A A^T$.

### Os 4 subespaços via SVD

| Subespaço | Base |
|---|---|
| $\text{Col}(A)$ | Primeiras $r$ colunas de $U$ |
| $\ker(A^T)$ | Últimas $m - r$ colunas de $U$ |
| $\text{Row}(A)$ | Primeiras $r$ colunas de $V$ |
| $\ker(A)$ | Últimas $n - r$ colunas de $V$ |

### Eckart-Young

Melhor aproximação de posto $k$ de $A$ (em norma Frobenius ou espectral):

$$A_k = \sum_{i=1}^k \sigma_i u_i v_i^T$$

Erro: $\|A - A_k\|_F = \sqrt{\sigma_{k+1}^2 + \cdots + \sigma_r^2}$, $\|A - A_k\|_2 = \sigma_{k+1}$.

### Pseudoinversa

$$A^+ = V \Sigma^+ U^T$$

com $\Sigma^+$ obtida invertendo singulares positivos. Resolve mínimos quadrados ($\min \|Ax - b\|$) com $x = A^+ b$ — solução de mínima norma.

Se você junta **só as primeiras camadas mais importantes**, ainda dá pra reconhecer a imagem, mas com muito **menos memória**. Essa é a base da **compressão**.

<Porta nivel="10" titulo="Pra criança de 10 anos">

Toda matriz pode ser **fatorada** em três pedaços: $A = U \Sigma V^T$.

- $U$ e $V$ são **rotações** (matrizes ortogonais).
- $\Sigma$ é só uma **escala** — diagonal com números $\geq 0$ chamados **valores singulares**.

Esses valores singulares te dizem **a importância de cada direção**. Se um valor singular é grande, a direção contribui muito; se é pequeno, contribui pouco. Se é zero, **não contribui nada**.

**Truque mágico**: se você manter só os 10 maiores valores singulares e zerar o resto, você obtém uma **versão comprimida** da matriz que mantém quase toda a informação.

Foto $1000 \times 1000$ ≈ 1 milhão de pixels. Comprimida com 50 valores singulares: $50 \times 1000 + 50 + 50 \times 1000 = 100050$ números — **10x menos**. Imagem ainda reconhecível.

É assim que **JPEG, MP3, recomendações da Netflix** funcionam.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

A SVD é provavelmente **a decomposição mais útil da matemática aplicada**. Ela combina diagonalização + ortogonalidade + generalidade total.

### Geometria

$A$ leva esfera unitária em **elipsoide** com semi-eixos $\sigma_1, \sigma_2, \ldots, \sigma_r$, alinhados aos vetores $u_1, \ldots, u_r$.

- $V^T$ rotaciona o espaço de entrada.
- $\Sigma$ estica os eixos por $\sigma_i$.
- $U$ rotaciona o espaço de saída.

### Comparação com diagonalização

| Aspecto | Diagonalização | SVD |
|---|---|---|
| Aplica a | Quadradas diagonalizáveis | Qualquer matriz |
| Existência | Pode falhar | Sempre existe |
| Bases | $P, P^{-1}$ (não ortogonais em geral) | $U, V$ ortogonais |
| Valores | Autovalores ($\mathbb{C}$ em geral) | Singulares ($\geq 0$ reais) |

### Aplicações imediatas

- **Imagem comprimida**: foto → matriz → trunca SVD.
- **PCA** (Aula 118): SVD da matriz de dados centralizada.
- **Pseudoinversa**: $A^+ = V \Sigma^+ U^T$ resolve mínimos quadrados.
- **Latent semantic analysis (LSA)**: SVD da matriz termo-documento.
- **Recomendação**: SVD da matriz usuário-item.

### Caso famoso: Netflix Prize

Netflix tem matriz $\sim 10^7$ usuários $\times 10^5$ filmes — esparsa (só ratings dados). Aproximação por baixo posto via **matrix factorization** (variante de SVD) prevê ratings ausentes. Time vencedor (BellKor's Pragmatic Chaos, 2009) ganhou US\$ 1 milhão.

### Cálculo

Para $A$ pequena: encontre autovetores de $A^T A$ → $V$ e $\sigma_i^2$. Calcule $u_i = A v_i / \sigma_i$ → $U$.

Para grande: use `np.linalg.svd` (LAPACK).

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

SVD é a **canivete suíço** da álgebra linear computacional. Resolve sistemas mal-condicionados, encontra estrutura latente, comprime dados.

### Aplicações por domínio

**Computer vision**:
- **Eigenfaces**: SVD de matriz de rostos centralizados → bases ortonormais para reconhecimento.
- **Stitching de panoramas**: SVD pra resolver homografia.
- **Compressão de imagem**: trunca SVD do canal de luminância.

**ML**:
- **Word embeddings**: GloVe usa SVD da matriz de coocorrência.
- **Pseudoinversa estável**: resolve $\min \|Ax - b\|$ com regularização Tikhonov via thresholding de $\sigma_i$.
- **Whitening**: $W = V \Sigma^{-1}$ — pré-processamento que descorrelaciona features.

**Finanças**:
- **PCA de yield curve**: 3 fatores principais (level, slope, curvature) capturam ~99% da variância.
- **Risco**: análise de fatores principais via SVD da covariância.

**Numérico**:
- **Número de condicionamento**: $\kappa_2(A) = \sigma_1 / \sigma_r$ — medida de mal-condicionamento.
- **Posto numérico**: $r_\epsilon = $ # de $\sigma_i > \epsilon \cdot \sigma_1$.
- **Rank-revealing**: SVD é o gold standard.

### Algoritmos

- **SVD completa**: $O(\min(mn^2, m^2 n))$ via Golub-Reinsch.
- **SVD truncada** (top $k$): $O(mnk)$ via Lanczos/Arnoldi para $A^TA$.
- **Randomized SVD**: $O(mn k)$ esperado, com erro probabilístico controlado (Halko-Martinsson-Tropp).

```python
import numpy as np
A = np.random.randn(100, 50)
U, s, Vt = np.linalg.svd(A, full_matrices=False)

# Aproximação por baixo posto
k = 10
A_k = U[:, :k] @ np.diag(s[:k]) @ Vt[:k, :]

# Erro Frobenius
err = np.linalg.norm(A - A_k, 'fro')
print("Erro relativo:", err / np.linalg.norm(A, 'fro'))

# Pseudoinversa
A_pinv = Vt.T @ np.diag(1/s) @ U.T
```

### Decisão de truncamento

- **Visual**: scree plot dos $\sigma_i$ — corte onde "joelha".
- **Variância explicada**: cumulative $\sum \sigma_i^2 / \sum \sigma_j^2 \geq 0,95$.
- **Marchenko-Pastur threshold**: para ruído puro, $\sigma_i$ esperado tem distribuição conhecida — corte acima.

### Quando NÃO usar SVD

- Matriz **muito grande** e esparsa: prefira métodos iterativos (Lanczos, randomized SVD).
- Matriz **estruturada** (Toeplitz, Hankel): há algoritmos $O(n \log n)$ específicos.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

### Generalização para tensores

Tensores 3D+ não têm SVD canônica única. Variantes:
- **CP (CANDECOMP/PARAFAC)**: $T = \sum_r \lambda_r \, u_r \otimes v_r \otimes w_r$.
- **Tucker**: $T = G \times_1 U \times_2 V \times_3 W$.
- **Tensor Train**: $T = G_1 \times G_2 \times \cdots$.

Aplicações: deep learning compression, quantum chemistry, recomendação multi-aspecto.

### Randomized methods

Halko-Martinsson-Tropp (2011): para matriz grande, multiplique $A \cdot \Omega$ (matriz aleatória), faça QR para obter base aproximada de $\text{Col}(A)$, projete e faça SVD pequena. **Custo**: $O(mn k)$ vs $O(mn \min(m,n))$.

Para Netflix Prize, este é o método indispensável.

### Algoritmos modernos

- **Krylov methods** (Lanczos para $A^TA$): top $k$ singulares em $O(\text{nnz}(A) k)$ para esparsas.
- **Block Lanczos**: paralelização.
- **PIRO** (power iterations + randomization): estado-da-arte 2023.

### Conexão com BS (Aula 119)

Em **multi-asset Black-Scholes**, $\Sigma$ (covariância) tem SVD = espectral. Truncar $\Sigma$ via top $k$ singulares = **factor model**: tipo Fama-French (3-fator) ou APT.

### Operadores compactos

Em Hilbert separável, operadores compactos têm SVD: $T = \sum \sigma_n \langle \cdot, e_n\rangle f_n$ com $\sigma_n \to 0$. Generaliza diagonalização espectral. Base de **regularização de problemas mal-postos** (Tikhonov, Landweber).

### Conexão com determinante

$|\det A| = \prod \sigma_i$ (matriz quadrada). Útil em ML para likelihood de modelos com transformações invertíveis (normalizing flows).

### Aplicações topo-de-pilha

- **Robust PCA** (Candes-Li-Ma-Wright): $A = L + S$ com $L$ baixo posto, $S$ esparsa. SDP relaxation.
- **Tensor completion**: extensão multi-modo do Netflix.
- **Online SVD**: atualização incremental quando colunas chegam streaming.
- **Quantum singular value transformation (QSVT)**: framework unificador para algoritmos quânticos.
- **Implicit bias do gradient descent**: em redes lineares, GD converge à solução de mínima norma — SVD-truncated.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

### Como calcular SVD na prática

```python
import numpy as np

A = np.array([[3, 1, 1],
              [-1, 3, 1]])
U, s, Vt = np.linalg.svd(A, full_matrices=False)
print("U:\n", U)
print("singulares:", s)
print("V^T:\n", Vt)

# Verificar
print("Erro:", np.linalg.norm(A - U @ np.diag(s) @ Vt))
```

### Caso uso 1: compressão de imagem

```python
from PIL import Image
import numpy as np

img = np.array(Image.open("foto.png").convert("L"), dtype=float)
U, s, Vt = np.linalg.svd(img, full_matrices=False)

for k in [5, 20, 50, 100]:
    img_k = (U[:, :k] * s[:k]) @ Vt[:k, :]
    Image.fromarray(img_k.clip(0, 255).astype("uint8")).save(f"k_{k}.png")
```

### Caso uso 2: pseudoinversa para mínimos quadrados

```python
# y = X @ beta + ruído, X $m \times n$
beta = np.linalg.pinv(X) @ y  # mais estável que solve(X.T @ X, X.T @ y)
```

### Caso uso 3: sistema de recomendação simples

```python
# R: matriz usuário x item, NaN onde sem rating
# Imputar com média, fazer SVD, truncar
R_fill = np.nan_to_num(R, nan=R.mean())
U, s, Vt = np.linalg.svd(R_fill, full_matrices=False)
k = 10
R_pred = U[:, :k] @ np.diag(s[:k]) @ Vt[:k, :]
```

### Quando preocupar?

- **Singulares muito pequenos**: $\sigma_r \approx 0$ → matriz mal-condicionada. Use **threshold**: zerar $\sigma_i < \epsilon$.
- **Matrizes esparsas**: SVD densa é cara. Use `scipy.sparse.linalg.svds` (Lanczos).
- **Identifiability**: SVD é única a menos de sinais e permutações. Em PCA, esta é fonte clássica de bug ("sign flipping").

### Para a próxima aula (PCA)

PCA = SVD da matriz de dados centralizada $X - \bar X$. Os componentes principais são as colunas de $V$, e as variâncias explicadas são $\sigma_i^2 / (n-1)$.
