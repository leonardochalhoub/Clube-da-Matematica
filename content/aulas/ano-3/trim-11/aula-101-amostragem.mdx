<Porta nivel="formal" titulo="Amostragem e estimação">

### População e amostra

- **População** $\mathcal P$: conjunto de interesse, parâmetros $\theta$ (média $\mu$, variância $\sigma^2$, proporção $p$, ...) — **fixos mas desconhecidos**.
- **Amostra** $X_1, \ldots, X_n$: realizações observadas — **aleatórias**.

### Amostragem aleatória simples (SRS)

$X_1, \ldots, X_n$ iid com mesma distribuição da população.

### Estimadores

$\hat\theta = T(X_1, \ldots, X_n)$ — função da amostra.

| Parâmetro | Estimador clássico |
|---|---|
| $\mu$ | $\bar X = \frac{1}{n}\sum X_i$ |
| $\sigma^2$ | $S^2 = \frac{1}{n-1}\sum (X_i - \bar X)^2$ |
| $p$ | $\hat p = \frac{1}{n}\sum \mathbb{1}\{X_i \in A\}$ |

### Propriedades desejáveis

- **Não-viés**: $E[\hat\theta] = \theta$.
- **Consistência**: $\hat\theta \xrightarrow{P} \theta$ quando $n \to \infty$.
- **Eficiência**: variância mínima na classe dos não-viesados (Cramér-Rao).
- **Suficiência**: $\hat\theta$ contém toda informação relevante.

### Distribuição amostral de $\bar X$

- $E[\bar X] = \mu$ (não-viesado).
- $\text{Var}(\bar X) = \sigma^2/n$ (decai com $n$).
- $\bar X \to \mathcal{N}(\mu, \sigma^2/n)$ pelo TCL.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

A **estatística inferencial** te ensina a **chutar** propriedades da população usando **uma amostra** aleatória.

**Conceitos**:

- **População**: todos os elementos de interesse (ex: todos brasileiros).
- **Amostra**: subconjunto sorteado.
- **Parâmetro**: número fixo da população (média $\mu$, proporção $p$).
- **Estatística**: número calculado da amostra (média $\bar X$, proporção $\hat p$).

A estatística é **aleatória** (depende de quem caiu na amostra). Ela tem **distribuição amostral**: se você repetir o experimento, qual é o intervalo provável?

**Propriedades mágicas**:

1. **A média amostral $\bar X$ tende ao $\mu$** quando $n$ cresce (Lei dos Grandes Números).
2. **A distribuição de $\bar X$ é Gaussiana** se $n$ é grande, **independente da população original** (Teorema Central do Limite).

Isso significa: pesquisa eleitoral com 1000 pessoas pode estimar o quê 200 milhões pensam, com **margem de erro pequena**.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

### Por que amostrar?

Censo (medir tudo) é **caro, demorado, impossível**. Amostragem te dá estimativa precisa **com 1/1000 do esforço**, **se feita direito**.

### Exemplos

- IBGE: PNAD com $\sim$ 200 mil domicílios estima quase tudo do Brasil.
- Datafolha: 2000 entrevistas estimam intenção de voto com erro $\pm$ 2%.
- Controle de qualidade: 100 chips testados estimam taxa de defeito do lote de 100 mil.

### Tipos de amostragem

- **SRS** (simples): cada um tem mesma probabilidade. Padrão teórico.
- **Estratificada**: divide população em estratos, sorteia em cada. Ex: por gênero, idade.
- **Sistemática**: cada $k$-ésimo elemento.
- **Por conglomerados**: sorteia grupos inteiros (ex: bairros).

### Vieses comuns

- **Seleção**: pesquisa online só pega quem tem internet.
- **Não-resposta**: quem responde difere de quem não responde.
- **Auto-seleção**: voluntários ≠ população.
- **Survivorship**: análise só de empresas que sobreviveram.

### Distribuição amostral

A média amostral $\bar X$ é uma **variável aleatória** com:
- $E[\bar X] = \mu$
- $\text{Var}(\bar X) = \sigma^2/n$
- $\bar X \to \mathcal{N}(\mu, \sigma^2/n)$ se $n$ grande (TCL)

Erro padrão $\text{SE}(\bar X) = \sigma/\sqrt n$. **Reduz com $\sqrt n$**.

### Margem de erro

Para 95% de confiança, $\text{ME} = 1{,}96 \cdot \sigma/\sqrt n$. **Quadruplique $n$ pra dividir ME por 2**.

### Exemplo

Pesquisa eleitoral $n = 2000$, $\hat p = 0{,}45$. ME = $1{,}96\sqrt{0{,}45 \cdot 0{,}55/2000} \approx 0{,}022$. Resultado: 45% $\pm$ 2,2%.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Amostragem é a **base de toda inferência empírica** em engenharia: testes de qualidade, ensaios clínicos, A/B testing.

### Lei dos Grandes Números (LLN)

$\bar X_n \xrightarrow{P} \mu$ quando $n \to \infty$. Garantia de consistência.

**Forte (Kolmogorov)**: $\bar X_n \xrightarrow{a.s.} \mu$.
**Fraca (Khinchin)**: $\bar X_n \xrightarrow{P} \mu$.

### Teorema Central do Limite

Para $X_i$ iid com $E[X] = \mu$, $\text{Var}(X) = \sigma^2 < \infty$:

$$\frac{\bar X - \mu}{\sigma/\sqrt n} \xrightarrow{d} \mathcal{N}(0, 1)$$

**Aplicabilidade**: $n \geq 30$ tipicamente, mas depende de skewness. Distribuição muito assimétrica precisa $n$ maior.

### Estimadores e propriedades

| Propriedade | Definição |
|---|---|
| Não-viés | $E[\hat\theta] = \theta$ |
| Consistência | $\hat\theta \xrightarrow{P} \theta$ |
| Eficiência | $\text{Var}(\hat\theta) = $ Cramér-Rao |
| Robustez | Insensível a outliers |
| Suficiência | $f(\mathbf x | \theta) = g(T(\mathbf x), \theta) h(\mathbf x)$ |

### Cramér-Rao

$$\text{Var}(\hat\theta) \geq \frac{1}{nI(\theta)}, \quad I(\theta) = E\left[\left(\frac{\partial}{\partial\theta}\log f\right)^2\right]$$

Estimador atinge igualdade $\Leftrightarrow$ é UMVUE (unif. min. var. não-viesado).

### Métodos de estimação

- **Método dos momentos**: iguale momentos amostrais aos populacionais.
- **MLE** (máxima verossimilhança): $\hat\theta_{ML} = \arg\max L(\theta | \mathbf x)$.
- **Bayesian**: $p(\theta | \mathbf x) \propto p(\mathbf x | \theta) p(\theta)$.
- **Bootstrap**: re-amostre os dados pra estimar variabilidade.

### Tamanho de amostra

Para precisão $E$ com confiança $1-\alpha$:

$$n = \left(\frac{z_{\alpha/2} \sigma}{E}\right)^2$$

Para proporção, max em $p = 0{,}5$: $n = (z/E)^2 \cdot 0{,}25$. Para 95% e $E = 0{,}03$: $n \approx 1067$.

### Aplicações em engenharia

- **Controle de qualidade**: cartas $\bar X$, $R$, $S$. Six Sigma.
- **Medição**: GUM (Guide to Uncertainty in Measurement) usa SE.
- **A/B testing**: comparar conversão entre versões.
- **Ensaios clínicos**: tamanho de amostra para detectar efeito.

### Análise de variância (ANOVA)

Compara médias de múltiplos grupos. Particiona variância: entre + dentro.

### Estatística não-paramétrica

Sem assumir distribuição. Mediana, mood, Wilcoxon, Mann-Whitney. Robustos a outliers.

### Cross-domínio: Mecanismo de Higgs

Detecção do bóson de Higgs no LHC envolveu inferência estatística sofisticada para distinguir sinal de fundo (5σ critério). François Englert e Peter Higgs receberam o [Prêmio Nobel de Física de 2013](https://www.nobelprize.org/prizes/physics/2013/summary/) pela predição teórica.

### Visão moderna

Amostragem é caso particular de **inferência baseada em desígnio (design-based)** e contrasta com **modelo-baseada (model-based)**.

### Estimador Horvitz-Thompson

Para amostragem complexa com probabilidades de inclusão $\pi_i$:

$$\hat T_{HT} = \sum_{i \in S} \frac{Y_i}{\pi_i}$$

Não-viesado mesmo em desígnios complexos. Variância via Sen-Yates-Grundy.

### Pós-estratificação e raking

Ajustes pós-amostragem usando informação auxiliar para reduzir erro padrão e corrigir não-resposta.

### Dependência: amostragem em redes

Amostragem em populações ocultas (HIV, drogas), redes sociais. Respondent-Driven Sampling (RDS), snowball.

### Estatística suficiente

Família exponencial $f(x|\theta) = h(x) \exp(\eta(\theta) T(x) - A(\theta))$. $T(x)$ é estatística suficiente. Lehmann-Scheffé.

### Estimação robusta

- M-estimators: $\arg\min \sum \rho(x_i - \theta)$. Huber-loss balanceia $L^2$ e $L^1$.
- Trimmed mean, Winsorized mean.
- Quantile regression.

### Bootstrap e variantes

- **Não-paramétrico**: re-amostra com reposição.
- **Paramétrico**: re-amostra do modelo ajustado.
- **Block bootstrap**: pra séries temporais.
- **Bayesian bootstrap**: Dirichlet prior em pesos.

### Concentração de medida

Bound não-assintótico: Hoeffding, Chernoff, McDiarmid. Hoeffding:

$$P(|\bar X - \mu| > t) \leq 2 \exp\left(-\frac{2nt^2}{(b-a)^2}\right)$$

para $X_i \in [a, b]$.

### Estatística sequencial

Wald sequential probability ratio test. Em A/B testing moderno: peeking, always-valid p-values.

### Big data

Amostragem em streaming (reservoir sampling), distributed estimators (mean reduction com 1 round). Differential privacy.

### Bayesian non-parametric

Dirichlet processes, Pólya urn, Chinese restaurant process. Não fixa número de classes a priori.

### Inferência causal

Randomized controlled trials (RCT) — gold standard. Quasi-experimental: matching, IV, RDD, DiD. Trabalho de Joshua Angrist e Guido Imbens em Nobel de Economia 2021.

### Métodos compostos

- **Plug-in**: $\hat\theta_{plug} = T(\hat F_n)$ onde $\hat F_n$ é distribuição empírica.
- **U-statistics**: para estimar funcionais simétricos.
- **Influence functions**: análise de robustez.

### Workflow de projeto amostral

1. **Defina população alvo**: quem é elegível?
2. **Frame amostral**: lista/cadastro pra sortear (e.g., CPF, IBGE).
3. **Tamanho**: $n = (z\sigma/E)^2$ ou via simulação.
4. **Desígnio**: SRS, estratificada, conglomerados.
5. **Coleta**: protocolo, treinamento.
6. **Análise**: estimativa pontual, intervalo de confiança.
7. **Diagnósticos**: vieses, missing data.

### Checklist de qualidade

- [ ] Frame cobre população alvo? (under-coverage?)
- [ ] Não-resposta < 20%?
- [ ] Variância dos estratos estimada?
- [ ] Desenho efetivo > 1? Ajuste $n$.
- [ ] Calibração com dados auxiliares?

### Cálculos rápidos

```python
import numpy as np
from scipy import stats

def sample_size_mean(sigma, error, conf=0.95):
    z = stats.norm.ppf(1 - (1-conf)/2)
    return int(np.ceil((z*sigma/error)**2))

def sample_size_prop(p, error, conf=0.95):
    z = stats.norm.ppf(1 - (1-conf)/2)
    return int(np.ceil(z**2 * p*(1-p) / error**2))

print(sample_size_prop(0.5, 0.03))  # margem 3% → 1067
```

### Erros comuns

- **Confundir SE com SD**: SE = SD/$\sqrt n$.
- **Pesquisa por conveniência tratada como SRS**: viés enorme.
- **Não considerar deff**: design effect aumenta variância em conglomerado.
- **Aplicar TCL com $n = 5$**: TCL exige $n$ moderado.
- **Estimativa pontual sem intervalo**: incompleto.

### A/B testing

```python
from scipy.stats import norm
n_a, n_b = 1000, 1000
p_a, p_b = 0.10, 0.12
se = np.sqrt(p_a*(1-p_a)/n_a + p_b*(1-p_b)/n_b)
z = (p_b - p_a)/se
p_value = 2*(1 - norm.cdf(abs(z)))
```
