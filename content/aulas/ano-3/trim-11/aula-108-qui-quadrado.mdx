<Porta nivel="formal" titulo="Definição formal">

### Estatística

$$\chi^2 = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i}$$

onde $O_i$ é frequência observada e $E_i$ esperada na categoria $i$.

Sob $H_0$, $\chi^2 \sim \chi^2_{df}$ assintoticamente.

### Goodness of fit (aderência)

$$H_0: p_i = p_i^0, \quad i=1, \dots, k$$

$E_i = n p_i^0$. Graus de liberdade: $df = k - 1 - r$, onde $r$ é número de parâmetros estimados.

### Independência (tabela de contingência)

Para tabela $r \times c$:

$$E_{ij} = \frac{(\text{total linha } i)(\text{total coluna } j)}{n}$$

$$\chi^2 = \sum_{i,j} \frac{(O_{ij} - E_{ij})^2}{E_{ij}}, \quad df = (r-1)(c-1)$$

### Suposições

- Observações independentes.
- $E_i \geq 5$ para todas as células (ou pelo menos 80% delas).
- Categorias mutuamente exclusivas e exaustivas.

### Correção de Yates (2x2)

$$\chi^2 = \sum \frac{(|O_{ij} - E_{ij}| - 0{,}5)^2}{E_{ij}}$$

Para tabelas $2 \times 2$ com $E$ pequenos.

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Você tem **6 jujubas de cores diferentes** num saco e diz que elas saem **na mesma proporção**. Mas quando você tira 60, sai muito mais vermelha que verde.

O teste qui-quadrado pergunta: "**O que eu observei está perto do que eu esperava**?" Se está bem diferente, alguma coisa está errada.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

### Dois usos principais

1. **Aderência (goodness of fit)**: dado é justo? Distribuição segue Poisson? etc.
2. **Independência**: cor de cabelo está relacionada com cor dos olhos? Ou são independentes?

### Receita básica

1. Conte **observado** ($O_i$).
2. Calcule **esperado** ($E_i$) sob $H_0$.
3. Some $(O - E)^2 / E$ para cada categoria.
4. Compare com tabela $\chi^2$ usando $df$.

### Exemplo: dado é justo?

Joga 60 vezes. Observa: 12, 8, 11, 9, 13, 7. Esperado se justo: 10 cada.

$\chi^2 = (12-10)^2/10 + (8-10)^2/10 + \dots = (4+4+1+1+9+9)/10 = 2{,}8$

$df = 6 - 1 = 5$. Crítico a 5%: $11{,}07$. **Não rejeita** — dado parece justo.

### Tabela 2x2

| | Curou | Não curou |
|---|---|---|
| Droga | 80 | 20 |
| Placebo | 60 | 40 |

Calcule esperados, $\chi^2$, $df=(2-1)(2-1)=1$. Compare com $3{,}84$ (5%).

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

### Por que essa fórmula?

Cada $(O-E)^2/E$ é uma "padronização" da diferença. Lembra $Z^2$. Para muitas categorias, soma desses $Z^2$ aproxima $\chi^2$ pela teoria assintótica.

### Distribuição $\chi^2_k$

Soma de $k$ normais padrão ao quadrado. Média $k$, variância $2k$. Para $k$ grande, $\chi^2_k \approx \mathcal{N}(k, 2k)$.

### Goodness of fit detalhado

**Exemplo**: distribuição de cor de bala (cervejaria de Mars):

- Vermelho: esperado 20%
- Verde: 20%
- Amarelo: 20%
- Marrom: 20%
- Laranja: 20%

Observa em $n=200$ balas: 30 vermelhas, 50 verdes, 35 amarelas, 45 marrons, 40 laranja.

Esperado: 40 cada. $\chi^2 = (10^2 + 10^2 + 5^2 + 5^2 + 0)/40 = 250/40 = 6{,}25$

$df = 5-1 = 4$. Crítico 5%: $9{,}488$. Não rejeita.

### Independência

Tabela $r \times c$. $E_{ij} = (R_i C_j)/n$. Subtração $(r-1)(c-1)$ porque marginais são fixos.

**Exemplo**: gênero $\times$ preferência política em 200 pessoas. $E_{ij}$ = produto das marginais sobre $n$. Teste $\chi^2$.

### Quando NÃO usar $\chi^2$

- $E_i < 5$ em mais de 20% das células: use Fisher exato.
- Amostras pareadas: use McNemar (diferente!).
- Variável ordinal: use Mann-Whitney ou Kendall.

### Tamanho de efeito

**Cramér's V**: $V = \sqrt{\chi^2/(n \cdot \min(r-1, c-1))}$. $V \in [0,1]$. $0{,}1$ pequeno; $0{,}3$ médio; $0{,}5$ grande.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

### Derivação

Pela teoria multinomial, sob $H_0$, contagens $(O_1, \dots, O_k)$ seguem multinomial$(n, p^0)$. A estatística $\chi^2$ aproxima $\sum Z_i^2 \approx \chi^2_{k-1}$ por aplicação do TCL multivariado e covariância das marginais.

### Df detalhado

- **Goodness of fit puro**: $df = k - 1$.
- **Goodness of fit estimando $r$ parâmetros**: $df = k - 1 - r$.
- **Independência $r \times c$**: $df = (r-1)(c-1)$.
- **Homogeneidade entre populações**: mesmo $df$ que independência.

### Validade da aproximação

Regra Cochran: $E_i \geq 5$ em todos. Modificação: $\geq 1$ em todos e $\geq 5$ em 80%. Se viola: Fisher exato.

### McNemar (pareado)

Para tabela 2x2 pareada (mesmo sujeito antes/depois):

$$\chi^2 = \frac{(b - c)^2}{b + c}, \quad df = 1$$

onde $b, c$ são discordâncias.

### Likelihood ratio test alternativo

$$G^2 = 2 \sum O_i \ln(O_i/E_i)$$

Equivalente a $\chi^2$ assintoticamente. Em log-linear models é o padrão.

### Aplicações

- **Genética**: testar leis de Mendel em proporções fenotípicas.
- **Marketing**: variação por região em conversão.
- **Engenharia de qualidade**: distribuição de defeitos uniforme entre máquinas?
- **Epidemiologia**: associação entre exposição e doença.

### Implementação

```python
from scipy.stats import chisquare, chi2_contingency

# Goodness of fit
chi2, p = chisquare(observed, expected)

# Independência (tabela)
chi2, p, dof, expected = chi2_contingency(table)
```

### Exemplo completo

Tabela $3 \times 2$: três marcas, conversão.

|        | Comprou | Não    |
|--------|---------|--------|
| Marca A | 40 | 60 |
| Marca B | 50 | 50 |
| Marca C | 30 | 70 |

Total $n = 300$. Marginais: 120 compras (40%), 100 cada marca.

$E_{ij}$: marca A: 40, 60. Marca B: 40, 60. Marca C: 40, 60.

$\chi^2 = 0 + 0 + 100/40 + 100/60 + 100/40 + 100/60 = 2{,}5 + 1{,}67 + 2{,}5 + 1{,}67 = 8{,}33$

$df = 2$. Crítico 5%: $5{,}99$. **Rejeita** — marcas diferem em conversão.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

### Conexão com modelos log-lineares

Tabela de contingência é caso particular de **GLM Poisson** com log link. Modelo de independência:

$$\log E_{ij} = \mu + \alpha_i + \beta_j$$

Modelo saturado adiciona $(\alpha\beta)_{ij}$. Comparação via LR test = $G^2 = \chi^2$ assintoticamente.

### Tamanho amostral

Para detectar Cramér's $V = 0{,}3$ em $3 \times 3$ a poder 80%, $\alpha=0{,}05$:

```python
from statsmodels.stats.power import GofChisquarePower
analysis = GofChisquarePower()
analysis.solve_power(effect_size=0.3, alpha=0.05, power=0.80, n_bins=9)
```

### Bayesian alternative

Bayes factor para tabelas via Dirichlet-multinomial. Pacote `BayesFactor::contingencyTableBF` em R.

### Múltiplas correções

Testar $k$ tabelas: Bonferroni $\alpha/k$. Para post-hoc após $\chi^2$ global: residuais padronizados $|r_{ij}| > 1{,}96$ indicam células anômalas.

### Fisher's exact

Em tabelas pequenas ou desbalanceadas, distribuição hipergeométrica exata em vez de aproximação. `scipy.stats.fisher_exact`. Computacionalmente caro para tabelas grandes.

### Cochran-Mantel-Haenszel

Combina múltiplas tabelas $2 \times 2$ ajustando por estrato (Simpson's paradox).

### Goodness of fit para distribuições contínuas

- **Kolmogorov-Smirnov**: max distance entre CDF empírica e teórica.
- **Anderson-Darling**: pesa caudas mais.
- **Shapiro-Wilk**: específico para normalidade.

$\chi^2$ requer agrupar em bins — perda de info.

### Multinomial GLM

Para resposta categórica com mais de 2 níveis, multinomial logistic regression. Tabela "long" com função link multinomial.

### Crítica

$\chi^2$ assintótico. Para $n$ pequeno ou $E$ pequeno: Fisher. Para $n$ grande: tudo significativo, foque em $V$.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita: qui-quadrado correto.**

1. **Identifique tipo**: aderência ou independência?
2. **Verifique $E_i \geq 5$**: senão, Fisher exato.
3. **Calcule $E_i$**: para aderência, $np_i^0$; para independência, $(R_i C_j)/n$.
4. **Compute $\chi^2$**.
5. **$df$**: $k-1-r$ (aderência) ou $(r-1)(c-1)$ (independência).
6. **Compare** com tabela ou software.
7. **Tamanho de efeito**: Cramér's V.
8. **Reporte**: $\chi^2$, $df$, $p$, $V$, $n$.

### Erros frequentes

- **Aplicar $\chi^2$ em dados pareados**: use McNemar.
- **Ignorar células com $E < 5$**: aproximação falha.
- **$n$ enorme + V minúsculo**: significância sem relevância.
- **Tabela com totais marginais zerados**: indefinido.

### Reportando

"Houve associação significativa entre marca e conversão ($\chi^2(2, N=300) = 8{,}33$; $p = 0{,}016$; Cramér's $V = 0{,}17$, efeito pequeno)."

### Workflow

```python
import pandas as pd
from scipy.stats import chi2_contingency

table = pd.crosstab(df['marca'], df['comprou'])
chi2, p, dof, expected = chi2_contingency(table)

# Cramér's V
import numpy as np
n = table.sum().sum()
v = np.sqrt(chi2 / (n * (min(table.shape) - 1)))
```

### Visualização

- **Mosaic plot**: tamanhos proporcionais a frequências.
- **Heatmap de resíduos padronizados**: identifica células anômalas.

### Quando outros testes são melhores

| Situação | Use |
|---|---|
| 2x2, $n$ pequeno | Fisher exato |
| Pareado | McNemar |
| Variável ordinal | Mann-Whitney, Kendall |
| Múltiplas tabelas | Cochran-Mantel-Haenszel |
| Continua | KS, AD, Shapiro |
| Bayesian | BF Dirichlet-multinomial |
