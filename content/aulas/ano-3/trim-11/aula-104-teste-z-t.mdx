<Porta nivel="formal" titulo="Definições formais">

### Teste z (uma amostra)

Suposição: $\sigma$ conhecido, dados aproximadamente normais ou $n \geq 30$.

$$Z = \frac{\bar X - \mu_0}{\sigma/\sqrt n} \sim \mathcal{N}(0,1) \text{ sob } H_0$$

### Teste t (uma amostra)

Suposição: $\sigma$ desconhecido, dados aproximadamente normais.

$$T = \frac{\bar X - \mu_0}{s/\sqrt n} \sim t_{n-1} \text{ sob } H_0$$

### Teste t para duas amostras independentes

**Variâncias iguais (pooled t)**:

$$T = \frac{\bar X_1 - \bar X_2}{s_p \sqrt{1/n_1 + 1/n_2}}, \quad s_p^2 = \frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}$$

$df = n_1 + n_2 - 2$.

**Variâncias diferentes (Welch's t)**:

$$T = \frac{\bar X_1 - \bar X_2}{\sqrt{s_1^2/n_1 + s_2^2/n_2}}$$

$df$ por aproximação de Welch-Satterthwaite.

### Teste t pareado

Aplicado a pares $(X_i, Y_i)$. Calcule $D_i = X_i - Y_i$.

$$T = \frac{\bar D - 0}{s_D/\sqrt n} \sim t_{n-1}$$

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Você quer comparar a **altura média de meninos e meninas** da turma. Ou ver se **uma planta cresce mais com adubo**.

Tem dois tipos:

1. **Teste com tudo separado**: compara meninos com meninas. Cada criança aparece uma vez só.
2. **Teste pareado**: mede a planta antes do adubo, depois mede a mesma planta com adubo. É a mesma planta nos dois momentos.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

### Quando usar cada teste?

- **Teste z**: você sabe o $\sigma$ exato (raro!).
- **Teste t**: o normal — você só tem $s$ da amostra.

E para comparação:

- **Duas amostras independentes**: dois grupos diferentes (escola A vs escola B).
- **Pareado**: mesmo indivíduo medido duas vezes (antes/depois).

### Receita

1. Identifique o tipo (uma amostra? duas? pareado?).
2. Calcule a estatística $T$.
3. Compare com a tabela $t$ usando $df$.
4. Se $|T|$ é maior que o crítico, rejeite $H_0$.

### Exemplo pareado

Mediu pulsação de 10 pessoas antes e depois de café:

| Pessoa | Antes | Depois | Diff |
|---|---|---|---|
| 1 | 70 | 78 | +8 |
| 2 | 65 | 72 | +7 |
| ... | | | |

Calcula $\bar D$ e $s_D$. Faz $T = \bar D / (s_D/\sqrt n)$. Pronto.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

### Por que t e não z na maioria dos casos?

$\sigma$ raramente é conhecido. Substituir por $s$ introduz incerteza extra. A distribuição $t$ tem **caudas mais pesadas** que normal. Quando $n \to \infty$, $t \to z$.

### Pooled vs Welch

**Pooled t** assume $\sigma_1 = \sigma_2$. Mais poderoso quando válido. **Welch** não assume — mais robusto, recomendado por padrão.

Regra prática (Ruxton 2006): **sempre use Welch**, salvo se realmente sabe que variâncias são iguais.

### Pareado vs independente

Pareado é **muito mais poderoso** quando há correlação entre pares (mesmo indivíduo, gêmeos, mesma loja em momentos diferentes). Reduz variância removendo efeito individual.

**Erro fatal**: usar teste independente em dados pareados perde poder enorme.

### Exemplo: nova dieta

Mediu peso de 12 pessoas antes e depois de 30 dias:

$$\bar D = -2{,}5 \text{ kg}, \quad s_D = 1{,}8 \text{ kg}, \quad n=12$$

$$T = -2{,}5 / (1{,}8/\sqrt{12}) = -4{,}81$$

$df = 11$. Crítico bilateral $\alpha=0{,}05$: $\pm 2{,}201$. **Rejeita** $H_0$. Dieta funciona.

### Quando teste t falha?

- Dados muito não-normais com $n$ pequeno.
- Outliers extremos.
- Dados ordinais (use Mann-Whitney ou Wilcoxon).

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

### Família t e seus parentes

A distribuição $t_\nu$ é a razão entre $Z \sim \mathcal{N}(0,1)$ e $\sqrt{\chi^2_\nu/\nu}$ independentes.

$$T = \frac{Z}{\sqrt{V/\nu}}, \quad V \sim \chi^2_\nu$$

**Fórmulas-chave por cenário:**

| Cenário | Estatística | df |
|---|---|---|
| Uma amostra, $\sigma$ conhecido | $Z = \frac{\bar X - \mu_0}{\sigma/\sqrt n}$ | — |
| Uma amostra, $\sigma$ estimado | $T = \frac{\bar X - \mu_0}{s/\sqrt n}$ | $n-1$ |
| Duas amostras pooled | $T_p = \frac{\bar X_1 - \bar X_2}{s_p \sqrt{1/n_1+1/n_2}}$ | $n_1+n_2-2$ |
| Duas amostras Welch | $T_w = \frac{\bar X_1 - \bar X_2}{\sqrt{s_1^2/n_1+s_2^2/n_2}}$ | Welch-Sat. |
| Pareado | $T = \frac{\bar D}{s_D/\sqrt n}$ | $n-1$ |

### Welch-Satterthwaite

$$df_w = \frac{(s_1^2/n_1 + s_2^2/n_2)^2}{\frac{(s_1^2/n_1)^2}{n_1-1} + \frac{(s_2^2/n_2)^2}{n_2-1}}$$

Geralmente fracionário; software calcula automaticamente.

### Tamanho de efeito

**Cohen's d** para 2 amostras:

$$d = \frac{\bar X_1 - \bar X_2}{s_p}$$

Convenção: $d=0{,}2$ pequeno, $d=0{,}5$ médio, $d=0{,}8$ grande.

### A/B testing prático

Para comparar tempo médio na página entre versões:

```python
from scipy import stats
t_stat, p_value = stats.ttest_ind(timeA, timeB, equal_var=False)
# Welch's t por padrão (equal_var=False é a recomendação moderna)
```

### Verificação de premissas

1. **Normalidade**: Shapiro-Wilk se $n < 50$. Q-Q plot sempre.
2. **Homocedasticidade** (pooled): teste F ou Levene. Para Welch, não precisa.
3. **Independência**: design experimental (não estatística).
4. **Outliers**: visualização.

### Quando t não dá

- Dados ordinais → Mann-Whitney U
- Não-normal forte + $n$ pequeno → Bootstrap ou permutation
- Múltiplos grupos → ANOVA (Aula 107)

### Cross-domínio

- **Engenharia**: comparação de processos (corte vs fundição) — t pareado em peças idênticas.
- **Medicina**: ensaio crossover — pareado é padrão.
- **Marketing**: A/B test em landing pages — Welch t para conversão.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

### Robustez do teste t

Teste t é **robusto à não-normalidade** moderada se $n \geq 30$ (TCL). É **frágil a outliers extremos** e **assimetria forte**.

### Alternativas modernas

- **Yuen's t**: trimmed means + Welch. Robusto a outliers, mantém poder.
- **Bootstrap-t**: usa percentis bootstrap em vez da distribuição t analítica.
- **Permutação exata**: $p$-valor via reembaralhamento de rótulos. Sem suposições paramétricas.

### Welch é default moderno

Pooled t domina quando $\sigma_1 = \sigma_2$ exatamente. Na prática, **Welch é quase igual em poder** quando isso vale, e **muito melhor** quando não vale. Hipóteses prévias quase nunca justificam pooled.

### Múltiplas comparações

Comparando $k$ grupos par a par: $\binom{k}{2}$ testes. Use Bonferroni ou Tukey HSD para controle de FWER.

### Bayesian t-test

JZS Bayes factor (Rouder et al. 2009) com prior Cauchy em $d$. Pacote `BayesFactor` em R. Reporta $BF_{10}$ — quão mais provável $H_1$ vs $H_0$.

### TOST (equivalence)

Para afirmar que dois grupos são **equivalentes**:

$$H_0: |\mu_1 - \mu_2| > \delta \quad H_1: |\mu_1 - \mu_2| \leq \delta$$

Implementado como dois testes t unilaterais. Padrão em farmacologia (bioequivalência genérico).

### Heterocedasticidade severa

Se $\sigma_1 \gg \sigma_2$, considere transformação (log, sqrt) ou modelo paramétrico explícito (GLM com variância função da média).

### Análise sequencial

Wald SPRT para testes t com early stopping. Pacote `confseq` permite p-valor anytime-valid.

### Implementação rigorosa

```python
import scipy.stats as st
import numpy as np

# Welch's t com tamanho de efeito e IC
t, p = st.ttest_ind(x, y, equal_var=False)
d = (x.mean() - y.mean()) / np.sqrt(((len(x)-1)*x.var(ddof=1) + (len(y)-1)*y.var(ddof=1)) / (len(x)+len(y)-2))
print(f"t={t:.2f}, p={p:.4f}, d={d:.2f}")
```

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita de bolo: como escolher o teste certo.**

```
Quantos grupos?
├─ 1: teste t uma amostra (vs valor de referência)
├─ 2:
│  ├─ Pareados (mesmo sujeito): t pareado
│  └─ Independentes:
│     ├─ Variâncias iguais? → t pooled
│     └─ Não sei: → Welch t (default seguro)
└─ 3+: ANOVA (Aula 107)
```

### Workflow completo

1. **Visualize** os dados (boxplot, histograma).
2. **Cheque outliers** e normalidade.
3. **Escolha o teste** pela árvore acima.
4. **Calcule** estatística, $p$, IC, tamanho de efeito.
5. **Reporte** todos: $t$, $df$, $p$, IC, $d$.

### Erros frequentes

- **Independente quando deveria ser pareado**: perda massiva de poder.
- **Pooled quando variâncias diferem**: $p$ inválido.
- **Ignorar premissas**: t-test não é mágica.
- **Múltiplos testes sem correção**: $p$-hacking.

### Exemplo: A/B test

Versão A vs B do site, métrica: tempo na página.

```python
from scipy import stats
import numpy as np

A = np.array([...])  # n=500
B = np.array([...])  # n=500
t, p = stats.ttest_ind(A, B, equal_var=False)
ci_diff = ...  # via bootstrap ou fórmula Welch
```

Reporte: "B teve média 125 s vs 118 s para A; Welch t(987) = $2{,}81$; p = $0{,}005$; diferença média 7 s (IC 95%: $2{,}1$ a $11{,}9$ s); Cohen's d = $0{,}18$ (efeito pequeno)."

### Quando não usar t

- Variável ordinal (escala 1-5): Wilcoxon ou Mann-Whitney.
- Dados de contagem: Poisson regression.
- Cauda muito pesada: trimmed mean t ou bootstrap.

### Pareamento natural

Sempre que possível, **paire**! Estudos crossover, before/after, gêmeos, irmãos, lojas pareadas por região. Reduz variância tremendamente.
