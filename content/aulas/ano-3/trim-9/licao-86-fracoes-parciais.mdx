<Porta nivel="formal" titulo="Decomposição e procedimento">

### Teorema (frações parciais sobre $\mathbb{R}$)

Se $P, Q \in \mathbb{R}[x]$ com $\deg P < \deg Q$, e $Q$ fatora como
$$Q(x) = \prod_i (x - r_i)^{m_i} \prod_j (x^2 + p_j x + q_j)^{n_j}$$
(com $p_j^2 - 4q_j < 0$, fatores irredutíveis), então
$$\frac{P}{Q} = \sum_i \sum_{k=1}^{m_i} \frac{A_{i,k}}{(x - r_i)^k} + \sum_j \sum_{l=1}^{n_j} \frac{B_{j,l} x + C_{j,l}}{(x^2 + p_j x + q_j)^l},$$
e a decomposição é única.

### Caso $\deg P \geq \deg Q$

Faça **divisão polinomial primeiro**: $P/Q = S(x) + R(x)/Q(x)$ com $\deg R < \deg Q$.

### Procedimento

1. Fatore $Q(x)$ em $\mathbb{R}$.
2. Escreva a decomposição genérica.
3. Multiplique tudo por $Q(x)$ e iguale numeradores.
4. Determine constantes:
   - **Cover-up** (raízes simples): substitua $x = r_i$.
   - **Comparação de coeficientes**: expanda e iguale.
5. Integre cada termo:
   - $\int A/(x-r)\, dx = A \ln\lvert x - r\rvert + C$.
   - $\int A/(x-r)^k\, dx = -A/((k-1)(x-r)^{k-1}) + C$ ($k > 1$).
   - $\int (Bx + C)/(x^2 + px + q)\, dx$: complete quadrado → $\ln$ + $\arctan$.

### Exemplo

$\dfrac{1}{x^2 - 1} = \dfrac{1}{2(x-1)} - \dfrac{1}{2(x+1)}$. Logo
$$\int \frac{dx}{x^2 - 1} = \frac{1}{2} \ln\left|\frac{x-1}{x+1}\right| + C.$$

Você pode **cortar de novo**, em pedaços normais — alguns triangulares, outros quadradinhos. Aí fica fácil.

Frações parciais é cortar uma fração esquisita em fraçõezinhas fáceis de integrar.

<Porta nivel="10" titulo="Pra criança de 10 anos">

Você sabe que $\frac{1}{2} + \frac{1}{3} = \frac{5}{6}$. Foi do **separado** pro **junto**.

Frações parciais é o **caminho de volta**: dado $\frac{5}{6}$ como "já somado", você descobre que era $\frac{1}{2} + \frac{1}{3}$.

Pra polinômios é igual: dado $\frac{1}{(x-1)(x+1)}$, descobre que é $\frac{1/2}{x-1} - \frac{1/2}{x+1}$.

Por que isso ajuda? Porque integrar $\frac{1}{x-1}$ é **fácil**: vira $\ln\lvert x - 1\rvert$. Integrar a fração junta é mais difícil.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Pra integrar $\int \frac{P(x)}{Q(x)}\, dx$ com $P, Q$ polinômios:

1. **Garanta** $\deg P < \deg Q$ (senão divide primeiro).
2. **Fatore $Q$**.
3. **Escreva decomposição** com letras.
4. **Resolva o sistema** pelas constantes.
5. **Integre termo a termo**.

### Tipos de termo

- **Raiz real simples** $(x - r)$ → $\frac{A}{x - r}$.
- **Raiz real de multiplicidade $m$** → $\frac{A_1}{x-r} + \frac{A_2}{(x-r)^2} + \cdots + \frac{A_m}{(x-r)^m}$.
- **Quadrático irredutível** $(x^2 + px + q)$ → $\frac{Bx + C}{x^2 + px + q}$.

### Exemplo trabalhado

$\int \frac{1}{x^2 - 1}\, dx$.

Fatora: $x^2 - 1 = (x-1)(x+1)$.

Decompõe: $\frac{1}{(x-1)(x+1)} = \frac{A}{x-1} + \frac{B}{x+1}$. Multiplicando: $1 = A(x+1) + B(x-1)$.

Cover-up: $x = 1 \Rightarrow A = 1/2$. $x = -1 \Rightarrow B = -1/2$.

Integra: $\int \frac{1/2}{x-1} - \int \frac{1/2}{x+1} = \frac{1}{2}\ln\lvert x-1\rvert - \frac{1}{2}\ln\lvert x+1\rvert + C$.

### Por que importa

Em **circuitos elétricos**, **Laplace inversa** depende disso. Em **probabilidade racional**, idem. É a base operacional pra **decompor qualquer função racional**.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Frações parciais é **a** técnica em transformada de Laplace inversa — convertendo $F(s)$ em $f(t)$.

### Caso geral

Se $Q(x) = (x - r_1)(x - r_2) \cdots$ com raízes simples:
$$\frac{P(x)}{Q(x)} = \sum_i \frac{P(r_i)}{Q'(r_i)} \cdot \frac{1}{x - r_i}$$
(fórmula de Heaviside).

### Quadrático irredutível

$\int \frac{Bx + C}{x^2 + px + q}\, dx$: 
1. Manipule numerador pra ter derivada do denominador (pra $\ln$).
2. Resto é constante × $1/((x + p/2)^2 + (\text{cte}))$, que vira $\arctan$.

### Multiplicidade alta

$\int \frac{A_k}{(x - r)^k}\, dx = \frac{-A_k}{(k-1)(x - r)^{k-1}}$ para $k > 1$.

### Conexão com Laplace

Pra resolver EDOs lineares de coef. constantes: $\mathcal{L}^{-1}\left\{\frac{P(s)}{Q(s)}\right\}$ via frações parciais. Cada termo $1/(s - r_i)^k$ inverte para $t^{k-1} e^{r_i t}/(k-1)!$.

### Em circuitos

Função de transferência $H(s) = N(s)/D(s)$. Frações parciais → resposta no tempo via Laplace inversa. Padrão em controle e sinais.

### CAS

```python
import sympy as sp
x = sp.symbols('x')
print(sp.apart(1/(x**2 - 1)))
# Out: -1/(2*(x + 1)) + 1/(2*(x - 1))
print(sp.integrate(1/(x**2 - 1), x))
```

`apart` faz a decomposição automaticamente.

### Cuidados

- Erro de sinal em raízes negativas.
- Esquecer divisão polinomial quando $\deg P \geq \deg Q$.
- Confundir multiplicidade.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

### Frações parciais sobre $\mathbb{C}$

Sobre $\mathbb{C}$, todo polinômio fatora em lineares, e a decomposição só tem termos $A/(x - r)^k$. Útil em análise complexa (cálculo de resíduos, transformadas).

### Resíduos

Para raiz simples $r$ de $Q$: $A = \text{Res}_{x = r} P/Q = P(r)/Q'(r)$. Conexão direta com **integrais de contorno** e **transformada de Laplace**.

### Hermiticidade e função de Green

Pra equação $L u = f$ com $L$ operador linear, $G(x, x') = \sum_n \phi_n(x) \phi_n(x') / \lambda_n$ (decomposição espectral). Frações parciais é caso 1D.

### Algoritmos

- **Bareiss / Lagrange** pra interpolação direta.
- **Gauss-Jordan** pra sistema de coeficientes.
- **Cover-up de Heaviside** + **derivadas iteradas** pra multiplicidade.

### Computação numérica

Em produção (controle, processamento de sinais), `scipy.signal.residue` retorna polos e resíduos. Cuidado com **polos próximos** — má-condicionamento numérico.

### Conexão com EDO sistemas

Sistemas lineares $\dot x = Ax$ resolvem-se via $e^{At}$ que, espectralmente, decompõe em $\sum c_i e^{\lambda_i t}$ — frações parciais em $(sI - A)^{-1}$.

### Limitação

Para função racional **transcendental** (com $\sin, \cos$, etc), frações parciais não se aplica — é técnica puramente algébrica.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita pra frações parciais.**

1. Verifique $\deg P < \deg Q$. Se não, divida.
2. Fatore $Q$ completamente em $\mathbb{R}$.
3. Escreva decomposição genérica (com letras).
4. Determine letras:
   - **Cover-up**: simples e rápido pra raízes simples.
   - **Comparação**: pra outros casos.
5. Integre cada termo.
6. Confira derivando.

### Erros comuns

- Esquecer divisão quando grau alto.
- Multiplicidade $> 1$: precisa termos $1/(x-r), 1/(x-r)^2, \ldots$.
- Quadrático irredutível: numerador é $Bx + C$, não só $B$.
- $\ln\lvert \cdot\rvert$ vs $\ln(\cdot)$.

### CAS

```python
sp.apart(P/Q)  # decomposição
sp.integrate(P/Q, x)  # direto
```

### Quando aparece

- Laplace inversa em controle.
- Resposta de circuitos RLC.
- Probabilidade de sistemas Markov racionais.
- EDOs lineares.

### Exemplo trabalhado

$\int \frac{2x + 3}{x^2 + 2x + 5}\, dx$.

Numerador = derivada do denominador (que é $2x + 2$) + $1$. Quebre:

$\int \frac{2x + 2}{x^2 + 2x + 5}\, dx + \int \frac{1}{x^2 + 2x + 5}\, dx$

= $\ln(x^2 + 2x + 5) + \int \frac{1}{(x+1)^2 + 4}\, dx$

= $\ln(x^2 + 2x + 5) + \frac{1}{2}\arctan\left(\frac{x+1}{2}\right) + C$.
