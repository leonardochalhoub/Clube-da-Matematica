<Porta nivel="formal" titulo="Definição e tabela">

<Definicao titulo="Antiderivada e integral indefinida">
$F$ é **antiderivada** de $f$ em $I$ se $F'(x) = f(x)$ para todo $x \in I$. A **integral indefinida** $\int f(x)\, dx$ denota a família de todas as antiderivadas; vale $F(x) + C$ onde $C \in \mathbb{R}$ é arbitrária.
</Definicao>

### Por que a constante?

Se $F' = f$, então $(F + C)' = F' + 0 = f$ também. Logo a antiderivada não é única — difere por constante.

### Tabela essencial

| $f(x)$ | $\int f(x)\, dx$ |
|---|---|
| $x^n$ ($n \neq -1$) | $\frac{x^{n+1}}{n+1} + C$ |
| $1/x$ | $\ln|x| + C$ |
| $e^x$ | $e^x + C$ |
| $a^x$ | $a^x / \ln a + C$ |
| $\sin x$ | $-\cos x + C$ |
| $\cos x$ | $\sin x + C$ |
| $\sec^2 x$ | $\tan x + C$ |
| $1/(1+x^2)$ | $\arctan x + C$ |
| $1/\sqrt{1-x^2}$ | $\arcsin x + C$ |

### Linearidade

$\int (\alpha f + \beta g)\, dx = \alpha \int f + \beta \int g$.

### Não-comutatividade do produto

**$\int (fg)\, dx \neq (\int f)(\int g)$** em geral. Para produtos use **integração por partes** (Aula 85).

### Exemplo

$\int (3x^2 - 4\sin x + e^x)\, dx = x^3 + 4\cos x + e^x + C$.

### Cuidado

- **Sempre** colocar $+C$. Esquecer é o erro #1 em provas.
- O domínio importa: $\int 1/x = \ln|x| + C$ — mas em $(0, \infty)$ e $(-\infty, 0)$ as constantes podem ser diferentes.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Antiderivada é "**desfazer derivada**". Se você sabe que $f'(x) = 2x$, qual era $f$? Resposta: $f(x) = x^2$ — porque $(x^2)' = 2x$.

Mas atenção: $f(x) = x^2 + 5$ também tem derivada $2x$. E $f(x) = x^2 - 17$. Qualquer constante somada funciona. Por isso escrevemos:

$$\int 2x\, dx = x^2 + C$$

O **C** é a "**constante de integração**" — admite que não temos info pra fixar a constante.

**Truque pra usar a tabela**: pra $\int x^n$, soma 1 ao expoente e divide pelo novo. Pra $x^3$: vira $x^4/4$. Pra $x^7$: vira $x^8/8$.

**Onde aparece**: o TFC (Aula 83) diz que pra calcular áreas e somatórios contínuos, basta achar a antiderivada. Por isso você está aprendendo isso primeiro.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Antiderivação é **operacionalmente** o oposto da derivação, mas **computacionalmente** muito mais difícil. Há funções elementares cuja antiderivada NÃO é elementar:

- $\int e^{-x^2}\, dx$ — relacionada à $\text{erf}$, sem fórmula fechada elementar.
- $\int \sin(x^2)\, dx$ — função de Fresnel.
- $\int \sin(x)/x\, dx$ — seno integral $\text{Si}(x)$.

**Liouville (1835)** caracteriza quais funções têm antiderivada elementar — resultado profundo, base de **algoritmos de integração simbólica** (Risch 1969) usados em Mathematica/SymPy.

### Estratégia geral de integração

Quando vê $\int f\, dx$:
1. **Tabela direta?** Use.
2. **Linearidade?** Quebre em soma/escalar.
3. **Substituição** $u$? Inversa da regra da cadeia (Aula 84).
4. **Por partes?** Inversa da regra do produto (Aula 85).
5. **Frações parciais** (Aula 86) se for racional.
6. **Identidade trigonométrica** (Aula 87).
7. **Sem antiderivada elementar?** Use série de Taylor ou método numérico.

### Em prática profissional

- **Mathematica/SymPy** resolvem 99% das integrais elementares automaticamente.
- O valor de aprender técnicas é **entender o resultado** + **reconhecer estrutura** em problemas mais amplos (séries, EDOs, transformadas).

</Porta>
