### Identidades fundamentais

- $\sin^2 x + \cos^2 x = 1$.
- $\sin^2 x = (1 - \cos 2x)/2$.
- $\cos^2 x = (1 + \cos 2x)/2$.
- $\sin x \cos x = \sin(2x)/2$.
- $\tan^2 x + 1 = \sec^2 x$.
- $\sin A \cos B = \frac{1}{2}[\sin(A+B) + \sin(A-B)]$.

### Padrões $\int \sin^n x \cos^m x\, dx$

| Caso | Estratégia |
|---|---|
| $n$ ímpar | Reserve $\sin x\, dx$, converta resto via $\sin^2 = 1 - \cos^2$, sub $u = \cos x$ |
| $m$ ímpar | Reserve $\cos x\, dx$, sub $u = \sin x$ |
| Ambos pares | Use $\sin^2 = (1-\cos 2x)/2$ e $\cos^2 = (1+\cos 2x)/2$ |

### Padrões $\int \tan^n x \sec^m x\, dx$

| Caso | Estratégia |
|---|---|
| $m$ par | Reserve $\sec^2 x\, dx$, converta via $\sec^2 = 1 + \tan^2$, sub $u = \tan x$ |
| $n$ ímpar | Reserve $\sec x \tan x\, dx$, sub $u = \sec x$ |

### Substituição trigonométrica

Quando vê **radical**, faça:

| Forma | Sub | $dx$ | Identidade |
|---|---|---|---|
| $\sqrt{a^2 - x^2}$ | $x = a \sin\theta$ | $a\cos\theta\, d\theta$ | $\sqrt{a^2 - x^2} = a\cos\theta$ |
| $\sqrt{a^2 + x^2}$ | $x = a \tan\theta$ | $a\sec^2\theta\, d\theta$ | $\sqrt{a^2 + x^2} = a\sec\theta$ |
| $\sqrt{x^2 - a^2}$ | $x = a \sec\theta$ | $a\sec\theta\tan\theta\, d\theta$ | $\sqrt{x^2 - a^2} = a\tan\theta$ |

Após integrar, **volte** com triângulo de referência.

### Fórmulas de redução

$$\int \sin^n x\, dx = -\frac{\sin^{n-1} x \cos x}{n} + \frac{n-1}{n} \int \sin^{n-2} x\, dx.$$

(Idem $\cos^n$, $\tan^n$, $\sec^n$.)

Sabe quando você usa um chapéu pra trocar de roupa sem ninguém ver? Substituição trig é assim: você troca $x$ por $\sin\theta$, integra fácil, e depois "tira o chapéu" voltando pra $x$.

<Porta nivel="10" titulo="Pra criança de 10 anos">

Tem um teorema da escola: $\sin^2\theta + \cos^2\theta = 1$. Isso significa $\sqrt{1 - \sin^2\theta} = \cos\theta$.

Então se você tem $\sqrt{1 - x^2}$ e troca $x = \sin\theta$, o radical desaparece e vira $\cos\theta$. Magia!

Depois de integrar com $\theta$, você "destroca" pra $x$ usando triângulo: $\sin\theta = x$ é o cateto oposto sobre hipotenusa, então um triângulo com cateto $x$ e hipotenusa $1$ tem cateto adjacente $\sqrt{1-x^2}$. Daí lê todos os $\sin, \cos, \tan$ de $\theta$ em função de $x$.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

### Caso 1: potências de seno e cosseno

$\int \sin^n x \cos^m x\, dx$ — estratégia depende dos expoentes.

**Exemplo (um ímpar)**: $\int \sin^3 x\, dx = \int (1 - \cos^2 x) \sin x\, dx$. Sub $u = \cos x$: $\int -(1 - u^2)\, du = -u + u^3/3 + C = -\cos x + \cos^3 x/3 + C$.

**Exemplo (ambos pares)**: $\int \sin^2 x\, dx = \int (1 - \cos 2x)/2\, dx = x/2 - \sin(2x)/4 + C$.

### Caso 2: radicais

Pra integrar coisas como $\int \sqrt{1 - x^2}\, dx$ (área de um setor circular!), faça $x = \sin\theta$. Aí $\sqrt{1 - x^2} = \cos\theta$ e $dx = \cos\theta\, d\theta$.

A integral vira $\int \cos^2\theta\, d\theta = \theta/2 + \sin(2\theta)/4 + C = \theta/2 + \sin\theta\cos\theta/2 + C$.

Volta pra $x$: $\theta = \arcsin x$, $\sin\theta = x$, $\cos\theta = \sqrt{1-x^2}$.

Resultado: $\arcsin(x)/2 + x\sqrt{1-x^2}/2 + C$.

### Triângulo de referência

Sempre desenhe! Se $x = a \sin\theta$, faça triângulo com hipotenusa $a$, cateto oposto $x$ e adjacente $\sqrt{a^2 - x^2}$. Lê todas as razões direto.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Conjunto de técnicas pra integrais com **trig** ou **radicais quadráticos**.

### Tabela mental

- $\int \sin^2 = (\theta - \sin\theta\cos\theta)/2$.
- $\int \cos^2 = (\theta + \sin\theta\cos\theta)/2$.
- $\int \tan = -\ln\lvert\cos\rvert$, $\int \cot = \ln\lvert\sin\rvert$.
- $\int \sec = \ln\lvert\sec + \tan\rvert$, $\int \csc = -\ln\lvert\csc + \cot\rvert$.
- $\int \sec^2 = \tan$, $\int \csc^2 = -\cot$.
- $\int \sec\tan = \sec$.

### Sub trig em circuitos

Reatância capacitiva, indutiva, ressonância — fórmulas envolvendo $\sqrt{R^2 + X^2}$. Sub trig aparece em integrais de transferência analítica.

### Aplicações

- **Áreas**: setor circular, elipse.
- **Comprimentos de arco**: $L = \int \sqrt{1 + (f')^2}$.
- **Trabalhos**: força ao longo de curva.
- **Probabilidade**: distribuição de Cauchy $f(x) = 1/(\pi(1+x^2))$ — sub trig $x = \tan\theta$.

### Conexão com complexos

Identidade de Euler $e^{i\theta} = \cos\theta + i\sin\theta$ permite escrever potências trig como exponenciais. Frequentemente mais limpo:
$$\sin^n x = \left(\frac{e^{ix} - e^{-ix}}{2i}\right)^n.$$

### Fórmulas de redução

Em prática profissional, pra $n$ alto: use redução, ou integre simbolicamente em CAS.

```python
import sympy as sp
x = sp.symbols('x')
print(sp.integrate(sp.sin(x)**5 * sp.cos(x)**2, x))
```

### Risco principal

Esquecer de **voltar pra $x$** com triângulo após sub trig.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

### Universal substitution (Weierstrass)

$t = \tan(x/2)$ transforma qualquer integral racional em $\sin, \cos$ em integral racional em $t$:
$$\sin x = \frac{2t}{1+t^2}, \quad \cos x = \frac{1-t^2}{1+t^2}, \quad dx = \frac{2\, dt}{1+t^2}.$$

Útil em CAS pra reduzir integrais trig a frações parciais.

### Fourier

Coeficientes $a_n, b_n$ de uma função periódica via integrais $\int f \cos(nx), \int f \sin(nx)$ — partes + integrais trig recorrentes.

### Quântica

Funções de onda do oscilador harmônico envolvem polinômios de Hermite × $e^{-x^2/2}$. Normalizações exigem integrais trig + Gaussianas.

### Geometria

Comprimento de elipse: $\int \sqrt{1 - k^2 \sin^2\theta}\, d\theta$ — **integral elíptica**, sem fórmula fechada elementar. Define $E(k)$. Tabela e numérico.

### Sub trig em alta dimensão

Mudança esférica, polar, cilíndrica — todas Jacobianas com $\sin, \cos$. Mecânica quântica (orbital), eletromagnetismo.

### Cuidados

- Em sub trig, a **escolha de ramo** importa: $\sec\theta = \sqrt{1 + \tan^2\theta}$ assume $\theta \in (-\pi/2, \pi/2)$.
- $\sqrt{x^2 - a^2}$: $x \geq a$ → $\theta \in [0, \pi/2)$; $x \leq -a$ → $\theta \in [\pi/2, \pi)$. Quebrar.

### Limitação

Integrais trig com $\sin(P(x))$, $P$ polinômio grau $\geq 2$ não são elementares — viram funções especiais (Fresnel).

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita pra integrais trig.**

1. Veja **expoentes**. Algum ímpar? Reserve uma cópia, converte resto.
2. Ambos pares? Use **half-angle** $\sin^2 = (1 - \cos 2x)/2$.
3. Radical $\sqrt{a^2 \pm x^2}$ ou $\sqrt{x^2 - a^2}$? **Sub trig**.
4. Após sub trig, **desenhe o triângulo** de referência pra voltar pra $x$.
5. Confira derivando.

### Atalhos mentais

- $\int \sin x \cos x\, dx = \sin^2 x / 2 + C$ (ou $-\cos^2 x / 2 + C$).
- $\int \tan x\, dx = -\ln\lvert \cos x\rvert + C = \ln\lvert \sec x\rvert + C$.
- $\int \sec x\, dx = \ln\lvert \sec x + \tan x\rvert + C$ (truque: multiplique por $(\sec + \tan)/(\sec + \tan)$).

### SymPy

```python
sp.integrate(sp.sqrt(1 - x**2), x)
# Out: x*sqrt(1 - x**2)/2 + asin(x)/2
```

### Erros comuns

- Esquecer Jacobiana $dx \to d\theta$ na sub trig.
- Não voltar de $\theta$ pra $x$.
- Confundir $\sin^2$ (potência) com $\sin(2x)$ (ângulo dobrado).
- Em sub $x = a\sec\theta$, esquecer escolha de ramo.

### Aplicação favorita

Área do círculo: $A = 4 \int_0^r \sqrt{r^2 - x^2}\, dx$. Sub $x = r\sin\theta$. Após contas: $A = \pi r^2$. Validação canônica.

</Porta>
