<Definicao titulo="Integral de Riemann">
Seja $f: [a,b] \to \mathbb{R}$ limitada. Para uma partição $P = \{a = x_0 < x_1 < \cdots < x_n = b\}$ e pontos amostrais $x_i^* \in [x_{i-1}, x_i]$, a **soma de Riemann** é $S(f, P) = \sum_{i=1}^n f(x_i^*) \Delta x_i$. Se $S(f, P) \to L$ quando $\|P\| \to 0$ (independente das escolhas), $f$ é **integrável** e $\int_a^b f(x)\, dx := L$.
</Definicao>

### Critério de integrabilidade

Toda $f$ **contínua** em $[a, b]$ é integrável. Mais geral: $f$ limitada com **conjunto de descontinuidades de medida nula** (Lebesgue) é integrável Riemann.

### Propriedades

- **Linearidade**: $\int (\alpha f + \beta g) = \alpha \int f + \beta \int g$.
- **Aditividade**: $\int_a^c f = \int_a^b f + \int_b^c f$.
- **Monotonicidade**: $f \leq g \Rightarrow \int_a^b f \leq \int_a^b g$.
- **Inversão**: $\int_b^a f = -\int_a^b f$.
- **TVM integral**: existe $c \in [a,b]$ com $\int_a^b f = f(c)(b-a)$ se $f$ contínua.

### Área orientada

Se $f \geq 0$, $\int_a^b f$ = área entre o gráfico e o eixo $x$. Se $f < 0$, contribui **negativamente**. Por isso "**orientada**".

### Soma de Darboux

Definição alternativa: somas inferior $L(f, P) = \sum \inf_{[x_{i-1}, x_i]} f \cdot \Delta x_i$ e superior $U(f, P) = \sum \sup f \cdot \Delta x_i$. $f$ integrável $\iff \sup_P L = \inf_P U$.

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Imagina o desenho de uma montanha. Você quer pintar tudo o que está **embaixo da montanha** com tinta. **Quanta tinta você precisa?**

Pra responder, você divide a montanha em **muitos quadradinhos** finos, mede cada um e soma. Quanto mais fininhos os quadradinhos, mais certo fica.

Isso é a integral: **somar quadradinhos pra medir uma área**.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

Você sabe calcular área de retângulo: base × altura. Mas e se a "altura" muda o tempo todo, como uma montanha?

A ideia é **dividir a base em pedaços pequenos**, fazer um retângulo pra cada pedaço (com a altura mais ou menos certa), e somar todas as áreas.

$$A \approx f(x_1)\Delta x + f(x_2)\Delta x + \cdots$$

Se você usa **muitos retângulos finos**, o erro fica pequeno. **Quando os retângulos somem (ficam infinitamente finos), a soma converge** pra área exata.

Esse limite é a **integral definida**: $\int_a^b f(x)\, dx$. O símbolo $\int$ é um "S esticado" — vem de "soma".

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

A integral $\int_a^b f(x)\, dx$ é a **área orientada** entre o gráfico de $f$ e o eixo $x$, no intervalo $[a, b]$.

**Como aproximar**: divide $[a, b]$ em $n$ pedaços iguais de largura $\Delta x = (b-a)/n$. Em cada pedaço, escolhe um ponto $x_i^*$ e faz o retângulo de altura $f(x_i^*)$.

$$\sum_{i=1}^n f(x_i^*) \Delta x \approx \int_a^b f$$

Quando $n \to \infty$, o erro vai a zero (se $f$ é decente, tipo contínua).

**Por que "orientada"**: se $f < 0$, a "área" entra negativa. $\int_0^{2\pi} \sin x\, dx = 0$ — metade positiva, metade negativa.

**Você não vai calcular Riemann sempre** — o TFC (Aula 83) faz isso virar uma operação de antiderivar. Mas a **definição** importa: ela conecta integral com **somatórios**, e isso aparece em probabilidade, física, economia.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Riemann é o **conceito**; numericamente você usa quadratura.

### Riemann esquerdo, direito, médio, Simpson

- **Esquerdo**: $f(x_i^*) = f(x_{i-1})$. Erro $O(1/n)$.
- **Direito**: $f(x_i^*) = f(x_i)$. Erro $O(1/n)$.
- **Médio**: $f(x_i^*) = f((x_{i-1}+x_i)/2)$. Erro $O(1/n^2)$.
- **Trapézio**: média de esquerdo e direito. $O(1/n^2)$.
- **Simpson** (parabólico): $O(1/n^4)$. Padrão em produção.

### Conexão com somatórios

Soma de Riemann é o **continuum limit** de somas discretas. Em probabilidade: $E[X] = \int x \rho(x)\, dx$ é limite de $\sum x_i p_i$. Em física: $\int \rho\, dV$ vira integral tripla.

### Aplicações canônicas

- **Trabalho**: $W = \int_a^b F(x)\, dx$.
- **Massa**: $m = \int_\Omega \rho\, dV$.
- **Carga**: $Q = \int i(t)\, dt$.
- **Probabilidade**: $P(a \leq X \leq b) = \int_a^b f_X(x)\, dx$.
- **Centro de massa**: $\bar x = \int x \rho / \int \rho$.

### Discretização

Em métodos numéricos, **derivadas viram diferenças finitas** e **integrais viram somas finitas**. CFD, FEA, simulação — tudo é Riemann disfarçado.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

### Riemann vs Lebesgue

Riemann particiona o **domínio**. Lebesgue particiona o **contradomínio** — mais robusto a descontinuidades. Para $f$ Riemann-integrável, ambos coincidem; mas Lebesgue lida com $\mathbf{1}_\mathbb{Q}$, limites pontuais (TCD, TCM), e é a base da teoria moderna.

### Quadratura adaptativa

`scipy.integrate.quad` é Gauss-Kronrod adaptativo: avalia subintervalos com tolerância e refina onde o erro é alto. Padrão para integrais 1D suaves.

### Alta dimensão

Quadratura tensorial em $d$ dimensões: $n^d$ avaliações — **maldição da dimensionalidade**. Soluções:
- **Monte Carlo**: $O(1/\sqrt N)$, independente de $d$.
- **Quasi-Monte Carlo** (Sobol, Halton): $O((\log N)^d/N)$, melhor pra dimensões moderadas.
- **Sparse grids** (Smolyak): bom pra $d \sim 10$.

### Singularidades e infinitos

Tanh-sinh (double exponential) é o estado da arte para integrais com singularidades de extremo. Em finanças, integrais até $\infty$ (Black-Scholes) usam transformação de variável + Gauss-Hermite.

### TFC e EDOs

A integral definida é **operador linear** entre espaços de funções. EDOs, equações integrais (Volterra, Fredholm) e transformadas (Laplace, Fourier) são edifícios sobre integral definida.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Quando aparece integral definida na vida real?**

- **Engenharia mecânica**: trabalho, momento, centroide.
- **Engenharia elétrica**: carga, energia, RMS.
- **Estatística**: probabilidade, esperança, variância.
- **Economia**: excedente do consumidor, valor presente.
- **Computação**: rendering (path tracing é Monte Carlo de integrais).

### Workflow

1. **Modele** o problema como integral.
2. **Identifique** $f$, $a$, $b$.
3. **Resolva** simbolicamente (TFC + técnicas) ou numericamente.
4. **Confira ordem de grandeza** com Riemann grosseira mental.

### Exemplo: trabalho de uma mola

$F(x) = kx$ (Hooke). Trabalho de $0$ a $L$: $W = \int_0^L kx\, dx = kL^2/2$. Famoso.

### SciPy

```python
from scipy.integrate import quad
import numpy as np
val, err = quad(lambda x: np.exp(-x**2), 0, np.inf)
# val ≈ √π/2
```

Sempre **confira o erro** retornado.
