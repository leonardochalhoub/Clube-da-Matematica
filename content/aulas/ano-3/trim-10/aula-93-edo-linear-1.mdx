<Porta nivel="formal" titulo="Fator integrante">

### Forma canônica

$$y' + p(x)\, y = q(x)$$

(Linear: $y$ e $y'$ aparecem com expoente 1, sem produto entre si.)

### Método do fator integrante

1. Calcule $\mu(x) = e^{\int p(x)\, dx}$.
2. Multiplique a EDO: $\mu y' + \mu p y = \mu q$.
3. O lado esquerdo é $(\mu y)'$ (pela escolha de $\mu$).
4. Integre: $\mu y = \int \mu q\, dx + C$.
5. Isole: $y = \mu^{-1}\left(\int \mu q\, dx + C\right)$.

### Estrutura da solução

$$y = \underbrace{y_h}_{\text{homogênea}} + \underbrace{y_p}_{\text{particular}}$$

- $y_h = C e^{-\int p\, dx}$ — resposta natural (regime livre).
- $y_p$ — resposta forçada (depende de $q$).

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Imagina um balde furado. Cai água do céu (entrada $q$) e sai pelo furo (taxa $p$ vezes o que tem dentro). Quanta água tem no balde?

A equação linear de 1ª ordem é exatamente **isso**: água entrando, água saindo proporcionalmente. A gente acha **quanta água tem em cada momento**.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

A aula passada (separáveis) só funcionava quando dava pra **separar** $x$ e $y$. Mas e se a equação for $y' + 3y = x$? **Não dá** pra separar.

Aí entra um **truque mágico**: o **fator integrante**. Você multiplica a equação inteira por uma função especial $\mu(x)$ escolhida de tal jeito que o lado esquerdo vira **derivada de produto** $(\mu y)'$.

Para a equação $y' + 3y = x$, o fator é $\mu = e^{3x}$:

$$e^{3x} y' + 3 e^{3x} y = x e^{3x} \Rightarrow (e^{3x} y)' = x e^{3x}$$

Aí você integra dos dois lados e pronto.

A receita é: **fator $= e^{\int p\, dx}$**.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

EDO **linear de 1ª ordem**: $y' + p(x)y = q(x)$. **Linear** quer dizer que $y$ aparece elevado a 1, sem produtos $y \cdot y'$ ou $y^2$.

### Por que o fator integrante funciona

Eu quero que $\mu y' + \mu p y = (\mu y)' = \mu' y + \mu y'$. Comparando, preciso $\mu' = \mu p$, ou seja $\mu'/\mu = p$, donde $\mu = e^{\int p}$.

### Exemplo completo

$y' + 2xy = 4x$, $y(0) = 5$.

- $p = 2x$, $\mu = e^{x^2}$.
- $(e^{x^2} y)' = 4x e^{x^2}$.
- $e^{x^2} y = 2 e^{x^2} + C \Rightarrow y = 2 + C e^{-x^2}$.
- $y(0) = 5 \Rightarrow C = 3$. Resposta: $y = 2 + 3 e^{-x^2}$.

### Estrutura linear

A solução geral é **homogênea + particular**: $y = y_h + y_p$. A homogênea $y_h = C e^{-\int p}$ representa a "memória" do sistema; a particular $y_p$ representa a **resposta** ao forçamento $q(x)$.

### Aplicação canônica — RC com fonte

Circuito $RC$ com fonte $V_s$: $\dfrac{dV_C}{dt} + \dfrac{V_C}{RC} = \dfrac{V_s(t)}{RC}$. Solução: $V_C \to V_s$ exponencialmente (constante de tempo $\tau = RC$).

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

EDOs lineares de 1ª ordem governam **toda dinâmica de 1ª ordem em engenharia**: termodinâmica, controle, química, biologia.

### Resposta de sistemas de 1ª ordem

$$\tau \dot y + y = K \cdot u(t)$$

- $\tau$: constante de tempo.
- $K$: ganho estático.
- $u$: entrada.

**Resposta a degrau** ($u = 1$ para $t \geq 0$): $y(t) = K(1 - e^{-t/\tau})$.
**Resposta a impulso**: $y(t) = (K/\tau) e^{-t/\tau}$.
**Resposta a senoide** $u = \sin\omega t$: senoide com atraso $\arctan(\omega\tau)$ e atenuação $1/\sqrt{1 + \omega^2 \tau^2}$.

### Princípio da superposição

Como a EDO é **linear**, vale: se $y_1$ resolve $\mathcal{L}y = q_1$ e $y_2$ resolve $\mathcal{L}y = q_2$, então $\alpha y_1 + \beta y_2$ resolve $\mathcal{L}y = \alpha q_1 + \beta q_2$. **Superposição** = base de toda análise via Fourier/Laplace.

### Caixa de ferramentas

- **Variação de parâmetros**: derive $y_p$ assumindo $C = C(x)$ na homogênea. Equivalente ao fator integrante.
- **Coeficientes a determinar**: chuta forma de $y_p$ baseada em $q$. Útil quando $q$ é polinômio, exp ou seno/cosseno.
- **Transformada de Laplace**: converte EDO em álgebra. Crucial em controle.

### Exemplos de engenharia

| Sistema | EDO | $\tau$ |
|---|---|---|
| RC | $RC \dot V + V = V_s$ | $RC$ |
| RL | $L \dot i / R + i = V/R$ | $L/R$ |
| Reator CSTR | $V \dot c + Q c = Q c_{in}$ | $V/Q$ |
| Termômetro | $\tau \dot T + T = T_{amb}$ | $\tau$ térmico |

### Cuidado: linearidade não é universal

$y' = y(1-y)$ (logística) **não** é linear em $y$. Exigirá técnica não-linear.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

EDOs lineares vivem na intersecção de **álgebra de operadores**, **análise funcional** e **teoria de controle**.

### Visão operatorial

Define-se $\mathcal{L} = \dfrac{d}{dx} + p(x) \cdot$ — operador linear no espaço de funções diferenciáveis. A EDO vira $\mathcal{L} y = q$. **Inverso** de $\mathcal{L}$ é a **convolução com a função de Green**:

$$y(x) = \int_{x_0}^x G(x, s) q(s)\, ds + y_h(x)$$

onde $G(x, s) = \exp\left(-\int_s^x p(t) dt\right)$.

### Função de transferência (Laplace)

Para $p$ constante: $\mathcal{L}\{y' + py\} = (s + p) Y(s) = Q(s) + y(0)$. **Função de transferência** $H(s) = 1/(s+p)$ — **polo único** em $s = -p$.

- $p > 0$: estável (polo no semiplano esquerdo).
- $p < 0$: instável.
- $p = 0$: integrador puro.

### Conexão com filtros

Sistema $\tau \dot y + y = u$ com $u = e^{j\omega t}$:

$$|H(j\omega)| = \frac{1}{\sqrt{1 + \omega^2\tau^2}}, \quad \angle H = -\arctan(\omega\tau)$$

Filtro **passa-baixas** de 1ª ordem. Frequência de corte $\omega_c = 1/\tau$.

### Equação de Riccati

$y' = a(x)y^2 + b(x)y + c(x)$ — **não-linear**. Substituição $y = -u'/(au)$ a converte em **linear de 2ª ordem** $u'' + ...$. Conexão com mecânica quântica.

### Distribuições e Green

A função de Green de $\dfrac{d}{dx} + p$ (com $p$ constante) satisfaz:

$$G'(x, s) + p G(x, s) = \delta(x - s), \quad G = H(x-s) e^{-p(x-s)}$$

onde $H$ é Heaviside. **Causal**.

### Lyapunov

Para $\dot y + p y = 0$ com $p > 0$, $V(y) = y^2/2$ é função de Lyapunov: $\dot V = -p y^2 \leq 0$. Estabilidade exponencial garantida.

### Cosmologia: solução exata para escalar inflacionário

Em modelos slow-roll, a perturbação $\delta\phi$ obedece a uma EDO linear de 1ª ordem em derivativa de Hubble. Saul Perlmutter, Brian Schmidt e Adam Riess receberam o [Prêmio Nobel de Física de 2011](https://www.nobelprize.org/prizes/physics/2011/summary/) pela expansão acelerada — formalismo perturbativo usa exatamente essa estrutura.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

### Receita do fator integrante

1. **Escreva na forma canônica**: $y' + p(x) y = q(x)$. Se vier $a(x) y' + b(x) y = c(x)$, divide por $a$.
2. **Calcule** $\mu = e^{\int p\, dx}$ (não precisa $+C$ aqui).
3. **Multiplique a EDO por $\mu$**: $(\mu y)' = \mu q$.
4. **Integre os dois lados**: $\mu y = \int \mu q\, dx + C$.
5. **Isole $y$**: $y = \mu^{-1}\left(\int \mu q\, dx + C\right)$.
6. **Use $y(x_0) = y_0$** pra achar $C$.

### Erros comuns

- **Forma canônica errada**: não dividir pelo coeficiente de $y'$.
- **$\mu$ com sinal errado**: cuidado com $-p$ vs $+p$.
- **Adicionar $+C$ no $\mu$**: não precisa, qualquer fator integrante serve.
- **Esquecer integração por partes** em $\int \mu q$.

### Cheat sheet de respostas a entradas

| Entrada $u(t)$ | Resposta $y(t)$ ($\tau \dot y + y = K u$) |
|---|---|
| Degrau $1$ | $K(1 - e^{-t/\tau})$ |
| Rampa $t$ | $K t - K\tau (1 - e^{-t/\tau})$ |
| Impulso $\delta$ | $(K/\tau) e^{-t/\tau}$ |
| $\sin\omega t$ | $\dfrac{K}{\sqrt{1+\omega^2\tau^2}} \sin(\omega t - \arctan\omega\tau)$ |

### Workflow numérico

```python
from scipy.integrate import odeint
import numpy as np

p, q = lambda x: 2*x, lambda x: 4*x
def rhs(y, x): return q(x) - p(x)*y

x = np.linspace(0, 3, 100)
y = odeint(rhs, 5, x)
```

Confere com analítica $y = 2 + 3 e^{-x^2}$.
