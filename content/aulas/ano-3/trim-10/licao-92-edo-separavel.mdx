<Porta nivel="formal" titulo="Definição e método">

### Forma canônica

Uma EDO de 1ª ordem é **separável** se pode ser escrita como:

$$\frac{dy}{dx} = g(x) \cdot h(y)$$

### Método de solução

1. Reescreva: $\dfrac{dy}{h(y)} = g(x)\, dx$ (assumindo $h(y) \neq 0$).
2. Integre os dois lados: $\displaystyle \int \frac{dy}{h(y)} = \int g(x)\, dx + C$.
3. Resolva para $y$ se possível (solução explícita); caso contrário, deixe na forma implícita.
4. **Soluções singulares**: valores $y^*$ com $h(y^*) = 0$ são soluções constantes que precisam ser checadas separadamente.

### Existência e unicidade (Picard)

Se $g$ é contínua e $h$ é contínua e Lipschitz em $y$ numa vizinhança de $(x_0, y_0)$, então existe **solução única** local do PVI $y(x_0) = y_0$.

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Imagina que você tem dois saquinhos de bala. Um saquinho só tem balas vermelhas (chama $x$). O outro só tem balas azuis (chama $y$). Eles estão **misturados**.

A gente quer **separar**: bala vermelha no saquinho do $x$, bala azul no saquinho do $y$. Depois conta cada saquinho **separado**.

É isso que faz uma "EDO separável": separa as duas coisas, conta cada uma sozinha.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

Uma **equação diferencial** é tipo um quebra-cabeça: alguém te dá a **velocidade** com que algo cresce, e você precisa achar **a função** que descreve esse algo.

Por exemplo: "a velocidade de crescimento de uma planta é o dobro da altura dela". Em conta: $\dfrac{dy}{dx} = 2y$.

Para resolver, a gente usa um **truque**: junta tudo de $y$ de um lado e tudo de $x$ do outro. Aí integra cada lado separado:

$$\frac{dy}{y} = 2\, dx \Rightarrow \ln|y| = 2x + C \Rightarrow y = A e^{2x}$$

Pronto: a altura cresce **exponencialmente**.

Esse método chama **separação de variáveis**. Funciona quando dá pra "puxar" o $y$ pro lado do $dy$ e o $x$ pro lado do $dx$.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Uma **EDO** (equação diferencial ordinária) relaciona uma função $y(x)$ com sua derivada $y'$. **Separável** é o tipo mais simples:

$$\frac{dy}{dx} = g(x) h(y)$$

### Receita

1. **Separe**: $\dfrac{dy}{h(y)} = g(x)\, dx$.
2. **Integre**: $\displaystyle \int \frac{dy}{h(y)} = \int g(x)\, dx + C$.
3. **Resolva** para $y$ (se conseguir).

### Exemplo padrão — decaimento radioativo

$\dfrac{dN}{dt} = -kN$ (a taxa de decaimento é proporcional ao que sobrou).

Separa: $\dfrac{dN}{N} = -k\, dt$. Integra: $\ln N = -kt + C$. Logo $N(t) = N_0 e^{-kt}$.

### Exemplo crescimento populacional simples

$\dfrac{dP}{dt} = rP \Rightarrow P(t) = P_0 e^{rt}$ — Malthus.

### Cuidado com soluções singulares

Se $h(y_0) = 0$, então $y(x) = y_0$ constante também é solução — não esqueça de verificar.

### Forma diferencial

Algumas EDOs aparecem como $M(x)\, dx + N(y)\, dy = 0$. Já está separada: integre direto.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Separáveis são **a primeira classe** integrável fechada que você encontra. Toda outra técnica (linear de 1ª ordem, exata, Bernoulli) tenta **reduzir o problema à separabilidade**.

### Modelagem básica em engenharia

| Fenômeno | EDO | Solução |
|---|---|---|
| Decaimento radioativo | $\dot N = -kN$ | $N_0 e^{-kt}$ |
| Carga capacitor (sem fonte) | $\dot V = -V/RC$ | $V_0 e^{-t/RC}$ |
| Resfriamento (Newton) | $\dot T = -k(T - T_a)$ | $(T_0 - T_a) e^{-kt} + T_a$ |
| Mistura tanque (saída) | $\dot c = -Q c / V$ | $c_0 e^{-Qt/V}$ |
| Logística | $\dot P = rP(1 - P/K)$ | $K/(1 + Ae^{-rt})$ |

### Frações parciais e não-linearidades

Em $\dfrac{dy}{y(1-y)} = dx$, decompõe-se $\dfrac{1}{y(1-y)} = \dfrac{1}{y} + \dfrac{1}{1-y}$. Integra:

$$\ln\left|\frac{y}{1-y}\right| = x + C \Rightarrow y(x) = \frac{1}{1 + Ae^{-x}}$$

Sigmoide logística — usada em redes neurais (ativação).

### Análise qualitativa via campo de direções

Antes de resolver, **plote** $g(x)h(y)$ no plano $(x, y)$. Pontos $h(y^*) = 0$ são **equilíbrios**. Análise de estabilidade: se $h'(y^*) < 0$, equilíbrio estável.

### Não-existência global

$\dot y = y^2$, $y(0) = 1 \Rightarrow y(x) = 1/(1-x)$ — explode em $x = 1$. **Solução local existe sempre, global não**.

### Cross-domínio

- **Química**: cinética de 1ª ordem $[A]' = -k[A]$.
- **Farmacocinética**: eliminação de droga $C(t) = C_0 e^{-k_e t}$ — modelo monocompartimental.
- **Finanças**: juros contínuos $\dot S = rS \Rightarrow S = S_0 e^{rt}$.
- **Biologia**: lei de Verhulst (logística), modelo de Gompertz.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

Separabilidade é um caso especial de **EDOs exatas** e tem estrutura de **simetria de Lie**.

### Visão geométrica

A EDO $\dot y = g(x) h(y)$ define um **campo vetorial** $(1, g(x)h(y))$ no plano. Soluções são **curvas integrais**. Separabilidade significa que o campo é **fatoravelmente decomponível** em direções.

### Conexão com variedades produto

Em superfície produto $X \times Y$, a EDO separável corresponde a um **fluxo desacoplado** após mudança de coordenadas $u = \int dy/h(y)$, $v = \int g(x)dx$. Em $(u, v)$, a EDO vira $du/dv = 1$ — fluxo trivial.

### Existência global e blow-up

Pela teoria de Picard-Lindelöf, $\dot y = h(y)$ tem solução global se e somente se $\int_{y_0}^\infty dy/h(y) = \infty$ (critério de Osgood). Caso contrário, blow-up em tempo finito.

### Aplicação a fluxos

EDPs de transporte $\partial_t u + c(x) \partial_x u = 0$ se reduzem, ao longo de **características**, a EDO separável.

### Lei de Hubble (cosmologia)

$\dot a = H a$ — a EDO mais famosa da cosmologia. Solução $a(t) = a_0 e^{Ht}$ corresponde a expansão exponencial do universo (de Sitter). Saul Perlmutter, Brian Schmidt e Adam Riess receberam o [Prêmio Nobel de Física de 2011](https://www.nobelprize.org/prizes/physics/2011/summary/) pela descoberta da expansão acelerada via Supernovas Tipo Ia.

### Numérico vs analítico

Embora separáveis sejam analíticas, a integral $\int dy/h(y)$ pode ser não-elementar (e.g. $h(y) = e^{y^2}$). Nesse caso, integra numericamente — aula 98.

### Bibliotecas

```python
import sympy as sp
from sympy import Function, dsolve, Eq, symbols
x = symbols('x'); y = Function('y')
ode = Eq(y(x).diff(x), y(x) * (1 - y(x)))
sol = dsolve(ode, y(x))
```

<Porta nivel="pratica" titulo="Aplicação prática">

### Receita rápida

1. Veja se você consegue escrever $\dfrac{dy}{dx} = g(x) \cdot h(y)$ — produto de função só de $x$ por função só de $y$.
2. Separe: tudo com $y$ pra um lado (com $dy$), tudo com $x$ pro outro (com $dx$).
3. Integre cada lado.
4. Adicione $+C$ em **um** lado só.
5. Resolva pra $y$.
6. Use a condição inicial pra achar $C$.

### Erros comuns

- **Esquecer $+C$**: solução incompleta.
- **Esquecer soluções constantes**: se $h(y^*) = 0$, $y = y^*$ é solução perdida.
- **Trocar sinais ao integrar $1/y$**: $\int dy/y = \ln|y|$ (módulo!).
- **Não usar a condição inicial pra fixar $C$**: aí sua resposta tem 1 parâmetro livre.

### Workflow

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint

def f(y, x): return -2*y  # dy/dx = -2y

x = np.linspace(0, 5, 100)
sol = odeint(f, y0=3, t=x)
plt.plot(x, sol); plt.show()
```

Compare com a analítica $y = 3 e^{-2x}$.

### Quando NÃO é separável

Se aparecer $y' = x + y$ ou $y' = (x+y)^2$, **não dá** pra separar — é linear de 1ª ordem (próxima aula) ou requer substituição.
