<Porta nivel="formal" titulo="Malthus e Verhulst">

### Modelo de Malthus (1798)

$$\dot P = rP, \quad P(0) = P_0$$

Solução: $P(t) = P_0 e^{rt}$.

- $r > 0$: crescimento exponencial.
- $r < 0$: decaimento exponencial.
- $r = 0$: estacionário.

### Modelo logístico (Verhulst, 1838)

$$\dot P = rP\left(1 - \frac{P}{K}\right)$$

- $r$: taxa intrínseca de crescimento.
- $K$: **capacidade de suporte** (carrying capacity).

### Solução fechada (separável + frações parciais)

$$P(t) = \frac{K}{1 + \left(\dfrac{K - P_0}{P_0}\right) e^{-rt}}$$

### Equilíbrios

$P^* = 0$ (instável) e $P^* = K$ (estável). Curva tem **inflexão em $P = K/2$**.

</Porta>

Mas a ilha é pequena, né? Um dia **acaba o capim**. Aí eles param de crescer. **A ilha tem limite**.

A matemática consegue desenhar essa história inteira: começo rapidinho, depois enche, depois fica parado.

<Porta nivel="10" titulo="Pra criança de 10 anos">

**Como uma população cresce?**

**Modelo simples (Malthus)**: cada coelho gera mais coelhos. Quanto mais coelhos, mais nascem. A população **dobra** sempre depois do mesmo tempo. Isso chama **crescimento exponencial**.

Equação: $\dot P = r P$. Solução: $P(t) = P_0 e^{rt}$.

**Problema**: na vida real, não dá pra crescer pra sempre. **Falta comida, espaço**. Aí entra o **modelo logístico**:

$$\dot P = rP\left(1 - \frac{P}{K}\right)$$

Onde $K$ é o **máximo** que o ambiente aguenta. Quando $P$ está perto de $K$, o crescimento desacelera. Quando $P = K$, **para**.

A curva fica em forma de **S** — começa rápido, no meio diminui, no final estabiliza.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

### Por que Malthus falha

Thomas Malthus (1798) previu **catástrofe** porque assumiu crescimento exponencial. Não funcionou — recursos crescem também, e há **limites** ecológicos.

### Verhulst conserta

Pierre-François Verhulst (1838) corrigiu: a taxa **diminui** conforme $P$ se aproxima de $K$:

$$\dot P = rP\left(1 - \frac{P}{K}\right) = rP - \frac{r}{K} P^2$$

O termo $-rP^2/K$ representa **competição intra-específica**.

### Solução

A EDO é separável (com frações parciais):

$$\frac{1}{P(1 - P/K)} = \frac{1}{P} + \frac{1/K}{1 - P/K}$$

Integrando: $\ln P - \ln(K - P) = rt + C$, donde:

$$P(t) = \frac{K}{1 + A e^{-rt}}, \quad A = \frac{K - P_0}{P_0}$$

### Análise qualitativa

- $P < K$: crescente.
- $P > K$: decrescente.
- $P = 0, P = K$: equilíbrios. $K$ é **atrator** (estável).

### Aplicações

- Populações reais (paramécios, leveduras): logística ajusta bem em laboratório.
- Difusão de inovações (Bass model).
- Epidemiologia básica (SIR simplificado).
- Adoção de tecnologia.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Modelos populacionais são a **introdução padrão a sistemas dinâmicos não-lineares**.

### Sistemas dinâmicos 1D

$\dot x = f(x)$. Análise:

1. **Equilíbrios**: $f(x^*) = 0$.
2. **Linearização**: $\dot \xi \approx f'(x^*) \xi$ — exponencial local.
3. **Estabilidade**: $f'(x^*) < 0$ → estável; $> 0$ → instável.

Para logístico, $f(P) = rP(1-P/K)$:
- $f(0) = 0$, $f'(0) = r > 0$ → instável.
- $f(K) = 0$, $f'(K) = -r < 0$ → estável.

### Generalizações

| Modelo | EDO | Interpretação |
|---|---|---|
| Gompertz | $\dot P = -rP \ln(P/K)$ | Tumores |
| Allee | $\dot P = rP(P/A - 1)(1 - P/K)$ | Extinção se $P < A$ |
| Lotka-Volterra | sistema 2D presa-predador | Oscilações |
| SIR | sistema 3D suscetível-infectado-recuperado | Epidemias |

### Predador-presa (Lotka-Volterra)

$$\dot x = ax - bxy, \quad \dot y = -cy + dxy$$

Conservação: $H(x, y) = dx + by - c\ln x - a\ln y$. **Órbitas fechadas** — oscilações periódicas.

### Modelos de doenças

SIR básico:

$$\dot S = -\beta SI, \quad \dot I = \beta SI - \gamma I, \quad \dot R = \gamma I$$

$R_0 = \beta S(0)/\gamma$ — número de reprodução básico. Se $R_0 > 1$, epidemia.

### Aplicação em engenharia

- **Difusão de inovações** (Bass): vendas de smartphones, EVs.
- **Epidemiologia computacional**: modelos COVID-19.
- **Pesca sustentável**: capacidade de captura $h(P)$ vs estabilidade.
- **Biotecnologia**: crescimento de leveduras em fermentador.

### Ferramentas

```python
from scipy.integrate import odeint
def logistic(P, t, r=0.5, K=1000):
    return r*P*(1 - P/K)
import numpy as np
t = np.linspace(0, 30, 200)
sol = odeint(logistic, 10, t)
```

<Porta nivel="40" titulo="Pra profissional sênior">

Logística é o **arquétipo de bifurcação supercrítica** e ponte para teoria de sistemas dinâmicos.

### Bifurcação transcrítica

Considere $\dot P = (r - r_c) P - \alpha P^2$. Para $r$ crescendo através de $r_c$:
- Equilíbrio $P^* = 0$ troca de estável para instável.
- Surge novo ramo $P^* = (r - r_c)/\alpha > 0$, estável.

### Mapa logístico (tempo discreto)

$x_{n+1} = r x_n (1 - x_n)$ — **caos** para $r > 3{,}57$. Cascata de duplicações de período. O trabalho de **Giorgio Parisi sobre sistemas complexos** (incluindo dinâmicas caóticas em sistemas desordenados) levou ao [Prêmio Nobel de Física de 2021](https://www.nobelprize.org/prizes/physics/2021/summary/).

### Equação de Fisher-Kolmogorov (logística + difusão)

$$\partial_t P = D \partial_{xx} P + r P(1 - P/K)$$

Frente viajante com velocidade $c = 2\sqrt{rD}$. Aplicações: invasão biológica, propagação de genes, cancer growth.

### Modelos com atraso (DDE)

$\dot P(t) = r P(t) [1 - P(t-\tau)/K]$ — **equação de Hutchinson**. Para $r\tau > \pi/2$, oscilações via bifurcação de Hopf.

### Métodos estocásticos

Adicionar ruído: $dP = rP(1-P/K)dt + \sigma P\, dW_t$ — SDE de Itô. Estado quase-estacionário, distribuição.

### Lotka-Volterra estocástico

Estados quase-estacionários, fenômeno de extinção rara, cálculo via Feynman-Kac.

### Calibração estatística

Ajuste pelo método dos mínimos quadrados não-linear; usar bootstrap pra IC nos parâmetros $r, K$. Identificabilidade: $r$ e $K$ são identificáveis se há observações em fase de crescimento **e** plateau.

<Porta nivel="pratica" titulo="Aplicação prática">

### Quando usar Malthus vs logístico

- **Curtos prazos, recursos abundantes**: Malthus serve.
- **Médio/longo prazo, ambiente limitado**: logístico.
- **Tumor** com angiogênese saturando: Gompertz.
- **Espécies em risco com Allee**: cuidado, pode haver extinção mesmo com $P > 0$.

### Como estimar $r$ e $K$ a partir de dados

1. Pegue série temporal $P(t_i)$.
2. **Fase exponencial inicial**: $\ln P \approx \ln P_0 + rt$ → regressão linear dá $r$.
3. **Plateau**: $K \approx \lim P(t)$.
4. Refine com fit não-linear (scipy.optimize.curve_fit).

### Avisos

- Se observa só fase inicial, **não dá** pra estimar $K$.
- Erro padrão em $r$ é alto pra dados ruidosos — bootstrap.
- **Não ajuste se mecanismo não bate**: tumor não é população.

### Workflow

```python
from scipy.optimize import curve_fit
import numpy as np
def logistic(t, r, K, P0):
    return K / (1 + ((K-P0)/P0) * np.exp(-r*t))

popt, pcov = curve_fit(logistic, t_obs, P_obs, p0=[0.5, 1000, 10])
r_est, K_est, P0_est = popt
```

### Sanity checks

- Plotar $P$ vs $t$ + curva ajustada.
- Resíduos brancos? Se não, modelo não captura tudo.
- $K_{est} > \max P_{obs}$? Sim, deveria.
