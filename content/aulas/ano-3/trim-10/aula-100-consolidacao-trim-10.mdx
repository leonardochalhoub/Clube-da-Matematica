<Porta nivel="formal" titulo="Mapa de técnicas">

### EDOs de 1ª ordem

| Tipo | Forma | Técnica |
|---|---|---|
| Separável | $y' = g(x)h(y)$ | Integrar $\int dy/h = \int g\, dx$ |
| Linear | $y' + p(x)y = q(x)$ | Fator integrante $\mu = e^{\int p}$ |
| Bernoulli | $y' + py = qy^n$ | Substituição $u = y^{1-n}$ |
| Exata | $M\, dx + N\, dy = 0$, $M_y = N_x$ | $F$ tal que $\nabla F = (M, N)$ |
| Autonôma | $y' = f(y)$ | Equilíbrios + análise de fase |

### EDOs de 2ª ordem lineares com coef. constantes

$ay'' + by' + cy = q(x)$. Característica $a\lambda^2 + b\lambda + c = 0$:

- $\Delta > 0$: $y_h = C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}$.
- $\Delta = 0$: $y_h = (C_1 + C_2 x) e^{\lambda x}$.
- $\Delta < 0$: $y_h = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$.

$y_p$ por chute (coef. determinar) ou variação de parâmetros.

### Aplicações canônicas

| Domínio | EDO | Técnica |
|---|---|---|
| Decaimento radioativo | $\dot N = -kN$ | Separável |
| Capacitor RC | $\dot V + V/(RC) = V_s/(RC)$ | Linear 1ª ordem |
| População logística | $\dot P = rP(1-P/K)$ | Separável c/ frações parciais |
| Massa-mola | $m\ddot x + c\dot x + kx = F$ | Linear 2ª ordem |
| RLC | $L\ddot Q + R\dot Q + Q/C = V$ | Linear 2ª ordem |
| Resfriamento | $\dot T = -k(T - T_a)$ | Separável |

### Numérico

Quando não há fórmula fechada: Euler ($O(h)$), RK4 ($O(h^4)$), implícito (stiff), simplético (Hamiltoniano).

</Porta>

A gente aprendeu a lidar com **equações que mudam no tempo**. Tipo: o quanto **vai esfriando**, o quanto **vai crescendo**, o quanto **balança**.

Aprendemos **truques diferentes** pra cada tipo. E quando o truque não funciona, a gente pede ajuda do **computador**, que faz passinho a passinho.

Agora a gente junta tudo!

<Porta nivel="10" titulo="Pra criança de 10 anos">

Pelo trimestre 10, vimos **8 técnicas** pra resolver EDOs. Cada problema do mundo real cai em uma delas:

1. **Separável** — variável $x$ e $y$ se separam.
2. **Linear 1ª ordem** — fator integrante.
3. **Populacional** — Malthus exponencial ou logístico.
4. **2ª ordem com coef. constantes** — equação característica.
5. **Vibrações** — massa-mola, regimes.
6. **RLC** — análogo ao massa-mola, mas elétrico.
7. **Numérico (Euler/RK4)** — quando não tem fórmula.
8. **Resfriamento** — Newton e calor.

**Como escolher?**

- Olhe a EDO: tem $y''$? → 2ª ordem.
- Só $y'$ e $y$: tenta separável primeiro. Se não dá, linear.
- Modela algo da vida real (vibração? RLC?): use a EDO específica daquela área.
- Não tem fórmula? Computador resolve!

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

### Fluxograma de decisão

```
EDO chegou:
├─ É de 1ª ordem? (só y' presente)
│  ├─ Separável? Integra direto.
│  ├─ Linear y'+py=q? Fator integrante.
│  └─ Outra? Tenta substituição (Bernoulli, exata).
└─ É de 2ª ordem?
   ├─ Coeficientes constantes?
   │  ├─ Homogênea? Característica.
   │  └─ Forçada? Coef. determinar ou variação.
   └─ Coeficientes variáveis? Cauchy-Euler ou série.

Se nada disso funciona → numérico (Euler/RK4).
```

### Exemplos integradores

**Problema 1**: Carro freia com força $F = -mv^2/L$ (drag aerodinâmico).
$$m\dot v = -mv^2/L \Rightarrow \dot v = -v^2/L$$
**Separável**: $-1/v = -t/L + C$, donde $v = L/(t + L/v_0)$. Não-exponencial!

**Problema 2**: Pêndulo amortecido pequeno.
$$\ddot\theta + 2\zeta\omega_0 \dot\theta + \omega_0^2\theta = 0$$
**2ª ordem coef. constantes**. Subamortecido se $\zeta < 1$.

**Problema 3**: Tanque de salmoura entrada variável.
$$\dot c + (Q/V) c = (Q/V) c_{in}(t)$$
**Linear 1ª ordem** com $c_{in}$ função do tempo.

### Verificação cruzada

Sempre **substitua a solução** na EDO pra confirmar. E **plote**: a forma deve fazer sentido fisicamente (decai? oscila? satura?).

### Erros típicos

- Aplicar **fator integrante** numa EDO **não-linear** ($y' = y^2$). Não funciona.
- Esquecer **soluções constantes** em separáveis: $h(y_0) = 0 \Rightarrow y \equiv y_0$ é solução.
- Confundir $\omega_0$ e $\omega_d$ no amortecido.
- Usar Euler com $h$ grande em problema rápido (instável).

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

EDOs são **a linguagem da engenharia**. Toda área tem suas EDOs canônicas:

### Mecânica

- **Cinemática**: $\dot v = a$, $\dot x = v$.
- **Dinâmica**: $m\ddot x = F$.
- **Vibrações**: $m\ddot x + c\dot x + kx = F$.
- **Estabilidade**: linearização + análise de autovalores.

### Eletricidade

- **RC, RL, LC, RLC**: análogos diretos do mecânico.
- **Linhas de transmissão**: equação do telegrafista.
- **Máquinas elétricas**: equações de Park.
- **Sistemas de potência**: dinâmica de geradores síncronos.

### Química

- **Cinética**: $\dot c = r(c)$, várias ordens.
- **CSTR, batch reactor**: balanço de massa.
- **Combustão**: chama estacionária.

### Biomédica

- **Farmacocinética**: $\dot C = -k_e C + r_a(t)$.
- **Compartimental**: sistemas multi-compartimento.
- **Epidemiologia**: SIR, SEIR.

### Finanças

- **Black-Scholes** (EDP, mas reduz a EDOs em coordenadas certas).
- **Vasicek, CIR**: SDEs lineares (taxa de juros).

### Workflow profissional

1. **Modele**: tire a EDO do balanço de massa/energia/momento.
2. **Adimensionalize**: identifique parâmetros sem dimensão (Re, Bi, $\zeta$, ...).
3. **Linearize** se possível (pequenos desvios).
4. **Resolva analiticamente** se cabe; senão numérico.
5. **Valide** com experimentos ou simulações alta-fidelidade.
6. **Otimize** parâmetros (controle, design).

### Software

- **MATLAB/Simulink**: padrão indústria controle, sistemas.
- **Python (scipy, control)**: open-source equivalente.
- **Modelica/Dymola**: orientado a equações para sistemas multi-físicos.
- **COMSOL, ANSYS**: EDPs (FEM/FVM).
- **CasADi, JuMP**: otimização baseada em modelos EDO.

### Perspectiva

Galileu, Newton, Euler, Lagrange, Laplace **inventaram** EDOs para descrever física. Hoje o ferramental se estende a controle ótimo, ML (gradient flows como EDOs!), neural ODEs, e análise inversa.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

### Visão moderna: sistemas dinâmicos

Toda EDO autonôma define um **fluxo** em variedade. Conceitos relevantes:

- **Equilíbrios**: $f(x^*) = 0$. Hartman-Grobman: localmente equivalente a linearização se $f'(x^*)$ não tem autovalor zero ou puramente imaginário.
- **Variedades estável/instável**: invariantes sob fluxo.
- **Bifurcações**: como retrato de fase muda com parâmetros.
- **Conjuntos atratores**: $\omega$-limites.
- **Caos**: sensibilidade a condições iniciais (Lyapunov).

### Métodos modernos

- **Geometric integration**: simpléticos para Hamiltoniano, Lie groups para mecânica de corpos rígidos.
- **Reduced-order models**: POD, DMD pra reduzir dimensão.
- **SINDy** (Sparse Identification): descobre EDO a partir de dados.
- **Neural ODEs**: redes neurais como EDOs contínuas (Chen et al., 2018).
- **Physics-informed NN (PINN)**: NN treinada respeitando EDOs.

### Controle

- **PID** controla planta de 2ª ordem.
- **State-feedback** $\mathbf u = -\mathbf K \mathbf x$ aloca polos.
- **Observador** estima estados.
- **MPC** (model predictive): otimização constrained baseada em EDO.

### EDP $\to$ EDO

Discretização espacial reduz EDPs a sistemas grandes de EDOs. Padrão em CFD, FEA estrutural, Maxwell.

### Estocástico

EDOs estocásticas (SDEs) modelam ruído inerente. Itô, Stratonovich. Aplicações: finanças, biologia, controle robusto.

### Cosmologia e relatividade

Equações de Friedmann (cosmologia) são EDOs. James Peebles teorizou estrutura cósmica e recebeu o [Prêmio Nobel de Física de 2019](https://www.nobelprize.org/prizes/physics/2019/summary/) por contribuições teóricas em cosmologia física.

### Inferência

Bayesian inverse problems: dado $y(t)$ medido, infere parâmetros $\theta$ tais que solução de $\dot x = f(x; \theta)$ encaixa. PyMC, Stan, Turing.jl.

### Otimização

Adjunto de EDO: gradiente eficiente em alta dimensão. Aplicações: design ótimo, ML.

### Onde vão as fronteiras

- **Federated EDO**: dinâmicas em rede.
- **Quantum ODEs**: simulação clássica de sistemas quânticos.
- **Multiphysics**: acoplamento estrutural-térmico-fluido.
- **Real-time control**: controle de partícula em LHC.

</Porta>

### Workshop: 8 problemas para você dominar

Resolva cada um e classifique no fluxograma de decisão. Esses são **exemplos canônicos** da indústria.

**1. Cápsula espacial reentrada**
$$m\dot v = -mg - C_d \rho A v^2/2$$
Modele $g(h)$, $\rho(h)$, integre numericamente.

**2. Reator químico CSTR**
$$V \dot c + Q c = Q c_{in}, \quad V \dot T = Q\rho c_p (T_{in} - T) + (-\Delta H_r) V r$$
Sistema acoplado. Equilíbrios e estabilidade térmica.

**3. Suspensão de carro 1/4-veículo**
$$m\ddot x + c\dot x + kx = c\dot u + ku$$
$u(t)$ = perfil da estrada. Resposta a impulso (lombada).

**4. Crescimento de tumor com tratamento**
$$\dot V = -r V \ln(V/K) - f(t) V$$
Gompertz com terapia $f(t)$ pulsátil.

**5. Filtro RLC para áudio**
Passa-banda 100Hz-10kHz, ganho 0dB em centro, atenuação 40dB/dec fora. Calcule $L, C, R$.

**6. Termômetro IoT**
$\tau = 30$s. Calibrar com input degrau. Algoritmo de correção de leitura.

**7. Pêndulo invertido**
$$(I + ml^2)\ddot\theta = mgl\sin\theta - bl\cos\theta \cdot u$$
Linearize, projete controle.

**8. Vacinação SIR**
$$\dot S = -\beta SI - vS, \quad \dot I = \beta SI - \gamma I, \quad \dot R = \gamma I + vS$$
$v$ = taxa de vacinação. Limiar pra extinção.

### Boas práticas

- **Sempre adimensionalize** antes de resolver.
- **Plote campo de direções** antes de computar.
- **Compare numérico vs analítico** em casos simples.
- **Valide com experimento** sempre que possível.
- **Documente** parâmetros, condições iniciais, tolerâncias.

### Cheat sheet

```
Separável: dy/h(y) = g(x)dx, integra
Linear:    μ=exp(∫p), (μy)'=μq, integra
Logística: P = K/(1+Ae^(-rt))
Newton:    T = T_amb + ΔT_0 e^(-kt)
Característica: aλ² + bλ + c = 0
Vibração: ω_0 = √(k/m), ζ = c/(2√(mk))
RLC:       ω_0 = 1/√(LC), ζ = (R/2)√(C/L)
Euler:     y_(n+1) = y_n + h f(t_n,y_n)
RK4:       4 etapas
```
