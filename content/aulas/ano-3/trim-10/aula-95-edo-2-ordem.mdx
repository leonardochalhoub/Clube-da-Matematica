<Porta nivel="formal" titulo="Solução pela equação característica">

### Problema

$$a y'' + b y' + c y = 0, \quad a, b, c \in \mathbb{R}, a \neq 0$$

### Ansatz

Substitua $y = e^{\lambda x}$:

$$(a\lambda^2 + b\lambda + c) e^{\lambda x} = 0 \Rightarrow a\lambda^2 + b\lambda + c = 0$$

### Três casos (discriminante $\Delta = b^2 - 4ac$)

**Caso 1 — $\Delta > 0$**: raízes reais distintas $\lambda_1, \lambda_2$.
$$y = C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}$$

**Caso 2 — $\Delta = 0$**: raiz dupla $\lambda$.
$$y = (C_1 + C_2 x) e^{\lambda x}$$

**Caso 3 — $\Delta < 0$**: complexas $\lambda = \alpha \pm i\beta$.
$$y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$$

### Não-homogênea

$ay'' + by' + cy = q(x)$. Solução: $y = y_h + y_p$. $y_p$ via **coeficientes a determinar** ou **variação de parâmetros**.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

Equação de **2ª ordem**: aparece $y''$ (segunda derivada — aceleração). Em vez de uma força que depende só da posição (1ª ordem), agora a EDO descreve sistemas que **oscilam, vibram, balançam**.

A receita é mágica: **chuta** $y = e^{\lambda x}$. Substitui na equação, e ela vira **equação do 2º grau** em $\lambda$:

$$a\lambda^2 + b\lambda + c = 0$$

Resolve com Bhaskara! Aí, dependendo das raízes:

- **Duas reais distintas**: $y = C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}$ (sistema "morre rápido", sem oscilar).
- **Uma raiz dupla**: $y = (C_1 + C_2 x) e^{\lambda x}$.
- **Complexas**: $y = e^{\alpha x} (C_1 \cos\beta x + C_2 \sin\beta x)$ — **oscilação amortecida**.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

EDOs de 2ª ordem aparecem **em todo lugar** que tem aceleração: massa-mola, pêndulo, RLC, vigas.

### Por que o ansatz $e^{\lambda x}$?

Porque exponenciais são **autofunções** do operador $D = d/dx$: $De^{\lambda x} = \lambda e^{\lambda x}$. Isso transforma EDO **linear** em equação **algébrica**.

### Os três regimes físicos

Tome a equação canônica $\ddot x + 2\zeta\omega_0 \dot x + \omega_0^2 x = 0$ (oscilador amortecido):

- $\zeta > 1$: **superamortecido** — duas raízes reais negativas, retorna ao equilíbrio sem oscilar.
- $\zeta = 1$: **criticamente amortecido** — raiz dupla, retorno mais rápido sem oscilar.
- $0 < \zeta < 1$: **subamortecido** — oscila com decaimento exponencial.
- $\zeta = 0$: **harmônico puro** — oscila eternamente.

### Conjunto fundamental

Duas soluções $y_1, y_2$ formam **conjunto fundamental** se são **linearmente independentes**: $W(y_1, y_2) = y_1 y_2' - y_1' y_2 \neq 0$ (Wronskiano).

### Exemplo

$y'' - 5y' + 6y = 0$. Característica: $\lambda^2 - 5\lambda + 6 = 0$, raízes $\lambda = 2, 3$. Solução: $y = C_1 e^{2x} + C_2 e^{3x}$.

$y'' + 4y = 0$. Característica: $\lambda^2 + 4 = 0$, $\lambda = \pm 2i$. Solução: $y = C_1 \cos 2x + C_2 \sin 2x$.

### Não-homogênea: $y_p$ por chute

$y'' + y = e^{2x}$. Chute $y_p = A e^{2x}$. Substitui: $4A + A = 1 \Rightarrow A = 1/5$. Solução geral: $y = C_1\cos x + C_2\sin x + \frac{1}{5}e^{2x}$.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Esta é a EDO **mais importante de toda a engenharia**. Domina vibrações mecânicas, circuitos RLC, linhas de transmissão, sismologia, controle clássico.

### Forma normalizada

$$\ddot x + 2\zeta\omega_0 \dot x + \omega_0^2 x = F(t)$$

- $\omega_0$: frequência natural.
- $\zeta$: fator de amortecimento.
- $F(t)$: forçamento externo.

### Equação característica e raízes

$$\lambda^2 + 2\zeta\omega_0 \lambda + \omega_0^2 = 0 \Rightarrow \lambda = \omega_0(-\zeta \pm \sqrt{\zeta^2 - 1})$$

### Tabela completa de regimes

| $\zeta$ | Tipo | Comportamento |
|---|---|---|
| $0$ | Não-amortecido | Oscilação pura $A\cos(\omega_0 t + \phi)$ |
| $0 < \zeta < 1$ | Subamortecido | $A e^{-\zeta\omega_0 t} \cos(\omega_d t + \phi)$, $\omega_d = \omega_0\sqrt{1-\zeta^2}$ |
| $\zeta = 1$ | Crítico | $(A + Bt) e^{-\omega_0 t}$ |
| $\zeta > 1$ | Superamortecido | $A e^{\lambda_1 t} + B e^{\lambda_2 t}$, ambos $< 0$ |

### Resposta forçada e ressonância

$\ddot x + \omega_0^2 x = F_0 \cos\omega t$. Particular:

$$x_p = \frac{F_0}{\omega_0^2 - \omega^2} \cos\omega t$$

**Diverge** em $\omega = \omega_0$ — **ressonância**! Com amortecimento, o pico fica finito mas afiado.

### Aplicações canônicas

- **Massa-mola-amortecedor**: $m\ddot x + c\dot x + kx = F(t)$. $\omega_0 = \sqrt{k/m}$, $\zeta = c/(2\sqrt{mk})$.
- **Pêndulo linearizado**: $\ddot\theta + (g/L)\theta = 0$.
- **RLC série**: $L\ddot q + R\dot q + q/C = V(t)$. $\omega_0 = 1/\sqrt{LC}$.
- **Galloping** em linhas de transmissão (vento).
- **Tacoma Narrows** (1940): ressonância aerodinâmica derrubou a ponte.

### Sistema de 1ª ordem equivalente

$$\dot{\mathbf x} = A \mathbf x, \quad \mathbf x = (x, \dot x)^T, \quad A = \begin{pmatrix} 0 & 1 \\ -\omega_0^2 & -2\zeta\omega_0\end{pmatrix}$$

Autovalores de $A$ = raízes da característica. **Estabilidade** = parte real $< 0$.

### Função de transferência

$H(s) = 1/(s^2 + 2\zeta\omega_0 s + \omega_0^2)$. **Polos** = raízes características. Diagrama de Bode.

### Cross-domínio

- Na física: equação de Schrödinger separável reduz a EDO 2ª ordem (tempo separado).
- Em controle: PID controla planta de 2ª ordem (típico).
- Em sismologia: edifícios alto-amortecidos para evitar ressonância em terremotos.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

EDOs lineares 2ª ordem são instâncias da **teoria de Sturm-Liouville** e **operadores auto-adjuntos**.

### Operador linear

$\mathcal{L} = a D^2 + b D + c$. **Espectro discreto** com autofunções $e^{\lambda_k x}$. Composição:

$$\mathcal{L} = a(D - \lambda_1)(D - \lambda_2)$$

— fatoração via raízes da característica.

### Forma de Sturm-Liouville

EDOs de 2ª ordem gerais: $-(p y')' + q y = \lambda w y$. Espectro real, autofunções ortogonais com peso $w$.

### Função de Green

Para problema com condições de contorno $y(0) = y(L) = 0$:

$$y(x) = \int_0^L G(x, s) f(s) \, ds$$

onde $G$ é a função de Green do operador.

### Teoria espectral

Polos de $H(s) = 1/p(s)$ no semiplano esquerdo $\Leftrightarrow$ EDO **estável**. Routh-Hurwitz dá critério algébrico.

### Análise modal

Em sistema MIMO $\mathbf{M}\ddot{\mathbf q} + \mathbf{C}\dot{\mathbf q} + \mathbf{K}\mathbf q = \mathbf 0$, autoproblema generalizado $(\mathbf K - \omega^2 \mathbf M)\mathbf v = 0$ dá **modos** e frequências naturais. Crucial em estruturas e vibrações.

### Conexão com mecânica quântica

Schrödinger estacionária 1D: $-\dfrac{\hbar^2}{2m} \psi'' + V(x)\psi = E\psi$. EDO linear de 2ª ordem com auto-valores $E$. Métodos de WKB, etc. Roy Glauber recebeu o [Prêmio Nobel de Física de 2005](https://www.nobelprize.org/prizes/physics/2005/summary/) pelo desenvolvimento da óptica quântica coerente — extensão dessas EDOs ao formalismo de operadores.

### Resolvente

$(\mathcal{L} - \lambda I)^{-1}$ é o **resolvente**. Continuação meromorfa em $\lambda$, polos = autovalores. Base da teoria de scattering.

### Numérico

```python
from scipy.signal import lti, step, impulse
sys = lti([1], [1, 0.4, 4])  # 1/(s^2 + 0.4s + 4)
t, y = step(sys)
```

### Identificação experimental

Resposta a impulso $\to$ FFT $\to$ pólos via algoritmos como Prony, ERA. Padrão em teste modal experimental.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

### Receita completa

1. **Escreva** $a y'' + b y' + c y = q(x)$.
2. **Característica**: $a\lambda^2 + b\lambda + c = 0$.
3. **Calcule $\Delta$**: identifique caso (real distintas, real dupla, complexas).
4. **Escreva $y_h$** conforme tabela.
5. Se $q \neq 0$: **chute** $y_p$ pela forma de $q$ (ver tabela).
6. Solução geral: $y = y_h + y_p$.
7. Use 2 condições iniciais $y(x_0), y'(x_0)$ pra fixar $C_1, C_2$.

### Tabela de $y_p$ (coeficientes a determinar)

| $q(x)$ | Chute para $y_p$ |
|---|---|
| Polinômio grau $n$ | Polinômio grau $n$ |
| $e^{\alpha x}$ | $A e^{\alpha x}$ |
| $\sin\beta x$ ou $\cos\beta x$ | $A\sin\beta x + B\cos\beta x$ |
| $e^{\alpha x}\sin\beta x$ | $e^{\alpha x}(A\sin\beta x + B\cos\beta x)$ |

**Cuidado**: se chute coincide com $y_h$, multiplique por $x$ (ressonância).

### Erros comuns

- **Esquecer fator $x$** na raiz dupla.
- **Trocar $\sin$ e $\cos$** no caso complexo.
- **Não verificar independência linear** das soluções.
- **Ignorar ressonância** quando $q$ é a frequência natural.

### Workflow numérico

```python
from scipy.integrate import solve_ivp
def rhs(t, y):
    return [y[1], -0.4*y[1] - 4*y[0]]  # x'' + 0.4x' + 4x = 0
sol = solve_ivp(rhs, [0, 20], [1, 0], dense_output=True, max_step=0.01)
```

### Diagnóstico rápido

Olhou para $\zeta = b/(2\sqrt{ac})$:
- $\zeta = 0$: oscila para sempre.
- $0 < \zeta < 1$: oscila e decai.
- $\zeta = 1$: decai sem oscilar (mais rápido).
- $\zeta > 1$: decai mais lento.

</Porta>
