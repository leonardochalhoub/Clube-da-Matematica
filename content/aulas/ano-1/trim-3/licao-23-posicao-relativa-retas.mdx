### Paralelismo e perpendicularidade

Dadas retas $r: y = m_r x + n_r$ e $s: y = m_s x + n_s$:

| Posição | Critério (forma reduzida) | Critério (forma geral) |
|---|---|---|
| Paralelas distintas | $m_r = m_s$, $n_r \neq n_s$ | $A_r/A_s = B_r/B_s \neq C_r/C_s$ |
| Coincidentes | $m_r = m_s$, $n_r = n_s$ | $A_r/A_s = B_r/B_s = C_r/C_s$ |
| Concorrentes | $m_r \neq m_s$ | $A_r B_s - A_s B_r \neq 0$ |
| Perpendiculares | $m_r \cdot m_s = -1$ | $A_r A_s + B_r B_s = 0$ |

Casos com retas verticais: $x = a$ é vertical, $y = b$ é horizontal. Verticais entre si são paralelas; vertical com horizontal são perpendiculares.

### Ângulo entre retas

$$\tan\theta = \left|\frac{m_r - m_s}{1 + m_r m_s}\right|$$

(Assumindo nenhuma vertical.)

### Distância ponto-reta

Ponto $P_0 = (x_0, y_0)$ e reta $Ax + By + C = 0$:

$$d(P_0, r) = \frac{|A x_0 + B y_0 + C|}{\sqrt{A^2 + B^2}}$$

### Distância entre retas paralelas

Para $r: Ax + By + C_1 = 0$ e $s: Ax + By + C_2 = 0$:
$$d(r, s) = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$

### Mediatriz

Conjunto de pontos equidistantes de $A$ e $B$. É reta perpendicular a $\overline{AB}$ passando pelo ponto médio.

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Imagina **dois trilhos de trem**. Eles vão lado a lado, **nunca se encontram**. Isso é **paralelo**.

Agora imagine duas **réguas formando um sinal de mais**, $+$. Elas se cruzam fazendo **canto reto**. Isso é **perpendicular**.

Quando duas retas não são paralelas e nem perpendiculares, elas só **se cruzam num ponto** torto.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

Duas retas no plano podem fazer 4 coisas:

1. **Paralelas**: nunca se encontram (mesma inclinação).
2. **Coincidentes**: são a mesma reta.
3. **Concorrentes**: cruzam num ponto.
4. **Perpendiculares**: cruzam fazendo 90°.

**Como saber?** Olha a inclinação $m$:
- $m_1 = m_2$: paralelas (ou coincidentes se $n$ também igual).
- $m_1 \cdot m_2 = -1$: perpendiculares.
- Senão: cruzam num ponto qualquer.

**Exemplo concreto**: $y = 2x + 1$ e $y = 2x - 5$ são paralelas (inclinação 2 nas duas). $y = 2x$ e $y = -x/2$ são perpendiculares ($2 \cdot (-1/2) = -1$).

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Duas retas no plano podem:

1. **Ser paralelas** (mesma inclinação, nunca se cruzam).
2. **Ser perpendiculares** (formam ângulo de 90°).
3. **Cruzar em um ponto** (concorrentes).
4. **Ser a mesma reta** (coincidentes).

**Critério de paralelismo**: $m_r = m_s$ — mesma inclinação.

**Critério de perpendicularidade**: $m_r \cdot m_s = -1$. Por que? Pense em $m_1 = \tan\alpha_1$. Se $\alpha_2 = \alpha_1 + 90°$, então $\tan\alpha_2 = -\cot\alpha_1 = -1/m_1$. Logo $m_1 m_2 = -1$.

**Exemplo numérico completo**: 
- $r: y = 2x + 1$ e $s: y = -x/2 + 5$. Produto $2 \cdot (-1/2) = -1$. **Perpendiculares**.
- Encontre o ponto de cruzamento: $2x + 1 = -x/2 + 5 \Rightarrow 5x/2 = 4 \Rightarrow x = 8/5$. Substitui: $y = 16/5 + 1 = 21/5$. Cruzam em $(8/5, 21/5)$.

**Distância de ponto a reta** é fundamental — usado em quase todo problema geométrico subsequente. A fórmula $|Ax_0 + By_0 + C|/\sqrt{A^2 + B^2}$ vem de projetar o ponto na direção normal à reta.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Em prática, **resolução de sistemas lineares 2x2** é interseção de retas. Cramer e escalonamento começam aqui.

### Sistema 2x2 e interpretação geométrica

$$\begin{cases} a_1 x + b_1 y = c_1 \\ a_2 x + b_2 y = c_2 \end{cases}$$

| Caso | Geometria | Determinante |
|---|---|---|
| Solução única | retas concorrentes | $\det \neq 0$ |
| Infinitas soluções | retas coincidentes | $\det = 0$ + consistente |
| Sem solução | retas paralelas distintas | $\det = 0$ + inconsistente |

### Aplicações imediatas

- **Linear regression**: a reta de melhor ajuste é perpendicular à direção de variância máxima do erro.
- **CG / SVG**: clipping de polígonos (Sutherland-Hodgman) testa lados de retas via sinal de $Ax + By + C$.
- **Robótica**: planejamento de trajetória evita obstáculos representados por meios-planos.
- **GIS**: análise de proximidade — distância de cada ponto à rede viária.

### Demonstração da fórmula $m_1 m_2 = -1$

Sejam $\vec u_1 = (1, m_1)$ e $\vec u_2 = (1, m_2)$ vetores diretores. Perpendicularidade $\iff \vec u_1 \cdot \vec u_2 = 0 \iff 1 + m_1 m_2 = 0 \iff m_1 m_2 = -1$.

### Mediatriz como lugar geométrico

Pontos $(x, y)$ equidistantes de $A = (x_A, y_A)$ e $B = (x_B, y_B)$:
$$(x - x_A)^2 + (y - y_A)^2 = (x - x_B)^2 + (y - y_B)^2$$

Expandindo, os quadráticos cancelam, sobra reta $\Rightarrow$ é perpendicular a $\overline{AB}$ pelo ponto médio.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

A noção de "ortogonalidade" entre retas generaliza-se para **ortogonalidade de subespaços** em qualquer espaço com produto interno.

### Visão álgebra linear

Reta = subespaço afim 1D. Direção é vetor $\vec v$. Duas retas $r_1, r_2$ paralelas $\iff \vec v_1 \parallel \vec v_2$. Perpendiculares $\iff \langle \vec v_1, \vec v_2 \rangle = 0$.

Em $\mathbb{R}^n$, **complemento ortogonal** de subespaço $V$ é $V^\perp = \{w : \langle v, w \rangle = 0 \forall v \in V\}$. $V \oplus V^\perp = \mathbb{R}^n$. Base de Gram-Schmidt, QR, e SVD.

### Aplicações em ML / quant

- **Regressão linear**: vetor de resíduos é ortogonal ao espaço-coluna de $X$ (princípio de ortogonalidade). Formalmente, $\hat\beta = (X^TX)^{-1} X^Ty$ projeta $y$ ortogonalmente.
- **PCA**: componentes principais são ortogonais — eigenvalores da matriz de covariância.
- **Quant finance**: portfolio rebalancing usa projeção ortogonal de retornos esperados no espaço de constraints.
- **Quantum**: estados ortogonais $\langle \psi | \phi \rangle = 0$ correspondem a "exclusão mútua" de medições.

### Distância via projeção

A fórmula $d = |Ax_0 + By_0 + C|/\sqrt{A^2+B^2}$ é a magnitude da projeção do vetor "ponto-reta" sobre a normal $(A, B)/\sqrt{A^2+B^2}$. Generaliza pra distância ponto-hiperplano em $\mathbb{R}^n$.

### Numérico

Cuidado com retas quase-paralelas em sistemas lineares — número de condição $\kappa(A) = \cot(\theta/2)$ explode quando $\theta \to 0$. Pivoting parcial em Gauss + reescalamento mitiga.

### SymPy

```python
import sympy as sp
from sympy import Line, Point
r = Line(Point(0, 0), Point(1, 2))
s = Line(Point(0, 5), Point(1, 7))
print(r.is_parallel(s))      # True
print(r.is_perpendicular(s)) # False
print(r.angle_between(s))    # 0
print(s.distance(Point(0, 0)))
```

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita: classificar duas retas em 3 passos.**

1. **Compare inclinações** $m_1, m_2$:
   - Iguais? Vão pra paralelas/coincidentes.
   - Produto $-1$? Perpendiculares.
   - Senão? Concorrentes — encontre interseção.
2. **Se paralelas**: compare $n_1, n_2$. Iguais $\Rightarrow$ coincidentes; senão distintas.
3. **Para encontrar ângulo**: use $\tan\theta = |(m_1 - m_2)/(1 + m_1 m_2)|$.

### Workflow completo

```python
import sympy as sp
from sympy import Line, Point, pi
r = Line(Point(0, 1), Point(1, 3))   # y = 2x + 1
s = Line(Point(0, 5), Point(2, 4))   # y = -x/2 + 5
print(r.is_perpendicular(s))                   # True
print(r.intersection(s))                       # ponto de cruzamento
print(sp.deg(r.angle_between(s)))              # 90
print(s.distance(Point(0, 0)))                 # distância origem-reta
```

### Quando preocupar?

- **Retas verticais**: $m$ não definido — use forma geral.
- **Quase-paralelas**: pequena perturbação muda muito a interseção. Em prática, calcule $\det$ e descarte se $|\det| < \epsilon$.
- **Distância vs sinal**: a fórmula $|Ax_0 + By_0 + C|$ tem módulo. Sem módulo, dá **lado** da reta — útil em algoritmos de clipping.
