<Porta nivel="formal" titulo="Equações e propriedades">

<Definicao titulo="Circunferência">
Conjunto dos pontos $(x, y) \in \mathbb{R}^2$ que estão a distância $r > 0$ de um centro $C = (a, b)$.
</Definicao>

### Forma reduzida

$$(x - a)^2 + (y - b)^2 = r^2$$

### Forma geral

Expandindo: $x^2 + y^2 + Dx + Ey + F = 0$, com $D = -2a$, $E = -2b$, $F = a^2 + b^2 - r^2$.

Centro recuperado: $(a, b) = (-D/2, -E/2)$. Raio: $r = \sqrt{D^2/4 + E^2/4 - F}$ (válido se positivo; senão, não é círculo real).

### Reta e circunferência

| Posição | Critério | Pontos comuns |
|---|---|---|
| Exterior | $d(C, r) > r$ | 0 |
| Tangente | $d(C, r) = r$ | 1 |
| Secante | $d(C, r) < r$ | 2 |

### Dois círculos

| Posição | Critério |
|---|---|
| Disjuntos externamente | $d(C_1, C_2) > r_1 + r_2$ |
| Tangentes externamente | $d = r_1 + r_2$ |
| Secantes | $\lvert r_1 - r_2 \rvert \lt d \lt r_1 + r_2$ |
| Tangentes internamente | $d = \lvert r_1 - r_2 \rvert$ |
| Um dentro do outro | $d \lt \lvert r_1 - r_2 \rvert$ |

### Propriedade tangente-radius

Tangente em $P$ é perpendicular ao raio $\overline{CP}$.

### Equação da tangente

Tangente em $(x_0, y_0)$ ao círculo $(x-a)^2 + (y-b)^2 = r^2$:
$$(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$$

<Porta nivel="10" titulo="Pra criança de 10 anos">

Circunferência é o **conjunto dos pontos a uma mesma distância** do centro. Tipo o desenho que o compasso faz: a distância da pontinha à pontinha do lápis é o **raio**.

A equação $(x - a)^2 + (y - b)^2 = r^2$ vem da fórmula da distância (Aula 21):
- $(a, b)$ é o centro.
- $r$ é o raio.
- Pra cada ponto $(x, y)$ da curva, a distância até o centro é $r$. Elevando ao quadrado, fica como acima.

**Exemplo concreto**: roda de bicicleta com centro no eixo. Cada ponto da borracha está à mesma distância do eixo.

**Aplicações reais**:
- Wi-Fi: alcance é círculo.
- Sirene de polícia: pessoas que ouvem estão dentro do círculo.
- Pista de atletismo: 400m de circunferência.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

A circunferência é o conjunto dos pontos **igualmente distantes** de um centro. Distância $= r$ (raio).

Aplicando a fórmula da distância da Aula 21:
$$\sqrt{(x - a)^2 + (y - b)^2} = r$$

Elevando ao quadrado: $(x - a)^2 + (y - b)^2 = r^2$. **Pronto, é só isso**.

Quando o centro está na origem ($a = b = 0$): $x^2 + y^2 = r^2$. **Círculo unitário** ($r = 1$): $x^2 + y^2 = 1$ — o que você usou no Aula 12 do círculo trigonométrico!

**Forma geral $\to$ reduzida**: completa quadrado. Exemplo: $x^2 + y^2 - 4x + 6y - 12 = 0$.
- Em $x$: $x^2 - 4x = (x-2)^2 - 4$.
- Em $y$: $y^2 + 6y = (y+3)^2 - 9$.
- Substitui: $(x-2)^2 - 4 + (y+3)^2 - 9 - 12 = 0 \Rightarrow (x-2)^2 + (y+3)^2 = 25$.
- Centro $(2, -3)$, raio 5.

**Cuidado**: nem toda equação $x^2 + y^2 + Dx + Ey + F = 0$ é círculo. Se $D^2/4 + E^2/4 - F < 0$, não há ponto real. Se $= 0$, é o ponto único (círculo "degenerado").

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

A circunferência é o exemplo mais simples de **lugar geométrico** (locus). Generalização imediata: **esfera** em $\mathbb{R}^3$ — $(x-a)^2 + (y-b)^2 + (z-c)^2 = r^2$. Em $\mathbb{R}^n$: $\|x - c\|^2 = r^2$ é a $(n-1)$-esfera.

### Aplicações cross-domínio

- **Telecomunicações**: alcance de antena = círculo. Cobertura múltipla = união/interseção de círculos. Diagramas de Voronoi (cada ponto associado à antena mais próxima).
- **GPS / Trilateração**: cada satélite define uma esfera; sua localização é interseção de 4 esferas. 3 dão posição + tempo, 4ª remove ambiguidade.
- **Robótica**: alcance de braço com junta esférica é uma esfera.
- **Computer graphics**: bounding sphere para colisão e culling.
- **Engenharia química**: equilíbrio de Gibbs em sistema reativo binário forma círculos no diagrama de fase.

### Demonstração: tangente é perpendicular ao raio

Seja $P_0 = (x_0, y_0)$ ponto do círculo, centro $C = (a, b)$. Vetor $\vec{CP_0} = (x_0 - a, y_0 - b)$. A tangente tem como vetor normal este mesmo $\vec{CP_0}$. Logo equação:
$$(x_0 - a)(x - x_0) + (y_0 - b)(y - y_0) = 0$$

Expandindo e usando que $(x_0 - a)^2 + (y_0 - b)^2 = r^2$:
$$(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$$

### Potência de ponto

Para ponto $P_0$ e círculo de centro $C$ raio $r$:
$$\text{pot}(P_0) = |P_0 C|^2 - r^2$$

- Negativa: $P_0$ dentro.
- Zero: $P_0$ na fronteira.
- Positiva: $P_0$ fora; $\sqrt{\text{pot}}$ é o comprimento da tangente desde $P_0$.

### Trilateração 2D

3 círculos passando por mesmo ponto: 2 equações lineares (subtraindo pares de equações de círculos cancela termos quadráticos). Sistema linear 2×2 resolve.

<Porta nivel="40" titulo="Pra profissional sênior">

Circunferência é a **curva fundamental** de geometria diferencial — curvatura constante $\kappa = 1/r$.

### Visão diferencial

Parametrização: $\gamma(t) = (a + r\cos t, b + r\sin t)$. Vetor tangente $\gamma'(t) = (-r\sin t, r\cos t)$ tem módulo $r$ — não unitário, mas constante. Vetor curvatura $\gamma''(t)/|\gamma'|^2$ aponta para o centro com módulo $1/r$.

### Generalização: círculo osculador

Para curva qualquer $\gamma(t)$ no plano, em cada ponto há um círculo osculador (best-fit circle): tangente igual à curva e curvatura igual. Raio $\rho = 1/\kappa$. Crucial em design de estradas (transição entre retas e curvas via clotoides).

### Aplicações em ML / quant

- **Regularização $\ell_2$ (Ridge)**: $\|\beta\|_2 \leq c$ é bola euclidiana — cabeça do círculo no espaço de coeficientes. Solução é onde a curva de nível MSE toca a fronteira do círculo.
- **Quant**: precificação de opções com Heston ou SABR usa esferas em espaço de parâmetros pra regiões de calibração estável.
- **Otimização Trust Region**: cada passo restrito a uma bola euclidiana.
- **Manifold learning**: t-SNE, UMAP preservam vizinhanças locais (= círculos pequenos no espaço original).

### Generalização para esferas n-dimensionais

Volume da n-esfera: $V_n(r) = \frac{\pi^{n/2}}{\Gamma(n/2 + 1)} r^n$. Concentração de medida: para $n$ grande, quase todo o volume está numa "casca" fina perto da superfície — fenômeno chave em ML em alta dimensão.

### Curiosidade histórica

Descartes (1643) descreve o "círculo dos quatro círculos" (Descartes circle theorem): para 4 círculos mutuamente tangentes com curvaturas $k_1, k_2, k_3, k_4$, vale $(k_1 + k_2 + k_3 + k_4)^2 = 2(k_1^2 + k_2^2 + k_3^2 + k_4^2)$. Apollonian gaskets fractal.

### SymPy

```python
import sympy as sp
from sympy import Circle, Point, Line
C = Circle(Point(2, 3), 4)
print(C.equation())                    # (x-2)**2 + (y-3)**2 - 16
L = Line(Point(0, 0), Point(1, 1))
print(C.intersection(L))               # pontos de interseção
print(C.tangent_lines(Point(10, 3)))   # 2 retas tangentes
```

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita: extrair info de uma equação de círculo.**

1. **Forma reduzida** $(x-a)^2 + (y-b)^2 = r^2$: leia direto centro $(a, b)$ e raio $r$.
2. **Forma geral** $x^2 + y^2 + Dx + Ey + F = 0$: complete quadrado (ou use $a = -D/2$, $b = -E/2$, $r^2 = a^2 + b^2 - F$).
3. **Verificação**: substitua um ponto e teste — não pode dar valores absurdos.

### Workflow de problema típico

1. **Posição relativa reta-círculo**: calcule $d(\text{centro}, \text{reta}) = |Aa + Bb + C|/\sqrt{A^2+B^2}$, compare com $r$.
2. **Tangente em ponto do círculo**: use $(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$.
3. **Tangente externa de ponto $P$**: comprimento $= \sqrt{|PC|^2 - r^2}$.
4. **Círculo por 3 pontos**: resolva sistema 3×3 com $x^2 + y^2 + Dx + Ey + F = 0$.

### Wolfram / SymPy

```python
import sympy as sp
from sympy import Circle, Point
C = Circle(Point(0, 0), 5)
print(C.area)         # 25π
print(C.circumference) # 10π
P = Point(3, 4)
print(P in C)         # True (na borda)
```

### Quando preocupar?

- **Equação geral pode não ser círculo**: $D^2/4 + E^2/4 - F < 0$ → ponto vazio. $= 0$ → ponto único.
- **Numérico**: completar quadrado em coeficientes próximos de zero é instável; calcule centro/raio direto via fórmulas.
- **Esfera vs círculo**: em 3D, tome cuidado — círculo em 3D é interseção de plano e esfera (curva 1D); precisa de duas equações.
