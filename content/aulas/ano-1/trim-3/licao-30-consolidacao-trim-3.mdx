**Revisão** do que vimos:
1. **Plano cartesiano**: pontos com coordenadas $(x, y)$.
2. **Reta**: equação $y = mx + n$.
3. **Círculo**: $(x - a)^2 + (y - b)^2 = r^2$.
4. **Cônicas**: elipse, parábola, hipérbole.
5. **Vetores**: setas no plano que somam.
6. **Produto escalar**: combina dois vetores e dá um número.
7. **Sistemas lineares**: várias equações ao mesmo tempo.

**Nesta aula**, problemas que usam tudo junto. Por exemplo: encontrar onde uma reta cruza um círculo (Aula 22 + 24). Ou achar a tensão em cabos sustentando uma carga (vetores + sistema linear).

**Estratégia para resolver problemas integrados**:
1. **Identifique os tópicos** envolvidos.
2. **Esboce o problema** geometricamente.
3. **Escreva equações** de cada parte.
4. **Resolva o sistema** resultante.
5. **Interprete** e verifique a resposta.

**Conexões importantes**:
- Reta $\to$ paramétrica $\to$ vetor diretor (Aulas 22, 26).
- Círculo $\to$ caso particular de elipse (24, 25).

### Síntese do método cartesiano

Descartes (1637) ensinou: **traduza qualquer problema geométrico para álgebra**, resolva, depois traduza de volta. Esta aula é o exercício final dessa tradução em 2D antes de ir para 3D no Trim 4.

### Próximos trimestres usam tudo isso

- **Trim 4 (Geometria Espacial)**: distância 3D, planos, esferas — extensão direta.
- **Trim 5 (Funções)**: estuda transformações de gráficos ($f(x-a) + b$) — translações vetoriais.
- **Trim 6 (Cálculo)**: derivada como inclinação, integral como área. Tudo geometria analítica acelerada.
- **Trim 12 (Álgebra Linear formal)**: vetores e sistemas em $\mathbb{R}^n$ rigorosamente.

<Porta nivel="40" titulo="Pra profissional sênior">

O Trim 3 cobre o que em literatura inglesa é "Coordinate Geometry / Analytic Geometry" — mas o aluno provavelmente não percebe que está aprendendo **álgebra linear elementar disfarçada**.

### Conexões formais

| Tópico do Trim 3 | Conceito de Álgebra Linear |
|---|---|
| Plano cartesiano | $\mathbb{R}^2$ como espaço vetorial |
| Equação da reta | hiperplano em $\mathbb{R}^2$ |
| Sistema 2x2 | mapa linear $\mathbb{R}^2 \to \mathbb{R}^2$ |
| Determinante | volume orientado, área de paralelogramo |
| Produto escalar | forma bilinear / produto interno |
| Vetor diretor | base do subespaço |
| Coincidente / paralelas | dependência linear / posto |
| Cônica | forma quadrática $\mathbf x^T A \mathbf x = 1$ |

### Aplicações em ML / quant

- **Atenção em Transformers**: $\text{softmax}(QK^T/\sqrt d)V$. $QK^T$ é matriz de produtos escalares.
- **Markowitz** (1952, [Nobel 1990](https://www.nobelprize.org/prizes/economic-sciences/1990/)): minimiza $\mathbf w^T \Sigma \mathbf w$ — produto escalar generalizado. A frontier é hipérbole.
- **Black-Scholes** (1973, [Nobel 1997](https://www.nobelprize.org/prizes/economic-sciences/1997/)): equação de calor parabólica em ($\ln S$, $t$). Simulação Monte Carlo trabalha em $\mathbb{R}^N$.
- **Word embeddings**: similaridade $= \cos\theta$ entre vetores em $\mathbb{R}^{300}$.

### Roadmap pós-Trim 3

Próximos passos após dominar Trim 3:

1. **Trim 4**: extensão a $\mathbb{R}^3$ (planos, esferas, vetores 3D, produto vetorial).
2. **Trim 6**: cálculo diferencial — derivada como inclinação local de tangente.
3. **Trim 12**: álgebra linear formal — espaços vetoriais abstratos, autovalores, SVD.

### Curiosidade

Praticamente todos os **prêmios Nobel em Economia** desde 1969 ([nobelprize.org](https://www.nobelprize.org/prizes/lists/all-prizes-in-economic-sciences/)) usam matemática vetorial e otimização linear/quadrática como base. Markowitz (1990), Sharpe (1990), Black-Scholes/Merton (1997), Fama (2013), Hansen (2013) — todos partem de $\mathbb{R}^n$.

### Workflow Wolfram / SymPy

```python
import sympy as sp
from sympy.geometry import Point, Line, Circle, Triangle

# Exemplo de problema típico: tangente de ponto externo a círculo
C = Circle(Point(0, 0), 5)
P = Point(7, 0)
print(C.tangent_lines(P))      # 2 retas tangentes
print(P.distance(C.center)**2 - C.radius**2)  # comprimento^2 da tangente

# Sistema de geometria
T = Triangle(Point(0,0), Point(6,0), Point(0,8))
print(T.incenter, T.incircle.radius)  # centro e raio do círculo inscrito
```

### Quando preocupar?

- **Misturar contextos**: cuidado para não aplicar fórmula de elipse a círculo (são equivalentes em $a = b$, mas notação muda).
- **Unidades**: trabalho em joules, força em newtons, distância em metros — sempre coerentes.
- **Sinal de cosseno**: produto escalar negativo significa ângulo obtuso, não erro.
- **Verificar ordem de grandeza**: tensão em cabo de 1000 kg sustentado deve ser ≥ 5000 N (gravidade) — qualquer coisa muito menor é erro.
