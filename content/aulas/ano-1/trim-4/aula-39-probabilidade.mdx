<Porta nivel="formal" titulo="Axiomas e fórmulas">

### Espaço amostral

$\Omega$: conjunto de todos os resultados possíveis do experimento. **Evento**: subconjunto de $\Omega$.

### Axiomas de Kolmogorov (1933)

1. $P(A) \geq 0$ para todo evento $A$.
2. $P(\Omega) = 1$.
3. **$\sigma$-aditividade**: se $A_1, A_2, \ldots$ disjuntos: $P(\bigcup A_i) = \sum P(A_i)$.

### Probabilidade clássica (espaço equiprovável)

$$P(A) = \frac{|A|}{|\Omega|}$$

### Propriedades imediatas

- $P(\emptyset) = 0$.
- $P(A^c) = 1 - P(A)$.
- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$ (inclusão-exclusão).
- $A \subseteq B \Rightarrow P(A) \leq P(B)$.
- $P(A) \in [0, 1]$.

### Probabilidade condicional

$$P(A | B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0$$

### Independência

$A$ e $B$ são **independentes** se $P(A \cap B) = P(A) P(B)$ (equivalente: $P(A|B) = P(A)$).

### Teorema de Bayes

$$P(A | B) = \frac{P(B | A) P(A)}{P(B)}$$

### Probabilidade total

Se $\{A_i\}$ partição de $\Omega$:
$$P(B) = \sum_i P(B | A_i) P(A_i)$$

### Variável aleatória discreta

$X: \Omega \to \mathbb R$. Distribuição: $P(X = x_i) = p_i$ com $\sum p_i = 1$.

| Distribuição | Fórmula | Aparição |
|---|---|---|
| Bernoulli | $P(X=1) = p$ | 1 ensaio |
| Binomial | $\binom{n}{k}p^k(1-p)^{n-k}$ | $n$ ensaios |
| Geométrica | $(1-p)^{k-1}p$ | tempo de espera |
| Poisson | $e^{-\lambda}\lambda^k/k!$ | eventos raros |

### Esperança e variância

- $E[X] = \sum x_i p_i$
- $\text{Var}(X) = E[(X - E[X])^2] = E[X^2] - E[X]^2$

<Porta nivel="5" titulo="Pra criança de 5 anos">

Probabilidade é "**chance** de algo acontecer". Vai de 0 (impossível) até 1 (com certeza).

Lança a moeda: chance de cara é meio (1/2), porque tem 2 resultados e 1 deles é cara.

Lança o dado: chance de tirar 6 é um sexto (1/6), porque o dado tem 6 lados e só 1 é o 6.

Quanto **mais possibilidades**, **mais difícil** acertar.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

**Probabilidade** = chance de algo acontecer, número entre 0 e 1.

**Regra clássica** (quando todos os resultados são igualmente prováveis):
$$P(\text{evento}) = \frac{\text{casos favoráveis}}{\text{casos totais}}$$

Exemplos:
- Dado: $P(\text{par}) = 3/6 = 1/2$.
- Moeda × 2: $P(\text{2 caras}) = 1/4$ (HH, HT, TH, TT — só HH).
- Cartas: $P(\text{rei}) = 4/52 = 1/13$.

**Probabilidade condicional**: "se já sei que B aconteceu, qual a chance de A?". Escreve-se $P(A|B)$.

**Independência**: o resultado de A não afeta B. Exemplo: lançar 2 dados — o segundo dado não liga pro primeiro.

**Bayes**: tem uma fórmula esperta pra "inverter" probabilidades. É como descobrir a causa a partir do efeito.

**Onde aparece**: sorteio, jogos, previsão do tempo, diagnóstico médico, filtro de spam, IA.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Probabilidade é "número que mede a chance" de algo acontecer. Vai de 0 (impossível) até 1 (certeza).

**Caso clássico**: experimento com $|\Omega|$ resultados igualmente prováveis (lançar moeda, lançar dado). $P(A)$ = "casos favoráveis dividido por casos possíveis".

Exemplo: lançar dado, $P(\text{par}) = 3/6 = 1/2$. Lançar moeda 2 vezes, $P(\text{2 caras}) = 1/4$.

**Probabilidade condicional**: "dado que B ocorreu, qual a probabilidade de A?". $P(A|B) = P(A \cap B)/P(B)$.

**Eventos independentes**: $P(A \cap B) = P(A) P(B)$. Lançar 2 dados — resultado do 2.º não afeta o 1.º.

**Bayes**: jeito de "inverter" condicional. Se você sabe $P(B|A)$, pode calcular $P(A|B)$. Usado em diagnóstico médico, filtro de spam, ML.

**Inclusão-exclusão**: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$. Quando os eventos não são disjuntos, sobra a interseção contada duas vezes — subtrai.

**Distribuição binomial**: número de sucessos em $n$ ensaios independentes:
$$P(X = k) = \binom{n}{k}p^k(1-p)^{n-k}$$

Exemplo: lançar moeda 10 vezes, qual a chance de exatamente 5 caras? $\binom{10}{5}(1/2)^{10} = 252/1024 \approx 0{,}246$.

**Combinatória + probabilidade andam juntas**: contagem dá $|A|$ e $|\Omega|$, divisão dá $P(A)$.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Probabilidade é a **base** da ciência de dados, ML, controle estocástico, finanças quantitativas.

### Variável aleatória

$X: \Omega \to \mathbb{R}$. Valor numérico associado a cada resultado. **Discreta** ou **contínua** (próximas séries).

### Esperança

$E[X] = \sum_i x_i P(X = x_i)$. Linear: $E[aX + bY] = aE[X] + bE[Y]$ — sempre, mesmo sem independência.

### Variância

$\text{Var}(X) = E[(X - \mu)^2] = E[X^2] - \mu^2$. **Não é linear**: $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)$.

Se $X, Y$ independentes: $\text{Cov} = 0$, daí $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y)$.

### Distribuição binomial

$X \sim \text{Bin}(n, p)$: número de "sucessos" em $n$ ensaios independentes com prob. $p$:
$$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}$$

Aparece em controle de qualidade, ensaios clínicos, A/B testing.

$E[X] = np$, $\text{Var}(X) = np(1-p)$.

### Bayes em prática — diagnóstico médico

Doença rara $D$ com $P(D) = 0{,}001$. Teste: $P(+|D) = 0{,}99$ (sensibilidade), $P(-|D^c) = 0{,}95$ (especificidade). Probabilidade de doença dado teste positivo:
$$P(D | +) = \frac{P(+|D) P(D)}{P(+)} = \frac{0{,}99 \cdot 0{,}001}{0{,}99 \cdot 0{,}001 + 0{,}05 \cdot 0{,}999} \approx 0{,}019$$

Apenas **1,9%** — base rate fallacy explicada por Bayes.

### Cross-domínio

- **Engenharia**: confiabilidade — sistemas em série/paralelo, MTBF.
- **Sinais**: detecção em ruído via Neyman-Pearson.
- **ML**: classificadores Bayesianos, redes neurais probabilísticas.
- **Quant**: pricing risk-neutral, value-at-risk.
- **Estatística**: testes de hipótese, p-valores, intervalos de confiança.

### Lei dos grandes números

$\bar X_n = \frac{1}{n}\sum X_i \xrightarrow{n\to\infty} E[X]$ (convergência em probabilidade ou quase certa). Justifica frequências relativas como estimativas de probabilidade.

### Teorema central do limite (preview)

$\frac{\bar X - \mu}{\sigma/\sqrt n} \xrightarrow{d} N(0, 1)$. Soma de muitas variáveis independentes tende à Gaussiana — explica ubiquidade da normal.

<Porta nivel="40" titulo="Pra profissional sênior">

Probabilidade moderna é construída sobre **espaços de medida** $(\Omega, \mathcal F, P)$ no formalismo de Kolmogorov (1933). Toda variável aleatória é função mensurável.

### Wolfram tour

```mathematica
Probability[X >= 5, X \[Distributed] BinomialDistribution[10, 0.5]]
Mean[BinomialDistribution[10, 0.5]]
PDF[NormalDistribution[0, 1], x]
RandomVariate[BinomialDistribution[100, 0.3], 10]
```

### Kolmogorov e medida

$\sigma$-álgebra $\mathcal F$ — eventos mensuráveis. Necessário porque em $\Omega$ contínuo (e.g., $[0,1]$) nem todo subconjunto é mensurável (Vitali). Em discreto, $\mathcal F = 2^\Omega$ funciona.

### Inferência Bayesiana

Posterior $\propto$ prior $\times$ likelihood. MCMC (Metropolis-Hastings, HMC, NUTS) amostra de posteriores arbitrárias. Stan, PyMC3, Pyro, NumPyro. Ferramentas de produção em ciência de dados moderna.

### Quant — measure change

Risk-neutral pricing: $V_0 = e^{-rT} E^Q[\text{payoff}]$. Mudança de Radon-Nikodym de $P$ (real-world) para $Q$ (risk-neutral). Girsanov para processos contínuos.

### Concentration inequalities

- **Markov**: $P(X \geq a) \leq E[X]/a$.
- **Chebyshev**: $P(|X-\mu| \geq k\sigma) \leq 1/k^2$.
- **Hoeffding**: $P(|\bar X_n - \mu| \geq t) \leq 2e^{-2nt^2/(b-a)^2}$ — base de aprendizado PAC.
- **Chernoff**: cauda exponencial — comunicação, RNG, hashing.

### ML e probabilidade

- **Cross-entropy loss**: $-\sum p_i \log q_i$.
- **KL divergence**: $D_{KL}(p\|q) = \sum p_i \log(p_i/q_i)$ — base de variational methods.
- **Score-based generative models** (diffusion): aprende $\nabla \log p$.
- **Neural networks** podem ser entendidos como aproximação universal de distribuições condicionais $P(Y|X)$.

### Pesquisa

- **Probabilidade livre** (Voiculescu): álgebra não-comutativa para matrizes aleatórias grandes.
- **Optimal transport** (Villani, Cuturi): distância entre distribuições — Wasserstein.
- **Stein's method**: cota erros em CLT e variações.
- **Markov chain mixing**: cutoff phenomena, Diaconis.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Workflow para problemas de probabilidade.**

1. **Defina $\Omega$ e o evento $A$ claramente**. 80% dos erros vêm de definição imprecisa.
2. **Equiprovável?** Use combinatória ($|A|/|\Omega|$).
3. **Não-equiprovável?** Use propriedades + axiomas.
4. **Condicional ou Bayes?** Identifique o que é dado e o que se quer.
5. **Independência?** Multiplica. Senão, condicione.

### Python receita

```python
import math
from math import comb
from scipy.stats import binom, norm, poisson

# Probabilidade clássica
def prob_classica(favoraveis, totais):
    return favoraveis / totais

print(prob_classica(comb(4, 2), comb(52, 5)))   # P(2 reis em 5 cartas) — adaptado

# Distribuição binomial
n, p = 10, 0.5
print(binom.pmf(5, n, p))     # P(X=5)
print(binom.cdf(3, n, p))     # P(X<=3)
print(binom.mean(n, p))       # n*p

# Bayes manual
def bayes(prior_A, likelihood_B_given_A, likelihood_B_given_notA):
    prior_notA = 1 - prior_A
    p_B = likelihood_B_given_A * prior_A + likelihood_B_given_notA * prior_notA
    return likelihood_B_given_A * prior_A / p_B

# Doença rara
print(bayes(0.001, 0.99, 0.05))   # P(D|+) ~ 0.019

# Simulação Monte Carlo
import numpy as np
np.random.seed(42)
samples = np.random.choice([0, 1], size=10000, p=[0.5, 0.5])
print((samples.sum()) / 10000)    # ~ 0.5
```

### Pegadinhas

- **Distinguir $P(A|B)$ de $P(B|A)$**. Bayes é exatamente sobre essa inversão.
- **Independência != mutualmente exclusivos**. Independentes podem co-ocorrer; mutuamente exclusivos não podem.
- **Casos não equiprováveis**: lançar 2 dados → 36 resultados equiprováveis, MAS somas 2 a 12 NÃO são equiprováveis.
- **Base rate fallacy**: ignora a base — a probabilidade a priori importa muito.
