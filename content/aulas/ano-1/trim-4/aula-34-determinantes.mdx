<Porta nivel="formal" titulo="Cálculo e propriedades">

### 2x2

$$\det \begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc$$

### 3x3 — Regra de Sarrus

$$\det \begin{pmatrix} a & b & c \\ d & e & f \\ g & h & i \end{pmatrix} = aei + bfg + cdh - ceg - bdi - afh$$

(Repita as 2 primeiras colunas à direita, 3 produtos descendentes − 3 ascendentes.)

### n×n — Expansão de Laplace (cofatores)

$$\det A = \sum_{j=1}^n (-1)^{i+j} a_{ij} M_{ij}$$

onde $M_{ij}$ é o **menor** (det da submatriz removendo linha $i$ e coluna $j$). Recursivo: reduz $n \times n$ a soma de $(n-1) \times (n-1)$.

### Definição via permutações (Leibniz)

$$\det A = \sum_{\sigma \in S_n} \text{sgn}(\sigma) \prod_{i=1}^n a_{i,\sigma(i)}$$

Soma sobre todas as $n!$ permutações.

### Propriedades

| # | Propriedade |
|---|---|
| 1 | $\det(I) = 1$ |
| 2 | $\det(A^T) = \det(A)$ |
| 3 | $\det(AB) = \det(A) \cdot \det(B)$ (Cauchy-Binet) |
| 4 | $\det(\alpha A) = \alpha^n \det(A)$ para $A_{n\times n}$ |
| 5 | $\det(A^{-1}) = 1/\det(A)$ |
| 6 | Trocar 2 linhas/colunas inverte o sinal |
| 7 | Linha de zeros ⟹ $\det = 0$ |
| 8 | Linhas iguais ⟹ $\det = 0$ |
| 9 | Linhas proporcionais ⟹ $\det = 0$ |
| 10 | Somar múltiplo de uma linha em outra **não altera** $\det$ |
| 11 | Multiplicar uma linha por $\alpha$ multiplica $\det$ por $\alpha$ |
| 12 | Triangular: $\det = $ produto da diagonal |

### Interpretação geométrica

- $|\det A|$ = volume do paralelepípedo gerado pelas colunas de $A$.
- $\det A > 0$: orientação preservada. $\det A < 0$: orientação invertida.
- $\det A = 0$: colunas linearmente dependentes (paralelepípedo "achatado").

### Critério de invertibilidade

$A$ invertível $\iff \det A \neq 0$.

Imagina que sua matriz é uma **caixa de papelão** de brinquedo. O **determinante** é só o **tamanho da caixa** — quanto cabe dentro.

Se a caixa for **chata** como um papel (achatada), o tamanho é zero — não cabe nada.

Tem caixa que vira "do avesso" quando você multiplica — aí o tamanho fica negativo. É só uma forma de dizer que a caixa virou.

<Porta nivel="10" titulo="Pra criança de 10 anos">

Determinante é um número que mede o **tamanho** da matriz, mais ou menos:
- Para matriz $2 \times 2$, é a **área** do paralelogramo formado pelas colunas.
- Para $3 \times 3$, é o **volume** do paralelepípedo.

**Conta direta para 2×2**: $\det\begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc$. Multiplica diagonal principal, subtrai diagonal secundária.

**Para 3×3 use Sarrus**: copia as 2 primeiras colunas à direita, soma os 3 produtos descendentes, subtrai os 3 ascendentes.

**Coisas legais que o determinante diz**:
- $\det = 0$ → matriz é "achatada", sem inversa.
- $\det \neq 0$ → matriz tem inversa.
- $\det$ negativo → a matriz "vira" o espaço.

**Aplicação**: a regra de Cramer dá solução de sistema usando determinantes — bonito, mas demora muito pra sistemas grandes.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Determinante é um **número** que se calcula a partir de uma matriz quadrada.

Para 2x2:
$$\det \begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc$$

Para 3x3 use **Sarrus**:
$$\det\begin{pmatrix} a & b & c \\ d & e & f \\ g & h & i \end{pmatrix} = aei + bfg + cdh - ceg - bdi - afh$$

Para 4×4 ou maior, use **Laplace** (expansão por cofatores) ou escalonamento (mais eficiente).

**Por que importa?**

1. **Invertibilidade**: $\det A \neq 0$ ⟺ $A$ é invertível.
2. **Cramer**: solução de $A\mathbf{x} = \mathbf{b}$ usa razões de determinantes.
3. **Área e volume**: para 2x2, $|\det A|$ é a **área** do paralelogramo formado pelas colunas. Para 3x3, é o **volume**.

**Geometricamente**: se as colunas de $A$ são "vetores que apontam pra mesma direção", o paralelogramo é fino e $\det A$ é pequeno. Se são perpendiculares, paralelogramo "gordo" e $\det A$ grande.

**Propriedades essenciais pra decorar**:
- $\det(AB) = \det A \cdot \det B$ — multiplicativo!
- Trocar duas linhas inverte o sinal.
- Linha de zeros ou duas linhas iguais ⟹ $\det = 0$.
- Triangular: produto da diagonal.

**Dica de cálculo**: pra matrizes grandes, escalone até triangular — propriedade 10 mantém o determinante. Depois multiplique a diagonal.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Determinante é o **único** funcional multilinear, alternado, que vale 1 na identidade. Esta caracterização axiomática (Weierstrass) define $\det$ unicamente.

### Axiomas (Weierstrass)

1. **Multilinear** nas linhas (e colunas).
2. **Alternado**: trocar duas linhas inverte o sinal.
3. **Normalizado**: $\det I = 1$.

Tudo o resto segue.

### Demonstração: $\det(AB) = \det A \det B$

Fixe $B$. Defina $f(A) = \det(AB)$. Então $f$ é multilinear e alternado nas linhas de $A$. Logo $f(A) = c \det A$ para algum $c$. Tome $A = I$: $f(I) = \det B = c$. Logo $\det(AB) = \det A \det B$.

### Cofatores e adjunta

$\text{adj}(A) = (C_{ij})^T$ onde $C_{ij} = (-1)^{i+j} M_{ij}$. Identidade fundamental:
$$A \cdot \text{adj}(A) = \det(A) \cdot I$$

Logo $A^{-1} = \frac{1}{\det A} \text{adj}(A)$ — útil teoricamente, péssimo numericamente.

### Cross-domínio

- **CG**: jacobiano $J = \det\frac{\partial(u,v)}{\partial(x,y)}$ corrige áreas em mudança de coordenadas. Integrais de superfície/volume.
- **Mecânica**: jacobiano de transformação corpo→referencial mede deformação.
- **Quântica**: determinante de Slater para férmions (antissimetria).
- **EDOs lineares**: Wronskiano $W(t) = \det\begin{pmatrix} y_1 & y_2 \\ y_1' & y_2' \end{pmatrix}$ — independência de soluções.
- **Sistemas dinâmicos**: $\det A$ = produto dos autovalores. Sinal de $\det$ relaciona-se a estabilidade.

### Volume orientado

Em $\mathbb R^n$, $|\det A|$ = volume $n$-dimensional gerado pelas colunas. Via SVD, $|\det A| = \prod \sigma_i$ (singular values).

### Cálculo numérico

- **Laplace** $O(n!)$ — só p/ pequenos $n$.
- **Via LU**: $\det A = \pm \prod u_{ii}$ ($O(n^3)$). **Padrão.**
- Cuidado com underflow para $n$ grande: use $\log|\det A|$.

### Conexão com autovalores

$$\det A = \prod_i \lambda_i, \quad \text{tr}(A) = \sum_i \lambda_i$$

Caracterizam matriz mais que aparenta. Em $2\times 2$: $\lambda_{1,2} = \frac{\text{tr} \pm \sqrt{\text{tr}^2 - 4\det}}{2}$.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

Determinante é o **único** invariante de matriz que generaliza naturalmente a **forma volume** em geometria diferencial — via $\det$ define-se a forma de volume canônica $dx_1 \wedge \cdots \wedge dx_n$ em $\mathbb R^n$.

### Visão tensorial

$\det A = \frac{1}{n!}\epsilon^{i_1\cdots i_n} \epsilon_{j_1 \cdots j_n} A^{j_1}_{i_1} \cdots A^{j_n}_{i_n}$ — contração com Levi-Civita. Define $\det$ em qualquer espaço sobre qualquer corpo (até característica positiva, com ressalvas).

### Wolfram tour

```mathematica
A = {{1, 2, 3}, {4, 5, 6}, {7, 8, 10}};
Det[A]
Minors[A, 2]                    (* todos os menores 2x2 *)
Cofactor[A, {1, 1}]
Eigenvalues[A]
```

### Identidade de Jacobi

$$\frac{d}{dt}\det A(t) = \det A(t) \cdot \text{tr}\left(A(t)^{-1} \dot A(t)\right)$$

Crucial em mecânica de meios contínuos (taxa de mudança de volume), processos estocásticos, fluxos hamiltonianos.

### Cauchy-Binet generalizado

Para $A_{m\times n}, B_{n\times m}$ com $m \leq n$:
$$\det(AB) = \sum_{S \in \binom{[n]}{m}} \det(A_{[m],S}) \det(B_{S,[m]})$$

Aparece em estatística (matriz de Gram) e física combinatória.

### Quant/finance — riscos e copulas

- $\det(\Sigma)$: medida de variância total agregada de portfolio.
- Mudança de medida via Radon-Nikodym usa jacobianos.
- Copulas multidimensionais: derivada parcial mista = $\det$ de Hessiana relacionada.

### Computação moderna

- **Determinant via Schur complement**: para blocos $\begin{pmatrix} A & B \\ C & D \end{pmatrix}$, $\det = \det(A) \det(D - CA^{-1}B)$.
- **Log-determinante** para evitar overflow: $\log\det = \sum \log|u_{ii}|$ via LU. Padrão em ML (priors gaussianos).

### Pesquisa e aplicações modernas

- **Determinantal point processes (DPPs)**: $P(S) \propto \det(K_S)$. Modela diversidade em recommender systems, quantum.
- **Computational complexity**: classe $\#P$ de Valiant — permanente é hard, determinante é fácil. Distinção fundamental.
- **Pfaffian**: $\text{Pf}(A)^2 = \det(A)$ para antissimétrica — combinatória de matchings perfeitos.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Workflow para calcular determinantes.**

1. **2×2**: $ad - bc$ direto. 4 multiplicações + 1 subtração.
2. **3×3**: Sarrus. Mais rápido que Laplace.
3. **n×n** (n ≥ 4): escalonamento → triangular → produto da diagonal.
4. **Numericamente**: use LU; nunca implemente Laplace em produção.

### NumPy receita

```python
import numpy as np

A = np.array([[1, 2, 3], [4, 5, 6], [7, 8, 10]], dtype=float)

# Determinante
print(np.linalg.det(A))        # via LU internamente

# Log-determinante (estável)
sign, logdet = np.linalg.slogdet(A)
print(sign * np.exp(logdet))   # equivalente a det

# Para matriz mal-condicionada, prefira slogdet
```

### SymPy simbólico

```python
import sympy as sp
A = sp.Matrix([[1, 2], [3, 4]])
print(A.det())                 # forma fechada exata
```

### Cramer pra 2×2 explícito

```python
def cramer_2x2(A, b):
    a, B, c, d = A[0, 0], A[0, 1], A[1, 0], A[1, 1]
    det = a * d - B * c
    if abs(det) < 1e-12:
        raise ValueError("Sistema singular")
    x = (b[0] * d - B * b[1]) / det
    y = (a * b[1] - b[0] * c) / det
    return np.array([x, y])
```

### Pegadinhas comuns

- **`np.linalg.det` retorna float64 mesmo para matriz inteira** — pode ter erros de arredondamento. Use SymPy para exato.
- **Determinante grande** pode ter overflow — use `slogdet`.
- **Cramer não escala**: $O(n^4)$ se ingenuamente, $O(n^3)$ otimizado, vs. Gauss $O(n^3)$ direto.
- **Sarrus** **NÃO funciona** para $4\times 4$ ou maior — só para $3\times 3$.
