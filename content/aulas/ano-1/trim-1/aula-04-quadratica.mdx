### Raízes — Fórmula de Bhaskara

<Equation numero="1" explicacao="A famosa fórmula resolutiva. Δ = b² − 4ac é o discriminante. Δ > 0: duas raízes reais distintas. Δ = 0: uma raiz dupla. Δ < 0: nenhuma raiz real (duas complexas conjugadas).">
{`x = \\frac{-b \\pm \\sqrt{b^2 - 4ac}}{2a}`}

### Vértice

<Equation numero="2" explicacao="Coordenada x do vértice — ponto crítico (máximo se a < 0, mínimo se a > 0). Demonstra-se completando o quadrado, ou pela média das raízes.">
{`x_V = -\\frac{b}{2a}, \\quad y_V = f(x_V) = -\\frac{\\Delta}{4a}`}
</Equation>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

A função quadrática $f(x) = ax^2 + bx + c$ é a primeira função "curva" que a gente estuda formalmente. O gráfico é uma **parábola** — abre pra cima se $a > 0$, abre pra baixo se $a < 0$.

**Três coisas pra saber:**

1. **Raízes**: onde a parábola cruza o eixo x. A fórmula de Bhaskara dá:
   $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

2. **Vértice**: o ponto mais alto (se $a < 0$) ou mais baixo (se $a > 0$). Coordenada x: $x_V = -b/(2a)$.

3. **Concavidade**: $a > 0$ é "sorrindo" (cup), $a < 0$ é "triste" (cap).

A trajetória de uma bola arremessada é uma parábola. A antena parabólica tem essa forma. O farol do carro também. **Por que parábolas aparecem em tantos lugares?** Porque elas têm uma propriedade geométrica: todo raio paralelo ao eixo é refletido para o foco. (Ou inverso: raios saindo do foco saem paralelos.) Você verá isso em Cálculo 3.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

A quadrática é **muito** mais que pré-cálculo:

- Equações diferenciais de 2ª ordem com coeficientes constantes (Trim 10) reduzem-se a equação **característica** quadrática $\lambda^2 + b\lambda + c = 0$.
- Otimização sem restrição de funções quadráticas é **convex programming** trivial — base da regressão.
- Fórmula de Bhaskara generaliza para Cardano (3.º grau) e Ferrari (4.º grau). Galois (1832) provou que não há fórmula geral para grau ≥ 5 com radicais.
- Discriminante $\Delta = b^2 - 4ac$ controla se as soluções são reais ou complexas. Aparece em mecânica vibratória: $\Delta > 0$ é amortecimento supercrítico (sem oscilação), $\Delta = 0$ é crítico, $\Delta < 0$ é subamortecido (oscila).

### Demonstração de Bhaskara (completar quadrado)

$ax^2 + bx + c = 0 \Rightarrow x^2 + \frac{b}{a}x = -\frac{c}{a}$

Soma $\left(\frac{b}{2a}\right)^2$ aos dois lados:

$\left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a^2}$

Tirando raiz: $x + \frac{b}{2a} = \pm\frac{\sqrt{b^2 - 4ac}}{2a}$

Logo $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$. ∎
