<Porta nivel="formal" titulo="Definição e fórmulas">

<Definicao titulo="Progressão Aritmética (PA)">
Sequência $(a_n)$ tal que $a_{n+1} - a_n = r$ (constante) para todo $n$. $r$ chama-se **razão**.
</Definicao>

### Termo geral

$$a_n = a_1 + (n-1) r$$

Demonstração por indução em $n$. Equivalente a $a_n = a_p + (n-p) r$ para qualquer $p$.

### Soma dos $n$ primeiros termos

$$S_n = \sum_{k=1}^n a_k = \frac{n(a_1 + a_n)}{2} = \frac{n(2a_1 + (n-1)r)}{2}$$

**Demonstração** (Gauss criança, $\sim$1789): escreva $S_n$ duas vezes, uma em ordem crescente, outra decrescente:

$$\begin{aligned} S_n &= a_1 + a_2 + \ldots + a_n \\ S_n &= a_n + a_{n-1} + \ldots + a_1 \end{aligned}$$

Somando termo a termo: $2 S_n = n(a_1 + a_n)$, pois cada par soma $a_1 + a_n$. ∎

### Propriedades

- **Média aritmética**: três termos consecutivos satisfazem $a_n = (a_{n-1} + a_{n+1})/2$.
- **Crescente** se $r > 0$, **decrescente** se $r < 0$, **constante** se $r = 0$.
- **Soma dos extremos** = soma dos termos equidistantes dos extremos: $a_1 + a_n = a_2 + a_{n-1} = \ldots$.

### Exemplo numérico

PA com $a_1 = 5$, $r = 3$: termos $5, 8, 11, 14, 17, \ldots$

$a_{20} = 5 + 19 \cdot 3 = 62$. $S_{20} = 20 \cdot (5 + 62)/2 = 670$.

### Soma de potências (preview)

$$\sum_{k=1}^n k = \frac{n(n+1)}{2}, \quad \sum_{k=1}^n k^2 = \frac{n(n+1)(2n+1)}{6}, \quad \sum_{k=1}^n k^3 = \left(\frac{n(n+1)}{2}\right)^2$$

</Porta>

<Porta nivel="5" titulo="Pra criança de 5 anos">

Andando uma escada com **degraus iguais**: 1 degrau, 2 degraus, 3 degraus... Cada degrau **soma a mesma altura**.

PA é assim: você começa num número e **soma sempre a mesma quantidade** pra ir pro próximo. Tipo 2, 4, 6, 8, 10 (sempre soma 2). Ou 5, 8, 11, 14 (sempre soma 3).

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

PA é uma lista de números onde, do primeiro pro segundo, do segundo pro terceiro, etc., a diferença é **sempre a mesma**. Essa diferença se chama **razão**.

- $1, 4, 7, 10, 13$ — razão 3.
- $20, 17, 14, 11$ — razão $-3$ (descendo).
- $0, 5, 10, 15, 20$ — razão 5.

**Quanto é o $n$-ésimo termo?** $a_n = a_1 + (n-1) r$.

Tipo: $1, 4, 7, 10, \ldots$ Qual o 100º? $a_{100} = 1 + 99 \cdot 3 = 298$.

**Soma da PA**: tem fórmula esperta. $S_n = n(a_1 + a_n)/2$.

História engraçada: o **Carl Friedrich Gauss aos 8 anos** o professor mandou somar 1 até 100 pra ocupar a turma. Em 30 segundos ele entregou: $5050$. Ele juntou $1 + 100 = 101$, $2 + 99 = 101$, ..., 50 pares de $101$. Total: $50 \cdot 101 = 5050$.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

PA é uma lista onde cada número é o anterior **mais um valor fixo**. O valor fixo se chama **razão**.

- $1, 4, 7, 10, 13, \ldots$ — PA de razão 3.
- $20, 17, 14, 11, 8, \ldots$ — PA de razão $-3$ (decrescente).
- $0, 5, 10, 15, 20, \ldots$ — PA de razão 5.

**Termo geral**: $a_n = a_1 + (n-1) r$. Tipo "qual o 50.º termo?": pega $a_1$, soma $49$ vezes a razão.

**Soma da PA**: $S_n = n(a_1 + a_n)/2$. Truque genial de **Gauss aos 8 anos**: pra somar $1+2+3+\ldots+100$, ele juntou em pares ($1+100, 2+99, 3+98, \ldots$) — cada par dá $101$, e tem $50$ pares. Total: $5.050$. **Sem somar tudo um por um.**

**Onde aparece?** Sapato 32, 33, 34, 35, ... (PA). Andares de prédio. Datas em calendário. Salário com aumento fixo todo ano (R$ 3.000, R$ 3.300, R$ 3.600, ...).

**Truque útil**: três termos consecutivos $a, b, c$ formam PA $\iff$ $b = (a+c)/2$ (média aritmética). Por isso "aritmética" no nome!

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

PA é a recorrência mais simples. Aparece como **base** em:

- **Soma de Riemann em escada**: aproximação de integral via somas finitas — somatório com índice em PA.
- **Discretização de tempo**: amostragem regular $t_n = t_0 + n \Delta t$ é PA.
- **Métodos numéricos**: malhas regulares são PAs em coordenadas.
- **Criptografia clássica**: cifras de César deslocam letras em PA.

### Demonstração rigorosa do termo geral

Por indução em $n$. Caso base $n = 1$: $a_1 = a_1 + 0 \cdot r$. ✓ Passo: suponha $a_n = a_1 + (n-1) r$. Como $a_{n+1} = a_n + r$, então $a_{n+1} = a_1 + (n-1) r + r = a_1 + n r = a_1 + ((n+1)-1) r$. ∎

### Demonstração da soma

A demonstração de Gauss já mostrada é equivalente a:

$$S_n = \sum_{k=1}^n [a_1 + (k-1)r] = n a_1 + r \sum_{k=1}^n (k-1) = n a_1 + r \cdot \frac{n(n-1)}{2}$$

### Soma de quadrados, cubos, etc.

PA não cobre tudo. Para somas $\sum n^2$ ou $\sum n^3$, fórmulas distintas (indução):

$$\sum_{k=1}^n k = \frac{n(n+1)}{2}, \quad \sum_{k=1}^n k^2 = \frac{n(n+1)(2n+1)}{6}, \quad \sum_{k=1}^n k^3 = \left(\frac{n(n+1)}{2}\right)^2$$

### Aplicações em DSP

- **Onda dente-de-serra**: amplitude muda linearmente — caracterizada por PA dentro de cada período.
- **Filtros FIR**: coeficientes em janela retangular formam constante; janela de Hamming os modula.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

PA é caso particular de **série aritmética** — a soma de PA é polinômio de grau 2 em $n$. Generalização: **somatórios polinomiais** $\sum k^m$ são polinômios de grau $m+1$ em $n$ (Faulhaber, números de Bernoulli).

### Faulhaber e Bernoulli

$$\sum_{k=1}^n k^m = \frac{1}{m+1} \sum_{j=0}^m \binom{m+1}{j} B_j n^{m+1-j}$$

onde $B_j$ são números de Bernoulli. Para $m = 1$ recupera Gauss.

### Aplicações modernas

- **Matemática combinatória**: sumação simbólica via algoritmo de Gosper / Zeilberger automatiza somas hipergeométricas (estende PA/PG).
- **Números $p$-ádicos**: sequências aritméticas em $\mathbb{Z}/p\mathbb{Z}$ são fundamentais em teoria dos números (Dirichlet: existem infinitos primos em PA $\{a + nd\}$ se $\gcd(a, d) = 1$).
- **Mathematica `Sum`**: avalia somas em forma fechada quando possível.

```python
import sympy as sp
n, k = sp.symbols('n k', integer=True, positive=True)
print(sp.summation(k, (k, 1, n)))    # n*(n+1)/2
print(sp.summation(k**2, (k, 1, n))) # n*(n+1)*(2n+1)/6
print(sp.summation(k**3, (k, 1, n))) # (n*(n+1)/2)**2
```

### Pitfalls

- **Off-by-one**: $\sum_{k=1}^n$ vs $\sum_{k=0}^{n-1}$ — diferem por inclusão/exclusão de $a_n$.
- **Soma vazia**: $\sum_{k=a}^{a-1}$ vale $0$ por convenção.
- **Numerical accumulation**: somar $10^6$ termos em float pode acumular erro $\sim 10^{-10}$. Usar **Kahan summation** se precisão crítica.

</Porta>

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita: dada uma PA, encontre tudo.**

1. **Identifique $a_1$ e $r$**.
2. **Termo geral**: $a_n = a_1 + (n-1) r$.
3. **Soma**: $S_n = n(a_1 + a_n)/2$.
4. **Confira** com 2 termos calculados manualmente.

### Workflow Wolfram / SymPy

```python
import sympy as sp
n, k = sp.symbols('n k', integer=True, positive=True)
a1, r = sp.symbols('a_1 r', real=True)

# Termo geral
a_n = a1 + (n-1)*r

# Soma
S_n = sp.summation(a1 + (k-1)*r, (k, 1, n))
print(sp.simplify(S_n))   # n*(2*a1 + (n-1)*r)/2

# Exemplo: PA 5, 8, 11, ... (a1=5, r=3), S_20:
print(S_n.subs({a1: 5, r: 3, n: 20}))   # 670
```

Wolfram Alpha: `sum_{k=1}^{100} (3 + 2(k-1))` ou `arithmetic series first term 5 common difference 3 sum 20 terms`.

### Quando preocupar?

- **PA "decrescente"**: $r < 0$. Não confunda com "decrescente" no sentido de tender a zero.
- **Soma negativa**: pode acontecer se $a_n < 0$ para vários termos. Verifique sinais.
- **Em problemas reais**: salários, juros simples, distância acumulada por velocidade constante. PA modela bem cenários **lineares**.

### O exercício mais importante

17.21 (poupança crescente), 17.23 (queda livre via PA), e a demonstração 17.31 — essenciais.
