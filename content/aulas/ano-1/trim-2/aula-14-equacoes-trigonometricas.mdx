### Equações com seno

Para $\sin x = a$ com $a \in [-1, 1]$:
$$x = \arcsin a + 2k\pi \quad \text{ou} \quad x = \pi - \arcsin a + 2k\pi, \quad k \in \mathbb{Z}$$

Onde $\arcsin: [-1, 1] \to [-\pi/2, \pi/2]$ é a inversa principal.

### Equações com cosseno

Para $\cos x = a$ com $a \in [-1, 1]$:
$$x = \pm \arccos a + 2k\pi, \quad k \in \mathbb{Z}$$

Onde $\arccos: [-1, 1] \to [0, \pi]$.

### Equações com tangente

Para $\tan x = a$ com $a \in \mathbb{R}$:
$$x = \arctan a + k\pi, \quad k \in \mathbb{Z}$$

Onde $\arctan: \mathbb{R} \to (-\pi/2, \pi/2)$.

### Receita geral

Para resolver $f(x) = 0$ trigonométrica:

1. **Reduzir** a equações básicas $\sin x = a$, $\cos x = a$, $\tan x = a$ via identidades.
2. **Listar** todas as soluções no intervalo $[0, 2\pi)$ (ou $[-\pi, \pi]$).
3. **Generalizar** somando $2k\pi$ (ou $k\pi$ para tangente).

### Tipos de equações reduzíveis

| Tipo | Exemplo | Substituição |
|---|---|---|
| Polinomial em $\sin$ | $2 \sin^2 x - 3 \sin x + 1 = 0$ | $u = \sin x$ |
| Polinomial em $\cos$ | $\cos^2 x = 1/2$ | $u = \cos x$ |
| $\sin x \pm \cos x$ | $\sin x + \cos x = 1$ | $\sqrt 2 \sin(x + \pi/4) = 1$ |
| Duplo ângulo | $\sin 2x = \sin x$ | $2 \sin x \cos x = \sin x$ |
| Soma-produto | $\sin x + \sin 3x = 0$ | $2 \sin 2x \cos x = 0$ |

<Porta nivel="5" titulo="Pra criança de 5 anos">

Imagine um relógio. Em quais momentos do dia o ponteiro grande está apontando para o **6** (lá embaixo)? Resposta: **6:00 e 18:00** (e também 30:00 se quiser mais voltas, mas não tem 30:00 no dia normal).

Equação trigonométrica é assim: você quer saber **em quais ângulos** o seno (ou cosseno) tem certo valor. E a resposta é uma **lista** de ângulos, porque a roda gira sempre.

</Porta>

<Porta nivel="10" titulo="Pra criança de 10 anos">

A equação $\sin x = 1/2$ não é como $x + 3 = 5$ (uma resposta só). Ela tem **infinitas respostas** porque seno se repete.

Imagine o círculo trigonométrico. Quais pontos têm $y = 1/2$? **Dois pontos**: um no primeiro quadrante (ângulo $\pi/6$) e outro no segundo (ângulo $5\pi/6$). E como você pode girar o círculo quantas vezes quiser, cada um desses dois pontos vira **infinitos**: $\pi/6, \pi/6 + 2\pi, \pi/6 + 4\pi, \ldots$

Por isso a resposta de equação trigonométrica vem em **conjunto**:
$$x = \pi/6 + 2k\pi \quad \text{ou} \quad x = 5\pi/6 + 2k\pi$$

para $k$ inteiro qualquer.

Geralmente o professor pede a resposta em $[0, 2\pi)$ — só uma volta. Aí é fácil: lista as soluções desse intervalo e pronto.

</Porta>

<Porta nivel="15" titulo="Pra adolescente de 15 anos">

Resolver $\sin x = 1/2$ não é tão simples como resolver $x + 3 = 5$.

Por quê? Porque seno se repete o tempo todo. Se $x = \pi/6$ é solução, $x = \pi/6 + 2\pi$ também é. E $x = \pi/6 + 4\pi$. E... infinitas.

Além disso, $\sin(\pi - \pi/6) = \sin(5\pi/6) = 1/2$ também. Então em $[0, 2\pi)$ tem **duas** soluções: $\pi/6$ e $5\pi/6$.

**Receita:**
1. Acha as soluções no círculo trigonométrico, em $[0, 2\pi)$.
2. Adiciona $2k\pi$ (todas as voltas inteiras).

Para tangente é diferente: período de $\tan$ é $\pi$ (não $2\pi$), porque $\tan(x + \pi) = \tan x$.

**Equações mais elaboradas** reduzem a básicas via identidades:
- $\cos 2x = \cos^2 x - \sin^2 x = 1 - 2 \sin^2 x$.
- $\sin 2x = 2 \sin x \cos x$.
- $\sin x + \cos x = \sqrt 2 \sin(x + \pi/4)$.

**Inequações**: idem mas com intervalo. $\sin x > 1/2$ em $[0, 2\pi)$? Os $x$ em que $\sin x$ está acima de $1/2$ — pelo gráfico, isso é $(\pi/6, 5\pi/6)$.

</Porta>

<Porta nivel="25" titulo="Pra estudante de engenharia">

Em prática, **não se resolve trigonometria à mão** depois do EM. Usa-se calculadora ou solver. Mas a estrutura das soluções é importante porque controla a forma das respostas em problemas de engenharia.

Em controle, **acoplamento de fase** entre dois sinais é dado por $\arctan$ da razão imaginária/real do quociente das transferências. Em mecânica, ângulo de equilíbrio é raiz de equação trigonométrica.

### Métodos numéricos

Para equações trigonométricas que não se reduzem a básicas (ex.: $x = \cos x$), usa-se **Newton-Raphson** ou **bisseção**. **Não há fórmula fechada**.

```python
from scipy.optimize import brentq
import numpy as np
sol = brentq(lambda x: x - np.cos(x), 0, 1)
print(sol)  # ~0.739085
```

### Soluções complexas (preview)

Para $\sin z = 5$ (impossível em reais), há solução em $\mathbb{C}$ usando Euler:
$$\sin z = (\e^{iz} - \e^{-iz})/(2i) = 5 \Rightarrow z = \arcsin 5 = \pi/2 - i \ln(5 + 2\sqrt 6)$$

### Aplicações em engenharia

- **Equilíbrio de forças**: $\sum F_i \cos \theta_i = 0$ — equação trig em $\theta$.
- **Cinemática inversa**: braço robótico ângulo de junta resolve equação $a \cos\theta + b \sin\theta = c$.
- **Acústica**: ressonância em sala fechada quando $\sin(k_x L_x) = 0$ — modos próprios.
- **Sistemas de controle**: zeros e pólos no círculo unitário — onde $z = \e^{j\omega T}$.

</Porta>

<Porta nivel="40" titulo="Pra profissional sênior">

Equações trigonométricas são caso particular de equações **transcendentes**. Em geral não admitem fórmula fechada — só fórmulas para casos polinomiais em $\sin, \cos, \tan$.

### Estrutura algébrica

O **anel das funções trigonométricas** $\mathbb{R}[\sin x, \cos x]$, módulo $\sin^2 x + \cos^2 x = 1$, é isomorfo a $\mathbb{R}[t, t^{-1}]/(t \bar t = 1)$ via Euler $t = \e^{ix}$. Isto reduz toda equação polinomial em $\sin x, \cos x$ a uma equação algébrica em $t$ no círculo unitário complexo.

```python
import sympy as sp
x = sp.symbols('x', real=True)

# Solução geral simbólica:
sp.solveset(sp.sin(x) - sp.Rational(1,2), x, sp.S.Reals)
# Union(ImageSet({2*pi*n + pi/6, n in Z}),
#       ImageSet({2*pi*n + 5*pi/6, n in Z}))

# Em intervalo:
sp.solveset(sp.sin(x) - sp.Rational(1,2), x, sp.Interval(0, 2*sp.pi))
```

### Padrões avançados

- **Equação de Kepler**: $M = E - e \sin E$ — resolução para órbita elíptica. Newton converge em $\sim 5$ iterações para $e < 0{,}9$.
- **Equação de Bessel**: zeros não-fechados, mas tabuláveis. Aparece em modos de vibração de tambor.
- **Identidades de Chebyshev**: $\cos(n\theta) = T_n(\cos\theta)$ com $T_n$ polinômio de grau $n$ — base de quadratura de Gauss-Chebyshev.

### Pitfalls

- **`np.arcsin` retorna em $[-\pi/2, \pi/2]$**: pra obter solução no quadrante II, usar $\pi - \arcsin a$.
- **Modo de calculadora**: deg vs rad em CFD/FEA — fonte clássica de bug em validação.
- **Soluções perdidas** ao multiplicar/dividir por $\sin x$ ou $\cos x$ (que pode ser zero): sempre conferir se solução adicional existe onde aquela função zera.

### O que vem depois

- Aulas 25-26 (Trim 3) introduzem **vetores e matrizes** — onde rotação por $\theta$ vira álgebra linear.
- Aula 41 (Trim 5) define limite formal — base do cálculo trigonométrico.

<Porta nivel="pratica" titulo="Aplicação prática">

**Receita master para resolver equação trigonométrica:**

1. **Tente reduzir a básica** ($\sin x = a$, $\cos x = a$, $\tan x = a$).
2. **Use identidades** (Pitágoras, soma, duplo ângulo) se necessário.
3. **Substitua** $u = \sin x$ ou $u = \cos x$ se for polinomial.
4. **Resolva** $u$ primeiro, depois volte para $x$.
5. **Liste** soluções no intervalo pedido.
6. **Confira** uma das soluções na equação original.

### Workflow Wolfram / SymPy

```python
import sympy as sp
x = sp.symbols('x', real=True)

# Equação simples:
sp.solveset(sp.sin(x) - sp.cos(x), x, sp.Interval(0, 2*sp.pi))
# {pi/4, 5*pi/4}

# Equação com identidade:
eq = sp.sin(2*x) - sp.sin(x)   # = 2 sin(x) cos(x) - sin(x)
sp.solveset(eq, x, sp.Interval(0, 2*sp.pi))

# Inequação:
sp.reduce_inequalities(sp.sin(x) > sp.Rational(1, 2))   # cuidado: pode falhar
```

Wolfram Alpha: `solve sin(x) + cos(x) = 1, x in [0, 2pi]` retorna $\{0, \pi/2\}$.

### Quando preocupar?

- **Soluções espúrias** ao quadrar: $\sin x = -\cos x$ tem solução, mas $\sin^2 x = \cos^2 x$ tem **mais** soluções (incluindo $\sin x = +\cos x$). Sempre verificar.
- **Ao dividir por $\cos x$**: você perde soluções com $\cos x = 0$. Trate esse caso à parte.
- **`tan` em ponto crítico**: $\tan x$ vai a $\pm \infty$ em $x = \pi/2$ — na inequação, esse ponto **não está** no domínio.
- **Em problemas físicos**, restrinja domínio: ângulos negativos podem não ter sentido.

### O exercício mais importante

Os exercícios 14.21 (sin + cos = 1) e 14.31 (sin x > cos x) — porque combinam identidades, intervalos e inequações simultaneamente.
