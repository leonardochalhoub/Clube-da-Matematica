# i18n — Sistema de internacionalização

## Visão geral

O Clube da Matemática suporta **16 idiomas** com auto-detecção e tradução pré-computada (não em tempo real).

**Idiomas suportados:**
🇧🇷 pt-BR · 🇺🇸 en · 🇪🇸 es · 🇨🇳 zh · 🇯🇵 ja · 🇩🇪 de · 🇫🇷 fr · 🇮🇹 it · 🇷🇺 ru · 🇰🇷 ko · 🇻🇳 vi · 🇵🇱 pl · 🇰🇪 sw · 🇸🇦 ar · 🇮🇳 hi · 🇪🇪 et

## Arquitetura

### Detecção
- Cliente ao montar: `navigator.language` → `localStorage` → fallback inglês
- Mapeamento por país: `BR/PT/AO/MZ` → pt-BR, `US/UK/AU/...` → en, etc.
- Salva preferência em `localStorage['clube-locale']`

### UI strings (curto)
- Definido em `src/lib/i18n/translations.ts`
- Hook `useLocale()` retorna `t(key, fallback)`
- Fallback inteligente: locale específico → en → key

### Content (MDX longos)
- **NÃO traduz em runtime.** Traduções pré-computadas.
- Source: `content/aulas/*.mdx` (PT-BR, fonte da verdade)
- Target: `content/translations/<locale>/aulas/*.mdx`
- Loader detecta locale ativo e carrega arquivo certo se existir.

### Audio (Web Speech API)
- Se há tradução pro locale: usa `lang = de-DE` + voz alemã
- Se NÃO há: cai pra PT-BR + voz PT-BR (sem sotaque artificial)
- Filtra vozes por match exato primeiro, depois prefix

## Como traduzir

### UI strings (rápido — minutos)
1. Adicione a chave nova no `translations.ts` com `pt-BR`
2. Rode `ANTHROPIC_API_KEY=... npx tsx scripts/translate-ui.ts`
3. Cole as traduções no `translations.ts`

### Content (lento — horas, custa $)
```bash
# Por idioma
ANTHROPIC_API_KEY=sk-ant-... npx tsx scripts/translate-content.ts en
ANTHROPIC_API_KEY=sk-ant-... npx tsx scripts/translate-content.ts es
# ...
```

**Custo estimado:** ~$90 por idioma com Claude Opus 4.7. Para 15 idiomas = ~$1350.

**Cache:** o script pula arquivos já existentes em `content/translations/<locale>/`. Pra re-traduzir, delete primeiro.

**Tempo:** ~3 horas por idioma (120 lições × 30s/lição com retries). Roda em paralelo se quiser (ajuste `--concurrency` no script).

## O que é traduzido

✅ UI strings (header, footer, botões)
✅ Lições MDX (texto, portas, exercícios)
✅ Provas (enunciado, passos)
✅ AudioReader (texto falado em voz nativa)

❌ Não traduzido:
- LaTeX matemático (universal)
- URLs e slugs
- Nomes de livros (mantém título original do autor)
- Tags `<Porta>`, `<Exercicio>`, etc. (componentes JSX)

## RTL (right-to-left) para árabe

`document.dir = 'rtl'` é setado automaticamente quando locale = ar.
Tailwind tem suporte nativo via classes `rtl:` quando necessário.

## Detecção por geolocalização (opcional)

Pode ser ativada chamando `https://ipapi.co/json/` ou similar para obter
país. Não implementado por padrão (privacy + custo). `navigator.language`
é suficiente em 95% dos casos.
